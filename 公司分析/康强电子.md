## 二、主营业务

### 核心产品

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **引线框架（Lead Frame）** | 半导体封装的核心材料，连接芯片与外部引脚 | 集成电路、分立器件封装 |
| **键合丝（Bonding Wire）** | 用于芯片内部互连的金丝/铜丝 | 存储芯片、高端IC封装 |
| **电极丝** | 精密加工材料 | 半导体制造设备 |

### 技术路线
- 冲压工艺引线框架
- 蚀刻工艺引线框架（高端细间距、多引脚）
- PRP蚀刻引线框架（进口替代重点）
- QFN封装用环氧模塑材料（已量产）

## 四、与存储芯片的关系

### 产业链定位

```
存储芯片产业链：
设计 → 制造（晶圆厂）→ 【封装测试】→ 终端应用
                          ↑
                    康强电子位置
                   （封装材料供应商）
```

### 具体关联

| 维度 | 说明 |
|------|------|
| **直接关系** | 引线框架、键合丝是存储芯片封装的必需材料 |
| **技术突破** | QFN封装用环氧模塑材料已成功量产，用于存储芯片封装 |
| **国产替代** | 高端PRP蚀刻引线框架项目推进中，可替代进口产品 |
| **市场需求** | 受益于存储芯片扩产、AI算力芯片封装需求增长 |

### 与HBM的关系
- HBM（高带宽存储器）需要先进封装技术
- 公司的高精密引线框架、键合丝可用于先进封装环节
- 属于间接受益标的

## 五、近期股价上涨原因

### 1️⃣ 行业板块热点
- "存储芯片"概念
- "先进封装"概念
- "专精特新"概念
- "芯片国产替代"主题

### 2️⃣ 基本面改善
- 2025年前三季度净利润+21.4%
- 毛利率回升至14%以上
- 经营现金流大幅改善（+463.73%）

### 3️⃣ 资金面推动
- 主力资金阶段性流入
- 游资与散户活跃
- 换手率上升，成交额放大

### 4️⃣ 政策与产业趋势
- 国产替代政策支持
- AI、5G、汽车电子拉动封装材料需求
- 公司参与国家级/省级先进封装项目
