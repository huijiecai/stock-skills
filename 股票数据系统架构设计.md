### 官方文档

| 数据源 | 文档地址 | 说明 |

### 数据源分工

| 数据源 | 优先级 | 核心能力 | 脚本 |
|--------|--------|----------|------|
| **adata** | 主力 | 当日分时、日线、指数、概念、资金流向、股票信息 | `skills/trading-system/tools/fetch_adata_data.py` |
| **Tushare** | 补充 | 涨跌停数据（连板高度）、历史分钟数据 | `skills/trading-system/tools/fetch_tushare_data.py` |

### 数据格式统一

**分时数据格式**（adata & tushare 统一）：
```json
{
  "2026-03-27": [
    {
      "time": "09:30",
      "price": 78.00,
      "change": 7.09,
      "change_pct": 10.00,
      "volume": 10000,
      "amount": 780000,
      "avg_price": 78.50
    }
  ]
}
```

**日线数据格式**（adata & tushare 统一）：
```json
{
  "2026-03-27": {
    "open": 73.33,
    "high": 78.00,
    "low": 72.72,
    "close": 78.00,
    "pre_close": 70.91,
    "change": 7.09,
    "change_pct": 9.99,
    "volume": 421223,
    "amount": 3145292965
  }
}
```

> 注：adata 日线额外包含 `turnover_ratio`（换手率）字段，tushare 无此字段。

### Task 3: 数据采集服务设计

**文件**: `backend/app/services/data_collector.py`, `backend/app/scheduler/`

#### 架构原则

1. **后端自闭环**：数据采集作为后端服务的内置能力，不依赖外部脚本
2. **定时调度**：使用 APScheduler 集成在 FastAPI 中，服务启动即自动调度
3. **查询触发**：支持 lazy loading，查询时若数据不存在则触发采集

#### 目录结构

```
backend/app/
├── scheduler/
│   ├── __init__.py
│   ├── jobs.py           # 定时任务定义
│   └── triggers.py       # 触发器（按日期、按代码等）
├── services/
│   ├── data_collector.py # 数据采集核心逻辑
│   └── data_service.py   # 数据查询服务
└── main.py               # 启动时初始化调度器
```

#### 定时任务配置

```python
# backend/app/scheduler/jobs.py
from apscheduler.schedulers.asyncio import AsyncIOScheduler
from apscheduler.triggers.cron import CronTrigger

scheduler = AsyncIOScheduler()

# 每个交易日 16:30 采集日线数据
scheduler.add_job(
    collect_daily_data,
    CronTrigger(day_of_week='mon-fri', hour=16, minute=30),
    id='daily_data_collector'
)

# 每个交易日 16:35 采集分时数据
scheduler.add_job(
    collect_intraday_data,
    CronTrigger(day_of_week='mon-fri', hour=16, minute=35),
    id='intraday_data_collector'
)

# 每个交易日 16:40 采集概念板块数据
scheduler.add_job(
    collect_concept_data,
    CronTrigger(day_of_week='mon-fri', hour=16, minute=40),
    id='concept_data_collector'
)

# 每周一 09:00 更新概念成分股映射
scheduler.add_job(
    update_concept_mapping,
    CronTrigger(day_of_week='mon', hour=9, minute=0),
    id='concept_mapping_updater'
)
```

```python
async def get_stock_daily(code: str, date: str) -> dict:
    """获取股票日线数据，不存在则触发采集"""
    # 1. 先查数据库
    data = await db.query_stock_daily(code, date)
    if data:
        return data
    
    # 2. 数据不存在，触发采集
    await data_collector.collect_stock_daily(code, date)
    
    # 3. 再次查询返回
    data = await db.query_stock_daily(code, date)
    return data
```

## 四、调度器启动与配置

### 服务启动时初始化

```python
# backend/app/main.py
from contextlib import asynccontextmanager
from apscheduler.schedulers.asyncio import AsyncIOScheduler
from app.scheduler.jobs import scheduler, init_jobs

@asynccontextmanager
async def lifespan(app: FastAPI):
    # 启动时：初始化调度器
    init_jobs()
    scheduler.start()
    yield
    # 关闭时：停止调度器
    scheduler.shutdown()

app = FastAPI(lifespan=lifespan)
```
