# 股票归属验证操作手册

> **目的**：为每只股票的概念归属和标的类型提供标准化的验证流程
> 
> **适用范围**：所有股票池中的162只股票
> 
> **预计工作量**：50分钟/只 × 162只 = 约135小时（分阶段完成）

---

## 📋 快速开始

### 工作文件

1. **参考文档**：`公司分析/概念股票池体系.md`
2. **概念体系**：参考当前已建立的概念树结构

### 工作流程

```
1. 查看当前股票池和概念归属
2. 选择一只股票（优先级：新增龙头 > 核心股 > 关联股）
3. 按照本手册步骤验证
4. 记录验证结论和信息来源
5. 标注需要调整的股票和概念
6. 继续下一只
```

## 🔍 验证步骤详解

### 第一步：信息收集（30分钟）

#### 1.1 查看公司官网（10分钟）

**访问路径**：
```
Google搜索：[股票代码] [股票名称] 官网
或
直接访问：http://www.[公司简称].com
```

**需要收集的信息**：
- [ ] 公司简介 → 核心业务是什么？
- [ ] 产品中心 → 有哪些主要产品？
- [ ] 业务板块 → 如何划分业务线？
- [ ] 技术实力 → 有什么核心技术/专利？
- [ ] 客户案例 → 主要客户是谁？

**示例：002230 科大讯飞**
```
官网：https://www.iflytek.com/
核心业务：
  ✓ 智能语音 → 匹配"AIGC应用"
  ✓ 讯飞星火大模型 → 匹配"AIGC应用"
  ✓ 教育、医疗AI → 垂直场景
  
结论：归入"AIGC应用"合理 ✅
```

#### 1.2 阅读最新年报（15分钟）

**获取渠道**：
- 巨潮资讯网：http://www.cninfo.com.cn
- 公司官网：投资者关系 → 定期报告
- 东方财富：个股页面 → 财务分析 → 财务报表

**重点查看章节**：
1. **第三节 管理层讨论与分析**
   - [ ] 主营业务分析 → 各业务营收占比
   - [ ] 行业地位 → 市占率、排名
   - [ ] 核心竞争力 → 技术优势

2. **第四节 公司财务状况**
   - [ ] 营业收入构成 → 按产品/地区分类
   - [ ] 毛利率分析 → 与同行对比

**关键数据记录**：
```
公司：[股票名称]
报告期：2025年年报

主营业务构成：
  - XX业务：XX亿元（XX%）
  - XX业务：XX亿元（XX%）
  
行业地位：
  - 市占率：XX%（行业第X）
  - 竞争对手：XXX、XXX
  
毛利率：XX%（行业平均XX%）
```

#### 1.3 搜索券商研报（5分钟）

**搜索渠道**：
- 东方财富研报：https://data.eastmoney.com/report/
- 慧博投研：http://www.hibor.com.cn
- 萝卜投研：https://robo.datayes.com

**搜索关键词**：
```
[股票名称] + 研报
[股票名称] + 行业地位
[股票名称] + [概念名称]
```

**需要提取的信息**：
- [ ] 研报标题 → 券商如何定位该公司？
- [ ] 投资逻辑 → 核心看点是什么？
- [ ] 行业对比 → 与同行比较如何？

**示例**：
```
科大讯飞 研报摘要：
  • "AI语音龙头" ← 多篇研报提及
  • "星火大模型追赶GPT" ← 核心看点
  • "教育+医疗双轮驱动" ← 应用场景
  
结论：AIGC应用核心标的 ✅
```

---

### 第二步：逻辑验证（15分钟）

#### 2.1 主营业务匹配度检查

**判断标准**：

| 营收占比 | 匹配度 | 是否合理 |
|---------|--------|---------|
| > 50% | 高度匹配 | ✅ 合理 |
| 30-50% | 中度匹配 | ⚠️ 可接受 |
| 10-30% | 低度匹配 | ⚠️ 建议标记为"相关标的" |
| < 10% | 几乎不匹配 | ❌ 不合理，需调整 |

**示例：300450 先导智能**
```
当前归属：锂电池
实际业务：锂电池生产设备制造

问题：混淆了"设备"和"电池"
建议：调整到"锂电设备"（需新建概念）

结论：当前归属不合理 ❌
```

#### 2.2 行业地位检查

**核心标的判定**（满足任意2项）：
- [ ] 市占率前3名
- [ ] 细分领域龙头（研报明确提及）
- [ ] 拥有核心技术壁垒
- [ ] 营收规模行业前5

**相关标的判定**：
- [ ] 有相关业务但不是主营
- [ ] 市占率排名靠后（5名以外）
- [ ] 正在布局转型中

**示例：601689 拓普集团**
```
当前标记：相关标的（机器人零部件）
实际情况：
  • 汽车底盘系统龙头
  • 智能驾驶领域布局
  • 配套特斯拉等头部车企

问题：实际是汽车零部件龙头，非机器人
建议：
  方案1：移除出机器人板块
  方案2：新建"智能驾驶"概念

结论：概念归属待调整 ⚠️
```

#### 2.3 技术/产品能力检查

**核心问题**：
- [ ] 该公司的核心产品是什么？
- [ ] 这些产品是否真正属于该概念？
- [ ] 技术水平如何？（领先/追赶/落后）

**红旗信号**（需警惕）：
- ⚠️ 公司刚宣布进入该领域（蹭热点？）
- ⚠️ 该业务尚未产生营收
- ⚠️ 官网/年报都没有详细介绍
- ⚠️ 研报对该业务一笔带过

---

### 第三步：结论记录（5分钟）

#### 3.1 记录验证结果

**建议记录的信息**：

1. **主营业务**：用一句话概括
   ```
   示例：AI语音识别技术及大模型应用
   ```

2. **营收占比**：该业务占比
   ```
   示例：AI业务 65%，教育业务 25%，其他 10%
   ```

3. **行业排名**：市占率或排名
   ```
   示例：AI语音市占率70%，行业第1
   ```

4. **归属逻辑**：为什么归入该概念？（核心）
   ```
   示例：
   • 讯飞星火大模型国内前3
   • AI语音技术龙头
   • AIGC工具矩阵完整
   → 归入"AIGC应用"合理且为核心标的
   ```

5. **信息来源**：列出3个主要来源
   ```
   示例：
   • 科大讯飞官网-产品中心
   • 2025年年报-管理层分析(P23-25)
   • 中金研报"科大讯飞：星火大模型加速"(2025-12-20)
   ```

6. **调整建议**：如果需要调整
   ```
   示例（不合理时）：
   • 当前：锂电池（相关标的）
   • 建议：移至"锂电设备"（核心标的）
   • 原因：主营锂电生产设备，非电池本身
   ```

**记录方式**：
- 在 `公司分析/概念股票池体系.md` 中更新对应股票的验证状态
- 或创建独立的验证记录文档（如 `公司分析/验证记录/002230_科大讯飞.md`）

#### 3.2 置信度评分（可选）

给自己的验证结果打分：

| 星级 | 含义 | 何时使用 |
|------|------|---------|
| ⭐⭐⭐⭐⭐ | 非常确定 | 官网+年报+研报信息一致，逻辑清晰 |
| ⭐⭐⭐⭐ | 比较确定 | 有2个以上可靠信息源 |
| ⭐⭐⭐ | 基本确定 | 信息充足但有小疑问 |
| ⭐⭐ | 不太确定 | 信息不足或矛盾 |
| ⭐ | 存疑 | 需要二次确认 |

## 🎯 常见问题处理

### Q1：找不到该公司的官网怎么办？

**解决方案**：
1. 尝试搜索：[股票代码].com / [公司拼音].com
2. 查看年报第二节"公司基本情况" → 公司网址
3. 如果确实没有官网（罕见），标注并跳过此步骤

### Q2：年报太长，如何快速找到关键信息？

**快捷方法**：
1. 使用PDF搜索功能（Ctrl+F）
2. 搜索关键词：
   - "主营业务"
   - "营业收入构成"
   - "行业地位"
   - [概念相关关键词]，如"锂电池"、"AI"等

### Q3：研报需要付费怎么办？

**免费替代方案**：
1. 东方财富股吧 → 用户分享的研报
2. 雪球 → 搜索该股票，查看专业用户分析
3. 公司公告 → 机构调研记录（问答环节有价值信息）
4. 百度资讯 → 搜索公司新闻

### Q4：营收占比数据找不到怎么办？

**估算方法**：
1. 查看年报"营业收入构成"表格
2. 如果按产品分类，直接使用
3. 如果只按地区分类，看"主营业务分析"文字描述
4. 实在找不到，标注"未披露"

### Q5：如何判断"核心标的"还是"相关标的"？

**决策树**：
```
该业务营收占比 > 30%？
  ├─ 是 → 行业排名前3？
  │      ├─ 是 → ✅ 核心标的
  │      └─ 否 → 营收规模大？
  │             ├─ 是 → ✅ 核心标的
  │             └─ 否 → ⚪ 相关标的
  └─ 否 → ⚪ 相关标的
```

### Q6：遇到跨界公司怎么处理？

**示例：比亚迪（002594）**
- 新能源汽车（主营）
- 锂电池（核心业务）
- 半导体（子公司比亚迪半导体）

**处理方法**：
1. 可以归入多个概念（这是合理的）
2. 主营业务归为"核心标的"
3. 重要但非主营归为"核心标的"
4. 布局中的业务归为"相关标的"

```
比亚迪合理归属：
  ✅ 新能源/锂电池（核心） - 营收占比高
  ✅ 半导体/芯片设计（相关） - 子公司业务
```

## 🔄 验证后续工作

### 概念树优化（按需审核）

**触发条件**：
- 发现3只以上股票归属同一个待建概念
- 某个概念只剩1-2只股票
- 发现更合理的分类逻辑

**优化流程**：
1. 提出优化建议（文档）
2. 讨论评审
3. 更新概念树文档
4. 调整股票归属（记录变更原因）
