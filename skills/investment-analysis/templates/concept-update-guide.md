# 概念股票池体系更新指南

> 当完成公司/行业分析后，如何正确更新概念股票池体系

---

## 一、更新时机

### 必须更新的场景

1. **完成公司分析后**
   - 发现公司属于某个概念
   - 公司为行业龙头或核心标的
   - 有明确的归属逻辑和数据支撑

2. **完成行业分析后**
   - 识别出该行业的龙头公司
   - 这些公司符合核心标的标准
   - 尚未纳入概念池

3. **用户明确要求**
   - 用户要求添加某股票到概念池
   - 用户要求调整股票的概念归属

### 不需要更新的场景

- 公司业务不明确，难以归类
- 公司不符合核心/相关标的标准
- 信息来源不足（少于2个）

**判定逻辑**：
```
IF 满足 >= 3 项标准:
    → 核心标的（✅ 核心标的）
ELSE:
    → 相关标的（⚪ 相关标的）
```

### 相关标的判定

- 有该业务但非主营（营收占比 < 50%）
- 市占率排名靠后（非前3）
- 转型中或布局中的业务
- 技术实力一般，无明显壁垒

## 三、更新流程

### 步骤1: 收集信息

#### 必需信息

| 信息项 | 来源 | 示例 |
|--------|------|------|
| **股票代码** | 分析过程中确定 | 002230 |
| **股票名称** | 官网、交易所 | 科大讯飞 |
| **归属概念** | 参考现有概念树 | AIGC应用 |
| **标的类型** | 根据判定标准 | 核心标的 |
| **归属逻辑** | 分析结论（50字内） | AI语音龙头，讯飞星火大模型国内前3 |
| **信息来源** | 至少2个可靠来源 | 官网+2025年报+中金研报 |

#### 归属逻辑撰写要点

**好的归属逻辑**（清晰、简洁、有数据）：
```
AI语音龙头，讯飞星火大模型国内前3，AIGC产品矩阵完整
```

**不好的归属逻辑**：
```
❌ 公司做AI的（太笼统）
❌ 是龙头公司（没有说明哪个领域）
❌ 很有潜力（主观判断，无数据支撑）
```

**格式规范**：
- 长度：30-50字
- 结构：主营业务描述 + 市场地位 + 核心优势
- 用词：客观、数据化、避免主观词汇

### 步骤2: 询问用户确认

**确认模板**：
```

[归属逻辑简述，50字以内]

📚 信息来源
━━━━━━━━━━━━━━━━━━━━━━
1. [来源1：如公司官网]
2. [来源2：如2025年报]
3. [来源3：如券商研报]

```

**用户回应处理**：
- **确认** → 执行步骤3
- **修改** → 根据用户意见调整，重新确认
- **取消** → 不执行更新，询问原因

## 四、特殊情况处理

### 情况1: 股票归属多个概念

**判断标准**：
- 公司有多个主营业务
- 每个业务都符合相关标的标准

**处理方式**：
- 允许同一股票出现在多个概念中
- 每个概念独立判定核心/相关
- 在归属逻辑中说明不同业务

**示例**：
```
深科技(000021) 同时在：
- "封装测试"（核心标的）：封装业务营收占比40%
- "存储封测"（核心标的）：存储芯片封测业务领先
```

   - 创建新的概念章节

### 情况3: 需要调整现有归属

**场景**：
- 发现某股票归属不准确
- 需要修改核心/相关标的标记
- 需要更新归属逻辑

**处理方式**：
1. 找到原有记录
2. 修改对应字段
3. 在"归属逻辑"或"信息来源"中注明调整原因
4. 如有重大调整，可在文档末尾添加更新记录

## 六、常见错误及避免方法

### 错误1: 归属逻辑过于笼统

❌ **错误示例**：
```
公司主营半导体业务
```

✅ **正确示例**：
```
刻蚀设备龙头，市占率国内第一，介质刻蚀技术达到国际先进水平
```

### 错误2: 信息来源不明确

❌ **错误示例**：
```
网上查的
```

✅ **正确示例**：
```
公司官网+2025年报P15-20+中金研报(2025-12-10)
```

### 错误3: 核心/相关判定主观

❌ **错误判定**：
```
感觉这个公司很厉害 → 核心标的
```

✅ **正确判定**：
```
市占率25%（行业第2）+ 营收占比60% + 技术专利200+ → 核心标的
```

### 错误4: 表格格式错误

❌ **错误格式**：
```markdown
|股票代码|股票名称|标的类型|
|002230|科大讯飞|核心标的|  # 缺少竖线
```
