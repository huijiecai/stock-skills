## 公司分析报告结构

### 标准格式示例

```markdown
# [公司名称]

> 更新日期：2026-03-15
> 数据来源：Tushare + 2025年报 + 券商研报

---

## 一、公司简介

| 项目 | 内容 |
|-----|------|
| 证券代码 | 600519 |
| 公司名称 | 贵州茅台酒股份有限公司 |
| 上市板块 | 上交所主板 |
| 所属行业 | 白酒制造 |
| 成立时间 | 1999年 |
| 上市时间 | 2001-08-27 |
| 市场地位 | 高端白酒龙头 |

## 二、主营业务

**核心产品**：
- 茅台酒系列（占营收95%）
- 系列酒（占营收5%）

**技术路线**：传统酱香型白酒酿造工艺
**业务模式**：自产自销，经销商渠道为主

## 三、财务与业绩

### 最新财务数据（2025年前三季度）

| 指标 | 金额 | 同比变化 |
|-----|------|---------|
| 营业收入 | 1309.04亿元 | +8.5% |
| 归母净利润 | 646.27亿元 | +12.3% |
| 毛利率 | 91.2% | 持平 |
| 净利率 | 49.4% | +1.5pp |

**分析**：
- 营收增长稳健，主要受益于产品涨价和销量提升
- 净利润增速超过营收，盈利能力持续提升
- 毛利率维持高位，定价权强

## 四、行业地位与产业链定位

**产业链位置**：白酒行业 - 高端白酒 - 酱香型龙头

**市场地位**：
- 市场份额：15%（高端白酒市场第一）
- 品牌价值：5000亿元（白酒行业第一）
- 竞争优势：品牌稀缺性、产能稀缺性

## 五、投资亮点

1. **品牌护城河**
   - 国酒地位稳固
   - 消费者认可度极高
   - 品牌溢价能力强

2. **提价能力强**
   - 过去5年平均年涨价5-10%
   - 提价对销量影响小
   - 业绩增长确定性高

3. **现金流充沛**
   - 经营现金流/净利润 > 1
   - 预收账款充足
   - 分红比例稳定

## 六、风险提示

1. **消费降级风险**
   - 经济下行可能影响高端白酒需求
   - 历史案例：2013-2015年营收下降20%

2. **政策风险**
   - 限酒政策可能收紧
   - 税收政策调整风险

3. **估值风险**
   - 当前PE(TTM) 19.66，高于5年平均值15
   - 股价存在回调风险

## 七、业绩预期与估值

**业绩预期**（2026-2027）：
- 2026年营收预期：1850亿元（+10%）
- 2027年营收预期：2035亿元（+10%）

**估值分析**：
- PE(TTM)：19.66（行业平均：20.5）
- PB：7.80（行业平均：5.2）
- 估值合理，略低于行业平均

**关键观察指标**：
- 出厂价调整情况
- 经销商库存变化
- 预收账款增长率

---

*免责声明：本报告仅供学习研究，不构成投资建议*
```

## 行业分析报告结构

### 标准格式示例

```markdown
# 有色金属行业分析

> 更新日期：2026-03-15

---

## 一、行业定义与范围

**核心概念**：有色金属是除黑色金属（铁、铬、锰）外的所有金属，包括黄金、稀土、铜等。

**产业链位置**：
```
资源开采 → 冶炼加工 → 材料制造 → 终端应用
```

**细分领域**：
- 黄金
- 稀土
- 铜
- 铝
- 锂钴镍
- 钨钼

## 二、市场规模与增长趋势

**全球市场**：
- 2025年市场规模：约5000亿美元
- 预计2030年：7000亿美元
- 年复合增长率：7%

**中国市场**：
- 2025年市场规模：约3.5万亿人民币
- 全球占比：40%
- 增长驱动：新能源、国防军工

## 三、竞争格局

**全球龙头**：
- 黄金：巴里克黄金、纽蒙特矿业
- 稀土：中国占全球90%产能
- 铜：必和必拓、自由港

**国内龙头**：
- 黄金：紫金矿业、山东黄金
- 稀土：北方稀土、广晟有色
- 铜：江西铜业、紫金矿业
```

## 核心要点

### 公司分析必备章节
1. ✅ 公司简介（表格形式）
2. ✅ 主营业务（产品+技术+模式）
3. ✅ 财务与业绩（最新+对比）
4. ✅ 行业地位（产业链+市占率）
5. ✅ 投资亮点（3-5条，有数据支撑）
6. ✅ 风险提示（3-5条，客观全面）
7. ✅ 业绩预期与估值（前瞻+估值分析）

### 行业分析必备章节
1. ✅ 行业定义与范围
2. ✅ 市场规模与增长
3. ✅ 竞争格局（全球+国内）
4. ✅ 产业链结构
5. ✅ A股投资标的（三层级分类）
6. ✅ 投资逻辑与风险

### 格式要求
- ✅ 使用标准模板
- ✅ 表格对齐，数值精确
- ✅ 章节结构清晰
- ✅ 数据标注来源和时间
- ✅ 必须包含风险提示
