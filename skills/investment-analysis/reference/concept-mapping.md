# 概念归属判定指南

> **文档目的**：教LLM如何判定公司应该归属哪个概念，以及如何处理新行业/新概念

---

## 一、核心原则

### 1.1 投资热度优先

**不是所有细分领域都需要独立概念**，按投资热度决定：

| 热度 | 处理方式 | 举例 |
|------|---------|------|
| ⭐⭐⭐ **投资热点** | 独立概念，完整展开，详细分层 | 黄金、稀土、锂业、AIGC |
| ⭐⭐ **周期性品种** | 独立概念，重点列举龙头 | 铜、铝、锌铅 |
| ⭐ **小众品种** | 独立概念，列举核心标的 | 钨、钼、锡等小金属 |

### 1.2 灵活判断，不固定规则

**重要**：本文档提供的是"判定方法"，不是"固定映射表"
- ❌ 不要记住"黄金→黄金概念"这种规则
- ✅ 应该学会"如何评估黄金是否值得独立概念"

## 二、投资热度评估方法

### 2.1 如何判断是否"投资热点"？

使用以下信号综合判断：

#### 信号1：研报提及频率
```
搜索："[细分领域] 投资机会 2026"
判断标准：
- >15篇研报/月 → ⭐⭐⭐ 热点
- 5-15篇/月 → ⭐⭐ 中等
- <5篇/月 → ⭐ 冷门
```

#### 信号2：政策支持
```
搜索："[细分领域] 国家政策 战略"
判断标准：
- 国家战略、重点支持 → +1星
- 产业政策提及 → +0.5星
- 无明确政策 → 0
```

#### 信号3：市场资金流向
```
搜索："[细分领域] 板块 资金流入"
判断标准：
- 持续净流入、板块活跃 → +1星
- 波动中性 → 0
- 持续流出 → -0.5星
```

#### 信号4：媒体关注度
```
搜索："[细分领域] 投资 机会 风口"
判断标准：
- 36氪、钛媒体等频繁报道 → +0.5星
- 偶尔提及 → 0
- 无报道 → 0
```

### 2.2 综合评分

```
总分 = 研报频率 + 政策支持 + 资金流向 + 媒体关注

>= 3.5星 → 独立概念，完整展开 ⭐⭐⭐
2-3.5星 → 独立概念，重点龙头 ⭐⭐
<2星 → 独立概念，核心标的 ⭐
```

**重要原则**：
- ✅ **已完成分析的公司都应纳入概念池** - 既然分析了，就值得关注
- ✅ 投资热度决定的是"概念展开的详细程度"，不是"要不要纳入"
- ❌ 不要让LLM主观决定某个领域"不重要，跳过"

## 三、概念归属判定流程

### 3.1 标准流程

```
分析完公司 
    ↓
提取主营业务（从公司报告）
    ↓
评估投资热度（用上述方法）
    ↓
查询现有概念树
    ↓
┌─────────────┬─────────────┬─────────────┐
│ 找到匹配概念  │ 没有匹配概念  │ 不确定      │
└─────────────┴─────────────┴─────────────┘
      ↓              ↓              ↓
  直接归属      评估是否创建    询问用户
                 新概念
      ↓              ↓              ↓
填写细分领域    提出建议        获得确认
    （从主营        给用户
     业务提取）
```

### 3.2 查询现有概念树

在更新概念池前，必须先查看 `docs/概念股票池体系.md`：

```python
# 伪代码
current_concepts = read("docs/概念股票池体系.md")
extracted_concepts = parse_concept_tree(current_concepts)

# 搜索匹配
for concept in extracted_concepts:
    if is_match(company_business, concept):
        return concept
```

## 四、细分领域提取规则

### 4.1 从哪里提取？

从公司分析报告中提取：
1. **主营业务章节**：核心产品/服务
2. **财务与业绩章节**：营收构成（占比>30%的业务）
3. **行业地位章节**：公司定位描述

### 4.2 如何标注？

| 场景 | 细分领域标注 | 示例 |
|------|------------|------|
| **单一业务** | 直接写领域名 | "黄金"、"稀土"、"锂盐" |
| **多业务（主次分明）** | 主营+次营，用"+"连接 | "铜+黄金"、"钴+钼" |
| **产业链上下游** | 标注完整链条 | "锂矿+锂盐"、"稀土+磁材" |
| **多领域均衡** | 标注为"综合" | "有色金属综合"、"设备综合" |

**提取规则**：
```
IF 某业务营收占比 > 50%:
    细分领域 = 该业务
ELIF 有2个业务占比都 > 30%:
    细分领域 = "业务1+业务2"（按占比排序）
ELSE:
    细分领域 = "综合" 或 主要业务名
```

## 五、特殊情况处理

### 5.1 跨大类的公司

**场景**：公司业务横跨多个大类（如有色金属+新能源）

**处理方式**：
```
允许多重归属：
  ├─ 天齐锂业 归入 "有色金属/锂业"
  └─ 天齐锂业 归入 "新能源/锂矿"

每个概念独立判定：
  - 在"有色金属/锂业"中：核心标的（资源视角）
  - 在"新能源/锂矿"中：核心标的（下游应用视角）

归属逻辑要调整：
  - 有色金属视角：强调"锂资源储量、成本优势"
  - 新能源视角：强调"新能源车需求、锂价弹性"
```

### 5.2 概念池中没有匹配概念

**场景**：分析的公司找不到对应概念

**判定流程**：
```
1. 评估投资热度（用第二章方法）
   ↓
2. 如果热度 >= 3.5星：
   - 建议创建新概念
   - 提供：建议名称、大类归属、投资逻辑、预计公司数
   - 询问用户确认
   ↓
3. 如果热度 2-3.5星：
   - 建议归入相关的"综合"概念
   - 或暂不纳入，等后续热度提升
   ↓
4. 如果热度 <2星：
   - 不建议纳入概念池
   - 分析报告保存在 docs/[行业]/[公司].md
   - 提示：该细分领域暂不是投资热点
```

**建议创建新概念的模板**：
```
检测到新领域"氢能源"，当前无对应概念：

投资热度评估：
  - 研报提及：18篇/月 ⭐⭐⭐
  - 政策支持：国家战略+地方补贴 ⭐⭐
  - 资金流向：板块持续净流入 ⭐
  - 综合评分：4星 → 建议独立概念 ✓

建议创建：
  大类：新能源
  概念名：氢能源
  细分概念：氢能源/制氢、氢能源/储运、氢能源/燃料电池
  投资逻辑：碳中和路径、氢能产业链布局、政策支持
  预计公司：12-15家（制氢5家、储运3家、燃料电池7家）

是否确认创建？[确认] / [修改] / [暂不创建]
```

### 5.3 细分领域不是投资热点

**场景**：公司已分析，但其细分领域热度低（<2星）

**处理方式**：
```
选项1：归入"综合"概念（推荐）
  - 如"有色金属/工业金属（综合）"
  - 只列龙头或不完整展开
  - 标注细分领域：铜、铝、锌铅等

选项2：暂不纳入概念池
  - 分析报告保存在 docs/[行业]/[公司].md
  - 在报告中标注：该领域暂不是投资热点
  - 等待未来热度提升时再纳入

选项3：询问用户意见
  - 说明：该细分领域投资热度较低（<2星）
  - 提供建议：归入综合 vs 暂不纳入
  - 等待用户决策
```

## 六、实战示例（动态判断）

### 示例1：有色金属行业（2026年2月）

**分析30家公司后，评估各细分领域热度**：

```
黄金（3家）：
  - 研报：20+篇/月（避险需求、金价新高）⭐⭐⭐
  - 政策：央行持续购金 ⭐
  - 资金：持续流入 ⭐
  - 总分：5星 → 独立概念 ✓

稀土（4家）：
  - 研报：15篇/月（新能源磁材、战略资源）⭐⭐⭐
  - 政策：国家战略资源 ⭐⭐
  - 资金：波动但整体正向 ⭐
  - 总分：6星 → 独立概念 ✓

锂业（3家）：
  - 研报：25+篇/月（新能源车核心）⭐⭐⭐
  - 政策：新能源产业链 ⭐⭐
  - 资金：大量流入 ⭐
  - 总分：6星 → 独立概念 ✓

铜（5家）：
  - 研报：8篇/月（基建、制造业）⭐⭐
  - 政策：无特别支持 0
  - 资金：波动 0
  - 总分：2星 → 合并为"工业金属" ⭐⭐

铝、锌铅、钨钼等（14家）：
  - 类似铜，总分1.5-2.5星
  - 合并为"工业金属（综合）"
  - 只列出各细分龙头
```

**最终决策**：
```
概念池更新：
  ├─ 有色金属/黄金（独立）：3家
  ├─ 有色金属/稀土（独立）：4家
  ├─ 有色金属/锂业（独立）：3家
  └─ 有色金属/工业金属（综合）：20家
      ├─ 铜：5家（列出）
      ├─ 铝：3家（列出）
      ├─ 钴镍：4家（列出龙头）
      ├─ 钨钼：4家（列出龙头）
      └─ 锌铅、锡：4家（合并说明）
```

### 示例2：如果是2024年呢？（假设）

**假设2024年市场环境不同**：

```
黄金：
  - 金价低迷，研报5篇/月 ⭐
  - 总分：2星 → 可能归入"避险资产"而非独立概念

锂业：
  - 锂价暴跌，产能过剩，研报8篇/月 ⭐⭐
  - 总分：2.5星 → 可能归入"新能源材料"综合概念

稀土：
  - 磁材需求旺盛，研报20+篇/月 ⭐⭐⭐
  - 总分：6星 → 仍然是独立概念 ✓
```

**这就是"动态判断"而非"固定规则"的意义！**

## 七、质量检查清单

更新概念池前，确认：

### 投资热度评估
- [ ] 已搜索研报提及频率（>15篇/月？）
- [ ] 已查询政策支持情况
- [ ] 已了解资金流向趋势
- [ ] 综合评分 >= 3.5星（独立概念）或 >= 2星（综合概念）

### 概念归属判定
- [ ] 已查询现有概念树（docs/概念股票池体系.md）
- [ ] 找到匹配概念 OR 建议创建新概念
- [ ] 如创建新概念，已征得用户确认

### 细分领域提取
- [ ] 从公司报告准确提取主营业务
- [ ] 多业务标注按营收占比排序（主+次）
- [ ] 产业链业务标注完整（如"锂矿+锂盐"）

### 归属逻辑
- [ ] 归属逻辑清晰（50字以内）
- [ ] 突出核心竞争力（市占率、技术、规模）
- [ ] 有数据支撑（不是主观判断）
