# 公司分析操作指南

> 详细说明如何分析单个公司并生成分析报告

---

## 完整流程概览

```
步骤1: 信息收集（30-40分钟）
  ↓
步骤2: 投资决策分析（核心，20-30分钟）
  ↓
步骤3: 生成报告（10-15分钟）
  ↓
步骤4: 行业归属判定与保存（2-3分钟）
```

**总计时间**：40-55分钟

---

## 步骤1: 信息收集

### 1.1 优先使用实时数据获取

```python
from scripts.realtime_data import get_stock_realtime_data, get_stock_financial_data
from scripts.stock_formatter import format_for_analysis

# 1. 获取实时行情和估值数据
realtime = get_stock_realtime_data('600519')  # 6位股票代码
# 包含：基本信息、最新行情、PE/PB/PS、市值等

# 2. 获取财务数据
financial = get_stock_financial_data('600519')
# 包含：营收、利润、资产负债、现金流等

# 3. 格式化为 Markdown（可选）
markdown = format_for_analysis(realtime, financial)
```

**优势**：
- 秒级响应，无需等待网络搜索
- 数据准确，来自官方API（Tushare）
- 自动计算估值指标和财务比率
- 避免搜索引擎的缓存延迟

**配置要求**：需要配置 Tushare Token

### 1.2 补充使用Web搜索

**官方信息**（最高权重）：
- 公司官网：主营业务、产品线、技术实力
- 巨潮资讯网：最新年报、季报、公告
- 交易所网站：重大事项、投资者关系

**专业研究**（高权重）：
- 券商研报：行业地位分析、估值模型
- 行业报告：竞争格局、技术趋势
- 专业媒体：公司动态、重大事件

**市场讨论**（参考）：
- 雪球：投资者讨论、一手信息
- 财联社：实时快讯、行业动态
- 36氪/钛媒体：科技公司深度报道

**搜索关键词模板**：
```
- "[公司名称] 官网"
- "[股票代码] 年报 2025"
- "[公司名称] 主营业务"
- "[公司名称] 研报 2025"
- "[公司名称] 技术突破 最新进展"
- "[公司名称] 核心竞争力"
- "[公司名称] 行业地位"
- "[公司名称] 市占率 排名"
```

**信息交叉验证**（必须）：
- ✅ 至少2个独立来源验证关键数据
- ✅ 官方公告 > 券商研报 > 媒体报道 > 市场讨论
- ✅ 财务数据必须以官方年报/季报为准
- ✅ 行业地位需多方信息印证
- ⚠️ 市场讨论仅作参考，不作为核心依据

**建议工作流**：
1. 先用 `scripts/realtime_data.py` 获取基础数据（1分钟）
2. 再用 Web 搜索补充深度信息（定性分析、战略规划等）
3. **交叉验证**：对比多个信息源，确保数据准确性
4. 结合两者生成完整报告

## 步骤2: 投资决策分析（核心）

> ⚠️ **重要**：基本面只是起点，投资的关键在于**识别机会类型+预期差+催化剂**！
> 
> 💡 **说明**：这个步骤的分析内容，将在步骤3中体现为报告的**第5-11章节**（核心投资逻辑、交易价值、催化剂、策略建议等）

### 分析框架总览

| 分析模块 | 核心问题 | 对应报告章节 |
|---------|---------|-------------|
| 2.1 识别投资机会类型 | 这是什么类型的股票？（周期/成长/价值/主题） | 第5章 |
| 2.2 预期差分析 | 市场预期vs实际，哪里超预期？ | 第5章 |
| 2.3 催化剂追踪 | 有什么催化剂？什么时候买？ | 第7章 |
| 2.4 情绪面与资金面 | 市场怎么看？资金什么态度？ | 第8章 |
| 2.5 估值与交易策略 | 贵还是便宜？怎么操作？ | 第6章+第9章 |

---

### 2.1 识别投资机会类型（最高优先级）

> 💡 **核心逻辑**：不同类型的股票，赚钱逻辑完全不同！

**4种投资类型快速判断**：

```
1. 行业是否有明显周期性？（产品价格波动大）
   → YES: 🔄 类型A（周期股）- 赚周期底→顶的钱
   → NO: 继续判断

2. 是否是新兴行业？渗透率<30%？
   → YES: 🚀 类型B（成长股）- 赚渗透率起飞的钱
   → NO: 继续判断

3. 估值是否极低？PE/PB历史低位？
   → YES: 💎 类型C（价值股）- 赚低估修复的钱
   → NO: 继续判断

4. 是否有重大政策/事件催化？
   → YES: 🔥 类型D（主题股）- 赚政策驱动的钱
   → NO: 可能不适合投资（没有明确逻辑）
```

**注意**：有些股票可能同时符合多个类型（如新能源车=成长+政策），这种**叠加逻辑最强**！

📖 **详细分析框架**：参见 [投资机会类型判定.md](./投资机会类型判定.md)

---

### 2.2 预期差分析（核心价值）

> 💡 **核心逻辑**：投资赚的是认知差的钱，市场预期vs实际表现的差异就是机会！

**对比维度表格**：

| 维度 | 市场预期（A） | 实际表现（B） | 预期差（B-A） |
|-----|-------------|-------------|--------------|
| 行业趋势 | 继续下行？ | 触底反弹？ | ⭐⭐⭐ |
| 公司业绩 | 同比+5%？ | 同比+50%？ | ⭐⭐⭐ |
| 现金流 | 恶化？ | 大幅改善？ | ⭐⭐ |
| 毛利率 | 下滑？ | 企稳回升？ | ⭐⭐ |
| 订单情况 | 持平？ | 大幅增长？ | ⭐⭐ |

**判断标准**：
- 🟢 **大幅超预期**（3个⭐以上）→ **强烈看多**
- 🟡 **略超预期**（1-2个⭐）→ **中性偏多**
- 🔴 **低于预期**（负向差异）→ **规避**

**搜索关键词**：
```
- "[公司名称] 机构评级 研报 2025"
- "[公司名称] 业绩预告 超预期"
- "[公司名称] 季报 点评"
- "[公司名称] 业绩快报"
```

---

### 2.3 催化剂追踪（交易时机）

> 💡 **核心逻辑**：有催化剂才有上涨动力，没有催化剂就是"买了不涨"！

**近期催化剂**（已发生，验证逻辑）：
- ✅ 行业价格触底反弹（最重要！）
- ✅ 季度业绩大幅超预期
- ✅ 重大订单/项目落地
- ✅ 新产能投产/放量

**未来催化剂**（待验证，布局机会）：
- 🔜 下季度业绩持续高增长（最关键！）
- 🔜 行业协会限产/提价
- 🔜 新产品/新技术突破
- 🔜 海外市场拓展
- 🔜 政策扶持/标准升级

**时机判断**：
- **激进**：近期催化剂已兑现，股价未充分反应 → **当前买入**
- **稳健**：等待股价回调或下一个催化剂前布局 → **分批建仓**
- **保守**：等待多个催化剂验证后确认趋势 → **追涨买入**

**搜索关键词**：
```
- "[公司名称] 2025 最新消息 重大事件"
- "[公司名称] 订单 新产品 合作"
- "[公司名称] 产能投产 扩产"
- "[公司名称] 政策 补贴 项目"
```

---

### 2.4 情绪面与资金面

> 💡 **核心逻辑**：冷门股+基本面改善 = 最佳配置窗口；热门妖股+估值透支 = 警惕泡沫！

**分析要点**：
- **股价表现**：年内涨跌幅，强于/弱于大盘？
- **资金态度**：主力资金净流入/流出趋势
- **市场情绪**：讨论热度，多空分歧
- **换手率**：活跃度，是否有新资金关注

**情绪判断**：
- 🟢 **冷门股+基本面改善** → **最佳配置窗口**
- 🟡 **适度关注+业绩验证** → **可以参与**
- 🔴 **热门妖股+估值透支** → **警惕泡沫**

**搜索关键词**：
```
- "[公司名称] 股价 资金流向 2025"
- "[公司名称] 主力资金 北向资金"
- "[公司名称] 涨跌 换手率"
- "雪球 [公司名称]"（查看散户情绪）
- "财联社 [公司名称]"（查看市场热度）
```

---

### 2.5 估值与交易策略

> 💡 **核心逻辑**：再好的公司，估值过高也要谨慎！

**估值水平**（从实时数据获取）：
```python
pe_ttm = realtime['pe_ttm']  # 市盈率
pb = realtime['pb']          # 市净率
ps_ttm = realtime['ps_ttm']  # 市销率
```

**对比维度**：
1. **历史估值**：当前PE在历史分位数？（低估/合理/高估）
2. **行业估值**：相对同行业平均水平？
3. **成长匹配**：PEG = PE / 增速（<1为低估，>2为高估）
4. **机构目标价**：券商一致预期价格

**交易策略**：

| 时间维度 | 看多条件 | 看空条件 | 操作方式 |
|---------|---------|---------|---------|
| **短期**（1-3个月） | 拐点明确 + 业绩超预期 + 估值合理 | 短期涨幅过大 + 催化剂兑现 + 估值透支 | 分批建仓/波段操作 |
| **中期**（6-12个月） | 行业反转逻辑清晰 + 公司竞争优势明显 | 行业拐点证伪 + 业绩不及预期 | 趋势持有，目标价卖出 |
| **长期**（1-2年） | 赛道优质 + 龙头地位 + 长期成长确定 | 行业天花板显现 + 竞争格局恶化 | 长期配置，定期再平衡 |

**基础信息**（第1-4章）：

9. **交易策略建议**（多空逻辑 + 短中长期策略）

## 步骤4: 行业归属判定与文档保存

### 4.1 判定行业归属

> ⚠️ **目的**：确保每个公司都能归类到正确的行业目录下，方便后续行业分析和横向对比。

**判定逻辑（4个优先级）**：

1. **按主营业务营收占比**：如果某项业务营收占比 >50%，归类到该行业
2. **按核心竞争力**：如果营收占比接近，看哪个业务是核心竞争力
3. **按未来战略方向**：如果公司正在转型，可归类到新业务（需注明）
4. **按产业链位置**：如果跨多个环节，选择产业链价值最高的环节

**快速判断决策树**：
```
1. 看主营业务营收占比（>50%？）
   → YES: 归类到该行业
   → NO: 继续判断

2. 看核心竞争力（哪个业务是核心？）
   → 归类到核心业务所在行业

3. 看未来战略（是否转型？）
   → 归类到新业务，但需注明原业务

4. 看产业链位置（跨多个环节？）
   → 归类到价值最高的环节
```

## ⚠️ 重要提醒

### 投资分析的本质

❌ **不是**：罗列财务数据 + PE比较 + 照搬研报结论  
✅ **而是**：找到**预期差**、判断**拐点**、识别**催化剂**

### 核心思考框架

```
为什么现在要买这只股票？
↓
1. 这是什么类型的股票？
   - 周期股？成长股？价值股？主题股？
   
2. 行业/公司有没有拐点或超预期？
   - 价格触底？渗透率起飞？估值修复？政策催化？
   
3. 有什么催化剂？
   - 短期什么事件会推动股价？
   
4. 估值是高估还是低估？
   - PE/PB/PEG合理吗？
   
5. 现在是不是好的入场时机？
   - 激进/稳健/保守策略？
```

### 避免常见错误

- ❌ 只看PE就判断便宜/贵（忽略行业周期、成长性）
- ❌ 只看基本面不看催化剂（买了就不涨）
- ❌ 不分类型用统一框架（周期股用成长股框架分析）
- ❌ 盲目追热点（高位接盘）
- ❌ 不设止损位（亏损扩大）

### 好的分析报告特征

✅ **逻辑清晰**：为什么看多/看空，理由充分  
✅ **接地气**：基于市场实际，而非理论模型  
✅ **可操作**：明确入场时机、目标价、止损位  
✅ **风险明确**：什么情况下逻辑失效  
✅ **动态更新**：跟踪催化剂兑现情况，及时调整判断
