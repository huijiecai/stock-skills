## 🎯 完整执行检查清单（必须完成所有步骤）

### ✅ 步骤1：获取基础数据（1-2分钟）

```python
# 执行脚本获取实时数据
from scripts.realtime_data import get_stock_realtime_data, get_stock_financial_data

realtime = get_stock_realtime_data('股票代码')  # 6位代码
financial = get_stock_financial_data('股票代码')
```

**必须获取**：
- [ ] 股票代码、公司全称、所属行业
- [ ] 当前股价、涨跌幅、市值
- [ ] PE/PB/PS估值指标
- [ ] 营收、净利润、毛利率、ROE等财务数据
- [ ] 最新年报/季报数据

---

### ✅ 步骤2：Web搜索补充信息（10-15分钟）

#### 必须搜索的内容（并行执行）

**基本面信息**：
```
- "[公司名称] 主营业务 产品"
- "[公司名称] 行业地位 市占率"
- "[股票代码] 年报 2025"（或最新年报）
- "[公司名称] 核心竞争力 技术"
```

**最新动态**（判断涨/跌/横盘原因）：
```
- "[公司名称] 2025 最新消息 重大事件"
- "[公司名称] 股价 涨跌 原因"
- "[公司名称] 订单 新产品 合作"
- "[行业名称] 2025 行业动态 拐点"
```

**宏观催化剂**（重要！容易忽略）：
```
- "[行业名称] 地缘政治 影响"（如：欧洲能源危机→绿电需求）
- "[公司名称] 海外市场 国际政策"（如：欧盟绿色新政、美国IRA法案）
- "[行业名称] 大宗商品 成本"（如：钢材价格影响风塔成本）
- "[货币] 汇率 影响"（如：人民币贬值利好出口）
```

**机构研究**（市场预期）：
```
- "[公司名称] 机构评级 研报 2025"
- "[公司名称] 业绩预告 预期"
- "[公司名称] 目标价 券商"
```

**资金与情绪**：
```
- "[公司名称] 资金流向 主力"
- "[公司名称] 北向资金 持仓"
- "雪球 [公司名称]"（散户情绪）
```

**必须获取的信息**：
- [ ] 主营业务和产品结构（占比）
- [ ] 行业地位和市占率（具体数字）
- [ ] 最新重大事件/公告
- [ ] 近期股价表现及原因
- [ ] 机构一致预期（目标价、业绩预期）
- [ ] 资金流向和市场情绪
- [ ] 行业近况（是否有拐点/催化剂）

---

### ✅ 步骤3：判断投资机会类型（5-10分钟）

**核心问题**：这只股票投的是什么预期？

#### 判断框架

| 预期类型 | 关键验证问题 | 数据来源 |
|---------|-------------|---------|
| 🔄 **周期股** | 行业价格是否触底反弹？产能是否出清？ | 行业价格数据、开工率、库存 |
| 🚀 **成长股** | 渗透率处于什么阶段？订单是否爆发？ | 渗透率数据、订单公告、技术突破 |
| 💎 **价值股** | 估值是否极低？基本面是否稳定？有无催化剂？ | PE/PB历史分位、ROE、政策 |
| 🔥 **主题股** | 政策/事件刚发生？业绩能否兑现？ | 政策文件、市场热度 |

**必须判断**：
- [ ] 主要投资类型（可能多个叠加）
- [ ] 关键信号是否满足（每个类型至少3个信号）
- [ ] 是否有叠加逻辑（最强机会）

📖 详细判断标准：[投资机会类型判定.md](./投资机会类型判定.md)

| 行业趋势 | [券商研报/市场共识] | [你的判断] | ⭐⭐⭐ / ⭐⭐ / ⭐ / 负向 |
| 公司业绩 | [一致预期增速/绝对值] | [实际/预测增速] | ⭐⭐⭐ / ⭐⭐ / ⭐ / 负向 |
| 现金流 | [市场预期] | [实际表现] | ⭐⭐⭐ / ⭐⭐ / ⭐ / 负向 |
| 毛利率 | [市场预期] | [实际表现] | ⭐⭐⭐ / ⭐⭐ / ⭐ / 负向 |
| 催化剂 | [是否预期] | [是否有催化剂] | ⭐⭐⭐ / ⭐⭐ / ⭐ / 负向 |

**预期差总分**：XX个⭐

**判断标准**：
- 🟢 大幅超预期（≥8个⭐）→ **强烈看多**
- 🟡 略超预期（4-7个⭐）→ **中性偏多**
- ⚪ 符合预期（1-3个⭐）→ **中性**
- 🔴 低于预期（负向）→ **规避**
