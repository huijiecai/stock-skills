# 公司分析报告模板规范

> 本文档详细定义公司分析报告的格式标准、必填字段、撰写要求。确保所有报告结构统一、信息完整、易于横向对比。

---

## 报告结构总览

### 完整章节清单（1-11章）

```
# [公司名称]（[股票代码]）投资分析报告

> 一句话亮点：[核心优势/市场地位]
> 更新日期：YYYY年MM月DD日
> 所属行业：[一级行业] / [二级行业]

1. 公司简介（表格 + 概况）
2. 主营业务（产品表 + 技术 + 模式）
3. 财务与业绩（数据表 + 分析）
4. 行业地位与产业链定位（定位图 + 竞争格局）
5. 核心投资逻辑（⭐⭐⭐ 最重要）
6. 当前交易价值判断（估值 + 机构态度）
7. 催化剂与交易时机（近期 + 未来）
8. 情绪面与资金面（股价 + 资金 + 情绪）
9. 交易策略建议（多空逻辑 + 操作）
10. 风险提示（三大风险）
11. 最终判断（短中长期评级）
```

### 分析→报告映射表

| 分析步骤 | 对应报告章节 | 核心内容 |
|---------|------------|---------|
| 步骤1：信息收集 | 第1-4章 | 基础信息、财务数据、行业地位 |
| 步骤2.1：识别类型 | 第5章 | 周期/成长/价值/主题类型判定 |
| 步骤2.2：预期差 | 第5章 | 市场预期vs实际对比 |
| 步骤2.3：催化剂 | 第7章 | 近期+未来催化剂清单 |
| 步骤2.4：情绪面 | 第8章 | 股价、资金、市场情绪 |
| 步骤2.5：估值策略 | 第6章+第9章 | 估值分析+操作建议 |
| 步骤3：风险分析 | 第10章 | 行业/公司/估值风险 |
| 步骤4：综合判断 | 第11章 | 短中长期评级 |

## 第1章: 公司简介

### 必填信息表格

| 项目 | 内容 |
|------|------|
| 公司全称 | [完整公司名称，如"许继电气股份有限公司"] |
| 证券代码 | [6位代码] |
| 上市板块 | 上海证券交易所（主板）/ 深圳证券交易所（主板/创业板/科创板） |
| 成立时间 | [YYYY年] |
| 上市时间 | [YYYY年]（可选） |
| 所属行业 | [一级行业 / 二级行业]（如：电力设备 / 柔性直流输电） |
| 市场地位 | [龙头/领先/区域龙头/新兴] + 具体说明（如：换流阀市占率超40%） |

### 公司概况（1-2段）

**撰写要求**：
- **第1段**：核心业务一句话概括（做什么的？主营产品/服务）
- **第2段**：核心优势/特色（市占率、技术领先性、股东背景等）

**示例**：
```markdown
**公司概况**：许继电气是国家电网旗下的输配电及控制设备研发、制造、销售和服务企业，在柔性直流输电（柔直）技术领域全国领先。公司在特高压直流换流阀市占率超50%，是特高压直流输电的核心设备供应商。
```

## 第2章: 主营业务

### 核心产品/服务表格

| 产品类型 | 说明 | 应用领域 | 占营收比重 |
|----------|------|----------|-----------|
| [产品A] | [简要说明（1句话）] | [下游应用] | [XX%]（必填） |
| [产品B] | [简要说明] | [下游应用] | [XX%] |
| [产品C] | [简要说明] | [下游应用] | [XX%] |

**撰写要求**：
- 至少列出2个主要产品/服务
- 占营收比重必须填写（体现业务结构）
- 说明部分一句话概括核心特点

### 技术优势

**撰写要求**：
- 列出3-5个核心技术/专利
- 如有多条技术路线，需说明选择及优劣
- 说明技术壁垒（别人为什么难以复制）

**示例**：
```markdown
**技术优势**：
- 研发成功6.5kV/4000A IGCT柔直换流阀（全球首创）
- 全球首台40.5千伏/4000安气体绝缘开关设备
- 柔直换流阀用自主可控干式电容器入选能源领域首台套重大装备
- 核心技术壁垒：高压大电流控制技术，需要10年以上技术积累
```

### 业务模式

**撰写要求**：
- **生产模式**：自产/代工/混合（说明产能规模）
- **销售模式**：直销/经销/招投标（说明主要渠道）
- **客户结构**：行业分布、头部客户占比、依赖度分析

**示例**：
```markdown
**业务模式**：
- 生产模式：自主生产为主，部分通用元器件外采，拥有郑州、许昌等生产基地
- 销售模式：招投标为主（国网/南网项目），部分直销（海外市场）
- 客户结构：国网/南网占比70%，海外市场占比20%，其他占比10%
```

## 第5章: 核心投资逻辑（⭐⭐⭐ 最重要）

> 这是报告的**灵魂章节**，决定投资价值！

### 5.1 投资机会类型判定

**撰写要求**：
- 明确属于哪种类型：周期股/成长股/价值股/主题股（参见 [投资机会类型判定.md](./投资机会类型判定.md)）
- 如果是叠加逻辑，需说明主次（如：成长股（主）+政策股（辅））
- 说明判定理由（满足哪些关键信号）

**示例**：
```markdown
### 投资机会类型

**类型**：🚀 **成长股的爆发** + 🔥 **主题股的催化**（叠加逻辑）

**判定理由**：
- **成长逻辑**：特高压柔直技术渗透率<10%，处于爆发前夜
- **主题逻辑**："十五五"特高压大建设，政策催化强
- **叠加优势**：技术领先+政策支持，双重催化

**持有周期建议**：中长期（6-24个月）
```

### 5.2 行业拐点判断（针对周期股）

**撰写要求**：
- 如果是周期股，必须分析行业拐点信号
- 列出价格、产能、库存、龙头动作等5个维度
- 判断是强拐点/弱拐点/伪拐点

**示例**：
```markdown
### 行业拐点判断

**拐点信号**：
- ✅ **价格反弹**：光伏组件价格从0.8元/W回升至1.0元/W（+25%）
- ✅ **产能出清**：落后产能退出，行业开工率从60%回升至75%
- ✅ **库存去化**：渠道库存从3个月降至1.5个月
- ⚠️ **龙头限产**：暂无明确限产动作
- ✅ **需求回暖**：海外订单环比+30%

**拐点强度**：🟢 **强拐点**（满足4/5个信号）

**结论**：行业周期底部确认，景气度进入上行通道
```

### 5.3 预期差分析

**撰写要求**：
- 列出市场预期vs实际表现对比表
- 标注预期差程度（⭐⭐⭐/⭐⭐/⭐）
- 计算预期差总分，判断看多/看空

**示例**：

| 维度 | 市场预期（A） | 实际表现（B） | 预期差（B-A） |
|-----|-------------|-------------|--------------|
| 行业趋势 | 继续下行 | 触底反弹 | ⭐⭐⭐ |
| 公司业绩 | 净利润+10% | 净利润+25% | ⭐⭐⭐ |
| 现金流 | 持平 | 大幅改善+50% | ⭐⭐ |
| 毛利率 | 下滑 | 提升2.78pct | ⭐⭐ |
| 订单情况 | 持平 | 重大项目落地 | ⭐⭐ |

**预期差总分**：12个⭐（大幅超预期）

**结论**：🟢 **强烈看多** - 市场低估了行业拐点和公司业绩改善

### 5.4 业绩增长驱动因素

**撰写要求**：
- 列出3-5个关键驱动因素
- 每个因素需量化影响（如："特高压建设拉动营收增长30%"）
- 区分短期驱动和长期驱动

**示例**：
```markdown
### 业绩增长驱动

**短期驱动**（1-2年）：
1. **特高压直流大建设**：预计"5直3交"开工，换流阀订单集中释放，拉动营收+30%
2. **在手订单饱满**：甘肃-浙江、青藏二期等项目落地，2026年业绩有保障
3. **毛利率提升**：技术升级+规模效应，毛利率有望从20%提升至22%

**长期驱动**（3-5年）：
1. **柔直技术爆发**：新能源并网需求旺盛，柔直市场空间>500亿
2. **海外市场拓展**：中东、南美市场突破，海外营收占比有望从20%提升至35%
3. **国产替代**：IGBT、IGCT等核心器件国产化，降低成本+提升毛利
```

## 第6章: 当前交易价值判断

### 估值水平对比

**撰写要求**：
- 列出当前PE/PB/PS估值
- 对比历史估值分位（10%/50%/90%分位）
- 对比同行业平均估值
- 计算PEG（PE/增速），判断成长性匹配度

**示例表格**：

| 估值指标 | 当前值 | 历史分位 | 行业平均 | 评估 |
|---------|-------|---------|---------|------|
| PE(TTM) | 25倍 | 30%分位（低估） | 30倍 | 低于行业 |
| PB | 3.5倍 | 40%分位 | 4.0倍 | 合理 |
| PS(TTM) | 1.8倍 | 35%分位 | 2.0倍 | 低于行业 |
| PEG | 0.8 | - | 1.2 | **低估** |

**估值分析**：
- PE处于历史30%分位，估值较低
- PEG<1，成长性未被充分定价
- 相对同行业折价15%，有估值修复空间

### 机构态度

**撰写要求**：
- 引用3-5家券商最新研报评级
- 列出一致预期目标价
- 说明机构看多/看空理由

**示例**：
```markdown
### 机构态度

| 券商 | 评级 | 目标价 | 日期 | 核心观点 |
|-----|------|--------|------|---------|
| 中信证券 | 买入 | ¥35 | 2026-03-10 | 特高压拐点确认，业绩进入兑现期 |
| 华泰证券 | 增持 | ¥32 | 2026-03-08 | 柔直技术领先，长期成长空间大 |
| 国泰君安 | 买入 | ¥38 | 2026-03-05 | 在手订单饱满，毛利率提升超预期 |

**一致预期目标价**：¥35（相对当前价格¥28，上涨空间25%）

**机构核心逻辑**：
1. 特高压建设进入新周期，换流阀需求爆发
2. 公司技术领先，市占率有望进一步提升
3. 毛利率提升+海外突破，业绩弹性大
```

## 第7章: 催化剂与交易时机

```markdown
- ✅ **2025年10月**：6.5kV IGCT柔直换流阀通过鉴定，技术突破 → 股价+5%

**逻辑兑现度**：🟢 **高**（4/4个催化剂均兑现，业绩和股价双双上涨）
```

### 未来催化剂（待验证，布局机会）

**撰写要求**：
- 列出未来3-12个月的潜在催化剂

- 说明如何验证（关键指标）

```markdown
### 未来催化剂

| 时间窗口 | 催化剂 | 重要性 | 验证指标 |

**最关键催化剂**：2026年Q3"5直3交"项目开工（板块性机会）
```

## 第8章: 情绪面与资金面

### 股价表现

**撰写要求**：
- 年内涨跌幅
- 相对大盘表现（强于/弱于）
- 关键时间点股价变化

**示例**：
```markdown
### 股价表现

- **年内涨跌幅**：+25%（2026年1月1日至今）
- **相对大盘**：强于沪深300指数15个百分点

  - 1月：中标甘肃-浙江项目后单日+8%
  - 2月：随板块回调-5%
  - 3月：业绩发布后再次上涨+12%
```

### 资金流向

**撰写要求**：
- 主力资金净流入/流出趋势（近1-3个月）
- 北向资金/机构持仓变化
- 换手率和成交量分析

**示例**：
```markdown
### 资金流向

- **主力资金**：近3个月净流入15亿，持续流入态势
- **北向资金**：持仓占比从2%提升至3.5%，外资加仓
- **机构持仓**：基金持仓占比8%，较Q4提升2个百分点
- **换手率**：3%（正常水平），资金活跃度适中
```

### 市场情绪

**撰写要求**：
- 讨论热度（雪球/财联社/东方财富）
- 多空分歧
- 市场认知（热门股/冷门股）

**示例**：

```markdown
### 市场情绪

- **讨论热度**：雪球关注度2万+，财联社提及频次中等
- **多空分歧**：
  - 多方观点：特高压拐点确认，业绩进入兑现期
  - 空方观点：估值已修复，短期涨幅较大需要消化
- **市场认知**：🟡 **适度关注**（不冷不热，机构配置但散户关注度一般）

**情绪判断**：🟢 **健康** - 适度关注+机构加仓，不是妖股，回调是机会
```

## 第9章: 交易策略建议

### 多头逻辑

**撰写要求**：
- 列出3-5个看涨理由
- 每个理由需有数据支撑
- 说明哪些条件下继续看多

**示例**：
```markdown
### 多头逻辑

**看涨理由**：
1. **行业拐点确认**：特高压进入新建设周期，换流阀需求爆发
2. **业绩超预期**：Q4净利润+25%，超市场预期15个百分点
3. **估值合理**：PE 25倍，PEG 0.8，相对行业折价15%
4. **催化剂密集**：Q1业绩、"5直3交"开工、海外突破
5. **资金态度积极**：主力净流入，北向资金加仓

**持续看多条件**：
- Q1净利润增速>30%
- "5直3交"项目如期开工
- 毛利率持续提升
- 估值不超过35倍PE
```

### 空头逻辑

**撰写要求**：
- 列出2-3个看跌风险
- 说明哪些条件下需要退出

**示例**：
```markdown
### 空头逻辑

**看跌风险**：
1. **拐点证伪**：特高压项目延期，订单不及预期
2. **业绩不及预期**：Q1净利润增速<15%
3. **估值透支**：短期涨幅过大，PE超过40倍
4. **竞争加剧**：其他厂商技术突破，市占率下滑

**退出条件**（止损）：
- 跌破¥25（短期止损）
- Q1业绩低于预期15%以上
- "5直3交"项目取消或大幅延期
```

### 短期/中期/长期策略

**撰写要求**：
- 明确三个时间维度的操作建议
- 给出目标价和止损位
- 说明仓位管理

**示例**：
```markdown
### 操作建议

#### 短期（1-3个月）

**策略**：波段操作，Q1业绩发布前布局
- **目标价**：¥32（+15%）
- **止损位**：¥25
- **仓位**：20-30%
- **操作**：¥26-28分批买入，¥32附近止盈

#### 中期（6-12个月）

**策略**：趋势持有，享受业绩兑现期
- **目标价**：¥38（+35%）
- **止损位**：¥28
- **仓位**：30-40%
- **操作**：
  - Q1业绩超预期后加仓
  - "5直3交"开工后继续持有
  - 目标价或估值过高（PE>40）时止盈

#### 长期（1-2年）

**策略**：核心配置，享受行业红利
- **目标价**：¥45-50（+60-80%）
- **止损位**：拐点证伪或行业天花板显现
- **仓位**：10-20%（作为组合配置的一部分）
- **操作**：
  - 长期持有，不频繁交易
  - 每季度复盘，调整判断
  - 渗透率>50%或竞争格局恶化时退出
```

**撰写要求**：

```markdown
| | 海外市场拓展不及预期 | 低 |

**风险应对**：

- 跟踪季度业绩，低于预期15%止损
- 估值超过40倍PE时止盈，避免泡沫
```
