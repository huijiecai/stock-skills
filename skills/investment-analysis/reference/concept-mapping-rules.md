# 行业-概念映射规则

> **文档目的**：定义行业分析产出的公司如何映射到概念股票池体系中的概念

---

## 一、映射原则

### 1.1 投资热度优先原则

**核心思路**：按投资热度选择概念，不是所有行业细分领域都需要独立概念

**分类标准**：
- ⭐⭐⭐ **投资热点**：独立概念，完整展开
- ⭐⭐ **周期性品种**：合并为"工业/综合"概念，只列龙头
- ⭐ **小众品种**：记录在行业分析中，暂不纳入概念池

### 1.2 细分领域说明原则

**所有概念都需要说明包含哪些细分领域**：

```markdown
#### 2.X.X 概念名称（X只）
> **细分领域**：[具体的行业细分]  
> **投资逻辑**：[为什么这是投资热点]  
> **对应行业细分**：[与docs/中的目录对应]
```

## 六、特殊情况处理

### 6.1 跨大类的公司

**场景**：一家公司的业务横跨多个大类

**处理方式**：
1. **允许多重归属**：同一公司可以出现在多个概念中
2. **每个概念独立判定**：在不同概念中的核心/相关标的可以不同
3. **细分领域标注要全面**：完整标注所有业务

**示例**：
```
紫金矿业（601899）:
  ├─ 有色金属/黄金：细分领域"铜+黄金"，核心标的（黄金营收>30%）
  ├─ 有色金属/工业金属：细分领域"铜+黄金"，核心标的（铜业务龙头）
  └─ 归属逻辑根据概念调整：
      - 在黄金概念中：强调"铜金双龙头，2026年产金105吨"
      - 在工业金属中：强调"全球矿业航母，产铜120万吨"
```

### 6.2 细分领域不在投资热点的公司

**场景**：公司分析已完成，但其细分领域不是投资热点

**处理方式**：
1. **方案1：归入综合概念**
   - 如工业金属（综合）、化工（综合）
   - 只列龙头或不完整展开

2. **方案2：暂不纳入概念池**
   - 分析报告保存在 `docs/[行业]/[公司名].md`
   - 等待该细分领域成为投资热点时再纳入

3. **方案3：创建新的小众概念**
   - 如果该细分有5+家公司，且有投资价值
   - 经用户确认后创建新概念

**示例**：
```
铜陵有色（铜冶炼龙头）:
  ├─ 选项1：归入"有色金属/工业金属（综合）"
  ├─ 选项2：暂不纳入概念池，仅保留分析报告
  └─ 选项3：如铜价大涨成为热点，可创建"有色金属/铜"独立概念
```

### 6.3 概念树中没有对应概念

**场景**：分析的行业在现有概念树中找不到对应

**处理方式**：
1. **询问用户**：是否创建新概念？
2. **提供建议**：
   - 建议的大类归属
   - 建议的概念名称
   - 预计包含的公司数量
3. **获得确认后**：
   - 在概念树总览中添加新行
   - 创建新的概念章节
   - 批量添加公司

**示例模板**：
```
检测到新行业"商业航天"尚未在概念池中：

建议创建新概念：
  大类：商业航天
  细分概念：
    - 卫星制造（3家）
    - 火箭发射（2家）
    - 卫星通信（4家）
  
是否创建？[确认] / [修改] / [取消]
```

## 七、映射流程总结

```
行业分析完成（30家公司）
        ↓
提取行业名称 + 细分领域
        ↓
查询映射规则表
        ↓
┌───────────────┬───────────────┬───────────────┐
│  投资热点     │  周期性品种    │  小众品种      │
│  (独立概念)   │  (综合概念)    │  (暂不纳入)    │
└───────────────┴───────────────┴───────────────┘
        ↓              ↓              ↓
  完整展开全部    只列龙头      保留分析报告
  (黄金、稀土、   (工业金属、   (docs/中查阅)
   锂业等)        化工等)
        ↓              ↓
  细分领域标注：单一业务 vs 多业务
        ↓
  批量更新概念股票池体系
```
