## 一、公司分析示例

### 示例：康强电子（002119）

**参考文件**：`docs/公司分析/康强电子.md`

#### 分析亮点

✅ **结构完整**：
- 公司简介用表格清晰展示基本信息
- 主营业务包含产品列表和技术路线
- 财务数据有多个时间段对比
- 行业地位用产业链图示说明

✅ **数据详实**：
- 财务数据具体到小数点
- 同比变化用百分比标注
- 营收构成用表格展示占比

✅ **分析客观**：
- 既有投资亮点也有风险提示
- 业绩预期分乐观/中性/悲观三种场景
- 估值分析与行业对比

#### 关键特点

**1. 产业链定位可视化**

```
存储芯片产业链：
设计 → 制造（晶圆厂）→ 【封装测试】→ 终端应用
                          ↑
                    康强电子位置
                   （封装材料供应商）
```

**2. 财务数据对比表**

| 指标 | 2024年 | 2025年前三季度 | 变化 |
|------|--------|----------------|------|
| 营业收入 | 19.65亿元 | 15.64亿元 | +5.16% |
| 归母净利润 | 8,318.97万元 | 9,641.49万元 | +21.4% |
| 毛利率 | 11.98% | 14.35% | 提升 |

**3. 风险提示全面**

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ 业绩波动 | 季度间利润波动剧烈，Q4曾大幅下滑 |
| ⚠️ 成本压力 | 铜、金、银等原材料价格波动影响利润 |
| ⚠️ 竞争加剧 | 国内外厂商均在加大封装材料投入 |

#### 学习要点

1. **用表格呈现结构化信息**（基本信息、财务数据、竞争格局）
2. **用图示说明复杂关系**（产业链位置）
3. **数据要有对比**（同比、环比、行业对比）
4. **客观呈现优势和风险**（不能只谈优点）
5. **标注信息来源和时间**（年报、研报、查询日期）

## 二、行业分析示例

### 示例：半导体设备与材料

**参考文件**：`docs/行业分析/半导体产业链/01_设备与材料.md`

#### 分析亮点

✅ **通俗易懂**：
- 用"造芯片的机器"、"印刷机印报纸"等比喻解释专业术语
- 表格对比不同设备类型的作用

✅ **数据可视化**：
```

✅ **竞争格局清晰**：

| 设备类型 | 全球龙头 | 市占率 | 国内龙头 | 国内市占率 |

✅ **国产替代分析**：
- 各细分领域国产化率
- 技术差距评估（星级难度）
- 突破方向和时间预期

#### 关键特点

```

#### 学习要点

1. **用通俗语言解释专业术语**
2. **用可视化符号表示数据**（进度条、星级）
3. **竞争格局用对比表**（全球vs国内）
4. **产业链用图示**（上中下游关系）
5. **投资标的要列明**（公司、代码、核心看点）

## 三、对比分析

### 公司分析 vs 行业分析

| 维度 | 公司分析 | 行业分析 |
|------|----------|----------|
| **聚焦对象** | 单个公司 | 整个行业/产业链 |
| **核心内容** | 财务、业绩、竞争力 | 市场规模、竞争格局、趋势 |
| **数据来源** | 年报、研报（公司维度） | 行业报告、协会数据 |
| **输出文档** | `docs/公司分析/[公司].md` | `docs/行业分析/[产业链]/XX_[行业].md` |
| **更新频率** | 季报/年报发布后 | 年度更新或重大变化时 |

### 核心标的 vs 相关标的

| 维度 | 核心标的 | 相关标的 |
|------|----------|----------|
| **市占率** | 前3名 | 排名靠后 |
| **营收占比** | >50% | <50% |
| **技术实力** | 领先，有壁垒 | 一般 |
| **研报提及** | 频繁提及龙头 | 较少提及 |
| **标记符号** | ✅ 核心标的 | ⚪ 相关标的 |

## 六、常见问题示例

### Q1: 财务数据不全怎么办？

**场景**：新上市公司，只有招股书，没有年报

**处理**：
```
1. 使用招股书财务数据（标注来源）
2. 查看上市后的季报数据
3. 参考东方财富F10的财务摘要
4. 标注"数据有限，待年报更新"
```

### Q2: 找不到行业龙头信息？

**场景**：细分领域太新，研报少

**处理**：
```
1. 扩大搜索范围到上级分类
   如：搜不到"CPO光模块龙头"
   → 搜"光模块龙头"，再细化到CPO
   
2. 查看行业协会网站
3. 参考东方财富概念板块的个股列表
4. 标注"行业初期，龙头格局未定"
```

### Q3: 概念归属不确定？

**场景**：公司业务多元，难以归类

**处理**：
```
1. 查看年报营收构成，找主营业务
2. 参考东方财富的概念板块归属
3. 查看券商研报如何定义该公司
4. 如果确实跨多个概念，可归入多个
5. 在归属逻辑中说明主次
```
