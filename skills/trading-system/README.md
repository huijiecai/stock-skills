# 预期驱动交易系统

> 核心：围绕预期做交易。形态只是确认预期的载体。
> 不是量化公式，是投资者的思维框架。

---

## 目录结构

```
trading-system/
├── README.md                    ← 本文件

├── daily/

│       ├── 模拟看盘.md           ← 盘后：分时回溯+信号模拟（数据允许时）

└── lessons/
    └── learned.md               ← 经验教训库（按阶段分类）
```

## 核心流程

```
盘前分析 → 交易预案 → 盘中执行 → 盘后复盘 → 经验教训 → 系统迭代
```

### 盘前（T-1日晚）
- 全市场催化扫描
- 梳理候选预期方向（产业 > 政策 > 个股 > 情绪）
- 每个方向回答：锚是什么？什么情况下证伪？
- 情景推演 + 交易预案

### 盘中（9:30-15:00）
- 执行计划内交易，不做计划外交易
- 每笔过检查清单：预期？确认了吗（资金+价格+锚）？仓位匹配？

### 盘后（15:00后）
- 实际数据 vs 盘前预期
- 信号触发记录
- 经验教训提炼
- 系统规则迭代

## 系统版本

- **v1**：4阶段情绪周期（冰点→修复→高潮→退潮），2026-03-27首次实战
- **v2**：7阶段主线生命周期（冰点→修复→分歧→再确认→主升→高潮→退潮），三维确认信号
- **v2.1**：补入实战教训（多主线并行、板块内部结构、全市场催化扫描、周末风险预案）

## 关键原则

- 低频，但确定
- 不猜、不赌、不扛
- 指标是感知信号，不是触发条件
- 每个决策回到核心问题：预期确认了？兑现了？失效了？
