# 交易系统：预期驱动交易法则

> 预期驱动交易法则：围绕预期做交易。市场在情绪拐点选择了哪个方向，就参与哪个方向。不猜、不赌、不扛。低频，但确定。

---

# 第一部分：认知框架

## 一、预期

### 核心定义

**预期 = 市场资金正在投票的方向。**

围绕预期做交易，不是量化公式，是投资者的思维框架。形态只是确认预期的载体。每一个数值都是感知信号，不是机械触发条件。

### 买入的两步前提（缺一不可）

1. **市场形成了新的预期** —— 相对此前而言，出现了**新的叙事、新的催化或新的可验证点**（旧故事没有更新、没有增量信息，不构成新的买入理由）。
2. **资金完成了确认，并认可了这个预期** —— 真金白银投票，用板块效应、价格行为、预期依据（见第五节三维确认）证明：**市场认**这个故事，而不只是你个人认为对。

缺一则不买：只有逻辑没有资金认，是空气；只有异动没有预期锚点，是噪音。

**确认 = 市场选择了这个方向，形成了板块效应。** 系统不预判哪个方向"应该"走强，只观察市场**实际选择了哪个方向**。市场选择的标准：
1. 在情绪修复或分歧后率先走强
2. 形成板块效应（不是一只票涨，是一批票联动）
3. 经过分歧测试后资金仍然选择它（分歧→再确认）

**确认就是系统的买点。证伪就是系统的卖点。**

| | 确认（买点） | 证伪（卖点） |
|--|------------|------------|
| 产业趋势方向 | 价格/数据持续走强 | 价格/数据逆转 |
| 叙事/消息方向 | 分歧后板块效应仍然强 | 分歧后资金不回来了 |
| 任何方向 | 市场选择了这个方向 | 市场放弃了这个方向 |

**预期不问出处，确认只看市场。**

### 系统的决策层次

本系统的决策分三层，从高到低：

| 层次 | 核心问题 | 决定什么 |
|------|---------|---------|
| **① 预期+确认** | 有新预期吗？资金认可了吗？ | **买不买**（核心决策层） |
| **② 预期强度** | 产业级/板块级/个股级？处于哪个阶段？ | **买多少**（仓位管理层） |
| **③ 情绪周期** | 市场整体情绪处在哪一段？ | **参考指标**：日常影响仓位轻重；极端时作为安全阀 |

**核心原则**：买不买由"预期+确认"决定，不由情绪周期决定。情绪周期是参考指标，不是门槛。

- 日常使用：情绪好→仓位可以更积极；情绪差→仓位收紧。但不会因为"情绪差"而错过一个被资金确认的强预期
- 极端安全阀：情绪极差时（涨停<30、大面积跌停、恐慌性下跌），即使有预期出现，资金也很难认可，空仓是更安全的选择

### 预期的来源

| 来源 | 说明 | 退出信号清晰度 | 仓位建议 |
|------|------|---------------|---------|
| **产业趋势** | 行业趋势、供需变化、技术突破 | 清晰（有价格/排产/业绩数据可跟踪） | 可给较高仓位 |
| **政策落地** | 政策方向、会议信号，有具体投资金额/项目进度 | 较清晰（有项目进度可跟踪） | 中等仓位 |
| **叙事/事件** | 两岸关系、重组预期、阅兵、航天发射等 | 取决于事件节点和资金行为 | 中等仓位，止盈更紧 |
| **情绪博弈** | 纯市场行为驱动，无明确逻辑支撑 | 最模糊（只能观察板块结构变化） | 低仓位或观察 |

预期没有高低贵贱之分。一个方向能否参与，不取决于它是什么类型的预期，而取决于**市场是否在情绪拐点选择了它**。类型只影响退出信号的清晰度和仓位管理策略。

**为什么产业趋势型预期的退出信号更清晰？** 因为可以用数据验证——碳酸锂跌3%就是信号，GPU租赁价回落就是信号。而叙事型预期（两岸概念、商业航天）没有这样的单一数据点，退出依赖板块结构观察，更模糊。但这不代表叙事型预期不能参与。关键看市场是否在情绪拐点选择了它，以及分歧后资金是否回来。

**注意**：情绪预期如果出现了具体的事件催化（如卫星发射验证了商业航天），则退出信号会更清晰。要区分"纯概念炒作"和"有事件验证的概念"——后者可持续性和确定性都会提升。

### 预期的两个致命问题

每次评估一个预期方向时，必须回答：

1. **这个预期的确认信号是什么？（资金+价格+预期依据）** — 如果答不出来，就无法判断该不该参与
2. **什么情况下这个预期会被证伪？** — 如果也想不出来，说明你对这个方向的理解还不够，不该参与

## 二、预期质量

预期本身就自带时间维度——"预期"意味着"未来会发生的事"。预期质量不是看"这个预期好不好"，而是看**"这个预期处于生命周期的哪个阶段"**和**"预期能持续多久"**。

### 统一规则：所有操作都基于预期

- **预期在**（本轮逻辑仍成立、尚未被本轮兑现充分定价）：可以持有，加仓、减仓按确定性调节。
- **本轮预期兑现**（业绩/事件/数据把本轮故事讲完、利好落地）：**走**。不因「以后可能还有下一季」而赖在仓位里——那是下一笔的事。
- **兑现之后**：空仓或轻仓等待；**等**新的市场信号、**等**新的预期形成，且**资金认可（确认）**后再进场。产业级与个股级的差别在于：**下一轮预期出现有多快、是否值得再接**，不是「兑现了还可以继续拿着不叫走」。

### 与情绪周期的关系

预期质量和第四节的情绪周期是两个独立但互相影响的维度：

- **情绪周期（第四节）**——市场整体层面的**参考指标**。日常影响仓位轻重（情绪好→仓位更积极，情绪差→仓位收紧）；极端时作为安全阀（情绪极差→即使有预期也空仓）。**不决定"买不买"。**
- **预期阶段（本节）**——单个方向层面，决定**"这个方向值不值得追"**。出现/确认期参与，兑现/结束期不参与。

两者独立但互相影响：同一个情绪周期下，不同方向可以处于不同的预期阶段；同一个预期在不同情绪周期下，参与策略也不同。

**注意**：买不买的唯一标准是"预期+确认"。情绪周期不影响这个判断。

### 预期阶段

```
出现期 → 确认期 → 加速期 → 兑现期 → 结束期
```

| 阶段 | 含义 | 操作 |
|------|------|------|
| 出现期 | 新预期刚出现，板块刚开始异动 | 最佳买点 |
| 确认期 | 市场对预期有了认知，板块效应形成 | 可以买 |
| 加速期 | 预期持续被验证，股价加速上涨 | 持有，分歧时加仓 |
| 兑现期 | 本轮预期中的事件发生（业绩公布、政策落地等），本轮定价完成 | **本轮兑现即走**；不追；然后等下一信号与下一预期，确认后再做 |
| 结束期 | 预期被完全消化 | 不参与 |

**注意**：市场对预期有反应（股价上涨）= 预期被认可的确认信号，不是"定价完了所以不能买"。没有市场反应，你根本不知道这个预期成立不成立。市场反应本身就是预期的确认信号。

**与产业级长周期的关系（避免误读）**：趋势线波动都来自**预期**；**没有**「兑现了还赖着不走、硬等下一季」这回事——**本轮**交易的预期一旦兑现，按纪律**先离场**（或收到观察仓），和持有周期长短无关。产业级趋势的「长」，指的是**同一条产业逻辑上，兑现后很快会出现下一个可验证的预期**（例如每个季报都是新的验证点），你可以在**空仓等待**之后，对**新一轮**预期在「市场信号 + 资金认可」确认后再进场。那是新的一笔交易，不是把上一笔扛穿兑现期。

### 预期强度

预期强度决定每个阶段持续多久，以及可给的仓位上限：

| 级别 | 定义 | 每个阶段持续多久 | 仓位上限 |
|------|------|---------------|---------|
| 产业级 | 行业趋势拐点，有可验证的数据支撑（季报、订单、行业价格） | 数月到数年 | 3成 |
| 板块级 | 有明确催化，有预期依据可跟踪 | 数周到数月 | 2成 |
| 个股级 | 纯情绪/连板炒作，缺乏预期依据 | 数天到数周 | 1成 |

### 关键区分

- **预期强度**决定仓位上限，不决定能不能参与
- **预期阶段**决定参与时机（出现/确认期参与，兑现/结束期不参与）
- **情绪周期**是参考指标，影响仓位轻重，但不决定是否进场。买不买的唯一标准是"预期+确认"（极端冰点除外）

不参与的判断来自预期阶段和市场阶段，不是来自预期强度。板块级出现期 = 可以给2成参与。

### 实例

**德明利（存储芯片）的预期生命周期**：
- 出现期：存储价格开始反弹，板块异动
- 确认期：佰维存储提前发布Q1业绩暴增429%，20%涨停，存储板块集体发酵，德明利涨停
- 加速期：德明利持续上涨，涨了近100%
- 兑现期：德明利公布Q1净利31.5-36.5亿，**本轮**预期被兑现 → **按纪律离场**（或收到观察仓），不混同于「继续持有等下一季」
- 兑现后：场外等待；若后续出现**新的**可验证预期（如Q2持续性、200亿营收目标被资金重新认可），那是**新一轮**确认与买点，再进场

**产业级 vs 个股情绪（时间尺度）**：二者都是「预期在就拿、兑现就走」。差别在于——**产业级**往往兑现后**很快**会有下一个可验证的预期锚点（例如每个季报都是新的验证点），同一条产业逻辑可以在多轮预期里反复参与，所以**日历上**可以很长；**个股纯情绪炒作**往往整段生命周期只有几天，**下一轮像样的预期**常常不存在或极短，所以看起来就是快进快出。不是产业级就可以「兑现了也不走」。

# 第二部分：市场节奏

## 三、情绪拐点

情绪拐点是系统确认的起点。如何判断市场情绪在转向？

### 拐点的常见信号

| 信号 | 拐点类型 | 具体表现 |
|------|---------|---------|
| 涨停数从极低（<30）快速回升到50+ | 冰点→修复 | 恐慌出清，第一批资金进场试错。修复期不是空仓期——如果有预期+确认，可以买入 |
| 指数在连续下跌后放量企稳 | 冰点→修复 | 抛压衰竭，增量资金开始承接 |
| 前期退潮的强势板块在指数企稳时率先领涨 | 冰点→修复 | 资金在重新选择方向 |
| 涨停数从高位（80+）快速回落到40以下 | 高潮→退潮 | 增量枯竭，资金开始撤退 |
| 龙头股断板后不再回封，跟风股全面溃散 | 高潮→退潮 | 主线确认结束 |
| 分歧日涨停数不降反升、板块联动恢复 | 分歧→再确认 | 资金经过筹码交换后重新选择了方向 |

**注意**：单个信号不足以确认拐点。多个信号共振时，拐点判断的确定性更高。同时要结合成交额变化——放量拐点比缩量拐点更可靠。

## 四、主线生命周期

情绪周期是**参考指标**，不是决策框架。它描述的是市场整体的情绪状态（冰点→修复→…→退潮），用来辅助判断**仓位轻重**和**持仓节奏**——但**不决定买不买**。

买不买的唯一标准是第一节的"预期+确认"（有新预期 + 资金认可）。情绪周期的角色：
1. **日常参考**：情绪好→仓位可以更积极；情绪差→仓位收紧
2. **极端安全阀**：涨停<30、大面积跌停、恐慌性下跌时，即使有预期出现，资金也很难认可，空仓是更安全的选择

情绪周期可能是"预期+确认"逻辑的自然结果——当多个方向的预期被资金确认时，市场自然进入主升期；当没有预期被确认时，市场自然进入冰点期。所以情绪周期更多是**观察**而非**决策**。

一条真正的主线（如锂电2026年3月），不是从"冰点"直线涨到"高潮"的。它经历的是一个完整的**情绪—资金**推进过程：

```
① 冰点        恐慌出清，市场否定了几乎所有方向
    ↓
② 修复        第一批资金进场试错，方向不明确，赚钱效应差
    ↓
③ 分歧        主线方向浮现，但资金意见不统一
              ↳ 这不是退潮！是筹码交换的自然阶段
              ↳ 分歧后能否重新走强，是判断主线真伪的关键
    ↓
④ 再确认      主线在分歧后重新走强 — 这恰恰是最安全的买点之一
              ↳ 经过分歧测试的确认，比未测试的确认更可靠
    ↓
⑤ 主升        共识形成，板块内梯队完整（龙头+中军+补涨）
              ↳ 持续时间最长，赚钱效应最强
    ↓
⑥ 高潮        全面爆发，后排补涨，连板高度达到顶峰
              ↳ 所有能买的都买了，增量资金开始枯竭
    ↓
⑦ 退潮        龙头见顶，板块联动消失，资金撤出
              ↳ 回到①
```

### 分歧 ≠ 退潮

| | 分歧 | 退潮 |
|---|---|---|
| 本质 | 筹码交换，资金在"要不要继续"上犹豫 | 资金已经放弃，在离场 |
| 龙头表现 | 震荡、断板但未大跌 | 大跌、A杀 |
| 板块联动 | 部分跟风回落但核心还在 | 全面溃散 |
| 后续可能 | 重新确认 → 主升 | 继续下杀 → 新冰点 |
| 操作意义 | **观察窗口**：看主线能否扛住分歧 | **空仓窗口**：等待新冰点 |

**3月27日实战验证**：3月26日看起来像退潮（超4400跌、涨停39），但电力方向实际上是分歧（华电辽能断板但未崩，华电能源震荡），锂电在分歧后重新确认（4连板+8只涨停）。如果简单判定为退潮→空仓，就错过了锂电的再确认买点。

### 多主线并行

不要默认市场同一时间只有一条主线。当成交额充裕（2万亿以上）时，市场有能力同时支撑2-3条并行主线（如锂电+创新药），前提是各主线之间不形成"抢钱"关系（一个涨另一个就跌）。判断方法是：看多条主线的资金流向是否共振——如果几条线都在涨且都有增量资金，说明是并行；如果此消彼长，说明资金在切换，只能选强的一条。

成交额不足（1.5万亿以下）时，资金大概率只能聚焦一条线，这时候多主线假设就不适用了。

### 每个阶段的预期状态

| 阶段 | 预期状态 | 投资者心态 | 正确做法 |
|------|---------|-----------|---------|
| 冰点 | 预期真空期，所有方向被否定 | 恐慌、绝望 | 空仓等待，观察哪个方向率先走出冰点 |
| 修复 | 新预期开始萌芽，方向不明确 | 试探、犹豫 | 有预期+确认的方向可以买入 |
| 分歧 | 预期被初步选择，但资金分歧 | 焦虑、多空争论 | 不急着站队，观察哪个预期在分歧中扛住 |
| 再确认 | 预期被分歧验证，确定性提升 | 信心恢复 | 经过分歧验证的确认更可靠，确认了可以加仓 |
| 主升 | 预期成为共识，板块梯队完整 | FOMO、怕踏空 | 持有主线，不寻找同主线的低位补涨机会 |
| 高潮 | 预期被充分定价，增量枯竭 | 贪婪、过度自信 | 开始减仓，锁定利润 |
| 退潮 | 预期被兑现或被新预期取代 | 后悔、不甘心 | 预期失效走人；但若出现新预期+确认，仍可买 |

### 阶段判断规则

阶段判断是整个系统的基础，但市场是模糊的——很多时间你无法100%确定当前处于哪个阶段。这时候的处理方式：

1. **不要怕说"不确定"**。如果数据同时支持两个阶段（如分歧和退潮），就两个都列出来
2. **列两个阶段的预案**。主判断写"分歧期"→ 对应分歧期操作；副判断写"退潮期"→ 写好退潮期触发条件。两边都准备好，盘中看市场选哪边
3. **用结构指标做判断，广度指标做辅助**。分歧vs退潮的核心区分是龙头表现和板块联动，不是涨跌家数。涨跌家数可以很差（分歧期资金从边缘撤退到核心），但只要核心还在就不是退潮
4. **盘中感知信号会告诉你答案**。如果盘前判断有误，盘中数据会打你的脸。这时候不要死扛判断，要尊重市场——感知信号出现时回到核心问题："阶段判断还对吗？"

# 第三部分：交易决策

## 五、买点：预期确认

**在满足第一节「新预期 + 资金确认认可」两点前提下**，当某个预期被市场资金选择（确认），就是买点。确认即「认可」的可观察形式；下方的三维确认是把它拆成可检查的清单。

### 买点分级体系

不是所有买点都要等"分歧后再确认"。根据预期强度和确认程度，买点分为三级：

| 买点类型 | 条件 | 仓位 | 风险 | 适用场景 | 实例 |
|---------|------|------|------|---------|------|
| **A级:分歧再确认** | 分歧后重新走强+三维确认齐全 | 满额(3/2/1成) | 最低 | 产业级首选 | 3/27锂电4板+碳酸锂突破16万 |
| **B级:出现期首板** | 新预期+首次板块爆发(≥5只涨停)+三维确认 | 半额 | 中等 | 产业级/板块级 | 4/1创新药首板(礼来收购催化) |
| **C级:加速期回调** | 已确认方向首次分歧回调(非退潮)+预期依据未破 | 1/3额 | 较高 | 仅限产业级(有数据锚) | 锂电主升期首次回踩5日线 |

**核心原则**：
1. **产业级预期可以参与B级和C级买点**——因为退出信号清晰，错了也能及时走
2. **板块级只参与A级和B级**——不参与C级（退出信号不够清晰，回调可能变退潮）
3. **个股级只参与A级**——纯情绪炒作，必须等分歧后再确认
4. **任何级别的买点，三维确认都是必须的**——分级不降低确认标准，只调节仓位

### 确认的三个维度

1. **资金确认** — 资金真金白银地在投票（**以盘中可观察为准**：联动、量价、封板与承接等）
   - 板块内多股联动（不是孤票）
   - 成交量变化（注意：不是看绝对值，是看相对于该股/板块日常水平的偏离）
   - **龙虎榜**（盘后披露）：**不参与**当日盘中是否「已确认」的判断，仅作**盘后复盘**（谁在用钱、风格是否一致）与**次日预案**参考，滞后一天属正常

2. **价格确认** — 价格行为验证了预期
   - 突破关键位置（前高、平台、均线）
   - 涨停板封住（最强确认）
   - 半路放量突破（中等确认）
   - 回踩关键位企稳（低吸确认）

3. **预期依据** — 预期背后的逻辑在被验证
   - 产业预期：相关商品期货/排产数据/价格指数在验证逻辑
   - 政策预期：政策文件落地、资金配套到位
   - 个股预期：业绩数据/公告事件落地
   - 叙事预期：市场资金持续选择、板块效应维持

**三个维度中，预期依据常被漏看：** 许多人只看量价。当你的**核心预期**要靠某类数据（如商品期货、排产、政策节点）来验证时，这类数据若与故事**反向**，就不能只靠图形说「已经确认」——例如产业逻辑说碳酸锂要涨，但期货价格在跌，则无论K线多好看，**预期依据这一维都未对齐**，不算真正确认。

### 板块内部结构

看板块不能只看龙头和低位核心。要观察**内部结构**——哪些在涨、哪些在跌、资金在往板块内的哪个细分方向集中。如果板块内部分化严重（如3月27日电力板块：新能泰山4板 vs 湖南发展大跌），说明资金在板块内部做选择而非整体看好，这种板块的参与难度远高于内部一致的板块。

### 买点判断流程

```
第一步：有没有新预期出现？
  └→ 有 → 进入第二步
  └→ 没有 → 不买

第二步：资金是否认可了这个预期？（三维确认：资金+价格+预期依据）
  └→ 确认了 → 进入第三步判断买点级别
  └→ 未确认 → 继续等待

第三步：买点级别判断
  └→ A级（分歧后再确认）→ 满额仓位，进入第四步
  └→ B级（出现期首板，仅限产业级/板块级）→ 半额仓位，进入第四步
  └→ C级（加速期回调，仅限产业级）→ 1/3额仓位，进入第四步

第四步：情绪周期参考
  └→ 情绪正常/偏暖 → 按买点级别给仓位（A级满额，B级半额，C级1/3额）
  └→ 情绪较差 → 仓位收紧（各档位减半）
  └→ 情绪极差（涨停<30、恐慌性下跌）→ 即使有预期+确认，空仓更安全（安全阀）

注意：情绪周期不决定买不买。只有"极端安全阀"可以否决一个已确认的买点。
```

### 买点分级与预期强度的关系

| 预期强度 | A级买点 | B级买点 | C级买点 |
|---------|---------|---------|--------|
| 产业级 | 3成（满额） | 1.5成（半额） | 1成（1/3额） |
| 板块级 | 2成（满额） | 1成（半额） | 不参与 |
| 个股级 | 1成（满额） | 不参与 | 不参与 |

## 六、卖点：预期兑现或失效

**预期兑现 = 利好落地，预期依据完成使命，该走。**
**预期失效 = 市场否定了这个方向，必须走。**

### 预期兑现的信号
- 利好正式公告（利好出尽）
- 产业数据见顶（如碳酸锂价格开始回落且持续）
- 预期逻辑被市场充分定价（后续无增量资金）
- 所有能讲的逻辑都被讲完了，没有新的增量信息

### 预期失效的信号（更重要，必须立刻走）
- 市场选择了另一个方向（新题材崛起，老题材被抽血）
- 龙头股走弱（断板后持续大跌，不是震荡）
- 板块联动性消失（跟风股不跟了，只剩龙头孤票）
- 预期依据被新事件推翻（政策转向、行业利空、供需格局逆转）
- **产业数据破位**（如碳酸锂期货跌破关键支撑）

### 一个容易犯的错：把分歧当失效

分歧时龙头可能断板、换手率升高、跟风回落——这看起来像失效，但实际上可能是再确认前的正常筹码交换。

**判断方法**：回到确认信号。预期依据还在被验证 → 分歧，等再确认。预期依据开始被否定 → 失效，走人。（分歧vs退潮的详细对比见第四节）

## 七、仓位管理

仓位由两个因素共同决定：**预期强度**（产业级/板块级/个股级）决定上限，**情绪周期**作为参考调节仓位轻重。核心决策仍然是"预期+确认"——只有确认了才涉及仓位问题。

### 按预期强度的仓位上限

| 级别 | 仓位上限 |
|------|---------|
| 产业级 | 3成 |
| 板块级 | 2成 |
| 个股级 | 1成 |

### 情绪周期对仓位的影响（参考，非门槛）

| 情绪状态 | 仓位调节 | 说明 |
|---------|---------|------|
| 正常/偏暖 | 不调节 | 按预期强度上限给仓位 |
| 较差（封板率<50%、涨跌比差） | 各档上限减半 | 市场风险偏好低，仓位收紧 |
| 极差（涨停<30、恐慌性下跌） | **空仓** | 安全阀：资金很难认可任何预期 |

**注意**：情绪差≠不买。如果有强预期（产业级）被资金确认，即使在情绪较差的环境下，也应该买入（只是仓位减半）。只有情绪极差（安全阀）时才空仓。

### 仓位计算示例

```
产业级预期+确认，情绪正常 → 3成
产业级预期+确认，情绪较差 → 1.5成（上限减半）
产业级预期+确认，情绪极差 → 空仓（安全阀）
板块级预期+确认，情绪正常 → 2成
个股级预期+确认，情绪正常 → 1成
```

**核心原则**：仓位跟随预期强度和确认确定性走。情绪周期是参考调节，不是门槛。

**最小交易单位约束**：A股最小交易单位100股。小资金（如1万元）在高价股上可能无法满足"退潮期1成"的仓位限制——1万×1成=1000元，买不了100股融捷（70+元/股）。此时实际仓位以最小交易单位为准，但不能因此突破确定性匹配原则。

## 八、盘中感知与动态调整

### 为什么需要盘中感知？

盘前预案是基于**昨天的信息**形成的。但市场每天都在变化——
- 可能出现盘前完全没预料到的新方向（3月27日的创新药）
- 可能出现盘前未预期到的个股异动（3月27日新能泰山一字板）
- 可能盘前的情绪判断在开盘就被证伪（3月27日退潮期预期被开盘数据推翻）

**盘中感知的目的不是推翻预案，而是发现预案没覆盖到的东西。**

### 三个触发条件

当以下任一情况出现时，从"执行预案"切换到"重新思考"：

| 触发 | 具体表现 | 实例 |
|------|---------|------|
| **意料外的板块异动** | 某方向突然出现2-3只以上涨停/一字板，且该方向不在盘前预案中 | 3月27日创新药板块多只20cm涨停，盘前完全未覆盖 |
| **确认信号突变** | 产业数据（期货价格）突破/跌破关键位，或突发政策/事件 | 碳酸锂期货单日涨6%突破16万；突发的产业政策 |
| **情绪指标矛盾** | 涨停数/涨跌家数/封板率与当前情绪周期判断严重偏离 | 退潮期预期下突然78涨停、4300股飘红 |

### 触发后的快速评估（三个问题，每个<1分钟）

**1. 这是什么类型的预期？**

| 类型 | 优先级 | 持续性 | 示例 |
|------|--------|--------|------|
| 产业预期 | ⭐⭐⭐ | 长，退出信号清晰 | 碳酸锂涨价、创新药BD数据 |
| 政策预期 | ⭐⭐ | 中，看落地 | 算电协同政策、降息 |
| 情绪预期 | ⭐ | 短，退出信号最模糊 | 冰点修复博反弹 |

**2. 退出信号是否清晰？**
- 清晰（产业数据可跟踪、事件有时间节点）→ 值得认真评估，纳入决策
- 模糊（纯跟风、无具体验证手段）→ 观望为主，可能是噪音

**3. 和我当前的计划什么关系？**

| 关系 | 信号清晰 | 信号模糊 |
|------|------|------|
| **加固我的方向** | 继续执行，可考虑加仓 | 继续执行 |
| **矛盾我的方向** | ⚠️ 暂停，重新评估预期强度 | 保持原计划，可能是噪音 |
| **无关** | 记录下来，纳入明天盘前分析 | 忽略 |

### 盘中感知的核心纪律

1. **不是每个异动都需要反应。** 大部分盘中波动是噪音。只有触发了上述三个条件之一，才值得花时间评估。
2. **快速评估，不深度研究。** 盘中的职责是感知，不是研究。发现新方向 → 记录下来 → 下一天再深入研究。
3. **发现 ≠ 立刻行动。** 盘中发现新预期后，第一反应是"这和我有什么关系"，而不是"我要不要买"。
4. **预案优先。** 如果盘中新预期和盘前预案矛盾，先确认预案的假设是否已被证伪。只有证伪了才调整，不是看到新的就想上。

# 第四部分：工具与纪律

## 九、感知信号

以下指标不是机械触发条件，是**提醒你注意的感知信号**。看到信号后，思考方向是：**这对当前的预期意味着什么？**

### 换手率
- **不是**：换手率>15%就卖
- **而是**：换手率显著偏离该股近期正常水平时，说明筹码交换在加剧——回到核心问题：这个预期是被确认了还是被兑现了？如果预期还在兑现途中，换手率高说明分歧大但方向未变；如果预期已经充分定价，换手率高就是出货信号

### 涨停数 / 封板率
- **不是**：涨停>100就高潮
- **而是**：涨停数和封板率的变化**趋势**——是在上升还是在下降？结合涨跌家数一起看，判断赚钱效应是在扩散还是在收窄

### 成交额
- **不是**：缩量就危险
- **而是**：关注成交额变化和价格方向的**背离**——缩量上涨说明增量资金不足，放量下跌说明抛压加重，缩量下跌可能是抛压衰竭
- **缩量上涨的持续性**：1天可能是正常的缩量修复，连续2-3天缩量上涨则说明增量资金不愿进场，反弹可能乏力。反过来，放量且价格也涨，才是预期被真正确认

### 连板高度 / 晋级率
- **不是**：连板高度降低就是退潮
- **而是**：连板高度降低 + 晋级率下降 = 高低位切换中，观察新低位核心能否接力。连板高度降低 + 涨停数也降 = 真退潮

### 产业数据指标（商品期货/行业数据）
- 碳酸锂期货、原油、铜等大宗价格
- 行业排产数据、PMI、社融等宏观指标
- **核心用法**：与板块走势做交叉验证。期货涨+板块涨=预期确认；期货跌+板块涨=脱离基本面炒作，警惕

## 十、纪律红线

1. **没有明确预期，不买。** 随手单 = 没有预期的单 = 亏损的根源。
2. **预期未确认，不买。** 看到预期方向 ≠ 资金已经确认，等确认信号。三个维度都要看（资金、价格、预期依据）。
3. **预期失效，必走。** 不扛单，不抱幻想，不"再看看"。
4. **预期兑现，必走。** 不贪最后一段，不因为涨了就觉得还能涨。
5. **情绪极差时空仓。** 涨停<30、大面积跌停、恐慌性下跌时，资金很难认可任何预期，空仓是更安全的选择。但情绪"较差"不等于"极差"——有强预期被确认时，即使情绪较差也应该买入（仓位收紧）。
6. **不机械执行。** 任何指标都只是感知信号，回到核心问题：预期确认了吗？兑现了吗？失效了吗？
7. **盘中感知不是盘中追涨。** 发现新预期 ≠ 立刻买。评估、验证、等确认。

## 十一、每日流程

### 盘前（9:00 前）
- 判断当前市场处于主线生命周期的哪个阶段（见第四节「阶段判断规则」）
- **全市场催化扫描**：系统性检查隔夜/盘前有哪些板块出现了新催化（政策、数据、事件、公告）。不能只盯着已有持仓和已知方向——3月27日创新药方向被完全忽略，就是因为预案只覆盖了"我在看的方向"
- 梳理今日候选预期方向（产业 > 政策 > 个股 > 情绪）
- 对每个方向回答两个问题：确认信号是什么？（资金+价格+预期依据）什么情况下会证伪？
- **对每个方向预设"改主意条件"**：什么信号出现时，我会放弃/改变对某个方向的判断？
- **预设"盘中新方向关注条件"**：什么情况下我会关注一个盘前未覆盖的新方向？（如：创新药板块3只以上涨停+有权重股异动）
- 制定交易预案：如果A预期被确认→买X，如果B预期被确认→买Y
- 设定最坏情况下的止损/止盈线
- **周五专用**：如果持仓处于高位（如连续板），评估周末可能出现的消息对持仓预期的影响。风险可控→继续持有；风险较大→可减仓至舒适水平

### 盘中（9:30 - 15:00）
- 执行计划内的交易，不做计划外的交易
- 每笔交易前过检查清单：预期是什么？确认了吗（资金+价格+预期依据）？确定性有多高？仓位匹配吗？
- 注意感知信号：换手率、成交量、板块联动的变化
- 感知信号出现时 → 回到核心问题：预期状态变了吗？阶段判断还对吗？
- **盘中感知触发时**：按第八节「触发后的快速评估」流程（什么预期？退出信号清晰吗？和我的计划什么关系？）
- **记录盘中发现**：即使不操作，也要记录盘中的新预期/新异动，纳入盘后复盘

### 盘后（15:00 后）
- 复盘今日交易：每笔交易的预期是什么？确认/兑现/失效了吗？
- 更新主线生命周期判断：市场从哪个阶段进入了哪个阶段？
- 记录感知信号的准确度：哪些信号提前预示了预期变化？
- **复盘盘中感知**：今天有没有遗漏的盘中异动？盘中感知的评估是否准确？
- 调整明日计划
- **更新虚拟账户**：记录今日交易、持仓、盈亏

## 十二、系统执行纪律

1. **系统说买就买，系统说卖就卖。** 不给自己加额外的门槛或条件。三维确认齐全=买点成立，不编"再等等"、"不回落就不买"。
2. **不确定性→可持续的确定性。** 系统无法保证每笔交易都赚钱，但严格执行可以拉开时间看到复利。单笔亏损是系统的成本，不是执行错误。
3. **不在已确认的买点上犹豫。** 犹豫=给自己加条件=脱离系统=变成随意交易。
4. **复盘时的判断标准**：这笔操作是否符合系统规则？而不是这笔操作是否赚钱。赚钱但不符合系统=错误；亏钱但符合系统=正确。
