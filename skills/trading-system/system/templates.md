# 每日文件模板 v2.2

> 每个交易日产生4个文件 + 更新2个文件
> 文件放在 `daily/YYYY-MM-DD/` 下
> 所有文件基于交易系统 v2.2 规则

---

## 文件清单

| 文件 | 写入时间 | 核心内容 |
|------|---------|---------|
| 盘前分析.md | 盘前（9:00前） | 情绪判断+预期方向+交易预案+改主意条件 |
| 模拟看盘.md | 盘后（15:00后） | 多股分时对照+盘中感知+虚拟账户操作 |
| 盘后复盘.md | 盘后（15:00后） | 预期验证+判断回顾+经验教训 |
| 信号记录.md | 盘后（15:00后） | 逐条信号触发记录+遗漏分析 |
| account/portfolio.md | 盘后更新 | 持仓+盈亏+交易记录 |
| lessons/learned.md | 盘后更新 | 新增经验教训 |

---

## 五、account/portfolio.md 更新规则

每个交易日盘后更新以下部分：

1. **账户状态**：更新总资产、持仓市值、可用现金、累计盈亏
2. **当前持仓**：更新现价、盈亏%、盈亏额
3. **交易记录**：追加当日交易
4. **每日快照**：追加当日行
5. **备忘**：记录当日未操作的原因 / 关键决策理由
