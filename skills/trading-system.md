# 交易系统：预期驱动交易法则

> 核心理念：不局限模式，围绕预期做交易。形态只是确认预期的载体。
> 不是量化公式，是投资者的思维框架。每一个数值都是感知信号，不是机械触发条件。

---

## 一、预期的定义

**预期 = 市场资金正在投票的方向。**

情绪拐点出现时，资金在重新选择方向。这些候选方向就是"预期"。

### 预期的四个来源（有优先级）

| 优先级 | 来源 | 说明 | 特征 | 示例 |
|--------|------|------|------|------|
| ⭐⭐⭐ | **产业预期** | 行业趋势、供需变化、技术突破 | 有量化指标可跟踪（价格、排产、产能利用率）；有业绩/供需兜底，分歧时有锚，不容易A杀 | 碳酸锂涨价+供给收缩、固态电池量产、算力基建需求 |
| ⭐⭐ | **政策预期** | 政策方向、会议信号、监管动作 | 有方向但难量化，落地节奏不确定；"利好"到底落实没有需要持续跟踪 | AI产业政策、降息、算电协同新基建 |
| ⭐⭐ | **个股预期** | 业绩、重组、事件驱动 | 确定性取决于具体事件，需要逐案分析 | 业绩预告超预期、并购重组、产品发布 |
| ⭐ | **情绪预期** | 市场整体情绪拐点 | 没有锚，纯靠资金博弈，持续性最差，退潮时A杀概率大 | 冰点修复博反弹、退潮末端博反抽 |

**为什么产业预期优先级最高？**

产业预期有锚——有价格数据、排产数据、供需平衡表可以每天验证。有锚的方向在分歧时能找到支撑位，资金敢在分歧后重新进场。情绪预期没有锚，退潮时资金无处可寻支撑，容易无底洞。

但这不是铁律——当情绪预期叠加产业催化（如电力+算电协同政策落地），优先级可以动态提升。核心是**判断这个预期有没有"可验证的锚"**。

### 预期的两个致命问题

每次评估一个预期方向时，必须回答：

1. **这个预期的锚是什么？** — 如果答不出来，这个预期就没有根基
2. **什么情况下这个预期会被证伪？** — 如果也想不出来，说明你对这个方向的理解还不够，不该参与

## 二、主线生命周期

一条真正的主线（如锂电2026年3月），不是从"冰点"直线涨到"高潮"的。它经历的是一个完整的预期生命周期：

```
① 冰点        恐慌出清，市场否定了几乎所有方向
    ↓
② 修复        第一批资金进场试错，方向不明确，赚钱效应差
    ↓
③ 分歧        主线方向浮现，但资金意见不统一
              ↳ 这不是退潮！是筹码交换的自然阶段
              ↳ 分歧后能否重新走强，是判断主线真伪的关键
    ↓
④ 再确认      主线在分歧后重新走强 — 这恰恰是最安全的买点之一
              ↳ 经过分歧测试的确认，比未测试的确认更可靠
    ↓
⑤ 主升        共识形成，板块内梯队完整（龙头+中军+补涨）
              ↳ 持续时间最长，赚钱效应最强
    ↓
⑥ 高潮        全面爆发，后排补涨，连板高度达到顶峰
              ↳ 所有能买的都买了，增量资金开始枯竭
    ↓
⑦ 退潮        龙头见顶，板块联动消失，资金撤出
              ↳ 回到①
```

### 关键认知：分歧 ≠ 退潮

| | 分歧 | 退潮 |
|---|---|---|
| 本质 | 筹码交换，资金在"要不要继续"上犹豫 | 资金已经放弃，在离场 |
| 龙头表现 | 震荡、断板但未大跌 | 大跌、A杀 |
| 板块联动 | 部分跟风回落但核心还在 | 全面溃散 |
| 后续可能 | 重新确认 → 主升 | 继续下杀 → 新冰点 |
| 操作意义 | **观察窗口**：看主线能否扛住分歧 | **空仓窗口**：等待新冰点 |

**3月27日实战验证**：3月26日看起来像退潮（超4400跌、涨停39），但电力方向实际上是分歧（华电辽能断板但未崩，华电能源震荡），锂电在分歧后重新确认（4连板+8只涨停）。如果简单判定为退潮→空仓，就错过了锂电的再确认买点。

### 每个阶段的预期状态

| 阶段 | 预期状态 | 投资者心态 | 正确做法 |
|------|---------|-----------|---------|
| 冰点 | 预期真空期，所有方向被否定 | 恐慌、绝望 | 空仓等待，观察哪个方向率先走出冰点 |
| 修复 | 新预期开始萌芽，方向不明确 | 试探、犹豫 | 轻仓试错，不恋战 |
| 分歧 | 预期被初步选择，但资金分歧 | 焦虑、多空争论 | 不急着站队，观察哪个预期在分歧中扛住 |
| 再确认 | 预期被分歧验证，确定性提升 | 信心恢复 | **核心买点**，这是最确定的入场时机 |
| 主升 | 预期成为共识，板块梯队完整 | FOMO、怕踏空 | 持有主线，不频繁换股 |
| 高潮 | 预期被充分定价，增量枯竭 | 贪婪、过度自信 | 开始减仓，锁定利润 |
| 退潮 | 预期被兑现或被新预期取代 | 后悔、不甘心 | 全部离场，回到空仓等待 |

## 三、买点：预期确认

**当某个预期被市场资金选择（确认），就是买点。**

### 确认的三个维度

1. **资金确认** — 资金真金白银地在投票
   - 板块内多股联动（不是孤票）
   - 成交量变化（注意：不是看绝对值，是看相对于该股/板块日常水平的偏离）
   - 龙虎榜资金性质（游资/机构/量化各自的参与度）

2. **价格确认** — 价格行为验证了预期
   - 突破关键位置（前高、平台、均线）
   - 涨停板封住（最强确认）
   - 半路放量突破（中等确认）
   - 回踩关键位企稳（低吸确认）

3. **锚的确认** — 预期的量化锚在走强
   - 产业预期：相关商品期货/排产数据/价格指数在验证逻辑
   - 政策预期：政策文件落地、资金配套到位
   - 个股预期：业绩数据/公告事件兑现
   - 情绪预期：市场情绪指标修复（涨停数、涨跌比、封板率）

**三个维度中，锚的确认是最被低估的。** 如果产业逻辑说碳酸锂要涨，但期货价格在跌，那不管K线多好看，预期都没有被真正确认。

### 买点判断流程

```
当前处于主线生命周期的哪个阶段？
  └→ 冰点/修复 → 观察哪些预期方向在萌芽，等确认
  └→ 分歧 → 观察哪些预期在分歧中扛住，准备再确认买点
  └→ 再确认 → 核心买点，重仓介入确认最强的主线
  └→ 主升 → 持有，寻找同主线的低位补涨机会
  └→ 高潮 → 不追高，准备减仓
  └→ 退潮 → 空仓等待
```

## 四、卖点：预期兑现或失效

**预期兑现 = 利好落地，逻辑完成使命，该走。**
**预期失效 = 市场否定了这个方向，必须走。**

### 预期兑现的信号
- 利好正式公告（利好出尽）
- 量化锚见顶（如碳酸锂价格开始回落且持续）
- 预期逻辑被市场充分定价（后续无增量资金）
- 所有能讲的逻辑都被讲完了，没有新的增量信息

### 预期失效的信号（更重要，必须立刻走）
- 市场选择了另一个方向（新题材崛起，老题材被抽血）
- 龙头股走弱（断板后持续大跌，不是震荡）
- 板块联动性消失（跟风股不跟了，只剩龙头孤票）
- 逻辑被新事件推翻（政策转向、行业利空、供需格局逆转）
- **量化锚破位**（如碳酸锂期货跌破关键支撑）

### 一个容易犯的错：把分歧当失效

分歧时龙头可能断板、换手率升高、跟风回落——这看起来像失效，但实际上可能是再确认前的正常筹码交换。

**判断方法**：回到锚。锚还在走强 → 分歧，等再确认。锚开始走弱 → 失效，走人。

## 五、仓位管理

仓位不是按百分比硬套，是按**预期确定性**来调节：

| 确定性 | 对应阶段 | 仓位思路 |
|--------|---------|---------|
| 极低 | 冰点 | 空仓 |
| 低 | 修复 | 试探仓，能亏得起的量 |
| 中 | 分歧 | 不加仓，观察 |
| 高 | 再确认 | **这是重仓点**，确定性最高 |
| 高→中 | 主升→高潮 | 逐步减仓，锁定利润 |
| 低 | 退潮 | 清仓，回到空仓 |

**核心原则**：仓位跟随确定性走，不跟随情绪走。高潮时最兴奋，恰恰应该仓位最轻；再确认时不显眼，恰恰应该仓位最重。

## 六、感知信号

以下指标不是机械触发条件，是**提醒你注意的感知信号**。看到信号后，思考方向是：**这对当前的预期意味着什么？**

### 换手率
- **不是**：换手率>15%就卖
- **而是**：换手率显著偏离该股近期正常水平时，说明筹码交换在加剧——回到核心问题：这个预期是被确认了还是被兑现了？如果预期还在兑现途中，换手率高说明分歧大但方向未变；如果预期已经充分定价，换手率高就是出货信号

### 涨停数 / 封板率
- **不是**：涨停>100就高潮
- **而是**：涨停数和封板率的变化**趋势**——是在上升还是在下降？结合涨跌家数一起看，判断赚钱效应是在扩散还是在收窄

### 成交额
- **不是**：缩量就危险
- **而是**：关注成交额变化和价格方向的**背离**——缩量上涨说明增量资金不足，放量下跌说明抛压加重，缩量下跌可能是抛压衰竭

### 连板高度 / 晋级率
- **不是**：连板高度降低就是退潮
- **而是**：连板高度降低 + 晋级率下降 = 高低位切换中，观察新低位核心能否接力。连板高度降低 + 涨停数也降 = 真退潮

### 量化锚（商品期货/数据指标）
- 碳酸锂期货、原油、铜等大宗价格
- 行业排产数据、PMI、社融等宏观指标
- **核心用法**：与板块走势做交叉验证。期货涨+板块涨=预期确认；期货跌+板块涨=脱离基本面炒作，警惕

## 七、纪律红线

1. **没有明确预期，不买。** 随手单 = 没有预期的单 = 亏损的根源。
2. **预期未确认，不买。** 看到预期方向 ≠ 资金已经确认，等确认信号。三个维度都要看（资金、价格、锚）。
3. **预期失效，必走。** 不扛单，不抱幻想，不"再看看"。
4. **预期兑现，必走。** 不贪最后一段，不因为涨了就觉得还能涨。
5. **退潮期必空。** 退潮不是分歧，退潮是资金已经离场。
6. **不机械执行。** 任何指标都只是感知信号，回到核心问题：预期确认了吗？兑现了吗？失效了吗？

## 八、每日流程

### 盘前（9:00 前）
- 判断当前市场处于主线生命周期的哪个阶段
- 梳理今日候选预期方向（产业 > 政策 > 个股 > 情绪）
- 对每个方向回答两个问题：锚是什么？什么情况下会证伪？
- 制定交易预案：如果A预期被确认→买X，如果B预期被确认→买Y
- 设定最坏情况下的止损/止盈线

### 盘中（9:30 - 15:00）
- 执行计划内的交易，不做计划外的交易
- 每笔交易前过检查清单：预期是什么？确认了吗（资金+价格+锚）？确定性有多高？仓位匹配吗？
- 注意感知信号：换手率、成交量、板块联动的变化
- 感知信号出现时 → 回到核心问题：预期状态变了吗？

### 盘后（15:00 后）
- 复盘今日交易：每笔交易的预期是什么？确认/兑现/失效了吗？
- 更新主线生命周期判断：市场从哪个阶段进入了哪个阶段？
- 记录感知信号的准确度：哪些信号提前预示了预期变化？
- 调整明日计划

---

*这套系统的核心不是预测，而是跟随。跟随市场的选择，在确认时介入，在兑现或失效时退出。*
*不猜、不赌、不扛。*
*低频，但确定。*
