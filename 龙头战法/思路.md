核心思想
短线交易的本质不是预测市场，而是管理预期——从建立预期到兑现预期的闭环过程。所有操作都围绕“建立预期 → 买入（确认预期） → 卖出（预期兑现）”展开。市场不可预测，但资金确认的预期节点（情绪拐点，如情绪冰点修复、龙头弱转强）具有最高确定性，因为市场已用真金白银投票验证，容错率大幅提升。永远敬畏市场，只做确定性最高的1-2个机会。
- 建立预期：提前识别资金可能聚焦的方向（缩小范围，避免大海捞针）。  
- 买入=确认预期：只在市场用行动“投票确认”时介入（如分时强承接、板块联动），而非主观预判。  
- 卖出=预期兑现：当预期被充分实现或证伪时离场（如分时走弱，逻辑被证伪）。


预期管理四步法：建立 → 确认 → 兑现 → 复盘
（1）建立预期：缩小范围，聚焦高概率机会（核心起点！）
筛选铁律（缺一不可）：  
- 人气底线：无关注度=无合力基础，资金无法形成共识标的。必须进入当日人气榜前30（涨停家数、龙虎榜曝光度、股吧热度）  f
- 逻辑正宗：逻辑越硬，预期越持久；蹭热点易被证伪，必须想明白炒作逻辑，逐条列出。
- 地位突出：三选一  
  - 身位最高（连板数/趋势高度领先）  
  - 领涨性强（率先涨停带动板块）  
  - PK胜出（分时强于同题材对手）  
- 市场状态：只做两类机会  
  - 冰点修复：昨日跌停＞15家 + 连板高度≤2板 → 今日聚焦抗分歧标的  
  - 增量主升：指数放量 + 情绪高涨 → 只做身位龙  

（2）买入（确认预期）：只在资金合力形成时介入
核心原则：买入不是预判，而是确认。市场会发出“资金投票”信号，表示哪些个股超预期，此时介入容错率最高。  
关键信号：分时承接 + 板块联动
确认信号（必须同时满足）：  
- 冰点修复确认：冰点存活=资金用脚投票；今日强承接=修复预期被确认
  - 个股昨日抗住冰点分歧（如板块大跌时仅微跌/红盘）  
  - 今日分时：开盘不破昨日低点 + 均线上方放量进攻  
- 龙头弱转强确认：分歧洗盘后资金回流=主线未结束；弱转强是情绪延续的铁证
  - 龙头爆量分歧次日
  - 今日分时：高开＞3% + 30分钟内不破分时均线 + 板块联动  
- 补涨分离确认：分离=资金主动切换；补涨确认=主线生命力延续
  - 龙头断板当日
  - 补涨股分时与龙头分离（龙头水下时逆势翻红/涨停）  

（3）卖出（预期兑现）：预期实现即离场
核心原则*卖点取决于预期是否兑现，而非主观臆测。  
兑现信号：  
- 通用止损：分时破均线，30分钟内无法快速拉回，避免“该强不强即弱”
- 主升行情：板块没有活口，承接不够 
- 补涨行情：补涨走势无法强于龙头，补涨被证伪

（4）复盘：验证预期，迭代认知
- 每日必做：  
  1. 检查自选股：哪些预期被确认？哪些被证伪？  
  2. 记录信号有效性：冰点修复/弱转强等模式是否触发？  
  3. 修正标准：如“人气前30”是否需调整为“前20”？  
- 关键问题：> “我的买入是否基于资金确认，而非主观幻想？”> “卖出是否在预期兑现/证伪的节点？”

确定性最高模式：情绪拐点三板斧
为什么是拐点？ 情绪冰点/高潮后的转折点，资金已用行动确认方向，确定性远高于主升期（主升期鱼龙混杂，拐点后存活者=真龙）。  
模式1：冰点修复（新节点）
- 触发条件：
  - 情绪冰点日：高位标识度大批量负反馈，连板高度被压缩
  - 个股抗分歧，如分时均线上攻，或者因为情绪被被动走弱
- 确认信号：
  - 次日开盘30分钟内：抗分歧股分时站稳均线 + 爆量拉升
模式2：龙头弱转强（主线延续）
- 触发条件：
  - 龙头爆量分歧日，当承接强
  - 板块未大面积退潮，有活口存活
- 确认信号：
  - 次日高开＞3% + 30分钟内不破均线 + 板块联动
模式3：补涨分离（主线以补涨延续）
- 触发条件：
  - 龙头断板当日
  - 低位股或中位股主动走强（非被动跟风）
- 确认信号：
  - 补涨股分时与龙头分离
  - 次日高开强承接

模式4：机构趋势（逻辑为王）
- 触发条件
  - 逻辑非常正宗并且完全独立
  - 要么有巨大的想象力难以被证伪，要么有业绩证明

✅ 模式使用原则：  
- 只做确认点：冰点修复买次日、弱转强买次日、补涨分离买次日  
- 容错率排序：冰点修复 ＞ 龙头弱转强 ＞ 补涨分离  
- 绝对禁止：主升期追高、冰点当日博弈、无分离的补涨

案例库（10大经典模式）
示例1：主升+冰点修复（二波+破局）——恒宝科技
- 背景：6/13情绪冰点，板块大跌（稳定币概念），恒宝科技维持红盘抗分歧
- 超预期信号：
  - 6/16修复日高开5%（远超预期）
  - 开盘后爆量快速拉升站稳均线（10分钟内涨幅＞7%）
- 操作逻辑：
  - 建立预期：冰点日恒宝抗分歧（6/13红盘 vs 板块跌）
  - 确认预期：修复日高开5%+快速拉升（突破阈值：高开＞3%）
  - 兑现卖出：6/19开盘下杀，30分钟无法站稳分时均线
示例2：主升+冰点修复（非典型主升）——红宝丽
- 背景：4/15情绪冰点逆势涨停，4/16爆量2连板完成突破趋势，标识度拉满

  - 4/17高开5%（市场共识为微微高开）
  - 分时强承接：开盘30分钟内不破分时均线，爆量拉升
  - 逻辑强化：化工涨价政策落地
- 操作逻辑：
  - 建立预期：长期震荡后突破平台（4/16爆量涨停）
  - 确认预期：高开5%+强承接（突破阈值：高开＞3%）
  - 兑现卖出：4/18涨停后反复炸板，出现历史天量

  - 兑现卖出：4/22补涨龙开板即走（高度达龙头75%）

示例7：补涨+走势分离——安妮股份
