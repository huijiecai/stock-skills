# AI超短龙头交易系统设计

## 一、系统定位

**超短+低频+龙头**
- 持仓周期：1-5个交易日
- 交易频率：每周1-3次（只做高确定性机会）
- 目标标的：板块核心龙头，有标识度的票
- 核心理念：在情绪拐点，做与大盘共振的主动走强个股

## 二、核心交易模式定义

### 2.1 什么是"情绪拐点"

#### 做多拐点（底部反转信号）
1. **市场情绪从冰点开始回暖**
   - 连续调整后首次放量反弹
   - 恐慌盘抛售衰竭
   - 前期强势股开始企稳
   
2. **量化指标**
   - 涨停板数量：从<20只 → >40只
   - 昨日涨停今日表现：炸板率从>70% → <40%
   - 北向资金：从持续流出 → 开始流入
   - 两市成交额：从萎缩 → 开始放量

3. **盘面特征**
   - 早盘低开后快速拉升
   - 午后资金继续涌入（而非冲高回落）
   - 板块开始轮动（不再是单一题材）

#### 做空拐点（顶部反转信号，空仓观望）
1. **市场情绪亢奋顶点**
   - 涨停板破百只
   - 妖股横行，逻辑混乱
   - 媒体开始报道赚钱效应
   
2. **量化指标**
   - 炸板率突然飙升（>60%）
   - 高位股集体闷杀
   - 成交额创阶段新高但指数滞涨

### 2.2 什么是"与大盘共振"

**正向共振（可做）**
- 大盘放量上涨 + 板块更强
- 大盘企稳反弹 + 板块率先启动
- 大盘突破压力 + 板块加速上涨

**逆向不共振（不做）**
- 大盘大跌，板块逆市走强（短期冲高后必定回落）
- 大盘大涨，板块滞涨（说明资金不认可）

**量化判断**
- 个股涨幅 > 板块涨幅 > 大盘涨幅
- 个股换手率 > 板块平均换手
- 个股相对强度指标 RSI(个股) - RSI(指数) > 20

### 2.3 什么是"主动走强"

**主动 vs 被动**

主动走强（真强）：
- 无明显利好，却率先启动
- 量价配合，资金坚决
- 分时图呈现攻击性（快速拉升、回踩不破关键位）
- 领先板块其他个股

被动跟随（假强）：
- 跟随板块补涨
- 量能不足，拉升乏力
- 分时图犹豫（反复试探、拉升后快速回落）

**量化特征**
```
主动强度 = (个股涨幅排名 / 板块个股总数) × 资金流入排名权重 × 时间领先度
时间领先度：率先涨停得分高，尾盘跟风得分低
```

### 2.4 什么是"有标识度的核心个股"

**标识度 = 辨识度 + 号召力**

#### 辨识度
1. **题材纯正度高**
   - 主营业务契合题材（不是边缘股）
   - 概念清晰，易于传播
   - 有明确的催化剂

2. **技术位置优秀**
   - 前期充分调整（至少回撤30%以上）
   - 突破关键压力位
   - 形态好看（符合技术派审美）

3. **流通市值适中**
   - 20-100亿最佳
   - 太小：容易被操纵，不稳定
   - 太大：弹性不足，资金推动难

#### 号召力
1. **市场认知度**
   - 是否是板块内的"老大"
   - 历史上是否做过龙头
   - 是否有标志性事件

2. **龙虎榜风格**
   - 有知名游资曾经参与
   - 席位风格是短线还是中线
   - 是否有接力资金

3. **市场地位**
   - 板块内首板、二板、三板地位
   - 空间板（高度最高）or 连板（连续性最强）
   - 能否带动板块情绪

## 三、交易模式定义

### 模式A：情绪反转首板突击
**触发条件**
- 市场情绪从冰点反转（做多拐点出现）
- 某板块率先启动，且与大盘共振
- 板块内龙头个股封涨停板

**进场时机**
- 涨停封板后，观察封单强度
- 封单 > 流通市值的1% 且持续放大
- 下午1:30后依然封死，可尾盘打板

**止损**
- 次日低开>5%直接竞价出
- 次日冲高回落破均价线出
- 次日炸板不回封，立即出

**止盈**
- 次日高开>5%，冲高减仓50%
- 连板则继续持有，直到炸板或情绪转弱
- 目标：首板10%，二板20%，三板35%+

### 模式B：二板接力确认龙头
**触发条件**
- 首板次日继续强势
- 板块持续发酵，非一日游
- 个股有望成为连板龙头

**进场时机**
- 次日高开后快速封板（10:30前）
- 或盘中回踩后再次快速封板
- 确保封单强度和资金意愿

**风控**
- 只做确定性高的二板
- 一旦犹豫（多次炸板），放弃
- 不追高位二板（板块已有5板以上妖股）

### 模式C：情绪高潮顶点空仓
**触发条件**
- 市场出现做空拐点信号
- 板块妖股高位炸板
- 资金开始离场

**操作**
- 立即清仓，不做任何新仓
- 观望3-5个交易日
- 等待下一个做多拐点

## 四、AI决策流程

### 4.1 每日盘前分析（9:00-9:15）
```
输入数据：
- 隔夜外盘情况
- A股市场前一日数据（涨停数、炸板率、龙虎榜）
- 北向资金、融资融券数据
- 政策消息、突发事件

输出判断：
- 今日市场情绪预判（冰点/回暖/正常/亢奋/顶部）
- 可能启动的板块（基于题材强度、资金流向）
- 关注标的池（5-10只）
```

### 4.2 盘中实时监控（9:30-15:00）
```
每5分钟扫描：
- 涨停板实时统计
- 板块异动监控
- 重点标的分时走势
- 大盘情绪指标

触发条件出现时：
- 发出预警信号
- 给出详细分析
- 提供操作建议（买入/观望/卖出）
```

### 4.3 盘后复盘（15:30-21:00）
```
数据收集：
- 龙虎榜解读
- 涨停原因分析
- 资金流向统计
- 情绪指标计算

输出：
- 今日市场情绪评分
- 强势板块和龙头标的
- 明日操作策略
- 持仓个股分析和建议
```

## 五、关键量化指标体系

### 5.1 市场情绪指标
```python
情绪指数 = (
    涨停数量权重 * 0.3 +
    炸板率权重 * 0.25 +
    昨日涨停今日表现权重 * 0.25 +
    成交额变化权重 * 0.1 +
    北向资金权重 * 0.1
)

情绪分级：
0-20：冰点（关注拐点）
20-40：低迷（观望）
40-60：正常（可操作）
60-80：活跃（黄金期）
80-100：亢奋（警惕）
```

### 5.2 板块强度指标
```python
板块强度 = (
    板块涨幅排名 * 0.3 +
    板块资金净流入排名 * 0.3 +
    板块涨停个股数量 * 0.2 +
    板块持续性（连续几日强势）* 0.2
)
```

### 5.3 个股龙头指标
```python
龙头指数 = (
    涨停时间（越早越好）* 0.25 +
    封单强度（封单/流通市值）* 0.25 +
    换手率（适中为好，8-15%）* 0.15 +
    流通市值（20-100亿最优）* 0.10 +
    题材纯正度（主营业务匹配度）* 0.15 +
    市场地位（首板/空间板/连板）* 0.10
)
```

## 六、风险控制

### 6.1 仓位管理
- 单只个股：不超过30%
- 单个板块：不超过50%
- 总持仓：根据市场情绪调整
  - 冰点/顶部：0-20%
  - 低迷：20-40%
  - 正常：40-60%
  - 活跃：60-80%
  - 亢奋：逐步降低至40%

### 6.2 止损纪律
- 个股止损：单只最大亏损不超过7%
- 账户止损：单日最大亏损不超过3%
- 连续止损：连续3次止损，暂停操作3天

### 6.3 回撤控制
- 最大回撤预警：15%
- 最大回撤止损：20%
- 达到后：降低仓位至20%，重新评估系统

## 七、系统优化迭代

### 7.1 数据积累
- 每笔交易详细记录

### 7.2 模式优化
- 每月复盘，总结得失
- 识别哪些模式有效，哪些失效
- 根据市场变化调整参数

### 7.3 AI学习
- 用历史交易数据训练模型
- 识别更多微妙的模式
- 提高情绪拐点判断准确率
