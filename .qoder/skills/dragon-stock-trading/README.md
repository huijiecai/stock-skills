# 股票分析工具使用说明

## 功能概述

这是一个专注于A股股票分析的工具，提供：
- ✅ 实时股票行情查询
- ✅ 基础技术面分析
- ✅ 成交量和价格趋势分析
- ✅ 支撑位和阻力位识别

## 使用方法

### 基本查询
直接输入6位股票代码即可获取分析结果：

```
用户：000547怎么样
AI：[返回详细分析结果]
```

### 获取股票代码
如果不知道股票代码，可通过以下方式获取：
1. **搜索引擎**：搜索"股票名称 股票代码"
2. **财经网站**：东方财富、同花顺、雪球等
3. **证券交易软件**：各券商APP

### 常用股票代码示例
- 宁德时代：300750
- 比亚迪：002594
- 贵州茅台：600519
- 红宝丽：002165
- 航天发展：000547

## 技术分析维度

### 价格分析
- 当前价格位置分析
- 涨跌幅和波动幅度
- 开盘价与收盘价关系

### 成交量分析
- 成交活跃度评估
- 量价配合关系
- 换手率水平分析

### 技术形态
- 短期趋势判断
- 关键价位识别
- 支撑阻力分析

## API配置

### 获取API密钥
1. 访问 [iTick官网](https://www.itick.org/) 注册账号
2. 登录后进入开发者中心获取API密钥
3. iTick提供免费套餐，适合个人开发者使用

### 配置环境变量
```bash
export ITICK_API_KEY="your_api_key_here"
```

## 限制说明

⚠️ **重要限制**：
- itick API不支持股票名称模糊搜索
- 必须使用6位数字股票代码查询
- 分析结果仅供参考，不构成投资建议
