# 战法知识库

本文档记录龙头战法的核心理念和操作规则。

---

## 🎯 核心理念

**预期管理四步法**：建立预期 → 确认预期（买入） → 兑现预期（卖出） → 复盘

**只做情绪拐点，强势板块的领涨个股**

---

## 📐 筛选铁律（缺一不可）

1. **人气底线**：进入当日人气榜前30
2. **逻辑正宗**：匹配当前热点逻辑，逻辑强度≥4星
3. **地位突出**：三选一
   - 身位最高（连板数领先）
   - 领涨性强（率先涨停）
   - PK胜出（同题材中最强）
4. **市场状态**：只做两类机会
   - 冰点修复：昨日跌停>15家 + 连板≤2板
   - 增量主升：连板高度≥3板

---

## 🎯 确定性最高的三种情绪拐点模式

### 1. 冰点修复（新节点）

- **触发条件**：情绪冰点 + 个股抗分歧
- **确认信号**：次日高开 + 站稳均线 + 爆量拉升
- **操作要点**：
  - 选择逻辑正宗、身位突出的个股
  - 等待次日高开确认，不追涨停板
  - 站稳均线后介入，破均线立即止损

### 2. 龙头弱转强（主线延续）

- **触发条件**：龙头爆量分歧 + 板块有活口
- **确认信号**：次日高开>3% + 30分钟不破均线
- **操作要点**：
  - 关注龙头分歧日表现
  - 次日高开>3%是强势信号
  - 30分钟内破均线止损

### 3. 补涨分离（主线以补涨延续）

- **触发条件**：龙头断板 + 低位股主动走强
- **确认信号**：补涨股与龙头分离 + 次日强承接
- **操作要点**：
  - 选择同逻辑内的低位股
  - 必须是核心受益股（非蹭热点）
  - 补涨走势要强于龙头

---

## 🚪 卖出信号

### 通用止损

- **分时破均线**：30分钟内无法快速拉回 → 立即止损
- **冲高回落**：涨停打开后无法再封 → 减仓或离场

### 主升行情

- **板块退潮**：板块没有活口，承接不够 → 降低仓位
- **龙头断板**：龙头断板且板块跟风弱 → 警惕退潮

### 补涨行情

- **走势弱于龙头**：补涨股走势无法强于龙头 → 离场
- **龙头二次断板**：龙头彻底走弱 → 补涨逻辑破裂

## 🌡️ 市场状态判断

### 冰点修复（做修复）

- **特征**：昨日跌停≥15家 + 连板高度≤2板
- **情绪**：极度悲观，市场冰点
- **策略**：只做抗分歧的逻辑正宗个股
- **重点关注**：冰点修复模式

### 增量主升（只做龙头）

- **特征**：连板高度≥3板
- **情绪**：强势上涨，赚钱效应强
- **策略**：只做身位最高的龙头，避免追高补涨
- **重点关注**：龙头弱转强、补涨分离

### 震荡观望（等待拐点）

- **特征**：跌停<15家 + 连板<3板
- **情绪**：方向不明，机会不清晰
- **策略**：观望为主，等待情绪拐点
- **重点关注**：耐心等待

## 📚 重要概念解释

### 人气排名

基于以下因素综合评分：
- 连板天数（权重最高）
- 首板时间（早于9:35加分）
- 龙虎榜上榜（加分）

### 逻辑正宗

- 股票概念与热点逻辑高度匹配
- 是逻辑的核心受益方，非蹭热点
- 逻辑强度≥4星

### 身位

- 连板高度在同题材中的排名
- 身位越高，市场认可度越强
- "身位最高"=连板数最多

### 地位判断

- **龙头**：身位最高 + 逻辑最正宗
- **次龙头**：身位仅次于龙头，或某细分逻辑龙头
- **补涨**：龙头已高位，低位股启动跟涨
- **跟风**：无明确逻辑优势，蹭热点

## ⚠️ 常见错误

### ❌ 追高

- **错误**：看到涨停就追，不看市场状态
- **正确**：只做确认信号出现后的介入

### ❌ 逻辑不正宗

- **错误**：买入概念边缘的蹭热点股
- **正确**：只买核心受益股（逻辑库明确标注）

### ❌ 不止损

- **错误**：破均线后继续持有，期待反抽
- **正确**：破均线30分钟不回，立即止损

### ❌ 盲目补涨

- **错误**：龙头断板后继续买补涨股
- **正确**：龙头断板警惕退潮，补涨股要谨慎
