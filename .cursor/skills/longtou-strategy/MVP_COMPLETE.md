## 🎓 战法核心

**预期管理四步法**：
1. 建立预期（筛选符合条件的股票）
2. 确认预期（买入：等待确认信号）
3. 兑现预期（卖出：预期实现即离场）
4. 复盘（验证预期，迭代认知）

**四种确定性模式**：
1. 冰点修复（新节点）
2. 龙头弱转强（主线延续）
3. 补涨分离（主线以补涨延续）
4. 机构趋势（逻辑为王）
