## 一、官方信息渠道（最高权重）

### 1. 上市公司官网

**用途**：
- 公司简介、发展历程
- 主营业务、产品介绍
- 技术实力、核心专利
- 客户案例、合作伙伴

**访问方法**：
```
Google搜索：[公司名称] 官网
或直接访问：http://www.[公司简拼].com

示例：
- 科大讯飞：https://www.iflytek.com/
- 比亚迪：https://www.byd.com/
```

**关键页面**：
- 首页 > 关于我们 > 公司简介
- 首页 > 产品中心 / 业务介绍
- 首页 > 投资者关系 > 公司公告

### 2. 巨潮资讯网（年报、季报、公告）

**网址**：http://www.cninfo.com.cn

**用途**：
- 年报、半年报、季报
- 重大事项公告
- 股东大会决议
- 财务数据详情

**访问方法**：
```
1. 进入巨潮资讯网
2. 搜索框输入股票代码或公司名称
3. 选择"定期报告" > "年度报告" / "季度报告"
4. 下载最新报告PDF

快速搜索：
巨潮资讯网 [股票代码] 年报
```

**年报重点章节**：
- 第三节 管理层讨论与分析（经营情况）
- 第四节 公司治理（主营业务分析）
- 第十节 财务报告（详细财务数据）

### 3. 交易所网站

**上海证券交易所**：http://www.sse.com.cn
**深圳证券交易所**：http://www.szse.cn
**北京证券交易所**：http://www.bse.cn

**用途**：
- 上市公司信息披露
- 交易规则、监管公告
- 停复牌信息

## 三、市场数据渠道（参考权重）

### 1. 东方财富

**网址**：http://www.eastmoney.com

**个股页面**：
```
格式：http://quote.eastmoney.com/[市场][股票代码].html

示例：
- 科大讯飞：http://quote.eastmoney.com/sz002230.html
- 比亚迪：http://quote.eastmoney.com/sz002594.html

市场代码：
- sz = 深圳
- sh = 上海
```

**可查询信息**：
- 实时行情、K线图
- 财务数据（营收、利润等）
- 主力资金流向
- 概念板块归属
- F10资料（公司资料、财务分析）

**F10资料使用**：
```
个股页面 > 点击"F10资料"

关键模块：
- 公司概况：基本信息、主营构成
- 财务分析：营收、利润、财务比率
- 股东研究：股东结构、持股变化
- 行业分析：行业地位、竞争对手
```

```
Google搜索 > 工具 > 任何时间 > 选择"过去一年"
```

## 五、数据质量评估

### 信息可信度等级

| 等级 | 来源类型 | 可信度 | 使用建议 |
|------|----------|--------|----------|
| ⭐⭐⭐⭐⭐ | 公司官网、年报 | 最高 | 优先使用，直接引用 |
| ⭐⭐⭐⭐ | 券商研报、行业协会 | 高 | 可信，但需注意时效性 |
| ⭐⭐⭐ | 财经媒体、专业网站 | 中等 | 参考，需交叉验证 |
| ⭐⭐ | 个人博客、论坛 | 低 | 仅供参考，不作依据 |
| ⭐ | 自媒体、营销文 | 很低 | 谨慎对待，可能有偏见 |

### 数据时效性检查

**检查清单**：
- [ ] 数据发布日期是否标注
- [ ] 是否为最新季度/年度数据
- [ ] 是否有更新的数据来源
- [ ] 行业数据是否过时（超过1年）

**最佳实践**：
```
✅ 使用最新年报数据（如2025年报）
✅ 补充最新季报数据（如2026Q1）
✅ 标注数据查询日期

❌ 使用2年前的数据
❌ 不标注数据时间
```

## 六、特定信息获取路径

### 获取财务数据

**路径1: 年报**
```
巨潮资讯网 > 搜索公司 > 年度报告 > 
第十节财务报告 > 合并利润表/合并资产负债表
```

**路径2: 东方财富F10**
```
东方财富 > 个股页面 > F10资料 > 财务分析 > 
主要指标/财务摘要
```

### 获取营收构成

**路径: 年报**
```
年报 > 第三节 管理层讨论与分析 > 
主营业务分析 > 分行业、分产品情况
```

### 获取市占率数据

**路径1: 券商研报**
```
搜索：[公司名称] 市场份额 研报
或
搜索：[行业名称] 竞争格局 研报
```

**路径2: 行业协会**
```
行业协会网站 > 统计数据 > 行业排名
```

### 获取技术专利

**路径1: 公司官网**
```
官网 > 关于我们 > 技术创新/研发实力
或
官网 > 投资者关系 > 公司介绍
```

**路径2: 国家知识产权局**
```
网址：http://www.cnipa.gov.cn
专利检索 > 输入公司名称
```

## 八、常见问题

**Q: 找不到公司年报怎么办？**
A: 
1. 确认公司是否已上市（上市时间）
2. 巨潮资讯网搜索股票代码（而非公司名称）
3. 新上市公司可能只有招股说明书

**Q: 券商研报需要付费吗？**
A: 
- 东方财富研报中心：大部分免费
- 部分深度报告可能需要付费
- 可查看报告摘要和核心观点

**Q: 如何判断信息是否过时？**
A: 
- 财务数据：使用最新季报/年报
- 行业数据：不超过1年
- 技术趋势：不超过6个月
- 政策文件：查看发布日期

**Q: 不同来源数据冲突怎么办？**
A: 
- 优先使用官方数据（年报、公告）
- 交叉验证多个来源
- 选择更权威的机构数据
- 注明数据来源和时间
