# 07 存储芯片

> 数据的"仓库"，AI时代的核心瓶颈之一

---

## 一、存储芯片是什么？

### 通俗理解

> **存储芯片 = 电子产品的"仓库"**
>
> 用来存放数据，不同类型的存储芯片就像不同用途的仓库。

### 存储芯片在系统中的位置

```
┌─────────────────────────────────────────────────────────────┐
│                    计算机存储层次                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   速度快                                                     │
│     ↑      ┌─────────┐                                      │
│     │      │ 寄存器  │   ← CPU内部，最快                     │
│     │      └────┬────┘                                      │
│     │           │                                           │
│     │      ┌────┴────┐                                      │
│     │      │ 缓存L1/2│   ← SRAM，很快很贵                   │
│     │      └────┬────┘                                      │
│     │           │                                           │
│     │      ┌────┴────┐                                      │
│     │      │  内存   │   ← DRAM，快且较贵                   │
│     │      └────┬────┘                                      │
│     │           │                                           │
│     │      ┌────┴────┐                                      │
│     ↓      │ 固态硬盘│   ← NAND Flash，较快较便宜           │

│                 │                                           │

│                                                             │

```

## 二、存储芯片分类

### 主要类型对比

| 类型 | 全称 | 特点 | 速度 | 断电 | 用途 |
|------|------|------|------|------|------|
| **SRAM** | 静态随机存储器 | 最快、最贵 | ⚡⚡⚡⚡⚡ | 丢失 | CPU缓存 |
| **DRAM** | 动态随机存储器 | 快、需刷新 | ⚡⚡⚡⚡ | 丢失 | 内存条 |
| **HBM** | 高带宽存储器 | 超高带宽 | ⚡⚡⚡⚡⚡ | 丢失 | AI芯片 |
| **NAND** | 闪存 | 断电保存 | ⚡⚡ | 保存 | SSD |
| **NOR** | 闪存 | 可执行代码 | ⚡⚡⚡ | 保存 | 启动程序 |

### DRAM vs NAND

```
DRAM (内存)                      NAND (闪存)
─────────────────               ─────────────────
• 断电数据丢失                   • 断电数据保存
• 速度极快                       • 速度较慢
• 每比特成本高                   • 每比特成本低
• 用于临时存储                   • 用于永久存储
• 代表：DDR4/DDR5内存条          • 代表：SSD固态硬盘
```

## 五、HBM详解（重点！）

### 什么是HBM？

> **HBM = High Bandwidth Memory = 高带宽存储器**
>
> 专门为AI/GPU设计的超高带宽内存

### 为什么AI需要HBM？

```
AI模型训练/推理的瓶颈：

GPU算力 ████████████████████████████████ 足够
内存带宽 ████████ 不够！

传统GDDR带宽：~512GB/s
HBM3e带宽：~1TB/s+ （翻倍！）

结论：没有HBM，GPU性能发挥不出来
```

### HBM结构

```
┌─────────────────────────────────────────┐
│              HBM结构图                   │
├─────────────────────────────────────────┤
│                                         │
│    ┌───┐ ┌───┐ ┌───┐ ┌───┐           │
│    │D12│ │D12│ │D12│ │D12│  ← 12层DRAM │
│    ├───┤ ├───┤ ├───┤ ├───┤           │
│    │...│ │...│ │...│ │...│           │
│    ├───┤ ├───┤ ├───┤ ├───┤           │
│    │D1 │ │D1 │ │D1 │ │D1 │           │
│    └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘           │
│      │TSV  │TSV  │TSV  │TSV  ← 硅通孔  │

### HBM为什么这么难做？

| 难点 | 说明 |
|------|------|
| **3D堆叠** | 要把12层DRAM精确堆叠 |
| **TSV工艺** | 硅通孔直径仅5-10微米 |
| **散热** | 堆叠后散热困难 |
| **良率** | 一层坏整颗报废 |
| **封装** | 需要CoWoS等先进封装配合 |

结论：NAND追赶最快，DRAM次之，HBM差距最大
```

## 七、存储周期

### 存储芯片的周期性

```
存储芯片是强周期行业：

价格上涨 → 厂商扩产 → 供过于求 → 价格下跌 → 减产 → 供不应求 → 价格上涨
    ↑                                                              │
    └──────────────────────────────────────────────────────────────┘

周期通常3-4年一轮
```

### 当前周期位置（2026年初）

```
2022-2023: 下行周期（价格暴跌、库存高企）
2024: 周期底部（去库存完成）
2025-2026: 上行周期（AI需求拉动、价格回升）

当前位置：上行周期中
```

## 九、投资逻辑

### 核心逻辑

```
1. AI带动HBM需求爆发
   • AI芯片必须配HBM
   • HBM供不应求，价格上涨
   • 利好：SK海力士、三星、美光

2. 存储周期上行
   • 价格见底回升
   • 库存去化完成
   • 利好：全行业

3. 国产替代
   • 长江存储、长鑫存储技术进步
   • 国内客户支持
   • 利好：兆易创新、澜起科技
```

### 风险提示

| 风险 | 说明 |
|------|------|
| ⚠️ 周期风险 | 存储价格波动大 |
| ⚠️ 技术差距 | HBM国产化困难 |
| ⚠️ 制裁风险 | 设备材料受限 |
| ⚠️ 竞争加剧 | 国内厂商相互竞争 |
