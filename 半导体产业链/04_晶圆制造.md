## 一、晶圆制造总览

### 1️⃣ 是什么？

> **晶圆制造 = 把芯片图纸"印刷"到硅片上**
>
> 就像把设计图变成实体建筑，晶圆制造把芯片设计变成真正的芯片。

```
┌─────────────────────────────────────────┐
│              晶圆（Wafer）               │
├─────────────────────────────────────────┤
│                                         │
│         ┌─────────────────┐            │
│        /                   \           │
│       /   ┌──┬──┬──┬──┐    \          │
│      │    │芯│芯│芯│芯│     │         │
│      │    ├──┼──┼──┼──┤     │  ← 一片晶圆上│
│      │    │片│片│片│片│     │    有几百颗芯片│
│      │    ├──┼──┼──┼──┤     │         │
│      │    │芯│芯│芯│芯│     │         │
│       \   └──┴──┴──┴──┘    /          │
│        \                   /           │
│         └─────────────────┘            │
│                                         │
│  直径：200mm (8寸) 或 300mm (12寸)      │
│  厚度：约 0.7mm                          │
│  材质：高纯度单晶硅                       │
└─────────────────────────────────────────┘
```

**中芯国际面临的挑战**

```
1. 设备受限
   • 无法获得EUV光刻机
   • DUV设备也受限
   • 7nm以下量产困难

2. 客户流失风险
   • 华为被制裁后订单减少
   • 高端客户可能流向台积电

3. 技术追赶
   • 与台积电差距2-3代
   • 良率提升需要时间

但也有机会：
   • 成熟工艺需求大（汽车、IoT）
   • 国产替代政策支持
   • 国内客户支持
```

## 二、制程节点详解

### 1️⃣ 是什么？

> **制程节点 = 芯片上最小结构的尺寸**
>
> 数字越小，晶体管越密集，性能越强，但制造越难。

### 2️⃣ 制程演进

| 制程 | 年代 | 晶体管密度 | 代表产品 | 能做的厂商 |
|------|------|------------|----------|------------|
| 180nm | 2000 | ~5百万/mm² | 奔腾III | 很多 |
| 90nm | 2004 | ~2千万/mm² | 奔腾4 | 很多 |
| 28nm | 2011 | ~1亿/mm² | 骁龙800 | 台积电、三星、中芯 |
| 14nm | 2014 | ~2亿/mm² | 骁龙810 | 台积电、三星、Intel、中芯 |
| **7nm** | 2018 | ~5亿/mm² | A12 | 台积电、三星 |
| **5nm** | 2020 | ~8亿/mm² | A14、M1 | 台积电、三星 |
| **3nm** | 2022 | ~15亿/mm² | A17、M3 | **仅台积电、三星** |
| **2nm** | 2025+ | ~20亿/mm² | 规划中 | 仅台积电、三星、Intel |

### 3️⃣ 制程与光刻机的关系

| 制程 | 需要的光刻机 | 能做的厂商 |
|------|--------------|------------|
| 28nm+ | ArF干式/浸没 | 很多厂商 |
| 14-7nm | ArF浸没（多重曝光）| 台积电、三星、Intel、中芯 |
| **7nm以下** | **EUV光刻机** | **仅台积电、三星** |

```
⚠️ 7nm是分水岭！没有EUV，就无法高效量产7nm以下芯片。
```

## 四、晶圆制造流程

### 简化流程

```
┌─────────────────────────────────────────────────────────────┐
│                   晶圆制造主要步骤                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 氧化    →  在硅片表面生成氧化层                          │
│      ↓                                                      │
│  2. 光刻    →  把电路图案转移到硅片上（最关键！）            │
│      ↓                                                      │
│  3. 刻蚀    →  去除不需要的材料                              │
│      ↓                                                      │
│  4. 离子注入 → 掺入杂质，改变导电性                          │
│      ↓                                                      │
│  5. 薄膜沉积 → 沉积各种材料层                                │
│      ↓                                                      │
│  6. CMP     →  平坦化处理                                    │
│      ↓                                                      │
│  7. 金属互连 → 制作金属连线                                  │
│      ↓                                                      │
│  重复上述步骤 50-100 次 → 完成一颗芯片                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 为什么晶圆厂这么贵？

| 成本项 | 金额 | 说明 |
|--------|------|------|
| 建厂 | $100-200亿 | 一座先进fab |
| 设备 | $50-100亿 | 光刻机等 |
| 运营 | $20-50亿/年 | 电、水、人工 |
| 研发 | $30-50亿/年 | 新工艺开发 |

```
一座先进晶圆厂总投资：200-400亿美元
这是全球资本最密集的产业之一！
```

### 核心逻辑

```
1. 国产替代
```
