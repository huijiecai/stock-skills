# 04 晶圆制造

> 把芯片设计变成实物的"施工队"，也是最难、最贵的环节

---

## 一、什么是晶圆制造？

### 通俗理解

> **晶圆制造 = 把芯片图纸"印刷"到硅片上**
>
> 就像把设计图变成实体建筑，晶圆制造把芯片设计变成真正的芯片。

### 晶圆是什么？

```
┌─────────────────────────────────────────┐
│              晶圆（Wafer）               │
├─────────────────────────────────────────┤
│                                         │
│         ┌─────────────────┐            │
│        /                   \           │
│       /   ┌──┬──┬──┬──┐    \          │
│      │    │芯│芯│芯│芯│     │         │
│      │    ├──┼──┼──┼──┤     │  ← 一片晶圆上│
│      │    │片│片│片│片│     │    有几百颗芯片│
│      │    ├──┼──┼──┼──┤     │         │
│      │    │芯│芯│芯│芯│     │         │
```

## 二、制程节点

### 什么是制程节点？

> **制程节点 = 芯片上最小结构的尺寸**
>
> 数字越小，晶体管越密集，性能越强，但制造越难。

### 制程演进

| 制程 | 年代 | 晶体管密度 | 代表产品 |
|------|------|------------|----------|
| 180nm | 2000 | ~5百万/mm² | 奔腾III |
| 90nm | 2004 | ~2千万/mm² | 奔腾4 |
| 65nm | 2006 | ~3千万/mm² | Core 2 |
| 45nm | 2008 | ~5千万/mm² | Core i7 |
| 28nm | 2011 | ~1亿/mm² | 骁龙800 |
| 14nm | 2014 | ~2亿/mm² | 骁龙810 |
| **7nm** | 2018 | ~5亿/mm² | A12、骁龙855 |
| **5nm** | 2020 | ~8亿/mm² | A14、M1 |
| **3nm** | 2022 | ~15亿/mm² | A17、M3 |
| **2nm** | 2025+ | ~20亿/mm² | 规划中 |

### 制程与光刻机的关系

| 制程 | 需要的光刻机 | 能做的厂商 |
|------|--------------|------------|
| 28nm+ | ArF干式/浸没 | 很多厂商 |
| 14-7nm | ArF浸没（多重曝光）| 台积电、三星、Intel、中芯 |
| **7nm以下** | **EUV光刻机** | **仅台积电、三星** |

⚠️ **7nm是分水岭！没有EUV，就无法高效量产7nm以下芯片。**

## 四、台积电的统治地位

### 为什么台积电这么强？

| 优势 | 说明 |
|------|------|
| **技术领先** | 制程领先三星6-12个月 |
| **良率最高** | 同样制程，良率比对手高 |
| **产能最大** | 全球最大的代工产能 |
| **客户信任** | 不与客户竞争（纯代工） |
| **生态完善** | 设计服务、IP、封装一条龙 |

## 五、中国大陆晶圆厂

### 主要厂商

| 公司 | 最先进制程 | 产能（12寸等效）| 定位 |
|------|------------|------------------|------|
| **中芯国际** | 7nm(有争议) | ~75万片/月 | 综合代工龙头 |
| **华虹** | 28nm | ~30万片/月 | 特色工艺（功率、嵌入式）|
| **晶合集成** | 55nm | ~10万片/月 | 面板驱动 |
| **合肥长鑫** | 1Xnm | DRAM专用 | 存储芯片 |
| **长江存储** | 232层 | NAND专用 | 存储芯片 |

### 中芯国际的处境

```
中芯国际面临的挑战：

1. 设备受限
   • 无法获得EUV光刻机
   • DUV设备也受限
   • 7nm以下量产困难

2. 客户流失风险
   • 华为被制裁后订单减少
   • 高端客户可能流向台积电

3. 技术追赶
   • 与台积电差距2-3代
   • 良率提升需要时间

但也有机会：
   • 成熟工艺需求大（汽车、IoT）
   • 国产替代政策支持
   • 国内客户支持
```

## 六、晶圆制造流程

### 简化流程

```
┌─────────────────────────────────────────────────────────────┐
│                   晶圆制造主要步骤                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 氧化    →  在硅片表面生成氧化层                          │
│      ↓                                                      │
│  2. 光刻    →  把电路图案转移到硅片上（最关键！）            │
│      ↓                                                      │
│  3. 刻蚀    →  去除不需要的材料                              │
│      ↓                                                      │
│  4. 离子注入 → 掺入杂质，改变导电性                          │
│      ↓                                                      │
│  5. 薄膜沉积 → 沉积各种材料层                                │
│      ↓                                                      │
│  6. CMP     →  平坦化处理                                    │
│      ↓                                                      │
│  7. 金属互连 → 制作金属连线                                  │
│      ↓                                                      │
│  重复上述步骤 50-100 次 → 完成一颗芯片                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 八、投资逻辑

### 核心逻辑

```
1. 国产替代
   • 国内芯片设计公司被迫选择中芯/华虹
   • 成熟工艺需求大

2. 产能扩张
   • 中芯/华虹持续扩产
   • 设备国产化带来成本下降

3. 政策支持
   • 大基金持续投资
   • 地方政府补贴
```

### 风险提示

| 风险 | 说明 |
|------|------|
| ⚠️ 设备受限 | 先进制程设备被禁 |
| ⚠️ 盈利压力 | 折旧大、良率爬坡 |
| ⚠️ 地缘政治 | 制裁可能加剧 |
| ⚠️ 周期波动 | 行业周期性明显 |
