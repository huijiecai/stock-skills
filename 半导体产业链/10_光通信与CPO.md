## 一、光通信总览

### 1️⃣ 是什么？

> **光通信 = 用光传输数据**
>
> 数据中心里，服务器之间靠光纤连接，光模块负责把电信号转成光信号。

```
数据传输流程：

服务器A                                      服务器B
┌─────┐   电信号   ┌───────┐  光信号  ┌───────┐   电信号   ┌─────┐
│芯片 │ ─────────→│光模块 │─────────→│光模块 │─────────→│芯片 │
└─────┘            └───────┘   光纤   └───────┘            └─────┘
                   (电→光)              (光→电)
```

## 三、CPO光电共封装

### 1️⃣ 是什么？

> **CPO = 把光模块和芯片"打包"在一起**
>
> 让数据传输更快、功耗更低的新技术。

```
传统方案（可插拔光模块）：
┌─────────┐    电线    ┌─────────┐     光纤     ┌─────────┐
│  芯片   │ ─────────→ │ 光模块  │ ───────────→ │  网络   │
└─────────┘            └─────────┘              └─────────┘
    │                      │
    │                      └── 分离式，距离远
    └── 电信号损耗大

CPO方案（光电共封装）：
┌───────────────────────┐     光纤     ┌─────────┐
│  芯片 + 光引擎(一体)  │ ───────────→ │  网络   │
└───────────────────────┘              └─────────┘
    │
    └── 集成在一起，电信号距离极短，损耗小
```

### 2️⃣ 市场规模

```
CPO市场规模预测：

2025年: ████ ~10亿美元
2027年: ████████████████ ~50亿美元
2030年: ████████████████████████████████ ~150亿美元

年增速：约 50%+（最快增长的光通信领域）
```

### 3️⃣ CPO的优势

| 指标 | 传统方案 | CPO | 提升 |
|------|----------|-----|------|
| **功耗** | 15pJ/bit | 5pJ/bit | **-70%** |
| **带宽密度** | 低 | 高 | **2-4倍** |
| **延迟** | 较高 | 低 | 显著降低 |
| **成本** | 高 | 低（长期）| 有优势 |

### 4️⃣ CPO结构

```
┌─────────────────────────────────────────────────────────────┐
│                      CPO封装结构                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────┐              │
│   │                                         │              │
│   │    ┌─────────┐       ┌─────────┐       │              │
│   │    │ 交换芯片 │       │  光引擎 │       │              │
│   │    │  (ASIC) │←─────→│(Optical │       │              │
│   │    │         │ 极短  │ Engine) │       │              │
│   │    └─────────┘ 电连接└────┬────┘       │              │
│   │                           │            │              │
│   └───────────────────────────┼────────────┘              │
│                               │                            │
│                          光纤接口                           │
│                               │                            │
│                          ═════╧═════                       │
│                            光纤                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘

关键点：光引擎与芯片集成在同一封装内，电信号距离极短
```

### 5️⃣ CPO vs 传统光模块

| 对比项 | 可插拔光模块 | CPO |

| 电信号损耗 | 高 | **极低** |
| 可维护性 | 可热插拔 | 需特殊设计 |

## 六、投资逻辑总结

### 核心逻辑

```
1. AI驱动带宽升级
   • AI集群需要超高带宽互连
   • 800G→1.6T→3.2T升级
   • 利好：中际旭创、新易盛

2. CPO是未来趋势
   • 传统光模块功耗/密度受限
   • CPO是必然趋势
   • 利好：光模块厂转型光引擎

3. 国产替代
   • 光芯片国产化进程加速
   • 光模块国内厂商已领先
   • 利好：光迅科技、源杰科技
```

### 投资节奏

| 阶段 | 时间 | 投资重点 |
|------|------|----------|
| **当前** | 2025-2026 | 800G/1.6T光模块放量 |
| **中期** | 2027-2028 | CPO光引擎开始放量 |
| **长期** | 2029+ | CPO大规模应用 |

### 风险提示

| 风险 | 说明 |
|------|------|
| ⚠️ 进度不及预期 | CPO商用化可能延迟 |
| ⚠️ 技术路线变化 | 可能出现替代方案 |
| ⚠️ 竞争加剧 | 国内厂商相互竞争 |
| ⚠️ 估值偏高 | AI概念已有炒作 |

### A股标的汇总

| 环节 | 核心标的 |
|------|----------|
| 光模块 | **中际旭创**、**新易盛** |
| 光芯片 | **光迅科技**、**源杰科技** |
| 无源器件 | **天孚通信**、太辰光 |

## 七、与其他环节的关系

### CPO不是所有芯片都需要

```
需要CPO/高速光模块的场景：
✅ AI芯片（数据中心级）
✅ 交换芯片（高端）
✅ 高性能计算

不需要CPO的场景：
❌ 手机芯片

❌ 普通存储芯片

```
