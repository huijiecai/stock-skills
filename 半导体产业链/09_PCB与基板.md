# 09 PCB与基板

> 电子产品的"骨架"，连接芯片与系统的桥梁

---

## 一、PCB总览

### 1️⃣ 是什么？

> **PCB = 电子产品的"骨架"和"血管"**
>
> 承载芯片并连接各个元器件，所有电子产品都需要PCB。

```
┌─────────────────────────────────────────────────────────────┐
│                    PCB的作用                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                      │
│   │ CPU │  │内存 │  │电源 │  │接口 │                      │
│   └──┬──┘  └──┬──┘  └──┬──┘  └──┬──┘                      │
│      │        │        │        │                          │
│   ═══╧════════╧════════╧════════╧═══════════               │
│              PCB电路板（连接所有芯片）                        │
│   ════════════════════════════════════════                  │
│                                                             │
│   作用：                                                     │
│   1. 机械支撑 - 承载所有元器件                               │
│   2. 电气连接 - 连接各芯片，传输信号                         │
│   3. 电源分配 - 给各芯片供电                                 │
│   4. 散热 - 帮助散发热量                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3️⃣ PCB分类

| 类型 | 层数 | 特点 | 应用 | 代表厂商 |
|------|------|------|------|----------|
| 单面板 | 1层 | 最简单 | 遥控器 | 很多 |
| 双面板 | 2层 | 两面走线 | 简单产品 | 很多 |
| 多层板 | 4-8层 | 主流 | 手机、电脑 | 鹏鼎、景旺 |
| 高多层板 | 10-20层 | 复杂 | 服务器 | 沪电、深南 |
| **超高多层板** | 20层+ | **最复杂** | **AI服务器** | **胜宏科技** |
| HDI板 | - | 高密度 | 手机、服务器 | **胜宏科技**、深南 |
| FPC | - | 柔性 | 手机内部 | 鹏鼎、东山 |
| IC基板 | - | 封装用 | 芯片封装 | 深南、兴森 |

## 二、PCB细分领域与龙头

### 细分领域一览

| 细分领域 | 技术要求 | 核心龙头 | 第二梯队 |
|----------|----------|----------|----------|
| **AI服务器/算力** | 高多层（20层+） | **胜宏科技** | 沪电、深南 |

### 各细分详解

#### 1️⃣ AI服务器/算力PCB

| 指标 | 说明 |

| **是什么** | AI服务器用的高端PCB |

| 指标 | 说明 |
|------|------|

| **龙头** | **沪电股份**（002463） |
| **看点** | 华为核心供应商，5G建设 |

#### 3️⃣ IC封装基板（最高端）

| 指标 | 说明 |

| **看点** | ABF基板国产替代空间大 |

#### 4️⃣ 消费电子PCB/FPC

## 三、IC基板详解（重点！）

### 1️⃣ 是什么？

> **IC基板 = 芯片和PCB之间的"桥梁"**
>
> 先进封装中的关键材料，技术要求最高。

```
┌─────────────────────────────────────────────────────────────┐
│                 IC基板在系统中的位置                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    ┌─────────┐                              │
│                    │  芯片   │                              │
│                    │  (Die)  │                              │
│                    └────┬────┘                              │
│                         │ 焊球                              │
│                    ┌────┴────┐                              │
│                    │ IC基板  │  ← 这里！                    │
│                    └────┬────┘                              │
│                         │ BGA焊球                           │
│                    ┌────┴────┐                              │
│                    │   PCB   │                              │
│                    └─────────┘                              │
│                                                             │
│   IC基板是芯片到PCB的"过渡层"                               │
│   线宽/线距要求比普通PCB高得多                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2️⃣ 市场规模

```
IC基板市场规模（2025年）：

市场规模：约 180亿美元
年增速：约 15-20%（AI芯片带动）
```

### 3️⃣ IC基板分类

| 类型 | 全称 | 应用 | 难度 | 国产化率 |
|------|------|------|------|----------|
| **BT基板** | Bismaleimide Triazine | 普通封装 | 中等 | ~30% |
| **ABF基板** | Ajinomoto Build-up Film | CPU、GPU、AI芯片 | **最高** | **<10%** |
| **FC-BGA** | Flip-Chip BGA | 高端封装 | 高 | <20% |

### 4️⃣ 竞争格局

```

台湾(欣兴、南电等):  ██████████████████████████████████ 35%
日本(Ibiden等):      ██████████████████████████████ 30%
韩国(三星电机等):    ████████████████████████ 25%
中国大陆:            ██████ 5%
其他:                ████ 5%

⚠️ 中国在IC基板领域差距较大，是重点追赶方向！
```

### 5️⃣ ABF基板为什么难？

| 难点 | 说明 |
|------|------|
| 材料 | ABF膜主要来自日本味之素 |
| 设备 | 高精度设备依赖进口 |
| 工艺 | 线宽/线距要求极高 |
| 认证 | 客户验证周期长 |
| 资金 | 投资门槛高 |

```
ABF基板是高端芯片的必需品：

CPU → 需要ABF基板
GPU → 需要ABF基板
AI芯片 → 需要ABF基板

⚠️ ABF基板是AI芯片的瓶颈之一！
```

## 五、投资逻辑总结

### 核心逻辑

```
1. AI服务器拉动
   • AI服务器用更多高端PCB
   • AI芯片需要ABF基板
   • 利好：胜宏科技、深南电路

2. 5G/汽车电子
   • 5G基站建设持续
   • 汽车电子渗透率提升
   • 利好：沪电、景旺

3. IC基板国产替代
   • ABF基板严重依赖进口
   • 国产厂商加速布局
   • 利好：深南电路、兴森
```

| 风险 | 说明 |
