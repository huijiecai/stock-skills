# 09 PCB与基板

> 电子产品的"骨架"，连接芯片与系统的桥梁

---

## 一、什么是PCB？

### 通俗理解

> **PCB = 电子产品的"骨架"和"血管"**
>
> 承载芯片并连接各个元器件，所有电子产品都需要PCB。

### PCB的作用

```
┌─────────────────────────────────────────────────────────────┐
│                    PCB的作用                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                      │
│   │ CPU │  │内存 │  │电源 │  │接口 │                      │
│   └──┬──┘  └──┬──┘  └──┬──┘  └──┬──┘                      │
│      │        │        │        │                          │
│   ═══╧════════╧════════╧════════╧═══════════               │
│              PCB电路板（连接所有芯片）                        │
│   ════════════════════════════════════════                  │
│                                                             │
│   作用：                                                     │
│   1. 机械支撑 - 承载所有元器件                               │
│   2. 电气连接 - 连接各芯片，传输信号                         │
│   3. 电源分配 - 给各芯片供电                                 │
│   4. 散热 - 帮助散发热量                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 二、PCB分类

### 按层数分类

| 类型 | 层数 | 特点 | 应用 |
|------|------|------|------|
| 单面板 | 1层 | 最简单、最便宜 | 计算器、遥控器 |
| 双面板 | 2层 | 两面走线 | 简单电子产品 |
| 多层板 | 4-8层 | 主流 | 手机、电脑 |
| 高多层板 | 10-20层 | 复杂 | 服务器、通信设备 |
| **超高多层板** | 20层+ | 最复杂 | AI服务器、高端路由器 |

### 按类型分类

| 类型 | 特点 | 应用 | 代表厂商 |
|------|------|------|----------|
| **普通PCB** | 刚性板 | 各种电子产品 | 深南、沪电 |
| **HDI板** | 高密度互连 | 手机、可穿戴 | 深南、景旺 |
| **高频板** | 高频信号 | 5G基站、雷达 | 沪电、生益 |
| **FPC** | 柔性板 | 手机内部连接 | 鹏鼎、东山 |
| **IC基板** | 封装用 | 芯片封装 | 深南、兴森 |

## 三、IC基板（重点！）

### 什么是IC基板？

> **IC基板 = 芯片和PCB之间的"桥梁"**
>
> 先进封装中的关键材料，技术要求最高。

### IC基板的位置

```
┌─────────────────────────────────────────────────────────────┐
│                 IC基板在系统中的位置                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    ┌─────────┐                              │
│                    │  芯片   │                              │
│                    │  (Die)  │                              │
│                    └────┬────┘                              │
│                         │ 焊球                              │
│                    ┌────┴────┐                              │
│                    │ IC基板  │  ← 这里！                    │
│                    └────┬────┘                              │
│                         │ BGA焊球                           │
│                    ┌────┴────┐                              │
│                    │   PCB   │                              │
│                    └─────────┘                              │
│                                                             │
│   IC基板是芯片到PCB的"过渡层"                               │
│   线宽/线距要求比普通PCB高得多                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### IC基板分类

| 类型 | 全称 | 应用 | 难度 |
|------|------|------|------|
| **BT基板** | Bismaleimide Triazine | 普通封装 | 中等 |
| **ABF基板** | Ajinomoto Build-up Film | CPU、GPU、AI芯片 | **最高** |
| **FC-BGA** | Flip-Chip BGA | 高端封装 | 高 |

### ABF基板为什么重要？

```
ABF基板是高端芯片的必需品：

CPU → 需要ABF基板
GPU → 需要ABF基板
AI芯片 → 需要ABF基板

但ABF基板产能有限：
需求：████████████████████████████████████████ 100%
供给：██████████████████████████████ 70%

⚠️ ABF基板是AI芯片的瓶颈之一！
```

## 五、PCB技术趋势

### 高速化

```
5G/AI时代对PCB的要求：

信号速度：从Gbps → 几十Gbps → 上百Gbps
材料要求：低介电常数、低损耗
工艺要求：更细线宽、更高精度
```

### 高密度化

| 指标 | 传统PCB | 先进PCB |
|------|---------|---------|
| 线宽/线距 | 100/100μm | 30/30μm |
| 层数 | 4-8层 | 20层+ |
| 孔径 | >0.3mm | <0.1mm |

### 高频化

```
5G基站要求：

频率：从4G的<3GHz → 5G的3-6GHz → 毫米波24GHz+
材料：从普通FR-4 → 高频材料（PTFE、LCP等）
工艺：更精密的阻抗控制
```

### IC基板

| 类型 | 国产化率 | 挑战 |
|------|----------|------|
| BT基板 | ~30% | 逐步突破 |
| **ABF基板** | <10% | **差距最大** |
| FC-BGA | <20% | 追赶中 |

### ABF基板为什么难？

| 难点 | 说明 |
|------|------|
| 材料 | ABF膜主要来自日本味之素 |
| 设备 | 高精度设备依赖进口 |
| 工艺 | 线宽/线距要求极高 |
| 认证 | 客户验证周期长 |
| 资金 | 投资门槛高 |

## 八、投资逻辑

### 核心逻辑

```
1. AI服务器拉动
   • AI服务器用更多高端PCB
   • AI芯片需要ABF基板
   • 利好：深南电路、沪电

2. 5G/汽车电子
   • 5G基站建设持续
   • 汽车电子渗透率提升
   • 利好：沪电、景旺

3. IC基板国产替代
   • ABF基板严重依赖进口
   • 国产厂商加速布局
   • 利好：深南电路、兴森
```
