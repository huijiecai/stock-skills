## 一、CPO总览

### 1️⃣ 是什么？

> **CPO = 把光通信模块和芯片"打包"在一起**
>
> 让数据传输更快、功耗更低的新技术。

```
传统方案（可插拔光模块）：
┌─────────┐    电线    ┌─────────┐     光纤     ┌─────────┐
│  芯片   │ ─────────→ │ 光模块  │ ───────────→ │  网络   │
└─────────┘            └─────────┘              └─────────┘
    │                      │
    │                      └── 分离式，距离远
    └── 电信号损耗大

CPO方案（光电共封装）：
┌───────────────────────┐     光纤     ┌─────────┐
│  芯片 + 光引擎(一体)  │ ───────────→ │  网络   │
└───────────────────────┘              └─────────┘
    │
    └── 集成在一起，电信号距离极短，损耗小
```

### 2️⃣ 市场规模

```
CPO市场规模预测：

2025年: ████ ~10亿美元
2027年: ████████████████ ~50亿美元
2030年: ████████████████████████████████ ~150亿美元

年增速：约 50%+（最快增长的光通信领域）
```

### 3️⃣ 为什么需要CPO？

```
AI数据中心的带宽需求：

2020年: ████████ 400Gbps
2023年: ████████████████ 800Gbps
2025年: ████████████████████████████████ 1.6Tbps
2027年: ████████████████████████████████████████ 3.2Tbps

问题：传统可插拔光模块无法满足！
原因：
• 功耗太高（光模块距离芯片远，电信号损耗大）
• 密度受限（可插拔模块占空间）
• 成本上升（功耗和散热成本增加）
```

### 4️⃣ CPO的优势

| 指标 | 传统方案 | CPO | 提升 |
|------|----------|-----|------|
| **功耗** | 15pJ/bit | 5pJ/bit | **-70%** |
| **带宽密度** | 低 | 高 | **2-4倍** |
| **延迟** | 较高 | 低 | 显著降低 |
| **成本** | 高 | 低（长期）| 有优势 |

| 可维护性 | 可热插拔 | 需特殊设计 |

## 八、投资逻辑总结

### 核心逻辑

```
1. AI驱动带宽升级
   • AI集群需要超高带宽互连
   • 传统光模块功耗/密度受限
   • CPO是必然趋势

2. 产业链受益
   • 光模块厂转型光引擎
   • 光芯片需求提升
   • 无源器件用量增加

3. 国产替代
   • 光芯片国产化进程加速
   • 光模块国内厂商领先
```

### 投资节奏

| 阶段 | 时间 | 投资重点 |
|------|------|----------|
| **当前** | 2025-2026 | 800G/1.6T光模块放量 |
| **中期** | 2027-2028 | CPO光引擎开始放量 |
| **长期** | 2029+ | CPO大规模应用 |

### 风险提示

| 风险 | 说明 |
|------|------|
| ⚠️ 进度不及预期 | CPO商用化可能延迟 |
| ⚠️ 技术路线变化 | 可能出现替代方案 |
| ⚠️ 竞争加剧 | 国内厂商相互竞争 |
| ⚠️ 估值偏高 | CPO概念已有炒作 |

### A股标的汇总

| 环节 | 核心标的 |
|------|----------|
| 光模块 | **中际旭创**、**新易盛** |
| 光芯片 | **光迅科技**、**源杰科技** |
| 无源器件 | **天孚通信**、太辰光 |

## 九、与其他芯片的关系

```
需要CPO的场景：
✅ AI芯片（数据中心级）

不需要CPO的场景：
❌ 手机芯片
❌ 汽车芯片

❌ 普通存储芯片

```
