## 一、AI芯片总览

### 1️⃣ 是什么？

> **AI芯片 = 专门做AI计算的芯片**
>
> 传统CPU是"通才"，AI芯片是"专才"，专门干矩阵乘法这件事。

```
┌─────────────────────────────────────────────────────────────┐
│                计算架构对比                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  CPU（通用处理器）          GPU（图形处理器）               │
│  ┌─────────────────┐       ┌─────────────────────────────┐ │
│  │ ████  ████      │       │ ▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪ │ │
│  │ 控制  缓存      │       │ ▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪ │ │
│  │ ┌──┬──┬──┬──┐   │       │ ▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪ │ │
│  │ │核│核│核│核│   │       │ ▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪ │ │
│  │ └──┴──┴──┴──┘   │       └─────────────────────────────┘ │
│  └─────────────────┘                                        │
│                                                             │
│  • 4-64个大核                • 数千个小核                   │
│  • 擅长复杂逻辑              • 擅长并行计算                  │
│  • 串行能力强                • 矩阵运算能力强                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 六、液冷散热详解

### 1️⃣ 是什么？

> **液冷 = 用液体给服务器"降温"**
>
> 用水或特殊冷却液代替空气带走芯片产生的热量。

```
为什么需要液冷？

传统风冷散热能力：~300W/芯片
AI芯片功耗：
• H100：700W
• H200：700W
• B200：1000W+

单机柜功耗：
• 传统服务器：5-10kW
• AI服务器：20-100kW

结论：风冷根本吹不动！必须用液冷
```

### 3️⃣ 液冷技术分类

| 类型 | 原理 | 优势 | 劣势 | 应用场景 |
|------|------|------|------|----------|
| **冷板式液冷** | 冷却液通过冷板带走热量 | 改造成本低 | 散热效率中等 | **主流方案** |
| **浸没式液冷** | 服务器浸泡在冷却液中 | 效率最高 | 成本高、维护难 | 超高功耗场景 |
| **喷淋式液冷** | 冷却液直接喷在芯片上 | 效率高 | 复杂度高 | 特殊场景 |
