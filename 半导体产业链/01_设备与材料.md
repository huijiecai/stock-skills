# 01 设备与材料

> 半导体产业链的"地基"，没有设备和材料，芯片就是空中楼阁

---

## 一、半导体设备

### 通俗理解

> 就像做菜需要锅碗瓢盆，造芯片需要各种专业设备。**设备决定了你能造出多先进的芯片。**

### 核心设备分类

| 设备类型 | 作用 | 通俗比喻 |
|----------|------|----------|
| **光刻机** | 把电路图案"印"到硅片上 | 印刷机印报纸 |
| **刻蚀机** | 把不需要的材料"刻掉" | 雕刻刀刻章 |
| **薄膜设备** | 在硅片上镀各种薄膜 | 给蛋糕涂奶油 |
| **离子注入** | 掺入杂质改变性质 | 炒菜加调料 |
| **CMP设备** | 把表面磨平 | 打磨地板 |
| **检测设备** | 检查有没有缺陷 | 质检员 |
| **清洗设备** | 清洗硅片 | 洗碗机 |

## 二、光刻机详解（最核心设备）

### 为什么光刻机最重要？

```
芯片制造的核心步骤：
1. 在硅片上涂光刻胶
2. 用光刻机把电路图案"曝光"到光刻胶上  ← 这一步决定精度！
3. 显影、刻蚀、沉积...
4. 重复上百次
```

### 光刻机代际

| 类型 | 波长 | 能做到的制程 | 价格 | 供应商 |
|------|------|--------------|------|--------|
| i-line | 365nm | >180nm | 便宜 | 多家 |
| KrF | 248nm | 180-90nm | 中等 | Nikon/Canon |
| ArF干式 | 193nm | 90-45nm | 较贵 | Nikon/Canon/ASML |
| **ArF浸没式** | 193nm | 45-7nm | ~1亿美元 | ASML为主 |
| **EUV** | 13.5nm | 7nm以下 | ~1.5亿美元 | **仅ASML** |
| **High-NA EUV** | 13.5nm | 2nm以下 | ~3.5亿美元 | **仅ASML** |

### ASML的垄断地位

```
EUV光刻机市场份额：

ASML: ████████████████████████████████████ 100%
其他:  （无）

⚠️ 这是全球唯一的EUV供应商！
```

### EUV光刻机为什么难造？

| 难点 | 说明 |
|------|------|
| 光源 | 用激光打锡滴产生EUV光，效率极低 |
| 反射镜 | 需要纳米级精度的多层反射镜 |
| 真空环境 | EUV会被空气吸收，必须在真空中工作 |
| 供应链 | 需要全球顶尖零部件商配合 |
| 专利壁垒 | 几十年技术积累，专利墙极高 |

### 通俗理解

> 就像做菜需要食材，造芯片需要各种高纯度材料。**纯度差一点点，芯片可能就废了。**

| **CMP材料** | 抛光用的浆料和垫片 | 高 |
| **靶材** | 薄膜溅射用 | 高 |
| **湿电子化学品** | 清洗和刻蚀用 | 极高 |

## 四、洁净室（无尘室）

### 通俗理解

> **洁净室 = 芯片制造的"手术室"**
>
> 一粒灰尘就能毁掉一颗芯片，所以必须在极度洁净的环境中生产。

### 为什么需要洁净室？

```
芯片线宽对比：

头发丝直径：      ~80,000nm
灰尘颗粒：        ~1,000-10,000nm
7nm芯片线宽：     7nm
3nm芯片线宽：     3nm

结论：一粒灰尘 = 几千倍的芯片线宽 → 必须无尘环境
```

### 洁净室等级

| 等级 | 含义（每立方米≥0.5μm颗粒数）| 应用场景 |
|------|------------------------------|----------|
| Class 1 | ≤1个 | 最先进制程（EUV光刻区）|
| Class 10 | ≤10个 | 先进制程区 |
| Class 100 | ≤100个 | 普通制程区 |
| Class 1000 | ≤1000个 | 封装测试 |

> 💡 对比：普通室内空气每立方米有**几十万到上百万**个颗粒

### 洁净室产业链

```
晶圆厂建设流程：

土建施工 → 【洁净室建设】→ 设备搬入 → 产线调试 → 量产
              ↑
        亚翔集成等承接
        
洁净室建设包括：
• 净化空调系统（HVAC）
• 地板/墙板/天花板
• 气体/化学品管道
• 废气处理系统
```

## 七、投资逻辑

### 核心逻辑：国产替代

```
美国制裁 → 国内晶圆厂被迫采购国产设备/材料 → 国产厂商订单暴增
```

### 关注要点

| 要点 | 说明 |
|------|------|
| ✅ 订单增速 | 国产设备厂的订单是否持续增长 |
| ✅ 验证进度 | 是否进入主流晶圆厂供应链 |
| ✅ 技术突破 | 高端产品是否有进展 |
| ⚠️ 估值 | 国产替代预期是否已充分反映 |
| ⚠️ 盈利能力 | 研发投入大，何时能盈利 |
