# 05 封装测试

> 芯片的"装修"环节，让芯片能够正常使用

---

## 一、什么是封装？

### 通俗理解

> **封装 = 给芯片穿"衣服"**
>
> 芯片从晶圆厂出来是"裸片"，非常脆弱，需要封装保护并提供电气连接。

### 封装的作用

| 作用 | 说明 |
|------|------|
| **保护芯片** | 防止物理损伤、湿气、腐蚀 |
| **电气连接** | 把芯片引脚引出来，方便焊接 |
| **散热** | 帮助芯片散发热量 |
| **机械支撑** | 提供稳定的物理结构 |

### 封装前后对比

```
封装前（裸片Die）                  封装后
┌─────────────────┐              ┌─────────────────────┐
│                 │              │   ┌───────────┐     │
│   非常小        │              │   │   芯片    │     │
│   非常脆弱      │   封装后 →   │   └─────┬─────┘     │
│   无法直接使用   │              │         │ 键合丝    │
│                 │              │   ══════╧══════    │
└─────────────────┘              │      引线框架      │
                                 │   ═══════════════  │
                                 │       引脚        │
                                 └─────────────────────┘
```

## 二、封装类型演进

### 传统封装

| 类型 | 全称 | 特点 | 应用 |
|------|------|------|------|
| **DIP** | Dual In-line Package | 双列直插 | 老式芯片 |
| **QFP** | Quad Flat Package | 四边扁平 | 消费电子 |
| **QFN** | Quad Flat No-lead | 无引脚扁平 | 手机、IoT |
| **BGA** | Ball Grid Array | 球栅阵列 | 高端芯片 |

### 封装演进图

```
封装技术演进：

DIP(1970s) → QFP(1980s) → BGA(1990s) → Flip-Chip(2000s) → 先进封装(2010s+)
   ↑            ↑            ↑             ↑               ↑
 最简单      引脚增多      更高密度      倒装焊接        多芯片集成
 成本最低    成本低        成本中等      成本较高        成本最高
```

### 主流封装对比

| 封装类型 | 引脚数 | 尺寸 | 成本 | 应用场景 |
|----------|--------|------|------|----------|
| QFN | 8-100 | 小 | 低 | 电源、MCU |
| BGA | 100-2000+ | 中 | 中 | CPU、GPU、手机SoC |
| Flip-Chip | 1000+ | 中 | 较高 | 高性能芯片 |
| 2.5D/3D | 极多 | 大 | 高 | AI芯片、HBM |

## 三、封装流程

### 标准封装流程

```
┌─────────────────────────────────────────────────────────────┐
│                    封装主要流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 晶圆减薄  →  把晶圆磨薄，方便切割                        │
│       ↓                                                     │
│  2. 晶圆切割  →  把晶圆切成一颗颗芯片（Die）                 │
│       ↓                                                     │
│  3. 芯片贴装  →  把芯片贴到基板或引线框架上                  │
│       ↓                                                     │
│  4. 引线键合  →  用金线/铜线连接芯片和基板                   │
│       ↓                                                     │
│  5. 塑封      →  用环氧树脂包裹保护                          │
│       ↓                                                     │
│  6. 切割分离  →  切割成单个封装                              │
│       ↓                                                     │
│  7. 测试打标  →  测试性能，打印标识                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 四、什么是测试？

### 通俗理解

> **测试 = 质检员检验产品**
>
> 确保每颗芯片都能正常工作，不良品不能流入市场。

### 测试类型

| 测试阶段 | 时机 | 内容 |
|----------|------|------|
| **晶圆测试(CP)** | 封装前 | 在晶圆上测试每颗Die |
| **成品测试(FT)** | 封装后 | 测试封装好的芯片 |
| **可靠性测试** | 抽样 | 老化、温度、湿度测试 |

### 测试设备

| 设备 | 作用 | 主要厂商 |
|------|------|----------|
| 探针台 | 晶圆测试时接触芯片 | 东京精密、长川科技 |
| 测试机 | 生成测试信号、分析结果 | 泰瑞达、爱德万 |
| 分选机 | 根据测试结果分类 | 爱德万、长川科技 |

## 八、投资逻辑

### 核心逻辑

```
1. 先进封装趋势
   • 先进封装价值量更高
   • 长电、通富布局先进封装

2. 国产替代
   • 国内设计公司优先选择国内封测
   • 封装材料国产替代

3. 产能扩张
   • 配合晶圆厂扩产
   • 新能源汽车带来增量
```

### 风险提示

| 风险 | 说明 |
|------|------|
| ⚠️ 毛利率低 | 封测环节毛利率普遍较低 |
| ⚠️ 竞争激烈 | 国内封测厂相互竞争 |
| ⚠️ 客户集中 | 大客户依赖度高 |
