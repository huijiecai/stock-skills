## 一、芯片设计是什么？

### 通俗理解

> **芯片设计 = 画芯片的图纸**
>
> 就像建筑师设计房屋图纸，芯片设计师设计芯片的电路结构。

### 设计流程

```
┌─────────────────────────────────────────────────────────────┐
│                     芯片设计流程                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   需求定义 → 架构设计 → RTL编写 → 功能验证                   │
│                                        ↓                    │
│   流片生产 ← 物理验证 ← 布局布线 ← 逻辑综合                  │
│       ↓                                                     │
│   封装测试 → 量产                                           │
│                                                             │
│   整个过程通常需要 1-3 年                                    │
└─────────────────────────────────────────────────────────────┘
```

## 二、芯片公司的商业模式

### 三种模式对比

| 模式 | 说明 | 代表公司 |
|------|------|----------|
| **Fabless** | 只设计，不生产 | 英伟达、高通、AMD、华为海思 |
| **IDM** | 设计+生产一体 | 英特尔、三星、美光、TI |
| **Foundry** | 只生产，不设计 | 台积电、中芯国际 |

### Fabless模式详解

```
Fabless公司的运作方式：

设计公司（如高通）                 代工厂（如台积电）
┌─────────────────┐              ┌─────────────────┐
│  1. 设计芯片     │  →  发送设计  │  3. 生产芯片    │
│  2. 设计验证     │  →  文件      │  4. 交付晶圆    │
│                 │              │                 │
│  5. 封装测试     │  ←  晶圆回来  │                 │
│  6. 销售芯片     │              │                 │
└─────────────────┘              └─────────────────┘

优势：轻资产，专注设计
劣势：受制于代工厂产能
```

## 五、CPU架构之争

### 主流CPU架构

| 架构 | 授权方 | 应用领域 | 代表芯片 |
|------|--------|----------|----------|
| **x86** | Intel/AMD | PC、服务器 | Intel Core、AMD Ryzen |
| **ARM** | ARM | 手机、Mac、服务器 | 苹果M系列、高通骁龙 |
| **RISC-V** | 开源 | IoT、嵌入式 | 平头哥、芯来 |
| **MIPS** | Wave Computing | 网络设备 | 龙芯（早期） |
| **LoongArch** | 龙芯 | 国产自主 | 龙芯3A6000 |

### 中国的选择

```
中国CPU路线：

1. x86授权路线
   海光 → AMD授权 → 服务器CPU
   兆芯 → VIA授权 → 桌面CPU
   优势：生态兼容
   风险：授权可能被断

2. ARM授权路线
   华为 → 鲲鹏920
   飞腾 → 腾云/飞腾
   优势：移动生态好
   风险：ARM可能被限制

3. 自主架构路线
   龙芯 → LoongArch
   申威 → SW64
   优势：完全自主
   劣势：生态需要重建

4. RISC-V开源路线
   芯来、平头哥等
   优势：无授权风险
   劣势：生态不成熟
```

## 七、投资逻辑

### 核心逻辑

```
1. 国产替代
   • 手机/PC/服务器芯片国产替代空间大
   • 信创（信息技术应用创新）政策推动

2. 新兴需求
   • AI芯片需求爆发
   • 汽车电子渗透率提升
   • IoT设备数量增长

3. 周期复苏
   • 半导体周期见底回升
   • 库存去化完成后的补库需求
```

### 风险提示

| 风险 | 说明 |
|------|------|
| ⚠️ 估值偏高 | 国产替代预期已部分反映 |
| ⚠️ 技术差距 | 高端领域差距仍大 |
| ⚠️ 周期波动 | 半导体行业周期性强 |
| ⚠️ 竞争加剧 | 国内公司相互竞争 |
