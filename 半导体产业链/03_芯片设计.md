## 一、芯片设计总览

### 1️⃣ 是什么？

> **芯片设计 = 画芯片的图纸**
>
> 就像建筑师设计房屋图纸，芯片设计师设计芯片的电路结构。

```
┌─────────────────────────────────────────────────────────────┐
│                     芯片设计流程                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   需求定义 → 架构设计 → RTL编写 → 功能验证                   │
│                                        ↓                    │
│   流片生产 ← 物理验证 ← 布局布线 ← 逻辑综合                  │
│       ↓                                                     │

│                                                             │
```

## 三、各领域龙头分析

### GPU/AI芯片（重点！）

#### 1️⃣ 是什么？

> **AI芯片 = 专门做AI计算的芯片**
>
> 传统CPU是"通才"，AI芯片是"专才"，专门干矩阵乘法这件事。

```
┌─────────────────────────────────────────────────────────────┐
│                计算架构对比                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  CPU（通用处理器）          GPU（图形处理器）               │
│  ┌─────────────────┐       ┌─────────────────────────────┐ │
│  │ ████  ████      │       │ ▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪ │ │
│  │ 控制  缓存      │       │ ▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪ │ │
│  │ ┌──┬──┬──┬──┐   │       │ ▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪ │ │
│  │ │核│核│核│核│   │       │ ▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪▪ │ │
│  │ └──┴──┴──┴──┘   │       └─────────────────────────────┘ │
│  └─────────────────┘                                        │
│                                                             │
│  • 4-64个大核                • 数千个小核                   │
│  • 擅长复杂逻辑              • 擅长并行计算                  │
│  • 串行能力强                • 矩阵运算能力强                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### CPU（处理器）

#### 1️⃣ 是什么？

> **CPU = 计算机的"大脑"**
>
> 负责执行计算机程序的核心芯片。

#### 2️⃣ 市场规模

```
CPU市场规模（2025年）：

PC/服务器CPU：约 800亿美元
移动处理器：约 400亿美元
```

#### 3️⃣ 竞争格局

| 领域 | 龙头 | 市占率 |
|------|------|--------|
| PC CPU | Intel、AMD | 99%（双寡头）|
| 服务器CPU | Intel、AMD | 95%+ |
| 手机SoC | 苹果、高通、联发科 | 90%+ |

#### 4️⃣ 国产替代

**中国CPU路线**

```
1. x86授权路线
   海光 → AMD授权 → 服务器CPU
   兆芯 → VIA授权 → 桌面CPU
   优势：生态兼容 | 风险：授权可能被断

2. ARM授权路线
   华为 → 鲲鹏920
   飞腾 → 腾云/飞腾
   优势：移动生态好 | 风险：ARM可能被限制

3. 自主架构路线
   龙芯 → LoongArch
   申威 → SW64
   优势：完全自主 | 劣势：生态需要重建

4. RISC-V开源路线
   芯来、平头哥等
   优势：无授权风险 | 劣势：生态不成熟
```

#### 5️⃣ A股标的

| 公司 | 代码 | 主营 | 架构 | 核心看点 |
|------|------|------|------|----------|
| **海光信息** | 688041 | CPU/DCU | x86 | 信创服务器 |
| **龙芯中科** | 688047 | CPU | LoongArch | 完全自主 |
| **景嘉微** | 300474 | GPU | 自研 | 军用GPU |
