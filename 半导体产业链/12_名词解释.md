# 12 名词解释

> 半导体产业链常用术语速查表

---

## 一、公司/商业模式类

| 术语 | 英文 | 解释 |
|------|------|------|
| **Fabless** | 无晶圆厂 | 只做芯片设计，不做生产的公司。如：英伟达、高通 |
| **IDM** | Integrated Device Manufacturer | 设计+生产一体的公司。如：英特尔、三星 |
| **Foundry** | 代工厂 | 只做芯片生产，不做设计的公司。如：台积电、中芯国际 |
| **OSAT** | 封装测试外包 | 专业做封装测试的公司。如：日月光、长电科技 |

---

## 二、芯片设计类

| 术语 | 英文 | 解释 |
|------|------|------|
| **EDA** | Electronic Design Automation | 芯片设计自动化软件，没有它就无法设计芯片 |
| **IP核** | Intellectual Property | 可复用的设计模块，如CPU核心、接口模块等 |
| **RTL** | Register Transfer Level | 寄存器传输级，芯片逻辑设计的描述方式 |
| **SoC** | System on Chip | 片上系统，把CPU/GPU/内存等集成在一颗芯片上 |
| **ASIC** | Application-Specific IC | 专用集成电路，为特定用途定制的芯片 |
| **FPGA** | Field-Programmable Gate Array | 现场可编程门阵列，可重新配置的芯片 |
| **MCU** | Microcontroller Unit | 微控制器，低功耗控制芯片 |
| **DSP** | Digital Signal Processor | 数字信号处理器 |
| **NPU** | Neural Processing Unit | 神经网络处理器，AI专用单元 |

---

## 三、晶圆制造类

| 术语 | 英文 | 解释 |
|------|------|------|
| **晶圆/Wafer** | - | 圆形硅片，芯片的载体，直径通常是200mm或300mm |
| **制程节点** | Process Node | 芯片制造工艺的尺寸，如7nm、5nm、3nm |
| **光刻** | Lithography | 用光把电路图案"印"到硅片上的工艺 |
| **EUV** | Extreme Ultraviolet | 极紫外光刻，用于7nm及以下先进制程 |
| **DUV** | Deep Ultraviolet | 深紫外光刻，用于成熟制程 |
| **刻蚀** | Etch | 去除不需要的材料的工艺 |
| **薄膜沉积** | Deposition | 在硅片上镀各种材料薄膜的工艺 |
| **CVD** | Chemical Vapor Deposition | 化学气相沉积 |
| **ALD** | Atomic Layer Deposition | 原子层沉积 |
| **PVD** | Physical Vapor Deposition | 物理气相沉积 |
| **CMP** | Chemical Mechanical Polish | 化学机械抛光，把表面磨平 |
| **良率** | Yield | 合格芯片的比例 |

---

## 四、封装类

| 术语 | 英文 | 解释 |
|------|------|------|
| **Die** | 裸片 | 从晶圆上切下来的单颗芯片 |
| **封装** | Packaging | 给裸片穿"衣服"，提供保护和连接 |
| **引线框架** | Lead Frame | 传统封装的载体和引脚 |
| **键合丝** | Bonding Wire | 连接芯片和封装的金属细丝 |
| **BGA** | Ball Grid Array | 球栅阵列封装，用焊球连接 |
| **QFN** | Quad Flat No-lead | 无引脚扁平封装 |
| **Flip-Chip** | 倒装焊 | 芯片倒置焊接的封装方式 |
| **TSV** | Through-Silicon Via | 硅通孔，3D封装的垂直连接技术 |
| **Interposer** | 硅中介层 | 2.5D封装中连接多个芯片的硅层 |

---

## 五、先进封装类

| 术语 | 英文 | 解释 |
|------|------|------|
| **2.5D封装** | - | 多芯片并排放在硅中介层上 |
| **3D封装** | - | 芯片垂直堆叠 |
| **Chiplet** | 芯粒 | 把大芯片拆成多个小芯片组合 |
| **CoWoS** | Chip on Wafer on Substrate | 台积电的2.5D封装技术 |
| **InFO** | Integrated Fan-Out | 台积电的扇出型封装技术 |
| **SoIC** | System on Integrated Chips | 台积电的3D封装技术 |
| **Fan-Out** | 扇出型封装 | 引脚从芯片四周扇出的封装 |
| **SiP** | System in Package | 系统级封装，多芯片集成 |
| **混合键合** | Hybrid Bonding | 超高密度的芯片连接技术 |

---

## 六、存储类

| 术语 | 英文 | 解释 |
|------|------|------|
| **DRAM** | Dynamic RAM | 动态随机存储器，断电丢失，用于内存 |
| **SRAM** | Static RAM | 静态随机存储器，最快，用于CPU缓存 |
| **NAND** | - | 闪存，断电保存，用于SSD |
| **NOR** | - | 闪存，可直接执行代码，用于启动程序 |
| **HBM** | High Bandwidth Memory | 高带宽存储器，AI芯片专用 |
| **DDR** | Double Data Rate | 双倍数据速率，内存类型 |
| **GDDR** | Graphics DDR | 显卡专用内存 |
| **LPDDR** | Low Power DDR | 低功耗内存，用于手机 |
| **3D NAND** | - | 垂直堆叠的NAND闪存 |

---

## 七、AI/算力类

| 术语 | 英文 | 解释 |
|------|------|------|
| **GPU** | Graphics Processing Unit | 图形处理器，也用于AI计算 |
| **TPU** | Tensor Processing Unit | 谷歌的AI专用芯片 |
| **CUDA** | - | 英伟达的并行计算平台 |
| **TFLOPS** | Tera FLOPS | 万亿次浮点运算/秒，衡量算力 |
| **训练** | Training | 用数据训练AI模型 |
| **推理** | Inference | 用训练好的模型做预测 |
| **NVLink** | - | 英伟达的高速芯片互连技术 |

---

## 八、PCB/光通信类

| 术语 | 英文 | 解释 |
|------|------|------|
| **PCB** | Printed Circuit Board | 印制电路板，电子产品的"骨架" |
| **HDI** | High Density Interconnect | 高密度互连PCB |
| **IC基板** | IC Substrate | 芯片封装用的高端基板 |
| **ABF基板** | Ajinomoto Build-up Film | 高端封装基板，用于CPU/GPU |
| **CCL** | Copper Clad Laminate | 覆铜板，PCB的原材料 |
| **光模块** | Optical Module | 光电转换模块 |
| **CPO** | Co-Packaged Optics | 光电共封装 |
| **光引擎** | Optical Engine | CPO中的光收发组件 |

## 九、其他常用术语

| 术语 | 英文 | 解释 |
|------|------|------|
| **国产替代** | - | 用国产产品替代进口 |
| **卡脖子** | - | 被国外技术/产品限制的关键环节 |
| **大基金** | - | 国家集成电路产业投资基金 |
| **信创** | - | 信息技术应用创新，国产化替代 |
| **制裁** | Sanctions | 美国对中国半导体的出口限制 |
| **周期** | Cycle | 半导体行业的景气度波动 |
| **去库存** | Destocking | 消化过剩库存 |
| **下游** | Downstream | 产业链的后端，接近终端应用 |
| **上游** | Upstream | 产业链的前端，原材料和设备 |
