# 金沃股份（002597）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：精密传动部件核心标的

---

## 一、公司简介

浙江金沃精工股份有限公司成立于2008年，是国内领先的精密传动部件供应商。公司主营精密传动部件、轴承套圈等产品，近年来布局机器人精密传动业务。

**核心业务**：
- 精密传动部件（主业）
- 轴承套圈
- 汽车零部件
- 机器人传动部件（新兴业务）

---

### 2.1 业务结构

**传动部件**：

- 轴承套圈

**技术优势**：
