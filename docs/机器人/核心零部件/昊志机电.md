# 昊志机电（300503）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：无框力矩电机核心标的

---

## 一、公司简介

广州市昊志机电股份有限公司成立于2001年，是国内领先的主轴供应商。公司主营电主轴、运动控制系统等产品，近年来战略布局人形机器人无框力矩电机业务。

**核心业务**：
- 电主轴（主业）
- 运动控制系统
- 无框力矩电机（新兴业务）
- 转台

---
