# 汉宇集团（300407）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：谐波减速器新兴标的

---

## 一、公司简介

广东汉宇集团股份有限公司成立于2002年，是国内领先的家电排水泵供应商。公司主营家电排水泵、谐波减速器等产品，近年来战略布局机器人谐波减速器业务。

**核心业务**：
- 家电排水泵（主业）
- 谐波减速器（新兴业务）
- 汽车配件
- 工业机器人

---
