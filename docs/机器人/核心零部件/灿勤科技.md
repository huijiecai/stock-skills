# 灿勤科技（688691）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：谐波减速器新兴标的

---

## 一、公司简介

江苏灿勤科技股份有限公司成立于2004年，是国内领先的微波介质陶瓷元器件供应商。公司主营滤波器、介质谐振器等产品，近年来战略布局谐波减速器业务。

**核心业务**：
- 微波介质陶瓷元器件（主业）
- 滤波器
- 谐波减速器（新兴业务）
- 电子元器件
