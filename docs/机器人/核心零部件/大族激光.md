# 大族激光（002008）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：伺服电机+激光装备双主业标的

---

## 一、公司简介

大族激光科技产业集团股份有限公司成立于1996年，是全球领先的激光装备供应商。公司主营激光加工设备、伺服电机等产品，近年来布局机器人伺服电机业务。

**核心业务**：
- 激光加工设备（主业）
- 伺服电机
- 激光器
- 自动化装备
