# 力星股份（300421）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：精密轴承钢球核心标的

---

## 一、公司简介

江苏力星通用钢球股份有限公司成立于1993年，是国内领先的精密钢球供应商。公司主营精密钢球、轴承等产品，近年来战略布局机器人轴承业务。

**核心业务**：
- 精密钢球（主业）
- 轴承
- 机器人轴承（新兴业务）
- 精密零部件

---

## 二、主营业务

### 2.1 业务结构

| 业务板块 | 营收占比 | 行业地位 |
|----------|---------|---------|
| 精密钢球 | 55% | 国内领先 |
| 轴承 | 25% | 国内前列 |
| 机器人轴承 | 12% | 核心增长点 |
| 其他 | 8% | - |

### 2.2 机器人轴承业务布局

**产品矩阵**：
- 机器人关节轴承
- 高精度轴承钢球
- 交叉滚子轴承
- 谐波减速器轴承

**技术优势**：
- 精密钢球加工技术领先
- 高精度、高一致性
- 适配机器人需求
