# 机器人行业分析报告

> **更新时间**：2026-02-28
> 
> **行业定位**：机器人是智能制造的核心载体，涵盖工业机器人、服务机器人、人形机器人三大方向

---

## 一、行业定义与范围

### 1.1 行业定义

机器人是一种能够半自主或全自主工作的智能机器，通过编程或人工智能技术实现各种任务。按照应用场景分为：

- **工业机器人**：用于制造业自动化生产（焊接、装配、搬运、喷涂等）
- **服务机器人**：用于家庭服务、医疗、物流、公共服务等
- **人形机器人**：具备人类形态的智能机器人（新兴赛道）

### 1.2 产业链结构

```
上游：核心零部件（成本占比60-70%）
├── 减速器（35-50%）→ 谐波减速器、RV减速器
├── 伺服系统（20-25%）→ 伺服电机、伺服驱动器
├── 控制器（10-15%）→ 运动控制器、数控系统
├── 传感器（5-10%）→ 力传感器、视觉传感器、编码器
└── 传动部件 → 丝杠、轴承、关节模组

中游：整机制造
├── 工业机器人本体
├── 服务机器人本体
└── 人形机器人本体

下游：系统集成与应用
├── 工业自动化系统集成
├── 智能工厂解决方案
└── 行业应用解决方案
```

## 五、投资逻辑与风险

### 5.1 核心投资逻辑

**1. 工业机器人国产替代**
- 四大家族市场份额从70%降至50%
- 国产减速器、伺服系统技术突破
- 汇川技术、绿的谐波等核心标的受益

**2. 人形机器人从0到1**
- 2025年定义为量产元年
- 特斯拉Optimus引领产业链
- 减速器、伺服电机、执行器价值量最高

**3. 服务机器人消费升级**
- 扫地机器人渗透率提升
- 医疗机器人蓝海市场
- AI赋能服务机器人智能化

### 5.2 关键风险因素

| 风险类型 | 具体风险 |
|----------|---------|
| 技术风险 | 核心零部件技术差距，高端市场仍被外资垄断 |
| 市场风险 | 人形机器人量产不及预期，商业化周期长 |
| 竞争风险 | 行业进入壁垒降低，价格战加剧 |
| 客户风险 | 部分企业客户集中度高，业绩波动大 |

### 5.3 重点关注指标

- **减速器**：国产化率、市占率、产能利用率
- **伺服系统**：技术水平、客户结构、毛利率
- **人形机器人**：量产进度、订单金额、成本下降
