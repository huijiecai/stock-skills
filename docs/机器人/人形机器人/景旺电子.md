# 景旺电子（603228）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：人形机器人PCB核心标的

---

## 一、公司简介

深圳市景旺电子股份有限公司成立于1993年，是国内领先的PCB制造商。公司主营双面/多层PCB、HDI、柔性电路板等产品，广泛应用于消费电子、汽车电子等领域，近年来布局人形机器人PCB业务。

**核心业务**：
- 双面/多层PCB（主业）
- HDI高密度互联板
- 柔性电路板（FPC）
- 人形机器人PCB（新兴业务）
