# 均普智能（688306）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：人形机器人整机制造标的

---

## 一、公司简介

宁波均普智能制造股份有限公司成立于2019年，是全球领先的智能制造装备供应商。公司主营工业自动化装备、智能生产线等，近年来战略布局人形机器人业务，自主研发"贾维斯"系列人形机器人。

**核心业务**：
- 工业自动化装备（主业）
- 智能生产线
- 人形机器人整机（新兴业务）
- 数字化工厂解决方案

## 六、风险提示

1. **研发风险**：人形机器人技术迭代快
2. **量产风险**：量产进度存在不确定性
3. **市场竞争**：特斯拉、优必选等竞争对手强劲
4. **盈利风险**：研发投入大，盈利承压
