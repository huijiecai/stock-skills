## 一、公司简介

江苏天奇自动化工程股份有限公司成立于1997年，是国内领先的智能装备和循环产业服务商。公司主营智能输送装备、循环产业装备等，近年来战略转型人形机器人领域，与银河通用机器人深度合作，成为A股人形机器人整机制造核心标的。

**核心业务**：
- 智能输送装备（主业）
- 循环产业装备（主业）
- 人形机器人整机（新兴业务）
- 汽车智能工厂解决方案
