# 兆威机电（003021）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：人形机器人关节驱动模组核心标的

---

## 一、公司简介

深圳市兆威机电股份有限公司成立于2001年，是国内领先的微型传动系统供应商。公司主营微型驱动系统、精密齿轮及齿轮箱等产品，广泛应用于汽车电子、智能家居、通信设备等领域，近年来重点布局人形机器人关节驱动模组业务。

**核心业务**：
- 微型驱动系统（主业）
- 精密齿轮及齿轮箱
- 汽车电子驱动系统
- 人形机器人关节驱动模组（新兴业务）

### 2.2 人形机器人业务布局

**关节驱动模组**：
- 人形机器人关节核心驱动部件
- 仿生灵巧手驱动模组适配特斯拉Optimus

- 产能快速建设中

**微型传动解决方案**：
- 微型齿轮箱
- 行星减速器
- 空心杯电机驱动模组

## 五、投资亮点

### 5.1 人形机器人机遇

- 关节驱动模组是人形机器人核心部件
- 已向特斯拉Optimus送样测试
- 单台人形机器人需要20+个关节驱动模组

### 5.2 微型传动龙头

- 微型传动市场份额国内领先
- 受益于汽车电子和智能家居升级
- 人形机器人打开第三增长曲线

### 5.3 特斯拉生态

- 已进入特斯拉汽车供应链
- 关节驱动模组适配Optimus
- 特斯拉生态优势明显

## 六、风险提示

1. **客户依赖风险**：特斯拉等大客户订单波动影响业绩
2. **技术迭代风险**：驱动技术路线变化快
3. **产能风险**：扩产需匹配市场需求
4. **竞争加剧风险**：鸣志电器等竞争对手激烈
