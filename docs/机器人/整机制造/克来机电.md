# 克来机电（603960）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：汽车电子机器人系统集成核心标的

---

## 一、公司简介

上海克来机电自动化工程股份有限公司成立于2002年，是国内领先的汽车电子领域机器人系统集成商。公司主营汽车电子机器人系统集成、自动化装备等产品。

**核心业务**：
- 汽车电子机器人系统（主业）
- 自动化装备
- 柔性生产线
- 智能制造解决方案
