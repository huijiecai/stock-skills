# 爱仕达（002403）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：炊具+工业机器人双主业标的

---

## 一、公司简介

爱仕达股份有限公司成立于1993年，是国内领先的炊具企业。公司主营炊具、工业机器人等产品，炊具+工业机器人双主业布局，钱江机器人是国内工业机器人知名品牌。

**核心业务**：
- 炊具（主业）
- 工业机器人
- 钱江机器人
- 智能制造解决方案
