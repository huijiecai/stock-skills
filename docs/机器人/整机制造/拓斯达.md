# 拓斯达（300607）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：工业机器人+系统集成核心标的

---

## 一、公司简介

广东拓斯达科技股份有限公司成立于2007年，是国内领先的工业机器人和智能制造解决方案供应商。公司主营工业机器人、自动化系统集成等产品，2023年多关节机器人出货量增长48.55%。

**核心业务**：
- 工业机器人（主业）
- 自动化系统集成
- 注塑机配套
- 智能制造解决方案
