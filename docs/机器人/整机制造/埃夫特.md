# 埃夫特（688165）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：工业机器人整机核心标的

---

## 一、公司简介

埃夫特智能装备股份有限公司成立于2007年，是国内领先的工业机器人企业。公司主营工业机器人、智能制造解决方案等产品，在汽车领域具有优势，计划建设机器人超级工厂。

**核心业务**：
- 工业机器人（主业）
- 智能制造解决方案
- 核心零部件
- 机器人系统集成
