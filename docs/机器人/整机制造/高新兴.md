# 高新兴（300098）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：机器人+安防双主业标的

---

## 一、公司简介

高新兴科技集团股份有限公司成立于1997年，是国内领先的安防和机器人企业。公司主营安防产品、巡逻机器人等产品，机器人+安防双主业布局。

**核心业务**：
- 安防产品（主业）
- 巡逻机器人
- 智慧交通
- 物联网解决方案

---
