# 汉威科技（300007）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：机器人触觉传感器核心标的

---

## 一、公司简介

汉威科技集团股份有限公司成立于1998年，是国内领先的传感器企业。公司主营气体传感器、触觉传感器等产品，触觉传感器应用于人形机器人灵巧手，全球前三。

**核心业务**：
- 气体传感器（主业）
- 触觉传感器
- 仪器仪表
- 物联网解决方案
