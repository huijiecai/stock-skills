# 恒立液压（601100）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：人形机器人丝杠/线性执行器核心标的

---

## 一、公司简介

江苏恒立液压股份有限公司成立于1990年，是国内液压元件龙头。公司主营液压油缸、液压泵阀等产品，广泛应用于工程机械、船舶等领域，近年来战略布局人形机器人丝杠和线性执行器业务。

**核心业务**：
- 液压油缸（主业）
- 液压泵阀
- 液压系统
- 人形机器人丝杠/线性执行器（新兴业务）
