# 斯菱股份（301556）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：人形机器人轴承核心标的

---

## 一、公司简介

浙江斯菱汽车轴承股份有限公司成立于2004年，是国内领先的汽车轴承供应商。公司主营汽车轴承、轮毂轴承单元等产品，近年来战略布局人形机器人轴承业务。

**核心业务**：
- 汽车轴承（主业）
- 轮毂轴承单元
- 机器人轴承（新兴业务）
- 精密轴承

---

## 二、主营业务

### 2.1 业务结构

| 业务板块 | 营收占比 | 行业地位 |
|----------|---------|---------|
| 汽车轴承 | 60% | 国内领先 |
| 轮毂轴承单元 | 30% | 国内前列 |
| 机器人轴承 | 6% | 核心增长点 |
| 其他 | 4% | - |

### 2.2 机器人轴承业务布局

**产品矩阵**：
- 人形机器人关节轴承
- 高精度薄壁轴承
- 交叉滚子轴承
- 谐波减速器轴承

**技术优势**：
- 精密轴承制造技术领先
- 高精度、长寿命
- 适配人形机器人需求
