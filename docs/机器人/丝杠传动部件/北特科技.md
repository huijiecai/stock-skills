# 北特科技（603239）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：人形机器人微型丝杠核心标的

---

## 一、公司简介

上海北特科技股份有限公司成立于2002年，是国内领先的汽车转向系统零部件供应商。公司主营转向器齿条、减震器活塞杆等产品，近年来战略布局人形机器人微型丝杠业务。

**核心业务**：
- 转向器齿条（主业）
- 减震器活塞杆
- 汽车精密零部件
- 人形机器人微型丝杠（新兴业务）

---

## 二、主营业务

### 2.1 业务结构

| 业务板块 | 营收占比 | 行业地位 |
|----------|---------|---------|
| 转向器齿条 | 50% | 国内龙头 |
| 减震器活塞杆 | 30% | 国内领先 |
| 汽车精密零部件 | 15% | 快速发展 |
| 机器人丝杠 | 3% | 核心增长点 |

### 2.2 机器人业务布局

**微型丝杠**：
- 灵巧手微型丝杠配套
- 投资18.5亿元建设生产基地
- 技术研发持续推进
- 目标进入人形机器人供应链

**产品特点**：
- 高精度、小尺寸
- 适配灵巧手关节需求
- 性能达到国际先进水平
