# 天智航（688027）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：医疗机器人核心标的

---

## 一、公司简介

北京天智航医疗科技股份有限公司成立于2005年，是国内领先的骨科手术机器人供应商。公司专注于骨科手术导航定位机器人研发生产，是国内首家获得CFDA认证的骨科手术机器人企业，也是科创板医疗器械领域首批上市公司之一。

**核心业务**：
- 骨科手术机器人（主业）
- 手术机器人耗材配件
- 手术机器人技术服务
- 远程手术解决方案
