# 亿嘉和（603666）投资分析报告

> **更新时间**：2026-02-28
> **行业定位**：特种机器人核心标的

---

## 一、公司简介

亿嘉和科技股份有限公司成立于1999年，是国内领先的特种机器人企业。公司主营电力巡检机器人、特种服务机器人等产品，在电力巡检机器人领域市场份额领先。

**核心业务**：
- 电力巡检机器人（主业）
- 特种机器人
- 智能运维系统
- 机器人解决方案

---
