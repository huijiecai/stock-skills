## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 多氟多新材料股份有限公司 |
| 证券代码 | 002407 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 2010年5月18日 |
| 所属行业 | 化学原料和化学制品制造业（氟化工） |
| 市场地位 | 氟化工+新能源双轮驱动，六氟磷酸锂龙头，电子级氢氟酸龙头 |

**公司概况**：
多氟多是国内氟化工与新能源材料双轮驱动企业。氟化工方面，电子级氢氟酸、无水氟化铝等产品国内领先；新能源方面，六氟磷酸锂产能全球前列，为锂电池电解液核心原料。2024年六氟磷酸锂价格受锂电需求影响波动，氟化工相对稳健。公司从氟化工延伸至新能源，具备产业链协同优势。若锂电需求复苏，六氟磷酸锂拐点逻辑明确。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **六氟磷酸锂** | 锂电池电解液核心原料，产能全球前列 | 动力电池、储能电池 |
| **电子级氢氟酸** | 半导体、光伏清洗用，国内龙头 | 半导体、光伏 |
| **无水氟化铝** | 电解铝助熔剂 | 电解铝 |
| **氟化锂** | 六氟磷酸锂原料 | 自用及外销 |
| **制冷剂** | R32等 | 空调 |
| **其他氟化工** | 含氟材料 | 化工、电子 |

### 业务模式

公司采用"氟化工+新能源"双轮驱动模式。氟化工为基盘，电子级氢氟酸、无水氟化铝等；新能源为成长极，六氟磷酸锂。从氟化工原料到六氟磷酸锂一体化，成本优势明显。下游与锂电、半导体、电解铝相关。

### 技术路线/核心技术

- **六氟磷酸锂**：锂电池电解液核心原料，技术壁垒高
- **电子级氢氟酸**：半导体、光伏级，国内龙头
- **氟化工一体化**：从萤石、氢氟酸到六氟磷酸锂
- **无水氟化铝**：电解铝助熔剂，规模领先

**技术优势**：
六氟磷酸锂产能全球前列，技术成熟。电子级氢氟酸国内龙头，半导体、光伏级。氟化工-新能源一体化，成本优势。

## 四、行业地位与产业链定位

### 产业链定位

```
氟化工-新能源产业链：
萤石 → 氢氟酸 → 【多氟多 电子级氢氟酸/六氟磷酸锂】→ 半导体/锂电
                        ↑
                    多氟多（双龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 六氟磷酸锂全球前列，电子级氢氟酸国内龙头 |
| **竞争优势** | 双龙头、一体化、技术 |
| **客户结构** | 电解液、半导体、光伏、电解铝 |
| **供应商** | 萤石、锂盐等 |

### 竞争格局

**主要竞争对手**：
- 六氟磷酸锂：天赐材料、天际股份、永太科技等
- 电子级氢氟酸：多氟多龙头，巨化、滨化等
- 氟化工：巨化股份、永和股份等

**公司优势**：
六氟磷酸锂产能全球前列，电子级氢氟酸国内龙头。氟化工-新能源一体化，成本优势。双轮驱动，分散单一周期风险。

## 五、投资亮点

### 核心亮点

- ✅ **六氟磷酸锂龙头**：产能全球前列
- ✅ **电子级氢氟酸龙头**：半导体、光伏级，国内第一
- ✅ **拐点逻辑**：六氟磷酸锂若锂电需求复苏，价格有望企稳反弹；电子级氢氟酸受益半导体、光伏国产化
- ✅ **双轮驱动**：氟化工+新能源，分散风险
- ✅ **一体化**：氟化工-六氟磷酸锂，成本优势

### 成长驱动因素

1. **六氟磷酸锂拐点**
   - 锂电需求若复苏
   - 六氟磷酸锂价格若企稳反弹
   - 产能全球前列，弹性大

2. **电子级氢氟酸**
   - 半导体、光伏国产化
   - 需求持续增长
   - 国内龙头，受益

3. **氟化工基盘**
   - 无水氟化铝、制冷剂等
   - 相对稳健
   - 一体化协同

### 催化剂

- **锂电需求**：动力电池、储能需求复苏
- **六氟磷酸锂价格**：触底反弹
- **半导体**：国产化，电子级氢氟酸需求
- **光伏**：电子级氢氟酸需求

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **六氟磷酸锂拐点** | 锂电需求若复苏，价格企稳反弹 | 锂电需求、价格反弹 |
| **电子级氢氟酸** | 半导体、光伏国产化，持续成长 | 半导体、光伏 |
| **业绩拐点** | 六氟磷酸锂若反弹，弹性大 | 锂电复苏 |
