## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江新化化工股份有限公司 |
| 证券代码 | 603867 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997年 |
| 上市时间 | 2019年6月27日 |
| 所属行业 | 化学原料和化学制品制造业（氟化工/精细化工） |
| 市场地位 | 脂肪胺龙头，异丙胺全球第一，氟化工有布局 |

**公司概况**：
新化股份是国内脂肪胺龙头，异丙胺产能全球第一，产品涵盖异丙胺、乙基胺、二异丙胺等。公司同时布局氟化工，生产氢氟酸、电子级氢氟酸等。脂肪胺下游为农药、医药、橡胶助剂；氟化工下游为制冷剂、半导体、光伏。2024年农药需求波动影响脂肪胺，氟化工受益于电子级氢氟酸需求。异丙胺全球第一，若农药拐点来临弹性大。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **异丙胺** | 产能全球第一 | 农药、医药、橡胶助剂 |
| **乙基胺** | 一乙胺、二乙胺、三乙胺 | 农药、医药、染料 |
| **二异丙胺** | 农药中间体 | 农药 |
| **氢氟酸** | 氟化工原料 | 制冷剂、含氟材料 |
| **电子级氢氟酸** | 半导体、光伏级 | 半导体、光伏 |
| **香料** | 合成香料 | 日化、食品 |

### 业务模式

公司采用"脂肪胺+氟化工+香料"多产品模式，脂肪胺为核心，异丙胺全球第一。氟化工为延伸，电子级氢氟酸受益半导体、光伏国产化。下游与农药、医药、半导体、光伏相关。

### 技术路线/核心技术

- **脂肪胺**：异丙胺、乙基胺等，催化胺化
- **异丙胺**：全球产能第一，技术领先
- **氟化工**：氢氟酸、电子级氢氟酸
- **香料**：合成香料

**技术优势**：
异丙胺全球第一，脂肪胺龙头。电子级氢氟酸有布局，受益半导体、光伏。多产品平台，分散风险。

## 四、行业地位与产业链定位

### 产业链定位

```
脂肪胺-氟化工产业链：
丙酮/乙醇 → 【新化股份 异丙胺/乙基胺】→ 农药/医药
萤石+硫酸 → 【新化股份 氢氟酸/电子级氢氟酸】→ 制冷剂/半导体
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 异丙胺全球第一，脂肪胺龙头 |
| **竞争优势** | 异丙胺全球第一、氟化工、多产品 |
| **客户结构** | 农药、医药、半导体、光伏 |
| **供应商** | 丙酮、乙醇、萤石等 |

### 竞争格局

**主要竞争对手**：
- 脂肪胺：新化股份异丙胺全球第一，其他胺类企业
- 氟化工：多氟多、巨化、永和等

**公司优势**：
异丙胺全球第一，脂肪胺龙头。电子级氢氟酸有布局。多产品平台，分散风险。

## 五、投资亮点

### 核心亮点

- ✅ **异丙胺全球第一**：脂肪胺龙头
- ✅ **拐点逻辑**：农药需求若复苏，脂肪胺（异丙胺）需求改善；电子级氢氟酸受益半导体、光伏国产化
- ✅ **氟化工布局**：电子级氢氟酸，成长性
- ✅ **多产品平台**：脂肪胺+氟化工+香料
- ✅ **农药周期**：农药5-6年周期，若拐点来临弹性大

### 成长驱动因素

1. **农药拐点**
   - 农药行业5-6年大周期
   - 异丙胺为农药中间体，需求若复苏
   - 龙头弹性大

2. **电子级氢氟酸**
   - 半导体、光伏国产化
   - 需求持续增长
   - 氟化工延伸

3. **脂肪胺**
   - 医药、橡胶助剂需求
   - 异丙胺全球第一，议价能力强

### 催化剂

- **农药需求**：农药行业拐点
- **异丙胺价格**：触底反弹
- **半导体**：电子级氢氟酸需求
- **光伏**：电子级氢氟酸需求

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **农药拐点** | 农药5-6年周期，异丙胺需求若复苏 | 农药行业拐点 |
| **氟化工拐点** | 电子级氢氟酸受益半导体、光伏 | 半导体、光伏国产化 |
| **业绩拐点** | 异丙胺全球第一，拐点来临弹性大 | 农药复苏、异丙胺价涨 |
