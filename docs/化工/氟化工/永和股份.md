## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江永和制冷股份有限公司 |
| 证券代码 | 605020 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2004年 |
| 上市时间 | 2021年7月9日 |
| 所属行业 | 化学原料和化学制品制造业（氟化工） |
| 市场地位 | 氟精细化学品龙头，制冷剂+含氟聚合物 |

**公司概况**：
永和股份是国内氟精细化学品龙头，产品涵盖制冷剂、含氟聚合物、含氟精细化学品等。公司拥有完整的氟化工产业链，从萤石、氢氟酸到制冷剂、FEP、PVDF等。2024年制冷剂配额制实施，制冷剂景气上行；含氟聚合物与新能源、半导体相关。氟精细化学品技术壁垒高，公司为国内少数具备全产业链能力的企业之一。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **制冷剂** | R32、R134a、R125等HFCs | 家用/商用空调、汽车空调 |
| **含氟聚合物** | FEP、PVDF、PFA等 | 新能源、半导体、高端制造 |
| **含氟精细化学品** | 含氟中间体、特种化学品 | 医药、农药、电子 |
| **氢氟酸** | 氟化工原料 | 制冷剂、含氟材料上游 |
| **萤石** | 氟化工原料 | 自给或外采 |

### 业务模式

公司采用氟化工全产业链一体化模式，从萤石、氢氟酸到制冷剂、含氟聚合物、含氟精细化学品全覆盖。制冷剂受益于配额制；含氟聚合物受益于新能源、半导体；含氟精细化学品技术壁垒高、盈利能力强。

### 技术路线/核心技术

- **制冷剂**：R32、R134a、R125等第三代制冷剂
- **含氟聚合物**：FEP、PVDF、PFA等
- **含氟精细化学品**：医药、农药中间体
- **氟化工一体化**：从萤石到终端产品

**技术优势**：
氟精细化学品技术壁垒高，国内少数全产业链企业。制冷剂配额制受益。含氟聚合物与新能源、半导体需求相关，成长性强。

## 四、行业地位与产业链定位

### 产业链定位

```
氟化工产业链：
萤石 → 氢氟酸 → 【永和股份 制冷剂/含氟聚合物/精细化学品】→ 空调/新能源/半导体
                        ↑
                    永和股份（氟精细龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 氟精细化学品龙头，制冷剂有配额 |
| **竞争优势** | 全产业链、技术壁垒、氟精细 |
| **客户结构** | 空调、新能源、半导体、医药 |
| **供应商** | 萤石、硫酸等 |

### 竞争格局

**主要竞争对手**：
- 巨化股份：制冷剂龙头，配额第一
- 三美股份：制冷剂
- 多氟多：氟化工、新能源
- 东岳集团：氟化工

**公司优势**：
氟精细化学品龙头，技术壁垒高。制冷剂有配额，配额制受益。含氟聚合物与新能源、半导体相关，成长性强。

## 五、投资亮点

### 核心亮点

- ✅ **氟精细化学品龙头**：技术壁垒高，国内少数全产业链
- ✅ **拐点逻辑**：制冷剂配额制已实施，景气上行；含氟聚合物若新能源、半导体需求复苏有弹性
- ✅ **制冷剂配额**：配额制受益，供给受限
- ✅ **含氟聚合物**：FEP、PVDF与新能源、半导体相关
- ✅ **氟精细**：医药、农药中间体，高毛利

### 成长驱动因素

1. **制冷剂景气**
   - 配额制实施，供给受限
   - 制冷剂价格上行
   - 公司有配额，受益

2. **含氟聚合物**
   - PVDF与锂电隔膜、光伏背板相关
   - FEP与高端线缆、半导体相关
   - 新能源、半导体需求若复苏

3. **氟精细化学品**
   - 医药、农药中间体
   - 技术壁垒高，盈利能力强
   - 进口替代

### 催化剂
