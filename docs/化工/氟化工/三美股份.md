## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江三美化工股份有限公司 |
| 证券代码 | 603379 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2007年3月21日（2001年始建） |
| 上市时间 | 2019年4月2日 |
| 所属行业 | 化学原料和化学制品制造业（氟化工） |
| 市场地位 | 制冷剂行业第二，三代制冷剂配额占比行业第二 |

**公司概况**：
三美股份是国内氟化工领域领先企业，主营制冷剂与含氟材料。公司获得三代制冷剂行业第二高的配额占比，R134a配额5.0万吨、R125配额3.1万吨、R32配额3.3万吨。2024年制冷剂价格持续上涨，公司实现量增价涨，业绩高速增长。同时推进六氟磷酸锂、PVDF/FEP、ECH等高附加值产品布局。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **第三代制冷剂** | R134a、R125、R32等HFCs | 家用/商用空调、冰箱、汽车空调 |
| **氟化氢** | 制冷剂及含氟材料原料 | 氟化工上游 |
| **含氟材料** | 六氟磷酸锂、PVDF、FEP等（在建） | 新能源电池、半导体 |
| **ECH** | 环氧氯丙烷（在建） | 环氧树脂、水处理 |

### 业务模式

公司以制冷剂为核心，制冷剂业务营收占比约85.8%。2024年三代制冷剂HFCs供应总量受控，公司凭借高配额占比充分受益。同时推进氟化工产业一体化布局，投资建设六氟磷酸锂、PVDF/FEP、ECH等高附加值产品项目，拓展成长空间。

### 技术路线/核心技术

- **第三代制冷剂全系列**：R134a、R125、R32主流品种
- **配额优势**：R134a 5.0万吨、R125 3.1万吨、R32 3.3万吨
- **氟化工延伸**：六氟磷酸锂、PVDF/FEP等新能源材料
- **一体化布局**：从氟化氢到制冷剂、含氟材料

**技术优势**：
三美股份制冷剂配额占比行业第二，R32产能6-7万吨/年。公司持续投入氟化工新材料研发，六氟磷酸锂、PVDF/FEP等项目有序推进，有望打开第二增长曲线。

## 四、行业地位与产业链定位

### 产业链定位

```
氟化工产业链：
萤石 → 氟化氢 → 【三美股份 制冷剂/含氟材料】→ 空调/新能源
                      ↑
                 三美股份
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 三代制冷剂配额占比行业第二，R134a 5.0万/R125 3.1万/R32 3.3万吨 |
| **竞争优势** | 制冷剂业务聚焦、配额领先、业绩弹性大 |
| **客户结构** | 覆盖空调、冰箱、汽车空调等下游客户 |
| **供应商** | 氟化氢等原料部分自给 |

### 竞争格局

**主要竞争对手**：
- 巨化股份：配额占比30.7%国内第一，R32市占率45.82%
- 永和股份：R32产能4-5万吨/年
- 东岳集团：氟化工竞争对手

**公司优势**：
三美股份制冷剂业务营收占比高（85.8%），业绩对制冷剂价格弹性大。配额占比行业第二，2024年制冷剂景气上行时充分受益，业绩增速领先。机构上调2024-2026年盈利预测至7.88/15.98/18.44亿元。

## 五、投资亮点

### 核心亮点

- ✅ **制冷剂配额行业第二**：R134a 5.0万、R125 3.1万、R32 3.3万吨
- ✅ **业绩高弹性**：制冷剂业务占比85.8%，价格上涨业绩弹性大
- ✅ **2024年业绩兑现**：归母净利润+178%，扣非+258%
- ✅ **2025Q1延续高增**：预告净利润+139%-178%，制冷剂均价+43%
- ✅ **氟化工延伸**：六氟磷酸锂、PVDF/FEP、ECH等项目布局

### 成长驱动因素

1. **制冷剂景气持续**
   - 2024年配额制实施，2025年配额进一步削减
   - 制冷剂价格持续上涨，2025Q1均价同比+43%
   - 公司高配额占比充分受益

2. **新材料项目落地**
   - 六氟磷酸锂：新能源电池电解液
   - PVDF/FEP：锂电池粘结剂、半导体
   - ECH：环氧树脂、水处理

3. **盈利预测上调**
   - 机构预期2024-2026年盈利7.88/15.98/18.44亿元
   - 制冷剂价格持续上涨支撑业绩

### 催化剂

- 2025年制冷剂配额进一步削减
- 空调旺季需求拉动
- 六氟磷酸锂、PVDF等项目投产
- 制冷剂均价持续上行
