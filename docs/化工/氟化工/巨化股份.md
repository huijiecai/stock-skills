## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江巨化股份有限公司 |
| 证券代码 | 600160 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年6月1日 |
| 上市时间 | 1998年6月26日 |
| 所属行业 | 化学原料和化学制品制造业（氟化工） |
| 市场地位 | 第三代制冷剂全球龙头，配额占比30.7%国内第一 |

**公司概况**：
巨化股份是国内氟化工行业龙头企业，第三代制冷剂（HFCs）配额总量占全国30.7%，位居国内第一。公司拥有完整的氟化工产业链，产品涵盖制冷剂、氟化工原料、含氟聚合物材料、基础化工产品等，混配小包装制冷剂市占率居全球第一。2024年制冷剂配额制实施后，公司实现量价齐升，业绩大幅增长。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **第三代制冷剂** | R32、R134a、R125等HFCs | 家用/商用空调、冰箱、汽车空调 |
| **氟化工原料** | 氢氟酸、甲烷氯化物等 | 制冷剂、含氟材料上游 |
| **含氟聚合物** | PTFE、PVDF、FEP等 | 新能源、半导体、高端制造 |
| **基础化工** | 烧碱、氯碱等 | 化工、造纸、纺织 |
| **石化材料** | 己内酰胺等 | 化纤、工程塑料 |

### 业务模式

公司采用氟化工全产业链一体化模式，从萤石、硫酸到制冷剂、含氟材料全覆盖，原料自给能力强，成本优势显著。2024年制冷剂配额制实施，供给受限，公司凭借高配额占比充分受益于行业景气上行。

### 技术路线/核心技术

- **第三代制冷剂全系列**：R32、R134a、R125等主流品种全覆盖
- **氟化工一体化**：从氢氟酸到制冷剂、含氟材料的完整产业链
- **配额优势**：2024年配额总量22.42万吨，占全国30.7%
- **混配制冷剂**：小包装制冷剂市占率全球第一

**技术优势**：
巨化股份拥有国内最完整的氟化工产业链，R32产能10万吨/年、R134a产能6万吨/年、R125产能5万吨/年。2025年R32市占率约45.82%，几乎占据国内市场半壁江山。在建、拟建第三代制冷剂产能7万吨，巩固市场优势。

## 四、行业地位与产业链定位

### 产业链定位

```
氟化工产业链：
萤石 → 氢氟酸 → 制冷剂原料 → 【巨化股份 制冷剂/含氟材料】→ 空调/新能源/半导体
                                    ↑
                               巨化股份
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 三代制冷剂配额占全国30.7%国内第一，R32市占率45.82% |
| **竞争优势** | 产业链一体化、原料自给、配额领先、混配小包装全球第一 |
| **客户结构** | 覆盖格力、美的、海尔等空调龙头，汽车空调、冰箱客户 |
| **供应商** | 萤石等原料自给，成本控制能力强 |

### 竞争格局

**主要竞争对手**：
- 三美股份：R32产能6-7万吨/年，制冷剂业务营收占比85.8%，配额行业第二
- 永和股份：R32产能4-5万吨/年
- 东岳集团：氟化工竞争对手

**公司优势**：
巨化股份配额总量、R32市占率均居国内第一，产业链完整度和成本优势优于同业。2024年配额制实施后，供给受限，公司高配额占比充分受益，业绩兑现能力强。市值已突破千亿，氟化工行业迎爆发期。

## 五、投资亮点

### 核心亮点

- ✅ **三代制冷剂全球龙头**：配额占比30.7%国内第一，R32市占率45.82%
- ✅ **配额制核心受益者**：2024年配额制实施，供给受限，量价齐升
- ✅ **产业链一体化**：从萤石到制冷剂全覆盖，成本优势显著
- ✅ **混配制冷剂全球第一**：小包装制冷剂市占率居全球第一
- ✅ **AI液冷需求**：数据中心液冷对制冷剂需求增长，长期受益

### 成长驱动因素

1. **制冷剂景气持续**
   - 2024年配额制实施，2025年配额进一步削减
   - 行业供需格局优化，制冷剂价格持续上行
   - 2025Q1均价同比+58%，景气度看好

2. **产能扩张**
   - 在建、拟建三代制冷剂产能7万吨（山东3万吨、本埠4万吨）
   - 应对旺季交付需求，巩固市场优势

3. **AI液冷等新需求**
   - 数据中心液冷对制冷剂需求增长
   - 新能源汽车热管理、冷链物流等增量需求

### 催化剂

- 2025年制冷剂配额进一步削减
- 空调旺季需求拉动
- AI数据中心液冷建设加速
- 含氟聚合物（PVDF等）新能源需求

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **制冷剂价格波动** | 价格高位后若需求不及预期，存在回调风险 |
| ⚠️ **配额政策变化** | 配额削减节奏若放缓，供给约束减弱 |

| ⚠️ **萤石成本** | 萤石价格波动影响成本 |
| ⚠️ **四代制冷剂替代** | 长期看HFO等低GWP产品可能替代三代 |
