## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 卫星化学股份有限公司 |
| 证券代码 | 002648 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2005年8月3日（创业于1992年） |
| 上市时间 | 2011年12月 |
| 所属行业 | 化学原料和化学制品制造业 |
| 市场地位 | 丙烯酸龙头，乙烷裂解成本优势显著 |

**公司概况**：
卫星化学是国内丙烯酸及酯龙头，拥有C2（乙烷裂解）和C3（丙烯酸）双产业链。公司乙烷裂解制乙烯路线成本优势显著，2025年2月乙烷裂解利润2546元/吨，远高于石脑油裂解(56.5元/吨)和煤制烯烃(1309.6元/吨)。2024年业绩创历史新高，归母净利润61亿元，同比+26.8%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **丙烯酸及酯** | 丙烯酸、丙烯酸丁酯等 | 涂料、胶粘剂、SAP、纺织 |
| **乙烯及下游** | 乙烷裂解制乙烯、聚乙烯、乙二醇等 | 新材料、聚酯、新能源 |
| **多碳醇** | 80万吨/年项目2024年7月投产 | C3产业链完整闭环 |
| **高分子新材料** | 高端聚烯烃、POE等 | 新能源、高端材料 |
| **环氧乙烷下游** | 乙醇胺等 | 表面活性剂、医药 |

### 业务模式

公司采用"C2+C3"双产业链模式。C2产业链以乙烷裂解为核心，原料从美国进口，成本优势显著。C3产业链以丙烯酸为核心，2024年7月80万吨多碳醇项目投产，实现C3完整闭环。高分子新材料板块高速增长，高端聚烯烃国产化加速。

### 技术路线/核心技术

- **乙烷裂解**：美国乙烷原料，裂解制乙烯，成本远优于石脑油和煤制烯烃
- **C3产业链**：丙烯酸-丙烯酸酯-多碳醇，完整闭环
- **高端聚烯烃**：α-烯烃、POE等新能源材料
- **连云港基地**：α-烯烃综合利用产业园，2025年底建成

**技术优势**：
乙烷裂解是三种乙烯路线中最具盈利优势的方案。2024年乙烷价格同比-22%，进一步扩大价差。公司C3产业链完整，丙烯酸毛利从2023年539元/吨提升至2024年723元/吨。高分子新材料毛利率35.05%，同比+5.49pct。

## 四、行业地位与产业链定位

### 产业链定位

```
C2+C3产业链：
乙烷/丙烯 → 【卫星化学 乙烯/丙烯酸】→ 聚乙烯/丙烯酸酯/新材料 → 新能源/涂料/高端材料
                    ↑
              卫星化学（乙烷裂解+丙烯酸龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 丙烯酸及酯龙头，乙烷裂解产能国内领先 |
| **竞争优势** | 乙烷裂解成本优势、C3产业链完整、新材料布局 |
| **客户结构** | 覆盖涂料、胶粘剂、新能源、高端材料等 |
| **供应商** | 美国乙烷进口，长协保障 |

### 竞争格局

**主要竞争对手**：
- 万华化学：石化、聚氨酯、新材料
- 荣盛石化：大炼化、EVA、高端聚烯烃
- 丙烯酸行业：华谊集团、江苏裕廊等

**公司优势**：
卫星化学乙烷裂解成本优势显著，为三种乙烯路线中最优。C3产业链完整，丙烯酸龙头地位稳固。高分子新材料、POE等新能源材料布局领先，高端聚烯烃国产化加速。机构预计2025-2027年归母净利润分别为75.6亿、102.5亿、120.7亿元。

## 五、投资亮点

### 核心亮点

- ✅ **乙烷裂解成本优势**：利润2546元/吨，远高于石脑油、煤制烯烃
- ✅ **丙烯酸龙头**：C3产业链完整，丙烯酸毛利持续提升
- ✅ **业绩创历史新高**：2024年净利61亿同比+26.8%，Q4单季23.8亿
- ✅ **新材料高增长**：高分子新材料营收119.87亿，毛利率35%
- ✅ **新产能释放**：多碳醇、α-烯烃、POE等项目陆续投产

### 成长驱动因素

1. **乙烷成本优势延续**
   - 乙烷价格处于相对低位
   - 乙烷裂解价差维持高位

2. **新项目投产**
   - 80万吨多碳醇项目已投产
   - α-烯烃综合利用产业园2025年底建成，2026年初投产
   - POE、高端聚烯烃等装置

3. **新能源材料**
   - 高端聚烯烃国产化替代
   - POE等光伏材料需求
   - 新能源材料高速增长

### 催化剂

- 乙烷价格维持低位
- 连云港α-烯烃、POE项目投产
- 新能源材料需求增长
- 丙烯酸价格企稳
