## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 龙佰集团股份有限公司 |
| 证券代码 | 002601 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年（前身可追溯至1955年焦作市硫磺矿） |
| 上市时间 | 2011年7月15日 |
| 所属行业 | 化学原料和化学制品制造业 |
| 市场地位 | 钛白粉全球龙头，产能占全球11.7% |

**公司概况**：
龙佰集团是全球第三大、中国第一大氯化法钛白粉生产企业，钛白粉总产能151万吨/年，其中氯化法占比约26%。公司拥有钛精矿-钛白粉-海绵钛全产业链，钛精矿自给率超50%，成本优势显著。总部位于河南焦作，在职员工近两万人，自2011年上市以来连续13年分红，累计分红超160亿元。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **硫酸法钛白粉** | 65万吨产能，2024年销售79.64万吨 | 涂料、塑料、造纸、油墨 |
| **氯化法钛白粉** | 36-66万吨产能，2024年销售36.25万吨 | 高端涂料、塑料、造纸 |
| **钛精矿** | 自给率超50%，2025年计划扩充至250万吨 | 钛白粉原料 |
| **海绵钛** | 8万吨/年产能，市占率22.41% | 航空航天、军工、化工 |
| **其他** | 硫酸、铁系产品等 | 配套、综合利用 |

### 业务模式

公司采用"钛精矿-钛白粉-海绵钛"全产业链模式，上游钛精矿自给率高，降低原料成本。硫酸法+氯化法双工艺，氯化法产品高端化、单耗低。海绵钛业务从净进口转为净出口，成为新增长点。持续布局产业链资源，钛精矿产能扩充中。

### 技术路线/核心技术

- **氯化法工艺**：单耗低7.5%-13%，产品高端，环保优势
- **硫酸法工艺**：成熟稳定，产能规模大
- **钛精矿自给**：自给率超50%，完全成本10,538元/吨远低于行业
- **全产业链**：从钛矿到钛白粉、海绵钛一体化

**技术优势**：
龙佰集团氯化法钛白粉产能国内第一，全球第三。钛精矿自给率超50%，成本优势显著，完全成本较行业平均低。氯化法工艺能耗、物耗低，符合环保趋势。海绵钛实现进口替代，2024年上半年零进口。

## 四、行业地位与产业链定位

### 产业链定位

```
钛白粉产业链：
钛精矿 → 【龙佰集团 钛白粉/海绵钛】→ 涂料/塑料/造纸 → 建筑/汽车/包装
                    ↑
              龙佰集团（全产业链）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 钛白粉产能151万吨/年，占全球11.7%、亚洲20.9% |
| **竞争优势** | 氯化法国内第一、钛精矿自给、全产业链、成本领先 |
| **客户结构** | 覆盖涂料、塑料、造纸等下游头部企业 |
| **供应商** | 钛精矿自给率超50%，上游可控 |

### 竞争格局

**主要竞争对手**：
- 科慕（Chemours）：全球钛白粉龙头
- 特诺（Tronox）：全球主要供应商
- 中核钛白：国内硫酸法钛白粉
- 金浦钛业、安纳达等：国内竞争者

**公司优势**：
龙佰集团凭借氯化法产能国内第一、钛精矿自给、全产业链布局、成本优势，在国内外竞争中具备显著优势。钛精矿景气延续，自给优势进一步凸显。海绵钛业务成为新增长点，进口替代完成。

## 五、投资亮点

### 核心亮点

- ✅ **钛白粉全球龙头**：产能151万吨，占全球11.7%，氯化法国内第一
- ✅ **全产业链成本优势**：钛精矿自给率超50%，完全成本10,538元/吨
- ✅ **业绩改善**：2024年净利+19.68%，毛利率+9.66pct
- ✅ **钛白粉涨价**：季度均价较2023年底+660元/吨
- ✅ **海绵钛新增长点**：8万吨产能，市占率22.41%，净出口
- ✅ **高分红**：连续13年分红，累计160.56亿元，分红率69.42%

### 成长驱动因素

1. **钛白粉景气**
   - 2024年价格企稳回升
   - 房地产政策有望拉动需求
   - 氯化法高端产品占比提升

2. **钛精矿自给**
   - 2025年计划扩充至250万吨
   - 自给率提升进一步降本

3. **海绵钛放量**
   - 进口替代完成，净出口
   - 航空航天、军工需求

### 催化剂

- 房地产政策带动涂料需求
- 钛白粉价格继续上行
- 钛精矿产能扩充
- 海绵钛产销放量
