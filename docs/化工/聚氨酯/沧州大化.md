## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 沧州大化股份有限公司 |
| 证券代码 | 600230 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年9月24日 |
| 上市时间 | 2000年4月6日 |
| 所属行业 | 化学原料和化学制品制造业 |
| 市场地位 | 国内TDI龙头，市占率超30% |

**公司概况**：
沧州大化是国内TDI（甲苯二异氰酸酯）龙头企业，TDI产能12万吨/年，计划扩建至41.5万吨/年。公司形成TDI、烧碱、PC（聚碳酸酯）、双酚A等多产品布局，产品出口至巴西、巴基斯坦等国际市场，TDI外单创历史新高。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **TDI** | 甲苯二异氰酸酯，聚氨酯软泡核心原料 | 软体家具、沙发、床垫、涂料、胶粘剂 |
| **烧碱** | 氢氧化钠 | 化工、造纸、纺织、水处理 |
| **PC** | 聚碳酸酯 | 汽车、电子、建材、医疗 |
| **双酚A** | PC及环氧树脂原料 | 聚碳酸酯、环氧树脂 |

### 业务模式

公司采用"生产-销售"一体化模式，以TDI为核心，拓展烧碱、PC、双酚A等产品，形成产业链延伸。2024年完成20项提质增效项目，增效目标达成率146%，不变价成本达历史最低水平。PC产品获欧盟REACH认证，建立海外长期合作关系。

### 技术路线/核心技术

- **光气法TDI工艺**：成熟的光气法生产技术
- **PC生产技术**：非光气法工艺，高端产品市场竞争力提升
- **提质增效**：通过技改降本，不变价成本创新低
- **新产品研发**：成功研发20%高硅含量共聚硅PC等新产品

**技术优势**：
2024年获得9项专利授权，PC产品获欧盟REACH认证。公司持续投入研发，高端PC产品市场竞争力提升，建立海外长期合作关系。

## 四、行业地位与产业链定位

### 产业链定位

```
聚氨酯产业链：
甲苯/硝酸 → TDI → 【沧州大化】→ 聚氨酯软泡 → 家具/涂料/胶粘剂
                    ↑
               沧州大化
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内TDI市占率超30%，现有产能12万吨/年，计划扩建至41.5万吨/年 |
| **竞争优势** | TDI产能国内第二梯队，计划扩产后有望跃居全国第一 |
| **客户结构** | 覆盖国内软体家具、涂料等客户，出口至巴西、巴基斯坦等 |
| **供应商** | 上游甲苯等原料依赖外部采购 |

### 竞争格局

**主要竞争对手**：
- 万华化学：国内TDI产能约30万吨/年，份额约51%，计划扩能至142万吨/年
- 科思创：全球TDI主要供应商
- 巴斯夫：欧洲TDI生产商
- 甘肃银光：国内TDI供应商

**公司优势**：
沧州大化TDI产能国内第二梯队，计划扩建至41.5万吨/年后将超过万华化学现有30万吨/年，成为中国最大TDI制造商。2024年TDI产量和销量创历史新高，国际市场占有率不断扩大。

## 五、投资亮点

### 核心亮点

- ✅ **国内TDI龙头**：市占率超30%，产能扩建后有望跃居全国第一
- ✅ **TDI量创新高**：2024年产量和销量创历史新高
- ✅ **烧碱高毛利**：烧碱毛利率41.37%，贡献稳定利润
- ✅ **PC高端突破**：获欧盟REACH认证，建立海外长期合作
- ✅ **成本控制**：不变价成本达历史最低，提质增效显著

### 成长驱动因素

1. **TDI产能扩建**
   - 计划扩建至41.5万吨/年（现有12万吨→16万吨+新建22.5万吨）
   - 扩建完成后产能将超过万华化学现有规模

2. **产品结构优化**
   - PC高端产品占比提升，20%高硅含量共聚硅PC等新产品
   - 烧碱业务保持高毛利

3. **海外市场拓展**
   - PC获欧盟REACH认证
   - TDI出口创历史新高

### 催化剂

- TDI价格触底反弹
- 产能扩建项目获批及投产
- 软体家具、涂料下游需求复苏
- 双酚A扭亏为盈
