## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 万华化学集团股份有限公司 |
| 证券代码 | 600309 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年12月20日 |
| 上市时间 | 2001年1月5日 |
| 所属行业 | 化学原料和化学制品制造业 |
| 市场地位 | 全球MDI龙头，市占率超30% |

**公司概况**：
万华化学是全球最大的MDI（二苯基甲烷二异氰酸酯）制造商，拥有第六代MDI自主知识产权技术，是中国首家掌握MDI制造技术的企业。公司业务涵盖聚氨酯、石化、精细化学品及新材料三大板块，已形成从原料到终端应用的完整产业链布局。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **MDI** | 异氰酸酯类产品，聚氨酯核心原料 | 建筑保温、家电、汽车、鞋材、胶粘剂 |
| **TDI** | 甲苯二异氰酸酯 | 软体家具、涂料、胶粘剂 |
| **聚醚多元醇** | 聚氨酯另一核心原料 | 与MDI配套使用 |
| **石化产品** | 丙烯、乙烯及下游衍生物 | 新材料、精细化工 |
| **精细化学品** | 特种胺、ADI等 | 高端材料、电子化学品 |

### 业务模式

公司采用"研发-生产-销售"一体化模式，以技术创新驱动产品升级，通过规模化生产降低成本，产品销往全球100多个国家和地区。2024年持续推进数智化转型，提升装置自主运行水平。

### 技术路线/核心技术

- **第六代MDI技术**：拥有完全自主知识产权，技术领先全球
- **光气法工艺**：MDI/TDI主流生产工艺，具备成熟产业化能力
- **一体化产业链**：从煤/石油到MDI的完整技术链条
- **数智化制造**：持续投入提升装置运行效率和安全性

**技术优势**：
万华化学是中国首家拥有MDI制造技术自主知识产权的企业，打破国外技术垄断。公司在烟台、宁波、福建福清（在建）和匈牙利设有MDI生产基地，全球产能约305万吨/年，居全球第一。

## 四、行业地位与产业链定位

### 产业链定位

```
聚氨酯产业链：
石油/煤炭 → 苯胺/光气 → 【万华化学 MDI/TDI】→ 聚氨酯制品 → 建筑/家电/汽车/鞋材
                              ↑
                        万华化学
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球MDI市占率超30%，产能305万吨/年居全球第一 |
| **竞争优势** | 第六代MDI技术自主可控，一体化成本优势，全球化布局 |
| **客户结构** | 覆盖全球头部聚氨酯制品企业，客户遍布100+国家 |
| **供应商** | 原料自给能力较强，具备上游一体化布局 |

### 竞争格局

**主要竞争对手**：
- 科思创（原拜耳材料科技）：全球第二大MDI供应商
- 巴斯夫：欧洲主要MDI生产商
- 亨斯迈：美国MDI生产商
- 日本东曹等：亚洲区域竞争者

**公司优势**：
万华化学凭借产能规模、技术自主、成本控制和全球化布局，已成为全球最大、最有竞争力的MDI制造商。中国产能占全球比重持续提升，公司龙头地位稳固。

## 五、投资亮点

### 核心亮点

- ✅ **全球MDI龙头**：市占率超30%，产能全球第一，定价权强
- ✅ **技术自主可控**：第六代MDI技术打破垄断，持续迭代升级
- ✅ **一体化产业链**：从原料到终端全覆盖，成本优势显著
- ✅ **多基地布局**：烟台、宁波、福建、匈牙利四大基地，分散风险
- ✅ **新材料拓展**：精细化学品及新材料板块成长空间大

### 成长驱动因素

1. **MDI需求稳健增长**
   - 建筑节能、家电升级、新能源汽车轻量化带动聚氨酯需求
   - 全球聚氨酯市场年复合增长率约5%

2. **产能持续扩张**
   - 福建基地、匈牙利基地产能爬坡
   - 技改扩能项目陆续投产

3. **新材料业务放量**
   - 特种胺、ADI、电子化学品等高附加值产品
   - 新能源材料、可降解材料等新方向

### 催化剂

- 房地产政策放松带动建筑保温需求
- 家电以旧换新政策刺激需求
- 原油/煤炭价格回落改善成本端
- 海外基地产能释放贡献增量

## 七、业绩预期与估值

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|--------------|--------------|
| **乐观** | MDI价格回升、需求旺盛 | 净利润增速15%+ | 净利润增速10%+ |
| **中性** | 价格企稳、成本可控 | 净利润增速0-10% | 净利润增速5-10% |

| 指标 | 当前水平 | 行业平均 | 评估 |
|------|----------|----------|------|
| PE(TTM) | 约15-20倍 | 化工行业约15倍 | 合理 |
