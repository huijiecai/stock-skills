## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 云南云天化股份有限公司 |
| 证券代码 | 600096 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997年 |
| 上市时间 | 1997年7月9日 |
| 所属行业 | 化学原料和化学制品制造业（磷化工） |
| 市场地位 | 全球磷肥龙头，磷肥产量全国第一，磷酸二铵市占率27% |

**公司概况**：
云天化是全球磷肥行业龙头企业，磷肥产能555万吨/年位居全国第二、全球第四，化肥总产能1000万吨/年。公司拥有"矿化一体"全产业链优势，磷矿储量约8亿吨，磷矿采选年生产能力1368万吨。2024年磷产业链维持高景气，公司实现业绩稳健增长，分红力度创历史新高。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **磷肥** | 磷酸一铵、磷酸二铵 | 农业肥料 |
| **尿素** | 氮肥 | 农业肥料 |
| **复合肥** | 氮磷钾复合肥 | 农业肥料 |
| **磷化工** | 磷酸盐等 | 工业、食品、新能源 |
| **磷矿石** | 磷矿采选 | 磷化工原料 |

### 业务模式

公司采用"矿化一体"全产业链运营模式，从磷矿开采到磷肥、磷化工、复合肥全覆盖。拥有丰富的磷矿和煤炭资源，原材料自给率高，上下游产业链完整。通过成本控制、产品结构优化，核心产品磷酸一铵、二铵毛利率远超同行。

### 技术路线/核心技术

- **磷矿采选**：年生产能力1368万吨，原矿开采能力1450万吨
- **湿法磷酸**：磷肥核心工艺
- **矿化一体**：磷矿-磷酸-磷肥一体化，成本优势突出
- **450万吨磷矿浮选**：项目建设推进中

**技术优势**：
云天化磷矿储量约8亿吨，磷矿和煤炭资源丰富，原材料自给率高。通过"矿化一体"模式，硫磺、原料煤采购成本同比下降6.46%、12.60%，成本控制能力强。

## 四、行业地位与产业链定位

### 产业链定位

```
磷化工产业链：
磷矿石 → 湿法磷酸 → 【云天化 磷肥/磷化工】→ 农业/工业/新能源
                         ↑
                    云天化
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 磷肥产量全国第一（2019-2021连续三年），磷酸二铵市占率约27%全国第一 |
| **竞争优势** | 矿化一体、磷矿储量8亿吨、磷肥产能555万吨/年全球第四 |
| **客户结构** | 覆盖国内农业主产区，出口东南亚等市场 |
| **供应商** | 磷矿、煤炭自给，原材料保障能力强 |

### 竞争格局

**主要竞争对手**：
- 贵州磷化：磷肥产能全国第一
- 兴发集团：磷化工、电子级磷酸
- 川恒股份：磷化工、磷酸铁锂
- 国际：美盛、OCP等全球磷肥巨头

**公司优势**：
云天化磷肥产量曾连续三年全国第一，磷酸二铵市占率27%全国第一。矿化一体成本优势突出，核心产品毛利率远超同行。2024年磷产业链高景气，公司业绩稳健增长，分红力度创历史新高。

## 五、投资亮点

### 核心亮点

- ✅ **全球磷肥龙头**：磷肥产能555万吨/年全球第四，磷酸二铵市占率27%全国第一
- ✅ **矿化一体**：磷矿储量8亿吨，原材料自给率高，成本优势突出
- ✅ **磷产业链高景气**：2024年磷矿石、磷肥价格高位，毛利率显著提升
- ✅ **高分红**：分红比例47.86%，分红25.52亿元创历史新高，未来三年不低于45%
- ✅ **磷酸铁锂布局**：推进新能源材料领域，450万吨磷矿浮选项目

### 成长驱动因素

1. **磷产业链景气延续**
   - 磷矿石价格保持高位
   - 磷肥、磷化工毛利率提升
   - 矿化一体成本优势巩固

2. **产品结构优化**
   - 磷肥、尿素、复合肥产销量稳步增长
   - 磷化工向高附加值延伸
   - 磷酸铁锂等新能源材料布局

3. **成本控制**
   - 硫磺、原料煤价格同比下降
   - 资产负债率降至52.26%
   - 财务费用、销售费用双降

### 催化剂

- 磷矿石价格维持高位
- 农业需求稳健
- 磷酸铁锂项目推进
- 450万吨磷矿浮选投产
