## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 贵州川恒化工股份有限公司 |
| 证券代码 | 002895 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2002年11月25日 |
| 上市时间 | 2017年8月25日 |
| 所属行业 | 化学原料和化学制品制造业（磷化工） |
| 市场地位 | 国内磷化工龙头，饲料级磷酸二氢钙市占率第一，磷酸铁锂前驱体新锐 |

**公司概况**：
川恒股份是国内磷化工龙头企业，形成"磷矿石开采→湿法磷酸→精细磷酸盐+新能源材料"全产业链布局。公司磷矿储量超5亿吨，自给率超70%，饲料级磷酸二氢钙国内市占率第一。2024年广西鹏越半水-二水磷酸项目投产拉动业绩高增，磷酸铁锂前驱体业务已进入宁德时代供应链，2025年规划产能扩至20万吨/年。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **饲料级磷酸二氢钙** | 饲料添加剂核心产品 | 畜禽水产养殖 |
| **磷酸一铵** | 磷肥/工业磷酸盐原料 | 农业肥料、新能源材料 |
| **磷酸铁** | 磷酸铁锂正极材料前驱体 | 动力电池、储能 |
| **磷矿石** | 磷矿采选 | 磷化工原料 |
| **精细磷酸盐** | 食品级、工业级磷酸盐 | 食品、电子、新能源 |

### 业务模式

公司采用"矿化一体"全产业链运营模式，从磷矿开采到湿法磷酸、精细磷酸盐、新能源材料全覆盖。拥有磷矿资源禀赋，原材料自给率高，通过半水湿法磷酸技术实现成本领先。磷酸铁业务客户绑定湖南裕能等龙头，已进入宁德时代供应链。

### 技术路线/核心技术

- **半水-二水湿法磷酸**：广西鹏越项目采用半水-二水磷酸工艺，单位成本较行业低15%-20%
- **磷矿采选**：2024年磷矿开采量314.84万吨，自给率超70%
- **磷石膏综合利用**：综合利用率超40%，远超行业平均20%
- **磷酸铁制备**：配套磷酸铁锂正极材料生产，产能10万吨/年

**技术优势**：
川恒股份半水湿法磷酸技术领先，单位成本较行业低15%-20%。磷石膏综合利用率超40%，环保优势突出。磷酸铁锂前驱体已实现规模化生产，2025年规划产能扩至20万吨/年。

## 四、行业地位与产业链定位

### 产业链定位

```
磷化工产业链：
磷矿石 → 湿法磷酸 → 【川恒股份 精细磷酸盐/磷酸铁】→ 饲料/肥料/动力电池
                         ↑
                    川恒股份
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 饲料级磷酸二氢钙国内市占率第一，磷酸铁产能10万吨/年 |
| **竞争优势** | 磷矿储量超5亿吨、自给率超70%，半水湿法磷酸成本领先15%-20% |
| **客户结构** | 饲料添加剂覆盖国内头部企业，磷酸铁已进入宁德时代、湖南裕能供应链 |
| **供应商** | 磷矿自给，原材料保障能力强 |

### 竞争格局

**主要竞争对手**：
- 云天化：磷肥龙头，磷化工延伸
- 兴发集团：磷化工、电子级磷酸
- 湖北宜化：磷肥、磷化工
- 湖南裕能、德方纳米：磷酸铁锂正极材料

**公司优势**：
川恒股份饲料级磷酸二氢钙市占率第一，磷矿资源禀赋突出。磷酸铁锂前驱体业务快速放量，2024年产能利用率41.29%，2025年规划扩至20万吨/年。半水湿法磷酸技术成本领先，磷石膏综合利用环保优势明显。

## 五、投资亮点

### 核心亮点

- ✅ **磷矿资源禀赋**：磷矿储量超5亿吨，自给率超70%，2024年开采量314.84万吨
- ✅ **饲料磷酸二氢钙龙头**：国内市占率第一，营收14.45亿元同比+20.16%
- ✅ **磷酸铁锂前驱体新锐**：产能10万吨/年，已进入宁德时代、湖南裕能供应链，2025年扩至20万吨
- ✅ **半水湿法磷酸技术**：单位成本较行业低15%-20%，成本优势突出
- ✅ **磷石膏综合利用**：利用率超40%，远超行业20%，环保优势明显

### 成长驱动因素

1. **磷酸铁锂前驱体放量**
   - 新能源车、储能需求带动磷酸铁锂需求
   - 2025年产能扩至20万吨/年
   - 客户绑定龙头，供应链优势巩固

2. **磷化工产业链延伸**
   - 广西鹏越项目投产拉动业绩
   - 精细磷酸盐向高附加值延伸
   - 磷矿-磷酸-磷酸铁一体化成本优势

3. **成本控制与环保**
   - 磷矿自给率高，原材料成本可控
   - 磷石膏综合利用降低环保成本
   - 资产负债率持续下降

### 催化剂

- 磷酸铁锂需求超预期
- 磷矿石价格维持高位
- 20万吨磷酸铁产能投产
- 新客户拓展进展
