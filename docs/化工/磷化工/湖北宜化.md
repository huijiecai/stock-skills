## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 湖北宜化化工股份有限公司 |
| 证券代码 | 000422 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 1996年8月15日 |
| 所属行业 | 化学原料和化学制品制造业（磷化工） |
| 市场地位 | 磷化工+氯碱龙头，磷铵、PVC、烧碱产能大 |

**公司概况**：
湖北宜化是国内磷化工+氯碱双龙头，磷铵产能约130万吨/年，PVC产能约130万吨/年，烧碱产能约80万吨/年。公司拥有磷矿资源，形成"磷矿-磷肥-PVC-烧碱"一体化布局。2024年磷肥景气度维持，PVC、烧碱价格波动影响业绩。磷化工与农业、新能源（磷酸铁）相关；氯碱与房地产、氧化铝相关。若磷肥景气持续、氯碱拐点来临，双轮驱动弹性大。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **磷铵** | 磷酸一铵、磷酸二铵，产能约130万吨/年 | 农业肥料 |
| **PVC** | 产能约130万吨/年，电石法 | 建材、管材、型材 |
| **烧碱** | 产能约80万吨/年 | 氧化铝、造纸、纺织 |
| **尿素** | 氮肥 | 农业肥料 |
| **磷酸** | 湿法磷酸 | 磷肥、磷酸铁 |
| **磷矿** | 自有磷矿 | 磷化工原料 |

### 业务模式

公司采用"磷化工+氯碱"双轮驱动模式，磷矿-磷肥-PVC-烧碱一体化。磷肥与农业需求相关；PVC与房地产、建材相关；烧碱与氧化铝相关。拥有磷矿资源，成本优势明显。

### 技术路线/核心技术

- **磷矿采选**：自有磷矿，原料保障
- **湿法磷酸**：磷肥核心工艺
- **电石法PVC**：煤炭-电石-PVC
- **离子膜烧碱**：氯碱工艺
- **矿化一体**：磷矿-磷肥一体化

**技术优势**：
磷矿资源禀赋，磷肥成本优势。磷化工+氯碱双轮，产品多元。一体化布局，成本可控。

## 四、行业地位与产业链定位

### 产业链定位

```
磷化工+氯碱产业链：
磷矿 → 【湖北宜化 磷铵/磷酸】→ 农业/磷酸铁
煤炭+原盐 → 【湖北宜化 PVC/烧碱】→ 建材/氧化铝
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 磷铵约130万吨、PVC约130万吨、烧碱约80万吨，行业前列 |
| **竞争优势** | 磷矿资源、双轮驱动、一体化 |
| **客户结构** | 农业、建材、氧化铝企业 |
| **供应商** | 磷矿自给，煤炭、原盐外采 |

### 竞争格局

**主要竞争对手**：
- 磷肥：云天化、兴发集团、新洋丰
- PVC/烧碱：中泰化学、氯碱化工

**公司优势**：
磷化工+氯碱双龙头，磷矿资源禀赋。双轮驱动，分散单一周期风险。一体化成本优势。

## 五、投资亮点

### 核心亮点

- ✅ **磷化工+氯碱双龙头**：磷铵、PVC、烧碱产能大
- ✅ **拐点逻辑**：磷肥景气若持续，磷化工受益；PVC、烧碱若拐点来临，氯碱弹性大；磷酸铁若需求复苏，磷酸延伸受益
- ✅ **磷矿资源**：自有磷矿，成本优势
- ✅ **双轮驱动**：磷化工+氯碱，分散风险
- ✅ **磷酸铁**：磷酸可延伸至磷酸铁，新能源布局

### 成长驱动因素

1. **磷肥景气**
   - 磷肥与农业需求、粮食安全相关
   - 磷矿资源稀缺，磷肥景气若持续
   - 磷矿-磷肥一体化受益

2. **氯碱拐点**
   - PVC与房地产、建材相关
   - 烧碱与氧化铝相关
   - 若拐点来临，双产品弹性大

3. **磷酸铁**
   - 磷酸可制磷酸铁
   - 磷酸铁锂需求若复苏
   - 新能源延伸

### 催化剂

- **磷肥价格**：磷肥景气持续
- **PVC/烧碱价格**：触底反弹
- **房地产**：PVC需求
- **磷酸铁锂**：磷酸铁需求
- **磷矿**：磷矿价格、供给

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **磷肥景气** | 磷矿稀缺，磷肥景气若持续 | 粮食安全、磷矿供给 |
| **氯碱拐点** | PVC、烧碱若拐点来临 | 房地产、氧化铝 |
| **磷酸铁拐点** | 磷酸延伸，若磷酸铁锂复苏 | 新能源需求 |
| **业绩拐点** | 双轮驱动，若同时改善弹性大 | 磷肥+氯碱价涨 |
