## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 湖北兴发化工集团股份有限公司 |
| 证券代码 | 600141 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1994年 |
| 上市时间 | 1999年6月16日 |
| 所属行业 | 化学原料和化学制品制造业（磷化工） |
| 市场地位 | 电子级磷酸市占率领先，磷矿资源丰富，磷酸铁锂一体化布局 |

**公司概况**：
兴发集团是磷化工龙头企业，拥有3.95亿吨磷矿资源储量，设计产能585万吨/年。子公司兴福电子电子级磷酸产能3万吨/年，在国内市场占有较高份额，客户包括中芯国际、华虹宏力、SK海力士、格罗方德等国内外知名半导体企业。公司形成磷矿采选、磷化工、精细化工、新材料一体化布局，持续推进磷酸铁锂等新能源材料拓展。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **电子级磷酸** | 超纯磷酸，半导体级 | 集成电路、显示面板、太阳能 |
| **电子级硫酸/氨水** | 电子化学品 | 半导体、显示、光伏 |
| **磷肥** | 磷酸一铵、二铵等 | 农业肥料 |
| **农药** | 草甘膦等 | 农用除草剂 |
| **有机硅** | 有机硅单体及下游 | 建筑、电子、新能源 |
| **磷矿石** | 磷矿采选 | 磷化工原料 |

### 业务模式

公司采用"矿化一体"模式，从磷矿开采到磷肥、精细化工、电子化学品全覆盖。2024年多项新材料项目陆续投产：兴福电子2万吨/年电子级氨水、宜都兴发湿法磷酸精制、湖北兴瑞40万吨/年有机硅（一期）等。电子级磷酸等高端电子化学品取得突破，供应国内外高端客户。

### 技术路线/核心技术

- **电子级磷酸**：3万吨/年产能，达国际先进水平
- **湿法磷酸精制**：宜都兴发项目建成，提升磷酸品质
- **磷矿选矿**：瓦屋100万吨光电选矿、后坪200万吨选矿已投运
- **有机硅**：40万吨/年项目一期投产

**技术优势**：
兴福电子电子级磷酸、硫酸等产品已达国际先进水平，积累中芯国际、华虹宏力、SK海力士、格罗方德等国内外知名半导体客户，在国内电子级磷酸市场占有较高市场份额。

## 四、行业地位与产业链定位

### 产业链定位

```
磷化工产业链：
磷矿石 → 湿法磷酸 → 【兴发集团 电子级磷酸/磷肥/农药】→ 半导体/农业/新能源
                          ↑
                     兴发集团
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 电子级磷酸国内市场份额领先，产能3万吨/年居同行前列 |
| **竞争优势** | 磷矿资源3.95亿吨、产能585万吨/年，电子化学品技术领先 |
| **客户结构** | 中芯国际、华虹宏力、SK海力士、格罗方德等半导体龙头 |
| **供应商** | 磷矿自给，原材料保障能力强 |

### 竞争格局

**主要竞争对手**：
- 云天化：磷肥龙头，磷矿储量约8亿吨
- 川恒股份：磷化工、磷酸铁锂
- 川发龙蟒：磷化工
- 其他电子化学品企业

**公司优势**：
兴发集团电子级磷酸技术达国际先进水平，客户覆盖国内外半导体龙头。磷矿资源丰富，矿山采选毛利率74.28%，成本优势突出。特种化学品、农药、肥料多业务协同，抗风险能力强。

## 五、投资亮点

### 核心亮点

- ✅ **电子级磷酸领先**：产能3万吨/年，国内市场份额领先，客户覆盖中芯、华虹、海力士等
- ✅ **磷矿资源丰富**：3.95亿吨储量，585万吨/年产能，矿山毛利率74.28%
- ✅ **磷酸铁锂一体化**：布局新能源材料，湿法磷酸精制等项目支撑
- ✅ **多业务协同**：磷矿、磷肥、农药、有机硅、电子化学品
- ✅ **高分红**：拟每10股派10元，分红比例68.89%

### 成长驱动因素

1. **电子化学品放量**
   - 半导体国产化带动电子级磷酸、硫酸、氨水需求
   - 兴福电子2万吨电子级氨水投产
   - 客户拓展、份额提升

2. **磷矿高景气**
   - 磷矿石价格高位运行
   - 矿山采选营收+65%，毛利率74.28%
   - 瓦屋、后坪选矿项目投运

3. **磷酸铁锂布局**
   - 湿法磷酸精制项目建成
   - 新能源材料一体化延伸
   - 宜都兴发等项目推进

### 催化剂

- 半导体国产化加速
- 磷矿石价格维持高位
- 磷酸铁锂项目投产
- 有机硅产能释放

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **磷矿价格波动** | 磷矿石价格若回落，矿山业务盈利承压 |
| ⚠️ **半导体周期** | 电子化学品需求受半导体景气度影响 |
| ⚠️ **农药景气** | 草甘膦等农药价格波动 |
| ⚠️ **环保压力** | 磷化工环保要求趋严，环保投入增加 |
| ⚠️ **磷酸铁锂竞争** | 新能源材料竞争激烈，盈利存在不确定性 |
