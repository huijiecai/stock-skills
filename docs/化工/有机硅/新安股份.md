## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江新安化工集团股份有限公司 |
| 证券代码 | 600596 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1965年 |
| 上市时间 | 2001年 |
| 所属行业 | 化学原料和化学制品制造业 |
| 市场地位 | 有机硅+草甘膦双龙头 |

**公司概况**：
新安股份是草甘膦和有机硅行业双龙头企业，拥有80余家控股子公司，产品畅销全球130多个国家和地区。公司形成有机硅+草甘膦双主业格局，有机硅单体权益产能40.2万吨/年，草甘膦原药产能8万吨，已形成3000余个有机硅终端产品，转化率超45%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **草甘膦** | 8万吨原药产能，国内市占率10.8% | 除草剂、农业 |
| **有机硅单体** | 50万吨/年（权益40.2万吨） | 有机硅深加工原料 |
| **有机硅深加工** | 硅橡胶、硅油、密封胶等3000+产品 | 建筑、汽车、电子、日化 |
| **制剂** | 草甘膦制剂等 | 农业植保 |
| **特种化学品** | 磷系、硅系特种产品 | 高端应用 |

### 业务模式

公司采用"原药+制剂""单体+深加工"双轮驱动模式。草甘膦业务覆盖原药和制剂，2024年上半年制剂销量同比+52%，产量创历史新高。有机硅业务从单体到终端产品全覆盖，终端转化率超45%，有效抵御行业周期波动。海外业务加速突破，出口创利同比+20%。

### 技术路线/核心技术

- **草甘膦工艺**：甘氨酸法、IDA法双工艺路线
- **有机硅一体化**：单体-中间体-深加工完整链条
- **终端转化**：3000余个终端产品，转化率超45%
- **氯硅烷循环**：有机硅副产盐酸用于草甘膦，循环经济

**技术优势**：
新安股份有机硅与草甘膦形成产业链协同，有机硅副产盐酸可用于草甘膦生产，实现资源循环。终端产品丰富，抗周期能力强。2025年有机硅行业有望景气改善，价格有望上行。

## 四、行业地位与产业链定位

### 产业链定位

```
有机硅+农化产业链：
硅石/黄磷 → 有机硅单体/草甘膦原药 → 【新安股份 深加工/制剂】→ 建筑/汽车/农业
                        ↑
                  新安股份（双主业）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 草甘膦国内市占率10.8%前三，有机硅产能占比7.9%前五 |
| **竞争优势** | 双主业协同、终端转化率高、海外布局 |
| **客户结构** | 覆盖农业、建筑、汽车、电子等下游 |
| **供应商** | 原料部分自给，产业链协同 |

### 竞争格局

**主要竞争对手**：
- 合盛硅业：工业硅+有机硅全球龙头
- 兴发集团：草甘膦、有机硅
- 江山股份：草甘膦
- 东岳硅材：有机硅

**公司优势**：
新安股份双主业分散风险，有机硅与草甘膦产业链协同。终端产品转化率超45%，抗周期能力强。海外业务加速突破，出口创利增长。机构预计2025年有机硅景气改善，公司有望受益。

## 五、投资亮点

### 核心亮点

- ✅ **双龙头**：草甘膦+有机硅双主业，产业链协同
- ✅ **终端转化**：3000+终端产品，转化率超45%，抗周期
- ✅ **海外突破**：出口创利同比+20%，全球化布局
- ✅ **草甘膦景气**：2024年制剂销量+52%，产量创历史新高
- ✅ **景气改善预期**：2025年有机硅行业有望景气改善

### 成长驱动因素

1. **有机硅景气改善**
   - 2025年行业有望景气改善，价格有望上行
   - 机构给予2026年20倍PE，目标价9.65元/股

2. **终端业务成长**
   - 终端产品持续拓展
   - 高附加值产品占比提升

3. **海外扩张**
   - 出口创利持续增长
   - 全球化布局深化

### 催化剂

- 有机硅价格企稳回升
- 草甘膦需求稳健
- 终端产品放量
- 海外业务持续突破
