## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 合盛硅业股份有限公司 |
| 证券代码 | 603260 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2005年 |
| 上市时间 | 2017年10月30日 |
| 所属行业 | 化学原料和化学制品制造业 |
| 市场地位 | 工业硅+有机硅全球龙头，煤电硅一体化 |

**公司概况**：
合盛硅业是全球工业硅和有机硅双龙头，2010年于石河子园区首创"煤电硅"一体化绿色循环经济模式，引领行业发展。公司拥有从工业硅到有机硅单体及下游深加工产品的完整产业链，在新疆、云南等地布局生产基地，煤电硅一体化成本优势显著。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **工业硅** | 多晶硅、有机硅原料 | 光伏、半导体、有机硅 |
| **有机硅单体** | 甲基氯硅烷等 | 有机硅深加工原料 |
| **硅橡胶** | 高温胶、室温胶 | 建筑、汽车、电子、新能源 |
| **硅油** | 各类硅油产品 | 日化、纺织、电子 |
| **环体硅氧烷** | DMC、D4等 | 有机硅中间体 |

### 业务模式

公司采用"煤电硅"一体化模式，自备电厂降低电力成本，原料、能源自给率高。从工业硅到有机硅单体再到硅橡胶、硅油等深加工产品，产业链完整。新疆石河子、云南昭通等基地依托当地煤炭、水电资源，成本优势突出。

### 技术路线/核心技术

- **煤电硅一体化**：2010年首创，自备电厂+工业硅+有机硅，成本领先
- **绿电硅项目**：2019年启动昭通绿电硅项目，顺应低碳趋势
- **全产业链**：工业硅→有机硅单体→硅橡胶/硅油，一体化布局
- **规模效应**：工业硅、有机硅产能全球领先

**技术优势**：
合盛硅业"煤电硅"一体化模式为行业首创，电力成本占工业硅成本约30-40%，自备电厂显著降低成本。公司持续扩产，昭通、鄯善等项目投产后产能将进一步扩大。

## 四、行业地位与产业链定位

### 产业链定位

```
有机硅产业链：
硅石/煤炭 → 工业硅 → 【合盛硅业 有机硅单体】→ 硅橡胶/硅油 → 建筑/汽车/电子/新能源
                    ↑
              合盛硅业（煤电硅一体化）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 工业硅、有机硅产能全球第一，双龙头地位 |
| **竞争优势** | 煤电硅一体化成本优势、规模效应、全产业链 |
| **客户结构** | 覆盖光伏、有机硅深加工、出口等 |
| **供应商** | 硅石、煤炭自给或就近采购，电力自备 |

### 竞争格局

**主要竞争对手**：
- 新安股份：有机硅+草甘膦双主业
- 东岳硅材：有机硅单体及深加工
- 兴发集团：有机硅、磷化工
- 通威股份、协鑫等：工业硅（多用于多晶硅）

**公司优势**：
合盛硅业凭借工业硅+有机硅双龙头、煤电硅一体化成本优势、新疆云南资源禀赋，在行业中具备显著竞争力。工业硅处于历史底部区间，下行空间有限，周期复苏时业绩弹性大。

## 五、投资亮点

### 核心亮点

- ✅ **双龙头地位**：工业硅+有机硅产能全球第一，定价权强
- ✅ **煤电硅一体化**：行业首创模式，电力成本优势显著
- ✅ **全产业链**：从工业硅到硅橡胶、硅油全覆盖
- ✅ **资源禀赋**：新疆、云南煤炭、水电资源，成本优势
- ✅ **周期底部**：工业硅价格处于历史底部，复苏弹性大

### 成长驱动因素

1. **行业周期复苏**
   - 工业硅价格处于历史底部，下行空间有限
   - 光伏、新能源需求长期支撑工业硅
   - 有机硅景气有望改善

2. **新产能扩张**
   - 昭通绿电硅、鄯善等项目
   - 煤电硅一体化二三期项目（拟投超76亿元）

3. **下游需求**
   - 光伏多晶硅对工业硅需求
   - 新能源、电子对有机硅需求
   - 建筑、汽车密封胶等传统需求

### 催化剂

- 工业硅、有机硅价格企稳回升
- 光伏装机超预期带动工业硅需求
- 新产能投产贡献增量
- 原材料成本回落
