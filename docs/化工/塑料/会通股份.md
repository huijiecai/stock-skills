## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 会通新材料股份有限公司 |
| 证券代码 | 688219 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2008年 |
| 上市时间 | 2020年11月18日 |
| 所属行业 | 橡胶和塑料制品业（改性塑料/特种工程塑料） |
| 市场地位 | 改性塑料第二梯队，特种工程塑料布局 |

**公司概况**：
会通股份是国内改性塑料行业第二梯队企业，产量在10-100万吨区间。产品涵盖改性塑料、特种工程塑料，应用于家电、汽车、电子、新能源等领域。公司聚焦高分子材料改性技术，与美的、格力等家电龙头合作，汽车、电子等应用拓展中。科创板上市，研发投入较高。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **改性塑料** | PP、ABS、PC、PA等改性 | 家电、汽车、电子 |
| **特种工程塑料** | 高温、高强等特种材料 | 电子、汽车、新能源 |
| **生物降解材料** | 可降解塑料 | 包装、环保 |
| **其他** | 复合材料等 | 多元化 |

### 业务模式

公司采用大客户战略，家电客户占比高。改性塑料为主业，特种工程塑料、生物降解材料等新业务拓展。研发驱动，科创板定位新材料。

### 技术路线/核心技术

- **改性技术**：填充、增强、阻燃、合金化、生物降解
- **特种工程塑料**：高温、高强、耐化学
- **研发投入**：科创板，研发占比高

**技术优势**：
改性塑料技术积累深厚，家电客户粘性强。特种工程塑料、生物降解等新方向布局。

                              ↑
                    会通股份（第二梯队）
