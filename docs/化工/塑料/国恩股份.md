## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 青岛国恩科技股份有限公司 |
| 证券代码 | 002768 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2000年 |
| 上市时间 | 2015年6月30日 |
| 所属行业 | 橡胶和塑料制品业（改性塑料） |
| 市场地位 | 改性塑料第二龙头，产量超100万吨 |

**公司概况**：
国恩股份是国内改性塑料行业第二梯队龙头，产量超100万吨，与金发科技同属第一梯队。产品应用于家电、汽车、电子、新能源等领域。2024年前三季度营收141.60亿元，同比+11.79%，规模优势突出。改性塑料行业集中度提升，龙头受益。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **改性塑料** | PP、ABS、PC等改性材料 | 家电、汽车、电子 |
| **复合材料** | 玻纤增强、碳纤维增强 | 汽车、新能源 |
| **人造草坪** | 体育、景观用人造草坪 | 体育场馆、市政 |
| **其他** | 光显材料等 | 多元化布局 |

### 业务模式

公司采用大客户战略，与家电、汽车龙头深度合作。改性塑料为主业，人造草坪等多元化业务贡献增量。规模采购、一体化生产降本增效。

### 技术路线/核心技术

- **改性技术**：填充、增强、阻燃、合金化
- **复合材料**：玻纤、碳纤维增强
- **一体化**：从原料到成品一体化

**技术优势**：
改性塑料规模国内第二，家电、汽车客户粘性强。产量超100万吨，规模效应显著。

## 四、行业地位与产业链定位

### 产业链定位

```
改性塑料产业链：
石化原料 → 【国恩股份 改性塑料】→ 家电/汽车/电子 → 终端产品
                    ↑
              国恩股份（第二龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 产量超100万吨，行业第二 |
| **竞争优势** | 规模、客户、一体化 |
| **客户结构** | 家电、汽车龙头 |
| **供应商** | 石化原料，规模采购 |

### 竞争格局

**主要竞争对手**：
- 金发科技：行业第一
- 普利特：汽车改性塑料
- 道恩股份：弹性体
- 会通股份：特种工程塑料

**公司优势**：
产量与金发科技同属第一梯队，规模优势突出。家电、汽车客户稳定，行业集中度提升受益。

## 五、投资亮点

### 核心亮点

- ✅ **改性塑料第二龙头**：产量超100万吨，规模优势
- ✅ **拐点逻辑**：①**行业集中度提升**——改性塑料龙头份额提升；②**下游需求**——家电、汽车、新能源需求支撑；③**规模效应**——百万吨级产能，成本优势；④**多元化**——人造草坪等业务分散风险
- ✅ **营收稳健**：2024年前三季度营收+11.79%
- ✅ **客户粘性**：家电、汽车龙头深度合作

### 成长驱动因素

1. **行业集中度**
   - 改性塑料龙头份额提升
   - 中小产能出清
   - 规模效应强化

2. **下游需求**
   - 家电更新换代
   - 汽车轻量化
   - 新能源配套

3. **多元化**
   - 人造草坪等新业务
   - 光显材料等
   - 分散单一领域风险

### 催化剂

- 家电、汽车需求回暖
- 行业集中度持续提升
- 新业务放量
- 原材料价格回落
