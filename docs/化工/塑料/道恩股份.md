## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 山东道恩高分子材料股份有限公司 |
| 证券代码 | 002838 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2002年 |
| 上市时间 | 2017年1月6日 |
| 所属行业 | 橡胶和塑料制品业（弹性体/改性塑料） |
| 市场地位 | 热塑性弹性体（TPV）龙头，改性塑料第二梯队 |

**公司概况**：
道恩股份是国内热塑性弹性体（TPV）龙头，产品涵盖TPV、TPE、改性塑料等，应用于汽车、家电、医疗、建筑等领域。2024年前三季度营收37.69亿元同比+12.05%；2025年三季度单季营收15.74亿元同比+8.84%，净利润同比大增48.19%，年初至三季度营收44.55亿元同比+18.23%。弹性体+改性塑料双轮驱动，业绩高增。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **热塑性弹性体（TPV）** | 动态硫化弹性体，核心产品 | 汽车密封条、建筑密封 |
| **热塑性聚氨酯（TPU）** | 聚氨酯弹性体 | 鞋材、管材、汽车 |
| **改性塑料** | PP、ABS等改性 | 汽车、家电 |
| **熔喷料** | 熔喷布原料（防疫相关） | 口罩等 |

### 业务模式

公司采用研产销一体化模式，TPV为核心产品，国内龙头。汽车密封条、建筑密封等应用需求稳健。改性塑料与弹性体协同，客户重叠度高。

### 技术路线/核心技术

- **TPV**：动态硫化技术，橡胶与塑料共混
- **TPU**：聚氨酯弹性体合成
- **改性塑料**：填充、增强、合金化

**技术优势**：
TPV国内龙头，汽车密封条等高端应用认证壁垒高。弹性体与改性塑料协同，产品线丰富。
