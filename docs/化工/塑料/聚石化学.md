## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广东聚石化学股份有限公司 |
| 证券代码 | 688669 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2007年 |
| 上市时间 | 2021年1月25日 |
| 所属行业 | 橡胶和塑料制品业（改性塑料） |
| 市场地位 | 改性塑料企业，阻燃材料、无卤环保材料特色 |

**公司概况**：
聚石化学是国内改性塑料行业上市公司，产品以阻燃材料、无卤环保材料为特色，应用于电子电器、家电、汽车、照明等领域。公司聚焦环保阻燃、无卤化趋势，与电子电器、家电龙头合作。科创板上市，研发与环保属性突出。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **阻燃改性塑料** | 阻燃PP、阻燃ABS、阻燃PC等 | 电子电器、家电、照明 |
| **无卤环保材料** | 无卤阻燃材料，环保合规 | 电子、家电、汽车 |
| **改性塑料** | 通用改性PP、ABS等 | 家电、汽车 |
| **其他** | 光学材料、导热材料等 | 多元化 |

### 业务模式

公司采用研产销一体化模式，阻燃、无卤材料为核心特色。电子电器、家电客户为主，汽车、照明等拓展。环保政策驱动无卤化趋势，公司受益。

### 技术路线/核心技术

- **阻燃技术**：溴系、磷系、无机阻燃，无卤化
- **环保材料**：无卤阻燃、可回收
- **改性技术**：填充、增强、合金化

**技术优势**：
阻燃材料技术积累深，无卤环保材料符合欧盟RoHS等法规。电子电器、家电客户认证壁垒高。

## 四、行业地位与产业链定位

### 产业链定位

```
改性塑料产业链：
石化原料+阻燃剂 → 【聚石化学 阻燃/无卤改性塑料】→ 电子电器/家电/汽车 → 终端产品
                              ↑
                    聚石化学（阻燃特色）
```

### 市场地位

- 会通股份：特种工程塑料

**公司优势**：
阻燃、无卤材料特色鲜明，环保政策驱动无卤化趋势。电子电器、家电客户认证壁垒高。

## 五、投资亮点

### 核心亮点

- ✅ **阻燃材料特色**：阻燃PP、ABS、PC等，无卤环保
- ✅ **拐点逻辑**：①**环保政策**——无卤阻燃材料需求增长，欧盟RoHS等法规驱动；②**电子电器**——电子电器阻燃要求高，客户粘性强；③**国产替代**——高端阻燃材料国产化；④**新能源**——新能源车、储能等阻燃需求
- ✅ **无卤化趋势**：环保合规，无卤材料替代溴系
- ✅ **科创板**：研发、资本优势

### 成长驱动因素

1. **环保政策**
   - 无卤阻燃材料需求
   - 欧盟RoHS、REACH等
   - 国内环保标准提升

2. **电子电器**
   - 电子电器阻燃要求
   - 家电能效、安全
   - 照明、汽车电子

3. **新能源**
   - 新能源车阻燃材料
   - 储能电池包阻燃
   - 充电桩等

### 催化剂

- 无卤化政策推进
- 电子电器需求回暖
- 新能源车、储能放量
- 原材料价格回落
