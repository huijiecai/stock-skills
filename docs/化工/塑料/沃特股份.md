## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 深圳市沃特新材料股份有限公司 |
| 证券代码 | 002886 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001年 |
| 上市时间 | 2017年6月27日 |
| 所属行业 | 橡胶和塑料制品业（特种工程塑料） |
| 市场地位 | 特种高分子材料领军企业，LCP/PEEK国产替代 |

**公司概况**：
沃特股份是中国特种高分子材料领军企业，专注于LCP（液晶高分子）、PEEK（聚醚醚酮）等特种工程塑料。2024年营收19.0亿元同比+23.5%，归母净利润3660万元同比+520.69%。特种材料营收9.22亿元，占近半营收，驱动业绩腾飞。5G天线材料、人形机器人关节等领域实现国产替代，研发费用突破1亿元达1.16亿元，技术驱动成长。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **LCP（液晶高分子）** | 高频低介电，5G天线材料 | 5G通信、电子连接器 |
| **PEEK（聚醚醚酮）** | 耐高温、高强，特种塑料 | 人形机器人关节、医疗、汽车 |
| **特种尼龙** | 高温尼龙、长链尼龙 | 汽车、电子 |
| **其他特种材料** | 聚砜、PPS等 | 多元化应用 |

### 业务模式

公司采用研发驱动模式，LCP、PEEK等特种材料技术壁垒高。5G天线、人形机器人等新兴应用放量，国产替代逻辑强化。研发投入持续加大，2024年达1.16亿元。

### 技术路线/核心技术

- **LCP**：液晶高分子合成与改性，5G天线、连接器
- **PEEK**：聚醚醚酮，人形机器人关节、医疗植入
- **特种尼龙**：高温、长链尼龙
- **研发**：研发费用1.16亿元，技术领先

**技术优势**：
LCP、PEEK等特种工程塑料国产替代领军，5G、人形机器人等新兴应用突破。研发投入高，技术壁垒强。
