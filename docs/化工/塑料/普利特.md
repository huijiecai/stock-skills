## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海普利特复合材料股份有限公司 |
| 证券代码 | 002324 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 2009年12月18日 |
| 所属行业 | 橡胶和塑料制品业（改性塑料） |
| 市场地位 | 汽车改性塑料龙头，储能/钠电新业务放量 |

**公司概况**：
普利特是国内汽车改性塑料龙头，产品应用于汽车内外饰、电子电器等。2024年前三季度营收57.38亿元同比-11.69%，但净利润预计3.21-3.51亿元，同比+53.48%-67.82%。核心驱动是储能电池与钠离子电池订单大幅增加，公司投建年产6GWh钠离子电池生产基地，新能源业务成为新增长极。改性塑料+储能/钠电双轮驱动。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **改性塑料** | 汽车用PP、ABS、PC等 | 汽车内外饰、电子电器 |
| **储能电池** | 储能系统、电池模组 | 储能电站、工商业储能 |
| **钠离子电池** | 钠离子电池及系统 | 储能、两轮车、A00级车 |
| **LCP材料** | 液晶高分子，5G等 | 电子连接器、天线 |

### 业务模式

公司采用"改性塑料+新能源"双轮驱动。改性塑料深耕汽车客户，储能/钠电业务通过订单放量快速成长。年产6GWh钠离子电池基地建设推进，新能源产能扩张。

### 技术路线/核心技术

- **汽车改性塑料**：汽车内外饰材料，客户覆盖主流车企
- **储能系统**：储能电池模组、系统集成
- **钠离子电池**：钠电正负极、电芯、系统
- **LCP**：5G天线、连接器材料

**技术优势**：
汽车改性塑料先发优势明显，客户粘性强。储能/钠电订单大幅增加，新能源转型成效显著。

## 四、行业地位与产业链定位

### 产业链定位

```
普利特双轮驱动：
石化原料 → 【普利特 改性塑料】→ 汽车 → 整车
                    +
锂/钠原料 → 【普利特 储能/钠电】→ 储能电站/两轮车
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 汽车改性塑料龙头，产量10-100万吨第二梯队 |
| **竞争优势** | 汽车客户、储能/钠电新业务 |
| **客户结构** | 汽车主机厂、储能客户 |
| **新业务** | 储能、钠电订单放量 |

### 竞争格局

**主要竞争对手**：
- 金发科技：改性塑料龙头
- 国恩股份：改性塑料第二
- 道恩股份：弹性体
- 储能/钠电：宁德时代、比亚迪、传艺科技等

**公司优势**：
汽车改性塑料龙头+储能/钠电新业务放量，双轮驱动。钠电布局前瞻，6GWh产能建设打开空间。

## 五、投资亮点

### 核心亮点

- ✅ **汽车改性塑料龙头**：先发优势，客户粘性强
- ✅ **拐点逻辑**：①**新能源需求**——储能、钠离子电池订单大幅增加，利润拐点；②**钠电布局**——年产6GWh钠离子电池基地，钠电产业化先行；③**利润拐点**——2024年前三季度净利润+53%-68%，增收不增利转为增利；④**汽车修复**——若汽车需求回暖，改性塑料营收修复
- ✅ **储能/钠电放量**：订单大幅增加，新增长极
- ✅ **双轮驱动**：改性塑料+新能源

### 成长驱动因素

1. **新能源需求**
   - 储能电站、工商业储能需求
   - 钠离子电池产业化
   - 两轮车、A00级车钠电应用
   - 6GWh钠电产能释放

2. **汽车修复**
   - 汽车销量回暖
   - 改性塑料需求恢复
   - 汽车轻量化趋势

3. **LCP等新材料**
   - 5G、电子连接器
   - 高附加值产品

### 催化剂

- 储能/钠电订单持续放量
- 6GWh钠电产能投产
- 汽车销量回暖
- 钠电政策支持

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 普利特拐点取决于：①**新能源需求**——储能、钠离子电池订单大幅增加，2024年前三季度净利润+53%-68%已确认利润拐点；②**钠电布局**——6GWh钠离子电池基地建设，钠电产业化先行；③**汽车修复**——若汽车需求回暖，改性塑料营收修复；④**利润结构**——储能/钠电高毛利贡献，利润增速高于营收。**拐点信号**：储能/钠电营收占比突破20%、6GWh钠电产能投产、单季营收同比转正。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 储能/钠电放量、汽车回暖、6GWh投产 | 净利润增速30%+ | 净利润增速25%+ |
| **中性** | 储能稳健、汽车企稳、钠电推进 | 净利润增速15-25% | 净利润增速15-20% |
| **悲观** | 汽车疲软、储能竞争加剧 | 净利润增速<15% | 净利润增速<15% |
