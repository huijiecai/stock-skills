## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 金发科技股份有限公司 |
| 证券代码 | 600143 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 2004年6月23日 |
| 所属行业 | 橡胶和塑料制品业（改性塑料） |
| 市场地位 | 改性塑料龙头，产量超100万吨行业第一 |

**公司概况**：
金发科技是国内改性塑料绝对龙头，产品涵盖改性塑料、特种工程塑料、完全生物降解塑料、环保高性能再生塑料等。2024年预计归母净利润8.00-9.50亿元，同比+152.58%-199.94%；扣非净利润7.71-9.21亿元，同比+288.43%-364.00%。改性塑料及特种工程塑料在汽车、家电、电子电工、消费电子等领域的销量和毛利均实现稳定增长，绿色石化板块亏损收窄，新材料布局持续推进。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **改性塑料** | PP、ABS、PC等改性，营收占比52%+ | 汽车、家电、电子电工 |
| **特种工程塑料** | LCP、PPSU、PES等 | 电子、汽车、医疗 |
| **完全生物降解塑料** | PLA、PBAT等 | 包装、农膜 |
| **环保再生塑料** | 再生PP、再生ABS等 | 循环经济 |
| **绿色石化** | 专用PP、ABS微改性，营收占比19% | 与改性塑料协同 |

### 业务模式

公司采用研产销一体化模式，改性塑料与绿色石化协同，微改性专用料售价高于纯树脂。海外基地（美国、欧洲、印度、马来西亚）2024年上半年产成品销量12.09万吨同比+28.55%，越南、墨西哥、波兰基地建设推进中。

### 技术路线/核心技术

- **改性技术**：填充、增强、阻燃、合金化
- **特种工程塑料**：LCP、PPSU、PES合成树脂
- **新材料**：连续碳纤维增强复合材料（机器人领域）

**技术优势**：
年产0.6万吨PPSU/PES装置2024年6月试车成功；年产1.5万吨LCP项目2024年四季度逐步投产。改性塑料技术积累深厚，汽车、家电客户粘性强。

## 四、行业地位与产业链定位

### 产业链定位

```
改性塑料产业链：
石化原料（PP/ABS/PC等）→ 【金发科技 改性塑料】→ 汽车/家电/电子 → 终端产品
                              ↑
                    金发科技（绝对龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 改性塑料产量超100万吨，行业第一 |
| **竞争优势** | 规模、技术、客户、海外布局 |
| **客户结构** | 汽车、家电、电子电工、消费电子龙头 |
| **供应商** | 石化原料采购，规模优势 |

### 竞争格局

**主要竞争对手**：
- 国恩股份：改性塑料第二梯队，产量超100万吨
- 普利特：汽车改性塑料
- 道恩股份：弹性体
- 会通股份：特种工程塑料

**公司优势**：
改性塑料绝对龙头，规模、技术、渠道全面领先。绿色石化协同、新材料布局、海外扩张打开成长空间。

## 五、投资亮点

### 核心亮点

- ✅ **改性塑料龙头**：产量超100万吨，行业第一
- ✅ **拐点逻辑**：①**业绩拐点**——2024年净利润同比+152%-200%，业绩反转确认；②**绿色石化修复**——亏损收窄，协同效应显现；③**新材料放量**——PPSU/PES、LCP投产，高附加值产品贡献；④**海外扩张**——海外销量+28.55%，越南、墨西哥、波兰布局
- ✅ **新材料布局**：LCP、PPSU/PES、碳纤维复合材料
- ✅ **海外高增**：海外基地销量+28.55%

### 成长驱动因素

1. **业绩修复**
   - 改性塑料销量、毛利双增
   - 绿色石化亏损收窄
   - 原材料价格企稳

2. **新材料**
   - LCP、PPSU/PES等高附加值产品
   - 机器人用碳纤维复合材料
   - 特种工程塑料国产替代

3. **海外扩张**
   - 美国、欧洲、印度、马来西亚基地
   - 越南、墨西哥、波兰新建
   - 全球化产能布局

### 催化剂

- 绿色石化扭亏
- LCP、PPSU/PES放量
- 海外基地产能释放
- 汽车、家电需求回暖

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 金发科技拐点取决于：①**业绩拐点**——2024年净利润同比+152%-200%，业绩反转已确认；②**绿色石化修复**——亏损收窄，若扭亏将显著增厚业绩；③**新材料放量**——PPSU/PES、LCP等高附加值产品投产贡献；④**海外扩张**——海外销量+28.55%，全球化布局打开空间。**拐点信号**：绿色石化单季扭亏、LCP营收占比突破5%、海外收入增速持续20%+。
