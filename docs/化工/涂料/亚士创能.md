## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 亚士创能科技（上海）股份有限公司 |
| 证券代码 | 603378 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2009年 |
| 上市时间 | 2017年9月28日 |
| 所属行业 | 化学原料和化学制品制造业（建筑涂料） |
| 市场地位 | 建筑涂料、保温装饰一体化板，工程渠道为主 |

**公司概况**：
亚士创能主营建筑涂料、保温装饰一体化板、保温材料等，产品应用于建筑外墙、保温节能。2024年营收约31.10亿元（行业第五），但2025年前三季度业绩极度承压：营收3.97亿元、同比-76.97%，归母净利润亏损3.11亿元。防水板块受冲击严重，防水卷材和防水涂料合计同比-86.66%。公司以工程渠道为主，地产下行冲击大。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **建筑涂料** | 外墙涂料、内墙涂料 | 建筑外墙、室内装修 |
| **保温装饰一体化板** | 保温+装饰一体 | 建筑外墙节能 |
| **保温材料** | 保温板、保温砂浆 | 建筑保温 |
| **防水材料** | 防水卷材、防水涂料 | 建筑防水 |

### 业务模式

公司采用"工程渠道为主"模式，客户以房企、工装项目为主。地产下行导致工程订单大幅下滑，营收和利润承压。防水业务受冲击尤为严重。

### 技术路线/核心技术

- **保温装饰一体化**：保温与装饰结合，施工便捷
- **建筑涂料**：外墙涂料、真石漆等
- **防水材料**：防水卷材、涂料

**技术优势**：
亚士创能保温装饰一体化板具有差异化，建筑涂料产品线完整。但工程渠道占比高，地产周期敏感性强。

## 四、行业地位与产业链定位

### 产业链定位

```
建筑涂料产业链：
树脂/钛白粉/保温材料 → 【亚士创能 涂料/保温/防水】→ 房地产 → 建筑
                              ↑
                        亚士创能（工程为主）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 2024年营收约31亿，行业第五 |
| **竞争优势** | 保温装饰一体化板差异化、产品线完整 |
| **客户结构** | 房企、工装项目，工程渠道为主 |
| **供应商** | 原材料外采 |

### 竞争格局

**主要竞争对手**：
- 三棵树：建筑涂料龙头，零售强
- 东方雨虹：防水龙头，涂料布局
- 科顺股份：防水材料
- 立邦、多乐士：外资品牌

**公司优势**：
亚士创能保温装饰一体化板具有差异化。但工程渠道占比高，地产下行期业绩波动大。2025年前三季度业绩大幅下滑，需等待地产拐点。

## 五、投资亮点

### 核心亮点

- ✅ **保温装饰一体化**：产品差异化，节能建筑受益
- ✅ **产品线完整**：涂料+保温+防水
- ✅ **行业地位**：建筑涂料、保温行业有一定份额
- ⚠️ **周期底部**：业绩承压，若地产企稳弹性大

### 成长驱动因素

1. **地产企稳**
   - 地产销售、新开工若企稳
   - 工程订单修复
   - 业绩弹性大

2. **保温节能政策**
   - 建筑节能标准提升
   - 保温装饰一体化板需求

3. **渠道调整**
   - 若加大零售、非房渠道
   - 降低地产依赖

### 催化剂

- 地产政策放松
- 地产销售企稳
- 工程订单恢复
- 保温节能政策加码

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 亚士创能处于**业绩底部**，拐点强烈依赖地产复苏：①**地产拐点**——工程渠道占比高，地产销售、新开工若企稳，工程订单将显著恢复，营收拐点出现；②**防水业务拐点**——防水卷材+涂料同比-86.66%，若地产企稳，防水需求修复；③**亏损收窄拐点**——若营收止跌、成本控制，亏损收窄至扭亏。**拐点信号**：单季营收同比降幅收窄至-30%以内、地产销售数据企稳、新签订单环比增长。**风险**：若地产持续低迷，业绩拐点可能推迟，需关注现金流和债务。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 地产企稳、工程订单恢复 | 扭亏为盈，净利润转正 | 净利润增速20%+ |
| **中性** | 地产弱复苏、订单企稳 | 亏损收窄 | 扭亏或微利 |
| **悲观** | 地产继续下行 | 持续亏损 | 亏损扩大 |

### 估值水平

| 指标 | 当前水平 | 行业平均 | 评估 |

**估值分析**：
亚士创能业绩处于底部，估值参考意义有限。若地产企稳、业绩拐点出现，弹性较大。当前适合作为地产复苏的弹性标的观察，需严格控制仓位和止损。
