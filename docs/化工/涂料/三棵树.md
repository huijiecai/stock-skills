## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 三棵树涂料股份有限公司 |

| 所属行业 | 化学原料和化学制品制造业（建筑涂料） |
| 市场地位 | 建筑涂料龙头，零售业务高增长 |

**公司概况**：
三棵树是国内建筑涂料行业龙头企业，产品涵盖墙面涂料、木器涂料、防水材料等。2024年营收121.05亿元、同比-2.97%，归母净利润3.32亿元、同比+91.27%，利润大幅增长主要受益于零售业务双位数增长、成本控制及产品结构优化。精细化工（涂料）收入103.21亿元、占比85.26%，防水卷材13.19亿元、同比+4.93%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **墙面涂料** | 内墙漆、外墙漆、艺术漆 | 住宅、公建装修 |
| **木器涂料** | 家具漆、地板漆 | 家具、木地板 |
| **防水卷材** | 防水材料 | 建筑防水 |
| **辅材** | 腻子、胶粘剂等 | 装修辅料 |

### 业务模式

公司采用"工程+零售"双渠道模式。工程渠道对接房企、工装项目；零售渠道通过经销商、专卖店覆盖C端家装市场。2024年零售业务成为增长引擎，实现双位数增长，工程渠道受地产影响承压。

### 技术路线/核心技术

- **涂料配方**：环保、净味、抗甲醛等
- **艺术漆**：高端产品，毛利率高
- **渠道建设**：专卖店、经销商、电商

**技术优势**：
三棵树品牌知名度高，零售渠道掌控力强。艺术漆、高端产品占比提升，产品结构优化。零售业务抗周期能力强，地产下行期表现优于工程渠道。

## 四、行业地位与产业链定位

### 产业链定位

```
建筑涂料产业链：
树脂/钛白粉/填料 → 【三棵树 涂料/防水】→ 房地产/家装 → 建筑
                        ↑
                  三棵树（品牌龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 建筑涂料市占率行业前列，零售渠道领先 |
| **竞争优势** | 品牌、零售渠道、产品结构优化 |
| **客户结构** | C端家装、房企、工装项目 |
| **供应商** | 钛白粉、树脂、乳液等外采 |

### 竞争格局

**主要竞争对手**：
- 东方雨虹：防水龙头，涂料布局
- 亚士创能：建筑涂料、保温
- 立邦、多乐士：外资品牌
- 嘉宝莉、巴德士：国内品牌

**公司优势**：
三棵树零售业务占比高，地产下行期抗周期能力强。2024年净利+91.27%体现零售增长、成本控制、产品结构优化成效。建筑涂料国产替代逻辑清晰。

## 五、投资亮点

### 核心亮点

- ✅ **建筑涂料龙头**：品牌、渠道、产品领先
- ✅ **零售高增长**：双位数增长，抗周期
- ✅ **利润拐点**：2024年净利+91.27%
- ✅ **产品结构优化**：艺术漆、高端产品占比提升
- ✅ **防水拓展**：防水卷材+4.93%，协同发展

### 成长驱动因素

1. **零售渠道放量**
   - 家装零售抗周期
   - 专卖店、经销商扩张
   - 电商渠道

2. **产品结构优化**
   - 艺术漆、高端产品
   - 毛利率提升

3. **地产复苏**
   - 若地产企稳，工程渠道修复
   - 存量房翻新需求

### 催化剂

- 地产政策放松
- 零售业务持续高增长
- 原材料价格回落
- 艺术漆渗透率提升

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 三棵树2024年已实现**利润拐点**（净利+91.27%），由零售增长、成本控制、产品结构优化驱动。下一阶段拐点取决于：①**收入拐点**——营收-2.97%，若地产企稳、工程渠道修复，或零售增速进一步加快，营收转正；②**毛利率拐点**——若原材料回落、高端产品占比提升，毛利率持续改善；③**地产拐点**——地产销售、新开工若企稳，工程渠道需求修复。**拐点信号**：营收同比转正、单季毛利率环比提升、地产销售数据企稳。
