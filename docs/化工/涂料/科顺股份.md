## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 科顺防水科技股份有限公司 |
| 证券代码 | 300737 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1996年 |
| 上市时间 | 2018年1月25日 |
| 所属行业 | 化学原料和化学制品制造业（防水材料） |
| 市场地位 | 防水材料行业第二，2024年营收79亿元 |

**公司概况**：
科顺股份是国内防水材料行业第二梯队龙头，产品涵盖防水卷材、防水涂料等。2024年营收79.44亿元，行业第三（仅次于东方雨虹、三棵树）。2025年前三季度业绩承压：营收47.03亿元、同比-8.43%，归母净利润2008.77万元、同比-84.08%。工程渠道为主，地产下行冲击明显。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **防水卷材** | SBS/APP改性沥青卷材、高分子卷材 | 建筑屋面、地下室防水 |
| **防水涂料** | 聚氨酯、丙烯酸、水泥基涂料 | 建筑防水、家装 |
| **其他** | 密封材料、堵漏材料等 | 建筑防水 |

### 业务模式

公司采用"工程渠道为主"模式，客户以房企、基建项目为主。地产下行导致工程订单下滑，营收和利润承压。零售渠道占比低于东方雨虹，抗周期能力较弱。

### 技术路线/核心技术

- **防水技术**：改性沥青、高分子、涂料
- **系统服务**：材料+施工
- **品牌**：科顺品牌在工程领域有知名度

**技术优势**：
科顺股份防水产品线完整，工程渠道客户基础好。但工程占比高，地产周期敏感性强。与东方雨虹相比，规模、品牌、零售渠道有差距。

## 四、行业地位与产业链定位

### 产业链定位

```
防水材料产业链：
沥青/树脂/乳液 → 【科顺股份 防水卷材/涂料】→ 房地产/基建 → 建筑
                        ↑
                  科顺股份（行业第二）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 防水行业第二梯队，营收79亿 |
| **竞争优势** | 工程渠道、产品线、品牌 |
| **客户结构** | 房企、基建项目，工程为主 |
| **供应商** | 沥青、树脂等外采 |

### 竞争格局

**主要竞争对手**：
- 东方雨虹：绝对龙头，规模、渠道全面领先
- 三棵树：涂料龙头，防水布局
- 凯伦股份：防水材料
- 北新建材：防水收购

**公司优势**：
科顺股份防水行业第二，工程渠道有基础。但龙头东方雨虹优势明显，科顺需在零售、非房领域拓展，降低地产依赖。

## 五、投资亮点

### 核心亮点

- ✅ **防水行业第二**：营收79亿，行业地位稳固
- ✅ **产品线完整**：卷材+涂料全覆盖
- ✅ **工程渠道**：房企、基建客户基础
- ⚠️ **周期底部**：业绩承压，若地产企稳弹性大

### 成长驱动因素

1. **地产企稳**
   - 地产销售、新开工若企稳
   - 工程订单修复
   - 业绩弹性

2. **渠道多元化**
   - 若加大零售、非房渠道
   - 降低地产依赖

3. **成本控制**
   - 原材料价格若回落
   - 毛利率改善

### 催化剂

- 地产政策放松
- 地产销售企稳
- 工程订单恢复
- 原材料价格回落

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 科顺股份拐点强烈依赖地产复苏：①**地产拐点**——工程渠道为主，地产销售、新开工若企稳，工程订单修复，营收拐点；②**利润拐点**——净利润-84.08%处于底部，若营收企稳、原材料回落、费用控制，利润拐点；③**渠道拐点**——若加大零售、非房、基建渠道，降低地产依赖，业绩稳定性提升。**拐点信号**：单季营收同比转正、净利润同比降幅收窄至-30%以内、地产销售数据企稳。**风险**：龙头东方雨虹优势明显，科顺拐点弹性可能弱于龙头。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 地产企稳、工程订单恢复 | 净利润增速30%+ | 净利润增速20%+ |
| **中性** | 地产弱复苏、订单企稳 | 净利润增速0-20% | 净利润增速10-20% |
| **悲观** | 地产继续下行 | 净利润增速<0% | 净利润增速<10% |
