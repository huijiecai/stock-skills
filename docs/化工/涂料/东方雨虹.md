## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 北京东方雨虹防水技术股份有限公司 |
| 证券代码 | 002271 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2008年9月10日 |
| 所属行业 | 化学原料和化学制品制造业（防水涂料） |
| 市场地位 | 防水涂料龙头，2024年营收328亿元行业第一 |

**公司概况**：
东方雨虹是国内防水材料行业绝对龙头，产品涵盖防水卷材、防水涂料、砂浆、保温等。2024年营收328.23亿元，行业第一。2025年前三季度营收206.01亿元，归母净利润8.10亿元，经营活动现金流净额同比+184.56%。工程与零售渠道合计创收114.06亿元、占比84.06%，海外收入同比+42.16%，出海成效显著。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **防水卷材** | SBS/APP改性沥青卷材、高分子卷材 | 建筑屋面、地下室防水 |
| **防水涂料** | 聚氨酯、丙烯酸、水泥基涂料 | 建筑防水、家装 |
| **砂浆粉料** | 瓷砖胶、腻子、抹灰砂浆 | 建筑装修 |
| **保温材料** | 保温板、保温砂浆 | 建筑节能 |

### 业务模式

公司采用"工程+零售+海外"多渠道模式。工程渠道对接房企、基建项目；零售渠道覆盖C端家装、建材店；海外业务快速增长，2025年前三季度同比+42.16%。多元化渠道分散地产风险。

### 技术路线/核心技术

- **防水技术**：改性沥青、高分子、涂料多品类
- **系统服务**：设计、施工、材料一体化
- **品牌渠道**：雨虹品牌知名度高

**技术优势**：
东方雨虹防水技术国内领先，产品线最全。工程渠道客户覆盖头部房企，零售渠道持续拓展。海外布局打开成长空间。

## 四、行业地位与产业链定位

### 产业链定位

```
防水材料产业链：
沥青/树脂/乳液 → 【东方雨虹 防水卷材/涂料】→ 房地产/基建 → 建筑
                        ↑
                  东方雨虹（绝对龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 防水行业市占率第一，营收328亿 |
| **竞争优势** | 品牌、渠道、产品线、规模 |
| **客户结构** | 头部房企、基建、零售、海外 |
| **供应商** | 沥青、树脂等外采，规模采购优势 |

### 竞争格局

**主要竞争对手**：
- 科顺股份：防水行业第二
- 三棵树：涂料龙头，防水布局
- 凯伦股份：防水材料
- 北新建材：防水收购布局

**公司优势**：
东方雨虹防水行业绝对龙头，市占率、品牌、渠道全面领先。海外拓展打开成长空间。现金流改善体现经营质量提升。

## 五、投资亮点

### 核心亮点

- ✅ **防水绝对龙头**：营收328亿，行业第一
- ✅ **现金流改善**：经营现金流+184.56%
- ✅ **海外高增长**：海外收入+42.16%
- ✅ **渠道多元**：工程+零售+海外
- ✅ **产品多元**：防水+砂浆+保温

### 成长驱动因素

1. **海外拓展**
   - 海外收入高增长
   - 全球化布局
   - 分散国内地产风险

2. **零售渠道**
   - C端家装、翻新需求
   - 抗周期

3. **地产复苏**
   - 若地产企稳，工程渠道修复
   - 存量+增量需求

### 催化剂

- 地产政策放松
- 海外业务持续高增长
- 现金流持续改善
- 基建投资加码

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 东方雨虹拐点取决于：①**地产拐点**——工程渠道若随地产企稳修复，营收增速转正；②**现金流拐点**——经营现金流+184.56%已现改善，若持续为正且覆盖资本开支，财务质量拐点确认；③**海外拐点**——海外+42.16%高增长若延续，成为业绩稳定器，降低地产依赖；④**利润拐点**——若原材料回落、规模效应，毛利率改善，净利润增速拐点。**拐点信号**：单季营收同比转正、经营现金流持续为正、海外收入占比突破20%。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |

| **乐观** | 地产企稳、海外高增、现金流改善 | 净利润增速20%+ | 净利润增速15%+ |
