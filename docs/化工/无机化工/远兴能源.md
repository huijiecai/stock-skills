## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 内蒙古远兴能源股份有限公司 |
| 证券代码 | 000683 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1997年 |
| 上市时间 | 1997年1月31日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工） |
| 市场地位 | 天然碱龙头，纯碱成本最低，产能持续扩张 |

**公司概况**：
远兴能源是国内天然碱行业绝对龙头，拥有河南桐柏、内蒙古阿拉善等天然碱矿资源。天然碱法成本显著低于氨碱法、联碱法，具备极强成本竞争力。公司纯碱产能持续扩张，阿拉善天然碱项目投产后将成为全球最大纯碱供应商。2024年纯碱价格大幅下跌，但天然碱成本优势仍能保持盈利，行业拐点来临将最大受益。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **纯碱** | 天然碱法，成本最低，产能持续扩张 | 玻璃、洗涤剂、氧化铝 |
| **小苏打** | 天然碱副产，食品级、工业级 | 食品、医药、饲料 |
| **尿素** | 煤化工配套 | 化肥、化工 |
| **甲醇** | 煤化工配套 | 化工、燃料 |

### 业务模式

公司采用"天然碱+煤化工"双轮驱动，天然碱为核心。天然碱法无需石灰石、煤炭，环保压力小，成本远低于氨碱法。阿拉善天然碱项目产能大、成本低，投产后将重塑行业格局。

### 技术路线/核心技术

- **天然碱法**：碱矿开采、加工，成本最低
- **碱矿资源**：河南桐柏、内蒙古阿拉善
- **小苏打**：天然碱副产，食品级品质
- **煤化工**：尿素、甲醇补充

**技术优势**：
天然碱法成本显著低于氨碱法、联碱法，为纯碱行业成本最低路线。碱矿资源稀缺，国内仅少数企业拥有。阿拉善项目投产后产能全球第一。

## 四、行业地位与产业链定位

### 产业链定位

```
天然碱产业链：
碱矿 → 【远兴能源 纯碱/小苏打】→ 玻璃/洗涤剂/氧化铝 → 终端
                ↑
           远兴能源（成本最低）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 天然碱产能国内第一，阿拉善投产后全球第一 |
| **竞争优势** | 成本最低、碱矿稀缺、产能扩张 |
| **客户结构** | 玻璃企业、洗涤剂、氧化铝等 |
| **供应商** | 碱矿自给，成本可控 |

### 竞争格局

**主要竞争对手**：
- 三友化工、山东海化、中盐化工：氨碱法
- 和邦生物：天然碱，规模较小

**公司优势**：
天然碱成本最低，行业亏损时仍能盈利。碱矿资源稀缺，国内仅少数企业拥有。阿拉善项目投产后产能全球第一，行业格局重塑。

## 五、投资亮点

### 核心亮点

- ✅ **天然碱龙头**：成本最低，行业亏损时仍能盈利
- ✅ **拐点逻辑**：纯碱价格2024年大幅下跌，天然碱成本优势凸显；行业拐点来临将最大受益
- ✅ **产能扩张**：阿拉善项目投产后全球第一
- ✅ **碱矿稀缺**：资源壁垒高，难以复制
- ✅ **小苏打**：盈利稳定，食品级品质

### 成长驱动因素

1. **纯碱拐点**
   - 纯碱价格大幅下跌，氨碱法亏损
   - 天然碱成本优势凸显，市占率提升
   - 行业拐点来临，天然碱龙头最大受益

2. **阿拉善产能**
   - 阿拉善天然碱项目产能大、成本低
   - 投产后纯碱产能全球第一
   - 行业格局重塑，龙头地位巩固

3. **小苏打**
   - 食品级、工业级需求稳定
   - 天然碱副产，盈利稳定

### 催化剂

- **纯碱价格**：触底反弹，天然碱弹性最大
- **阿拉善投产**：产能释放，市占率提升
- **供给收缩**：氨碱法高成本产能退出
- **光伏玻璃**：需求超预期

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **纯碱拐点** | 价格大跌，天然碱成本优势凸显；拐点来临最大受益 | 价格反弹、供给收缩 |
| **市占率拐点** | 阿拉善投产，产能全球第一 | 产能释放 |
| **业绩拐点** | 成本最低，价格反弹时弹性最大 | 纯碱价涨 |
