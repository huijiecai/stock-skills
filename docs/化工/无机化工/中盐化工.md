## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中盐内蒙古化工股份有限公司 |
| 证券代码 | 600328 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2000年12月22日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工） |
| 市场地位 | 纯碱+烧碱双龙头，中盐集团旗下核心化工平台 |

**公司概况**：
中盐化工是中盐集团旗下核心化工企业，拥有纯碱、烧碱、PVC、金属钠等产品线。纯碱产能约400万吨/年，烧碱产能约100万吨/年，金属钠产能全球领先。公司依托内蒙古煤炭、原盐资源，具备成本优势。2024年纯碱、烧碱价格波动影响业绩，但双产品龙头地位稳固。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **纯碱** | 产能约400万吨/年，氨碱法 | 玻璃、洗涤剂、氧化铝 |
| **烧碱** | 产能约100万吨/年 | 造纸、纺织、化工、氧化铝 |
| **PVC** | 氯碱配套 | 建材、管材、型材 |
| **金属钠** | 产能全球领先 | 医药、染料、电池 |
| **氯酸钠** | 配套产品 | 造纸、水处理 |

### 业务模式

公司采用"纯碱+烧碱+PVC+金属钠"多产品模式，纯碱、烧碱为核心。依托内蒙古煤炭、原盐资源，一体化成本优势明显。中盐集团央企背景，资源整合能力强。

### 技术路线/核心技术

- **氨碱法纯碱**：规模大、工艺成熟
- **离子膜烧碱**：先进工艺，能耗低
- **金属钠**：全球产能领先，技术壁垒高
- **氯碱平衡**：烧碱、PVC、液氯协同

**技术优势**：
纯碱+烧碱双龙头，规模效应显著。金属钠全球领先，细分领域垄断。内蒙古资源禀赋，原料成本低。

## 四、行业地位与产业链定位

### 产业链定位

```
纯碱/烧碱产业链：
原盐+煤炭 → 【中盐化工 纯碱/烧碱】→ 玻璃/造纸/氧化铝 → 终端
                    ↑
               中盐化工（双龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 纯碱约400万吨、烧碱约100万吨，行业前列 |
| **竞争优势** | 双产品龙头、资源禀赋、央企背景 |
| **客户结构** | 玻璃、造纸、氧化铝、纺织等 |
| **供应商** | 原盐、煤炭自给或就近采购 |

### 竞争格局

**主要竞争对手**：
- 纯碱：远兴能源、三友化工、山东海化
- 烧碱：氯碱化工、中泰化学、滨化股份

**公司优势**：
纯碱+烧碱双龙头，规模效应显著。金属钠全球领先，细分垄断。中盐集团央企资源，整合能力强。

## 五、投资亮点

### 核心亮点

- ✅ **纯碱+烧碱双龙头**：产能规模行业前列
- ✅ **拐点逻辑**：纯碱、烧碱价格2024年大幅下跌，若供需改善价格有望反弹
- ✅ **金属钠全球领先**：细分领域垄断，盈利稳定
- ✅ **资源禀赋**：内蒙古煤炭、原盐，成本优势
- ✅ **央企背景**：中盐集团，融资成本低

### 成长驱动因素

1. **纯碱拐点**
   - 纯碱价格大幅下跌，行业亏损面扩大
   - 光伏玻璃、浮法玻璃需求若复苏
   - 供给收缩，高成本产能退出

2. **烧碱拐点**
   - 烧碱价格触底，氧化铝、造纸需求
   - 氯碱平衡，PVC景气影响

3. **金属钠**
   - 医药、染料、钠电池等新兴需求
   - 全球产能领先，议价能力强

### 催化剂

- **纯碱/烧碱价格**：触底反弹
- **光伏/浮法玻璃**：需求超预期
- **氧化铝**：电解铝扩产带动烧碱需求
- **钠电池**：金属钠新兴应用

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **纯碱拐点** | 价格大跌后供需有望改善 | 光伏需求、供给收缩 |
| **烧碱拐点** | 氯碱景气触底，氧化铝需求 | 氧化铝扩产、出口 |
| **业绩拐点** | 双产品若同时反弹弹性大 | 纯碱+烧碱价涨 |
