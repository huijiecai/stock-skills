## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 安徽安纳达钛业股份有限公司 |
| 证券代码 | 002136 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1994年 |
| 上市时间 | 2007年5月30日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工/钛白粉） |
| 市场地位 | 钛白粉+磷酸铁双主业，铜陵有色旗下 |

**公司概况**：
安纳达是铜陵有色旗下化工企业，钛白粉产能约8万吨/年，同时布局磷酸铁新能源材料。公司采用硫酸法工艺，钛白粉生产副产硫酸亚铁可制磷酸铁，实现资源综合利用。2024年钛白粉价格波动，磷酸铁产能过剩盈利承压，双主业均面临周期压力。铜陵有色国企背景，资源整合能力强。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **钛白粉** | 产能约8万吨/年，硫酸法 | 涂料、塑料、造纸 |
| **磷酸铁** | 硫酸亚铁制磷酸铁 | 磷酸铁锂正极材料 |
| **金红石型** | 钛白粉高端产品 | 涂料、塑料 |
| **锐钛型** | 钛白粉通用产品 | 造纸、化纤 |

### 业务模式

公司采用"钛白粉+磷酸铁"双主业模式，钛白粉副产硫酸亚铁制磷酸铁，实现产业链延伸。铜陵有色旗下，具备国企资源。下游与涂料、新能源电池相关。

### 技术路线/核心技术

- **硫酸法钛白粉**：工艺成熟
- **磷酸铁**：硫酸亚铁综合利用，新能源延伸
- **一体化**：钛白粉-磷酸铁协同
- **国企背景**：铜陵有色

**技术优势**：
钛白粉-磷酸铁一体化，副产综合利用。磷酸铁布局较早，具备先发优势。规模较龙头小，需关注成本控制。

## 四、行业地位与产业链定位

### 产业链定位

```
钛白粉-磷酸铁产业链：
钛矿+硫酸 → 【安纳达 钛白粉】→ 涂料/塑料
              ↓ 硫酸亚铁
          【安纳达 磷酸铁】→ 磷酸铁锂
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 钛白粉约8万吨，规模较小；磷酸铁有布局 |
| **竞争优势** | 一体化、国企背景 |
| **客户结构** | 涂料、塑料、新能源电池 |
| **供应商** | 钛矿、硫酸外采 |

### 竞争格局

**主要竞争对手**：
- 钛白粉：龙佰集团、中核钛白、金浦钛业、惠云钛业
- 磷酸铁：湖南裕能、德方纳米、中核钛白等

**公司优势**：
钛白粉-磷酸铁一体化，副产综合利用。铜陵有色国企资源。规模较小，弹性与风险并存。

## 五、投资亮点

### 核心亮点

- ✅ **双主业**：钛白粉+磷酸铁，一体化布局
- ✅ **拐点逻辑**：钛白粉若价格反弹有弹性；磷酸铁产能过剩后若出清、需求复苏有弹性
- ✅ **国企背景**：铜陵有色，资源整合
- ⚠️ **规模**：较龙头小，双主业均面临周期压力

### 成长驱动因素

1. **钛白粉拐点**
   - 价格波动后，若需求复苏企稳
   - 房地产、涂料需求

2. **磷酸铁拐点**
   - 产能过剩，行业出清
   - 磷酸铁锂需求若复苏
   - 一体化成本优势

3. **一体化**
   - 硫酸亚铁制磷酸铁，副产利用
   - 成本协同

### 催化剂

- **钛白粉价格**：触底反弹
- **磷酸铁锂**：新能源需求复苏
- **房地产**：涂料需求
- **产能出清**：磷酸铁行业整合

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **钛白粉拐点** | 价格波动后，若需求复苏 | 房地产、出口 |
| **磷酸铁拐点** | 产能过剩后出清，若需求复苏 | 磷酸铁锂需求 |
| **业绩拐点** | 双主业若同时改善弹性大 | 钛白粉+磷酸铁价涨 |
