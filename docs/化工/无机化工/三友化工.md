## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 唐山三友化工股份有限公司 |
| 证券代码 | 600409 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 2003年6月18日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工） |
| 市场地位 | 纯碱行业龙头，粘胶短纤重要供应商 |

**公司概况**：
三友化工是河北省属国有控股企业，纯碱产能约340万吨/年，位居行业前列。公司形成"纯碱+粘胶短纤+烧碱+有机硅"多产品格局，纯碱为核心盈利来源。2024年纯碱价格波动较大，行业供需格局变化影响业绩。公司依托唐山港区位优势，物流成本低，具备成本竞争力。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **纯碱** | 产能约340万吨/年，氨碱法工艺 | 玻璃、洗涤剂、氧化铝、化工 |
| **粘胶短纤** | 产能约50万吨/年 | 纺织、无纺布、混纺 |
| **烧碱** | 氯碱配套 | 造纸、纺织、化工 |
| **有机硅** | 单体及下游 | 硅橡胶、密封胶、电子 |
| **PVC** | 氯碱配套 | 建材、管材、型材 |

### 业务模式

公司采用"纯碱+粘胶+氯碱+有机硅"多产品平台模式，纯碱为核心。通过规模化生产、产业链协同和区位优势降本增效。纯碱下游主要为浮法玻璃、光伏玻璃，需求与房地产、新能源相关。

### 技术路线/核心技术

- **氨碱法纯碱**：传统工艺，规模大、成本可控
- **粘胶短纤**：差别化、高端化产品
- **氯碱平衡**：烧碱、PVC、液氯平衡生产
- **有机硅**：单体及深加工

**技术优势**：
纯碱产能规模大，氨碱法工艺成熟。唐山港区位优势，原盐、煤炭等原料运输成本低。多产品平台分散单一产品周期风险。

## 四、行业地位与产业链定位

### 产业链定位

```
纯碱产业链：
原盐+石灰石+煤炭 → 【三友化工 纯碱】→ 玻璃/洗涤剂/氧化铝 → 终端应用
                        ↑
                   三友化工（龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 纯碱产能约340万吨，行业前列 |
| **竞争优势** | 规模效应、区位优势、多产品平台 |
| **客户结构** | 玻璃企业、洗涤剂、氧化铝等 |
| **供应商** | 原盐、石灰石、煤炭等，唐山港物流优势 |

### 竞争格局

**主要竞争对手**：
- 远兴能源：天然碱龙头，成本最低
- 山东海化：纯碱产能大
- 中盐化工：纯碱+烧碱
- 和邦生物：天然碱

**公司优势**：
纯碱产能规模大，氨碱法工艺成熟。唐山港区位优势显著，多产品平台分散风险。国企背景融资成本低。

## 五、投资亮点

### 核心亮点

- ✅ **纯碱龙头**：产能约340万吨，行业前列
- ✅ **拐点逻辑**：纯碱价格2024年大幅下跌后，光伏玻璃、浮法玻璃需求若复苏，价格有望企稳反弹
- ✅ **多产品平台**：纯碱+粘胶+氯碱+有机硅，分散周期风险
- ✅ **区位优势**：唐山港物流成本低
- ✅ **国企背景**：融资成本低、管理规范

### 成长驱动因素

1. **纯碱拐点**
   - 纯碱价格2024年大幅下跌，行业亏损面扩大
   - 光伏玻璃产能投放放缓，供需有望改善
   - 浮法玻璃若房地产企稳，需求边际改善

2. **供给收缩**
   - 高成本产能退出，行业集中度提升
   - 环保、能耗双控限制新增产能

3. **粘胶短纤**
   - 若纺织出口复苏，粘胶需求改善
   - 差别化产品占比提升

### 催化剂

- **纯碱价格**：价格触底反弹，突破成本线
- **光伏玻璃**：装机需求超预期，纯碱需求增加
- **房地产**：浮法玻璃需求边际改善
- **供给收缩**：高成本产能退出

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **纯碱拐点** | 价格大幅下跌后，供需有望改善 | 光伏需求、供给收缩 |
| **业绩拐点** | 纯碱盈利触底，若价格反弹弹性大 | 纯碱价涨、需求复苏 |
| **成本拐点** | 原盐、煤炭价格若回落 | 原料成本下降 |
