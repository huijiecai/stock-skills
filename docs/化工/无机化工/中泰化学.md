## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 新疆中泰化学股份有限公司 |
| 证券代码 | 002092 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001年 |
| 上市时间 | 2006年12月8日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工/氯碱） |
| 市场地位 | 西北氯碱龙头，烧碱+PVC，新疆国企 |

**公司概况**：
中泰化学是新疆维吾尔自治区国资委控股的氯碱龙头，烧碱产能约150万吨/年，PVC产能约200万吨/年，位居全国前列。公司依托新疆煤炭、原盐、石灰石资源，形成"煤炭-电石-PVC-烧碱"一体化布局，成本优势显著。2024年氯碱行业景气度下行，烧碱、PVC价格承压，业绩波动。西北龙头，若行业拐点来临，一体化成本优势将充分体现。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **烧碱** | 产能约150万吨/年，离子膜法 | 氧化铝、造纸、纺织、化工 |
| **PVC** | 产能约200万吨/年，电石法 | 建材、管材、型材、薄膜 |
| **电石** | 自产，PVC原料 | 自用 |
| **粘胶纤维** | 纺织原料 | 纺织、服装 |
| **纱线** | 粘胶下游 | 纺织 |

### 业务模式

公司采用"煤炭-电石-PVC-烧碱-粘胶"一体化模式，从煤炭到PVC、烧碱全覆盖，并延伸至粘胶纤维。新疆煤炭、原盐、石灰石资源丰富，电石自给成本低。烧碱与氧化铝需求相关，PVC与房地产、建材相关。粘胶与纺织出口相关。

### 技术路线/核心技术

- **电石法PVC**：煤炭-电石-PVC一体化，成本低
- **离子膜烧碱**：先进工艺
- **粘胶纤维**：纺织延伸
- **新疆资源**：煤炭、原盐、石灰石禀赋

**技术优势**：
西北氯碱龙头，烧碱+PVC产能全国前列。新疆资源禀赋，煤炭-电石-PVC一体化成本最低。粘胶纤维延伸，产业链完整。

## 四、行业地位与产业链定位

### 产业链定位

```
氯碱产业链：
煤炭+原盐+石灰石 → 电石 → 【中泰化学 PVC】→ 建材
        ↓
        【中泰化学 烧碱】→ 氧化铝/造纸
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 烧碱约150万吨、PVC约200万吨，全国前列 |
| **竞争优势** | 新疆资源、一体化、规模 |
| **客户结构** | 氧化铝、建材、纺织企业 |
| **供应商** | 煤炭、原盐、石灰石自给或就近 |

### 竞争格局

**主要竞争对手**：
- 氯碱化工：华东龙头
- 滨化股份：烧碱+环氧丙烷
- 新疆天业：新疆氯碱
- 君正集团：内蒙氯碱

**公司优势**：
西北氯碱龙头，产能全国前列。新疆资源禀赋，煤炭-电石-PVC一体化成本最低。新疆国企，政策支持强。

## 五、投资亮点

### 核心亮点

- ✅ **西北氯碱龙头**：烧碱150万吨、PVC200万吨，全国前列
- ✅ **拐点逻辑**：烧碱、PVC价格2024年承压，若氧化铝、房地产需求复苏，氯碱景气有望改善；一体化成本优势，拐点来临弹性大
- ✅ **一体化成本最低**：煤炭-电石-PVC，新疆资源禀赋
- ✅ **规模优势**：产能全国前列
- ✅ **新疆国企**：政策支持

### 成长驱动因素

1. **烧碱拐点**
   - 氧化铝、电解铝需求
   - 新疆烧碱外销华东、华南
   - 价格若触底反弹

2. **PVC拐点**
   - 房地产、建材需求
   - 电石法成本若回落
   - 一体化成本优势

3. **粘胶**
   - 纺织出口若复苏
   - 粘胶需求改善
   - 当前产能过剩，需出清

### 催化剂

- **烧碱价格**：氧化铝需求
- **PVC价格**：房地产边际改善
- **电解铝**：扩产带动氧化铝、烧碱
- **电石**：原料价格回落

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **烧碱拐点** | 氧化铝需求若复苏 | 电解铝扩产 |
| **PVC拐点** | 房地产若企稳 | 房地产 |
| **业绩拐点** | 一体化成本最低，拐点来临弹性最大 | 烧碱+PVC价涨 |
