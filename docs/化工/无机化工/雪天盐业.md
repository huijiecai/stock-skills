## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 雪天盐业集团股份有限公司 |
| 证券代码 | 600929 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2011年 |
| 上市时间 | 2018年3月26日 |
| 所属行业 | 食品制造业/盐业（无机化工） |
| 市场地位 | 湖南省盐业龙头，盐业改革后跨区域整合 |

**公司概况**：
雪天盐业是湖南省盐业龙头，由原湖南省轻工盐业集团改制而来。公司拥有食用盐、工业盐、纯碱、烧碱等产品线，形成"盐-碱"一体化布局。盐业改革后公司积极跨区域整合，收购重庆、江西等地盐业资产，规模持续扩张。食用盐为刚需，工业盐、纯碱与化工景气相关。2024年盐业稳健，纯碱价格波动影响业绩。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **食用盐** | 加碘盐、高端盐、湖盐等 | 居民消费、食品加工 |
| **工业盐** | 两碱用盐、融雪剂等 | 纯碱、烧碱、化工 |
| **纯碱** | 收购整合 | 玻璃、洗涤剂 |
| **烧碱** | 氯碱配套 | 造纸、纺织、化工 |
| **芒硝** | 盐矿副产 | 洗涤剂、化工 |

### 业务模式

公司采用"盐-碱"一体化模式，从盐矿开采到食用盐、工业盐、纯碱、烧碱全覆盖。盐业改革后跨区域并购，湖南、重庆、江西等地布局。食用盐为刚需，盈利稳定；工业盐、纯碱与化工景气相关。

### 技术路线/核心技术

- **盐矿开采**：井矿盐、湖盐，资源禀赋
- **食用盐**：加碘、高端盐、差异化
- **盐-碱一体化**：工业盐制纯碱、烧碱
- **跨区域整合**：盐业改革后并购扩张

**技术优势**：
盐矿资源禀赋，湖南、重庆等地布局。食用盐刚需，盈利稳定。盐-碱一体化，工业盐自用降本。跨区域整合，规模持续扩张。

## 四、行业地位与产业链定位

### 产业链定位

```
盐业产业链：
盐矿 → 【雪天盐业 食用盐/工业盐】→ 居民/食品/两碱
              ↓
        纯碱/烧碱 → 玻璃/造纸等
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 湖南省盐业龙头，跨区域整合后规模扩大 |
| **竞争优势** | 盐矿资源、食用盐刚需、盐-碱一体化 |
| **客户结构** | 居民、食品加工、两碱企业 |
| **供应商** | 盐矿自给 |

### 竞争格局

**主要竞争对手**：
- 中盐集团：全国盐业龙头
- 各省盐业公司：区域龙头
- 纯碱：远兴能源、三友化工等

**公司优势**：
湖南省盐业龙头，盐业改革后跨区域整合。食用盐刚需，盈利稳定。盐-碱一体化，工业盐自用。纯碱若拐点来临有弹性。

- ✅ **盐业龙头**：湖南省龙头，跨区域整合
- ✅ **拐点逻辑**：食用盐刚需稳定；纯碱若拐点来临，盐-碱一体化受益

- ✅ **盐业改革**：跨区域整合，规模扩张

### 成长驱动因素

   - 刚需，盈利稳定

2. **纯碱拐点**
   - 纯碱若拐点来临
   - 盐-碱一体化受益

3. **跨区域**
   - 盐业改革后并购
   - 重庆、江西等地布局
   - 规模持续扩张
