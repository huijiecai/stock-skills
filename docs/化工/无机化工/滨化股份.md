## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 滨化集团股份有限公司 |
| 证券代码 | 601678 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2010年2月23日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工/氯碱） |
| 市场地位 | 烧碱+环氧丙烷龙头，山东氯碱重要企业 |

**公司概况**：
滨化股份是山东省氯碱重要企业，烧碱产能约65万吨/年，环氧丙烷产能约28万吨/年。环氧丙烷为聚醚、聚氨酯原料，下游应用广泛。公司采用氯醇法环氧丙烷工艺，同时布局PO/MTBE联产。2024年烧碱、环氧丙烷价格波动，氯碱行业景气度下行，业绩承压。环氧丙烷与聚醚、聚氨酯需求相关，若需求复苏拐点逻辑明确。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **烧碱** | 产能约65万吨/年，离子膜法 | 氧化铝、造纸、纺织、化工 |
| **环氧丙烷** | 产能约28万吨/年，氯醇法 | 聚醚、聚氨酯、丙二醇 |
| **三氯乙烯** | 氯碱下游 | 清洗剂、制冷剂 |
| **四氯乙烯** | 氯碱下游 | 干洗、制冷剂 |
| **环氧氯丙烷** | 精细化工 | 环氧树脂、水处理 |

### 业务模式

公司采用"烧碱+环氧丙烷"双核心模式，烧碱为氯碱基础产品，环氧丙烷为高附加值延伸。环氧丙烷下游为聚醚（红宝丽硬泡聚醚原料）、聚氨酯、丙二醇等。氯醇法工艺，与万华化学HPPO法、红宝丽等形成竞争。

### 技术路线/核心技术

- **离子膜烧碱**：先进工艺
- **氯醇法环氧丙烷**：传统工艺，技术成熟
- **PO/MTBE联产**：联产装置，综合利用
- **氯碱平衡**：烧碱、环氧丙烷、液氯协同

**技术优势**：
环氧丙烷产能国内前列，烧碱+环氧丙烷双核心。氯醇法工艺成熟，PO/MTBE联产降本。山东化工产业集群，区位优势。

## 四、行业地位与产业链定位

### 产业链定位

```
烧碱+环氧丙烷产业链：
原盐 → 【滨化股份 烧碱】→ 氧化铝/造纸
  +
丙烯 → 【滨化股份 环氧丙烷】→ 聚醚/聚氨酯 → 终端
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 烧碱约65万吨、环氧丙烷约28万吨，国内前列 |
| **竞争优势** | 双核心、规模、区位 |
| **客户结构** | 氧化铝、聚醚、聚氨酯企业 |
| **供应商** | 原盐、丙烯外采 |

### 竞争格局

**主要竞争对手**：
- 环氧丙烷：万华化学（HPPO法）、红宝丽（自产）、金陵亨斯迈等
- 烧碱：氯碱化工、中泰化学、中盐化工

**公司优势**：
环氧丙烷产能国内前列，烧碱+环氧丙烷双核心。氯醇法工艺成熟，PO/MTBE联产。山东化工产业集群，区位优势。

## 五、投资亮点

### 核心亮点

- ✅ **烧碱+环氧丙烷双核心**：产能国内前列
- ✅ **拐点逻辑**：环氧丙烷与聚醚、聚氨酯需求相关，若家电、建筑需求复苏，环氧丙烷景气有望改善；烧碱若氧化铝需求复苏
- ✅ **环氧丙烷**：聚醚、聚氨酯核心原料，需求广泛
- ✅ **双产品**：烧碱+环氧丙烷，分散风险
- ✅ **山东区位**：化工产业集群

### 成长驱动因素

1. **环氧丙烷拐点**
   - 环氧丙烷与聚醚、聚氨酯、丙二醇需求相关
   - 家电、建筑保温、涂料需求若复苏
   - 价格若触底反弹

2. **烧碱拐点**
   - 氧化铝、电解铝需求
   - 烧碱价格若企稳

3. **PO/MTBE**
   - 联产装置综合利用
   - MTBE为汽油添加剂

### 催化剂

- **环氧丙烷价格**：聚醚、聚氨酯需求
- **烧碱价格**：氧化铝需求
- **家电**：聚醚需求（红宝丽硬泡聚醚）
- **建筑**：聚氨酯保温

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **环氧丙烷拐点** | 聚醚、聚氨酯需求若复苏 | 家电、建筑 |
| **烧碱拐点** | 氧化铝需求若复苏 | 电解铝扩产 |
| **业绩拐点** | 双核心若同时改善弹性大 | 环氧丙烷+烧碱价涨 |
