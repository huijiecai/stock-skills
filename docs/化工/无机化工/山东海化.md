## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 山东海化股份有限公司 |
| 证券代码 | 000822 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 1998年7月3日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工） |
| 市场地位 | 纯碱行业重要企业，海盐资源禀赋 |

**公司概况**：
山东海化是潍坊市属国有控股企业，纯碱产能约280万吨/年，位居行业前列。公司依托渤海湾海盐资源，采用氨碱法工艺，具备原料成本优势。2024年纯碱价格大幅波动，光伏玻璃扩产放缓影响需求，业绩承压。公司纯碱产能规模大，行业地位稳固。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **纯碱** | 产能约280万吨/年，氨碱法 | 玻璃、洗涤剂、氧化铝、化工 |
| **溴素** | 海水提溴，资源禀赋 | 阻燃剂、医药、染料 |
| **原盐** | 海盐生产，自用及外销 | 纯碱原料、食用盐 |
| **氯化钙** | 纯碱副产 | 融雪剂、干燥剂、化工 |

### 业务模式

公司采用"原盐-纯碱-溴素"一体化模式，依托渤海湾海盐资源，从原盐到纯碱全产业链覆盖。溴素为海水提溴，资源稀缺，盈利稳定。纯碱下游主要为浮法玻璃、光伏玻璃。

### 技术路线/核心技术

- **氨碱法纯碱**：海盐为原料，成本优势
- **海水提溴**：资源禀赋，溴素稀缺
- **原盐生产**：海盐滩田，自给能力强
- **氯化钙**：纯碱副产综合利用

**技术优势**：
渤海湾海盐资源禀赋，原盐自给成本低。溴素为海水提溴，资源稀缺性强。纯碱规模大，氨碱法工艺成熟。

## 四、行业地位与产业链定位

### 产业链定位

```
纯碱产业链：
海盐+石灰石+煤炭 → 【山东海化 纯碱】→ 玻璃/洗涤剂/氧化铝 → 终端
                        ↑
                   山东海化（海盐禀赋）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 纯碱产能约280万吨，行业前列 |
| **竞争优势** | 海盐资源、溴素稀缺、一体化 |
| **客户结构** | 玻璃企业、洗涤剂、氧化铝等 |
| **供应商** | 原盐自给，石灰石、煤炭外采 |

### 竞争格局

**主要竞争对手**：
- 远兴能源：天然碱龙头，成本最低
- 三友化工：纯碱龙头
- 中盐化工：纯碱+烧碱
- 和邦生物：天然碱

**公司优势**：
海盐资源禀赋，原盐成本低。溴素资源稀缺，盈利稳定。纯碱规模大，行业地位稳固。

## 五、投资亮点

### 核心亮点

- ✅ **纯碱重要企业**：产能约280万吨，行业前列
- ✅ **拐点逻辑**：纯碱价格2024年大幅下跌，若供需改善价格有望反弹
- ✅ **溴素稀缺**：海水提溴，资源禀赋，盈利稳定
- ✅ **海盐禀赋**：原盐自给，成本优势
- ✅ **一体化**：原盐-纯碱-溴素，产业链协同

### 成长驱动因素

1. **纯碱拐点**
   - 纯碱价格大幅下跌，行业亏损面扩大
   - 光伏玻璃、浮法玻璃需求若复苏
   - 供给收缩，高成本产能退出

2. **溴素**
   - 阻燃剂、医药、染料需求
   - 资源稀缺，价格相对稳定
   - 海水提溴壁垒高

3. **海盐**
   - 原盐自给降本
   - 食用盐、工业盐需求

### 催化剂

- **纯碱价格**：触底反弹
- **光伏玻璃**：装机需求超预期
- **溴素价格**：阻燃剂等需求增长
- **供给收缩**：高成本纯碱产能退出

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **纯碱拐点** | 价格大跌后供需有望改善 | 光伏需求、供给收缩 |
| **业绩拐点** | 纯碱盈利触底，若价格反弹弹性大 | 纯碱价涨 |
| **溴素稳定** | 资源稀缺，盈利相对稳定 | 阻燃剂需求 |
