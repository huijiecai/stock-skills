## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 金浦钛业股份有限公司 |
| 证券代码 | 000545 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 1993年12月15日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工/钛白粉） |
| 市场地位 | 钛白粉行业重要企业，硫酸法工艺 |

**公司概况**：
金浦钛业是国内钛白粉行业重要企业，采用硫酸法工艺，产品涵盖锐钛型、金红石型钛白粉。公司产能约15万吨/年，主要应用于涂料、塑料、造纸等领域。2024年钛白粉价格受供需影响波动，出口需求、房地产景气度影响业绩。公司与龙佰集团、中核钛白等共同构成国内钛白粉第一梯队。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **金红石型钛白粉** | 高端产品，耐候性好 | 涂料、塑料、油墨 |
| **锐钛型钛白粉** | 通用产品 | 造纸、涂料、化纤 |
| **钛白粉** | 总产能约15万吨/年 | 涂料、塑料、造纸、油墨 |

### 业务模式

公司采用硫酸法工艺，从钛矿到钛白粉一体化生产。产品以金红石型为主，应用于高端涂料、塑料等。下游与房地产、汽车、包装等行业景气度相关。

### 技术路线/核心技术

- **硫酸法**：传统工艺，技术成熟
- **金红石型**：高端产品，耐候性、遮盖力好
- **锐钛型**：通用产品，成本较低
- **环保升级**：硫酸法环保改造

**技术优势**：
硫酸法工艺成熟，金红石型产品品质稳定。规模适中，具备一定成本控制能力。与龙佰集团等龙头相比规模较小，但区域市场有优势。

## 四、行业地位与产业链定位

### 产业链定位

```
钛白粉产业链：
钛矿+硫酸 → 【金浦钛业 钛白粉】→ 涂料/塑料/造纸 → 终端应用
                    ↑
                金浦钛业
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 产能约15万吨，行业重要企业 |
| **竞争优势** | 金红石型产品、区域市场 |
| **客户结构** | 涂料、塑料、造纸企业 |
| **供应商** | 钛矿、硫酸外采 |

### 竞争格局

**主要竞争对手**：
- 龙佰集团：全球龙头，氯化法+硫酸法
- 中核钛白：产能大，一体化
- 惠云钛业：硫酸法
- 安纳达：硫酸法，规模较小

**公司优势**：
金红石型产品结构，高端应用占比高。区域市场有优势。与龙头相比规模较小，需关注成本控制。

## 五、投资亮点

### 核心亮点

- ✅ **钛白粉重要企业**：产能约15万吨，行业地位稳固
- ✅ **拐点逻辑**：钛白粉价格2024年波动，若房地产、涂料需求复苏，价格有望企稳反弹
- ✅ **金红石型**：高端产品占比高，盈利能力强
- ⚠️ **规模**：与龙头有差距，弹性相对较小

### 成长驱动因素

1. **钛白粉拐点**
   - 钛白粉价格2024年波动，行业盈利承压
   - 房地产若企稳，涂料需求边际改善
   - 出口需求若复苏

2. **产品结构**
   - 金红石型占比提升
   - 高端应用盈利能力强

3. **成本控制**
   - 钛矿、硫酸价格若回落
   - 环保改造后成本优化

### 催化剂

- **钛白粉价格**：触底反弹
- **房地产**：涂料需求边际改善
- **出口**：钛白粉出口增长

| 拐点类型 | 逻辑 | 催化剂 |
