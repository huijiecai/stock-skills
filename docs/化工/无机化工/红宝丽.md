## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 红宝丽集团股份有限公司 |
| 证券代码 | 002165 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1994年 |
| 上市时间 | 2007年9月13日 |
| 所属行业 | 化学原料和化学制品制造业（聚氨酯/硬泡聚醚） |
| 市场地位 | 硬泡聚醚龙头，冰箱冷柜保温材料核心供应商 |

**公司概况**：
红宝丽是国内硬泡聚醚龙头，产品主要应用于冰箱、冷柜等冷藏设备保温层，以及建筑保温、管道保温等领域。公司硬泡聚醚产能约15万吨/年，与美的、海尔、海信等家电龙头深度合作。2024年家电需求波动、原料环氧丙烷价格影响业绩。硬泡聚醚与冰箱冷柜产量高度相关，房地产、消费复苏为拐点逻辑。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **硬泡聚醚** | 聚氨酯硬泡原料，产能约15万吨/年 | 冰箱冷柜保温、建筑保温、管道保温 |
| **异丙醇胺** | 表面活性剂、水泥助磨剂 | 日化、建材、农药 |
| **环氧丙烷** | 自产，硬泡聚醚原料 | 自用及外销 |
| **聚氨酯保温板** | 下游延伸 | 建筑保温 |

### 业务模式

公司采用"环氧丙烷-硬泡聚醚-保温材料"一体化模式，从原料到终端部分覆盖。硬泡聚醚为核心，下游主要为冰箱冷柜企业。与家电龙头深度绑定，客户粘性强。

### 技术路线/核心技术

- **硬泡聚醚**：聚氨酯硬泡专用聚醚，技术成熟
- **环氧丙烷** | 自产配套，原料保障
- **异丙醇胺** | 细分领域龙头
- **一体化** | 环氧丙烷-硬泡聚醚协同

**技术优势**：
硬泡聚醚龙头，冰箱冷柜保温材料核心供应商。与美的、海尔等深度合作，客户粘性强。环氧丙烷自产，原料成本可控。

## 四、行业地位与产业链定位

### 产业链定位

```
硬泡聚醚产业链：
环氧丙烷 → 【红宝丽 硬泡聚醚】→ 冰箱冷柜/建筑保温 → 终端
                    ↑
                红宝丽（龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 硬泡聚醚龙头，冰箱冷柜保温核心供应商 |
| **竞争优势** | 客户粘性、一体化、技术 |
| **客户结构** | 美的、海尔、海信等家电龙头 |
| **供应商** | 环氧丙烷自产，其他原料外采 |

### 竞争格局

**主要竞争对手**：
- 硬泡聚醚：红宝丽龙头，其他聚醚企业
- 环氧丙烷：滨化股份、万华化学等

**公司优势**：
硬泡聚醚龙头，冰箱冷柜客户深度绑定。环氧丙烷自产，一体化成本优势。异丙醇胺细分龙头。

## 五、投资亮点

### 核心亮点

- ✅ **硬泡聚醚龙头**：冰箱冷柜保温核心供应商
- ✅ **拐点逻辑**：家电需求若复苏，冰箱冷柜产量回升，硬泡聚醚需求改善；环氧丙烷价格若企稳，盈利修复
- ✅ **客户粘性**：美的、海尔等深度合作
- ✅ **一体化**：环氧丙烷自产，成本可控
- ✅ **异丙醇胺**：细分龙头，盈利稳定

### 成长驱动因素

1. **家电拐点**
   - 冰箱冷柜产量与房地产、消费相关
   - 家电以旧换新、消费复苏
   - 硬泡聚醚需求若改善

2. **环氧丙烷**
   - 环氧丙烷价格若企稳
   - 自产配套，价差扩大时受益

3. **建筑保温**
   - 建筑节能、保温需求
   - 聚氨酯保温板渗透率提升

### 催化剂

- **家电需求**：冰箱冷柜产量回升
- **以旧换新**：政策刺激家电消费
- **环氧丙烷**：价格企稳
- **房地产**：建筑保温需求

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **需求拐点** | 家电需求若复苏，硬泡聚醚需求改善 | 以旧换新、消费复苏 |
| **业绩拐点** | 环氧丙烷价格若企稳，盈利修复 | 环氧丙烷价稳 |
| **成本拐点** | 原料价格若回落，毛利率改善 | 环氧丙烷、丙二醇 |
