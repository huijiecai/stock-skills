## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广东惠云钛业股份有限公司 |
| 证券代码 | 300891 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2003年 |
| 上市时间 | 2020年9月17日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工/钛白粉） |
| 市场地位 | 钛白粉行业重要企业，硫酸法，华南区域龙头 |

**公司概况**：
惠云钛业是华南地区钛白粉重要企业，采用硫酸法工艺，产能约10万吨/年。公司位于广东云浮，毗邻珠三角涂料、塑料产业集群，具备区位优势。2024年钛白粉价格波动，出口需求、房地产景气度影响业绩。公司积极拓展高端金红石型产品，产品结构持续优化。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **金红石型钛白粉** | 高端产品，占比提升 | 涂料、塑料、油墨 |
| **锐钛型钛白粉** | 通用产品 | 造纸、涂料、化纤 |
| **钛白粉** | 总产能约10万吨/年 | 涂料、塑料、造纸 |

### 业务模式

公司采用硫酸法工艺，从钛矿到钛白粉生产。产品以金红石型为主，应用于高端涂料、塑料。华南区位优势，贴近珠三角下游客户，物流成本低。

### 技术路线/核心技术

- **硫酸法**：传统工艺，技术成熟
- **金红石型**：高端产品，品质稳定
- **环保升级**：硫酸法环保改造
- **区位优势**：华南，贴近下游

**技术优势**：
华南区域龙头，珠三角涂料、塑料产业集群客户粘性强。金红石型产品结构优化。规模适中，需关注成本控制。

## 四、行业地位与产业链定位

### 产业链定位

```
钛白粉产业链：
钛矿+硫酸 → 【惠云钛业 钛白粉】→ 涂料/塑料/造纸 → 终端
                    ↑
                惠云钛业（华南龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 产能约10万吨，华南区域龙头 |
| **竞争优势** | 区位优势、金红石型、客户粘性 |
| **客户结构** | 珠三角涂料、塑料企业 |
| **供应商** | 钛矿、硫酸外采 |

### 竞争格局

**主要竞争对手**：
- 龙佰集团、中核钛白：全国龙头
- 金浦钛业、安纳达：规模相当
- 华南区域：惠云钛业龙头

**公司优势**：
华南区域龙头，珠三角客户粘性强。金红石型产品结构。区位优势物流成本低。

## 五、投资亮点

### 核心亮点

- ✅ **华南区域龙头**：珠三角涂料、塑料客户粘性强
- ✅ **拐点逻辑**：钛白粉价格2024年波动，若需求复苏价格有望企稳
- ✅ **金红石型**：高端产品占比提升
- ✅ **区位优势**：贴近下游，物流成本低
- ⚠️ **规模**：较龙头小，弹性有限

### 成长驱动因素

1. **钛白粉拐点**
   - 价格波动后，若需求复苏企稳
   - 珠三角涂料、塑料需求
   - 出口需求若复苏

2. **产品结构**
   - 金红石型占比提升
   - 高端应用盈利能力强

3. **区位**
   - 华南下游集中
   - 客户服务响应快

### 催化剂

- **钛白粉价格**：触底反弹
- **珠三角**：涂料、塑料需求
- **出口**：钛白粉出口增长
- **房地产**：边际改善

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
