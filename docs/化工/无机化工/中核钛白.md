## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中核华原钛白股份有限公司 |
| 证券代码 | 002145 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1989年 |
| 上市时间 | 2007年8月3日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工/钛白粉） |
| 市场地位 | 钛白粉行业龙头之一，产能第二，一体化布局 |

**公司概况**：
中核钛白是国内钛白粉行业龙头之一，产能约40万吨/年，仅次于龙佰集团。公司采用硫酸法工艺，形成"钛矿-钛白粉-磷酸铁"一体化布局，积极拓展新能源材料。2024年钛白粉价格波动，但公司规模优势、一体化布局提供成本竞争力。磷酸铁锂正极材料前驱体等新能源布局为长期看点。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **钛白粉** | 产能约40万吨/年，硫酸法 | 涂料、塑料、造纸、油墨 |
| **金红石型** | 高端产品为主 | 涂料、塑料 |
| **磷酸铁** | 钛白粉副产硫酸亚铁制磷酸铁 | 磷酸铁锂正极材料 |
| **钛矿** | 自有矿山，原料保障 | 钛白粉原料 |

### 业务模式

公司采用"钛矿-钛白粉-磷酸铁"一体化模式，从原料到钛白粉全覆盖，并延伸至新能源材料。钛白粉生产副产硫酸亚铁，可制磷酸铁，实现资源综合利用。下游与涂料、塑料、新能源电池相关。

### 技术路线/核心技术

- **硫酸法钛白粉**：工艺成熟，规模大
- **钛矿自给**：原料保障，成本可控
- **磷酸铁**：硫酸亚铁综合利用，新能源延伸
- **一体化**：钛矿-钛白粉-磷酸铁产业链

**技术优势**：
钛白粉产能约40万吨，行业第二。钛矿自给降低原料波动风险。磷酸铁延伸实现副产综合利用，新能源布局为长期看点。

## 四、行业地位与产业链定位

### 产业链定位

```
钛白粉产业链：
钛矿 → 【中核钛白 钛白粉】→ 涂料/塑料/造纸 → 终端
              ↓
         磷酸铁 → 磷酸铁锂
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 产能约40万吨，行业第二 |
| **竞争优势** | 规模、一体化、钛矿自给 |
| **客户结构** | 涂料、塑料、造纸、新能源 |
| **供应商** | 钛矿自给，硫酸等外采 |

### 竞争格局

**主要竞争对手**：
- 龙佰集团：全球龙头，氯化法+硫酸法
- 金浦钛业、惠云钛业、安纳达：规模较小

**公司优势**：
产能行业第二，规模优势明显。钛矿自给、一体化布局成本竞争力强。磷酸铁新能源延伸，长期成长空间。

## 五、投资亮点

### 核心亮点

- ✅ **钛白粉龙头之一**：产能约40万吨，行业第二
- ✅ **拐点逻辑**：钛白粉价格2024年波动，若房地产、涂料需求复苏，龙头弹性大
- ✅ **一体化布局**：钛矿-钛白粉-磷酸铁，成本优势
- ✅ **新能源延伸**：磷酸铁布局，长期看点
- ✅ **钛矿自给**：原料保障，成本可控

### 成长驱动因素

1. **钛白粉拐点**
   - 钛白粉价格波动，行业盈利承压
   - 房地产若企稳，涂料需求边际改善
   - 龙头规模优势，拐点来临弹性大

2. **一体化**
   - 钛矿自给降本
   - 硫酸亚铁制磷酸铁，副产综合利用

3. **磷酸铁**
   - 磷酸铁锂需求若复苏
   - 当前产能过剩，需等待行业出清

### 催化剂

- **钛白粉价格**：触底反弹
- **房地产**：涂料需求边际改善
- **磷酸铁锂**：新能源需求复苏
- **出口**：钛白粉出口增长

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **钛白粉拐点** | 价格波动后，龙头弹性大 | 房地产、出口 |
| **业绩拐点** | 一体化成本优势，价格反弹弹性大 | 钛白粉价涨 |
| **磷酸铁拐点** | 产能过剩后出清，若需求复苏 | 磷酸铁锂需求 |
