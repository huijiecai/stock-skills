## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海氯碱化工股份有限公司 |
| 证券代码 | 600618 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1992年 |
| 上市时间 | 1992年11月13日 |
| 所属行业 | 化学原料和化学制品制造业（无机化工/氯碱） |
| 市场地位 | 华东氯碱龙头，烧碱+PVC，华谊集团旗下 |

**公司概况**：
氯碱化工是华谊集团旗下氯碱企业，烧碱产能约72万吨/年，PVC产能约50万吨/年，位居华东前列。公司位于上海化工区，具备区位、物流优势。烧碱下游主要为氧化铝、造纸、纺织；PVC下游为建材、管材。2024年氯碱行业景气度下行，烧碱、PVC价格承压，业绩波动。华东区域龙头，若行业拐点来临有望受益。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **烧碱** | 产能约72万吨/年，离子膜法 | 氧化铝、造纸、纺织、化工 |
| **PVC** | 产能约50万吨/年，电石法 | 建材、管材、型材、薄膜 |
| **液氯** | 氯碱副产 | 化工、消毒 |
| **盐酸** | 氯碱副产 | 化工、冶金 |
| **糊树脂** | PVC特种型号 | 手套、人造革 |

### 业务模式

公司采用氯碱平衡模式，烧碱、PVC为核心产品，液氯、盐酸为副产。烧碱与氧化铝、造纸需求相关；PVC与房地产、建材相关。华东区位，下游客户集中。

### 技术路线/核心技术

- **离子膜烧碱**：先进工艺，能耗低
- **电石法PVC**：传统工艺，电石原料
- **糊树脂**：特种PVC，高端应用
- **氯碱平衡**：烧碱、PVC、液氯协同

**技术优势**：
华东氯碱龙头，规模大。离子膜烧碱工艺先进。上海化工区区位优势，物流、配套完善。华谊集团国企背景。

## 四、行业地位与产业链定位

### 产业链定位

```
氯碱产业链：
原盐+电石 → 【氯碱化工 烧碱+PVC】→ 氧化铝/造纸/建材 → 终端
                    ↑
                氯碱化工（华东龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 烧碱约72万吨、PVC约50万吨，华东前列 |
| **竞争优势** | 区位、规模、国企背景 |
| **客户结构** | 氧化铝、造纸、建材企业 |
| **供应商** | 原盐、电石外采 |

### 竞争格局

**主要竞争对手**：
- 中泰化学：烧碱+PVC，西北龙头
- 滨化股份：烧碱+环氧丙烷
- 中盐化工：烧碱+纯碱
- 新疆天业：氯碱

**公司优势**：
华东龙头，区位优势明显。烧碱+PVC双产品，规模大。华谊集团国企背景，管理规范。

## 五、投资亮点

### 核心亮点

- ✅ **华东氯碱龙头**：烧碱72万吨、PVC50万吨
- ✅ **拐点逻辑**：烧碱、PVC价格2024年承压，若氧化铝、房地产需求复苏，氯碱景气有望改善
- ✅ **区位优势**：上海化工区，华东下游集中
- ✅ **双产品**：烧碱+PVC，分散单一产品风险
- ✅ **国企背景**：华谊集团

### 成长驱动因素

1. **烧碱拐点**
   - 烧碱与氧化铝、造纸需求相关
   - 电解铝扩产带动氧化铝、烧碱需求
   - 价格若触底反弹

2. **PVC拐点**
   - PVC与房地产、建材相关
   - 房地产若企稳，PVC需求边际改善
   - 电石法成本若回落

3. **糊树脂**
   - 特种PVC，手套等需求
   - 高端产品盈利能力强

### 催化剂

- **烧碱价格**：氧化铝需求带动
- **PVC价格**：房地产边际改善
- **电解铝**：扩产带动氧化铝、烧碱
- **出口**：烧碱出口增长

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **烧碱拐点** | 氧化铝需求若复苏，烧碱景气改善 | 电解铝扩产 |
| **PVC拐点** | 房地产若企稳，PVC需求改善 | 房地产 |
| **业绩拐点** | 双产品若同时改善弹性大 | 烧碱+PVC价涨 |
