## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 三角轮胎股份有限公司 |
| 证券代码 | 601163 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2001年 |
| 上市时间 | 2016年9月9日 |
| 所属行业 | 橡胶和塑料制品业（轮胎） |
| 市场地位 | 全钢胎、工程胎，2024年业绩承压 |

**公司概况**：
三角轮胎主营全钢子午线轮胎、工程胎、乘用车胎等。2024年营业收入101.56亿元、同比-2.55%，归母净利润11.03亿元、同比-21.03%。在2024年轮胎行业整体丰收的背景下，公司营收、利润双降，表现弱于行业。或与产品结构、客户结构、原材料成本等因素相关。品牌"三角"在卡客车胎、工程胎领域有知名度。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **全钢胎** | 卡客车子午线轮胎 | 卡车、客车 |
| **工程胎** | 工程机械轮胎 | 挖掘机、装载机 |
| **半钢胎** | 乘用车胎 | 轿车、SUV |
| **其他** | 特种轮胎 | 特种车辆 |

### 业务模式

公司采用"配套+替换"双渠道模式，国内基地为主。全钢胎、工程胎占比大，与物流、基建、矿山景气相关。配套客户包括国内商用车企、工程机械厂。

### 技术路线/核心技术

- **全钢胎**：卡客车胎技术成熟
- **工程胎**：矿用、装载机胎
- **智能制造**：自动化生产

**技术优势**：
三角轮胎全钢胎、工程胎产品线完整，品牌有积淀。但2024年业绩承压，需关注产品结构、成本控制、市场拓展。

## 四、行业地位与产业链定位

### 产业链定位

```
轮胎产业链：
天然胶/合成胶/炭黑 → 【三角轮胎 轮胎】→ 整车厂/替换市场 → 汽车/工程机械
                              ↑
                        三角轮胎（全钢+工程）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全钢胎、工程胎行业有一定份额 |
| **竞争优势** | 品牌、产品线、配套客户 |
| **客户结构** | 商用车企、工程机械厂、经销商 |
| **供应商** | 天然胶、合成胶等外采 |

### 竞争格局

**主要竞争对手**：
- 玲珑轮胎：全品类龙头
- 赛轮轮胎：规模领先
- 贵州轮胎：工程胎龙头
- 双钱轮胎：全钢胎
- 通用股份：全钢胎

**公司优势**：
三角轮胎品牌有积淀，全钢胎、工程胎产品线完整。但2024年业绩承压，需通过产品升级、成本控制、市场拓展实现拐点。

## 五、投资亮点

### 核心亮点

- ✅ **品牌积淀**：三角品牌在卡客车、工程胎领域有知名度
- ✅ **产品线完整**：全钢胎+工程胎+半钢胎
- ✅ **配套客户**：商用车、工程机械配套
- ⚠️ **周期底部**：2024年业绩承压，若拐点出现弹性大

### 成长驱动因素

1. **物流、基建景气**
   - 全钢胎、工程胎与物流、基建相关
   - 若景气回升，需求修复

2. **成本控制**
   - 原材料价格若回落
   - 毛利率改善

3. **产品结构优化**
   - 高端产品占比提升
   - 毛利率改善

### 催化剂

- 物流、基建景气回升
- 原材料价格回落
- 产品结构优化
- 海外市场拓展

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 三角轮胎处于**业绩底部**，拐点取决于：①**营收拐点**——营收-2.55%，若物流、基建景气回升，全钢胎、工程胎需求修复，营收拐点；②**利润拐点**——净利-21.03%，若原材料回落、成本控制、产品结构优化，利润拐点；③**行业拐点**——公司2024年逆势下滑，若通过内部调整（产品、成本、渠道）实现与行业同步或超越，则拐点确认。**拐点信号**：单季营收同比转正、单季净利润同比降幅收窄至-10%以内、毛利率环比提升。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 物流基建景气、原材料回落、内部调整见效 | 净利润增速20%+ | 净利润增速15%+ |
| **中性** | 行业企稳、成本可控 | 净利润增速0-15% | 净利润增速5-15% |
