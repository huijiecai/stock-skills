## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 青岛森麒麟轮胎股份有限公司 |
| 证券代码 | 002984 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2007年 |
| 上市时间 | 2020年9月11日 |
| 所属行业 | 橡胶和塑料制品业（轮胎） |
| 市场地位 | 半钢胎龙头，2024年净利+59.74%增速行业第一 |

**公司概况**：
森麒麟专注高性能半钢子午线轮胎，产品以乘用车胎为主。2024年营业总收入85.11亿元、同比+8.53%，归母净利润21.86亿元、同比+59.74%，净利润增速行业第一。产销量3222.61万条/3140.87万条，同比+10.22%/+7.34%。高性能乘用车胎在美国市场表现出色，与中国轮胎盈利前四的玲珑、赛轮、中策齐名。2024年现金流同比-17.19%，需关注。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **半钢胎** | 乘用车胎，高性能、大尺寸 | 轿车、SUV、新能源车 |
| **航空胎** | 航空轮胎 | 民航、通航 |
| **其他** | 特种轮胎 | 特种车辆 |

### 业务模式

公司采用"高端化+美国市场"策略，专注半钢胎，不做全钢胎。产品以高性能、大尺寸为主，毛利率高。美国市场占比大，泰国基地供应。航空胎为差异化布局，技术壁垒高。

### 技术路线/核心技术

- **半钢胎**：高性能、大尺寸、低滚阻
- **航空胎**：民航轮胎，国产替代
- **智能制造**：自动化程度高

**技术优势**：
森麒麟专注半钢胎，产品高端化。美国市场双反税率优势，泰国基地供应。航空胎国产替代，成长空间大。2024年净利增速59.74%行业第一，盈利能力突出。

## 四、行业地位与产业链定位

### 产业链定位

```
轮胎产业链：
天然胶/合成胶/炭黑 → 【森麒麟 半钢胎】→ 整车厂/替换市场 → 汽车
                        ↑
                  森麒麟（半钢胎龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 半钢胎细分龙头，盈利前四 |
| **竞争优势** | 高端化、美国市场、航空胎、盈利能力 |
| **客户结构** | 美国替换市场、国内车企配套 |
| **供应商** | 天然胶、合成胶等外采 |

### 竞争格局

**主要竞争对手**：
- 玲珑轮胎：全品类龙头
- 赛轮轮胎：液体黄金、多品类
- 中策橡胶：国内龙头
- 国际：米其林、普利司通等

**公司优势**：
森麒麟专注半钢胎，高端化策略清晰。美国市场双反税率优势，泰国基地供应。2024年净利增速59.74%行业第一，盈利能力突出。航空胎差异化，国产替代空间大。

## 五、投资亮点

### 核心亮点

- ✅ **净利增速第一**：2024年+59.74%，行业最快
- ✅ **半钢胎龙头**：专注高端，毛利率高
- ✅ **美国市场**：高性能胎表现出色
- ✅ **航空胎**：国产替代，成长空间
- ✅ **盈利前四**：与玲珑、赛轮、中策齐名

### 成长驱动因素

1. **美国市场**
   - 高性能胎需求
   - 双反税率优势
   - 泰国基地供应

2. **产品高端化**
   - 大尺寸、高性能占比提升
   - 毛利率提升

3. **航空胎**
   - 民航轮胎国产替代
   - 技术壁垒高
   - 成长空间大

### 催化剂

- 美国轮胎需求稳健
- 航空胎认证、放量
- 大尺寸胎渗透率提升
- 原材料价格回落

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 森麒麟2024年已实现**盈利拐点**（净利+59.74%），拐点由美国市场、产品高端化、盈利能力提升驱动。下一阶段拐点取决于：①**航空胎拐点**——若航空胎认证通过、批量供货，第二增长曲线打开；②**现金流拐点**——经营现金流-17.19%需改善，若应收账款回收、存货周转优化，现金流拐点；③**美国占比拐点**——若美国营收占比突破50%，海外依赖度拐点，抗国内周期。**拐点信号**：航空胎营收突破1亿、经营现金流同比转正、美国营收占比超50%。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 美国高增、航空胎放量、现金流改善 | 净利润增速25%+ | 净利润增速20%+ |
| **中性** | 美国稳健、产品高端化延续 | 净利润增速15-25% | 净利润增速15-20% |
