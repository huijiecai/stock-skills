## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏通用科技股份有限公司 |
| 证券代码 | 601500 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2002年 |
| 上市时间 | 2016年9月19日 |
| 所属行业 | 橡胶和塑料制品业（轮胎） |
| 市场地位 | 全钢胎为主，国内+泰国双基地 |

**公司概况**：
通用股份主营全钢子午线轮胎、半钢胎，产品以卡客车胎为主。拥有国内江苏、安徽基地，海外泰国基地，双基地布局应对贸易政策。品牌"千里马""赤兔马"在卡客车胎领域有知名度。2024年轮胎行业整体景气，公司受益于行业复苏。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **全钢胎** | 卡客车子午线轮胎 | 卡车、客车 |
| **半钢胎** | 乘用车胎 | 轿车、SUV |
| **其他** | 斜交胎等 | 特种车辆 |

### 业务模式

公司采用"国内+泰国"双基地模式，泰国基地供应海外市场，规避双反。配套+替换双渠道，卡客车胎以替换市场为主。产品以性价比著称，中高端市场持续拓展。

### 技术路线/核心技术

- **全钢胎**：卡客车胎技术成熟
- **双基地**：国内+泰国，贸易政策应对
- **智能制造**：自动化生产

**技术优势**：
通用股份全钢胎产能规模行业前列，泰国基地具备双反优势。产品性价比突出，替换市场渠道完善。

## 四、行业地位与产业链定位

### 产业链定位

```
轮胎产业链：
天然胶/合成胶/炭黑/钢丝 → 【通用股份 轮胎】→ 整车厂/替换市场 → 汽车/物流
                              ↑
                        通用股份（全钢胎）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全钢胎行业有一定份额，规模中等 |
| **竞争优势** | 双基地、性价比、替换市场渠道 |
| **客户结构** | 物流车队、经销商、整车厂 |
| **供应商** | 天然胶、合成胶等外采 |

### 竞争格局

**主要竞争对手**：
- 玲珑轮胎：全品类龙头
- 赛轮轮胎：规模领先
- 贵州轮胎：工程胎
- 三角轮胎：全钢胎
- 双钱轮胎：全钢胎

**公司优势**：
通用股份双基地布局，泰国基地应对贸易政策。全钢胎性价比突出，替换市场渠道完善。2024年行业景气，公司受益。

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 通用股份拐点取决于：①**行业拐点**——2024年轮胎行业已现景气，若延续则公司受益；②**泰国拐点**——泰国基地若满产、出口放量，海外收入拐点；③**原材料拐点**——天然胶、炭黑若回落，毛利率改善；④**物流拐点**——卡客车胎与物流相关，若物流景气回升，需求拐点。**拐点信号**：单季营收同比转正或加速、泰国基地产能利用率超90%、毛利率环比提升。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 行业景气延续、泰国放量、原材料回落 | 净利润增速20%+ | 净利润增速15%+ |
| **中性** | 行业稳健、泰国稳步释放 | 净利润增速10-20% | 净利润增速10-15% |
| **悲观** | 行业回落、原材料涨价 | 净利润增速<10% | 净利润增速<10% |
