## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 山东玲珑轮胎股份有限公司 |
| 证券代码 | 601966 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1975年 |
| 上市时间 | 2016年7月6日 |
| 所属行业 | 橡胶和塑料制品业（轮胎） |
| 市场地位 | 轮胎龙头，2024年营收220.6亿、净利17.5亿、+26% |

**公司概况**：
玲珑轮胎是国内轮胎行业龙头企业，产品涵盖乘用车胎、卡客车胎、工程胎等。2024年营业收入220.6亿元、同比+9.4%，归母净利润17.5亿元、同比+26%，业绩稳健增长。公司在中国轮胎企业盈利排名前四，高性能乘用车轮胎在美国市场表现出色。国内+海外双基地布局，塞尔维亚工厂推进中。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **乘用车胎** | 半钢子午线轮胎 | 轿车、SUV |
| **卡客车胎** | 全钢子午线轮胎 | 卡车、客车 |
| **工程胎** | 工程机械轮胎 | 挖掘机、装载机 |
| **其他** | 特种轮胎 | 航空、特种车辆 |

### 业务模式

公司采用"国内+海外"双基地模式，国内山东、广西、湖北等基地，海外泰国、塞尔维亚（在建）。配套+替换双渠道，配套客户包括国内外主流车企，替换市场通过经销商覆盖。品牌"玲珑"知名度高。

### 技术路线/核心技术

- **子午线轮胎**：半钢、全钢技术成熟
- **高性能轮胎**：美国市场表现出色
- **智能制造**：自动化、数字化生产

**技术优势**：
玲珑轮胎产品品质获国际认可，高性能乘用车胎在美国市场有竞争力。国内龙头，规模、品牌、渠道优势突出。

## 四、行业地位与产业链定位

### 产业链定位

```
轮胎产业链：
天然胶/合成胶/炭黑/钢丝 → 【玲珑轮胎 轮胎】→ 整车厂/替换市场 → 汽车
                              ↑
                        玲珑轮胎（龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内轮胎龙头，全球前列 |
| **竞争优势** | 规模、品牌、美国市场、双基地 |
| **客户结构** | 国内外车企配套、替换市场经销商 |
| **供应商** | 天然胶、合成胶、炭黑等外采 |

### 竞争格局

**主要竞争对手**：
- 赛轮轮胎：营收、净利规模领先
- 森麒麟：半钢胎、美国市场高增长
- 中策橡胶：国内龙头
- 国际：米其林、普利司通、固特异等

**公司优势**：
玲珑轮胎国内龙头地位稳固，美国市场突破带来增量。2024年业绩稳健增长，盈利能力提升。双基地布局应对贸易政策。

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 玲珑轮胎已处于业绩上行通道，拐点更多体现为**增速能否维持或加速**。核心观察：①**美国市场拐点**——高性能胎美国放量若持续，海外占比提升，业绩弹性增强；②**原材料拐点**——天然胶、炭黑若回落，毛利率改善，利润弹性；③**塞尔维亚拐点**——欧洲工厂若投产，欧洲市场突破，全球化布局完善。**拐点信号**：美国市场营收占比突破30%、单季毛利率环比提升、塞尔维亚工厂投产。
