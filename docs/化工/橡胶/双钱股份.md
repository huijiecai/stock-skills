## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海华谊集团股份有限公司（原双钱股份） |
| 证券代码 | 600623 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1992年（双钱品牌始于1929年） |
| 上市时间 | 1992年 |
| 所属行业 | 化学原料和化学制品制造业（轮胎+化工） |
| 市场地位 | 双钱轮胎营收109亿创历史新高，全球75强第22位 |

**公司概况**：
华谊集团（600623）拥有双钱轮胎等业务，双钱轮胎为全资子公司，拥有"双钱""回力"两大民族品牌。2024年双钱集团轮胎营收109.41亿元、同比+7.43%，再创历史新高；生产1705.89万条、销售1623.97万条、同比+9.1%。重庆、新疆基地产能利用率超100%，江苏载重胎项目67%产能，安徽二期建成投产。全球轮胎75强排名第22位。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **全钢胎** | 卡客车子午线轮胎 | 卡车、客车 |
| **工程胎** | 工程机械轮胎 | 挖掘机、装载机 |
| **半钢胎** | 乘用车胎（回力品牌） | 轿车、SUV |
| **其他** | 特种轮胎 | 特种车辆 |

### 业务模式

公司采用"国内+泰国"多基地模式，重庆、新疆、江苏、安徽、泰国等基地。双钱品牌主打全钢胎、工程胎，回力品牌主打半钢胎。配套+替换双渠道，2024年产能利用率高位，产销两旺。

### 技术路线/核心技术

- **全钢胎**：卡客车胎技术成熟
- **工程胎**：矿用、装载机胎
- **多基地**：重庆、新疆、江苏、安徽、泰国

**技术优势**：
双钱轮胎中华老字号，品牌积淀深厚。多基地布局，产能利用率高。2024年营收创历史新高，贡献华谊集团业绩增长。

### 产业链定位

```
轮胎产业链：
天然胶/合成胶/炭黑 → 【双钱轮胎 轮胎】→ 整车厂/替换市场 → 汽车/工程机械
                        ↑
                  双钱轮胎（华谊集团）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球轮胎75强第22位，国内前列 |

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 双钱轮胎（华谊集团轮胎业务）2024年已处于**业绩上行**（营收创历史新高），拐点更多体现为**增速能否维持或加速**。核心观察：①**安徽二期拐点**——若安徽二期满产，产能拐点，产销进一步放量；②**江苏载重胎拐点**——产能利用率67%，若爬坡至90%+，贡献增量；③**集团拐点**——华谊集团为化工+轮胎，若化工业务企稳，集团业绩拐点。**拐点信号**：双钱轮胎营收增速突破10%、安徽二期产能利用率超90%、华谊集团净利润增速加速。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
