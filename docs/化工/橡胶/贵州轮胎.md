## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 贵州轮胎股份有限公司 |
| 证券代码 | 000589 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1958年 |
| 上市时间 | 1996年3月8日 |
| 所属行业 | 橡胶和塑料制品业（轮胎） |
| 市场地位 | 工程胎龙头，卡客车胎、工程胎双轮驱动 |

**公司概况**：
贵州轮胎是国内工程胎领域龙头企业，产品涵盖工程机械轮胎、卡客车胎等。2024年营业总收入106.91亿元、同比+11.35%，归母净利润6.15亿元、同比-26.08%。轮胎产量961.09万条、同比+7.42%，销量960.18万条、同比+11.81%。营收增长、利润下滑，主要受原材料价格、产品结构及竞争影响。经营现金流7.32亿元，拟每10股派1.3元。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **工程胎** | 工程机械轮胎，矿用、装载机等 | 挖掘机、装载机、矿卡 |
| **卡客车胎** | 全钢子午线轮胎 | 卡车、客车 |
| **其他** | 特种轮胎 | 特种车辆 |

### 业务模式

公司采用"工程胎+卡客车胎"双轮驱动，工程胎为特色优势产品，毛利率相对较高。配套+替换双渠道，工程胎配套工程机械龙头。国内基地为主，越南基地推进中。

### 技术路线/核心技术

- **工程胎**：矿用、装载机胎，技术壁垒高
- **全钢胎**：卡客车胎技术成熟
- **越南基地**：海外布局推进

**技术优势**：
贵州轮胎工程胎国内龙头，矿用、装载机胎等细分领域优势明显。工程机械与基建、矿山相关，周期性强。卡客车胎补充规模。

## 四、行业地位与产业链定位

### 产业链定位

```
轮胎产业链：
天然胶/合成胶/炭黑 → 【贵州轮胎 工程胎/卡客车胎】→ 工程机械/物流 → 矿山/基建
                              ↑
                        贵州轮胎（工程胎龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 工程胎龙头，卡客车胎有一定份额 |
| **竞争优势** | 工程胎技术、细分领域龙头 |
| **客户结构** | 工程机械厂、物流车队、经销商 |
| **供应商** | 天然胶、合成胶等外采 |

### 竞争格局

**主要竞争对手**：
- 玲珑轮胎：全品类龙头
- 赛轮轮胎：规模领先
- 三角轮胎：全钢胎
- 双钱轮胎：全钢胎、工程胎

**公司优势**：
贵州轮胎工程胎国内龙头，矿用、装载机胎等细分优势明显。工程机械与基建、矿山景气相关，若基建加码，工程胎需求弹性大。

## 五、投资亮点

### 核心亮点

- ✅ **工程胎龙头**：矿用、装载机胎国内领先
- ✅ **营收增长**：2024年+11.35%，产销量增长
- ✅ **双轮驱动**：工程胎+卡客车胎
- ✅ **现金流**：经营现金流7.32亿
- ✅ **分红**：拟每10股派1.3元

### 成长驱动因素

1. **基建、矿山景气**
   - 工程胎与基建、矿山相关
   - 若基建加码，需求弹性大

2. **工程胎高端化**
   - 矿用、大型工程机械胎
   - 毛利率提升

3. **越南基地**
   - 海外布局
   - 产能释放

### 催化剂

- 基建投资加码
- 矿山开工率提升
- 原材料价格回落
- 越南基地投产

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 贵州轮胎拐点取决于：①**利润拐点**——净利-26.08%处于承压期，若原材料回落、产品结构优化、费用控制，利润拐点；②**工程机械拐点**——工程胎与基建、矿山相关，若基建加码、矿山开工率提升，需求拐点；③**毛利率拐点**——若原材料回落、工程胎高端产品占比提升，毛利率拐点。**拐点信号**：单季净利润同比降幅收窄至-10%以内、工程机械销量同比转正、毛利率环比提升。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 基建加码、原材料回落、毛利率改善 | 净利润增速20%+ | 净利润增速15%+ |
| **中性** | 工程机械企稳、成本可控 | 净利润增速0-15% | 净利润增速5-15% |
| **悲观** | 工程机械下行、原材料涨价 | 净利润增速<0% | 净利润增速<5% |
