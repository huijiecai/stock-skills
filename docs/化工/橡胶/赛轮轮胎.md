## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 赛轮集团股份有限公司 |
| 证券代码 | 601058 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2002年 |
| 上市时间 | 2011年6月30日 |
| 所属行业 | 橡胶和塑料制品业（轮胎） |
| 市场地位 | 轮胎行业领先，2024年营收、净利创历史新高 |

**公司概况**：
赛轮轮胎是国内轮胎行业领军企业，2024年营业总收入318.02亿元、同比+22.42%，净利润40.63亿元、同比+31.42%，营收和净利均创历史新高。轮胎产量7481.11万条、同比+27.59%，销量7215.58万条、同比+29.34%，产销量高增长。多元布局打开长期发展空间，国内+海外双基地，高性能轮胎在美国市场表现出色。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **半钢胎** | 乘用车胎 | 轿车、SUV |
| **全钢胎** | 卡客车胎 | 卡车、客车 |
| **工程胎** | 工程机械轮胎 | 挖掘机、装载机 |
| **其他** | 特种轮胎 | 航空、特种车辆 |

### 业务模式

公司采用"国内+海外"多基地模式，国内青岛、东营、沈阳等，海外越南、柬埔寨等。配套+替换双渠道，高性能乘用车胎在美国市场表现突出。液体黄金等新材料技术应用，产品差异化。

### 技术路线/核心技术

- **液体黄金**：新型橡胶材料，滚动阻力低、耐磨
- **智能制造**：自动化、数字化
- **多基地**：国内+越南+柬埔寨，分散贸易风险

**技术优势**：
赛轮轮胎液体黄金等新材料技术领先，产品性能优异。美国市场突破带来高增长。产销量规模行业领先，规模效应显著。

## 四、行业地位与产业链定位

### 产业链定位

```
轮胎产业链：
天然胶/合成胶/炭黑/钢丝 → 【赛轮轮胎 轮胎】→ 整车厂/替换市场 → 汽车
                              ↑
                        赛轮轮胎（领先）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 营收318亿，产销量行业领先 |
| **竞争优势** | 液体黄金技术、美国市场、多基地、规模 |
| **客户结构** | 国内外车企、替换市场 |
| **供应商** | 天然胶、合成胶等外采，液体黄金自研 |

### 竞争格局

**主要竞争对手**：
- 玲珑轮胎：国内龙头
- 森麒麟：半钢胎、美国市场高增长
- 中策橡胶：国内龙头
- 国际：米其林、普利司通等

**公司优势**：
赛轮轮胎2024年营收、净利规模及增速行业领先。液体黄金技术差异化，美国市场突破。产销量高增长，规模效应释放。多元布局打开长期空间。

   - 毛利率提升

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 赛轮轮胎已处于业绩高增长通道，拐点更多体现为**增速能否维持**。核心观察：①**液体黄金拐点**——若液体黄金产品占比突破、毛利率显著提升，则技术拐点兑现；②**美国市场拐点**——美国营收占比若持续提升至40%+，海外依赖度拐点，抗国内周期；③**产能拐点**——柬埔寨等新基地若满产，产销量拐点，规模效应进一步释放。**拐点信号**：液体黄金营收占比突破10%、美国市场营收占比超40%、单季产销量增速维持20%+。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |

| **乐观** | 产销量高增延续、液体黄金放量 | 净利润增速25%+ | 净利润增速20%+ |
