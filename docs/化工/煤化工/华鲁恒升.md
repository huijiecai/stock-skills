## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 山东华鲁恒升化工股份有限公司 |
| 证券代码 | 600426 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2000年 |
| 上市时间 | 2002年6月20日 |
| 所属行业 | 化学原料和化学制品制造业（煤化工） |
| 市场地位 | 煤化工"一头多线"模式典范，醋酸产能全国前三，DMF全球最大、全国制造业单项冠军 |

**公司概况**：
华鲁恒升是国内重要的基础化工原料制造商和新型煤化工行业领军企业，采用"一头多线"柔性生产模式，以煤为原料通过气化平台生产合成气，灵活调配生产醋酸、DMF、尿素、己内酰胺、碳酸二甲酯等产品。公司醋酸产能150万吨/年进入全国前三，DMF全球最大、市占率全国首位，入选第七批全国制造业单项冠军。2024年营收342.26亿元同比+25.55%，归母净利润39.03亿元同比+9.14%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **醋酸及衍生品** | 醋酸、醋酐等，产能150万吨/年 | PTA、醋酸乙烯、农药、医药 |
| **DMF** | 二甲基甲酰胺，全球最大 | 聚氨酯、电子、医药 |
| **新能源新材料** | 己内酰胺、尼龙6、己二酸、碳酸二甲酯、碳酸甲乙酯 | 尼龙、锂电池电解液 |
| **化学肥料** | 尿素 | 农业肥料 |
| **有机胺** | 一甲胺、二甲胺、三甲胺 | 农药、医药、染料 |

### 业务模式

公司采用"一头多线"柔性生产模式：以煤为原料经气化制合成气（CO+H2），根据市场供需灵活调配生产醋酸、DMF、尿素、己内酰胺、碳酸二甲酯等产品。山东德州、湖北荆州两大基地，总资产450亿元，员工5700人。通过柔性调配实现效益最大化，成本控制能力强。

### 技术路线/核心技术

- **煤气化**：水煤浆气化技术，合成气平台
- **一头多线**：根据产品价差灵活调配产能，效益最大化
- **醋酸**：甲醇羰基化工艺，产能150万吨/年
- **DMF**：二甲胺与一氧化碳合成，全球最大产能
- **己内酰胺/尼龙6**：环己酮-羟胺路线

**技术优势**：
华鲁恒升"一头多线"模式为行业典范，可根据醋酸、DMF、尿素、己内酰胺等产品价差灵活调配，抗周期能力强。DMF入选全国制造业单项冠军，醋酸产能全国前三，成本控制领先。

## 四、行业地位与产业链定位

### 产业链定位

```
煤化工产业链：
煤炭 → 煤气化 → 合成气 → 【华鲁恒升 醋酸/DMF/尿素/己内酰胺】→ PTA/聚氨酯/肥料/尼龙
                              ↑
                         华鲁恒升
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 醋酸产能150万吨/年全国前三，DMF全球最大、市占率全国首位 |
| **竞争优势** | "一头多线"柔性生产，成本控制强，DMF制造业单项冠军 |
| **客户结构** | 覆盖PTA、聚氨酯、农业、新能源等下游头部企业 |
| **供应商** | 煤炭采购，规模与长协优势 |

### 竞争格局

**主要竞争对手**：
- 江苏索普：醋酸产能
- 兖矿能源：煤化工
- 宝丰能源：煤制烯烃
- 万华化学：己内酰胺、尼龙

**公司优势**：
华鲁恒升"一头多线"模式可根据产品价差灵活调配，抗周期能力突出。DMF全球最大、制造业单项冠军，醋酸全国前三。新能源新材料板块（己内酰胺、碳酸二甲酯等）快速放量，荆州基地贡献增量。

## 五、投资亮点

### 核心亮点

- ✅ **"一头多线"模式**：煤化工柔性生产典范，根据价差灵活调配，抗周期强
- ✅ **醋酸全国前三**：产能150万吨/年，PTA、醋酸乙烯等下游需求稳健
- ✅ **DMF全球龙头**：全球最大DMF企业，制造业单项冠军，市占率全国首位
- ✅ **新能源新材料放量**：己内酰胺、碳酸二甲酯等，营收164.33亿元
- ✅ **双基地布局**：德州+荆州，产能持续扩张

### 成长驱动因素

1. **产品结构优化**
   - 新能源新材料占比提升
   - 己内酰胺、碳酸二甲酯放量
   - 荆州基地产能释放

2. **柔性调配效益**
   - 根据醋酸、DMF、尿素、己内酰胺价差调配
   - 单产品景气波动时切换至高价差产品
   - 综合效益最大化

3. **成本控制**
   - 煤气化一体化
   - 规模优势
   - 煤炭采购议价

### 催化剂

- 醋酸、DMF价格企稳回升
- 新能源新材料需求增长
- 荆州基地产能释放
- 煤炭价格回落改善成本
