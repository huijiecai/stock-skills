## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 阳煤化工股份有限公司 |
| 证券代码 | 600691 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 1996年 |
| 所属行业 | 化学原料和化学制品制造业（煤化工） |
| 市场地位 | 山西煤化工龙头，尿素+甲醇+乙二醇，华阳集团旗下 |

**公司概况**：
阳煤化工是山西省煤化工龙头，华阳集团（原阳煤集团）旗下化工平台。公司产品涵盖尿素、甲醇、乙二醇、烧碱、PVC等，形成"煤炭-煤气化-化肥/化工"一体化布局。2024年尿素价格受农业需求、出口影响；甲醇、乙二醇受化工景气影响。煤化工+化肥双主业，若农业景气、化工拐点来临有望受益。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **尿素** | 氮肥，产能约200万吨/年 | 农业肥料 |
| **甲醇** | 煤制甲醇 | 化工、燃料、MTBE |
| **乙二醇** | 煤制乙二醇 | 聚酯、防冻液 |
| **烧碱** | 氯碱 | 氧化铝、造纸 |
| **PVC** | 氯碱 | 建材、管材 |
| **复合肥** | 氮磷钾复合肥 | 农业肥料 |
| **三聚氰胺** | 化工产品 | 板材、涂料 |

### 业务模式

公司采用"煤炭-煤气化-化肥/化工"一体化模式，以煤为原料经气化制合成气，生产尿素、甲醇、乙二醇等。尿素与农业需求相关；甲醇、乙二醇与化工、聚酯相关。山西煤炭资源丰富，原料成本优势。

### 技术路线/核心技术

- **煤气化**：水煤浆/粉煤气化
- **尿素**：合成氨-尿素
- **甲醇**：合成气制甲醇
- **乙二醇**：合成气制乙二醇
- **煤化工一体化**：多产品联产

**技术优势**：
山西煤炭资源禀赋，原料成本低。煤化工一体化，多产品联产。尿素、甲醇、乙二醇产能大，规模效应。

## 四、行业地位与产业链定位

### 产业链定位

```
煤化工产业链：
煤炭 → 煤气化 → 合成气 → 【阳煤化工 尿素/甲醇/乙二醇】→ 农业/化工/聚酯
                            ↑
                        阳煤化工
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 尿素、甲醇、乙二醇产能大，山西龙头 |
| **竞争优势** | 煤炭资源、一体化、国企 |
| **客户结构** | 农业、化工、聚酯企业 |
| **供应商** | 煤炭自给或集团供应 |

### 竞争格局

**主要竞争对手**：
- 尿素：云天化、华鲁恒升、湖北宜化
- 甲醇：宝丰能源、华鲁恒升
- 乙二醇：华鲁恒升、恒力石化

**公司优势**：
山西煤化工龙头，煤炭资源禀赋。尿素+甲醇+乙二醇多产品。华阳集团国企背景。需关注经营效率、债务压力。

## 五、投资亮点

### 核心亮点

- ✅ **煤化工+化肥龙头**：尿素、甲醇、乙二醇产能大
- ✅ **拐点逻辑**：尿素若农业景气、出口改善；甲醇、乙二醇若化工拐点来临
- ✅ **煤炭资源**：山西煤炭，成本优势
- ✅ **多产品**：尿素+甲醇+乙二醇，分散风险
- ✅ **国企背景**：华阳集团
- ⚠️ **经营**：需关注经营效率、债务

### 成长驱动因素

1. **尿素拐点**
   - 农业需求、粮食安全
   - 尿素出口若改善
   - 化肥景气若持续

2. **甲醇/乙二醇拐点**
   - 甲醇与化工、MTBE需求相关
   - 乙二醇与聚酯需求相关
   - 若化工拐点来临

3. **煤炭**
   - 山西煤炭成本优势
   - 煤化工一体化

### 催化剂

- **尿素价格**：农业景气、出口
- **甲醇价格**：化工需求
- **乙二醇价格**：聚酯需求
- **煤炭价格**：若回落，成本改善

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **尿素拐点** | 农业景气、出口若改善 | 粮食安全、出口 |
| **甲醇拐点** | 化工需求若复苏 | 化工景气 |
| **乙二醇拐点** | 聚酯需求若复苏 | 聚酯景气 |
| **业绩拐点** | 多产品若同时改善 | 尿素+甲醇+乙二醇价涨 |
