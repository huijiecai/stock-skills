## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 宁夏宝丰能源集团股份有限公司 |
| 证券代码 | 600989 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2005年 |
| 上市时间 | 2019年5月16日 |
| 所属行业 | 石油加工、炼焦和核燃料加工业（煤化工） |
| 市场地位 | 煤制烯烃龙头，全球单厂规模最大煤制烯烃项目，绿氢耦合制烯烃先行者 |

**公司概况**：
宝丰能源是国内煤制烯烃龙头企业，采用煤-甲醇-烯烃一体化工艺，拥有宁夏、内蒙古两大基地。内蒙古项目为全球单厂规模最大的煤制烯烃项目，采用第三代DMTO技术，配套40万吨/年绿氢耦合制烯烃为全球首个规模化绿氢替代化石能源生产烯烃项目。2024年营收329.8亿元同比+13.2%，归母净利润63.4亿元同比+12.2%，煤价下跌优势凸显。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **聚乙烯** | 煤制烯烃下游，2024年销量113.52万吨 | 薄膜、管材、注塑 |
| **聚丙烯** | 煤制烯烃下游，2024年销量116.49万吨 | 注塑、纤维、薄膜 |
| **焦炭** | 炼焦副产品 | 钢铁冶炼 |
| **精细化工** | 纯苯、改质沥青等 | 化工原料 |
| **绿氢** | 风光制氢，已建6亿立方/年 | 绿氢耦合制烯烃 |

### 业务模式

公司采用"煤-焦-化-烯烃"一体化模式，从煤炭开采到焦炭、甲醇、烯烃全覆盖。内蒙古项目采用第三代DMTO技术，单套烯烃装置产能超100万吨，2.65吨纯甲醇转化1吨烯烃。配套40万吨/年绿氢耦合制烯烃，用绿氢替代原料煤，全球首个规模化项目。宁夏、内蒙古双基地，新疆规划400万吨煤制烯烃。

### 技术路线/核心技术

- **第三代DMTO技术**：中科院大化所开发，单套产能超100万吨
- **煤-甲醇-烯烃**：2.65吨甲醇转化1吨烯烃
- **绿氢耦合制烯烃**：40万吨/年绿氢替代原料煤，减碳631万吨/年
- **风光制氢**：绿氢年产能6亿立方，全球单厂规模最大

**技术优势**：
宝丰能源内蒙古项目为全球单厂规模最大煤制烯烃项目，采用第三代DMTO技术。配套40万吨/年绿氢耦合制烯烃为全球首个规模化项目，可节约标准煤253万吨/年、减少碳排放631万吨/年。已建成全球单厂规模最大绿氢生产厂，绿氢年产能6亿立方。

## 四、行业地位与产业链定位

### 产业链定位

```
煤化工产业链：
煤炭 → 甲醇 → 【宝丰能源 烯烃】→ 聚乙烯/聚丙烯 → 薄膜/管材/注塑
                    ↑
               宝丰能源（+绿氢耦合）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 煤制烯烃龙头，内蒙古项目占全国烯烃产量23.4%，待全部投产后产能520万吨 |
| **竞争优势** | 全球单厂规模最大、绿氢耦合制烯烃先行、第三代DMTO技术 |
| **客户结构** | 聚乙烯、聚丙烯覆盖塑料加工、薄膜、管材等下游 |
| **供应商** | 煤炭自给+外购，绿氢自产 |

### 竞争格局

**主要竞争对手**：
- 中煤能源：煤制烯烃
- 中国神华：煤制烯烃
- 延长石油：煤制烯烃
- 油头烯烃：中石化、中石油等

**公司优势**：
宝丰能源内蒙古项目为全球单厂规模最大煤制烯烃项目，仅用18个月建成投产。绿氢耦合制烯烃为全球首个规模化项目，绿氢先行者地位明确。煤价下跌时成本优势凸显，聚乙烯、聚丙烯价差改善。新疆规划400万吨煤制烯烃，成长空间大。

## 五、投资亮点

### 核心亮点

- ✅ **煤制烯烃龙头**：全球单厂规模最大，内蒙古项目占全国产量23.4%
- ✅ **绿氢先行者**：40万吨/年绿氢耦合制烯烃，全球首个规模化项目，减碳631万吨/年
- ✅ **成本优势**：煤价下跌时价差改善，2024年聚乙烯、聚丙烯价差分别+8.8%、+7.7%
- ✅ **产能扩张**：300万吨烯烃陆续投产，待全部投产后产能达520万吨
- ✅ **高分红**：2024年分红30.1亿元，分红比例47.44%

### 成长驱动因素

1. **产能释放**
   - 内蒙古第一、二、三系列300万吨烯烃陆续投产
   - 待全部投产后烯烃产能达520万吨
   - 新疆400万吨煤制烯烃规划

2. **绿氢耦合**
   - 40万吨/年绿氢替代原料煤
   - 节约标准煤253万吨/年
   - 碳减排、ESG优势

3. **成本与价差**
   - 煤价下跌改善成本
   - 聚乙烯、聚丙烯价差提升
   - 一体化成本优势

### 催化剂

- 内蒙古300万吨烯烃满产
- 煤价维持低位
- 烯烃价格企稳
- 绿氢耦合项目投产
- 新疆项目推进
