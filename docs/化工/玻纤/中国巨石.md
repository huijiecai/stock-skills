## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国巨石股份有限公司 |
| 证券代码 | 600176 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 1999年 |
| 所属行业 | 非金属矿物制品业（玻璃纤维） |
| 市场地位 | 全球玻纤龙头，玻纤纱年产能超300万吨，产能规模全球第一 |

**公司概况**：
中国巨石是中国建材股份有限公司的玻璃纤维业务核心企业，全球玻纤工业领导者。公司玻纤纱年产能超300万吨，拥有浙江桐乡、江西九江、四川成都、江苏淮安、埃及苏伊士、美国南卡六大生产基地，产品覆盖20多个大类3000多个规格品种。2024年玻纤纱销量首次突破300万吨达302.50万吨，电子布销量8.75亿米，但受产品价格下降影响，归母净利润同比-19.70%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **玻璃纤维纱** | 粗纱、细纱、短切纱等 | 复合材料、风电、建筑、汽车 |
| **电子布** | 电子级玻璃纤维布 | PCB、覆铜板 |
| **热固粗纱** | 热固性复合材料用粗纱 | 风电叶片、管道 |
| **热塑短切** | 热塑性复合材料用短切纱 | 汽车轻量化、家电 |
| **电子薄布** | 超薄电子布 | 高端PCB |

### 业务模式

公司采用"研发-生产-销售"一体化模式，六大基地覆盖国内及埃及、美国，可满足约50%的海外市场需求。产品销往100多个国家和地区，客户包括众多世界500强企业。通过规模化生产、智能制造降本增效，持续开发E7、E8、E9系列高模量玻璃配方。

### 技术路线/核心技术

- **E7/E8/E9高模量玻璃**：应用于汽车轻量化、5G、风电、光伏等领域
- **智能制造基地**：热固粗纱、热塑短切、电子薄布全球第一
- **零碳制造**：2025年开工建设淮安零碳智能制造基地，年产10万吨电子级玻纤
- **海外布局**：埃及、美国基地满足海外需求

**技术优势**：
中国巨石智能制造基地实现"三个第一"：热固粗纱全球第一、热塑短切全球第一、电子薄布全球第一。2024年获得125件专利授权，E7、E8、E9系列高模量配方技术领先。

## 四、行业地位与产业链定位

### 产业链定位

```
玻纤产业链：
叶腊石/石英砂 → 玻璃熔制 → 【中国巨石 玻纤纱/电子布】→ 复合材料 → 风电/汽车/PCB/建筑
                              ↑
                         中国巨石
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 玻纤纱年产能超300万吨，产能规模全球第一，销量302.50万吨 |
| **竞争优势** | 六大基地、智能制造、E7/E8/E9高模量配方、海外布局 |
| **客户结构** | 100多个国家和地区，众多世界500强企业 |
| **供应商** | 叶腊石等原料采购，规模采购优势 |

### 竞争格局

**主要竞争对手**：
- 泰山玻纤：国内第二大玻纤企业
- 重庆国际：国内主要玻纤企业
- 欧文斯科宁：美国玻纤龙头
- 日本电气硝子：电子布领域

**公司优势**：
中国巨石玻纤纱产能超300万吨全球第一，智能制造基地热固粗纱、热塑短切、电子薄布均达全球第一。埃及、美国基地可满足约50%海外需求，全球化布局领先。E7、E8、E9高模量配方应用于风电、汽车轻量化、5G、光伏等高端领域。

## 五、投资亮点

### 核心亮点

- ✅ **全球玻纤龙头**：玻纤纱产能超300万吨全球第一，销量首次突破300万吨
- ✅ **智能制造标杆**：热固粗纱、热塑短切、电子薄布全球第一
- ✅ **六大基地布局**：桐乡、九江、成都、淮安、埃及、美国，海外满足50%需求
- ✅ **高端产品**：E7/E8/E9高模量配方，风电、汽车、5G、光伏应用
- ✅ **现金流改善**：经营现金流同比+134.35%，资产负债率降至41.08%

### 成长驱动因素

1. **高端领域需求**
   - 风电叶片大型化带动高模量玻纤需求
   - 汽车轻量化、热塑复合材料增长
   - 5G、PCB、电子布需求
   - 光伏边框等新应用

2. **产能与产品升级**
   - 淮安零碳智能制造基地年产10万吨电子级玻纤
   - E7/E8/E9系列持续放量
   - 海外基地产能利用提升

3. **成本与效率**
   - 智能制造降本增效
   - 规模优势巩固
   - 原材料采购议价能力

### 催化剂

- 玻纤价格企稳回升
- 风电装机超预期
- 汽车轻量化渗透提升
- 淮安电子级玻纤项目投产
