## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 新洋丰农业科技股份有限公司 |
| 证券代码 | 000902 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 1999年4月8日 |
| 所属行业 | 化学原料和化学制品制造业（复合肥） |
| 市场地位 | 复合肥龙头，新型肥料占比持续提升 |

**公司概况**：
新洋丰是国内复合肥行业龙头企业，拥有磷矿-磷酸-复合肥一体化产业链。2024年营收156亿元、净利润13.1亿元，复合肥销量548.6万吨。新型肥料销量138万吨、同比+22.9%，占比提升至31.7%，成为核心增长引擎。2025年Q1复合肥销量同比+35%，创同期新高。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **传统复合肥** | 氮磷钾复合肥 | 大田作物、经济作物 |
| **新型复合肥** | 缓控释肥、水溶肥、专用肥 | 经济作物、设施农业 |
| **磷肥** | 磷酸一铵等 | 复合肥原料、直接销售 |
| **磷矿** | 磷矿采选 | 自用、外销 |

### 业务模式

公司采用"矿肥一体化"模式，拥有磷矿资源，磷酸一铵自给，成本可控。渠道下沉至县级经销商，品牌"洋丰""澳特尔"知名度高。新型肥料毛利率21.22%高于传统肥16.66%，产品结构持续优化。

### 技术路线/核心技术

- **矿肥一体**：磷矿-磷酸-复合肥产业链
- **新型肥料**：缓控释、水溶肥、生物肥等
- **渠道优势**：县级经销商网络覆盖全国

**技术优势**：
新型肥料增速快于传统肥，毛利率更高。公司持续加大新型肥研发和推广，产品结构优化驱动盈利能力提升。

## 四、行业地位与产业链定位

### 产业链定位

```
复合肥产业链：
磷矿/钾肥/尿素 → 【新洋丰 复合肥】→ 农资渠道 → 农业
                        ↑
                  新洋丰（矿肥一体）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 复合肥销量548万吨，行业前列 |
| **竞争优势** | 矿肥一体、品牌渠道、新型肥占比高 |
| **客户结构** | 县级经销商、种植大户、合作社 |
| **供应商** | 磷矿自给，钾肥、尿素外采 |

### 竞争格局

**主要竞争对手**：
- 云图控股：磷复肥一体化
- 史丹利：复合肥龙头
- 金正大：缓控释肥
- 芭田股份：复合肥、磷矿

**公司优势**：
新洋丰矿肥一体成本优势突出，新型肥料增速快、毛利率高，产品结构优化领先行业。渠道和品牌积淀深，业绩稳定性强。

## 五、投资亮点

### 核心亮点

- ✅ **复合肥龙头**：销量548万吨，行业领先
- ✅ **新型肥高增长**：销量+22.9%，占比31.7%，毛利率更高
- ✅ **矿肥一体**：磷矿自给，成本可控
- ✅ **业绩稳健**：2024年净利+9%，2025H1+29%
- ✅ **渠道品牌**：洋丰、澳特尔品牌知名度高

### 成长驱动因素

1. **新型肥料放量**
   - 缓控释、水溶肥、专用肥需求增长
   - 经济作物、设施农业渗透率提升
   - 毛利率高于传统肥

2. **矿肥一体深化**
   - 磷矿资源保障
   - 成本控制能力强

3. **渠道下沉**
   - 县级网络覆盖
   - 种植大户、合作社直供

### 催化剂

- 农业政策支持
- 新型肥料渗透率提升
- 磷矿价格企稳
- 销量持续高增长

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 新洋丰为成长型复合肥龙头，拐点特征不明显，更侧重**结构性成长**。核心观察：①**新型肥料占比突破**——若新型肥销收占比突破40%，毛利率将显著抬升；②**销量增速拐点**——2025年Q1销量+35%已现高增长，若持续2-3个季度验证，则成长逻辑强化；③**磷矿自给率提升**——若磷矿扩产或收购，成本端进一步优化。**拐点信号**：新型肥占比超40%、单季销量增速连续3季度超20%、毛利率突破18%。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 新型肥放量、销量高增 | 净利润增速15%+ | 净利润增速12%+ |
| **中性** | 稳健增长、结构优化 | 净利润增速8-12% | 净利润增速8-10% |
| **悲观** | 需求疲软、竞争加剧 | 净利润增速<5% | 净利润增速<5% |

**估值分析**：
新洋丰业绩稳健，新型肥料成长逻辑清晰。若新型肥占比持续提升、销量高增长延续，估值有望享受成长溢价。复合肥龙头中业绩确定性较强。
