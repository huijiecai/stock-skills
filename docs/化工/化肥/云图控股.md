## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 成都云图控股股份有限公司 |
| 证券代码 | 002539 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1995年 |
| 上市时间 | 2011年1月25日 |
| 所属行业 | 化学原料和化学制品制造业（复合肥） |
| 市场地位 | 磷复肥一体化龙头，产业链完整 |

**公司概况**：
云图控股是国内磷复肥一体化龙头企业，拥有磷矿-磷酸-磷肥-复合肥完整产业链，同时布局调味品等业务。2024年营收203.81亿元，归母净利润8.04亿元。磷复肥业务营收114.30亿元、同比+13.24%，成为主要支撑；贸易业务大幅下降拖累整体营收。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **磷复肥** | 磷酸一铵、磷酸二铵、复合肥 | 农业肥料 |
| **磷矿** | 磷矿采选 | 磷肥原料、外销 |
| **调味品** | 食用盐、品种盐 | 食品加工、餐饮 |
| **贸易** | 化肥、化工品贸易 | 渠道分销 |

### 业务模式

公司采用"矿化一体+多元经营"模式，磷矿-磷肥-复合肥产业链完整，成本优势突出。调味品业务提供稳定现金流。贸易业务受市场环境影响波动较大。

### 技术路线/核心技术

- **矿化一体**：磷矿-湿法磷酸-磷肥-复合肥
- **产业链完整**：上游资源到终端产品全覆盖
- **多元化**：化肥+调味品双轮驱动

**技术优势**：
磷矿资源保障磷肥成本，一体化布局抗周期能力较强。磷复肥业务在2024年逆势增长13.24%，体现主业韧性。

## 四、行业地位与产业链定位

### 产业链定位

```
磷复肥产业链：
磷矿石 → 湿法磷酸 → 【云图控股 磷肥/复合肥】→ 农业
                        ↑
                  云图控股（矿化一体）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 磷复肥产能行业前列，一体化龙头 |
| **竞争优势** | 矿化一体、产业链完整、成本可控 |
| **客户结构** | 农资经销商、复合肥企业、农业主产区 |
| **供应商** | 磷矿自给，钾肥、尿素外采 |

### 竞争格局

**主要竞争对手**：
- 云天化：磷肥龙头
- 新洋丰：复合肥龙头
- 史丹利：复合肥
- 兴发集团：磷化工

**公司优势**：
云图控股磷复肥一体化程度高，磷矿资源保障成本。调味品业务提供现金流缓冲。2024年磷复肥逆势增长体现主业竞争力。

## 五、投资亮点

### 核心亮点

- ✅ **磷复肥一体化**：矿化一体，成本优势
- ✅ **主业逆势增长**：磷复肥营收+13.24%
- ✅ **产业链完整**：磷矿到复合肥全覆盖
- ✅ **多元化缓冲**：调味品提供稳定现金流
- ✅ **盈利能力改善**：产品结构优化

### 成长驱动因素

1. **磷复肥景气**
   - 磷矿价格高位支撑磷肥盈利
   - 农业需求刚性
   - 一体化成本优势

2. **贸易业务收缩**
   - 主动收缩低毛利贸易，聚焦主业
   - 营收质量提升

3. **调味品稳健**
   - 食用盐刚需，现金流稳定
   - 品种盐差异化

### 催化剂

- 磷矿价格维持
- 磷复肥需求稳健
- 贸易业务出清
- 负债率下降

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 云图控股拐点取决于：①**贸易业务出清**——贸易业务大幅下滑拖累整体，若贸易占比降至低位或剥离，营收质量提升，业绩将更真实反映磷复肥主业；②**磷复肥景气延续**——磷矿价格维持、农业需求稳健，磷复肥+13.24%增速若延续，主业驱动业绩拐点；③**负债率下降**——有息负债增加推高财务费用，若通过经营现金流偿还债务，利润释放。**拐点信号**：磷复肥营收占比超70%、贸易业务同比降幅收窄或企稳、资产负债率降至60%以下。
