## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 深圳市芭田生态工程股份有限公司 |
| 证券代码 | 002170 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1995年 |
| 上市时间 | 2007年9月19日 |
| 所属行业 | 化学原料和化学制品制造业（复合肥） |
| 市场地位 | 复合肥+磷矿，小高寨磷矿扩至200万吨/年 |

**公司概况**：
芭田股份是复合肥企业，拥有贵州小高寨磷矿资源，形成"磷矿-磷肥-复合肥"产业链。2024年营收33.13亿元、同比+2.15%，归母净利润4.09亿元、同比+57.67%，业绩大幅增长主要受益于磷矿石放量。小高寨磷矿规模扩至200万吨/年，磷矿业务成为核心增长点。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **磷矿石** | 磷矿采选，小高寨磷矿200万吨/年 | 磷肥原料、外销 |
| **复合肥** | 氮磷钾复合肥、专用肥 | 农业肥料 |
| **磷肥** | 磷酸等 | 复合肥原料、外销 |

### 业务模式

公司采用"矿肥一体"模式，小高寨磷矿自给并外销。磷矿扩产驱动业绩高增长，2024年磷矿放量是净利润+57.67%的核心驱动力。复合肥业务提供稳定基础。

### 技术路线/核心技术

- **磷矿开采**：小高寨磷矿，规模扩至200万吨/年
- **矿肥一体**：磷矿-磷酸-复合肥
- **贵州资源**：磷矿品位、开采条件

**技术优势**：
芭田股份从复合肥企业转型为"磷矿+复合肥"双轮驱动。磷矿资源稀缺，扩产带来业绩弹性。2024年磷矿放量是业绩高增长的核心。

## 四、行业地位与产业链定位

### 产业链定位

```
磷肥产业链：
磷矿石 → 【芭田股份 磷矿/磷肥/复合肥】→ 农业
                    ↑
              芭田股份（矿肥一体）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 磷矿200万吨/年，复合肥行业中等规模 |
| **竞争优势** | 磷矿资源、矿肥一体、磷矿放量弹性 |
| **客户结构** | 磷肥企业、复合肥企业、农资渠道 |
| **供应商** | 磷矿自给，钾肥、尿素外采 |

### 竞争格局

**主要竞争对手**：
- 云天化：磷肥龙头，磷矿储量巨大
- 兴发集团：磷化工、磷矿
- 川恒股份：磷矿、磷化工
- 云图控股：磷复肥一体化

**公司优势**：
芭田股份磷矿规模虽不及云天化等龙头，但小高寨磷矿扩产带来业绩弹性。从纯复合肥转型矿肥一体，资源价值凸显。2024年净利+57.67%体现磷矿放量威力。

## 五、投资亮点

### 核心亮点

- ✅ **磷矿放量**：小高寨扩至200万吨/年，业绩高增
- ✅ **矿肥一体**：磷矿自给，成本优势
- ✅ **业绩弹性**：2024年净利+57.67%
- ✅ **资源稀缺**：磷矿资源价值凸显
- ✅ **转型成功**：从复合肥到矿肥一体

### 成长驱动因素

1. **磷矿持续放量**
   - 200万吨/年产能爬坡
   - 磷矿价格高位，利润贡献大
   - 外销+自用双轮驱动

2. **矿肥一体深化**
   - 磷矿-磷肥-复合肥协同
   - 成本控制

3. **磷矿景气**
   - 磷矿价格维持
   - 供给紧张

### 催化剂

- 磷矿价格维持高位
- 小高寨磷矿满产
- 磷矿外销放量
- 复合肥需求稳健

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 芭田股份2024年已实现**业绩拐点**（净利+57.67%），拐点由磷矿放量驱动。下一阶段拐点取决于：①**磷矿满产**——200万吨/年若满产运行，量增逻辑延续；②**磷矿价格维持**——磷矿价格高位是利润核心，若维持则业绩高增可持续；③**复合肥拐点**——营收增速仅+2.15%，若复合肥业务企稳或增长，营收端弹性增强。**拐点信号**：磷矿销量环比持续增长、磷矿均价维持、复合肥营收同比转正或加速。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
