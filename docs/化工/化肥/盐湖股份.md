## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 青海盐湖工业股份有限公司 |
| 证券代码 | 000792 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1958年 |
| 上市时间 | 1997年9月4日 |
| 所属行业 | 化学原料和化学制品制造业（钾肥） |
| 市场地位 | 国内钾肥龙头，氯化钾产能占全国87.3% |

**公司概况**：
盐湖股份是国内最大的钾肥生产企业，氯化钾产能500万吨/年，占全国总产量87.3%。公司依托察尔汗盐湖资源，拥有"钾肥+锂盐"双主业。钾肥业务成本极低、盈利稳定；碳酸锂业务依托盐湖提锂技术，具备成本优势。2024年受钾肥、碳酸锂价格下行影响业绩承压，2025年Q1预计净利润同比增20%-28%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **氯化钾** | 钾肥核心品种 | 农业肥料、复合肥原料 |
| **碳酸锂** | 盐湖提锂 | 动力电池、储能 |
| **其他** | 氢氧化钾、金属镁等 | 化工、冶金 |

### 业务模式

公司采用"盐湖资源-钾肥/锂盐"一体化模式，依托察尔汗盐湖得天独厚的卤水资源，钾肥生产成本远低于海外进口。碳酸锂采用吸附法盐湖提锂工艺，成本优势显著。钾肥以国内销售为主，碳酸锂面向动力电池产业链。

### 技术路线/核心技术

- **钾肥开采**：察尔汗盐湖卤水蒸发结晶，成本极低
- **盐湖提锂**：吸附法工艺，4万吨/年基础锂盐一体化项目推进中
- **资源禀赋**：察尔汗盐湖氯化钾储量约5.4亿吨，锂资源丰富

**技术优势**：
国内钾肥资源稀缺，进口依存度约50%。盐湖股份掌握国内最大钾肥产能，资源税补缴后成本仍具竞争力。盐湖提锂成本较锂云母、锂辉石路线更低。

## 四、行业地位与产业链定位

### 产业链定位

```
钾肥产业链：
盐湖卤水 → 【盐湖股份 氯化钾】→ 复合肥/直接施用 → 农业
                    ↑
              盐湖股份（资源垄断）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 氯化钾产能500万吨/年，占全国87.3% |
| **竞争优势** | 察尔汗盐湖资源垄断、成本极低、国内钾肥之王 |
| **客户结构** | 复合肥企业、农资经销商、农业主产区 |
| **供应商** | 盐湖资源自给，无上游依赖 |

### 竞争格局

**主要竞争对手**：
- 亚钾国际：老挝钾盐，海外扩产
- 藏格矿业：国内盐湖提钾
- 国际：加拿大钾肥、白俄罗斯钾肥等

**公司优势**：
盐湖股份是国内钾肥绝对龙头，资源禀赋独一无二。进口钾肥受地缘政治、海运影响，公司国内供应保障能力强。钾肥需求刚性，长期看资源价值凸显。

## 五、投资亮点

### 核心亮点

- ✅ **国内钾肥之王**：产能占全国87.3%，资源垄断
- ✅ **成本极低**：盐湖卤水开采，成本远低于进口
- ✅ **盐湖提锂**：4万吨锂盐项目2025年建成，成本优势
- ✅ **业绩拐点**：2025年Q1预计净利同比+20%-28%
- ✅ **粮食安全**：钾肥关乎粮食安全，政策支持

### 成长驱动因素

1. **钾肥价格企稳**
   - 全球粮价支撑化肥需求
   - 进口钾肥供应不确定性
   - 国内钾肥定价权

2. **锂盐产能释放**
   - 4万吨基础锂盐项目2025年建成
   - 盐湖提锂成本优势
   - 锂价若企稳，贡献增量

3. **资源税影响消化**
   - 2024年一次性补缴后，后续影响减弱

### 催化剂

- 钾肥价格企稳回升
- 碳酸锂价格反弹
- 4万吨锂盐项目投产
- 粮食安全政策加码

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 盐湖股份业绩拐点取决于：①**钾肥价格企稳**——全球粮价、进口供应、国内需求共同决定，若钾肥价格止跌企稳，公司毛利率将显著改善；②**碳酸锂价格反弹**——锂价若从底部回升，锂盐业务从拖累转为贡献；③**锂盐产能释放**——4万吨项目2025年建成，量增对冲价跌；④**资源税一次性影响消化**——2024年补缴后，2025年同比基数正常化。**拐点信号**：钾肥均价环比止跌、碳酸锂价格企稳、Q1业绩预告兑现。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 钾肥、锂价双企稳，锂盐放量 | 净利润增速15%+ | 净利润增速10%+ |
| **中性** | 钾肥企稳、锂价底部震荡 | 净利润增速0-10% | 净利润增速5-10% |
| **悲观** | 钾肥、锂价继续下行 | 净利润增速<0% | 净利润增速<5% |

### 估值水平

**估值分析**：
盐湖股份作为钾肥龙头，资源稀缺性赋予估值溢价。2024年业绩承压导致PE抬升，若2025年钾肥企稳、锂盐放量，业绩拐点兑现后估值有望修复。长期看资源价值凸显，具备配置价值。
