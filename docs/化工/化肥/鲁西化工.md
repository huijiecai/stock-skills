## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 鲁西化工集团股份有限公司 |
| 证券代码 | 000830 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 1998年8月7日 |
| 所属行业 | 化学原料和化学制品制造业（化肥+化工） |
| 市场地位 | 煤化工+化肥一体化，中化集团控股 |

**公司概况**：
鲁西化工是山东省大型煤化工企业，产品涵盖化肥、化工新材料、基础化工等。2020年加入中化集团，成为中化能源板块重要成员。拥有"煤-合成气-甲醇-化肥"及"煤-焦化-化工"等多条产业链，尿素、复合肥、己内酰胺、PC等产品产能行业前列。化肥与化工双轮驱动，周期互补。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **化肥** | 尿素、复合肥、磷酸二铵 | 农业肥料 |
| **化工新材料** | 己内酰胺、尼龙6、PC | 纺织、工程塑料 |
| **基础化工** | 甲醇、烧碱、双氧水 | 化工原料 |
| **其他** | 甲酸、DMF等 | 精细化工 |

### 业务模式

公司采用"煤化工+化肥"一体化模式，以煤炭为原料，通过煤气化生产合成气，延伸至甲醇、尿素、己内酰胺、PC等。产业链长，产品多元，周期互补。中化入主后管理、采购、销售协同增强。

### 技术路线/核心技术

- **煤气化**：水煤浆气化、粉煤气化
- **煤-化肥**：合成气-甲醇-尿素
- **煤-化工**：己内酰胺、PC、尼龙6

**技术优势**：
鲁西化工煤化工技术成熟，产业链完整。中化入主后获得资金、管理、渠道支持。化肥与化工双轮驱动，分散单一产品周期风险。

## 四、行业地位与产业链定位

### 产业链定位

```
煤化工产业链：
煤炭 → 煤气化 → 合成气 → 【鲁西化工 化肥/化工】→ 农业/纺织/塑料
                              ↑
                        鲁西化工（煤化一体）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 尿素、己内酰胺、PC产能行业前列 |
| **竞争优势** | 煤化工一体、中化控股、产品多元 |
| **客户结构** | 农业、纺织、塑料、化工等 |
| **供应商** | 煤炭采购，中化渠道协同 |

### 竞争格局

**主要竞争对手**：
- 华鲁恒升：煤化工龙头
- 云天化：磷肥龙头
- 恒力石化：炼化一体化
- 万华化学：化工龙头

**公司优势**：
鲁西化工煤化工产业链完整，化肥+化工双轮驱动。中化入主后资源整合，长期竞争力增强。山东化工大省，区位优势明显。

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 鲁西化工拐点取决于多产品周期共振：①**化肥拐点**——尿素、磷肥价格若企稳回升，化肥板块利润修复；②**化工拐点**——己内酰胺、PC若供需改善、价格上行，化工板块贡献增量；③**成本拐点**——煤炭价格若回落，煤化工成本端改善；④**中化协同拐点**——入主后协同效应若在2-3年内显著体现，管理效率、采购成本、销售渠道全面优化。**拐点信号**：尿素/己内酰胺价格环比止跌、单季净利润同比转正、毛利率环比改善。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 化肥+化工双景气、煤炭回落 | 净利润增速20%+ | 净利润增速15%+ |
| **中性** | 部分产品企稳、中化协同 | 净利润增速0-15% | 净利润增速5-15% |
| **悲观** | 多产品价格下行、成本高企 | 净利润增速<0% | 净利润增速<5% |

| PE(TTM) | - | 煤化工约10-15倍 | - |

鲁西化工煤化工一体、中化控股，长期具备配置价值。周期底部布局需等待化肥、化工价格拐点。双轮驱动分散风险，若多产品景气共振，业绩弹性大。
