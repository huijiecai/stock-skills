## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 史丹利农业集团股份有限公司 |
| 证券代码 | 002588 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1992年 |
| 上市时间 | 2011年6月10日 |
| 所属行业 | 化学原料和化学制品制造业（复合肥） |
| 市场地位 | 复合肥龙头，营收破百亿，连续两年双位数净利增长 |

**公司概况**：
史丹利是国内复合肥行业龙头企业，2024年营业收入首次突破百亿达102.63亿元，归母净利润8.26亿元、同比+17.89%，连续两年保持双位数增长。复合肥销量334.64万吨、同比+12.68%，毛利率从17.20%提升至18.59%。品牌"史丹利"知名度高，渠道覆盖全国。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **复合肥** | 氮磷钾复合肥、专用肥 | 大田作物、经济作物 |
| **新型肥料** | 缓控释肥、水溶肥 | 经济作物、设施农业 |
| **原料贸易** | 磷肥、钾肥等 | 渠道分销 |

### 业务模式

公司采用"品牌+渠道"模式，史丹利品牌在农资领域知名度高。渠道下沉至县级经销商，与种植大户、合作社合作。原料外采，通过规模采购和供应链管理控制成本。

### 技术路线/核心技术

- **产品研发**：专用肥、新型肥持续推出
- **渠道管理**：县级经销商网络、农化服务
- **品牌营销**：央视等媒体投放，品牌溢价

**技术优势**：
史丹利品牌价值高，渠道掌控力强。毛利率持续提升反映产品结构优化和定价能力。无磷矿资源，但通过规模采购和供应链优化控制成本。

## 四、行业地位与产业链定位

### 产业链定位

```
复合肥产业链：
磷肥/钾肥/尿素 → 【史丹利 复合肥】→ 农资渠道 → 农业
                        ↑
                  史丹利（品牌渠道）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 复合肥销量334万吨，行业前列 |
| **竞争优势** | 品牌知名度高、渠道覆盖广、毛利率提升 |
| **客户结构** | 县级经销商、种植大户、合作社 |
| **供应商** | 磷肥、钾肥、尿素外采，无上游资源 |

### 竞争格局

**主要竞争对手**：
- 新洋丰：矿肥一体、复合肥龙头
- 云图控股：磷复肥一体化
- 金正大：缓控释肥
- 芭田股份：复合肥、磷矿

**公司优势**：
史丹利虽无磷矿资源，但品牌和渠道优势突出。毛利率持续提升反映产品结构优化和定价能力。2024年净利增速17.89%在磷复肥龙头中领先，业绩弹性强。

3. **规模效应**

- 农业政策支持
- 复合肥需求稳健
- 毛利率继续提升

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 史丹利已处于业绩上行通道，拐点更多体现为**增速能否维持**。核心观察：①**毛利率拐点**——毛利率从17.20%提升至18.59%，若突破20%则盈利弹性显著；②**销量增速拐点**——+12.68%增速若维持2-3年，规模效应将进一步释放；③**负债拐点**——有息负债大增推高财务费用，若经营现金流改善、负债率下降，利润释放。**拐点信号**：毛利率突破20%、单季销量增速超15%、有息负债增速放缓或下降。
