## 一、行业定义与范围

### 是什么？

> **化工行业 = 将原油、天然气、煤炭等基础原料，通过化学反应转化为各种化学品的产业**
>
> 化工行业是国民经济的基础产业，涵盖从基础化学品到精细化学品、新材料的完整产业链，产品广泛应用于纺织、汽车、建筑、电子、新能源等领域。

| 细分领域 | 说明 | 典型应用 |
|----------|------|----------|
| **聚氨酯** | MDI、TDI等异氰酸酯产品 | 保温材料、汽车内饰、家具 |
| **氟化工** | 制冷剂、含氟材料 | 空调制冷、锂电池、半导体 |
| **磷化工** | 磷肥、磷酸、磷酸铁锂 | 农业、新能源电池 |
| **染料化工** | 分散染料、还原染料 | 纺织印染、涂料 |
| **玻纤** | 玻璃纤维及制品 | 复合材料、新能源、建筑 |
| **煤化工** | 醋酸、DMF、甲醇 | 化工原料、能源 |
| **炼化一体化** | PTA、聚酯、烯烃 | 纺织、包装、汽车 |
| **有机硅** | 硅油、硅橡胶、硅树脂 | 电子、建筑、汽车 |
| **新材料** | 高端聚烯烃、电子化学品 | 半导体、新能源、5G |

### 产业链定位

```
上游：原油、天然气、煤炭
    ↓
【中游：基础化工】
乙烯、丙烯、苯、甲醇等基础有机化学品
    ↓
【中游：精细化工】
聚乙烯、聚丙烯、聚酯、染料、农药等
    ↓
【下游：新材料】
高端聚烯烃、电子化学品、功能性膜材料
    ↓
终端应用：纺织、汽车、建筑、电子、新能源
```

**说明**：
化工行业位于产业链中游，上承能源原料，下启终端制造。基础化工将原料转化为基础单体，精细化工进一步加工为功能性产品，新材料则代表高附加值的发展方向。

### 增长驱动因素

1. **"反内卷"政策推动行业复苏**
   - 2025年Q3在建工程同比增速由正转负，本轮扩产周期接近尾声
   - 行业龙头通过联合减产、自律控产优化供需格局
   - 政策引导落后产能出清，行业集中度提升

2. **新材料打开成长空间**
   - 高性能聚烯烃（POE）、芳纶、超高分子量聚乙烯纤维实现国产化
   - 电子化学品、半导体材料、OLED材料等高端产品需求旺盛
   - 2026年多个新材料项目投产，打开高成长空间

3. **下游需求复苏**
   - 汽车、化纤、新能源等下游行业需求持续向好
   - AI液冷、储能电池等新兴领域带动氟化工、磷化工需求增长
   - 全球央行降息周期和财政刺激推动需求复苏

4. **成本优势凸显**
   - 2026年全球原油供大于求，布伦特油价中枢预计50-70美元/桶
   - 低油价有利于下游化工企业加工利润恢复
   - 煤化工、炼化一体化企业成本优势进一步强化

### 关键环节分析

**上游炼化环节**：
- 原油、天然气通过炼化产生乙烯、丙烯等"七大烯烃"
- 煤炭通过煤化工产生甲醇、醋酸、DMF等产品
- 成本受油价、煤价影响大，2026年低油价利好下游

**中游精细化工**：
- 将基础单体通过聚合、共聚、改性等工艺生产功能性产品
- 技术壁垒较高，行业集中度提升明显
- 龙头企业通过一体化布局降低成本

**下游新材料**：
- 高附加值产品，毛利率通常30%+
- 国产替代空间大，政策支持力度强
- 2026年多个新材料项目投产，打开成长空间

## 六、投资逻辑与风险

### 投资逻辑

**1. 行业周期拐点已近**
- 2025年Q3在建工程同比增速由正转负，本轮扩产周期接近尾声
- 111个化工品种中70种价格处于历史低位，向上弹性大
- 行业利润降幅从2023年-20.7%收窄至2024年-8.8%，拐点信号明确

**2. "反内卷"政策催化**
- 行业龙头通过联合减产、自律控产优化供需格局
- 政策引导落后产能出清，行业集中度提升
- PTA、长丝、有机硅、纯碱等细分领域率先受益

**3. 新材料打开成长空间**
- POE、己二腈、电子级磷酸等高端产品实现国产化
- AI液冷、储能电池等新兴领域带动需求爆发
- 2026年多个新材料项目投产，业绩增长确定性强

**4. 低估值+业绩见底**
- 化工龙头PE普遍低于历史中位数
- 市场增量资金优先配置化工ETF权重股
- 业绩基本见底，向上拐点明确

**5. 成本优势凸显**
- 2026年布伦特油价中枢预计50-70美元/桶，低油价利好下游
- 煤化工企业煤制烯烃成本优势进一步强化
- 一体化布局企业抗周期能力强

### 风险提示

**1. 宏观经济风险**
- 全球经济衰退导致需求下滑
- 终端消费疲软影响下游需求
- 贸易摩擦加剧出口受限

**2. 产能过剩风险**
- 部分细分领域产能扩张超预期
- "反内卷"执行力度不及预期
- 行业价格战卷土重来

**3. 原料价格波动**
- 油价大幅上涨侵蚀利润
- 煤炭价格上涨影响煤化工企业
- 天然气供应紧张推高成本

**4. 政策监管风险**
- 环保政策趋严增加合规成本
- 安全生产事故导致停产整顿
- 碳中和政策影响高耗能企业

**5. 技术迭代风险**
- 新技术路线替代传统产品
- 国际巨头技术封锁
- 国产替代进度不及预期

### 2026年投资策略

**配置思路**：稳健配置（龙头）+ 涨价品种（主题）

**核心持仓（60%）**：
