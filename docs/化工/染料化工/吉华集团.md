## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江吉华集团股份有限公司 |
| 证券代码 | 603980 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2003年 |
| 上市时间 | 2017年6月15日 |
| 所属行业 | 化学原料和化学制品制造业（染料化工） |
| 市场地位 | 分散染料重要企业，活性染料有布局 |

**公司概况**：
吉华集团是国内染料行业重要企业，分散染料产能约7万吨/年，活性染料亦有布局。公司位于浙江杭州湾，与浙江龙盛、闰土股份同处染料产业集群。染料下游为纺织印染，与服装出口、内需相关。2024年纺织需求波动、染料价格影响业绩。染料行业集中度高，龙盛、闰土为双寡头，吉华为重要跟随者，若纺织拐点来临有望受益。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **分散染料** | 产能约7万吨/年 | 纺织印染 |
| **活性染料** | 棉麻等纤维素纤维染色 | 纺织印染 |
| **染料中间体** | H酸、对位酯等 | 染料生产 |
| **硫化染料** | 传统染料 | 纺织印染 |
| **其他化工** | 配套产品 | 化工 |

### 业务模式

公司采用"染料+中间体"模式，分散染料为核心，活性染料、硫化染料补充。中间体部分自给。下游纺织印染与服装出口、内需相关。与龙盛、闰土同处浙江染料产业集群。

### 技术路线/核心技术

- **分散染料**：涤纶染色用染料
- **活性染料**：棉麻染色
- **中间体**：H酸、对位酯等
- **环保**：染料废水处理

**技术优势**：
染料技术成熟，产品品质稳定。中间体部分自给。浙江染料产业集群，配套完善。规模较龙盛、闰土小，成本控制需加强。

## 四、行业地位与产业链定位

### 产业链定位

```
染料产业链：
中间体 → 【吉华集团 分散染料/活性染料】→ 纺织印染 → 服装
                    ↑
                吉华集团（重要企业）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 分散染料约7万吨，重要企业 |
| **竞争优势** | 浙江产业集群、产品多元 |
| **客户结构** | 纺织印染企业 |
| **供应商** | 中间体部分自给 |

### 竞争格局

**主要竞争对手**：
- 浙江龙盛：分散染料龙头，中间体优势强
- 闰土股份：分散染料龙头，与龙盛双寡头
- 亚邦股份：染料、颜料

**公司优势**：
染料重要企业，产品多元。浙江产业集群，配套完善。规模较龙头小，需关注成本控制、差异化竞争。

## 五、投资亮点

### 核心亮点

- ✅ **染料重要企业**：分散染料约7万吨
- ✅ **拐点逻辑**：纺织出口若复苏，染料需求改善；染料价格若企稳，行业受益
- ✅ **浙江产业集群**：杭州湾，配套完善
- ✅ **产品多元**：分散、活性、硫化染料
- ⚠️ **规模**：较龙盛、闰土小，弹性相对有限

### 成长驱动因素

1. **纺织拐点**
   - 纺织服装出口若复苏
   - 染料需求边际改善
   - 行业整体受益

2. **染料价格**
   - 染料价格若企稳反弹
   - 龙头提价，跟随者受益

3. **差异化**
   - 活性染料、特种染料
   - 细分市场拓展

### 催化剂

- **纺织出口**：服装出口数据改善
- **染料价格**：触底反弹
- **环保**：小产能退出
- **内需**：服装消费复苏

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **需求拐点** | 纺织出口若复苏，染料需求改善 | 服装出口 |
| **价格拐点** | 染料价格若企稳，行业受益 | 供需改善 |
| **业绩拐点** | 规模较小，弹性相对有限 | 染料价涨、需求复苏 |
