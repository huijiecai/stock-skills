## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江闰土股份有限公司 |
| 证券代码 | 002440 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1996年 |
| 上市时间 | 2010年7月6日 |
| 所属行业 | 化学原料和化学制品制造业（染料化工） |
| 市场地位 | 分散染料龙头之一，与浙江龙盛并列产能最大 |

**公司概况**：
闰土股份是国内分散染料龙头之一，与浙江龙盛并列为中国分散染料行业产能最大的两家企业。公司分散染料产能约14万吨/年，活性染料、染料中间体亦有布局。染料下游为纺织印染，与服装出口、内需消费相关。2024年纺织出口波动、染料价格影响业绩。染料行业集中度高，龙头具备定价权，若纺织需求复苏拐点逻辑明确。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **分散染料** | 涤纶等化纤染色，产能约14万吨/年 | 纺织印染 |
| **活性染料** | 棉麻等纤维素纤维染色 | 纺织印染 |
| **染料中间体** | H酸、对位酯等 | 染料生产 |
| **助剂** | 纺织印染助剂 | 纺织印染 |
| **保险粉** | 还原剂 | 纺织、造纸 |

### 业务模式

公司采用"染料+中间体"一体化模式，分散染料为核心，活性染料、中间体补充。与浙江龙盛共同主导分散染料市场，行业集中度高。下游纺织印染与服装出口、内需相关。产品销往国内外。

### 技术路线/核心技术

- **分散染料**：蒽醌系、偶氮系等全系列
- **中间体**：H酸、对位酯等自给
- **一体化**：从中间体到染料
- **环保**：染料废水处理、清洁生产

**技术优势**：
分散染料产能与浙江龙盛并列第一，行业双寡头。中间体自给降低成本。染料技术成熟，产品品质稳定。

## 四、行业地位与产业链定位

### 产业链定位

```
染料产业链：
苯/硝基苯 → 中间体 → 【闰土股份 分散染料/活性染料】→ 纺织印染 → 服装
                        ↑
                    闰土股份（双寡头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 分散染料与浙江龙盛并列产能最大 |
| **竞争优势** | 规模、中间体自给、双寡头 |
| **客户结构** | 纺织印染企业，国内外 |
| **供应商** | 中间体自给，苯等外采 |

### 竞争格局

**主要竞争对手**：
- 浙江龙盛：分散染料龙头，中间体优势更强
- 吉华集团：分散染料重要企业
- 亚邦股份：染料、颜料

**公司优势**：
分散染料与龙盛双寡头，行业集中度高。中间体自给，成本优势。染料技术成熟，产品品质稳定。

## 五、投资亮点

### 核心亮点

- ✅ **分散染料龙头**：与浙江龙盛并列产能最大
- ✅ **拐点逻辑**：纺织出口若复苏，染料需求改善；染料价格若企稳反弹，龙头弹性大
- ✅ **双寡头格局**：行业集中度高，定价权
- ✅ **中间体自给**：成本优势
- ✅ **一体化**：染料+中间体

### 成长驱动因素

1. **纺织拐点**
   - 纺织服装出口若复苏
   - 内需消费若改善
   - 染料需求边际改善

2. **染料价格**
   - 染料价格若企稳反弹
   - 双寡头定价权
   - 龙头弹性大

3. **中间体**
   - 中间体自给降本
   - 环保趋严，小产能退出
   - 龙头市占率提升

### 催化剂

- **纺织出口**：服装出口数据改善
- **染料价格**：触底反弹
- **环保**：小产能退出，集中度提升
- **内需**：服装消费复苏

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **需求拐点** | 纺织出口若复苏，染料需求改善 | 服装出口 |
| **价格拐点** | 染料价格若企稳反弹，双寡头弹性大 | 供需改善 |
| **业绩拐点** | 龙头成本优势，拐点来临弹性大 | 染料价涨、需求复苏 |
