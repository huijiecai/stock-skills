## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江龙盛集团股份有限公司 |
| 证券代码 | 600352 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1970年 |
| 上市时间 | 2003年8月1日 |
| 所属行业 | 化学原料和化学制品制造业（染料化工） |
| 市场地位 | 全球分散染料龙头，分散染料市占率23.9%国内第一，染料总产能约30万吨/年 |

**公司概况**：
浙江龙盛是全球染料与纺织化学品龙头，主营以染料、助剂为主的纺织用化学品和以间苯二胺、间苯二酚为主的中间体业务。公司分散染料市占率23.9%国内第一，间苯二胺、间苯二酚分别占全球有效产能的65%和51%，形成显著的一体化壁垒。2024年归母净利润20.3亿元，同比+32.36%，经营现金流同比+236.79%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **分散染料** | 涤纶等化纤染色用染料，产能14万吨/年 | 纺织印染 |
| **活性染料** | 棉麻等纤维素纤维染色，产能8.8万吨/年 | 纺织印染 |
| **间苯二胺** | 染料中间体，市占率约80% | 染料、芳纶 |
| **间苯二酚** | 染料中间体，全球产能51% | 染料、橡胶、木材 |
| **助剂** | 纺织印染助剂 | 纺织印染 |

### 业务模式

公司采用"染料+中间体"一体化模式，通过控制间苯二胺、间苯二酚等核心中间体，形成从原料到染料的完整产业链。染料总产能约30万吨/年，与闰土股份并列为中国分散染料行业产能最大的两家企业。产品销往全球，客户覆盖纺织印染头部企业。

### 技术路线/核心技术

- **分散染料合成**：蒽醌系、偶氮系等分散染料全系列
- **间苯二胺/间苯二酚**：核心中间体自给，全球产能占比65%、51%
- **一体化产业链**：从苯、硝基苯到染料的全链条控制
- **环保工艺**：染料废水处理、清洁生产技术

**技术优势**：
浙江龙盛间苯二胺市场占有率达80%，间苯二胺和间苯二酚分别占全球有效产能的65%和51%，形成显著的技术壁垒和成本优势。分散染料持续保持全球第一的市场地位。

## 四、行业地位与产业链定位

### 产业链定位

```
染料化工产业链：
苯/硝基苯 → 间苯二胺/间苯二酚 → 【浙江龙盛 分散染料/活性染料】→ 纺织印染 → 服装/家纺
                                    ↑
                               浙江龙盛
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 分散染料市占率23.9%国内第一，染料总产能约30万吨/年 |
| **竞争优势** | 间苯二胺/间苯二酚全球产能65%/51%，一体化成本优势 |
| **客户结构** | 覆盖国内纺织印染头部企业，产品出口全球 |
| **供应商** | 核心中间体自给，上游依赖度低 |

### 竞争格局

**主要竞争对手**：
- 闰土股份：分散染料市占率20.2%，产能与龙盛相当
- 吉华集团：分散染料第三梯队
- 亚邦股份：蒽醌结构分散染料龙头
- 国际：亨斯迈、德司达等

**公司优势**：
浙江龙盛分散染料市占率23.9%国内第一，远高于第二名闰土股份的20.2%。间苯二胺、间苯二酚全球产能占比65%、51%，中间体一体化壁垒显著。中国分散染料行业CR3为57%，CR5为74%，公司稳居第一梯队。

## 五、投资亮点

### 核心亮点

- ✅ **全球分散染料龙头**：市占率23.9%国内第一，染料总产能约30万吨/年
- ✅ **中间体一体化**：间苯二胺全球产能65%、间苯二酚51%，成本与供应壁垒突出
- ✅ **2024年业绩高增**：归母净利润+32.36%，经营现金流+236.79%
- ✅ **现金流大幅改善**：经营现金流92.65亿元，货币资金增65.75%
- ✅ **分红回报**：拟每10股派现2.5元（含税）

### 成长驱动因素

1. **染料行业景气修复**
   - 纺织服装需求复苏
   - 印染订单回暖
   - 产品价格企稳

2. **中间体壁垒巩固**
   - 间苯二胺、间苯二酚全球产能集中
   - 一体化成本优势持续
   - 环保趋严淘汰中小产能

3. **财务结构优化**
   - 长期借款大幅压降
   - 现金流改善支撑分红与投资
   - 存货管理优化空间

### 催化剂

- 纺织服装出口回暖
- 染料价格企稳回升
- 中间体供应紧张推升染料价格
- 环保政策淘汰落后产能
