## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏亚邦染料股份有限公司 |
| 证券代码 | 603188 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 2014年9月9日 |
| 所属行业 | 化学原料和化学制品制造业（染料化工） |
| 市场地位 | 国内蒽醌系列染料及中间体生产基地，蒽醌结构分散染料龙头 |

**公司概况**：
亚邦股份是国内蒽醌系列染料及中间体的生产基地，主要生产经营染料及染料中间体。公司蒽醌结构分散染料在高端领域具有技术优势，产品应用于化纤、棉纺等印染领域。2024年染料行业面临下游需求转淡、原料价格居高、产品价格下跌等压力，公司通过优化产品结构、拓宽销售渠道、控制成本等措施，上半年实现扭亏为盈，全年亏损幅度同比收窄。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **蒽醌分散染料** | 蒽醌结构分散染料，高端品种 | 化纤印染 |
| **分散染料** | 偶氮系等分散染料 | 涤纶印染 |
| **染料中间体** | 蒽醌、溴氨酸等中间体 | 染料合成 |
| **农药** | 农药生产、销售 | 农业 |

### 业务模式

公司采用"中间体+染料"一体化模式，以蒽醌系列染料及中间体为核心，产品定位中高端。通过自产中间体降低成本，产品销往国内纺织印染企业。2024年采取优化产品结构、拓宽销售渠道、积极控制成本等措施应对行业压力。

### 技术路线/核心技术

- **蒽醌分散染料合成**：蒽醌结构分散染料技术壁垒较高
- **染料中间体**：蒽醌、溴氨酸等中间体自产
- **高端染料**：适用于高端化纤、特种纤维染色
- **农药**：农药制剂生产

**技术优势**：
亚邦股份是国内蒽醌系列染料及中间体的生产基地，蒽醌结构分散染料在高端领域具有技术优势，与浙江龙盛、闰土股份的偶氮系分散染料形成差异化竞争。

## 四、行业地位与产业链定位

### 产业链定位

```
染料化工产业链：
蒽醌/溴氨酸 → 【亚邦股份 蒽醌分散染料】→ 化纤印染 → 服装/家纺
                    ↑
               亚邦股份
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 蒽醌结构分散染料龙头，国内蒽醌染料及中间体生产基地 |
| **竞争优势** | 蒽醌染料技术壁垒高，高端领域差异化 |
| **客户结构** | 覆盖化纤印染企业，产品定位中高端 |
| **供应商** | 中间体部分自给 |

### 竞争格局

**主要竞争对手**：
- 浙江龙盛：分散染料全球龙头，市占率23.9%
- 闰土股份：分散染料市占率20.2%
- 吉华集团：分散染料第三梯队
- 其他蒽醌染料企业

**公司优势**：
亚邦股份在蒽醌结构分散染料领域具有技术优势，与浙江龙盛、闰土股份的偶氮系分散染料形成差异化。蒽醌染料适用于高端化纤、特种纤维，细分市场有定价权。但公司规模较小，抗风险能力弱于龙头。

## 五、投资亮点

### 核心亮点

- ✅ **蒽醌分散染料龙头**：国内蒽醌系列染料及中间体生产基地，技术壁垒较高
- ✅ **高端差异化**：蒽醌染料适用于高端化纤，与偶氮系形成差异化竞争
- ✅ **亏损收窄**：2024年亏损幅度同比收窄54.21%，经营改善
- ✅ **成本控制**：优化产品结构、拓宽渠道、控制成本初见成效
- ✅ **扭亏潜力**：上半年曾实现扭亏，具备盈利修复空间

### 成长驱动因素

1. **染料行业景气修复**
   - 纺织印染需求回暖
   - 蒽醌染料高端需求增长
   - 产品价格企稳

2. **成本与运营优化**
   - 产品结构优化
   - 销售渠道拓宽
   - 成本控制持续

3. **差异化竞争**
   - 蒽醌染料技术壁垒
   - 高端客户粘性
   - 细分市场定价权

### 催化剂

- 纺织服装需求复苏
- 蒽醌染料价格回升
- 产能整合、环保淘汰中小产能
- 公司扭亏为盈
