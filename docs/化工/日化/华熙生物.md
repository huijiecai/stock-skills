## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 华熙生物科技股份有限公司 |
| 证券代码 | 688363 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2000年 |
| 上市时间 | 2019年11月6日 |
| 所属行业 | 化学原料和化学制品制造业/生物科技（透明质酸） |
| 市场地位 | 全球透明质酸龙头，原料+终端全产业链，玻尿酸市占率全球第一 |

**公司概况**：
华熙生物是全球透明质酸（玻尿酸）龙头，透明质酸原料市占率全球第一。公司形成"原料+医疗终端+功能性护肤品"三驾马车，从玻尿酸原料到润百颜、夸迪、米蓓尔等终端品牌全覆盖。2024年医美监管趋严、护肤品竞争加剧，公司推进原料业务增长、终端品牌优化。透明质酸技术壁垒高，原料龙头地位稳固，若终端品牌改善、医美复苏，拐点逻辑明确。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **透明质酸原料** | 医药级、化妆品级、食品级，全球市占率第一 | 医美、护肤、食品、医药 |
| **功能性护肤品** | 润百颜、夸迪、米蓓尔、BM肌活 | 护肤、次抛精华 |
| **医疗终端** | 玻尿酸注射填充、骨科、眼科 | 医美、医疗 |
| **功能性食品** | 玻尿酸食品 | 口服美容 |
| **其他生物活性物** | 依克多因、麦角硫因等 | 护肤原料 |

### 业务模式

公司采用"原料+医疗终端+功能性护肤品"三驾马车模式。原料为基盘，全球龙头；医疗终端为医美注射、骨科眼科等；功能性护肤品为润百颜、夸迪等品牌。从原料到终端全产业链，技术壁垒高。

### 技术路线/核心技术

- **透明质酸发酵**：微生物发酵，技术全球领先
- **梯度3D交联**：玻尿酸注射剂技术
- **次抛精华**：润百颜、夸迪等
- **生物活性物**：依克多因、麦角硫因等

**技术优势**：
透明质酸原料全球龙头，市占率第一。发酵技术、纯度、分子量控制领先。原料-终端全产业链，技术壁垒高。生物活性物拓展，依克多因等。

## 五、投资亮点

### 核心亮点

- ✅ **透明质酸全球龙头**：原料市占率全球第一
- ✅ **拐点逻辑**：原料业务稳健；功能性护肤品若产品创新、渠道优化见效；医美若监管缓和、需求复苏
- ✅ **全产业链**：原料+医疗终端+护肤品
- ✅ **技术壁垒**：发酵技术、纯度、分子量控制领先
- ✅ **生物活性物**：依克多因、麦角硫因等拓展

### 成长驱动因素

1. **原料**
   - 透明质酸原料全球龙头，稳健
   - 食品级、化妆品级需求增长
   - 出口、国内双驱动

2. **功能性护肤品**
   - 润百颜、夸迪、米蓓尔、BM肌活
   - 次抛精华差异化
   - 若产品创新、渠道优化见效

3. **医美**
   - 医美监管若缓和
   - 玻尿酸注射需求
   - 医疗终端恢复

4. **生物活性物**
   - 依克多因、麦角硫因等
   - 原料拓展
   - 成长性

### 催化剂

- **原料**：透明质酸需求增长
- **护肤品**：润百颜、夸迪新品、放量
- **医美**：监管缓和、需求复苏
- **生物活性物**：依克多因等放量

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **原料拐点** | 原料龙头，稳健增长 | 透明质酸需求 |
| **护肤品拐点** | 润百颜、夸迪若产品创新、渠道优化见效 | 新品、渠道 |
| **医美拐点** | 医美若监管缓和、需求复苏 | 医美复苏 |
| **业绩拐点** | 原料稳健+护肤品若改善+医美若复苏 | 三驾马车协同 |
