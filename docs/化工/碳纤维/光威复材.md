## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 威海光威复合材料股份有限公司 |
| 证券代码 | 300699 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1992年 |
| 上市时间 | 2017年9月1日 |
| 所属行业 | 化学纤维制造业（碳纤维） |
| 市场地位 | 国产碳纤维龙头，航空航天碳纤维核心供应商 |

**公司概况**：
光威复材是国内碳纤维行业龙头，产品覆盖T300/T700/T800/T1000等全系列碳纤维及织物、预浸料、碳梁等。2024年业绩承压，营收24.5亿元同比-2.69%，净利润7.41-7.47亿元同比-14.43%。碳纤维及织物业务受定型纤维交付节奏放缓影响，但风电碳梁实现量利修复，高端T800级碳纤维同比增长64.64%，国产替代逻辑持续强化。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **碳纤维及织物** | T300/T700/T800/T1000级碳纤维、碳纤维织物 | 航空航天、军工、风电 |
| **能源新材料（碳梁）** | 风电叶片用碳纤维复合材料 | 风电叶片主梁 |
| **预浸料** | 碳纤维预浸料 | 航空航天、体育休闲 |
| **精密机械** | 碳纤维生产设备 | 碳纤维产业链 |

### 业务模式

公司采用研产销一体化模式，从原丝、碳纤维到织物、预浸料、复合材料制件全产业链布局。与国内军工、航空航天客户深度绑定，2024年与客户A签订36.6亿元重大合同（2024-2027年），其中2025年11.9亿元、2026年18.7亿元。

### 技术路线/核心技术

- **碳纤维制备**：湿法纺丝、干喷湿纺，T300-T1000全系列
- **航空航天认证**：T800级已批量供货，T1000级在研
- **风电碳梁**：全球主流风电叶片碳梁供应商，海外销量占比94.5%

**技术优势**：
国产碳纤维技术领军，航空航天领域认证周期长（3-5年），壁垒高。T800级碳纤维2024年收入4.72亿元同比+64.64%，高端产品放量加速。

## 四、行业地位与产业链定位

### 产业链定位

```
碳纤维产业链：
丙烯腈 → 原丝 → 【光威复材 碳纤维/织物/预浸料】→ 复合材料制件 → 航空航天/风电/军工
                              ↑
                         光威复材（国产龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国产碳纤维龙头，航空航天碳纤维核心供应商 |
| **竞争优势** | 全系列产品、军工认证、重大合同保障 |
| **客户结构** | 军工、航空航天、风电龙头（维斯塔斯等） |
| **供应商** | 丙烯腈等化工原料，原丝自产 |

### 竞争格局

**主要竞争对手**：
- 中复神鹰：大丝束碳纤维、民用市场
- 吉林碳谷：原丝龙头

**公司优势**：
航空航天碳纤维国产替代龙头，T800级放量、36.6亿元重大合同体现客户认可。风电碳梁海外占比高，分散单一市场风险。

## 五、投资亮点

### 核心亮点

- ✅ **国产碳纤维龙头**：航空航天碳纤维核心供应商，T800级放量
- ✅ **拐点逻辑**：①**国产替代**——军工、航空航天碳纤维国产化率提升，36.6亿元重大合同落地；②**高端放量**——T800级收入+64.64%，T1000级工程化推进；③**风电修复**——碳梁量利修复，海外需求旺盛
- ✅ **重大合同**：与客户A签订36.6亿元合同（2024-2027），2025年11.9亿、2026年18.7亿
- ✅ **全产业链**：原丝-碳纤维-织物-预浸料-复合材料一体化

### 成长驱动因素

1. **国产替代**
   - 航空航天碳纤维国产化率提升
   - 军工定型产品批量交付
   - T800/T1000高端产品认证放量

2. **风电碳梁**
   - 海外风电需求，碳梁渗透率提升
   - 销量从409万米增至1140万米

3. **新能源需求**
   - 氢能储罐、压力容器碳纤维
   - 光伏、锂电设备轻量化

### 催化剂

- 重大合同按计划交付
- T800/T1000级产品认证突破
- 风电碳梁海外订单持续增长
- 军工采购节奏恢复
