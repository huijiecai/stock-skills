## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中复神鹰碳纤维股份有限公司 |
| 证券代码 | 688295 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2006年 |
| 上市时间 | 2022年4月6日 |
| 所属行业 | 化学纤维制造业（碳纤维） |
| 市场地位 | 碳纤维龙头，大丝束碳纤维国内领先 |

**公司概况**：
中复神鹰是中国建材集团旗下碳纤维企业，国内碳纤维行业龙头之一。公司聚焦碳纤维行业新技术和新业态，产品涵盖小丝束、大丝束碳纤维，应用于航空航天、风电、压力容器、碳碳复合材料等领域。干喷湿纺技术国内领先，大丝束碳纤维产能扩张支撑民用市场放量，长期成长前景被机构看好。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **小丝束碳纤维** | T700/T800级，干喷湿纺 | 航空航天、军工、体育休闲 |
| **大丝束碳纤维** | 24K/48K等，低成本 | 风电、压力容器、碳碳复材 |
| **碳纤维织物** | 碳纤维编织物 | 复合材料增强 |
| **预浸料** | 碳纤维预浸料 | 复合材料成型 |

### 业务模式

公司采用原丝-碳纤维一体化生产模式，干喷湿纺技术实现高效率、低成本。西宁万吨级基地、上海碳纤维项目支撑产能扩张。产品覆盖航空航天、风电叶片、氢能储罐、碳碳热场、轨道交通等多领域。

### 技术路线/核心技术

- **干喷湿纺**：国内率先突破，纺丝速度高、成本低
- **大丝束**：24K/48K大丝束碳纤维，民用市场主力
- **一体化**：原丝-碳纤维-织物-预浸料产业链

**技术优势**：
干喷湿纺技术国内领先，大丝束碳纤维性价比突出。西宁基地规模效应显著，成本竞争力强。

## 四、行业地位与产业链定位

### 产业链定位

```
碳纤维产业链：
丙烯腈 → 原丝 → 【中复神鹰 碳纤维】→ 织物/预浸料 → 风电/压力容器/航空航天
                              ↑
                    中复神鹰（大丝束龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 碳纤维龙头，大丝束国内领先 |
| **竞争优势** | 干喷湿纺、大丝束、规模效应 |
| **客户结构** | 风电、压力容器、碳碳复材、航空航天 |
| **股东背景** | 中国建材集团，央企资源 |

### 竞争格局

**主要竞争对手**：
- 光威复材：航空航天碳纤维龙头
- 吉林碳谷：原丝龙头
- 恒神股份：全产业链
- 上海石化：大丝束碳纤维

**公司优势**：
大丝束碳纤维产能与成本优势突出，风电、氢能、碳碳复材等民用市场需求旺盛，成长空间大。中国建材系资源支持。

## 五、投资亮点

### 核心亮点

- ✅ **大丝束碳纤维龙头**：24K/48K大丝束国内领先，民用市场主力
- ✅ **拐点逻辑**：①**新能源需求**——风电碳梁、氢能储罐、碳碳热场需求爆发；②**国产替代**——碳纤维国产化率提升，进口依赖度下降；③**成本优势**——干喷湿纺+西宁基地规模效应，性价比突出
- ✅ **产能扩张**：西宁万吨级基地、上海项目，产能持续释放
- ✅ **央企背景**：中国建材集团，资源与客户优势

### 成长驱动因素

1. **新能源需求**
   - 风电叶片碳纤维渗透率提升
   - 氢能储罐IV型瓶碳纤维需求
   - 光伏碳碳热场材料
   - 锂电设备轻量化

2. **国产替代**
   - 碳纤维进口替代加速
   - 大丝束国产化率提升
   - 高端应用逐步突破

3. **成本下降**
   - 大丝束规模化降本
   - 西宁基地规模效应
   - 原丝自产一体化

### 催化剂

- 风电装机量超预期
- 氢能储罐政策与需求放量
- 大丝束碳纤维价格企稳
- 新产能投产释放

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 中复神鹰拐点取决于：①**新能源需求**——风电碳梁、氢能储罐、碳碳热场需求爆发，大丝束碳纤维放量；②**国产替代**——碳纤维国产化率提升，进口替代加速；③**产能释放**——西宁、上海等基地产能释放，规模效应降本；④**价格企稳**——碳纤维价格经历调整后企稳，量增驱动业绩。**拐点信号**：大丝束销量同比高增、毛利率企稳回升、新产能满产满销。
