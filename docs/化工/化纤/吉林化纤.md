## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 吉林化纤股份有限公司 |
| 证券代码 | 000420 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1988年 |
| 上市时间 | 1996年8月2日 |
| 所属行业 | 化学纤维制造业 |
| 市场地位 | 粘胶纤维+碳纤维原丝，双轮驱动 |

**公司概况**：
吉林化纤主营粘胶纤维（粘胶长丝、短纤）和碳纤维原丝。2024年上半年粘胶纤维营收占比80.86%（2.13亿元），碳纤维产品占比16.83%（0.44亿元）。粘胶长丝毛利率20.01%是主要利润来源，碳纤维产品毛利率-2.01%仍亏损。公司计划募资不超过12亿元投资1.2万吨碳纤维复材项目（风电拉挤板材），碳纤维是长期成长方向。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **粘胶长丝** | 人造丝长丝，营收占比56.30% | 高端服装、丝绸替代 |
| **粘胶短纤** | 人造棉短纤，营收占比24.55% | 服装、家纺、非织造 |
| **碳纤维原丝** | 碳纤维前体，1K、3K等 | 碳纤维、复合材料 |
| **碳纤维复材** | 拉挤板材（规划1.2万吨） | 风电叶片 |
| **碳纤维** | 凯美克600吨1K、3K生产线 | 高端应用 |

### 业务模式

公司采用"粘胶+碳纤维"双轮驱动。粘胶纤维是传统主业，贡献主要利润；碳纤维原丝、碳纤维是战略方向，凯美克600吨1K、3K碳纤维已投产，1.2万吨碳纤维复材（风电拉挤板材）项目推进中。碳纤维受益于风电、航空航天、汽车轻量化等需求。

### 技术路线/核心技术

- **粘胶工艺**：浆粕-粘胶长丝/短纤
- **碳纤维原丝**：PAN基原丝
- **碳纤维**：凯美克1K、3K小丝束
- **碳纤维复材**：拉挤工艺，风电叶片

**技术优势**：
粘胶长丝毛利率20.01%，盈利能力强。碳纤维原丝、碳纤维具备技术壁垒，风电拉挤板材是成长方向。与中石油吉林石化千吨级碳纤维项目有协同。

## 四、行业地位与产业链定位

### 产业链定位

```
粘胶+碳纤维产业链：
浆粕 → 【吉林化纤 粘胶】→ 纺织 → 服装/家纺
                    ↑
PAN原丝 → 【吉林化纤 碳纤维原丝/碳纤维】→ 风电/航空航天/汽车
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **粘胶** | 粘胶长丝、短纤，毛利率20% |
| **碳纤维** | 原丝、1K/3K小丝束，风电复材规划 |
| **竞争优势** | 粘胶盈利稳，碳纤维成长空间大 |
| **客户结构** | 纺织、风电、航空航天、汽车 |

### 竞争格局

**主要竞争对手**：
- 粘胶：新乡化纤、南京化纤
- 碳纤维：光威复材、中简科技、吉林碳谷
- 碳纤维原丝：吉林碳谷等

**公司优势**：
粘胶+碳纤维双轮驱动，粘胶贡献利润、碳纤维贡献成长。碳纤维原丝、1K/3K小丝束、风电拉挤板材布局完整。与中石油吉林石化碳纤维项目协同。

## 五、投资亮点

### 核心亮点

- ✅ **粘胶盈利稳**：粘胶长丝毛利率20.01%，主要利润来源
- ✅ **碳纤维成长**：1.2万吨复材项目、凯美克600吨碳纤维
- ✅ **双轮驱动**：粘胶稳业绩，碳纤维拓成长
- ✅ **风电需求**：碳纤维拉挤板材用于风电叶片
- ✅ **碳纤维拐点**：若碳纤维扭亏、放量，业绩弹性大

### 成长驱动因素

1. **碳纤维放量**
   - 1.2万吨复材项目投产
   - 凯美克600吨碳纤维产能爬坡
   - 风电、航空航天、汽车需求

2. **碳纤维扭亏**
   - 规模效应、产能利用率提升
   - 毛利率从-2.01%改善
   - 产品结构优化

3. **粘胶稳健**
   - 粘胶长丝毛利率20%，稳定贡献
   - 粘胶拐点若到来，锦上添花

### 催化剂

- 1.2万吨碳纤维复材项目投产
- 碳纤维毛利率转正
- 风电需求增长
- 募资12亿元落地

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **碳纤维拐点** | 当前亏损，若扭亏弹性大 | 产能放量、规模效应 |
| **粘胶拐点** | 粘胶价格若反弹 | 需求复苏、价差修复 |
| **业绩拐点** | 碳纤维贡献利润 | 复材项目投产、扭亏 |
