## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏华西村股份有限公司 |
| 证券代码 | 000936 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1991年 |
| 上市时间 | 1999年8月10日 |
| 所属行业 | 化学纤维制造业 |
| 市场地位 | 江阴国资委旗下涤纶化纤，石化仓储物流 |

**公司概况**：
华西股份是江阴市国资委旗下的涤纶化纤厂商，主营涤纶化纤研发、生产和销售，以及石化仓储物流服务。2024年营收25.33亿元、净利润3.29亿元，同比分别+0.88%、+132.98%，净利润大幅增长反映盈利能力改善。上半年净利润1.66亿元，拟10派0.2元。总股本8.86亿股，每股未分配利润4.22元。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **涤纶化纤** | 涤纶长丝、短纤等 | 纺织服装、家纺 |
| **石化仓储物流** | 石化产品仓储、物流 | 化工物流 |
| **聚酯** | 聚酯切片等 | 涤纶原料、瓶片 |

### 业务模式

公司采用"涤纶化纤+石化仓储"双轮驱动，涤纶化纤是传统主业，石化仓储物流提供稳定现金流。2024年营收基本持平但净利润大幅增长，反映产品结构优化、成本控制或非经常性收益贡献。

### 技术路线/核心技术

- **涤纶技术**：聚酯-涤纶长丝/短纤
- **石化仓储**：仓储设施、物流服务
- **成本控制**：规模效应、管理优化

**技术优势**：
江阴国资委旗下，区域资源优势。涤纶+石化仓储协同，现金流相对稳定。2024年盈利能力显著改善。

## 四、行业地位与产业链定位

### 产业链定位

```
涤纶产业链：
PTA → 聚酯 → 【华西股份 涤纶化纤】→ 纺织 → 服装/家纺
                    ↑
              华西股份+石化仓储
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 涤纶行业中小规模，江阴区域 |
| **竞争优势** | 国资委背景、石化仓储协同、盈利改善 |
| **客户结构** | 纺织、服装下游 |
| **供应商** | PTA、聚酯外采 |

### 竞争格局

**主要竞争对手**：
- 桐昆股份、新凤鸣：涤纶长丝龙头
- 恒逸石化、恒力石化：PTA+聚酯
- 其他区域涤纶企业

**公司优势**：
规模虽不及龙头，但2024年盈利能力大幅改善。国资委背景、石化仓储提供稳定现金流。若涤纶拐点持续，业绩弹性可期。
