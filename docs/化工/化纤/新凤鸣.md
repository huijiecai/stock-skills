## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 新凤鸣集团股份有限公司 |
| 证券代码 | 603225 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2007年 |
| 上市时间 | 2017年4月18日 |
| 所属行业 | 化学纤维制造业 |
| 市场地位 | 聚酯长丝行业第二，聚酯短纤国内第一 |

**公司概况**：
新凤鸣专注聚酯长丝、短纤，2024年营收670.91亿元、归母净利润11亿元，同比稳健增长。聚酯长丝产能805.5万吨、市占率超12%，国内第二；聚酯短纤产能120万吨，国内第一。PTA产能770万吨，2025年末预计超1000万吨，产业链一体化持续强化。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **聚酯长丝** | POY、FDY、DTY，产能805.5万吨 | 纺织服装、家纺 |
| **聚酯短纤** | 产能120万吨，国内第一 | 填充、非织造布 |
| **PTA** | 产能770万吨，2025年超1000万吨 | 自用及外销 |
| **聚酯切片** | 中间产品 | 瓶片、薄膜 |

### 业务模式

公司采用"PTA-聚酯"一体化模式，专注聚酯长丝、短纤，扩产节奏快。2024年聚酯长丝产量745.24万吨、销量755.39万吨，同比均增约11%。江苏新拓XCP03聚酯项目2024年9月投产，产能持续释放。

### 技术路线/核心技术

- **大容量聚酯**：单线产能大，能耗低
- **差别化纤维**：功能性、差异化产品
- **PTA配套**：原料自给，成本可控
- **智能制造**：数字化、自动化

**技术优势**：
聚酯长丝产能国内第二，扩产节奏快、弹性大。聚酯短纤国内第一，产品结构多元。PTA产能快速扩张，2025年超1000万吨。

## 四、行业地位与产业链定位

### 产业链定位

```
PTA/聚酯产业链：
原油 → PX → PTA → 【新凤鸣 聚酯长丝/短纤】→ 纺织/非织造 → 终端消费
                        ↑
              新凤鸣（长丝第二、短纤第一）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 聚酯长丝市占率超12%，国内第二；聚酯短纤国内第一 |
| **竞争优势** | 扩产节奏快、弹性大、PTA配套强化 |
| **客户结构** | 纺织、服装、家纺、非织造 |
| **供应商** | PTA自给率提升，2025年超1000万吨 |

### 竞争格局

**主要竞争对手**：
- 桐昆股份：聚酯长丝第一
- 恒逸石化：PTA+聚酯
- 恒力石化：炼化+PTA+聚酯
- 东方盛虹：炼化+化纤

**公司优势**：
新凤鸣专注聚酯，扩产节奏快，四季度利润同比+77.92%显示拐点信号。聚酯短纤国内第一，产品结构多元。PTA产能快速扩张，一体化优势强化。

## 五、投资亮点

### 核心亮点

- ✅ **聚酯长丝第二**：产能805.5万吨，市占率超12%
- ✅ **聚酯短纤第一**：产能120万吨，国内龙头
- ✅ **拐点信号**：四季度净利润同比+77.92%
- ✅ **PTA扩张**：2025年末超1000万吨，一体化强化
- ✅ **扩产弹性**：XCP03项目投产，产能持续释放

### 成长驱动因素

1. **PTA/聚酯拐点**
   - PTA扩产周期结束，价差有望修复
   - 聚酯长丝运行平稳，四季度利润大幅改善
   - 聚酯出口高速增长

2. **产能释放**
   - XCP03聚酯项目2024年9月投产
   - PTA产能2025年超1000万吨
   - 规模效应、成本优势

3. **产品结构**
   - 聚酯短纤国内第一，非织造需求
   - 差别化、功能性产品占比提升

### 催化剂

- 聚酯价差修复，四季度已现拐点
- PTA扩产周期结束
- 纺织出口复苏
- PTA产能超1000万吨投产

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **聚酯拐点** | 四季度净利润+77.92%，拐点信号 | 价差修复、需求复苏 |
| **PTA拐点** | 扩产结束，一体化企业受益 | 反内卷、产能出清 |
