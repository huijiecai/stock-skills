## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 恒逸石化股份有限公司 |
| 证券代码 | 000703 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1994年 |
| 上市时间 | 1997年3月28日 |
| 所属行业 | 石油加工、化学纤维制造业 |
| 市场地位 | PTA+聚酯一体化龙头，逸盛石化产能领先 |

**公司概况**：
恒逸石化是"炼化-PTA-聚酯"一体化龙头企业，控股逸盛石化（PTA产能约2150万吨，行业第一）、参股恒逸文莱炼化。公司聚酯长丝、瓶片产能规模大，2024年聚酯业务景气回升，化纤营收63.5亿元、毛利24.7亿元。文莱炼化2024年亏损拖累业绩，国内PTA/聚酯板块仍是核心。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **PTA** | 对苯二甲酸，逸盛石化产能约2150万吨 | 聚酯原料、涤纶、瓶片 |
| **聚酯长丝** | 涤纶POY、FDY、DTY | 纺织服装、家纺 |
| **聚酯瓶片** | 食品级聚酯 | 饮料瓶、包装 |
| **炼化产品** | 文莱炼化成品油、芳烃 | 能源、化工原料 |
| **己内酰胺** | 钦州项目，锦纶原料 | 锦纶纤维 |

### 业务模式

公司采用"炼化-PTA-聚酯"全产业链模式，国内以PTA、聚酯为核心，海外文莱炼化拓展上游。2024年聚酯行业景气回升，出口强劲（聚酯出口1287万吨，同比+15%），化纤板块贡献主要利润。钦州120万吨己内酰胺-聚酰胺一体化项目预计2025年投产。

### 技术路线/核心技术

- **PTA技术**：逸盛石化采用先进PTA工艺，规模效应显著
- **聚酯一体化**：PTA-聚酯配套，成本可控
- **炼化**：文莱炼化项目，海外原料基地
- **己内酰胺**：延伸至锦纶产业链

**技术优势**：
逸盛石化PTA产能行业第一，CR5集中度70%。聚酯新增产能2024年仅128万吨，增速放缓至2%，行业供给格局优化。

## 四、行业地位与产业链定位

### 产业链定位

```
PTA/聚酯产业链：
原油 → 炼化 → PX → 【恒逸石化 PTA】→ 聚酯长丝/瓶片 → 纺织/包装
                        ↑
              逸盛石化（PTA产能第一）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 逸盛石化PTA产能约2150万吨，行业第一 |
| **竞争优势** | PTA+聚酯一体化、规模效应 |
| **客户结构** | 覆盖聚酯、纺织、包装下游 |
| **供应商** | 文莱炼化自给部分原料，国内依赖外采 |

### 竞争格局

**主要竞争对手**：
- 恒力石化：PTA产能1660万吨，炼化一体化
- 桐昆股份：涤纶长丝龙头
- 新凤鸣：聚酯长丝第二
- 荣盛石化：浙石化大炼化

**公司优势**：
逸盛石化PTA产能行业第一，PTA扩产周期结束、行业拐点渐显下，龙头有望受益。聚酯出口强劲，化纤板块景气回升。

## 五、投资亮点

### 核心亮点

- ✅ **PTA产能第一**：逸盛石化约2150万吨，CR5达70%
- ✅ **拐点逻辑**：PTA扩产周期结束，行业拐点渐显
- ✅ **聚酯景气回升**：2024年化纤毛利24.7亿，出口+15%
- ✅ **钦州项目**：120万吨己内酰胺-聚酰胺2025年投产，延伸锦纶
- ✅ **现金流改善**：经营现金流+32.33%

### 成长驱动因素

1. **PTA拐点**
   - 扩产周期结束，2026-2027年无新增
   - 反内卷政策，老旧产能退出
   - 价差有望修复

2. **聚酯出口**
   - 2024年聚酯出口1287万吨，+15%
   - 纺织出口复苏带动需求
   - 聚酯新增产能增速放缓至2%

3. **文莱炼化修复**
   - 文莱二期推进
   - 若炼化景气改善，亏损有望收窄

### 催化剂

- PTA反内卷政策落地，开工率控制
- 聚酯出口持续高增长
- 钦州己内酰胺项目投产
- 文莱炼化景气修复

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **PTA拐点** | 扩产结束，CR5达70% | 反内卷、产能出清 |
| **聚酯拐点** | 新增产能增速放缓，出口强劲 | 纺织出口、价差修复 |
| **文莱拐点** | 炼化景气若改善，亏损收窄 | 原油价差、二期投产 |
