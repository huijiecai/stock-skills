## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 桐昆集团股份有限公司 |
| 证券代码 | 601233 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 2011年5月18日 |
| 所属行业 | 化学纤维制造业 |
| 市场地位 | 涤纶长丝全球龙头，市占率第一 |

**公司概况**：
桐昆股份是中国涤纶长丝行业绝对龙头，涤纶长丝产能、产量稳居全球第一。公司形成"PTA-聚酯-涤纶长丝"一体化布局，2024年涤纶长丝盈利好转，受益于PTA扩产周期结束、聚酯产业链景气回升。与恒逸石化、新凤鸣等六家A股企业合计产能占全行业约79%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **涤纶预取向丝(POY)** | 涤纶长丝半成品，占比约61% | 纺织、加弹、织造 |
| **涤纶牵伸丝(FDY)** | 全拉伸丝，直接用于织造 | 服装、家纺、产业用布 |
| **涤纶加弹丝(DTY)** | 加弹变形丝，手感柔软 | 针织、梭织、运动服 |
| **PTA** | 精对苯二甲酸，聚酯原料 | 自用及外销 |
| **聚酯切片** | 聚酯中间产品 | 瓶片、薄膜、短纤 |

### 业务模式

公司采用"PTA-聚酯-涤纶长丝"一体化模式，从原料到终端长丝全覆盖。通过规模化生产、技术升级和产业链协同降本增效。2024年聚酯成本改善、投资收益共同驱动业绩增长。

### 技术路线/核心技术

- **大容量聚酯技术**：单线产能大，能耗低，成本优势明显
- **差别化纤维**：功能性、差异化产品占比提升
- **一体化布局**：PTA自给降低原料波动风险
- **智能制造**：数字化、自动化提升生产效率

**技术优势**：
涤纶长丝产能规模全球第一，规模效应显著。PTA配套产能保障原料供应，产业链一体化成本优势突出。

## 四、行业地位与产业链定位

### 产业链定位

```
PTA/聚酯产业链：
原油 → PX → PTA → 【桐昆股份 涤纶长丝】→ 纺织服装 → 终端消费
                        ↑
                  桐昆股份（全球第一）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 涤纶长丝产能全球第一，市占率领先 |
| **竞争优势** | 规模效应、一体化布局、成本控制 |
| **客户结构** | 覆盖纺织、服装、家纺等下游头部企业 |
| **供应商** | PTA自给，原料保障能力强 |

### 竞争格局

**主要竞争对手**：
- 新凤鸣：聚酯长丝第二，产能扩张快
- 恒逸石化：PTA+聚酯一体化
- 恒力石化：炼化+PTA+聚酯全产业链
- 东方盛虹：炼化+化纤
- 荣盛石化：大炼化+聚酯

**公司优势**：
桐昆股份专注涤纶长丝，产能规模全球第一，市占率极高。2024年PTA扩产周期结束、聚酯景气回升，龙头受益于价差修复和集中度提升。

## 五、投资亮点

### 核心亮点

- ✅ **涤纶长丝全球龙头**：产能、产量全球第一，市占率极高
- ✅ **拐点逻辑明确**：PTA扩产周期结束（2026-2027年基本无新增），行业拐点渐显
- ✅ **盈利好转**：2024年涤纶长丝盈利改善，聚酯成本下降
- ✅ **产业链一体化**：PTA自给，原料波动风险可控
- ✅ **行业集中度提升**：六家龙头合计市占率79%，寡头格局

### 成长驱动因素

1. **PTA/聚酯产业链拐点**
   - PTA扩产周期结束，供给格局优化
   - 涤纶-PTA价差有望修复
   - 纺织出口复苏带动需求

2. **价差修复**
   - PTA-PX价差、涤纶-PTA价差处于低位
   - 盈利低迷已持续多年，反弹空间大
   - 新装置占比较高的头部企业受益

3. **需求复苏**
   - 纺织品出口维持增长
   - 社会消费品零售增速回升
   - 聚酯出口高速增长

### 催化剂

- **PTA反内卷政策**：工信部鼓励老旧产能退出，开工率回落至79%
- **纺织出口复苏**：化纤景气度与纺织品出口高度正相关
- **价差修复**：PTA、涤纶价差筑底反弹
- **产能出清**：老旧产能退出，格局优化

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **PTA拐点** | 扩产周期结束，2026-2027年无新增产能 | 反内卷政策、开工率控制 |
| **涤纶拐点** | 价差处于低位，盈利筑底反弹 | 需求复苏、出口增长 |
| **集中度拐点** | CR5达70%，龙头议价权增强 | 产能出清、寡头格局 |
