## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏东方盛虹股份有限公司 |
| 证券代码 | 000301 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2000年5月29日 |
| 所属行业 | 石油加工、化学纤维制造业 |
| 市场地位 | 炼化+化纤一体化，盛虹炼化1600万吨 |

**公司概况**：
东方盛虹是炼化+化纤一体化企业，控股盛虹炼化（1600万吨/年炼化一体化项目）、斯尔邦石化（EVA、丙烯腈等）、盛虹化纤（涤纶长丝）。2024年面临显著困难，预计全年亏损最高24亿元，同比下滑超400%。炼化景气低迷、化纤价差收窄是主因，公司计划不分配现金红利。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **炼化产品** | 成品油、芳烃、烯烃 | 能源、化工原料 |
| **PTA** | 对苯二甲酸 | 聚酯原料 |
| **涤纶长丝** | POY、FDY、DTY | 纺织服装、家纺 |
| **EVA** | 乙烯-醋酸乙烯共聚物 | 光伏胶膜、发泡 |
| **丙烯腈** | 斯尔邦石化 | 腈纶、碳纤维、ABS |
| **乙二醇** | 聚酯原料 | 聚酯生产 |

### 业务模式

公司采用"炼化-石化-化纤"一体化模式，盛虹炼化1600万吨炼化项目为核心，斯尔邦石化布局高端化工，盛虹化纤延伸至涤纶长丝。2024年炼化价差低迷、化纤景气波动，业绩承压。

### 技术路线/核心技术

- **大炼化一体化**：1600万吨炼化，PX、乙烯等配套
- **斯尔邦高端化工**：EVA、丙烯腈、MMA等
- **化纤一体化**：PTA-聚酯-涤纶
- **新能源材料**：EVA光伏胶膜料

**技术优势**：
炼化一体化规模大，斯尔邦EVA、丙烯腈技术领先。化纤与炼化协同，产业链完整。2024年景气下行，盈利承压。

## 四、行业地位与产业链定位

### 产业链定位

```
炼化+化纤产业链：
原油 → 【东方盛虹 炼化】→ PX/乙烯 → PTA/乙二醇 → 涤纶长丝 → 纺织
              ↑
        盛虹炼化+斯尔邦+化纤
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 炼化1600万吨，化纤产能规模大 |
| **竞争优势** | 炼化+石化+化纤一体化 |
| **客户结构** | 能源、化工、纺织下游 |
| **供应商** | 炼化自给原料 |

### 竞争格局

**主要竞争对手**：
- 恒力石化：炼化+PTA+聚酯
- 荣盛石化：浙石化4000万吨炼化
- 恒逸石化：PTA+聚酯
- 桐昆股份、新凤鸣：聚酯长丝

**公司优势**：
炼化一体化布局完整，斯尔邦EVA、丙烯腈高端化工有技术壁垒。2024年景气下行，若炼化、化纤拐点到来，业绩弹性大。

## 五、投资亮点

### 核心亮点

- ✅ **炼化一体化**：1600万吨炼化，产业链完整
- ✅ **斯尔邦高端化工**：EVA光伏胶膜料、丙烯腈
- ✅ **拐点弹性**：若炼化、化纤景气修复，业绩弹性大
- ✅ **一体化优势**：原油到涤纶全覆盖
- ⚠️ **当前承压**：2024年亏损，需等待拐点

### 成长驱动因素

1. **炼化拐点**
   - 炼化价差若修复，盈利改善
   - 原油价格企稳，加工利润回升

2. **化纤拐点**
   - PTA/聚酯扩产周期结束
   - 涤纶价差修复

3. **EVA/丙烯腈**
   - 光伏需求，EVA景气
   - 碳纤维、ABS需求，丙烯腈

### 催化剂

- 炼化价差修复
- PTA/聚酯行业拐点
- 光伏EVA需求复苏
- 纺织出口复苏

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **炼化拐点** | 价差处于低位，若修复弹性大 | 原油企稳、需求复苏 |
| **化纤拐点** | PTA扩产结束，涤纶价差修复 | 反内卷、出口复苏 |
| **业绩拐点** | 当前亏损，拐点后弹性大 | 炼化+化纤双拐点 |
