## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 际华集团股份有限公司 |
| 证券代码 | 601718 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2009年 |
| 上市时间 | 2010年8月16日 |
| 所属行业 | 纺织服装、防护装具 |
| 市场地位 | 军需被装、防护装具、功能纤维材料 |

**公司概况**：
际华集团主营纺织服装、防护装具及装备，涉及功能纤维材料。2024年营收98.86亿元（-14.49%），归母净利润亏损42.28亿元，经营面临困难。公司防护装具业务涉及环保滤材（PPS、P84、PTFE、NOMEX等）、防弹衣、防弹头盔等功能性材料；纺织印染涉及针织面料、特种功能性服装。2024年未进行利润分配。运动休闲服饰、职业装、医用防护等细分市场增速超行业平均，绿色纤维政策鼓励循环再利用化学纤维。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **环保滤材** | PPS、P84、PTFE复合、NOMEX、ZMS、玻璃纤维滤料 | 工业除尘、环保 |
| **防护装备** | 防弹衣、防弹头盔、防弹插板 | 军警、安保 |
| **功能性装具** | 帐篷、携行具 | 军需、户外 |
| **针织面料** | 针织面料、针织成衣 | 纺织服装 |
| **特种功能性服装** | 职业装、医用防护 | 职业装、医疗 |
| **染色布** | 染色面料 | 纺织 |

### 业务模式

公司采用"纺织-服装-防护装具"一体化模式，以军需被装为基础，拓展防护装具、功能性服装。功能纤维材料应用于滤材、防弹等，与化纤行业有交叉。2024年纺织服装行业平稳，运动休闲、职业装、医用防护增速超行业，绿色化趋势明显。

### 技术路线/核心技术

- **功能滤材**：PPS、P84、PTFE、NOMEX等耐高温、耐腐蚀滤料
- **防弹材料**：防弹纤维、复合材料
- **功能性面料**：特种服装、医用防护
- **绿色纤维**：循环再利用化学纤维（政策鼓励）

**技术优势**：
功能纤维材料在滤材、防弹领域有技术壁垒。军需被装、防护装具具备资质和客户优势。绿色纤维、循环再利用是政策方向。

## 四、行业地位与产业链定位

### 产业链定位

```
功能纤维产业链：
化纤原料 → 【际华集团 功能滤材/防弹材料】→ 环保/军警/服装
                    ↑
              际华集团（功能纤维应用）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 军需被装、防护装具重要供应商 |
| **竞争优势** | 资质、客户、功能材料技术 |
| **客户结构** | 军警、环保、医疗、职业装 |
| **供应商** | 化纤、化工原料 |

### 竞争格局

**主要竞争对手**：
- 军需被装：其他军工纺织企业
- 滤材：环保滤料企业
- 防弹：防护装备企业
- 功能性服装：职业装、医用防护企业

**公司优势**：
军需被装、防护装具资质和客户优势。功能纤维材料在滤材、防弹有技术壁垒。绿色纤维政策鼓励，循环再利用化学纤维是方向。

## 五、投资亮点

### 核心亮点

- ✅ **功能纤维**：PPS、P84、PTFE、NOMEX等滤材，技术壁垒
- ✅ **防护装具**：防弹衣、头盔，军警需求
- ✅ **绿色纤维**：政策鼓励循环再利用化学纤维
- ✅ **细分景气**：运动休闲、职业装、医用防护增速超行业
- ⚠️ **当前承压**：2024年亏损42亿，需等待拐点

### 成长驱动因素

1. **功能纤维拐点**
   - 环保滤材需求（工业除尘、环保）
   - 防弹装备需求（军警、安保）
   - 绿色纤维政策支持

2. **细分景气**
   - 运动休闲、职业装、医用防护
   - 增速超行业平均
   - 功能性服装升级

3. **扭亏拐点**
   - 若亏损收窄、扭亏
   - 业绩弹性大
   - 需观察订单、成本

### 催化剂

- 军需订单恢复
- 环保滤材需求增长
- 绿色纤维政策落地
- 亏损收窄、扭亏

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **业绩拐点** | 当前亏损42亿，若扭亏弹性大 | 订单恢复、成本控制 |
| **功能纤维拐点** | 滤材、防弹需求增长 | 环保、军警需求 |
