## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 华峰化学股份有限公司 |
| 证券代码 | 002064 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 2006年8月23日 |
| 所属行业 | 化学纤维制造业 |
| 市场地位 | 氨纶全球龙头，产能产量全球第二、中国第一 |

**公司概况**：
华峰化学是全球氨纶行业龙头企业，氨纶产能与产量位居全球第二、中国第一。同时聚氨酯原液产量全球第一，己二酸产能135.5万吨、产能利用率94.96%，全球领先。2024年营收269.31亿元、归母净利润22.2亿元，同比有所下滑，主因氨纶、己二酸需求不足及原材料价格波动。公司拥有30万吨/年差别化氨纶项目，部分已达产。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **氨纶** | 弹性纤维，产能全球第二、中国第一 | 瑜伽服、泳衣、紧身裤、口罩耳带、运动服 |
| **聚氨酯原液** | 产量全球第一 | 鞋底、汽车内饰、软泡 |
| **己二酸** | 产能135.5万吨，产能利用率94.96% | 尼龙66、聚氨酯、增塑剂 |
| **环己酮** | 己二酸原料 | 己二酸生产 |

### 业务模式

公司采用"氨纶+聚氨酯+己二酸"多产品平台模式，氨纶为核心，聚氨酯原液、己二酸全球领先。通过规模化生产、成本控制和技术升级保持竞争力。2024年全球宏观经济复杂，需求不足、原料波动导致产品售价及利润承压。

### 技术路线/核心技术

- **氨纶技术**：差别化氨纶，30万吨项目部分达产
- **聚氨酯原液**：产量全球第一，鞋底等应用
- **己二酸**：135.5万吨产能，产能利用率高
- **成本控制**：规模效应、一体化降本

**技术优势**：
氨纶全球龙头，成本优势极强。聚氨酯原液、己二酸全球领先，多产品协同。差别化氨纶项目持续贡献增量。

## 四、行业地位与产业链定位

### 产业链定位

```
氨纶产业链：
PTMEG/纯MDI → 【华峰化学 氨纶】→ 纺织/织造 → 瑜伽服/泳衣/运动服
                    ↑
              华峰化学（全球龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 氨纶全球第二、中国第一，聚氨酯原液全球第一 |
| **竞争优势** | 成本优势极强，规模效应显著 |
| **客户结构** | 纺织、服装、运动服饰下游 |
| **供应商** | 原料PTMEG、纯MDI外采 |

### 竞争格局

**主要竞争对手**：
- 晓星氨纶、泰和新材等
- 聚氨酯原液：华峰化学全球第一
- 己二酸：华峰化学产能领先

**公司优势**：
氨纶成本优势极强，氨纶价格波动的最大受益者。聚氨酯原液、己二酸全球领先，多产品平台协同。30万吨差别化氨纶项目持续贡献增量。

## 五、投资亮点

### 核心亮点

- ✅ **氨纶全球龙头**：产能产量全球第二、中国第一
- ✅ **拐点弹性**：氨纶价格波动时，龙头受益最大
- ✅ **成本优势**：规模效应、成本控制极强
- ✅ **多产品平台**：聚氨酯原液、己二酸全球领先
- ✅ **现金流稳健**：经营现金流+22.53%，分红持续

### 成长驱动因素

1. **氨纶拐点**
   - 氨纶价格若触底反弹，龙头弹性大
   - 需求复苏（运动服饰、瑜伽服等）
   - 差别化氨纶占比提升

2. **己二酸**
   - 尼龙66、聚氨酯需求
   - 产能利用率94.96%，运行良好

3. **聚氨酯原液**
   - 鞋底、汽车等需求
   - 全球第一，稳定贡献

### 催化剂

- 氨纶价格触底反弹
- 运动服饰、瑜伽服需求复苏
- 原料PTMEG、纯MDI价格回落
- 差别化氨纶产能释放

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **氨纶拐点** | 价格处于低位，龙头成本优势强 | 需求复苏、价格反弹 |
| **业绩拐点** | 2024年利润下滑，若价格反弹弹性大 | 氨纶价涨、需求改善 |
| **成本拐点** | 原料价格若回落，利润改善 | PTMEG、MDI价格 |
