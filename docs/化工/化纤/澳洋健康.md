## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏澳洋健康产业股份有限公司 |
| 证券代码 | 002172 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001年 |
| 上市时间 | 2007年9月21日 |
| 所属行业 | 医疗服务、医药物流（原化纤） |
| 市场地位 | 原锦纶企业，2024年12月已完成化纤业务剥离 |

**公司概况**：
澳洋健康原为锦纶（尼龙）纤维企业，2024年12月15日公告已完成化纤业务剥离，战略转型至医疗健康和医药物流。本报告主要从化纤行业视角记录其历史定位及转型情况。转型后公司专注于医疗服务、医药物流，不再从事锦纶生产。

## 二、主营业务

### 原核心产品/服务（已剥离）

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **锦纶（尼龙）** | 尼龙6、尼龙66纤维 | 丝袜、内衣、冲锋衣、安全气囊 |
| **己内酰胺** | 锦纶原料 | 锦纶聚合 |

### 现核心业务（转型后）

| 业务类型 | 说明 |
|----------|------|
| **医疗服务** | 医院、康复等 |
| **医药物流** | 药品配送、流通 |

### 业务模式

公司已完成从化纤产业向医疗健康产业的战略转变。化纤业务于2024年12月剥离完毕，目前专注于医疗服务和医药物流。

### 技术路线/核心技术

- **原锦纶技术**：己内酰胺-锦纶聚合-纺丝（已剥离）
- **现医疗健康**：医疗服务、医药物流

**技术优势**：
化纤业务已剥离，本报告仅作历史记录。锦纶行业需关注华鼎股份、台华新材等标的。
