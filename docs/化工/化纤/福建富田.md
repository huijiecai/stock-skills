| **锦纶切片** | 己内酰胺聚合 | 纺丝、工程塑料 |
| **己内酰胺** | 锦纶原料（若一体化） | 锦纶聚合 |
| **差别化锦纶** | 功能性、差异化纤维 | 高端纺织、产业用布 |

### 业务模式

福建地区锦纶企业多采用"己内酰胺-锦纶聚合-锦纶长丝"一体化或分段模式。锦纶下游与纺织服装、汽车（安全气囊）相关。福建永荣锦江等具备己内酰胺+锦纶全产业链能力。

### 技术路线/核心技术

- **己内酰胺**：环己酮-己内酰胺（若一体化）
- **锦纶聚合**：己内酰胺开环聚合
- **锦纶纺丝**：熔融纺丝、差别化
- **一体化**：部分企业己内酰胺自给

**技术优势**：
锦纶与涤纶差异化，高端纺织、产业用布有优势。己内酰胺-锦纶一体化企业成本优势明显。福建地区锦纶产业集聚。
