## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏三房巷聚材股份有限公司 |
| 证券代码 | 600370 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1994年 |
| 上市时间 | 2003年3月6日 |
| 所属行业 | 化学纤维制造业 |
| 市场地位 | 涤纶、聚酯瓶片，参与PTA扩产 |

**公司概况**：
三房巷主营涤纶、聚酯瓶片，控股海伦石化（PTA）。2024年面临盈利压力，加工费持续承压、供需错配导致竞争激烈，主要产品价格下降、加工差缩小，毛利率及盈利水平下降。2024年营业毛利率-0.08%，税后净利约-4.88亿元。公司加快推进海伦石化年产320万吨PTA技改扩能项目，积极拓展资源循环和再生材料。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **聚酯瓶片** | 食品级聚酯 | 饮料瓶、包装（软饮料占比73.25%） |
| **涤纶** | 涤纶长丝、短纤 | 纺织服装、家纺 |
| **PTA** | 海伦石化，320万吨技改扩能 | 聚酯原料 |
| **再生PET** | 功能性再生材料 | 循环经济、环保包装 |

### 业务模式

公司采用"PTA-聚酯"一体化模式，海伦石化PTA配套聚酯瓶片、涤纶。2024年聚酯瓶片产量1556万吨（行业）、消费量1497万吨，同比均增约18-21%。公司参与"反内卷"共同维护市场健康，拓展再生材料提升竞争力。

### 技术路线/核心技术

- **PTA技术**：海伦石化320万吨技改扩能
- **聚酯瓶片**：食品级聚酯，软饮料为主
- **再生材料**：功能性再生PET、资源循环
- **一体化**：PTA-聚酯配套

**技术优势**：
PTA-聚酯一体化，海伦石化扩产强化原料保障。再生材料、循环经济拓展新方向。

## 四、行业地位与产业链定位

### 产业链定位

```
PTA/聚酯产业链：
原油 → PX → PTA → 【三房巷 聚酯瓶片/涤纶】→ 饮料瓶/纺织
                        ↑
                  海伦石化PTA
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 聚酯瓶片行业CR4达80%以上 |
| **竞争优势** | PTA-聚酯一体化，海伦石化扩产 |
| **客户结构** | 软饮料、纺织下游 |
| **供应商** | PTA自给 |

### 竞争格局

**主要竞争对手**：
- 逸盛石化、恒力石化等PTA龙头
- 聚酯瓶片：华润材料、万凯新材等
- 涤纶：桐昆、新凤鸣等

**公司优势**：
PTA-聚酯一体化，海伦石化320万吨技改扩能强化规模。再生材料、循环经济拓展差异化。2024年承压，需等待行业拐点。

- ✅ **PTA扩产**：海伦石化320万吨技改扩能推进
- ✅ **一体化布局**：PTA-聚酯瓶片-涤纶

### 成长驱动因素

1. **PTA拐点**

2. **再生材料**

   - 环保、循环经济政策支持

3. **行业自律**
   - 反内卷政策，开工率控制
