## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江海利得新材料股份有限公司 |
| 证券代码 | 002206 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001年 |
| 上市时间 | 2008年1月28日 |
| 所属行业 | 化学纤维制造业、橡胶和塑料制品业 |
| 市场地位 | 涤纶工业丝龙头，轮胎帘布、石塑地板 |

**公司概况**：
海利得是综合性化纤材料企业，主营涤纶工业丝、轮胎帘布、数字印花材料、环保石塑地板等。涤纶工业丝与民用涤纶差异化，应用于轮胎帘子布、安全带、工业用布等。公司拥有从PET切片到工业丝、轮胎帘布的完整产业链，2024年化学纤维制造营收31.7亿元（+5.58%），海外收入37.4亿元占比高。涤纶长丝产能32万吨，聚酯切片30万吨，轮胎帘布6万吨，石塑地板1200万平方米。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **涤纶工业丝** | 高强工业用涤纶，产能32万吨 | 轮胎帘子布、安全带、工业用布 |
| **轮胎帘布** | 涤纶帘子布，产能6万吨 | 汽车轮胎骨架材料 |
| **聚酯切片** | 产能30万吨 | 工业丝原料、瓶片 |
| **数字印花材料** | 印花材料 | 纺织印花 |
| **环保石塑地板** | 产能1200万平米 | 建筑装饰、家居 |
| **塑料材料** | 产能2亿平米 | 包装、建材 |

### 业务模式

公司采用"聚酯-涤纶工业丝-轮胎帘布"一体化模式，从原料到终端帘布全覆盖。涤纶工业丝与民用涤纶差异化，技术门槛更高，应用于汽车、工业领域。海外收入占比高，2024年海外37.4亿元、国内21.6亿元。石塑地板、数字印花拓展新方向。

### 技术路线/核心技术

- **涤纶工业丝**：高强、低缩、耐疲劳
- **轮胎帘布**：汽车轮胎骨架，技术壁垒
- **一体化**：PET切片-工业丝-帘布
- **石塑地板**：环保建材，出口导向

**技术优势**：
涤纶工业丝与民用涤纶差异化，轮胎帘布具备技术壁垒。从PET到帘布完整产业链，汽车用丝等领域有质量优势。海外收入占比高，国际化布局。

## 四、行业地位与产业链定位

### 产业链定位

```
涤纶工业丝产业链：
PTA → 聚酯 → 【海利得 涤纶工业丝】→ 轮胎帘布 → 汽车轮胎
                    ↑
              海利得（工业丝龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 涤纶工业丝、轮胎帘布龙头 |
| **竞争优势** | 一体化、技术壁垒、海外布局 |
| **客户结构** | 轮胎企业、汽车零部件、工业 |
| **供应商** | 聚酯自给，PTA外采 |

### 竞争格局

**主要竞争对手**：
- 涤纶工业丝：古纤道、尤夫股份等
- 轮胎帘布：与轮胎企业深度绑定
- 石塑地板：出口型建材企业

**公司优势**：
涤纶工业丝与民用涤纶差异化，轮胎帘布技术壁垒高。一体化从PET到帘布，汽车用丝质量优势。海外收入占比高，抗国内周期波动。

## 五、投资亮点

### 核心亮点

- ✅ **涤纶工业丝龙头**：与民用涤纶差异化，技术壁垒高
- ✅ **轮胎帘布**：汽车轮胎骨架，与轮胎企业绑定
- ✅ **一体化**：PET-工业丝-帘布，成本、质量优势
- ✅ **海外占比高**：海外37.4亿，抗国内周期
- ✅ **拐点弹性**：若涤纶工业丝价差修复，盈利改善

### 成长驱动因素

1. **涤纶工业丝拐点**
   - 与PTA/聚酯产业链相关
   - 工业丝价差若修复
   - 汽车、轮胎需求复苏

2. **汽车需求**
   - 轮胎帘布与汽车产销相关
   - 新能源汽车、出口增长
   - 轮胎配套、替换需求

3. **石塑地板**
   - 环保建材，出口导向
   - 产能1200万平米
   - 海外需求

### 催化剂

- 涤纶工业丝价差修复
- 汽车产销、轮胎需求复苏
- 海外需求增长
- 石塑地板放量

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **涤纶工业丝拐点** | 与PTA/聚酯相关，价差修复 | 原料价差、需求复苏 |
| **汽车拐点** | 轮胎帘布与汽车相关 | 汽车产销、出口 |
| **业绩拐点** | 营收稳健增长，若价差修复弹性大 | 价差修复、需求增长 |
