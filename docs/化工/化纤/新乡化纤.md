## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 新乡化纤股份有限公司 |
| 证券代码 | 000949 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1994年 |
| 上市时间 | 1999年10月21日 |
| 所属行业 | 化学纤维制造业 |
| 市场地位 | 粘胶纤维龙头，粘胶长丝、短纤 |

**公司概况**：
新乡化纤主营粘胶纤维（人造丝），包括粘胶长丝和粘胶短纤。粘胶纤维以植物纤维（浆粕）为原料，具有吸水性好、手感柔软等特点，应用于人造丝衣服、面膜布、湿纸巾等。2024-2025年业绩承压，2025年上半年营收37.38亿元、净利润6274.69万元，同比分别-1.52%、-58.58%，主因粘胶价格、需求波动。机构预测2025-2027年业绩有望逐步改善。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **粘胶长丝** | 人造丝长丝 | 高端服装、丝绸替代 |
| **粘胶短纤** | 人造棉短纤 | 服装、家纺、非织造布、面膜 |
| **氨纶** | 弹性纤维（部分布局） | 纺织 |
| **浆粕** | 粘胶原料 | 自用 |

### 业务模式

公司采用"浆粕-粘胶"一体化模式，从植物纤维浆粕到粘胶长丝、短纤。粘胶纤维与涤纶、锦纶形成差异化，吸水性好、透气，在高端服装、面膜、湿巾等有应用。2026年2月粘胶短纤市场报价约12800元/吨。

### 技术路线/核心技术

- **粘胶工艺**：浆粕溶解、纺丝、后处理
- **差别化粘胶**：功能性、差异化产品
- **一体化**：浆粕自给或外采
- **环保**：粘胶生产环保要求高

**技术优势**：
粘胶纤维龙头，粘胶长丝、短纤产能规模大。差别化、功能性粘胶是升级方向。与涤纶形成差异化，在吸湿、透气领域有优势。

## 四、行业地位与产业链定位

### 产业链定位

```
粘胶纤维产业链：
浆粕（植物纤维）→ 【新乡化纤 粘胶长丝/短纤】→ 纺织/非织造 → 服装/面膜/湿巾
                        ↑
                  新乡化纤（龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 粘胶纤维龙头，长丝、短纤产能领先 |
| **竞争优势** | 一体化、差别化、规模效应 |
| **客户结构** | 纺织、服装、非织造、面膜 |
| **供应商** | 浆粕自给或外采 |

### 竞争格局

**主要竞争对手**：
- 吉林化纤：粘胶+碳纤维
- 南京化纤：粘胶
- 澳洋健康：已剥离化纤
- 中泰化学：粘胶

**公司优势**：
粘胶纤维龙头，粘胶长丝、短纤产能规模大。与涤纶形成差异化，在吸湿、透气、高端服装领域有优势。面膜、湿巾等非织造需求增长。

## 五、投资亮点

### 核心亮点

- ✅ **粘胶纤维龙头**：粘胶长丝、短纤产能领先
- ✅ **差异化**：吸水性好、透气，与涤纶差异化
- ✅ **应用拓展**：面膜、湿巾等非织造增长
- ✅ **机构看好**：2025-2027年业绩预测改善
- ✅ **拐点弹性**：若粘胶价格反弹，龙头受益

### 成长驱动因素

1. **粘胶拐点**
   - 粘胶价格若触底反弹
   - 浆粕-粘胶价差修复
   - 需求复苏

2. **应用拓展**
   - 面膜、湿巾等非织造需求
   - 高端服装、人造丝
   - 消费升级

3. **差别化**
   - 功能性粘胶
   - 高端产品占比提升

### 催化剂

- 粘胶价格反弹
- 纺织、非织造需求复苏
- 面膜、湿巾需求增长
- 浆粕价格回落

### 拐点逻辑标注

| 拐点类型 | 逻辑 | 催化剂 |
|----------|------|--------|
| **粘胶拐点** | 价格处于低位，若反弹龙头受益 | 需求复苏、价差修复 |
| **业绩拐点** | 2025年承压，机构预测2026-2027改善 | 价格反弹、需求增长 |
| **应用拐点** | 面膜、湿巾等非织造增长 | 消费升级 |
