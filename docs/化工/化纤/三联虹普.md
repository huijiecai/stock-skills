## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 北京三联虹普新合纤技术服务股份有限公司 |
| 证券代码 | 300384 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1999年 |
| 上市时间 | 2014年8月1日 |
| 所属行业 | 专用设备制造业 |
| 市场地位 | 聚酯/化纤成套设备龙头，工业AI智能体 |

**公司概况**：
三联虹普是国际先进的聚合物成套生产工艺技术提供商，在聚酯及相关材料领域提供从方案咨询、研发设计、生产制造、施工管理到运营管理的"交钥匙"系统集成解决方案。2024年营收10.7亿元（-14.1%）、归母净利润3.18亿元（+9.6%），利润逆势增长。公司布局功能性聚酯(vPET)、再生聚酯(rPET)、再生聚酰胺(rPA)、生物基化学纤维等，工业AI智能体有望带来新增长。2024年获国家科技进步二等奖、部级科技进步一等奖，知识产权268项。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **聚酯成套设备** | 功能性聚酯(vPET)、PC、PBT生产线 | 化纤、工程塑料 |
| **再生材料设备** | 再生聚酯(rPET)、再生聚酰胺(rPA) | 循环经济、环保 |
| **生物基纤维设备** | 生物基化学纤维生产线 | 绿色化纤 |
| **系统集成** | 交钥匙工程，咨询-设计-制造-施工-运营 | 化纤企业 |
| **工业AI智能体** | 化纤工业智能体解决方案V1.0 | 智能制造 |

### 业务模式

公司采用"技术+装备+服务"模式，为化纤、新材料企业提供成套生产线及系统集成服务。订单驱动，项目周期较长。2024年营收下滑主因订单节奏，净利润增长反映毛利率提升、费用控制或高毛利订单占比提升。再生材料、可降解材料是成长方向。

### 技术路线/核心技术

- **聚酯技术**：vPET、PC、PBT等聚合物工艺
- **再生技术**：化学法再生rPET、rPA
- **聚合+SSP**：系统解决方案
- **工业AI**：化纤工业智能体，融入核心装备
- **知识产权**：268项，国家科技进步二等奖

**技术优势**：
聚酯/化纤成套设备技术领先，交钥匙能力突出。再生材料、可降解材料是政策方向，化学法再生技术布局领先。工业AI与装备融合，智能制造赋能。尼龙在轨道交通、电力电子、汽车零部件等需求释放。

## 四、行业地位与产业链定位

### 产业链定位

```
化纤设备产业链：
【三联虹普 聚酯/化纤成套设备】→ 化纤企业扩产/技改 → 涤纶/锦纶/再生材料
            ↑
      三联虹普（设备龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 聚酯/化纤成套设备龙头 |
| **竞争优势** | 技术领先、交钥匙能力、再生材料布局 |
| **客户结构** | 桐昆、新凤鸣、恒逸等化纤龙头 |
| **供应商** | 零部件、原材料 |

### 竞争格局

**主要竞争对手**：
- 聚酯设备：国外企业、国内设备商
- 再生材料设备：新兴赛道，公司布局领先
- 工业AI：装备+AI融合，差异化

**公司优势**：
聚酯/化纤成套设备技术领先，客户覆盖行业龙头。再生材料、可降解材料设备是政策方向，化学法再生布局领先。工业AI智能体赋能，差异化竞争。若化纤拐点到来、扩产重启，设备需求复苏。

## 五、投资亮点

### 核心亮点

- ✅ **聚酯设备龙头**：化纤企业扩产、技改核心供应商
- ✅ **拐点受益**：若PTA/聚酯拐点→扩产重启→设备需求
- ✅ **再生材料**：rPET、rPA设备，政策方向
- ✅ **工业AI**：智能体解决方案，新增长点
- ✅ **高盈利**：净利率约30%，技术壁垒高

### 成长驱动因素

1. **化纤拐点传导**
   - PTA/聚酯拐点→化纤企业盈利改善
   - 扩产、技改重启→设备订单
   - 公司是核心供应商

2. **再生材料**
   - 化学法再生rPET、rPA
   - 可降解材料、生物基纤维
   - 政策鼓励，需求增长

3. **工业AI**
   - 化纤工业智能体V1.0
   - 装备+AI融合
   - 智能制造赋能

### 催化剂

- PTA/聚酯拐点→化纤扩产重启
- 再生材料政策、订单
- 工业AI智能体商业化
- 尼龙在轨道交通、汽车等需求
