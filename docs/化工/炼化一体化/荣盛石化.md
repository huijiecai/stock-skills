## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 荣盛石化股份有限公司 |
| 证券代码 | 002493 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2007年6月18日 |
| 上市时间 | 2010年11月2日 |
| 所属行业 | 石油加工、炼焦和核燃料加工业 |
| 市场地位 | 民营大炼化龙头，浙石化4000万吨炼化 |

**公司概况**：
荣盛石化是民营石化龙头企业，由李水荣家族创立。公司控股浙江石油化工有限公司（浙石化）51%股权，浙石化拥有4000万吨/年炼油、880万吨/年对二甲苯、420万吨/年乙烯的加工能力，是"十三五"期间最大的民营石化项目。2019年投产至2024年底累计加工原油1.8亿吨，工业产值突破1万亿元。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **炼化产品** | 成品油、芳烃、烯烃及下游 | 能源、化工原料 |
| **对二甲苯(PX)** | 880万吨/年产能 | PTA原料、聚酯产业链 |
| **乙烯及下游** | 420万吨/年乙烯 | 聚乙烯、乙二醇、苯乙烯等 |
| **PTA** | 对苯二甲酸 | 聚酯原料 |
| **聚酯** | 涤纶长丝、瓶片等 | 纺织、包装 |
| **新材料** | 高端聚烯烃、EVA等 | 新能源、高端材料 |

### 业务模式

公司采用"炼化-石化-新材料"一体化模式，浙石化项目位于浙江舟山绿色石化基地，总投资约1730亿元。一期2019年末投产，二期2022年1月投产。公司持续向高附加值新材料延伸，强链补链，提升产业链竞争力。

### 技术路线/核心技术

- **大炼化一体化**：4000万吨炼油+880万吨PX+420万吨乙烯，全球领先规模
- **混合所有制**：民企控股、国企参股，资源协同
- **舟山基地**：临港优势，物流成本低，出口便利
- **新材料布局**：EVA、高端聚烯烃、POE等新能源材料

**技术优势**：
浙石化采用国际先进炼化技术，装置规模大、能耗低、产品结构优。从炼油到芳烃、烯烃及下游新材料全覆盖，具备向高附加值产品延伸的能力。

## 四、行业地位与产业链定位

### 产业链定位

```
炼化一体化产业链：
原油 → 炼化 → 【荣盛石化 芳烃/烯烃/新材料】→ PTA/聚酯/新材料 → 纺织/新能源/高端材料
                    ↑
              荣盛石化（浙石化）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 4000万吨炼化规模国内民营第一，PX、乙烯产能领先 |
| **竞争优势** | 大炼化规模效应、舟山临港优势、新材料布局 |
| **客户结构** | 覆盖聚酯、新材料、新能源等下游头部企业 |
| **供应商** | 原油进口，与沙特阿美等战略合作 |

### 竞争格局

**主要竞争对手**：
- 恒力石化：PTA全球第一，炼化一体化
- 恒逸石化：PTA、聚酯产能
- 中国石化、中国石油：国有炼化龙头

**公司优势**：
荣盛石化凭借浙石化4000万吨炼化规模、舟山基地区位优势、以及向EVA、高端聚烯烃等新材料延伸的布局，在民营大炼化中具备独特竞争力。现金流充沛支撑后续资本开支和债务偿还。

## 五、投资亮点

### 核心亮点

- ✅ **大炼化龙头**：4000万吨炼化、880万吨PX、420万吨乙烯，规模国内民营第一
- ✅ **舟山基地**：临港优势显著，物流成本低，出口便利
- ✅ **新材料布局**：EVA、高端聚烯烃、POE等新能源材料，成长空间大
- ✅ **现金流充沛**：346亿元经营现金流，支撑稳健经营和债务管理
- ✅ **强链补链**：持续向高附加值产业延伸，提升盈利能力

### 成长驱动因素

1. **炼化景气修复**
   - 炼化价差有望随原油价格企稳改善
   - 下游需求复苏带动芳烃、烯烃需求

2. **新材料放量**
   - EVA光伏料需求增长
   - 高端聚烯烃国产替代
   - POE等新能源材料

3. **产能利用率提升**
   - 装置运行效率优化
   - 产品结构持续调整

### 催化剂

- 原油价格企稳改善炼化利润
- 光伏EVA需求增长
- 新材料项目投产
- 与沙特阿美等战略合作深化
