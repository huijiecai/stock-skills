## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 恒力石化股份有限公司 |
| 证券代码 | 600346 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1999年3月9日 |
| 上市时间 | 2001年8月20日（2016年资产重组后更名） |
| 所属行业 | 石油加工、炼焦和核燃料加工业 |
| 市场地位 | 民营炼化龙头，PTA产能全球第一 |

**公司概况**：
恒力石化是行业内首家实现"原油-芳烃、烯烃-PTA、乙二醇-聚酯新材料"全产业链一体化的民营炼化龙头。公司PTA年产能达1700万吨，稳居全球首位，拥有2000万吨/年炼化一体化项目及500万吨/年现代煤化工装置，大连长兴岛1200万吨PTA基地为全球最大单一PTA生产基地。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **PTA** | 对苯二甲酸，聚酯核心原料 | 涤纶纤维、聚酯瓶片、薄膜 |
| **炼化产品** | 芳烃、烯烃及下游衍生物 | 化工原料、新材料 |
| **聚酯产品** | 涤纶长丝、聚酯切片、瓶片 | 纺织服装、包装、薄膜 |
| **乙二醇** | 聚酯另一核心原料 | 与PTA配套生产聚酯 |
| **煤化工产品** | 甲醇、乙二醇等 | 化工原料 |

### 业务模式

公司采用"炼化-石化-聚酯"全产业链一体化模式，从原油加工到终端聚酯产品全覆盖。通过规模化生产、技术升级和数字化转型降本增效，产品销往国内外市场。2024年持续强化上游产业链布局、优化产业结构。

### 技术路线/核心技术

- **英威达PTA技术**：大连、惠州基地采用国际先进的英威达绿色化、智能化PTA生产工艺
- **炼化一体化**：2000万吨/年炼化项目，实现原油到芳烃、烯烃的深度加工
- **煤化工耦合**：500万吨/年现代煤化工装置，多元化原料来源
- **一体化成本优势**：原料自给率高，中间环节损耗低

**技术优势**：
两处PTA基地均采用行业领先的绿色化、智能化生产工艺，具有"投资规模大、生产能力强、工艺水平高、能源消耗低"等特点。惠州产业园500万吨PTA项目于2023年全面投产，进一步巩固全球产能领先地位。

## 四、行业地位与产业链定位

### 产业链定位

```
炼化一体化产业链：
原油/煤炭 → 炼化 → 【恒力石化 PTA/乙二醇/聚酯】→ 涤纶/瓶片/薄膜 → 纺织/包装/新材料
                        ↑
                  恒力石化
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | PTA产能1700万吨/年全球第一，市占率领先 |
| **竞争优势** | 全产业链一体化、规模效应、技术领先、成本优势 |
| **客户结构** | 覆盖聚酯、纺织、包装等下游头部企业 |
| **供应商** | 炼化自给原料，上游一体化程度高 |

### 竞争格局

**主要竞争对手**：
- 荣盛石化：浙石化4000万吨炼化，大炼化+新材料布局
- 恒逸石化：PTA、聚酯产能规模较大
- 桐昆股份：聚酯长丝龙头
- 新凤鸣：聚酯产能扩张

**公司优势**：
恒力石化凭借PTA全球第一产能、炼化一体化全产业链布局、大连+惠州双基地协同，在民营炼化中具备显著规模与成本优势。2024年产品提价能力增强，盈利修复明显。

## 五、投资亮点

### 核心亮点

- ✅ **PTA全球龙头**：1700万吨产能全球第一，定价权强，规模效应显著
- ✅ **全产业链一体化**：原油到聚酯全覆盖，中间环节损耗低，成本优势突出
- ✅ **双基地布局**：大连长兴岛1200万吨+惠州500万吨，分散区域风险
- ✅ **盈利修复**：2024年上半年净利同比+31.77%，毛利率大幅提升
- ✅ **高分红**：2024年派现0.45元/股，现金流充沛支撑持续分红

### 成长驱动因素

1. **炼化景气修复**
   - 原油价格企稳，炼化价差有望改善
   - 下游聚酯需求稳健，PTA供需格局向好

2. **产能释放与技改**
   - 惠州PTA项目全面投产贡献增量
   - 持续技改提升装置运行效率

3. **产品结构优化**
   - 向高附加值新材料延伸
   - 高端聚酯、可降解材料等新方向

### 催化剂

- 原油价格回落改善炼化利润
- 纺织服装出口复苏带动聚酯需求
- 产品提价传导至业绩
- 新材料项目投产贡献增量
