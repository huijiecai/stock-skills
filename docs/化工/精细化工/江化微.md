## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江阴江化微电子材料股份有限公司 |
| 证券代码 | 603078 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2001年 |
| 上市时间 | 2017年4月10日 |
| 所属行业 | 化学原料和化学制品制造业（湿电子化学品） |
| 市场地位 | 湿电子化学品国产替代先锋，显示面板市占率超80% |

**公司概况**：
江化微专注超净高纯试剂和光刻胶配套试剂，是国内湿电子化学品国产替代先锋。2024年营收10.99亿元，净利润9863万元，毛利率25.31%。在显示面板中低端市场占有率超80%，半导体领域国产化率提升至44%。2025年一季度营收2.75亿元同比+12.92%，半导体领域持续突破。湿电子化学品+光刻胶双轮驱动。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **超净高纯试剂** | 湿法工艺用高纯化学品 | 半导体、显示面板 |
| **光刻胶配套试剂** | 显影液、剥离液等 | 半导体、显示光刻 |
| **湿电子化学品** | 刻蚀液、清洗液等 | 半导体制造 |
| **显示用化学品** | 显示面板用 | 显示面板 |

### 业务模式

公司采用研产销一体化模式，超净高纯试剂和光刻胶配套试剂为核心。显示面板中低端市占率超80%，半导体领域国产化率提升至44%。光刻胶突围书写新篇章，半导体高端市场拓展。

### 技术路线/核心技术

- **超净高纯**：G2-G5级湿电子化学品
- **光刻胶配套**：显影液、剥离液
- **半导体**：刻蚀、清洗等湿法工艺

**技术优势**：
湿电子化学品国产替代先锋，显示面板市占率超80%。半导体领域国产化率44%，持续提升。光刻胶配套试剂拓展。

## 五、投资亮点

### 核心亮点

- ✅ **湿电子化学品国产替代先锋**：显示市占率超80%，半导体44%
- ✅ **拐点逻辑**：①**国产替代**——湿电子化学品国产化率提升，半导体领域44%；②**半导体周期**——若半导体景气回升，湿电子化学品需求放量；③**光刻胶突围**——光刻胶配套试剂拓展新篇章；④**业绩拐点**——2025年一季度营收+12.92%
- ✅ **显示龙头**：中低端市占率超80%
- ✅ **半导体突破**：国产化率44%，持续提升

### 成长驱动因素

1. **国产替代**
   - 湿电子化学品国产化
   - 半导体领域44%→更高
   - 高端产品突破

2. **半导体周期**
   - 半导体景气回升
   - 湿电子化学品需求
   - 晶圆厂扩产

3. **光刻胶配套**
   - 显影液、剥离液等
   - 光刻胶产业链延伸
   - 高附加值

### 催化剂

- 半导体景气回升
- 半导体客户突破
- 光刻胶配套放量
- 高端产品认证
