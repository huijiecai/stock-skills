## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏联瑞新材料股份有限公司 |
| 证券代码 | 688300 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2002年 |
| 上市时间 | 2019年11月15日 |
| 所属行业 | 非金属矿物制品业（功能性硅烷/填料） |
| 市场地位 | 功能性硅烷、球形硅微粉等电子封装材料龙头 |

**公司概况**：
联瑞新材主营功能性硅烷、球形硅微粉、球形氧化铝等产品，应用于电子封装、半导体、覆铜板、导热材料等领域。公司为生益科技等覆铜板龙头供应商，电子封装材料国产替代核心标的。功能性硅烷、球形填料技术壁垒高，半导体、新能源等高景气下游驱动成长。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **球形硅微粉** | 高纯球形二氧化硅 | 电子封装、覆铜板、半导体 |
| **球形氧化铝** | 导热填料 | 导热材料、电子封装 |
| **功能性硅烷** | 硅烷偶联剂等 | 复合材料、电子、新能源 |
| **角形硅微粉** | 角形二氧化硅 | 覆铜板、封装 |
| **其他** | 熔融硅微粉等 | 多元化 |

### 业务模式

公司采用研产销一体化模式，球形硅微粉、球形氧化铝、功能性硅烷为核心。覆铜板、电子封装、半导体、导热材料等下游。生益科技等龙头客户，国产替代逻辑。

### 技术路线/核心技术

- **球形硅微粉**：高纯球形二氧化硅，电子级
- **球形氧化铝**：导热填料，球形化技术
- **功能性硅烷**：硅烷偶联剂
- **电子封装**：覆铜板、半导体封装材料

**技术优势**：
球形硅微粉、球形氧化铝技术壁垒高，电子封装材料国产替代。功能性硅烷品种丰富，复合材料、新能源应用。

## 四、行业地位与产业链定位

### 产业链定位

```
电子封装材料产业链：
石英/铝等 → 【联瑞新材 球形硅微粉/氧化铝/硅烷】→ 覆铜板/封装/半导体/导热 → 电子
                              ↑
                    联瑞新材（封装材料龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 球形硅微粉、球形氧化铝龙头 |
| **竞争优势** | 球形化技术、生益科技等客户、国产替代 |
| **客户结构** | 覆铜板、封装、半导体、导热 |
| **供应商** | 石英、铝等原料 |

### 竞争格局

**主要竞争对手**：
- 雅克科技：硅微粉（华飞电子）
- 三孚股份：硅烷偶联剂
- 宏柏新材：功能性硅烷
- 国际：电化、雅都玛等

**公司优势**：
球形硅微粉、球形氧化铝龙头，生益科技等覆铜板客户。电子封装材料国产替代，功能性硅烷品种丰富。

## 五、投资亮点

### 核心亮点

- ✅ **功能性硅烷/球形填料龙头**：球形硅微粉、球形氧化铝
- ✅ **拐点逻辑**：①**国产替代**——电子封装材料国产化，球形硅微粉、氧化铝替代进口；②**半导体周期**——若半导体景气回升，封装材料需求放量；③**新能源需求**——导热材料、复合材料新能源应用；④**覆铜板**——生益科技等覆铜板客户，AI、服务器需求
- ✅ **生益科技客户**：覆铜板龙头
- ✅ **球形化技术**：技术壁垒高

### 成长驱动因素

1. **国产替代**
   - 电子封装材料国产化
   - 球形硅微粉、氧化铝替代进口
   - 半导体封装

2. **半导体周期**
   - 半导体景气回升
   - 封装材料需求
   - 覆铜板、封装

3. **新能源需求**
   - 导热材料
   - 复合材料
   - 新能源车、储能

### 催化剂

- 半导体景气回升
- AI、服务器覆铜板需求
- 导热材料放量
- 新客户突破

> **【拐点逻辑】** 联瑞新材拐点取决于：①**国产替代**——电子封装材料国产化，球形硅微粉、球形氧化铝替代进口；②**半导体周期**——若半导体景气回升，封装材料、覆铜板需求放量；③**新能源需求**——导热材料、复合材料新能源应用；④**AI/服务器**——覆铜板、封装材料AI、服务器需求。**拐点信号**：半导体材料营收同比转正、球形填料销量增长、新客户订单落地。
