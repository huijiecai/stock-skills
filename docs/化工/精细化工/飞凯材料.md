## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海飞凯材料科技股份有限公司 |
| 证券代码 | 300398 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2002年 |
| 上市时间 | 2014年10月9日 |
| 所属行业 | 化学原料和化学制品制造业（电子化学品） |
| 市场地位 | 电子化学品企业，紫外固化材料、显示材料 |

**公司概况**：
飞凯材料是国内电子化学品企业，产品涵盖紫外固化材料、显示材料、半导体材料等，应用于光纤光缆、显示面板、半导体封装等领域。公司通过并购拓展半导体材料，电子化学品多品类布局。显示材料受周期影响，半导体材料为成长方向。国产替代逻辑支撑长期估值。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **紫外固化材料** | 光纤涂覆材料等 | 光纤光缆 |
| **显示材料** | 液晶、混晶等 | 显示面板 |
| **半导体材料** | 封装材料、湿电子化学品等 | 半导体封装、制造 |
| **屏幕显示材料** | 显示用化学品 | 显示面板 |

### 业务模式

公司采用研产销一体化+并购整合模式，紫外固化材料为传统优势，显示材料、半导体材料通过并购和自研拓展。多品类电子化学品，客户覆盖光纤、显示、半导体。

### 技术路线/核心技术

- **紫外固化**：光纤涂覆材料，技术成熟
- **显示材料**：液晶、混晶
- **半导体**：封装材料、湿电子化学品

**技术优势**：
紫外固化材料国内领先，显示材料、半导体材料布局。电子化学品多品类，国产替代受益。

## 五、投资亮点

### 核心亮点

- ✅ **电子化学品多品类**：紫外固化、显示、半导体
- ✅ **拐点逻辑**：①**国产替代**——电子化学品国产化率提升；②**半导体周期**——若半导体景气回升，半导体材料需求修复；③**显示修复**——显示材料周期见底；④**紫外固化**——光纤需求稳定
- ✅ **半导体材料**：封装、湿电子化学品成长
- ✅ **并购整合**：通过并购拓展半导体

### 成长驱动因素

1. **国产替代**
   - 电子化学品国产化
   - 半导体材料突破
   - 客户导入

2. **半导体周期**
   - 半导体景气回升
   - 半导体材料需求
   - 封装材料

3. **显示修复**
   - 显示面板周期见底
   - 显示材料需求企稳

### 催化剂

- 半导体景气回升
- 半导体材料客户突破
- 显示材料价格企稳
- 紫外固化稳健增长

## 七、业绩预期与估值

### 拐点逻辑
