## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江永太科技股份有限公司 |
| 证券代码 | 002326 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 2009年12月22日 |
| 所属行业 | 化学原料和化学制品制造业（含氟精细化学品） |
| 市场地位 | 含氟精细化学品龙头，医药/农药/电子化学品 |

**公司概况**：
永太科技是国内含氟精细化学品龙头，产品涵盖医药中间体、农药中间体、锂电材料、电子化学品等。含氟化学技术壁垒高，产品应用于医药、农药、新能源、半导体等领域。医药CDMO、锂电材料（如LiFSI）为成长方向。新能源需求与医药需求双轮驱动。

### 技术路线/核心技术

- **含氟化学**：氟化、氟烷基化等，技术壁垒高
- **医药**：含氟医药中间体、CDMO
- **锂电**：LiFSI、六氟磷酸锂等

**技术优势**：
含氟精细化学品龙头，技术壁垒高。医药、农药、锂电、电子多领域，产品线丰富。

## 五、投资亮点

### 核心亮点

- ✅ **含氟精细化学品龙头**：技术壁垒高，多领域
- ✅ **拐点逻辑**：①**新能源需求**——锂电材料LiFSI等，若新能源景气回升；②**医药需求**——医药CDMO、含氟医药中间体成长；③**国产替代**——含氟电子化学品国产化；④**农药**——含氟农药中间体稳定
- ✅ **多领域布局**：医药、农药、锂电、电子
- ✅ **LiFSI**：新型锂盐，成长空间

### 成长驱动因素

1. **新能源需求**
   - 锂电材料LiFSI
   - 六氟磷酸锂等
   - 新能源车、储能

2. **医药需求**
   - 医药CDMO
   - 含氟医药中间体
   - 创新药需求

3. **国产替代**
   - 含氟电子化学品
   - 半导体、显示
   - 高端产品

### 催化剂

- 新能源景气回升
- 锂电材料价格企稳
- 医药CDMO订单放量
- LiFSI渗透率提升
