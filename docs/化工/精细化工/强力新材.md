## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 常州强力电子新材料股份有限公司 |

| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2000年 |
| 上市时间 | 2015年3月24日 |
| 所属行业 | 化学原料和化学制品制造业（光刻胶） |
| 市场地位 | 光刻胶专用化学品领先企业，PCB/LCD/半导体全覆盖 |

**公司概况**：
强力新材是国内光刻胶专用化学品领域的领先企业，产品覆盖PCB干膜光刻胶、LCD光刻胶、半导体光刻胶等主要光刻胶种类的关键原材料。2023-2025年预测收入分别为9.27、15.2、20.97亿元，半导体领域再应用创新打造第二增长曲线。光刻胶国产替代核心标的，技术壁垒高。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **PCB光刻胶原料** | 干膜光刻胶用光引发剂等 | PCB制造 |
| **LCD光刻胶原料** | 彩色光刻胶、黑色光刻胶用 | 显示面板 |
| **半导体光刻胶原料** | g线、i线、KrF、ArF光刻胶用 | 半导体制造 |
| **其他电子化学品** | 光刻胶配套材料 | 电子制造 |

### 业务模式

公司采用研产销一体化模式，光刻胶原料为主业。PCB、LCD市场成熟，半导体光刻胶原料为成长重点。半导体领域再应用创新，打造第二增长曲线。

### 技术路线/核心技术

- **光引发剂**：光刻胶用光引发剂合成
- **光刻胶原料**：PCB、LCD、半导体全覆盖
- **半导体**：g线、i线、KrF、ArF光刻胶原料

**技术优势**：
光刻胶专用化学品国内领先，PCB、LCD、半导体全覆盖。半导体光刻胶原料国产替代核心，技术壁垒高。

## 五、投资亮点

### 核心亮点

- ✅ **光刻胶原料龙头**：PCB、LCD、半导体全覆盖
- ✅ **拐点逻辑**：①**国产替代**——光刻胶国产化率提升，g线/i线约10%、KrF约10%、ArF全部进口，替代空间大；②**半导体周期**——若半导体景气回升，半导体光刻胶需求放量；③**第二增长曲线**——半导体领域再应用创新；④**收入高增**——2024-2025年预测15.2、20.97亿元
- ✅ **半导体放量**：半导体光刻胶原料成长重点
- ✅ **技术壁垒**：光刻胶原料技术门槛高

### 成长驱动因素

1. **国产替代**
   - 光刻胶国产化率提升
   - 半导体光刻胶原料突破
   - ArF等高端原料研发

2. **半导体周期**
   - 半导体景气回升
   - 半导体光刻胶需求
   - 晶圆厂扩产

3. **显示/PCB**
   - 显示面板、PCB稳定需求
   - 高端产品渗透

### 催化剂

- 半导体景气回升
- 半导体光刻胶客户突破
- ArF光刻胶原料进展
- 光刻胶国产化政策

## 七、业绩预期与估值

### 拐点逻辑

> **【拐点逻辑】** 强力新材拐点取决于：①**国产替代**——光刻胶国产化率提升，半导体光刻胶原料放量；②**半导体周期**——若半导体景气回升，半导体光刻胶需求放量；③**第二增长曲线**——半导体领域再应用创新，收入预测2024年15.2亿、2025年20.97亿；④**高端突破**——KrF、ArF光刻胶原料进展。**拐点信号**：半导体光刻胶原料营收占比突破30%、半导体客户批量供货、单季营收同比高增。

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 半导体景气回升、半导体原料放量、高端突破 | 营收20亿+，净利润增速30%+ | 营收25亿+，净利润增速25%+ |
| **中性** | 半导体企稳、半导体原料稳步放量 | 营收18-22亿，净利润增速15-25% | 营收22-26亿，净利润增速15-20% |
| **悲观** | 半导体低迷、半导体原料不及预期 | 营收<18亿，净利润增速<15% | 营收<22亿，净利润增速<15% |

### 估值水平

| 指标 | 当前水平 | 行业平均 | 评估 |
|------|----------|----------|------|
| PE(TTM) | - | 光刻胶约40-60倍 | 国产替代高弹性 |
| PB | - | - | - |
| PS | - | - | - |

**估值分析**：
强力新材光刻胶原料龙头，国产替代核心标的。半导体第二增长曲线，若半导体景气回升弹性大。
