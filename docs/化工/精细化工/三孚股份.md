## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 唐山三孚硅业股份有限公司 |
| 证券代码 | 603938 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2017年6月（股份公司） |
| 上市时间 | 2017年6月28日 |
| 所属行业 | 化学原料和化学制品制造业（三氯氢硅/硅烷） |
| 市场地位 | 三氯氢硅龙头，功能性硅烷、电子级产品布局 |

**公司概况**：
三孚股份主营硅系列（三氯氢硅、高纯四氯化硅、电子级二氯二氢硅、电子级三氯氢硅）、钾系列（氢氧化钾、硫酸钾）、硅烷偶联剂等产品。旗下唐山三孚新材料建立涵盖10个系列50多个品种的功能性硅烷产品系列，应用于新能源、半导体、航空航天、复合材料等领域。2024年前三季度营收13.44亿元同比-18.21%，净利润5053万元同比-71.04%，主要因产品价格大幅下降（硅系列-46.28%、钾系列-11.64%、硅烷偶联剂-9.33%）。电子级产品已通过IATF16949认证，半导体领域供应。价格拐点后有望修复。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **三氯氢硅** | 多晶硅原料、硅烷偶联剂原料 | 光伏、硅烷 |
| **高纯四氯化硅** | 光纤、半导体 | 光纤、半导体 |
| **电子级二氯二氢硅** | 电子级硅烷 | 半导体 |
| **电子级三氯氢硅** | 电子级硅烷 | 半导体 |
| **氢氧化钾/硫酸钾** | 钾系列产品 | 化工、农业 |
| **硅烷偶联剂** | 10系列50多品种 | 新能源、半导体、复合材料 |

### 业务模式

公司采用一体化生产模式，三氯氢硅为核心，延伸至高纯四氯化硅、电子级硅烷、硅烷偶联剂。钾系列为配套。产品应用于光伏、半导体、新能源、复合材料等。电子级产品半导体认证。

### 技术路线/核心技术

- **三氯氢硅**：多晶硅、硅烷原料
- **电子级**：二氯二氢硅、三氯氢硅，半导体用
- **硅烷偶联剂**：10系列50多品种
- **认证**：IATF16949、ISO9001等

**技术优势**：
三氯氢硅龙头，电子级产品半导体认证。功能性硅烷10系列50多品种，新能源、半导体、航空航天应用。
