## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 成都硅宝科技股份有限公司 |
| 证券代码 | 300019 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1998年 |
| 上市时间 | 2009年10月30日 |
| 所属行业 | 化学原料和化学制品制造业（有机硅密封胶） |
| 市场地位 | 有机硅密封胶龙头，国家制造业单项冠军，高端胶粘剂国产先锋 |

**公司概况**：
硅宝科技以有机硅材料为主，是国家制造业单项冠军示范企业。2024年拥有9大生产基地、年产能37万吨，近三年工业用胶领域销售业绩年均增长超40%。产品应用于比亚迪、宁德时代等头部企业，高端胶粘剂国产先锋。建筑用胶、工业用胶双轮驱动，新能源、电子等高景气下游。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **建筑用胶** | 门窗密封、幕墙、中空玻璃 | 建筑 |
| **工业用胶** | 电子、新能源、汽车等 | 电子、新能源、汽车 |
| **光伏胶** | 光伏组件封装 | 光伏 |
| **电子胶** | 电子电器密封、导热 | 电子 |
| **新能源车胶** | 电池包、车身等 | 新能源汽车 |
| **其他** | 锂电池用胶等 | 多元化 |

### 业务模式

公司采用研产销一体化模式，有机硅密封胶为核心。9大生产基地、年产能37万吨，规模优势。建筑用胶为传统优势，工业用胶（电子、新能源、汽车）近三年年均增长超40%。比亚迪、宁德时代等头部客户。

### 技术路线/核心技术

- **有机硅**：硅酮密封胶、导热胶
- **建筑用胶**：门窗、幕墙、中空玻璃
- **工业用胶**：电子、新能源、汽车
- **高端应用**：锂电池、光伏、电子

**技术优势**：
有机硅密封胶国家制造业单项冠军，高端胶粘剂国产先锋。工业用胶近三年年均增长超40%，比亚迪、宁德时代等头部客户。

## 四、行业地位与产业链定位

### 产业链定位

```
有机硅密封胶产业链：
有机硅单体 → 【硅宝科技 有机硅密封胶】→ 建筑/电子/新能源/汽车 → 终端
                              ↑
                    硅宝科技（有机硅龙头）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 有机硅密封胶龙头，胶粘剂三巨头第二 |
| **竞争优势** | 国家制造业单项冠军、比亚迪/宁德时代客户、产能 |
| **客户结构** | 建筑、电子、新能源、汽车龙头 |
| **供应商** | 有机硅单体 |

### 竞争格局

**主要竞争对手**：
- 回天新材：胶粘剂第一
- 康达新材：环氧、丙烯酸酯
- 其他有机硅企业

**公司优势**：
有机硅密封胶国家制造业单项冠军，高端胶粘剂国产先锋。工业用胶高增长，比亚迪、宁德时代等头部客户。

## 七、业绩预期与估值
