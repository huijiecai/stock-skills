## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海康达新材料集团股份有限公司 |
| 证券代码 | 002669 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1988年 |
| 上市时间 | 2012年4月16日 |
| 所属行业 | 化学原料和化学制品制造业（胶粘剂） |
| 市场地位 | 胶粘剂三巨头之一，军工电子、半导体材料布局 |

**公司概况**：
康达新材是国内胶粘剂行业三巨头之一，2024年营收31.01亿元同比+11.05%。在环氧树脂胶、丙烯酸酯胶等领域优势明显，通过并购整合（京瀚禹、晶材科技等）拓展至军工电子、半导体材料等高附加值领域。2024年受商誉减值约1.3-1.5亿元、新产能爬坡成本、下游需求收缩等因素影响，净利润亏损2.46亿元。胶粘剂主业稳健，军工电子、半导体为成长方向，商誉减值释放后有望轻装上阵。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **环氧树脂胶** | 结构胶、灌封胶等 | 风电、电子、军工 |
| **丙烯酸酯胶** | 丙烯酸类胶粘剂 | 汽车、电子 |
| **军工电子** | 京瀚禹等，电子检测 | 军工 |
| **半导体材料** | 晶材科技等 | 半导体 |
| **其他胶粘剂** | 聚氨酯、硅酮等 | 多元化 |

### 业务模式

公司采用"胶粘剂+军工电子+半导体"多元布局。胶粘剂为主业，环氧树脂胶、丙烯酸酯胶优势明显。通过并购京瀚禹、晶材科技等拓展军工电子、半导体材料。新产能爬坡、商誉减值短期拖累业绩。

### 技术路线/核心技术

- **环氧树脂胶**：结构胶、灌封胶，风电、电子
- **丙烯酸酯胶**：汽车、电子
- **军工电子**：京瀚禹电子检测
- **半导体**：晶材科技半导体材料

**技术优势**：
胶粘剂技术积累深，环氧、丙烯酸酯优势。军工电子、半导体高附加值布局。并购整合拓展成长空间。
