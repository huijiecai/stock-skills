## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 湖北回天新材料股份有限公司 |
| 证券代码 | 300041 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1998年 |
| 上市时间 | 2010年1月8日 |
| 所属行业 | 化学原料和化学制品制造业（胶粘剂） |
| 市场地位 | 国内工程胶粘剂首家上市公司，胶粘剂行业第一 |

**公司概况**：
回天新材是国内工程胶粘剂首家上市公司，胶粘剂行业营收第一，2024年约39.89亿元。产品线丰富，覆盖光伏新能源、新能源汽车、电子电器等高端领域。胶粘剂三巨头争霸格局中稳居第一，光伏、新能源车、电子等高景气下游驱动成长。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **光伏胶粘剂** | 光伏组件封装、边框密封等 | 光伏新能源 |
| **新能源汽车胶** | 电池包、电机、车身胶 | 新能源汽车 |
| **电子胶** | 电子电器用胶粘剂 | 电子电器 |
| **工程胶** | 建筑、工业用胶 | 建筑、工业 |
| **其他** | 包装、轨道交通等 | 多元化 |

### 业务模式

公司采用大客户战略，光伏、新能源车、电子等高端领域客户为主。产品线丰富，工程胶粘剂全覆盖。光伏、新能源车高景气下游，驱动业绩增长。

### 技术路线/核心技术

- **光伏胶**：组件封装、边框密封，光伏专用
- **新能源车胶**：电池包、电机、车身
- **电子胶**：电子电器密封、导热
- **工程胶**：硅酮、聚氨酯、环氧等

**技术优势**：
工程胶粘剂技术积累深，光伏、新能源车、电子高端应用。国内工程胶粘剂首家上市公司，行业第一。

## 四、行业地位与产业链定位

### 产业链定位

```
胶粘剂产业链：
化工原料 → 【回天新材 工程胶粘剂】→ 光伏/新能源车/电子/建筑 → 终端
                              ↑
                    回天新材（行业第一）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 胶粘剂行业第一，营收约40亿 |
| **竞争优势** | 光伏、新能源车、电子、产品线 |
| **客户结构** | 光伏龙头、新能源车、电子企业 |
| **供应商** | 化工原料 |

### 竞争格局

**主要竞争对手**：
- 硅宝科技：有机硅胶，营收约32亿
- 康达新材：环氧、丙烯酸酯，营收约31亿
- 其他胶粘剂企业

**公司优势**：
胶粘剂行业第一，光伏、新能源车、电子高端领域。产品线丰富，三巨头争霸领先。

## 五、投资亮点

### 核心亮点

- ✅ **胶粘剂行业第一**：营收约40亿，三巨头领先
- ✅ **拐点逻辑**：①**新能源需求**——光伏、新能源车高景气，胶粘剂需求旺盛；②**高端应用**——光伏、新能源车、电子高端领域；③**国产替代**——高端胶粘剂国产化；④**产品线**——工程胶粘剂全覆盖，客户粘性强
- ✅ **光伏+新能源车**：高景气下游
- ✅ **产品线丰富**：工程胶粘剂全覆盖

### 成长驱动因素

1. **新能源需求**
   - 光伏装机量
   - 新能源车产销
   - 储能、电池包
   - 高端胶粘剂需求

2. **高端应用**
   - 光伏组件封装
   - 新能源车电池包、电机
   - 电子电器
   - 国产替代

3. **产品线**
   - 工程胶粘剂全覆盖
   - 多下游分散风险
   - 客户粘性强

### 催化剂

- 光伏装机超预期
- 新能源车产销高增
- 电子需求回暖
- 高端胶粘剂突破
