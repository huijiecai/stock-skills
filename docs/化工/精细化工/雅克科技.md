## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏雅克科技股份有限公司 |
| 证券代码 | 002409 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1997年 |
| 上市时间 | 2010年5月25日 |
| 所属行业 | 化学原料和化学制品制造业（半导体材料） |
| 市场地位 | 半导体材料平台型龙头，前驱体、电子特气、光刻胶 |

**公司概况**：
雅克科技是国内半导体材料平台型龙头，通过并购整合形成前驱体、电子特气、光刻胶、硅微粉等产品矩阵。2024年上半年营收32.57亿元同比+40.21%，归母净利润5.2亿元同比+52.19%。业绩增长受益于集成电路行业恢复性增长、AI芯片对HBM等高端存储芯片需求增加。商誉高企至18.57亿元，需关注减值风险。半导体材料国产替代核心标的。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **前驱体** | 半导体薄膜沉积前驱体 | 半导体制造 |
| **电子特气** | 半导体用特种气体 | 半导体制造 |
| **光刻胶** | 半导体光刻胶 | 半导体制造 |
| **硅微粉** | 封装用硅微粉 | 半导体封装 |
| **LNG保温材料** | 液化天然气保温 | 能源 |

### 业务模式

公司采用并购整合模式，通过收购UP Chemical、科美特、华飞电子等形成半导体材料平台。前驱体、电子特气、光刻胶、硅微粉覆盖半导体制造多环节。LNG保温材料为传统业务。

### 技术路线/核心技术

- **前驱体**：半导体薄膜沉积用前驱体，高壁垒
- **电子特气**：半导体制造用特种气体
- **光刻胶**：半导体光刻胶
- **硅微粉**：封装材料

**技术优势**：
半导体材料平台型，前驱体、电子特气、光刻胶多品类。AI芯片、HBM等高端存储需求驱动。
