## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海新阳半导体材料股份有限公司 |
| 证券代码 | 300236 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1999年 |
| 上市时间 | 2011年6月29日 |
| 所属行业 | 化学原料和化学制品制造业（电子化学品） |
| 市场地位 | 电子电镀、电子清洗龙头，193nm ArF光刻胶研发中 |

**公司概况**：
上海新阳拥有电子电镀和电子清洗两大核心技术，产品已达世界先进水平。已立项研发集成电路制造用193nm ArF光刻胶及配套材料，高端光刻胶突破在即。电子化学品覆盖半导体制造多环节，国产替代核心标的。半导体周期与国产替代双逻辑。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **电子电镀** | 电镀液、添加剂等 | 半导体制造、封装 |
| **电子清洗** | 清洗液、去胶液等 | 半导体制造 |
| **光刻胶** | 193nm ArF光刻胶研发中 | 半导体制造 |
| **其他电子化学品** | 半导体制造用化学品 | 半导体 |

### 业务模式

公司采用研产销一体化模式，电子电镀、电子清洗为核心。193nm ArF光刻胶及配套材料研发立项，高端光刻胶突破。半导体制造多环节覆盖，客户粘性强。

### 技术路线/核心技术

- **电子电镀**：电镀液、添加剂，世界先进水平
- **电子清洗**：清洗液、去胶液
- **光刻胶**：193nm ArF光刻胶及配套材料研发

**技术优势**：
电子电镀、电子清洗技术世界先进水平。193nm ArF光刻胶研发，高端光刻胶国产替代。半导体制造多环节覆盖。
