## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 安集微电子科技（上海）股份有限公司 |
| 证券代码 | 688019 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2006年 |
| 上市时间 | 2019年7月22日 |
| 所属行业 | 化学原料和化学制品制造业（CMP材料） |
| 市场地位 | 国产CMP抛光液领军者，28nm及以上全面进口替代 |

**公司概况**：
安集科技深耕半导体材料研发，是国产CMP抛光液领军者。2024年营收18.3亿元，综合毛利率37.80%，净利率17.80%。铜互连CMP抛光液28nm及以上工艺全面进口替代，在国产芯片制造领域市场占有率第一，产品进入中芯国际、长江存储等主流晶圆厂。CMP材料国产替代核心标的，半导体周期与国产替代双逻辑。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **CMP抛光液** | 铜互连、钨、硅、氧化物等 | 半导体制造 |
| **光刻胶去除剂** | 光刻胶剥离、清洗 | 半导体制造 |
| **电镀液** | 半导体电镀 | 半导体制造 |
| **其他** | 半导体制造用化学品 | 半导体 |

### 业务模式

公司采用研产销一体化模式，CMP抛光液为核心产品。铜互连CMP抛光液28nm及以上全面进口替代，国产芯片制造市占率第一。产品矩阵拓展，光刻胶去除剂、电镀液等。客户覆盖中芯国际、长江存储等主流晶圆厂。

### 技术路线/核心技术

- **CMP抛光液**：铜、钨、硅、氧化物等，28nm及以上
- **光刻胶去除剂**：光刻胶剥离、清洗
- **电镀液**：半导体电镀
- **产品矩阵**：CMP+去除剂+电镀液等

**技术优势**：
CMP抛光液国产领军，28nm及以上全面进口替代。国产芯片制造市占率第一，客户粘性强。产品矩阵拓展，打造半导体材料平台。
