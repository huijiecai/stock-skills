## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 雅本化学股份有限公司 |
| 证券代码 | 300261 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2006年 |
| 上市时间 | 2011年9月6日 |
| 所属行业 | 化学原料和化学制品制造业（农药中间体、医药中间体） |
| 市场地位 | 农药中间体、医药创新中间体 |

**公司概况**：
雅本化学主要从事农药中间体、医药中间体的研发、生产和销售。2024年受农化市场景气度低位、核心农药产品竞争加剧、销售价格明显下滑影响，预计亏损1.85-2.65亿元。农药、医药创新中间体新项目尚处于立项、建设或产能爬坡阶段，预计计提大额商誉减值。第二季度下游需求有所回暖，农药产品销售额相较一季度回升，但全年仍承压。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **农药中间体** | 农药合成关键中间体 | 农药原药生产 |
| **医药中间体** | 医药创新中间体 | 医药研发生产 |
| **农药原药** | 农药原药 | 农化产业链 |
| **医药CDMO** | 医药定制研发生产 | 医药企业 |

### 业务模式

公司采用农药+医药双轮驱动模式。农药中间体面向农化企业，医药中间体面向医药企业。新项目（农药、医药创新中间体）尚处于立项、建设或产能爬坡阶段，未产生积极贡献。2024年农化市场景气度低位，业绩承压。

### 技术路线/核心技术

- **农药中间体**：农药合成关键中间体工艺
- **医药中间体**：医药创新中间体、CDMO
- **新项目**：农药、医药创新中间体拓展中

**技术优势**：
农药中间体、医药中间体双轮驱动。新项目储备长期成长，但短期处于投入期。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
化工原料 → 【雅本化学 农药中间体】→ 农药原药 → 制剂 → 农业种植
                    ↑
                雅本化学
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 农药中间体主要供应商 |
| **竞争优势** | 农药+医药双轮、新项目储备 |
| **客户结构** | 农化企业、医药企业 |
| **供应商** | 化工原料采购 |

### 竞争格局

**主要竞争对手**：
- 联化科技：农药中间体、医药
- 苏利股份：农药中间体、阻燃剂
- 福华通达：农药中间体

**公司现状**：
2024年业绩承压，新项目投入期。若农药行业拐点、新项目爬坡，有望改善。

## 五、投资亮点

### 核心亮点

- ✅ **农药+医药双轮**：分散单一行业风险，医药创新中间体长期空间
- ✅ **新项目储备**：农药、医药创新中间体拓展，爬坡后有望贡献
- ✅ **拐点逻辑**：农药行业拐点来临，中间体需求回升
- ⚠️ **短期承压**：2024年亏损扩大，新项目投入期
- ✅ **Q2回暖**：第二季度下游需求有所回暖，农药销售额回升

### 成长驱动因素

1. **农药行业拐点**
   - 农药行业去库尾声、补库启动
   - 中间体需求回升
   - 产品价格企稳

2. **新项目爬坡**
   - 农药、医药创新中间体项目投产
   - 产能爬坡后贡献业绩

3. **医药业务**
   - 医药创新中间体、CDMO
   - 长期成长空间

### 催化剂

- **农药价格**：农药行业拐点、中间体需求回升
- **新项目**：农药、医药创新中间体项目投产
- **商誉**：商誉减值一次性影响后轻装上阵
