## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏长青农化股份有限公司 |
| 证券代码 | 002391 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001年 |
| 上市时间 | 2010年4月16日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 麦草畏龙头，产能约1.1万吨/年 |

**公司概况**：
长青股份是国内麦草畏龙头，主要从事农药原药、制剂的生产和销售。麦草畏产能约1.1万吨/年，2020年美国禁用麦草畏后需求持续受压。2024年全年净亏损1.20亿元，但2025年迎来关键转机：美国EPA提议批准三款含麦草畏除草剂，政策环境边际改善，麦草畏量价修复可期。2025年上半年业绩已现改善，营收+7.28%，净利润+117.75%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **麦草畏** | 选择性除草剂，产能约1.1万吨/年 | 转基因大豆、玉米除草 |
| **农药原药** | 多种农药原药品种 | 除草剂、杀虫剂、杀菌剂 |
| **农药制剂** | 终端制剂产品 | 农业植保 |

### 业务模式

公司采用原药-制剂一体化模式，麦草畏为核心品种。产品主要出口美国、巴西等转基因作物种植区。2020年美国EPA禁用麦草畏后，麦草畏需求大幅下滑，公司业绩持续承压。2025年7月美国EPA提议批准含麦草畏产品，政策拐点来临。

### 技术路线/核心技术

- **麦草畏合成**：麦草畏全流程工艺，产能国内领先
- **多品种布局**：除草剂、杀虫剂、杀菌剂多品类
- **出口导向**：主要面向海外转基因作物市场

**技术优势**：
麦草畏产能约1.1万吨/年，国内龙头。2025年出口收入同比+15.82%，主要产品价格出现拐点。
