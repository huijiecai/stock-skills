## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏苏利精细化工股份有限公司 |
| 证券代码 | 603585 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2004年 |
| 上市时间 | 2016年12月14日 |
| 所属行业 | 化学原料和化学制品制造业（农药中间体、阻燃剂） |
| 市场地位 | 农药中间体、阻燃剂及中间体 |

**公司概况**：
苏利股份主要从事农药及农药中间体、阻燃剂及阻燃剂中间体的研发、生产和销售。2024年农药及农药中间体销量4.16万吨同比+58.72%，阻燃剂及阻燃剂中间体销量2.06万吨同比+38.75%，产销两旺。但受行业景气度下滑、新项目折旧压力、财务费用增加影响，净利润亏损1085万元。部分农化产品已显示价格回暖趋势，新项目全面投产后盈利能力有望增强。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **农药中间体** | 农药合成关键中间体 | 农药原药生产 |
| **农药** | 农药原药及制剂 | 农业植保 |
| **阻燃剂** | 阻燃剂及中间体 | 塑料、建材、电子 |
| **阻燃剂中间体** | 阻燃剂合成中间体 | 阻燃剂生产 |

### 业务模式

公司采用农药+阻燃剂双主业模式。农药及农药中间体面向农化企业，阻燃剂面向塑料、建材、电子等行业。2024年苏利宁夏、百力化学、大连永达苏利等新建项目陆续转固，产能爬坡中。产销两旺但折旧压力大，新项目全面投产后有望释放盈利。

### 技术路线/核心技术

- **农药中间体**：农药合成关键中间体工艺
- **阻燃剂**：阻燃剂及中间体技术
- **产能扩张**：宁夏、百力化学、大连永达苏利等项目

**技术优势**：
农药中间体、阻燃剂双主业分散风险。2024年农药中间体销量+58.72%，阻燃剂+38.75%，产销两旺。新项目投产后产品结构丰富，收入有望释放。

## 五、投资亮点

### 核心亮点

- ✅ **产销两旺**：2024年农药中间体销量+58.72%，阻燃剂+38.75%
- ✅ **产能扩张**：苏利宁夏、百力化学、大连永达苏利等项目推进
- ✅ **拐点逻辑**：部分农化产品价格已现回暖，新项目爬坡后盈利释放
- ✅ **催化剂**：农药价格回暖、新项目全面投产、产能利用率提升
- ✅ **双主业**：农药+阻燃剂分散单一行业风险

### 成长驱动因素

1. **农药价格回暖**
   - 部分农化产品已显示价格回暖趋势
   - 农药行业去库尾声、补库启动
   - 价格修复后毛利率改善

2. **新项目爬坡**
   - 苏利宁夏、百力化学、大连永达苏利产能爬坡
   - 产出覆盖折旧后盈利释放
   - 产品结构进一步丰富

3. **阻燃剂增长**
   - 阻燃剂销量+38.75%
   - 塑料、建材、电子需求

### 催化剂

- **农药价格**：农化产品价格回暖
- **产能爬坡**：新项目全面投产、产能利用率提升
- **库存周期**：农药补库周期
