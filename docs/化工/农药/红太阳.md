## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 南京红太阳股份有限公司 |
| 证券代码 | 000525（ST红太阳） |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1991年 |
| 上市时间 | 1993年10月28日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 吡啶碱龙头，生物L-草铵膦、生物新吡啶碱布局 |

**公司概况**：
红太阳（ST红太阳）主要从事农药的研发、生产和销售，农药销售占96.68%，国内50.67%、国外49.33%。公司在吡啶碱、2-氯吡啶、2,3-二氯吡啶、联吡啶等产品技术上取得重要突破，成功攻克合成生物产业链"卡脖子"技术难题。首期"双万吨级"生物酶及延伸物产业链项目在桠溪生产基地顺利投产。2024年归母净利润3.88亿元同比+199.73%，资产负债率从89.89%降至59.35%，财务显著改善。**注意：公司为ST状态，投资需关注风险。**

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **吡啶碱** | 农药关键中间体，技术突破 | 农药、医药 |
| **2-氯吡啶** | 农药中间体 | 农药合成 |
| **2,3-二氯吡啶** | 农药中间体 | 农药合成 |
| **联吡啶** | 农药中间体 | 农药合成 |
| **生物L-草铵膦** | 生物法草铵膦，药效更高 | 农业除草 |
| **农药** | 农药原药及制剂 | 农业植保 |

### 业务模式

公司采用"链长制"、激励技术进步，巩固绿色农药在全球的"链主"地位。农药销售占96.68%，国内国外各约50%。吡啶碱等关键产品技术取得突破，生物酶及延伸物产业链项目投产。战略聚焦生物L-草铵膦、生物新吡啶碱和生化双酰胺类新杀虫剂。

### 技术路线/核心技术

- **吡啶碱**：合成生物产业链"卡脖子"技术突破
- **生物L-草铵膦**：生物法，药效更高
- **双万吨级生物酶**：桠溪基地投产
- **2-氯吡啶、2,3-二氯吡啶、联吡啶**：技术突破

**技术优势**：
吡啶碱为农药关键中间体，技术突破打破"卡脖子"。生物L-草铵膦、生物新吡啶碱为战略方向。双万吨级生物酶项目投产。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
化工原料 → 【红太阳 吡啶碱/中间体】→ 农药原药 → 制剂 → 农业种植
                    ↑
                ST红太阳
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 吡啶碱龙头，绿色农药全球"链主" |
| **竞争优势** | 吡啶碱技术突破、生物L-草铵膦、生物新吡啶碱 |
| **客户结构** | 国内外农化企业，国内国外各约50% |
| **供应商** | 化工原料，生物法降低依赖 |

### 竞争格局

**主要竞争对手**：
- 利尔化学：草铵膦
- 扬农化工：菊酯
- 红太阳：吡啶碱、生物L-草铵膦

**公司优势**：
吡啶碱技术突破打破"卡脖子"，生物L-草铵膦、生物新吡啶碱战略布局。双万吨级生物酶项目投产。2024年业绩、财务显著改善。

## 五、投资亮点

### 核心亮点

- ✅ **吡啶碱技术突破**：攻克合成生物产业链"卡脖子"技术，吡啶碱、2-氯吡啶、2,3-二氯吡啶、联吡啶突破
- ✅ **业绩大幅改善**：2024年净利润+199.73%，资产负债率从89.89%降至59.35%
- ✅ **生物L-草铵膦**：生物法草铵膦，药效更高，战略方向
- ✅ **双万吨级项目投产**：生物酶及延伸物产业链桠溪基地投产
- ⚠️ **ST状态**：投资需关注ST风险

### 成长驱动因素

1. **吡啶碱技术突破**
   - 打破"卡脖子"，自主可控
   - 2-氯吡啶、2,3-二氯吡啶、联吡啶延伸
   - 绿色农药全球"链主"

2. **生物L-草铵膦**
   - 生物法，药效更高
   - 符合农药减量增效
   - 与利尔化学精草铵膦竞争

3. **财务改善**
   - 资产负债率大幅下降
   - 业绩扭亏并高增

### 催化剂

- **技术产业化**：吡啶碱、生物L-草铵膦放量
- **农药价格**：农药行业拐点
- **ST撤销**：若持续盈利、规范运作，申请撤销ST
