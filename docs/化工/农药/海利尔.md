## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 海利尔药业集团股份有限公司 |
| 证券代码 | 603639 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 2017年1月12日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 杀菌剂龙头，杀虫剂、杀菌剂一体化 |

**公司概况**：
海利尔是国内农药制剂、原药和中间体一体化企业，产品涵盖杀虫剂（吡虫啉、噻虫嗪、甲维盐等）和杀菌剂（吡唑醚菌酯、丙硫菌唑、苯醚甲环唑等）。2024年受农药行业去库周期影响，营收-11.16%，净利润-61.67%，但杀菌剂龙头地位稳固，行业拐点来临有望受益。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **杀菌剂** | 吡唑醚菌酯、丙硫菌唑、苯醚甲环唑等 | 作物病害防治 |
| **杀虫剂** | 吡虫啉、噻虫嗪、噻虫胺、甲维盐等 | 农业害虫防治 |
| **农药制剂** | 终端制剂产品 | 农业植保 |
| **农药原药** | 原药生产与销售 | 农化产业链 |
| **农药中间体** | 合成中间体 | 原药生产 |

### 业务模式

公司采用制剂-原药-中间体一体化模式，从中间体到制剂全产业链覆盖。制剂面向终端用户，原药供应国内外客户。2024年农药行业去库周期导致产品价格下行，业绩承压。

### 技术路线/核心技术

- **杀菌剂合成**：吡唑醚菌酯、丙硫菌唑等主流杀菌剂工艺
- **杀虫剂合成**：新烟碱类、甲维盐等
- **制剂配方**：终端制剂研发与生产

**技术优势**：
杀菌剂品种丰富，吡唑醚菌酯、丙硫菌唑等为主流品种。杀虫剂新烟碱类、甲维盐布局完善。制剂-原药一体化成本优势。

## 五、投资亮点

### 核心亮点

- ✅ **杀菌剂龙头**：吡唑醚菌酯、丙硫菌唑、苯醚甲环唑等主流品种
- ✅ **制剂-原药一体化**：全产业链布局，成本与渠道优势
- ✅ **拐点逻辑**：农药行业5-6年周期，2025年去库尾声、补库启动
- ✅ **催化剂**：杀菌剂价格企稳、农药补库周期、作物病害防治需求
- ✅ **品种丰富**：杀虫剂+杀菌剂双轮驱动，分散单一品种风险

### 催化剂

- **农药价格**：杀菌剂、杀虫剂价格企稳回升
- **库存周期**：海外补库、制剂出口改善
- **政策**：农药减量增效、绿色农业利好合规龙头
