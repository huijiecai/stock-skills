## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 山东潍坊润丰化工股份有限公司 |
| 证券代码 | 301035 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2005年 |
| 上市时间 | 2021年7月28日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 农药制剂出口龙头，境外自主登记模式 |

**公司概况**：
润丰股份主要从事农药产品的研发、生产和销售，采用传统出口模式（以原药出口为主）和境外自主登记模式。产品涵盖除草剂、杀虫剂、杀菌剂，除草剂营收占比最高。公司推进TO C业务发展，C模式占比从33%增至38%，加大北美、欧盟业务规模。2024年营收133.0亿元同比+15.8%，但归母净利润4.5亿元同比-41.6%，主要受汇兑损失、毛利率下降、销售费用增加影响。经营现金流7.51亿元同比+155.6%，现金流改善。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **除草剂** | 核心品类，营收占比最高 | 农业除草 |
| **杀虫剂** | 杀虫剂产品 | 农业害虫防治 |
| **杀菌剂** | 杀菌剂产品 | 作物病害防治 |
| **农药原药** | 传统出口以原药为主 | 农化产业链 |
| **农药制剂** | 境外自主登记模式 | 终端市场 |

### 业务模式

公司采用"传统出口+境外自主登记"双模式。传统出口以原药为主，面向农化企业。境外自主登记模式（TO C）在海外自主登记、销售制剂，直面终端，C模式占比从33%增至38%。加大北美、欧盟业务规模。2024年TO B业务竞争激烈导致毛利率下降。

### 技术路线/核心技术

- **境外登记**：多国农药登记，自主品牌
- **双模式**：传统出口+境外自主登记
- **TO C拓展**：北美、欧盟加大布局

**技术优势**：
境外自主登记网络完善，TO C模式提升毛利率潜力。除草剂、杀虫剂、杀菌剂产品线丰富。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
原药生产 → 【润丰股份 出口/境外登记】→ 全球农资 → 农业种植
                    ↑
                润丰股份
            （传统出口+境外自主登记）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 农药制剂出口龙头，营收133亿 |
| **竞争优势** | 境外自主登记、TO C模式、全球布局 |
| **客户结构** | 全球农化企业、终端用户（TO C） |
| **供应商** | 原药自产+外采 |

### 竞争格局

**主要竞争对手**：
- 安道麦A：全球仿制农药
- 扬农化工：菊酯、出口
- 中农立华：农药流通

**公司优势**：
农药制剂出口龙头，境外自主登记模式差异化。TO C占比提升，北美、欧盟拓展。营收规模133亿，行业领先。

## 五、投资亮点

### 核心亮点

- ✅ **农药制剂出口龙头**：营收133亿，+15.8%规模增长
- ✅ **境外自主登记**：自主品牌、多国登记，TO C模式
- ✅ **拐点逻辑**：农药行业拐点、汇兑影响减弱、TO C占比提升
- ✅ **催化剂**：农药补库周期、汇率企稳、TO C放量
- ✅ **现金流改善**：经营现金流+155.6%

### 成长驱动因素

1. **TO C模式拓展**
   - C模式占比33%→38%
   - 北美、欧盟加大规模
   - TO C毛利率高于TO B

2. **农药行业拐点**
   - 农药补库周期
   - 出口需求回升
   - 产品价格企稳

3. **汇兑影响减弱**
   - 2024年汇兑损失约2.96亿
   - 汇率企稳后影响减弱

### 催化剂

- **农药价格**：农药补库周期、出口改善
- **汇率**：汇率企稳，汇兑损失减少
- **TO C**：境外自主登记、TO C占比提升
