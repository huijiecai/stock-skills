## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 利民控股集团股份有限公司 |
| 证券代码 | 002734 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1996年 |
| 上市时间 | 2015年1月27日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 杀菌剂、杀虫剂领先企业，AI+生物合成技术布局 |

**公司概况**：
利民股份主要从事农药（杀菌剂、杀虫剂、除草剂）及兽药的生产销售。在杀菌剂和生物农药领域处于领先地位，采用AI+生物合成技术，致力于技术创新与绿色转型。2024年在农药行业去库周期中逆势实现净利润+31.1%，毛利率提升2.5个百分点，表现突出。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **农用杀菌剂** | 核心产品，毛利率26.07% | 作物病害防治 |
| **农用杀虫剂** | 杀虫剂原药及制剂 | 农业害虫防治 |
| **农用除草剂** | 除草剂产品 | 农业除草 |
| **兽药** | 兽用药品 | 畜禽养殖 |

### 业务模式

公司采用原药-制剂一体化模式，杀菌剂为核心优势品类。2024年顺应粮食安全需求，推进AI+生物合成技术、绿色转型，适应行业机遇。在农药价格下行周期中通过产品结构优化、成本控制实现逆势增长。

### 技术路线/核心技术

- **杀菌剂工艺**：主流杀菌剂合成技术，毛利率26.07%
- **AI+生物合成**：生物农药、绿色技术布局
- **多品类**：杀菌剂、杀虫剂、除草剂、兽药

**技术优势**：
杀菌剂毛利率显著高于杀虫剂，产品结构优化。AI+生物合成技术顺应农药减量增效、绿色农业政策方向。

## 五、投资亮点

### 核心亮点

- ✅ **逆势增长**：2024年净利润+31.1%，毛利率+2.5pct，行业下行期表现突出
- ✅ **杀菌剂优势**：杀菌剂毛利率26.07%，产品结构优化
- ✅ **拐点逻辑**：农药行业拐点来临，公司已展现抗周期能力，拐点后弹性可期
- ✅ **催化剂**：农药补库周期、杀菌剂需求刚性、粮食安全政策
- ✅ **AI+生物合成**：绿色技术布局，顺应农药减量增效政策

### 成长驱动因素

1. **行业拐点受益**
   - 农药行业去库尾声、补库启动
   - 公司已展现抗周期能力，拐点后量价齐升
   - 杀菌剂需求相对刚性

2. **产品结构优化**
   - 杀菌剂毛利率高，占比提升
   - AI+生物合成技术拓展生物农药

3. **粮食安全机遇**
   - 粮食安全需求下农药刚需
   - 绿色、高效农药受益政策

### 催化剂

- **农药价格**：杀菌剂、杀虫剂价格企稳回升
- **库存周期**：补库周期启动
- **政策**：粮食安全、农药减量增效、绿色农业
