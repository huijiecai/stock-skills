## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 南通江山农药化工股份有限公司 |
| 证券代码 | 600389 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1990年 |
| 上市时间 | 2001年1月10日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 草甘膦主要供应商，新材料业务高增长 |

**公司概况**：
江山股份主要从事农药、化工、热电及新材料产品的研发、生产和销售。草甘膦等除草剂为核心品种，2024年农药产品收入28.11亿元(-3.66%)。新材料产品收入11.62亿元同比+97.56%，成为主要增长驱动力。公司持续推进产业链延伸，研发投入2.07亿元同比+38.86%。2024年受农药价格下行影响净利润-20.7%，但新材料业务打开成长空间。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **草甘膦** | 非选择性除草剂，核心品种 | 农业除草、转基因作物 |
| **农药产品** | 除草剂、杀虫剂、杀菌剂 | 农业植保 |
| **化工产品** | 化工中间体 | 化工产业链 |
| **热电产品** | 热电联产 | 园区配套 |
| **新材料产品** | 新产品，高增长 | 新兴应用 |

### 业务模式

公司采用农药-化工-热电-新材料一体化模式。农药草甘膦为核心，化工、热电配套。新材料业务2024年收入+97.56%，成为主要增长点。研发投入持续加大，产业链延伸。

### 技术路线/核心技术

- **草甘膦合成**：草甘膦全流程工艺
- **热电联产**：园区配套，成本优势
- **新材料**：新产品研发与产业化

**技术优势**：
草甘膦产能规模国内领先。新材料业务2024年+97.56%，成为主要增长驱动力。研发投入2.07亿元同比+38.86%。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
化工原料 → 中间体 → 【江山股份 草甘膦/农药】→ 农资流通 → 农业种植
                         ↑
                    江山股份
                 （+新材料延伸）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 草甘膦主要供应商，新材料高增长 |
| **竞争优势** | 农药-化工-热电一体化、新材料延伸 |
| **客户结构** | 国内外农化企业 |
| **供应商** | 化工原料采购 |

### 竞争格局

**主要竞争对手**：
- 广信股份：草甘膦
- 福华通达：草甘膦
- 扬农化工：菊酯

**公司优势**：
草甘膦主要供应商，新材料业务打开成长空间。农药-化工-热电一体化成本优势。研发投入加大，产业链延伸。

## 五、投资亮点

### 核心亮点

- ✅ **新材料高增长**：2024年收入11.62亿元+97.56%，主要增长驱动力
- ✅ **草甘膦核心供应商**：草甘膦价格弹性大，行业拐点后弹性显著
- ✅ **拐点逻辑**：农药行业拐点来临，草甘膦价格企稳后业绩修复
- ✅ **催化剂**：草甘膦价格企稳、新材料放量、农药补库周期
- ✅ **研发投入**：2.07亿元同比+38.86%，产业链延伸

### 成长驱动因素

1. **新材料放量**
   - 2024年新材料+97.56%，成为主要增长点
   - 新产品产业化，降低农药周期依赖

2. **草甘膦价格修复**
   - 农药行业拐点，草甘膦价格企稳
   - 草甘膦库存下行、补库启动

3. **产业链一体化**
   - 农药-化工-热电协同
   - 研发投入加大，新产品储备

### 催化剂

- **农药价格**：草甘膦价格企稳回升
- **新材料**：新产品放量持续
- **库存周期**：农药补库周期
