## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 北京颖泰嘉和生物科技股份有限公司 |
| 证券代码 | 833819 |
| 上市板块 | 北京证券交易所 |
| 成立时间 | 2005年 |
| 上市时间 | 2015年新三板挂牌，2021年北交所上市 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 三嗪酮类、二苯醚类、酰胺类除草剂国内前列 |

**公司概况**：
颖泰生物是北交所上市公司，从事农药原药、中间体及制剂产品的研发、生产、销售，在农化行业深耕二十余年。已形成除草剂、杀菌剂、杀虫剂三大品类百余种产品，三嗪酮类、二苯醚类、酰胺类除草剂市场占有率居国内前列。2024年受客户需求放缓、市场竞争加剧、产品价格走低、联营单位亏损、资产减值影响，亏损5.87亿元。2025年Q1已扭亏为盈，经营呈逐步向好趋势。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **三嗪酮类除草剂** | 市场占有率国内前列 | 农业除草 |
| **二苯醚类除草剂** | 市场占有率国内前列 | 农业除草 |
| **酰胺类除草剂** | 市场占有率国内前列 | 农业除草 |
| **杀菌剂** | 百余种产品 | 作物病害防治 |
| **杀虫剂** | 百余种产品 | 农业害虫防治 |
| **农药制剂** | 终端制剂产品 | 农业植保 |

### 业务模式

公司采用原药-中间体-制剂一体化模式，除草剂、杀菌剂、杀虫剂三大品类百余种产品。三嗪酮类、二苯醚类、酰胺类除草剂国内前列。2024年通过强化技术研发、打造核心产品、聚焦高潜力产品、推行精益管理等措施创新提质、挖潜降本。

### 技术路线/核心技术

- **除草剂**：三嗪酮类、二苯醚类、酰胺类技术领先
- **多品类**：除草剂、杀菌剂、杀虫剂百余种
- **一体化**：原药-中间体-制剂

**技术优势**：
三嗪酮类、二苯醚类、酰胺类除草剂市场占有率国内前列。多品类分散风险，一体化成本优势。
