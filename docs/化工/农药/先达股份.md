## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 山东先达农化股份有限公司 |
| 证券代码 | 603086 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2002年 |
| 上市时间 | 2017年5月11日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 高端除草剂创制化合物领先企业 |

**公司概况**：
先达股份坚持创制化合物战略，聚焦高端除草剂领域，构建以喹草酮、吡唑喹草酯、苯丙草酮为核心的创制产品矩阵。建立"销售、市场、技术"铁三角营销模式，同步推进国内稳健增长和海外登记推广。2024年大幅减亏75.42%，2025年Q1已扭亏为盈，创制品种放量可期。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **喹草酮** | 创制除草剂，核心品种 | 高粱等作物除草 |
| **吡唑喹草酯** | 创制除草剂 | 水稻等作物除草 |
| **苯丙草酮** | 创制除草剂 | 农业除草 |
| **其他除草剂** | 烯草酮、烯禾啶等 | 农业除草 |

### 业务模式

公司采用创制化合物战略，聚焦高端除草剂。创制品种具有专利保护、毛利率高、竞争格局好等特点。建立"销售、市场、技术"铁三角营销模式，国内稳健增长、海外登记推广双轮驱动。2024年大幅减亏，2025年Q1扭亏，创制品种逐步放量。

### 技术路线/核心技术

- **创制化合物**：喹草酮、吡唑喹草酯、苯丙草酮等自主创制
- **专利保护**：创制品种专利壁垒，竞争格局好
- **登记推广**：国内+海外双线推进

**技术优势**：
创制化合物具有专利保护、毛利率高。喹草酮、吡唑喹草酯、苯丙草酮为核心矩阵，差异化竞争。
