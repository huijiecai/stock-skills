## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 安徽广信农化股份有限公司 |
| 证券代码 | 603599 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2000年 |
| 上市时间 | 2015年5月13日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 草甘膦、多菌灵、敌草隆等农药及光气化产品 |

**公司概况**：
广信股份主要从事农药原药、制剂及光气化产品的研发、生产和销售。核心产品包括草甘膦、多菌灵、甲基硫菌灵、敌草隆等。2024年受农药产品价格大幅下滑影响，业绩承压，草甘膦价格同比-53%。公司持续推进产业链一体化建设，年产30万吨离子膜烧碱项目已投产，静待农药需求复苏。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **草甘膦** | 非选择性除草剂，核心品种 | 农业除草、转基因作物 |
| **多菌灵** | 杀菌剂 | 作物病害防治 |
| **甲基硫菌灵** | 杀菌剂 | 作物病害防治 |
| **敌草隆** | 除草剂 | 农业除草 |
| **光气化产品** | 农药中间体、精细化工 | 农药、医药 |
| **烧碱** | 离子膜烧碱，年产30万吨 | 化工原料 |

### 业务模式

公司采用农药-光气化一体化模式，草甘膦为核心品种。光气为稀缺资源，公司具备光气生产资质。年产30万吨离子膜烧碱项目已投产，产业链延伸。2024年农药价格大幅下滑，业绩承压，静待需求复苏。

### 技术路线/核心技术

- **草甘膦合成**：草甘膦全流程工艺
- **光气化**：光气为农药中间体关键原料，资质稀缺
- **产业链一体化**：农药-烧碱-光气化协同

**技术优势**：
光气生产资质稀缺，农药-光气化一体化成本优势。烧碱项目投产增强产业链协同。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
化工原料 → 光气/烧碱 → 【广信股份 草甘膦/农药原药】→ 农资流通 → 农业种植
                            ↑
                       广信股份
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 草甘膦、多菌灵、敌草隆主要供应商 |
| **竞争优势** | 光气资质稀缺、农药-烧碱-光气化一体化 |
| **客户结构** | 国内外农化企业 |
| **供应商** | 化工原料，烧碱自产 |

### 竞争格局

**主要竞争对手**：
- 江山股份：草甘膦
- 福华通达：草甘膦
- 扬农化工：菊酯

**公司优势**：
光气生产资质稀缺，农药-光气化一体化成本优势。烧碱项目投产增强产业链。草甘膦价格弹性大，行业拐点后业绩弹性显著。

## 五、投资亮点

### 核心亮点

- ✅ **草甘膦核心供应商**：草甘膦价格弹性大，行业拐点后弹性显著
- ✅ **光气资质稀缺**：光气为农药中间体关键原料，资质壁垒高
- ✅ **拐点逻辑**：草甘膦库存快速下行，2025年价格有望突破3万元/吨
- ✅ **催化剂**：草甘膦价格企稳回升、农药补库周期、黄磷成本支撑
- ✅ **产业链延伸**：年产30万吨烧碱投产，多产品布局

### 成长驱动因素

1. **草甘膦价格修复**
   - 2024年草甘膦价格同比-53%至2.59万元/吨
   - 2025年库存下行、补库启动，价格有望突破3万元
   - 黄磷价格上涨传导成本支撑

2. **产业链一体化**
   - 烧碱项目投产，产业链协同
   - 光气化产品延伸
   - 多产品分散风险

3. **农药需求复苏**
   - 海外去库尾声、补库启动
   - 草甘膦为全球最大除草剂，需求刚性

### 催化剂

- **农药价格**：草甘膦价格企稳回升（关键）
- **库存周期**：草甘膦库存下行、补库启动
- **成本**：黄磷价格上涨传导
