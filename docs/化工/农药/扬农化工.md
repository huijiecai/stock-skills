## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏扬农化工股份有限公司 |
| 证券代码 | 600486 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1999年12月10日 |
| 上市时间 | 2002年4月25日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 菊酯农药全球龙头，拟除虫菊酯原药全国第一 |

**公司概况**：
扬农化工是中国中化控股旗下成员企业，全球拟除虫菊酯原药核心供应商，核心品种拟除虫菊酯为全国制造业单项冠军产品、中国工业大奖获得者。公司拥有南北两个研发中心，专利授权850余个，产品远销全球80多个国家和地区。2024年受农药行业去库周期影响业绩承压，但菊酯龙头地位稳固，行业拐点来临有望率先受益。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **拟除虫菊酯原药** | 拟除虫菊酯类杀虫剂，全国制造业单项冠军 | 农业害虫防治、卫生杀虫 |
| **农药原药** | 在产原药品种近70个 | 杀虫剂、除草剂、杀菌剂 |
| **农药制剂** | 终端制剂产品 | 农业植保 |
| **贸易业务** | 农药产品贸易 | 国内外农化市场 |

### 业务模式

公司采用研产销一体化运营模式，从原药研发生产到制剂销售全覆盖。拥有扬州植保研究院和沈阳中化农药化工研发有限公司南北两个研发中心，与全球主要农化跨国巨头形成紧密合作。旗下五家工厂，江苏优嘉为工信部首批"绿色工厂"、国家级智能制造示范工厂。

### 技术路线/核心技术

- **拟除虫菊酯合成**：菊酯类农药全产业链技术，成本与环保领先
- **绿色工艺**：优嘉工厂为绿色工厂标杆，HSE管控行业领先
- **产品创制**：国家级重点实验室、工程技术研究中心、博士后工作站

**技术优势**：
拟除虫菊酯规模名列全国第一，专利授权850余个。新农药产品创制及次新药产品创新具有显著优势，与跨国农化巨头深度绑定。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
化工原料 → 农药中间体 → 【扬农化工 原药/制剂】→ 农资流通 → 农业种植
                              ↑
                         扬农化工
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 拟除虫菊酯原药全国第一，全球核心供应商 |
| **竞争优势** | 菊酯全产业链、研产销一体化、中化系资源 |
| **客户结构** | 与拜耳、先正达、科迪华等跨国巨头深度合作 |
| **供应商** | 化工原料采购，产业链整合能力强 |

### 竞争格局

**主要竞争对手**：
- 利尔化学：草铵膦龙头
- 海利尔：杀菌剂龙头
- 安道麦A：全球仿制农药龙头

**公司优势**：
菊酯农药全球龙头，技术壁垒高，客户粘性强。2024年行业去库周期业绩承压，但龙头抗风险能力强，行业拐点来临将率先受益。

## 五、投资亮点

### 核心亮点

- ✅ **菊酯农药全球龙头**：拟除虫菊酯全国制造业单项冠军，中国工业大奖
- ✅ **研产销一体化**：南北双研发中心，专利850余个，与跨国巨头深度绑定
- ✅ **拐点逻辑**：农药行业5-6年大周期，2022Q3以来下行近3年，2025年去库尾声、补库启动
- ✅ **催化剂**：草甘膦等原药价格企稳回升，海外采购需求回暖
- ✅ **中化系背景**：央企资源、绿色工厂标杆、管理规范

### 成长驱动因素

1. **行业拐点来临**
   - 农药行业20年规律：5-6年大周期，上行2-3年
   - 2025年海外去库渐进尾声，美国农化品库存同比转正
   - 草甘膦库存从高位快速下行，与2020年周期启动相似

2. **农药价格修复**
   - 2024年原药价格大幅下跌后，2025年有望企稳回升
   - 黄磷等原料价格上涨传导至农药成本支撑
   - 国内企业调价函频发，涨价幅度5%-15%

3. **菊酯龙头优势**
   - 菊酯技术壁垒高，竞争格局稳定
   - 全球80+国家地区销售，抗单一市场风险

### 催化剂

- **农药价格**：草甘膦、菊酯等原药价格企稳回升，突破关键关口
- **库存周期**：海外去库尾声、补库启动，制剂出口数据改善
- **政策**：农药减量增效、绿色农业政策利好合规龙头
