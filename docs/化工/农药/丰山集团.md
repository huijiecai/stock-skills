## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏丰山集团股份有限公司 |
| 证券代码 | 603810 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1988年 |
| 上市时间 | 2018年9月17日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 氟乐灵、精喹禾灵等除草剂先进企业 |

**公司概况**：
丰山集团成立于江苏盐城，主要从事高效、低毒、低残留、环保型农药原药、制剂及农药中间体的研发、生产和销售。除草剂为核心业务，氟乐灵、精喹禾灵等营收占比超60%。2024年Q3净利润同比+606.40%，业绩拐点已现。公司正在拓展固态电池电解液等新领域，但农药仍为主业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **氟乐灵** | 除草剂，核心品种 | 农业除草 |
| **精喹禾灵** | 除草剂，核心品种 | 农业除草 |
| **毒死蜱** | 杀虫剂 | 农业害虫防治 |
| **杀菌剂** | 杀菌剂产品 | 作物病害防治 |
| **农药中间体** | 农药合成中间体 | 原药生产 |

### 业务模式

公司采用原药-制剂一体化模式，除草剂为核心，占比超60%。产品以高效、低毒、低残留、环保型为特色。2024年农药行业拐点，业绩显著改善。正在拓展固态电池电解液等新领域，但农药仍为主业。

### 技术路线/核心技术

- **除草剂工艺**：氟乐灵、精喹禾灵等先进工艺
- **环保型农药**：高效、低毒、低残留
- **新领域**：固态电池电解液研发中

**技术优势**：
氟乐灵、精喹禾灵等除草剂技术先进，营收占比超60%。环保型农药符合农药减量增效政策。
