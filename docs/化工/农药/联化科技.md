## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 联化科技股份有限公司 |
| 证券代码 | 002250 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2008年6月19日 |
| 所属行业 | 化学原料和化学制品制造业（农药中间体、医药） |
| 市场地位 | 农药中间体龙头，植保+医药+功能化学品 |

**公司概况**：
联化科技主要从事农药中间体（植保业务）、医药中间体及功能化学品的研发、生产和销售。2024年实现扭亏为盈，归母净利润1.03亿元，扣非净利润1.21亿元。植保业务收入35.90亿元(-15.39%)，但毛利率同比提升5.78个百分点至20.1%。医药业务毛利率47.95%同比+13.13pct。机构预测2025-2027年归母净利润分别为3.05亿、4.15亿、5.36亿元，业绩底部向上、逐步修复。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **农药中间体（植保）** | 农药合成关键中间体 | 农药原药生产 |
| **医药中间体** | 医药CRO、CDMO | 医药研发生产 |
| **功能化学品** | 新能源等精细化工 | 新能源、电子 |

### 业务模式

公司采用植保+医药+功能化学品三轮驱动模式。植保（农药中间体）为核心，面向全球农化企业。医药业务CRO商业化产品18个、临床III期产品30个。功能化学品完成4个新能源产品试生产及客户导入。英国子公司、江苏联化2024年减亏明显。

### 技术路线/核心技术

- **农药中间体**：植保业务，全球供应链
- **医药CRO/CDMO**：商业化产品18个，临床III期30个
- **功能化学品**：新能源产品试生产

**技术优势**：
农药中间体龙头，植保业务毛利率2024年同比+5.78pct。医药业务毛利率47.95%高毛利。功能化学品新能源布局。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
化工原料 → 【联化科技 农药中间体】→ 农药原药 → 制剂 → 农业种植
                    ↑
                联化科技
            （+医药+功能化学品）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 农药中间体龙头，植保收入35.90亿 |
| **竞争优势** | 植保+医药+功能化学品、毛利率改善 |
| **客户结构** | 全球农化企业、医药企业 |
| **供应商** | 化工原料采购 |

### 竞争格局

**主要竞争对手**：
- 雅本化学：农药中间体、医药
- 苏利股份：农药中间体
- 福华通达：农药中间体

**公司优势**：
农药中间体龙头，2024年扭亏为盈。植保毛利率改善，医药高毛利。机构预测2025-2027年净利润3.05亿、4.15亿、5.36亿，业绩底部向上。

## 五、投资亮点

### 核心亮点

- ✅ **扭亏为盈**：2024年归母净利润1.03亿元，扣非1.21亿元
- ✅ **农药中间体龙头**：植保收入35.90亿，毛利率+5.78pct改善
- ✅ **拐点逻辑**：业绩底部向上，机构预测2025-2027年净利润3.05/4.15/5.36亿
- ✅ **催化剂**：农药中间体需求回升、医药业务增长、新能源产品商业化
- ✅ **医药高毛利**：毛利率47.95%同比+13.13pct，CRO商业化产品18个

### 成长驱动因素

1. **农药中间体需求回升**
   - 农药行业拐点，中间体需求回升
   - 植保毛利率已改善+5.78pct
   - 海外基地减亏

2. **医药业务**
   - 毛利率47.95%高毛利
   - CRO商业化产品18个，临床III期30个
   - 2025年上半年医药收入同比+42.80%

3. **功能化学品**
   - 4个新能源产品试生产及客户导入
   - 新能源布局

### 催化剂

- **农药价格**：农药行业拐点、中间体需求回升
- **医药**：医药业务增长、商业化产品放量
- **新能源**：功能化学品新能源产品商业化
