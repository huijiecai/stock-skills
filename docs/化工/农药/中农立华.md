## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中农立华生物科技股份有限公司 |
| 证券代码 | 603970 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2009年 |
| 上市时间 | 2017年11月16日 |
| 所属行业 | 批发业（农药流通） |
| 市场地位 | 农药流通龙头，央企背景 |

**公司概况**：
中农立华是中农集团控股的农药流通龙头企业，主要从事农药流通及植保技术服务，为种植户提供作物健康解决方案和专业的植保技术服务。公司采取"流通+服务"模式，在农药行业去库周期中抗风险能力较强。2024年营收微增1.3%，净利润下滑9.5%，但经营现金流净额大增132.6%，财务质量改善。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **农药流通** | 农药原药、制剂采购与分销 | 农资渠道、种植户 |
| **植保技术服务** | 作物健康解决方案、病虫害防治 | 农业种植 |
| **农药制剂** | 自有品牌制剂产品 | 农业植保 |
| **农化服务** | 技术推广、用药指导 | 农业服务 |

### 业务模式

公司采用"流通+服务"双轮驱动模式。流通业务连接上游农药生产企业和下游农资经销商、种植户；植保技术服务提升客户粘性，差异化竞争。严控成本支出、提升技术服务能力、加强与合作伙伴合作，应对行业不利形势。

### 技术路线/核心技术

- **渠道网络**：覆盖全国的农资流通网络
- **植保技术**：作物健康解决方案、病虫害防治技术
- **供应链管理**：采购、仓储、物流一体化

**技术优势**：
流通龙头渠道优势明显，央企中农集团背景资源丰富。技术服务能力提升客户粘性，在行业下行期抗风险能力强。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
原药生产 → 【中农立华 流通/服务】→ 农资经销商 → 种植户
                ↑
           中农立华
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 农药流通龙头，营收超百亿 |
| **竞争优势** | 央企中农集团背景、渠道网络、技术服务 |
| **客户结构** | 覆盖农资经销商、种植户、规模种植主体 |
| **供应商** | 与扬农、利尔等原药企业合作 |

### 竞争格局

**主要竞争对手**：
- 诺普信：农药制剂、流通
- 润丰股份：农药制剂出口
- 辉丰股份：农药原药、流通

**公司优势**：
农药流通龙头，央企背景资源丰富。流通业务在行业下行期相对抗跌，不直接承受原药价格波动。经营现金流改善显示管理能力。

## 五、投资亮点

### 核心亮点

- ✅ **农药流通龙头**：营收超百亿，渠道网络覆盖全国
- ✅ **央企背景**：中农集团控股，资源与信用优势
- ✅ **拐点逻辑**：流通环节在行业下行期抗跌，行业拐点后量价齐升弹性
- ✅ **催化剂**：农药补库周期、原药价格企稳、农资需求回暖
- ✅ **现金流改善**：2024年经营现金流+132.6%，回款质量提升

### 成长驱动因素

1. **行业拐点受益**
   - 农药行业去库尾声、补库启动
   - 流通环节连接供需，拐点后周转加快
   - 原药价格企稳后，流通毛利改善

2. **服务差异化**
   - 植保技术服务提升客户粘性
   - 引进新产品和推广应对市场变化
   - 作物健康解决方案差异化竞争

3. **渠道下沉**
   - 规模种植、家庭农场等新型主体增加
   - 农资服务需求增长

### 催化剂

- **农药价格**：原药价格企稳，流通毛利改善
- **库存周期**：补库周期启动，流通量增加
- **政策**：农业社会化服务、农资保供政策
