## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 福华通达化学股份公司/四川福华农科投资集团 |
| 证券代码 | **未上市**（IPO 2023年终止） |
| 上市板块 | - |
| 成立时间 | 2007年12月10日 |
| 上市时间 | - |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 草甘膦全球前三、国内前列，产能15.3万吨/年 |

**公司概况**：
福华通达（福华化学）位于四川省乐山市，草甘膦原药产能15.3万吨/年，位居全球前三、国内前列。草甘膦为全球第一大除草剂，与转基因作物种植关联度高。2021年出口额位居中国农药年度出口前4，2022年出口额突破10亿美元。离子膜烧碱50万吨/年西南第一。坚持绿色发展理念，打造循环经济产业链，拥有磷矿、盐矿等草甘膦生产必需原材料，正在完善黄磷生产环节。2018年获工信部"绿色工厂"及"单项冠军产品"认证。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **草甘膦** | 产能15.3万吨/年，全球前三 | 农业除草、转基因作物 |
| **离子膜烧碱** | 50万吨/年，西南第一 | 化工原料 |
| **农药中间体** | 草甘膦产业链中间体 | 农药生产 |
| **其他化工** | 磷矿、盐矿等 | 产业链配套 |

### 业务模式

公司采用草甘膦-烧碱-矿产资源一体化模式。草甘膦为核心，烧碱配套。拥有磷矿、盐矿等原材料，正在完善黄磷生产，打通全产业链。循环经济、绿色工厂。产品主要出口，2022年出口额突破10亿美元。

### 技术路线/核心技术

- **草甘膦合成**：草甘膦全流程工艺，产能15.3万吨/年
- **循环经济**：磷矿、盐矿、烧碱、黄磷一体化
- **绿色工厂**：工信部"绿色工厂"、"单项冠军产品"

**技术优势**：
草甘膦产能全球前三，产业链一体化成本优势。磷矿、盐矿资源，黄磷环节完善中。绿色工厂认证。
