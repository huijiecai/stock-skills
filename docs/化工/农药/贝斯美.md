## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 绍兴贝斯美化工股份有限公司 |
| 证券代码 | 300796 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2003年 |
| 上市时间 | 2019年11月15日 |
| 所属行业 | 化学原料和化学制品制造业（农药、新材料） |
| 市场地位 | 二甲戊灵全产业链龙头，碳五新材料布局 |

**公司概况**：
贝斯美是国内仅有的具备二甲戊灵原药、中间体、制剂全产业链生产研发能力的农药企业。二甲戊灵为高效、低毒、低残留的新型环保除草剂。2024年营收13.28亿元同比+89.97%，但受控股子公司铜陵贝斯美亏损及二甲戊灵、农药中间体去库存周期影响，净利润亏损3286万元。碳五新材料布局推进，年产8500吨戊酮系列项目2024年投产，年产12100吨环戊烷系列项目试生产，环烯烃共聚物(COC)等高端新材料研发中。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **二甲戊灵** | 除草剂，原药-中间体-制剂全产业链 | 农业除草 |
| **农药中间体** | 二甲戊灵合成中间体 | 农药生产 |
| **戊酮系列** | 正戊烯、3-戊酮、2-戊酮等 | 新材料 |
| **环戊烷系列** | 环戊烷绿色新材料 | 新材料 |
| **COC** | 环烯烃共聚物，研发中 | 高端新材料 |

### 业务模式

公司采用农药+新材料双轮驱动。农药以二甲戊灵全产业链为核心，国内唯一。新材料布局碳五综合利用，戊酮系列2024年投产，环戊烷系列试生产，COC研发中。2024年营收+89.97%主要来自业务扩张，但去库存周期、铜陵贝斯美亏损导致净利润亏损。

### 技术路线/核心技术

- **二甲戊灵**：原药-中间体-制剂全产业链，国内唯一
- **碳五综合利用**：戊酮、环戊烷系列，国内领先
- **COC**：环烯烃共聚物高端新材料研发

**技术优势**：
二甲戊灵全产业链国内唯一，高效、低毒、低残留环保农药。碳五综合利用国内领先，戊酮、环戊烷、COC布局。

                    贝斯美

### 市场地位

|------|------|

**公司优势**：
二甲戊灵全产业链国内唯一，高效、低毒、低残留环保农药。碳五新材料布局，戊酮、环戊烷、COC延伸。

## 五、投资亮点

### 核心亮点

- ✅ **二甲戊灵全产业链**：国内唯一，原药-中间体-制剂一体化
- ✅ **营收高增长**：2024年营收+89.97%，业务扩张
- ✅ **拐点逻辑**：去库存周期后，二甲戊灵需求回升；碳五新材料爬坡
- ✅ **催化剂**：农药补库周期、戊酮/环戊烷放量、COC产业化
- ✅ **碳五新材料**：戊酮系列投产，环戊烷试生产，COC研发

### 成长驱动因素

1. **农药拐点**
   - 二甲戊灵去库存周期后需求回升
   - 高效、低毒、低残留符合政策

2. **碳五新材料**
   - 戊酮系列2024年投产，爬坡
   - 环戊烷系列试生产
   - COC高端新材料研发

3. **全产业链优势**
   - 二甲戊灵国内唯一全产业链
   - 成本、质量优势

### 催化剂

- **农药价格**：二甲戊灵价格企稳
- **库存周期**：农药补库周期
- **新材料**：戊酮、环戊烷放量，COC产业化
