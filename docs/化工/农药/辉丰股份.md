## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏辉丰生物农业股份有限公司 |
| 证券代码 | 002496（*ST辉丰） |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1989年 |
| 上市时间 | 2010年11月9日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 农药原药及化肥，**退市风险警示(*ST)** |

**公司概况**：
辉丰股份主要从事农药及化肥的研发、生产和销售。因2024年预计净利润为负且营业收入低于3亿元，公司股票被实施退市风险警示（*ST）。主营业务承压，新产品研发和市场开拓未达预期，联营企业安道麦辉丰持续亏损，金融资产公允价值波动带来损失。**投资需高度警惕退市风险。**

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **农药原药** | 农药原药生产 | 农化产业链 |
| **农药制剂** | 终端制剂产品 | 农业植保 |
| **化肥** | 化肥产品 | 农业种植 |

### 业务模式

公司曾为农药原药主要供应商，后因环保、停产整改等因素业务收缩。目前主营业务承压，新产品研发和市场开拓未达预期。联营企业安道麦辉丰(江苏)有限公司持续亏损，拖累业绩。持有金融资产公允价值波动带来显著损失。

### 技术路线/核心技术

- **农药原药**：传统农药原药工艺
- **化肥**：化肥生产
- **转型**：农业产业链变革中转型成效尚未显现

**技术优势**：
曾具备农药原药产能，目前业务收缩。转型成效待观察。

## 五、投资亮点

### 核心亮点

- ⚠️ **退市风险高**：营收<3亿、净利润为负，*ST状态
- ⚠️ 主营业务承压，转型成效待观察
- ⚠️ 联营企业亏损、金融资产损失
- ⚠️ 现金流薄弱，流动比率0.35

**潜在观察点**（高风险）：
- 若公司成功扭亏、营收恢复至3亿以上，可申请撤销*ST
- 资产重组、业务转型等可能性（不确定性高）

### 成长驱动因素

1. **扭亏保壳**
   - 需实现净利润转正且营收≥3亿元
   - 主营业务改善或资产处置

2. **转型**
   - 农业产业链变革中转型
   - 新产品、新市场开拓

3. **联营企业**
   - 安道麦辉丰若扭亏，减少拖累

### 催化剂

- **保壳**：扭亏、营收恢复（高度不确定）
- **重组**：资产重组、业务整合（不确定性高）
