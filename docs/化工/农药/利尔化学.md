## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 利尔化学股份有限公司 |
| 证券代码 | 002258 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 2008年7月8日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 草铵膦龙头，国内最大草铵膦原药生产企业 |

**公司概况**：
利尔化学是国内草铵膦龙头，主要从事农药原药、制剂及中间体的研发、生产和销售。草铵膦产能规模国内领先，2024年受草铵膦价格大幅下跌影响业绩承压，但产能建设稳步推进，精草铵膦项目主体建设进行中。2025年Q1业绩预告显示回暖迹象，行业整合背景下龙头优势有望凸显。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **草铵膦** | 非选择性除草剂，抗草甘膦杂草替代方案 | 农业除草、转基因作物 |
| **精草铵膦** | 草铵膦L-型异构体，药效更高 | 农业除草 |
| **农药原药** | 多种农药原药品种 | 杀虫剂、除草剂、杀菌剂 |
| **农药制剂** | 终端制剂产品 | 农业植保 |
| **农药中间体** | 农药合成中间体 | 原药生产 |

### 业务模式

公司采用原药-制剂一体化模式，从中间体到原药、制剂全产业链覆盖。2024年产销量14.90万吨同比+23.94%，量增价减。原药产能8.632万吨/年，利用率94.40%，在建产能2.2万吨/年。拟在荆州、岳阳等基地推进重点项目，优化生产工艺，加快海外市场布局。

### 技术路线/核心技术

- **草铵膦合成**：草铵膦全流程工艺，产能规模国内领先
- **精草铵膦**：L-型异构体技术，药效提升，项目推进中
- **连续化生产**：工艺优化，成本控制

**技术优势**：
草铵膦产能国内第一，精草铵膦项目推进中。农药中间体2024年逆势增长42.49%，显示技术延伸能力。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
化工原料 → 中间体 → 【利尔化学 草铵膦原药/制剂】→ 农资流通 → 农业种植
                         ↑
                    利尔化学
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 草铵膦产能国内第一，原药产能8.632万吨/年 |
| **竞争优势** | 草铵膦全产业链、精草铵膦技术储备 |
| **客户结构** | 国内外农化企业、制剂厂商 |
| **供应商** | 化工原料采购 |

### 竞争格局

**主要竞争对手**：
- 扬农化工：菊酯龙头
- 红太阳：吡啶碱、草铵膦
- 福华通达：草甘膦、草铵膦

**公司优势**：
草铵膦龙头，产能规模领先。精草铵膦项目推进中，药效更高、用量更少，有望打开成长空间。2024年业绩承压但产能利用率仍达94.40%，行业整合后龙头有望受益。

## 五、投资亮点

### 核心亮点

- ✅ **草铵膦龙头**：产能国内第一，原药产能8.632万吨/年，利用率94.40%
- ✅ **精草铵膦布局**：L-型异构体药效更高，项目主体建设推进中
- ✅ **拐点逻辑**：草铵膦价格创近5年新低，行业亏损倒逼产能出清，价格触底后弹性大
- ✅ **催化剂**：农药行业补库周期、草铵膦替代草甘膦需求、转基因作物推广
- ✅ **中间体逆势增长**：2024年中间体收入+42.49%，技术延伸能力强

### 成长驱动因素

1. **草铵膦价格修复**
   - 2024年价格跌至4.55万元/吨近5年新低，行业亏损面扩大
   - 产能出清后价格弹性大
   - 抗草甘膦杂草增加，草铵膦替代需求长期存在

2. **精草铵膦放量**
   - 精草铵膦药效更高、用量更少，符合农药减量增效政策
   - 项目建成后有望贡献增量

3. **产能建设推进**
   - 荆州、岳阳基地重点项目推进
   - 在建产能2.2万吨/年

### 催化剂

- **农药价格**：草铵膦价格触底反弹
- **库存周期**：海外补库、制剂出口改善
- **政策**：转基因作物推广、农药减量增效利好高效品种
