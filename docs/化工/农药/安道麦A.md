## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 安道麦股份有限公司 |
| 证券代码 | 000553 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 1993年12月3日 |
| 所属行业 | 化学原料和化学制品制造业（农药） |
| 市场地位 | 全球仿制农药龙头，全球市场份额约5% |

**公司概况**：
安道麦A是全球作物保护领域领军企业，主要从事非专利（仿制）农药原药的开发、生产和销售，产品涵盖除草剂、杀虫剂和杀菌剂。2024年全球植保行业销售规模下滑、原药价格处于历史低位，公司亏损29.03亿元同比扩大80.79%。但经营现金流37.61亿元同比+43.65%，连续2年上涨。全球仿制农药龙头地位稳固，行业拐点来临有望受益。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **除草剂** | 非专利除草剂原药及制剂 | 农业除草 |
| **杀虫剂** | 非专利杀虫剂原药及制剂 | 农业害虫防治 |
| **杀菌剂** | 非专利杀菌剂原药及制剂 | 作物病害防治 |

### 业务模式

公司采用"原药+制剂"一体化模式，面向全球市场。非专利农药（仿制农药）为主，与专利农药形成差异化。全球布局，产品销往100多个国家。2024年实施"奋进"计划重组，销售费用增加。

### 技术路线/核心技术

- **仿制农药**：非专利农药开发、生产
- **全球登记**：多国农药登记，全球销售
- **原药-制剂一体化**：从原药到制剂全产业链

**技术优势**：
全球仿制农药龙头，全球市场份额约5%。产品线丰富，覆盖除草剂、杀虫剂、杀菌剂。全球登记网络完善。

## 四、行业地位与产业链定位

### 产业链定位

```
农药产业链：
化工原料 → 中间体 → 【安道麦A 仿制农药原药/制剂】→ 全球农资 → 农业种植
                            ↑
                       安道麦A
                    （全球市场份额约5%）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球仿制农药龙头，全球市场份额约5% |
| **竞争优势** | 全球登记网络、产品线丰富、原药-制剂一体化 |
| **客户结构** | 全球100+国家，覆盖主要农业市场 |
| **供应商** | 原药自产+外采 |

### 竞争格局

**主要竞争对手**：
- 先正达：专利+仿制农药
- 拜耳：专利农药
- 扬农化工、利尔化学、润丰股份：中国农药龙头

**公司优势**：
全球仿制农药龙头，全球市场份额约5%。产品线丰富，全球布局。行业拐点来临，仿制农药需求弹性大。

## 五、投资亮点

### 核心亮点

- ✅ **全球仿制农药龙头**：全球市场份额约5%，产品销往100+国家
- ✅ **经营现金流**：37.61亿元同比+43.65%，连续2年上涨，现金流稳健
- ✅ **拐点逻辑**：全球植保行业拐点，仿制农药价格弹性大
- ✅ **催化剂**：农药价格企稳、全球补库、"奋进"计划成效
- ✅ **毛利率改善**：22.85%同比+2.12pct，成本控制见效

### 成长驱动因素

1. **行业拐点**
   - 全球植保行业销售规模下滑后有望复苏
   - 原药价格历史低位，企稳后弹性大
   - 仿制农药性价比优势，需求恢复

2. **"奋进"计划**
   - 重组优化，成本控制
   - 产品结构优化

3. **全球布局**
   - 100+国家销售，分散风险
   - 全球登记网络完善

### 催化剂

- **农药价格**：全球原药价格企稳回升
- **库存周期**：全球农化补库周期
- **重组成效**："奋进"计划成本优化
