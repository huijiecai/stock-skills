## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 凯莱英医药集团（天津）股份有限公司 |
| 证券代码 | 002821（深交所主板）；H 股 06821.HK |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998 年（集团发展以官网披露为准） |
| 上市时间 | 2016 年 11 月 18 日（A 股） |
| 所属行业 | 生物制药 / 医药研发外包（CDMO） |
| 市场地位 | 国内小分子 CDMO 第一梯队，全球化运营 |

**公司概况**：  
凯莱英为全球制药企业提供一站式医药外包服务，核心为小分子 CDMO，并持续向化学大分子、生物药 CDMO、临床 CRO 及新兴业务延伸。客户以跨国药企与 Biotech 为主，产能与质量体系对标国际监管要求。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **小分子 CDMO** | 临床至商业化阶段的工艺开发与生产 | 创新药原料药与中间体 |
| **化学大分子 / 多肽等** | 寡核苷酸、多肽等新兴化学药 CDMO | 抗肿瘤、代谢、罕见病等 |
| **生物大分子 CDMO** | 抗体、ADC 等生物药开发与生产 | 生物创新药、ADC |
| **新兴服务与制剂** | 制剂、临床供应链、技术输出等 | 全生命周期外包 |

### 业务模式

订单驱动型定制研发生产：按项目里程碑与供货量确认收入，大单商业化项目对营收与利润波动影响显著（历史上曾出现新冠相关大订单带来的高基数）。

### 技术路线/核心技术

- **连续反应与绿色化学**：降本、降耗、规模化生产。
- **工艺开发与放大**：从实验室到吨级生产的工程化能力。
- **全球合规体系**：满足 FDA、EMA、NMPA 等审计要求。

**技术优势**：  
在高活化合物、复杂合成路线、连续生产等领域积累深，与头部药企长期合作形成认证壁垒；新兴业务（化学大分子、生物药）为第二增长曲线。

## 四、行业地位与产业链定位

### 产业链定位

```
创新药产业链：
靶点发现 / 临床前 → 临床 CRO → 【CDMO/CMO：凯莱英】→ 制剂与商业化 → 患者
                                    ↑
                              工艺开发与规模化生产
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球小分子 CDMO 重要参与者；国内第一梯队 |
| **竞争优势** | 国际化质量体系、大客户绑定、连续反应与复杂合成能力 |
| **客户结构** | 大型跨国药企 + 中小创新药企；2025 年大型药企与中小药企收入约 29.16 亿 / 37.55 亿元（同比均正增长） |
| **资本开支** | 新兴业务与产能扩建需持续跟踪折旧与爬坡节奏 |

### 竞争格局

**主要竞争对手**：

- **药明康德**：一体化 CXO 龙头，规模与业务广度领先。
- **博腾股份、九洲药业、药石科技等**：小分子 CDMO 与工具化合物等细分重叠。
- **康龙化成**：实验室服务 + CDMO 协同。

**公司优势**：  
在小分子深度制造与国际化交付上品牌力强；新兴业务若持续放量，有助于平滑单一业务周期。

## 五、投资亮点

### 核心亮点

- ✅ **行业β**：全球创新药外包率提升与国内工程师红利长期逻辑仍在。
- ✅ **业绩修复能见度**：2025 年年报显示收入与利润同比回升，验证大订单退坡后的再增长能力。
- ✅ **项目梯队**：临床后期与商业化项目数量决定未来 2–3 年转化。
- ✅ **新兴业务**：化学大分子、生物药 CDMO、ADC 相关需求为长期选项价值。
- ✅ **财务稳健**：资产负债率长期处于行业偏低水平（具体以各期年报为准）。

### 成长驱动因素

1. **海外客户订单与欧美产能布局**  
   - 地缘与关税政策变化下，海外交付能力与合规记录是关键。

2. **国内创新药投融资与 IND 数量**  
   - Biotech 融资环境影响临床阶段项目增量。

3. **新兴业务爬坡**  
   - 多肽、ADC、制剂等占比提升有望优化长期收入结构。

### 催化剂

- 大额商业化订单签约或延长供应协议。
- 新兴业务（化学大分子/生物药）连续季度收入占比提升。
- 行业层面 FDA 检查、海外需求回暖等积极信号。

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **订单波动** | 大订单依赖史导致业绩波动；未来若再出现单项目占比过高，表观增速易失真。 |

| ⚠️ **监管与 ESG** | 环保、安全生产及药品监管审计趋严，合规成本上升。 |
