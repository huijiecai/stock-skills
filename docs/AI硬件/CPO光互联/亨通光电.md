# 亨通光电（600487）投资分析报告

## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏亨通光电股份有限公司 |
| 证券代码 | 600487 |
| 上市板块 | 上海证券交易所（主板） |
| 所属行业 | 光纤光缆与光通信 |
| 市场地位 | 全球光纤光缆龙头之一 |

**公司概况**：
亨通光电是全球领先的光纤光缆与光通信解决方案提供商，业务涵盖光纤预制棒、光纤光缆、光模块、海缆系统等。公司在光通信产业链具有垂直一体化优势，从上游光纤预制棒到下游光模块均有布局，是全球光通信基础设施的重要供应商。

## 二、主营业务分析

### 2.1 业务结构

| 业务板块 | 主要产品/服务 | 应用领域 | 营收占比（估） |
|---------|--------------|---------|---------------|
| **光纤光缆** | 光纤、光缆、光纤预制棒 | 电信网络、数据中心、5G基站 | ~50% |
| **光模块与器件** | 10G~800G光模块、光芯片 | AI数据中心、光通信 | ~20% |
| **海缆系统** | 海底光缆、海底电缆 | 跨海通信、海上风电 | ~20% |
| **其他** | 量子通信、工业互联网等 | 新兴领域 | ~10% |

### 2.2 核心竞争力

1. **光纤光缆产业链垂直整合**
   - 从光纤预制棒到光缆的完整产业链，成本控制能力强
   - 全球光纤光缆市场份额领先

2. **光模块业务快速成长**
   - 400G/800G光模块已量产，积极布局CPO技术
   - 受益AI数据中心光互联需求增长

3. **海缆系统全球领先**
   - 在海底光缆和海上风电电缆领域技术领先
   - 随"一带一路"及全球海缆建设受益

4. **全球化布局**
   - 在全球多地设有生产基地和研发中心
   - 客户覆盖全球主要电信运营商和云厂商

### 2.3 AI硬件相关业务亮点

- **高速光模块**：400G/800G光模块已进入AI数据中心供应链
- **CPO技术布局**：积极参与CPO技术研发，寻求在新一代光互联方案中的突破
- **光纤光缆基础设施**：AI数据中心建设带动光纤光缆需求增长

## 四、投资亮点

1. **AI数据中心光互联需求增长**：光模块和光缆需求双重受益
2. **光纤光缆全球龙头地位**：市场份额稳固，基本盘稳健
3. **海缆系统高成长性**：海上风电、跨海通信等新兴需求旺盛
4. **CPO技术布局**：积极参与前沿技术，寻求增量市场机会

## 五、风险提示

1. **光纤光缆价格波动**：行业周期性明显，价格战压力

3. **海外业务风险**：汇率波动、地缘政治风险
