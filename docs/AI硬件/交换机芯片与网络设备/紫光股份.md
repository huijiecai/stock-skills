## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 紫光股份有限公司 |
| 证券代码 | 000063 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1999年03月 |
| 上市时间 | 1999年11月 |
| 所属行业 | 网络设备、ICT解决方案 |
| 市场地位 | 国内网络设备龙头，ICT解决方案领先 |

**公司概况**：
紫光股份是国内领先的ICT基础设施及服务提供商，旗下新华三是核心业务主体。公司产品涵盖网络设备、计算存储、云计算、安全等，是国内少数能够提供全栈ICT解决方案的企业。在AI数据中心领域，公司是交换机和网络设备的核心供应商。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **网络设备** | 交换机、路由器、WLAN设备 | 数据中心、企业网络、运营商网络 |
| **计算存储** | 服务器、存储系统 | 数据中心、云计算 |
| **云计算** | 私有云、混合云解决方案 | 政企客户、数据中心 |
| **安全产品** | 防火墙、安全网关、安全服务 | 网络安全 |

### 业务模式

公司采用"产品+解决方案+服务"一体化模式，以直销为主（60%）、渠道为辅（40%）。新华三品牌定位中高端市场，客户涵盖政府、金融、教育、互联网等行业。

### 技术路线/核心技术

- **交换机技术**：自研数据中心交换机，支持400G/800G高速互联
- **网络操作系统**：自主研发Comware网络操作系统，功能丰富稳定
- **AI网络解决方案**：AI Fabric智能网络方案，支持AI训练集群

**技术优势**：国内网络设备技术领先企业；自研芯片和操作系统；全栈ICT解决方案能力；服务能力强。

## 四、行业地位与产业链定位

### 产业链定位

```
AI数据中心网络产业链：
芯片 → 设备商 → 【紫光股份】→ 云厂商/企业 → AI应用
(博通等) (交换机)  (网络解决方案)  (阿里、腾讯等)  (AI模型)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内企业网交换机市场份额第一（约35%） |
| **竞争优势** | 全栈ICT解决方案，服务能力强，品牌认知度高 |
| **客户结构** | 政府、金融、教育、互联网、运营商等 |
| **供应商** | 交换芯片来自博通、Marvell等国际厂商 |

### 竞争格局

**主要竞争对手**：华为（ICT龙头）、思科（国际龙头）、锐捷网络（国内领先）

**公司优势**：企业网交换机市占率第一；全栈解决方案能力；政企客户资源丰富；国产替代受益。

## 五、投资亮点

### 核心亮点

- ✅ **企业网交换机龙头**：国内市场份额第一，品牌认知度高
- ✅ **全栈ICT解决方案**：网络+计算+云+安全一站式服务
- ✅ **AI数据中心受益**：400G/800G交换机需求增长
- ✅ **国产替代加速**：政企市场国产化替代空间大
- ✅ **国企改革红利**：紫光集团重整完成，治理改善

### 成长驱动因素

1. **AI数据中心建设**：AI训练集群高速互联需求增加
2. **国产替代**：政企市场网络设备国产化加速
3. **云计算投资**：政企云计算投资持续增长

### 催化剂

- 大型数据中心交换机订单
- 政企国产替代项目
- AI网络解决方案落地
- 海外市场突破
