## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 福建星网锐捷通讯股份有限公司 |
| 证券代码 | 002396 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1996年03月 |
| 上市时间 | 2010年06月 |
| 所属行业 | 网络设备、企业ICT |
| 市场地位 | 国内企业网络设备领先企业 |

**公司概况**：
星网锐捷是国内领先的企业级网络设备和ICT解决方案提供商，旗下锐捷网络是核心业务主体。公司产品涵盖交换机、路由器、WLAN、云桌面、支付终端等，在教育、政府、企业等细分市场具有较强竞争力。在AI数据中心领域，公司是数据中心交换机和网络设备的供应商。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **网络设备** | 交换机、路由器、WLAN设备 | 教育园区、企业网络、数据中心 |
| **云桌面** | 云终端、虚拟桌面解决方案 | 教育、办公、政务 |
| **支付终端** | POS机、智能收银设备 | 零售、金融 |
| **其他业务** | 视频信息应用、智能家居 | 政府、家庭 |

### 业务模式

公司采用"产品+解决方案+服务"模式，以直销为主（55%）、渠道为辅（45%）。锐捷网络在教育、政府细分市场具有优势，采用差异化竞争策略。支付终端业务以ODM为主。

### 技术路线/核心技术

- **交换机技术**：自研数据中心交换机，支持100G/400G高速互联
- **WLAN技术**：Wi-Fi 6/7无线网络解决方案
- **云桌面技术**：虚拟桌面云解决方案，支持大规模部署

**技术优势**：网络设备研发能力强；细分市场深耕；产品性价比高；服务能力强。

## 四、行业地位与产业链定位

### 产业链定位

```
企业网络设备产业链：
芯片 → 设备商 → 【星网锐捷】→ 企业客户 → 业务应用
(博通等) (交换机)  (网络解决方案)  (教育、政府)  (办公/教学)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内企业网交换机市场份额前三（约20%） |
| **竞争优势** | 细分市场深耕，性价比高，服务能力强 |
| **客户结构** | 教育机构、政府部门、企业客户 |
| **供应商** | 交换芯片来自博通、Marvell等国际厂商 |

### 竞争格局

**主要竞争对手**：华为（ICT龙头）、紫光股份（企业网龙头）、思科（国际龙头）

**公司优势**：教育市场领先；性价比高；服务能力强；细分市场深耕。

## 五、投资亮点

### 核心亮点

- ✅ **企业网络设备领先**：教育市场交换机市场份额第一
- ✅ **细分市场深耕**：教育、政府差异化竞争
- ✅ **数据中心业务拓展**：AI数据中心交换机需求增长
- ✅ **盈利能力改善**：毛利率持续提升
- ✅ **国企背景**：福建电子信息集团控股，资源支持

### 成长驱动因素

1. **教育信息化**：智慧校园建设持续推进
2. **AI数据中心建设**：数据中心交换机需求增长
3. **国产替代**：政企市场网络设备国产化加速

### 催化剂

- 教育信息化项目订单
- 数据中心交换机突破
- 国产替代项目落地
- 新产品线拓展
