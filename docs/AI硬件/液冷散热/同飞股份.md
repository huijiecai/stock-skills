# 同飞股份（300990）投资分析报告

## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 三河同飞制冷股份有限公司 |
| 证券代码 | 300990 |
| 上市板块 | 深圳证券交易所（创业板） |
| 所属行业 | 水冷却设备与液冷温控 |
| 市场地位 | 液冷温控设备重要供应商 |

**公司概况**：
同飞股份成立于2001年，专注于水（油）冷却机、激光水冷却机等液冷设备。2024年营业收入21.6亿元，净利润1.534亿元。公司计划定增募资不超12亿元加码液冷温控领域布局，提升产能和技术水平。

---

## 二、主营业务

- **液冷设备**：水冷却机、激光水冷却机、数据中心液冷系统
- **应用领域**：数据中心、新能源储能、半导体、激光加工

---

## 三、投资亮点

1. **液冷赛道高景气**：AI算力和数据中心液冷需求快速增长
2. **产能扩张**：12亿定增资金用于三河液冷设备生产线，华南市场拓展
3. **技术积累**：20年液冷技术经验，产品性能稳定
4. **多领域应用**：数据中心、储能、半导体多点开花

---

## 四、风险提示

1. **竞争加剧**：液冷市场进入者众多，竞争激烈
2. **下游需求波动**：AI算力投资周期性波动影响订单
3. **技术迭代风险**：液冷技术快速更新，需持续研发投入
