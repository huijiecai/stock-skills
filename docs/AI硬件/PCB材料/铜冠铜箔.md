# 铜冠铜箔（301217）投资分析报告

## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 铜陵有色金属集团铜冠铜箔有限公司 |
| 证券代码 | 301217 |
| 上市板块 | 深圳证券交易所（创业板） |
| 所属行业 | PCB材料-电子铜箔 |
| 市场地位 | 新一代电子铜箔行业龙头 |

**公司概况**：
铜冠铜箔成立于2010年，专注于PCB铜箔和锂电池铜箔的研发、制造与销售。公司PCB铜箔产品包括HTE、RTF、HTE-W和HVLP铜箔，在高频高速基板用铜箔（HVLP+RTF）方面具有显著市场优势。2024年底总产能达8万吨（PCB铜箔3.5万吨，锂电池铜箔4.5万吨），2025年上半年营收29.97亿元，同比增长45%，其中铜箔业务占比94%。

---

## 二、主营业务

- **HVLP铜箔**：超低轮廓铜箔，表面粗糙度Rz≤2.0μm，降低信号传输损耗
- **HTE铜箔**：高温高延铜箔
- **应用领域**：AI服务器、通信、计算机、消费电子、新能源锂电

---

## 三、投资亮点

1. **AI服务器PCB核心材料**：HVLP铜箔是M9 CCL的关键材料，需求高增长
2. **产能规模领先**：PCB铜箔产能3.5万吨，全球领先
3. **技术优势显著**：HVLP铜箔Rz≤2.0μm，满足超5G和高性能计算需求
4. **业绩高增长**：2025年上半年营收同比增长45%，铜箔业务占比94%
5. **国资背景**：铜陵有色金属集团持股72.38%，资源保障能力强

---

## 四、风险提示

1. **铜价波动**：原材料铜价波动影响成本和毛利率
2. **下游需求波动**：AI服务器资本开支波动影响订单
3. **竞争加剧**：HVLP铜箔市场竞争激烈
4. **技术迭代压力**：铜箔技术快速更新
