# 联瑞新材（688300）投资分析报告

## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏联瑞新材料股份有限公司 |
| 证券代码 | 688300 |
| 上市板块 | 上海证券交易所（科创板） |
| 所属行业 | PCB材料-球形硅微粉（填料） |
| 市场地位 | 高性能球形硅微粉领先企业 |

**公司概况**：
联瑞新材专注于高性能球形硅微粉等电子材料的研发和生产，产品是高频高速覆铜板（CCL）的关键填料。随着AI服务器对M9材料的需求提升，球形硅微粉在M9 CCL中的使用比例预计超过40%，公司有望充分受益。

---

## 二、主营业务

- **球形硅微粉**：高性能填料，应用于高频高速CCL
- **应用领域**：AI服务器PCB、5G通信、高端电子

---

## 三、投资亮点

1. **M9材料升级核心受益**：球形硅微粉在M9 CCL中占比超40%，需求大增
2. **技术壁垒高**：高性能球形硅微粉制备技术门槛高
3. **AI服务器需求拉动**：AI服务器PCB价值量是传统产品的5-7倍
4. **供应紧张**：高性能填料供给紧张，价格有支撑

---

## 四、风险提示

1. **下游需求波动**：AI服务器资本开支波动影响订单
2. **竞争加剧**：PCB材料市场竞争激烈
3. **技术迭代风险**：材料技术快速更新
