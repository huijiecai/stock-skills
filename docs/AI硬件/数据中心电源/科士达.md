**公司概况**：
科士达是国内领先的电源设备和新能源解决方案提供商，UPS电源市场份额位居国内前三。公司业务涵盖数据中心关键基础设施、光伏逆变器、储能系统、充电桩等，是国内为数不多具备"光储充"一体化解决方案能力的企业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **UPS电源** | 在线式UPS、模块化UPS、工业UPS | 数据中心、金融、政府、医疗 |
| **光伏逆变器** | 组串式逆变器、集中式逆变器 | 光伏电站、分布式光伏 |
| **储能系统** | 储能变流器、储能集成系统 | 电网侧储能、工商业储能 |
| **充电桩** | 直流快充、交流慢充、充电模块 | 公共充电站、小区充电 |

### 业务模式

公司采用"自主品牌+OEM/ODM"双轮驱动模式，国内市场以自有品牌直销为主，海外市场以OEM/ODM和渠道销售为主。新能源业务采用"产品+系统集成"模式，提供从设备到运维的一站式服务。

### 技术路线/核心技术

- **模块化UPS技术**：自主研发高功率密度模块化UPS，支持热插拔和N+X冗余
- **光伏逆变器技术**：组串式逆变器效率达98.5%以上，支持智能运维
- **储能系统集成**：光储一体化解决方案，支持削峰填谷和备电场景

**技术优势**：30年电源技术积累，UPS和逆变器专利超300项；光伏逆变器出货量全球前十；储能系统集成能力领先。

## 四、行业地位与产业链定位

### 产业链定位

```
光储充产业链：
上游器件 → 中游设备 → 【科士达】→ 下游客户 → 终端应用
(电池、硅料) (逆变器/UPS) (光储充一体) (电站、数据中心) (新能源/云计算)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 光伏逆变器全球前十，UPS国内前三 |
| **竞争优势** | 光储充一体化解决方案，成本控制能力强 |
| **客户结构** | 国家电网、南方电网、大型电站开发商、互联网客户 |
| **供应商** | IGBT进口为主，电池来自宁德时代、亿纬锂能 |

### 竞争格局

**主要竞争对手**：阳光电源（光伏逆变器龙头）、华为（逆变器领先）、科华数据（UPS竞争）

**公司优势**：产品线覆盖广，光储充一体化能力；性价比优势明显；海外渠道布局完善；客户资源丰富。

## 五、投资亮点

### 核心亮点

- ✅ **光储充一体化**：业内少数具备完整光储充解决方案能力的企业
- ✅ **储能业务高增长**：储能收入占比提升，毛利率较高
- ✅ **海外业务拓展**：出口收入增长快，海外毛利高于国内
- ✅ **UPS业务稳健**：数据中心需求支撑，国产替代受益
- ✅ **充电桩布局完善**：覆盖交直流全产品线，受益新能源车普及

### 成长驱动因素

1. **储能需求爆发**：电网侧储能、工商业储能快速增长
2. **光伏装机持续**：分布式光伏渗透率提升，逆变器需求稳定
3. **数据中心建设**：AI驱动数据中心投资，UPS需求增加

### 催化剂

- 大型储能项目中标
- 海外市场突破
- 光储充一体化项目落地
- 数据中心大客户签约
