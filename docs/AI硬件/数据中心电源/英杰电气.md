## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 四川英杰电气股份有限公司 |
| 证券代码 | 300820 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1996年04月 |
| 上市时间 | 2020年02月 |
| 所属行业 | 工业电源、半导体设备电源 |
| 市场地位 | 国内工业电源龙头，半导体设备电源领先 |

**公司概况**：
英杰电气是国内领先的工业电源设备制造商，专注于功率控制电源、特种电源领域。公司是半导体设备电源的核心供应商，产品应用于光伏、半导体、冶金、化工等行业。在AI数据中心电源领域，公司布局半导体电源和功率控制设备。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **功率控制电源** | 晶闸管电源、直流电源、脉冲电源 | 光伏、半导体、冶金、化工 |
| **半导体设备电源** | 单晶炉电源、PECVD电源、刻蚀电源 | 光伏硅片、半导体晶圆制造 |
| **特种电源** | 高频开关电源、电化学电源 | 表面处理、电镀、电解 |
| **工业配套电源** | 大功率整流器、电解电源 | 有色冶金、化工 |

### 业务模式

公司采用"定制化设计+生产+服务"模式，以直销为主（80%）、经销为辅（20%）。产品以定制化为主，根据客户工艺需求设计和制造。光伏和半导体行业客户贡献主要收入。

### 技术路线/核心技术

- **功率控制技术**：自主研发晶闸管控制技术，功率达兆瓦级
- **半导体设备电源**：单晶炉、PECVD、刻蚀等设备电源，精度和稳定性行业领先
- **高频开关电源**：高效率、高功率密度设计，效率达95%以上

**技术优势**：20+年工业电源技术积累，专利超100项；半导体设备电源国产替代领先；与晶盛机电、北方华创等设备商深度合作。

## 四、行业地位与产业链定位

### 产业链定位

```
半导体设备电源产业链：
上游器件 → 中游电源 → 【英杰电气】→ 设备商 → 晶圆厂
(IGBT、晶闸管) (设备电源)  (设备集成)  (晶盛、北方华创) (中芯国际等)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 光伏单晶炉电源市占率超50%，半导体设备电源国产领先 |
| **竞争优势** | 技术门槛高，定制化能力强，客户粘性强 |
| **客户结构** | 晶盛机电、北方华创、TCL中环、隆基绿能等头部企业 |
| **供应商** | IGBT进口为主，晶闸管国产替代进行中 |

### 竞争格局

**主要竞争对手**：国际电源巨头（如日本变压器株式会社）、国内工业电源企业

**公司优势**：半导体设备电源国产替代领先；高毛利、高净利率；与头部设备商深度绑定；技术壁垒高。

## 五、投资亮点

### 核心亮点

- ✅ **半导体设备电源国产替代**：核心设备电源国产化，自主可控
- ✅ **毛利率行业领先**：毛利率43%，净利率24%，盈利能力优异
- ✅ **绑定头部设备商**：晶盛机电、北方华创等核心供应商
- ✅ **光伏电源龙头**：单晶炉电源市占率超50%
- ✅ **AI数据中心电源布局**：功率控制电源可应用于数据中心供配电

### 成长驱动因素

1. **半导体设备国产化**：国产半导体设备替代加速，电源需求增长
2. **光伏扩产持续**：光伏硅片产能扩张，设备电源需求稳定
3. **AI数据中心电源**：高功率电源需求增加，新业务拓展

### 催化剂

- 半导体设备电源新客户认证
- 大型半导体设备订单
- AI数据中心电源项目落地
- 新应用领域拓展
