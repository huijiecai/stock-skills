## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 易事特集团股份有限公司 |
| 证券代码 | 300376 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2001年06月 |
| 上市时间 | 2014年01月 |
| 所属行业 | 电源设备、新能源 |
| 市场地位 | 国内UPS领先企业，智慧能源解决方案提供商 |

**公司概况**：
易事特是国内领先的电源设备和智慧能源解决方案提供商，UPS电源市场份额位居国内前列。公司业务涵盖数据中心电源、光伏储能、充电桩、智慧能源等，是"电源+新能源"双主业布局的典型代表。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **UPS电源** | 高端UPS、模块化UPS、工业UPS | 数据中心、金融、政府、医疗 |
| **光伏逆变器** | 组串式逆变器、集中式逆变器 | 光伏电站、分布式光伏 |
| **储能系统** | 储能变流器、储能集成系统 | 电网侧储能、工商业储能 |
| **充电桩** | 直流快充、交流慢充、充电模块 | 公共充电站、小区充电 |
| **数据中心解决方案** | 模块化数据中心、精密空调 | 云计算、企业机房 |

### 业务模式

公司采用"产品+解决方案+服务"一体化模式，以直销为主（55%）、渠道为辅（45%）。数据中心业务采用解决方案模式，新能源业务采用EPC和产品销售模式。近年来新能源业务占比持续提升。

### 技术路线/核心技术

- **高端UPS技术**：自主研发大功率UPS，功率密度和效率行业领先
- **光储一体化技术**：光伏+储能智能调度系统，支持多种应用场景
- **智慧能源管理**：能源管理平台，支持多能源协同优化

**技术优势**：20+年电源技术积累，UPS和新能源专利超200项；参与多项国家标准制定；自主研发能力强。

## 四、行业地位与产业链定位

### 产业链定位

```
智慧能源产业链：
上游器件 → 中游设备 → 【易事特】→ 下游客户 → 终端应用
(电池、IGBT) (电源/逆变器) (智慧能源方案) (电网、企业) (新能源/数据中心)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | UPS国内前五，光伏逆变器国内中游 |
| **竞争优势** | 电源+新能源双主业，解决方案能力强 |
| **客户结构** | 国家电网、南方电网、数据中心运营商、企业客户 |
| **供应商** | IGBT进口为主，电池来自宁德时代等 |

### 竞争格局

**主要竞争对手**：阳光电源（光伏逆变器龙头）、科华数据（UPS龙头）、科士达（光储充一体）

**公司优势**：双主业协同，电源技术积累深厚；解决方案能力强；客户资源丰富。

## 五、投资亮点

### 核心亮点

- ✅ **电源+新能源双主业**：UPS和新能源业务协同发展
- ✅ **储能业务高增长**：储能收入占比提升至30%，毛利较高
- ✅ **数据中心需求支撑**：AI驱动数据中心投资，UPS需求稳定
- ✅ **智慧能源布局**：能源管理平台，支持多场景应用
- ✅ **国企背景**：广东恒健入主，资源协同

### 成长驱动因素

1. **储能需求爆发**：电网侧储能、工商业储能快速增长
2. **数据中心建设**：AI驱动数据中心投资，高端UPS需求增加
3. **国企改革红利**：国资背景带来资源和渠道支持

### 催化剂

- 大型储能项目中标
- 数据中心大客户签约
- 国企改革进展
- 智慧能源项目落地
