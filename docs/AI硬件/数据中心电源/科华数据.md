## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 科华数据股份有限公司 |
| 证券代码 | 002335 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1988年06月 |
| 上市时间 | 2010年01月 |
| 所属行业 | 电源设备、数据中心基础设施 |
| 市场地位 | 国产UPS电源龙头，数据中心基础设施领先 |

**公司概况**：
科华数据是国内领先的高端电源设备制造商，UPS电源市场份额国产第一。公司业务涵盖数据中心基础设施、智慧电能、新能源三大板块，是AI数据中心电源的核心供应商，产品广泛应用于金融、政府、互联网等关键领域。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **UPS电源** | 大功率UPS、模块化UPS、工业级UPS | 数据中心、金融、政府、医疗 |
| **数据中心基础设施** | 模块化数据中心、微模块、精密空调 | 云计算、互联网、企业机房 |
| **新能源产品** | 光伏逆变器、储能系统、充电桩 | 光伏电站、储能项目、充电设施 |
| **智慧电能** | 精密空调、配电系统、动环监控 | 机房运维、智能建筑 |

### 业务模式

公司采用"产品+解决方案+服务"一体化模式，直销为主（60%）、渠道为辅（40%）。数据中心业务采用总包和分包模式，新能源业务采用EPC模式。毛利率较高的高端UPS和数据中心解决方案占比持续提升。

### 技术路线/核心技术

- **高端UPS技术**：自主研发IGBT模块化UPS，效率达96%以上，功率密度行业领先
- **数据中心液冷技术**：布局浸没式液冷数据中心方案，PUE可达1.1以下
- **储能集成技术**：光储一体化解决方案，支持削峰填谷和备电场景

**技术优势**：30+年电源技术积累，UPS专利超200项；参与制定多项国家标准；自主研发能力国内领先。

## 四、行业地位与产业链定位

### 产业链定位

```
数据中心电源产业链：
上游器件 → 中游设备 → 【科华数据】→ 下游客户 → 终端应用
(电池、IGBT) (UPS整机)   (电源+数据中心)  (互联网、金融) (AI/云计算)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国产UPS市占率第一，数据中心基础设施国内前三 |
| **竞争优势** | 高端UPS技术自主可控，客户资源优质，品牌认知度高 |
| **客户结构** | 阿里、腾讯、华为、工农中建等头部客户 |
| **供应商** | 电池来自南都、雄韬；IGBT依赖进口（英飞凌等） |

### 竞争格局

**主要竞争对手**：施耐德（国际龙头）、伊顿（国际领先）、华为（数据中心能源）

**公司优势**：国产UPS技术最成熟；数据中心全栈解决方案能力；客户粘性强；本土化服务优势明显。

## 五、投资亮点

### 核心亮点

- ✅ **国产UPS龙头**：高端UPS技术自主可控，国产替代空间大
- ✅ **AI数据中心核心供应商**：大型数据中心UPS需求爆发
- ✅ **新能源业务高增长**：储能、充电桩业务快速放量
- ✅ **液冷数据中心领先**：布局浸没式液冷方案，PUE行业领先
- ✅ **客户资源优质**：互联网大厂、金融机构深度合作

### 成长驱动因素

1. **AI算力需求爆发**：AI服务器功耗提升，高端UPS需求增加
2. **数据中心建设加速**：东数西算推动数据中心投资
3. **新能源政策支持**：储能、充电桩政策红利持续

### 催化剂

- AI数据中心大项目中标
- 液冷数据中心商用落地
- 储能项目大单签约
- 国产替代政策加码
