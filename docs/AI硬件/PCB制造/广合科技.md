# 广合科技（001389）投资分析报告

## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广东广合科技股份有限公司 |
| 证券代码 | 001389 |
| 上市板块 | 深圳证券交易所（主板） |
| 所属行业 | 印制电路板（PCB）制造 |
| 市场地位 | AI服务器PCB领先厂商 |

**公司概况**：
广合科技专注于高端多层印制电路板的研发和生产，尤其在AI服务器PCB领域表现突出。公司服务器PCB收入占比约70%，客户包括全球顶尖的服务器ODM和云计算公司（如亚马逊、英伟达）。

---

## 二、主营业务

- **AI服务器PCB**：高速、高频产品，PCIe 6.0技术达到国际先进水平
- **高阶HDI与多层板**：拥有407项专利，技术壁垒显著
- **全球客户资源**：与亚马逊、英伟达等头部客户深度合作

---

## 三、投资亮点

1. **AI服务器PCB龙头**：服务器PCB收入占比70%，全球市场占有率12.4%
2. **技术壁垒高**：PCIe 6.0技术领先，客户粘性强
3. **高毛利率**：约35%毛利率，盈利质量优异
4. **产能布局完善**：广州、东莞、黄石、泰国四地布局，规模效应显著

---

## 四、风险提示

1. **客户集中度高**：主要收入依赖头部客户，订单波动风险
2. **AI服务器需求波动**：下游资本开支波动影响业绩
3. **技术迭代压力**：PCB技术快速更新，需持续研发投入
