# 崇达技术（002815）投资分析报告

## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 崇达技术股份有限公司 |
| 证券代码 | 002815 |
| 上市板块 | 深圳证券交易所（主板） |
| 所属行业 | 印制电路板（PCB）制造 |
| 市场地位 | 国内高端PCB重要厂商 |

**公司概况**：
崇达技术成立于2004年，专注于高频高速、高多层PCB的研发和生产。公司产品广泛应用于通信、工业控制、AI服务器等领域，70层高多层板和800G光模块PCB已通过英伟达GB300架构认证，打破海外垄断。

---

## 二、主营业务

- **高多层PCB**：70层超高层板，单板价值是传统产品的5-8倍
- **AI服务器PCB**：与国产GPU厂商合作，PCB产品已应用于AI服务器
- **通信PCB**：通过安费诺等供应商间接为英伟达等客户供货通信背板PCB

---

## 三、投资亮点

1. **AI服务器PCB龙头**：70层高多层板通过英伟达认证，技术壁垒高
2. **国产GPU合作**：与国产GPU厂商深度合作，受益国产替代
3. **产能扩张**：泰国新厂即将投产，提升供应能力
4. **高附加值产品**：超高层板单价显著高于传统产品，毛利率提升空间大

---

## 四、风险提示

1. **下游资本开支波动**：AI服务器需求波动影响订单
2. **技术迭代风险**：PCB技术快速更新，需持续研发投入
3. **竞争加剧**：高端PCB市场竞争激烈
