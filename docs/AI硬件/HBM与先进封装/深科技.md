## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 深圳长城开发科技股份有限公司 |
| 证券代码 | 000021 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1985年 |
| 上市时间 | 1994年02月 |
| 所属行业 | 半导体封装测试/存储制造 |
| 市场地位 | HBM3封装黑马，国内最大DRAM封测企业 |

**公司概况**：
深科技是中国电子信息产业集团下属上市公司，2024年营收约112.8亿元，净利润7.557亿元。公司是国内最大的DRAM封测企业，与长鑫存储深度合作，在HBM3封装领域技术领先，是存储芯片封测赛道的核心标的。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **DRAM封测** | 存储芯片封装测试 | 数据中心、消费电子 |
| **HBM封装** | HBM3等高带宽存储封装 | AI训练、数据中心 |
| **硬盘磁头** | 硬盘驱动器磁头制造 | 数据存储 |
| **智能电表** | 智能电网终端设备 | 电力行业 |

### 业务模式

公司采用多元化业务模式，存储芯片封测业务占比约35%，硬盘磁头业务占比约30%，智能电表业务占比约20%。与长鑫存储深度合作，是国内DRAM封测的核心供应商。

### 技术路线/核心技术

- **HBM3封装**：技术领先，良率优势明显
- **DRAM封测**：国内最大，与长鑫存储深度绑定
- **先进封装**：掌握TSV、2.5D等先进封装技术

**技术优势**：与长鑫存储战略合作；HBM3封装技术领先；封测良率高。

## 四、行业地位与产业链定位

### 产业链定位

```
存储芯片产业链：
晶圆制造 → 【深科技】→ 存储成品 → 终端应用
(长鑫存储)  (封装测试)  (DRAM/HBM) (数据中心)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内DRAM封测规模最大 |
| **竞争优势** | 与长鑫存储深度合作，HBM3技术领先 |
| **客户结构** | 长鑫存储占比高，客户关系稳定 |
| **供应商** | 设备来自国际主流厂商 |

### 竞争格局

**主要竞争对手**：长电科技、通富微电、华天科技等封测企业

**公司优势**：与长鑫存储深度绑定；HBM3封装技术领先；国内DRAM封测龙头。

## 五、投资亮点

### 核心亮点

- ✅ **国内DRAM封测龙头**：与长鑫存储深度合作
- ✅ **HBM3封装领先**：技术优势明显，2026年扩产预期
- ✅ **多元化业务**：存储+硬盘+电表，业务结构稳健
- ✅ **盈利能力改善**：净利润逆势增长
- ✅ **央企背景**：中电集团下属，资源优势明显

### 成长驱动因素

1. **国产存储芯片放量**：长鑫存储产能扩张
2. **HBM需求爆发**：AI训练对HBM需求激增
3. **存储周期复苏**：存储行业景气度回升

### 催化剂

- 长鑫存储产能扩张
- HBM3封装订单增长
- 存储行业景气复苏
- 新客户拓展
