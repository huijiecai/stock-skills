## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏长电科技股份有限公司 |
| 证券代码 | 600584 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1972年 |
| 上市时间 | 2003年06月 |
| 所属行业 | 半导体封装测试 |
| 市场地位 | 全球第三大封测企业，CoWoS/HBM技术领先 |

**公司概况**：
长电科技是全球领先的集成电路封测企业，2024年营收359.62亿元（+21.24%），净利润16.10亿元（+9.44%）。公司在先进封装领域布局深入，CoWoS、HBM封装、XDFOI等2.5D/3D封装技术领先，是AI芯片封装的核心供应商。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **先进封装** | CoWoS、2.5D/3D封装、HBM封装 | AI芯片、GPU、高性能计算 |
| **传统封装** | 引线键合、QFN、BGA等传统封装 | 消费电子、汽车电子 |
| **晶圆级封装** | WLP、Fan-out等晶圆级封装 | 移动设备、物联网 |
| **测试服务** | 晶圆测试、成品测试 | 半导体全产业链 |

### 业务模式

公司采用IDM模式，提供从晶圆级封装到成品测试的一站式服务。先进封装业务占比逐年提升（2024年约35%），毛利率高于传统封装10-15个百分点。客户涵盖全球主流芯片设计公司和晶圆厂。

### 技术路线/核心技术

- **CoWoS封装**：台积电CoWoS供应链核心合作伙伴，月产能0.5-0.8万片
- **HBM封装**：掌握HBM2E/3/3E堆叠封装技术，良率达75-80%
- **XDFOI平台**：自主开发的高密度扇出型封装技术，支持多芯片集成

**技术优势**：全球第三大封测企业，市场份额36.94%，拥有完整的技术专利池和规模化生产能力。

## 四、行业地位与产业链定位

### 产业链定位

```
半导体封装产业链：
晶圆制造 → 【长电科技】→ 芯片成品 → 终端应用
(TSMC等)   (封装测试)   (AI芯片)   (数据中心)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球封测市占率36.94%，排名第三 |
| **竞争优势** | 技术全面、产能规模大、客户资源丰富 |
| **客户结构** | 英伟达、AMD、华为海思等顶级芯片公司 |
| **供应商** | 设备来自ASM、K&S等国际厂商 |

### 竞争格局

**主要竞争对手**：日月光（全球第一）、安靠（全球第二）、通富微电（国内最大）

**公司优势**：先进封装技术领先，与台积电CoWoS合作深入；国内产能布局完善；HBM封装国内领先。

## 五、投资亮点

### 核心亮点

- ✅ **先进封装龙头**：全球第三大封测企业，AI芯片封装核心供应商
- ✅ **CoWoS产能稀缺**：台积电CoWoS供应链核心，产能供不应求
- ✅ **HBM封装领先**：掌握HBM2E/3E封装技术，受益AI算力需求爆发
- ✅ **业绩加速增长**：2025年净利润增速预计39%，成长性确立
- ✅ **国产替代机遇**：先进封装国产化率低，替代空间大

### 成长驱动因素

1. **AI芯片需求爆发**：GPU、AI芯片封装需求大幅增长
2. **CoWoS产能扩张**：台积电CoWoS产能持续紧张，公司受益
3. **HBM需求旺盛**：AI训练对HBM需求激增，封装环节受益

### 催化剂

- 新增CoWoS产能释放
- HBM封装新客户认证
- AI芯片订单增长
- 先进封装技术突破
