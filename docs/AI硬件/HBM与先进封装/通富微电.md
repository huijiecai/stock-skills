## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 通富微电子股份有限公司 |
| 证券代码 | 002156 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1994年 |
| 上市时间 | 2007年08月 |
| 所属行业 | 半导体封装测试 |
| 市场地位 | 国内最大封测企业，AMD核心合作伙伴 |

**公司概况**：
通富微电是国内规模最大的集成电路封测企业，2024年营收约270亿元（+18%），净利润约6.8亿元。公司与AMD深度合作（占AMD订单80%+），在HBM封装、2.5D/3D封装、CPO光电合封领域技术领先，是AI时代封测赛道的核心标的。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **先进封装** | HBM封装、2.5D/3D封装、Chiplet | AI芯片、GPU、高性能计算 |
| **CPU/GPU封装** | 高性能处理器封装 | 数据中心、云计算 |
| **传统封装** | 引线键合、QFN、BGA等 | 消费电子、汽车电子 |
| **CPO封装** | 光电混合封装技术 | 数据中心光互联 |

### 业务模式

公司采用专业封测代工模式，与AMD等国际芯片巨头建立深度战略合作。AMD业务占比约40-50%，是公司最大单一客户。先进封装业务占比逐年提升，毛利率高于传统封装。

### 技术路线/核心技术

- **HBM封装**：HBM2.5D封装良率达98%，技术成熟
- **CPO封装**：光电合封技术已通过可靠性测试，2025年取得突破
- **Chiplet技术**：多芯片集成封装，支持异构计算

**技术优势**：与AMD合作超过15年，在高性能CPU/GPU封装领域技术积累深厚，国内领先。

## 四、行业地位与产业链定位

### 产业链定位

```
半导体封装产业链：
晶圆制造 → 【通富微电】→ 芯片成品 → 终端应用
(TSMC等)   (封装测试)   (CPU/GPU)  (数据中心)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内封测企业规模第一，全球排名第五 |
| **竞争优势** | AMD核心合作伙伴，高性能封装技术领先 |
| **客户结构** | AMD占比40-50%，其他客户包括国内芯片设计公司 |
| **供应商** | 设备来自国际主流厂商 |

### 竞争格局

**主要竞争对手**：长电科技（国内技术领先）、华天科技（国内第三）、日月光（全球龙头）

**公司优势**：国内规模最大；与AMD深度绑定；HBM封装良率领先；CPO技术突破。

## 五、投资亮点

### 核心亮点

- ✅ **国内规模最大**：封测产能国内第一，规模效应显著
- ✅ **AMD核心伙伴**：占AMD订单80%+，合作关系稳定
- ✅ **HBM技术领先**：HBM2.5D良率98%，行业领先
- ✅ **CPO技术突破**：光电合封已通过验证，增量空间大
- ✅ **业绩高增长**：2025年净利润增速62-99%

### 成长驱动因素

1. **AMD AI芯片放量**：MI300系列芯片封装需求增长
2. **HBM需求爆发**：AI训练对HBM需求激增
3. **国产替代**：先进封装国产化空间大

### 催化剂

- AMD新一代AI芯片订单
- HBM封装新客户拓展
- CPO技术商业化
- 国产芯片封装订单增长
