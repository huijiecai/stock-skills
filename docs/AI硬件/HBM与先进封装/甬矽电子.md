## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 甬矽电子（宁波）股份有限公司 |
| 证券代码 | 688362 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2018年 |
| 上市时间 | 2023年05月 |
| 所属行业 | 半导体封装测试 |
| 市场地位 | AIoT封装专业企业，成长型封测厂商 |

**公司概况**：
甬矽电子是国内新兴的集成电路封测企业，2024年营收36.09亿元（+50.96%），净利润约6600万元（扭亏为盈）。公司专注于AIoT、射频前端芯片、汽车电子等领域的封装测试，在SiP、FC类封装等细分领域具有技术优势。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **SiP封装** | 系统级封装，高集成度 | AIoT、可穿戴设备 |
| **FC类封装** | 倒装芯片封装 | 射频前端、通信芯片 |
| **晶圆级封装** | WLP、Fan-out等 | 移动设备、物联网 |
| **QFN/DFN封装** | 无引脚封装 | 消费电子、汽车电子 |

### 业务模式

公司采用专业封测代工模式，客户涵盖国内外芯片设计公司。AIoT和射频前端芯片封装是核心业务，占比约60%。公司专注细分市场，与长电、通富等形成差异化竞争。

### 技术路线/核心技术

- **FHBSAP平台**：集成多种前沿封装技术
- **SiP封装**：高集成度系统级封装
- **FC封装**：倒装芯片封装技术

**技术优势**：专注AIoT细分市场；SiP封装技术领先；产能持续扩张。

## 五、投资亮点

### 核心亮点

- ✅ **AIoT封装专业**：细分市场领先，需求旺盛
- ✅ **扭亏为盈**：业绩拐点确立，进入盈利期

1. **AIoT需求增长**：物联网和智能设备需求旺盛
2. **产能释放**：新增产能支撑营收增长
3. **国产替代**：先进封装国产化空间大

### 催化剂

- AIoT芯片订单增长
- 新客户拓展
- 产能利用率提升
