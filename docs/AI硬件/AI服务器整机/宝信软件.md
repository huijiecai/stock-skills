## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海宝信软件股份有限公司 |
| 证券代码 | 600845 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2001年03月26日 |
| 上市时间 | 2007年02月05日 |
| 所属行业 | 数据中心与服务器集成 |
| 市场地位 | 数据中心一体化解决方案服务商 |

**公司概况**：
宝信软件是宝钢集团下属上市公司，从事数据中心、服务器集成及IT运维服务，2025年前三季度营收70.52亿元（-27.72%），归母净利润10.98亿元（-43.13%）。公司业绩面临较大压力，但作为数据中心解决方案商，在AI时代具有独特的IDM资源优势。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **服务器集成** | 服务器整机集成和定制化改造 | 金融、大型企业、互联网厂商 |
| **数据中心方案** | 数据中心建设、设计、集成方案 | 大型企业、互联网、金融 |
| **IT运维服务** | 数据中心运维、IT基础设施管理 | 金融、大型企业、互联网 |
| **AI服务器集成** | 高端AI服务器集成和优化 | 大模型计算、AI推理应用 |

### 业务模式

公司采用"产品集成+运维服务"的商业模式，其中服务器集成占比约60-70%（毛利率20-30%），数据中心建设和IT运维占比30-40%（毛利率40-50%）。主要客户为宝钢等大型企业和金融机构，议价能力强。

### 技术路线/核心技术

- **服务器集成技术**：自主研发服务器定制化能力
- **数据中心设计**：高效能耗优化、液冷应用设计
- **IT运维管理**：自主研发数据中心管理平台

**技术优势**：数据中心一体化设计能力强，IT运维服务经验丰富（客户粘性高），毛利率在行业高位。

## 四、行业地位与产业链定位

### 产业链定位

```
数据中心解决方案产业链：
上游硬件供应商 → 【宝信软件】→ 大型企业客户 → 终端应用
(服务器、网络设备)  (集成+运维)  (宝钢、金融等)  (业务系统)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 钢铁行业信息化服务龙头，数据中心集成区域领先 |
| **竞争优势** | 宝钢背景、客户资源稳定、IT运维服务能力强 |
| **客户结构** | 宝钢集团占比40%+，金融、政府占比30%，其他企业30% |
| **供应商** | 与主流服务器厂商（浪潮、曙光等）建立合作 |

### 竞争格局

**主要竞争对手**：神州数码（IT集成龙头）、东华软件（系统集成）、华为企业业务

**公司优势**：宝钢集团背景带来稳定客户资源；在钢铁行业信息化领域市占率第一；数据中心运维服务毛利率高。

## 五、投资亮点

### 核心亮点

- ✅ **宝钢背景客户稳定**：宝钢集团提供稳定订单来源
- ✅ **IT运维毛利率高**：运维服务毛利率40%+，盈利能力强
- ✅ **数据中心布局完善**：拥有多个自建数据中心，IDC资源稀缺
- ✅ **AI服务器集成潜力**：可承接AI服务器集成业务，拓展新增长点
- ✅ **国企改革预期**：作为央企下属上市公司，改革预期较强

### 成长驱动因素

1. **AI数据中心需求增长**：AI算力需求带动数据中心建设
2. **IT运维服务拓展**：从钢铁行业向其他行业扩展
3. **IDC资源变现**：自建数据中心租赁业务增长

### 催化剂

- 宝钢集团IT投资增加
- 数据中心客户拓展
- AI服务器集成订单
- 国企改革政策推进
