## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浪潮电子信息产业股份有限公司 |

| 上市时间 | 1996年11月26日 |
| 所属行业 | 服务器与IT基础设施 |

**公司概况**：
浪潮信息是全球领先的IT基础设施供应商，2025年前三季度营收1206.69亿元（+44.85%），AI服务器业务全球市场份额第一。公司深度绑定互联网头部客户，是AI时代基础设施的核心供应商，拥有领先的液冷散热和系统优化技术。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **AI服务器** | M6系列超大规模AI服务器、高端定制AI服务器 | 大模型训练、AI推理、云计算基础设施 |
| **通用服务器** | 机架式、塔式、刀片式服务器 | 企业数据中心、云计算、边缘计算 |
| **存储产品** | 分布式存储系统、全闪存存储阵列 | 数据中心存储、云计算、大数据 |
| **网络产品** | 数据中心交换机、路由器 | 数据中心互连、云网融合 |

### 业务模式

公司采用"硬件+软件+服务"一体化商业模式，70%营收来自互联网大客户（BAT等）直销，20-25%来自渠道商拓展政府金融等传统行业。毛利率主要受产品结构和市场竞争影响。

### 技术路线/核心技术

- **AI服务器架构优化**：M6系列多GPU并行计算设计，支持最新高端芯片
- **液冷散热技术**：浸没式液冷方案商用，功耗降低20-30%
- **系统管理软件**：自研性能调优和管理软件，提升算力利用率

**技术优势**：30+项AI服务器专利，液冷技术行业领先，与英伟达、AMD深度合作开发定制化方案。

## 四、行业地位与产业链定位

### 产业链定位

```
AI服务器产业链：
芯片厂商 → 配件供应 → 【浪潮信息】→ 云厂商/企业 → 终端应用
(英伟达、AMD)  (存储、网络)   (服务器整机)   (BAT等)   (AI应用)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球AI服务器出货量第一，中国服务器市场份额前三 |
| **竞争优势** | AI服务器技术领先，大客户资源丰富，品牌认知度高 |
| **客户结构** | 深度绑定BAT等互联网大厂（占营收60%+），政府金融基础扎实 |
| **供应商** | 与英伟达、AMD保持战略合作，核心器件供应稳定 |

### 竞争格局

**主要竞争对手**：戴尔科技（全球服务器龙头）、中科曙光（国内HPC和AI服务器）、惠普企业（国际老牌厂商）

**公司优势**：AI服务器细分领域绝对领先，出货量全球第一；与互联网大客户关系深度绑定；液冷和定制化方案成熟领先。

## 五、投资亮点

### 核心亮点

- ✅ **AI服务器全球龙头**：市场份额第一，充分受益AI算力爆发
- ✅ **大客户资源优质粘性强**：BAT等深度合作，订单稳定确定性高
- ✅ **液冷技术商业化领先**：浸没式方案大规模商用，差异化优势明显
- ✅ **盈利能力修复可期**：扣非净利润增速超营收增速
- ✅ **国产替代政策加持**：关键领域渗透率提升，市场空间扩大

### 成长驱动因素

1. **AI大模型商业化爆发**：生成式AI推动超大规模服务器需求，2025-2027年年均增长30%+
2. **液冷成为刚需**：高功耗GPU发热量大，液冷成本更优，芯片功耗可降30%
3. **国产芯片适配产生新需求**：国产GPU推出需适配新服务器，替代政策推动

### 催化剂

- AI大模型新品发布及商业化进展
- 云厂商资本开支增加，数据中心建设加速
- 国产AI芯片大规模应用
- 公司定制化产品新发布
