## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 曙光信息产业股份有限公司 |
| 证券代码 | 603019 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997年10月31日 |
| 上市时间 | 2010年11月18日 |
| 所属行业 | 高性能计算与AI服务器 |
| 市场地位 | 国内HPC和AI服务器领先者 |

**公司概况**：
中科曙光是国内领先的高性能计算和AI服务器制造商，2025年营收149.70亿元（+13.86%），归母净利润21.13亿元（+10.54%）。公司拥有扎实的HPC技术基础，推出全球首款单机柜级640卡超节点scaleX640，在AI服务器领域积极布局。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **AI服务器** | scaleX640等高端AI服务器、通用AI服务器 | 大模型训练、科学计算、AI推理 |
| **通用服务器** | 机架式、高密度服务器 | 企业IT、云计算、边缘计算 |
| **存储系统** | 高性能分布式存储、冷存储 | 数据中心、云计算、大数据 |
| **AI训推一体机** | 集成AI训练推理一体化方案 | AI应用部署、边缘AI计算 |

### 业务模式

公司采用"产品+服务+解决方案"的商业模式。通过政府、金融、科研机构等传统行业渠道销售高毛利HPC服务器（毛利率30%+）；通过互联网客户渠道销售通用AI服务器（毛利率15-20%）；服务收入占比约5-8%。

### 技术路线/核心技术

- **HPC技术积累深厚**：20年以上高性能计算技术积累，国内领先
- **scaleX系列AI服务器**：全球首创640卡单机柜架构，密度业界最高
- **智能化管理系统**：自研管理软件和节能优化技术

**技术优势**：拥有HPC和AI服务器相关专利50+项，政府科研合作深度强，国产化能力强。

## 四、行业地位与产业链定位

### 产业链定位

```
高性能计算产业链：
芯片厂商 → 配件供应 → 【中科曙光】→ 科研/云厂商 → 终端应用
(英伟达、AMD)  (存储、网络)  (HPC/AI服务器) (高校、政府)  (科学计算)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内HPC市场占有率第一，AI服务器市场份额持续提升 |
| **竞争优势** | HPC技术深厚，政府科研客户资源优质，高毛利产品占比高 |
| **客户结构** | 政府科研机构占比40%+，互联网客户占比30-35% |
| **供应商** | 与英伟达、AMD、Intel合作深入，国产化替代进行中 |

### 竞争格局

**主要竞争对手**：浪潮信息（AI服务器市场领先者）、同方计算（HPC领域竞争对手）、戴尔、HP等国际厂商

**公司优势**：HPC技术领先，政府科研客户关系密切；产品矩阵完整；毛利率领先行业。

## 五、投资亮点

### 核心亮点

- ✅ **HPC技术深厚积累**：20年技术沉淀，国内市场份额第一，竞争壁垒强
- ✅ **AI服务器创新领先**：640卡scaleX架构全球首创，密度最高
- ✅ **毛利率领先行业**：产品综合毛利率维持24%高位，盈利能力强
- ✅ **扣非净利润高增长**：2025年增速30.17%，盈利质量优秀
- ✅ **政府科研客户稳定**：政府、科研机构订单稳定，长期合作深度强

### 成长驱动因素

1. **国产化替代提速**：国产芯片和系统适配需求增加，曙光政府关系优势明显
2. **AI计算需求增长**：大模型、科学AI等领域计算需求持续增长
3. **高性能存储扩展**：AI数据存储需求增加，冷存储产品快速增长

### 催化剂

- 国产AI芯片（昇腾、飞腾等）商业化应用
- 可转换公司债发行，募资80亿元投入AI产业
- 政府部门和高校AI采购增加
- scaleX系列新产品订单释放
