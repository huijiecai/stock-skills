## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 紫光股份有限公司 |
| 证券代码 | 000938 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1993年08月25日 |
| 上市时间 | 1999年07月01日 |
| 所属行业 | IT基础设施与ICT解决方案 |
| 市场地位 | 新华三服务器与网络设备领导者 |

**公司概况**：
紫光股份通过控股子公司新华三集团从事IT基础设施业务，2025年前三季度营收773.22亿元（+31.41%）。新华三服务器业务营收596.23亿元（+48.07%），国内政企业务表现突出，同时布局AI服务器和网络设备两大核心赛道。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **AI服务器** | 新华三自主研发高密度AI服务器 | 大模型训练、AI推理、云计算 |
| **通用服务器** | 机架式、塔式服务器等 | 企业IT、云计算、政府部门 |
| **网络设备** | 交换机、路由器、防火墙 | 数据中心、园区网、云网融合 |
| **存储系统** | 存储阵列、备份系统 | 数据中心、云计算、大数据 |
| **IT服务** | 咨询、集成、运维服务 | 政府、金融、运营商等 |

### 业务模式

紫光通过新华三进行IT基础设施业务，分为国内政企业务（占比60%+，毛利率20-25%）和互联网定制化服务器业务（占比30%，毛利率10-15%）。政企业务稳定性强，互联网业务增长快但竞争激烈。

### 技术路线/核心技术

- **国产化替代方案**：自主研发支持国产芯片的服务器
- **AI服务器创新**：高密度、高性能定制化AI服务器设计
- **网络设备技术**：自主研发交换机芯片和系统软件

**技术优势**：IT产品线完整，可提供一体化解决方案；国产化替代方案完善；与国产芯片厂商深度合作。

## 四、行业地位与产业链定位

### 产业链定位

```
IT基础设施产业链：
芯片厂商 → 配件供应 → 【紫光/新华三】→ 云厂商/政府/企业 → 终端应用
(国产芯片)  (存储、网络)    (服务器整机)   (BAT、政府)   (IT应用)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内政企市场服务器份额前三，互联网定制化服务器市场份额快速提升 |
| **竞争优势** | 政府金融客户关系深度强，国产化替代完整方案，产品线完整 |
| **客户结构** | 政府金融占60%+，互联网大厂定制化服务器占30%，运营商占5-10% |
| **供应商** | 与飞腾、龙芯等国产芯片商合作深入，国产化布局完善 |

### 竞争格局

**主要竞争对手**：浪潮信息（AI服务器市场龙头）、中科曙光（HPC和AI服务器竞争者）、联想、戴尔等传统国际厂商

**公司优势**：政府金融客户基础扎实，国产化替代方案完整，产品线覆盖广（服务器+网络+存储）。

## 五、投资亮点

### 核心亮点

- ✅ **政企市场龙头地位坚实**：政府金融等传统行业占60%+，订单稳定
- ✅ **互联网定制化服务器高增长**：2025年增速48%+，成为新增长引擎
- ✅ **国产化替代政策加持**：国产芯片方案完整，市场空间巨大
- ✅ **产品线完整竞争力强**：服务器、网络、存储、服务一体化方案
- ✅ **扣非净利润持续增长**：扣非增速5.15%，盈利质量改善

### 成长驱动因素

1. **互联网定制化服务器需求爆发**：BAT等定制AI服务器需求增加
2. **国产替代加速进行**：政府推进国产化，订单增长机会
3. **数据中心投资增加**：云计算、AI驱动数据中心建设加速

### 催化剂

- 互联网定制化服务器订单加速释放
- 政府国产化采购增加
- AI服务器产品新发布
- 新华三整体上市预期

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **毛利率压力持续** | 互联网定制化服务器竞争激烈，毛利率下降幅度较大 |
| ⚠️ **应收账款风险巨大** | 应收账款占净利润比例高，现金流压力极大 |
| ⚠️ **利润增速不及营收** | 增收不增利，投资回报率下滑 |
| ⚠️ **国产化技术风险** | 国产芯片和系统成熟度不足，技术适配周期长 |
| ⚠️ **市场竞争加剧** | 浪潮等互联网市场竞争压力，定制化服务器市场竞争激化 |
