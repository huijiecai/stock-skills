## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 工业富联（鸿海）集团股份有限公司 |
| 证券代码 | 601138 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2018年02月08日 |
| 上市时间 | 2018年06月08日 |
| 所属行业 | 电子代工与精密制造 |
| 市场地位 | 全球服务器代工龙头 |

**公司概况**：
工业富联是富士康集团在大陆上市的主体，全球服务器代工龙头，2025年前三季度营收6039.31亿元（+38.4%），归母净利润224.87亿元（+48.52%）。公司AI服务器和云计算业务增长强劲，预计2025年全年净利润351-357亿元（+51-54%），是AI时代受益最大的代工企业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **AI服务器代工** | 超大规模AI服务器、高端定制AI服务器代工 | 云计算、大模型训练、AI推理 |
| **通用服务器代工** | 机架式、刀片式、高密度服务器代工 | 企业数据中心、云计算 |
| **通信设备代工** | 高速交换机、路由器、光模块代工 | 数据中心网络、云网融合 |
| **工业互联网** | 工业物联网、边缘计算平台 | 智能制造、工业4.0应用 |

### 业务模式

公司采用代工模式（EMS/ODM），为云服务商（AWS、Google等）、互联网厂商（BAT）和传统IT厂商代工生产。2025年云计算业务占比约40%（营收增速65%+），毛利率30%；通信设备占比35%，毛利率25%。规模优势显著，成本控制业界领先。

### 技术路线/核心技术

- **大规模代工制造**：全球顶级制造产能和工艺技术
- **高端芯片组装**：先进封装、BGA贴装等技术
- **供应链管理**：全球化供应链和成本优化能力

**技术优势**：自动化率85%领先行业，单位制造成本比浪潮等竞争对手低10%，与英伟达、AMD合作代工最新芯片方案。

## 四、行业地位与产业链定位

### 产业链定位

```
服务器代工产业链：
芯片厂商 → 配件采购 → 【工业富联】→ 云厂商/系统商 → 终端应用
(英伟达、AMD)  (全球采购) (代工组装)  (AWS、BAT)   (AI应用)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球服务器代工市场份额第一（约35%），AI服务器代工市场份额20%+ |
| **竞争优势** | 规模最大，成本最低，客户包括全球TOP云服务商，供应链最稳定 |
| **客户结构** | 云服务商占60%+（AWS、Google、Meta等），互联网大厂占25-30% |
| **供应商** | 与英伟达、AMD等芯片商战略合作，芯片分配优先 |

### 竞争格局

**主要竞争对手**：纬创资通（全球服务器代工市场第二）、广达电脑（服务器代工重要参与者）、浪潮等ODM厂商（代工+自有品牌模式）

**公司优势**：规模最大、成本最低、客户最优质，全球化布局（中国、美国、墨西哥等）。

## 五、投资亮点

### 核心亮点

- ✅ **AI服务器代工龙头地位牢固**：2025年AI服务器营收增速超120%，全球TOP云厂商合作
- ✅ **云计算业务高增长**：云服务商服务器订单爆发，营收增速80%+
- ✅ **规模和成本优势显著**：自动化率85%，单位成本比竞争对手低10%
- ✅ **业绩增速加快确定性强**：2025年净利润增速48%+，预计2026-2027年继续高增
- ✅ **芯片分配优先地位**：与英伟达、AMD等深度合作，芯片供应有保障

### 成长驱动因素

1. **AI服务器需求爆炸式增长**：大模型训练、推理推动超大规模AI服务器订单增加
2. **云服务商资本开支高增**：AWS、Google、Meta等云厂商AI投资超预期
3. **高端芯片应用加速**：H100、H200等最新高端芯片需求增加

### 催化剂

- 云厂商AI服务器订单持续超预期
- AI大模型新产品商业化应用
- 高端芯片（H200等）大规模部署
- 800G以上高速交换机市场爆发

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **单一客户集中度风险** | AWS、Google等TOP客户占比过高（可能30-40%），订单波动影响大 |
| ⚠️ **AI周期风险** | AI投资可能出现周期性调整，云厂商资本开支下降直接冲击业绩 |
| ⚠️ **地缘政治风险** | 美国芯片禁令和贸易摩擦可能影响供应链和客户拓展 |
| ⚠️ **应收账款风险** | 大客户账期较长，应收账款占净利润369.48%，现金流压力较大 |
| ⚠️ **代工利润空间受限** | 代工模式毛利率通常较低（6-7%），长期利润增速受限 |
