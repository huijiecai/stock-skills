## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 芯原微电子（上海）股份有限公司 |
| 证券代码 | 688521 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2001年08月 |
| 上市时间 | 2020年08月 |
| 所属行业 | 芯片设计服务、IP授权 |
| 市场地位 | 中国大陆第一的芯片设计服务企业 |

**公司概况**：
芯原股份是中国大陆排名第一的芯片设计服务企业，专注于半导体IP授权和芯片设计服务。公司拥有丰富的IP库，包括GPU、NPU、VPU、DSP等多种处理器IP，是AI芯片设计的重要服务商。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **IP授权** | GPU、NPU、VPU、DSP等处理器IP | 芯片设计、SoC集成 |
| **芯片设计服务** | 一站式芯片设计服务 | 互联网企业、芯片公司 |
| **NPU IP** | 人工智能处理器IP | AI芯片、边缘计算 |
| **GPU IP** | 图形处理器IP | 汽车电子、智能显示 |

### 业务模式

公司采用"IP授权+设计服务"模式，收入来源包括一次性IP授权费和版税收入。客户涵盖芯片设计公司、互联网企业、汽车电子厂商等。NPU IP和GPU IP是增长最快的业务。

### 技术路线/核心技术

- **NPU架构**：自研神经网络处理器架构，支持主流AI框架
- **GPU架构**：自研图形处理器架构，支持OpenGL/Vulkan
- **VPU架构**：视频编解码处理器IP

**技术优势**：中国大陆最大芯片设计服务商；IP库丰富；一站式服务能力；客户覆盖广。

## 四、行业地位与产业链定位

### 产业链定位

```
芯片设计产业链：
IP库 → 设计服务 → 【芯原股份】→ 芯片公司 → 终端应用
(处理器IP) (一站式服务) (设计服务商)  (客户设计)  (AI/汽车)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 中国大陆芯片设计服务市场第一 |
| **竞争优势** | IP库丰富，一站式服务能力强，客户覆盖广 |
| **客户结构** | 芯片设计公司、互联网企业、汽车电子厂商 |
| **供应商** | 晶圆代工、EDA工具等 |

### 竞争格局

**主要竞争对手**：ARM（全球IP龙头）、Imagination（GPU IP领先）、创意电子（台湾设计服务龙头）

**公司优势**：中国大陆第一设计服务商；IP库丰富国产替代；一站式服务能力强；客户资源优质。

## 五、投资亮点

### 核心亮点

- ✅ **中国大陆第一设计服务商**：市场份额领先
- ✅ **IP库丰富**：GPU、NPU、VPU等多种处理器IP
- ✅ **AI芯片设计受益**：NPU IP需求增长快
- ✅ **版税收入提升**：收入结构持续优化
- ✅ **国产替代加速**：IP国产化替代空间大

### 成长驱动因素

1. **AI芯片设计需求**：AI芯片公司增加，设计服务需求增长
2. **IP国产替代**：国产IP替代ARM等国际巨头
3. **汽车电子发展**：汽车芯片需求增长，IP授权增加

### 催化剂

- 大型IP授权项目
- 新客户拓展
- NPU IP放量
- 汽车电子IP突破
