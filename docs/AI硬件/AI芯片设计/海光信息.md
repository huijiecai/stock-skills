## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 海光信息技术股份有限公司 |
| 证券代码 | 688041 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2014年10月 |
| 上市时间 | 2022年08月 |
| 所属行业 | CPU、DCU芯片设计 |
| 市场地位 | 国产CPU龙头，DCU加速器领先 |

**公司概况**：
海光信息是国内领先的处理器芯片设计企业，专注于高端通用处理器（CPU）和协处理器（DCU）的研发。公司是国内少数能够提供高端服务器CPU的企业，产品基于x86架构授权。在AI算力领域，DCU产品是国产AI加速器的重要选择。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **海光CPU** | 海光系列服务器CPU | 服务器、数据中心、政务 |
| **海光DCU** | 深算系列AI加速器 | AI训练、AI推理、HPC |
| **工作站CPU** | 海光桌面CPU | 政务办公、企业计算 |

### 业务模式

公司采用Fabless模式，专注于芯片设计，晶圆制造和封装测试委托给代工厂。收入主要来自CPU和DCU芯片销售。客户涵盖政府机构、电信运营商、金融企业、互联网公司等。

### 技术路线/核心技术

- **CPU架构**：基于x86架构授权，自主优化设计
- **DCU架构**：自研GPGPU架构，支持AI计算
- **先进工艺**：采用先进制程工艺，性能持续提升

**技术优势**：国产高端CPU领先；x86生态兼容性强；DCU产品填补国产空白；专利储备丰富。

## 四、行业地位与产业链定位

### 产业链定位

```
处理器芯片产业链：
架构授权 → 设计 → 代工 → 【海光信息】→ 服务器厂商 → 数据中心
(x86)    (芯片设计) (TSMC等) (CPU/DCU)   (浪潮、联想等)  (AI算力)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国产服务器CPU市场份额第一，国产DCU领先 |
| **竞争优势** | x86生态兼容，性能国产领先，客户资源丰富 |
| **客户结构** | 政府、电信、金融、互联网企业 |
| **供应商** | 晶圆代工依赖TSMC，封装测试国内为主 |

### 竞争格局

**主要竞争对手**：英特尔（全球CPU龙头）、AMD（全球CPU/DCU领先）、华为鲲鹏（国产竞争）

**公司优势**：x86生态兼容性强；国产CPU性能领先；DCU填补国产空白；客户资源优质。

## 五、投资亮点

### 核心亮点

- ✅ **国产CPU龙头**：国产服务器CPU市场份额第一
- ✅ **x86生态兼容**：基于x86授权，软件生态完善
- ✅ **DCU高速增长**：AI加速器收入增长75%
- ✅ **盈利能力优异**：净利率24%，毛利率57%
- ✅ **国产替代核心**：政企市场国产化主力

### 成长驱动因素

1. **AI算力需求爆发**：DCU产品受益AI训练和推理需求
2. **国产替代加速**：CPU国产化率提升空间大
3. **产品迭代升级**：新一代CPU和DCU性能提升

### 催化剂

- 大型服务器CPU订单
- DCU新客户认证
- 新一代产品发布
- 国产替代政策加码
