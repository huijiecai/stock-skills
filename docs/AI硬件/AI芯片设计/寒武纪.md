## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中科寒武纪科技股份有限公司 |
| 证券代码 | 688256 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2016年03月 |
| 上市时间 | 2020年07月 |
| 所属行业 | AI芯片设计 |
| 市场地位 | 国产AI芯片龙头，智能处理器领先 |

**公司概况**：
寒武纪是国内领先的AI芯片设计企业，专注于人工智能核心处理器和芯片的研发。公司是国产AI芯片的标杆企业，产品涵盖云端训练芯片、云端推理芯片、边缘端芯片等。在AI算力自主可控背景下，公司是国产AI芯片的核心力量。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **云端训练芯片** | 思元系列训练芯片（MLU290/MLU370） | 大模型训练、AI集群 |
| **云端推理芯片** | 思元系列推理芯片（MLU220/MLU270） | AI推理服务、云计算 |
| **边缘端芯片** | 寒武纪边缘AI芯片 | 智能安防、自动驾驶 |
| **智能处理器IP** | AI处理器IP授权 | 芯片设计授权 |

### 业务模式

公司采用Fabless模式，专注于芯片设计，晶圆制造和封装测试委托给代工厂。收入来源包括芯片销售和IP授权。客户涵盖互联网企业、政府机构、科研院所等。

### 技术路线/核心技术

- **智能处理器架构**：自主研发MLU架构，针对AI计算优化
- **训练芯片技术**：支持大规模分布式训练，FP16/FP32精度
- **推理芯片技术**：高性能推理，支持INT8/INT4量化

**技术优势**：国内AI芯片技术领先企业；自研智能处理器架构；产品覆盖云端到边缘；专利储备丰富。

## 四、行业地位与产业链定位

### 产业链定位

```
AI芯片产业链：
设计 → 代工 → 【寒武纪】→ 云厂商/企业 → AI应用
(芯片设计) (TSMC等)  (AI芯片)  (互联网、政府)  (大模型)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国产AI云端芯片市场份额前三 |
| **竞争优势** | 自研架构，技术自主可控，国产替代唯一选择 |
| **客户结构** | 互联网企业、政府机构、科研院所 |
| **供应商** | 晶圆代工依赖TSMC，封装测试国内为主 |

### 竞争格局

**主要竞争对手**：英伟达（全球AI芯片龙头）、华为海思（国产竞争）、AMD（国际竞争）

**公司优势**：国产AI芯片技术领先；自研架构自主可控；政策支持力度大；客户资源优质。

## 五、投资亮点

### 核心亮点

- ✅ **国产AI芯片龙头**：云端AI芯片国产替代核心标的
- ✅ **技术自主可控**：自研MLU架构，打破国外垄断
- ✅ **营收高速增长**：2025年前三季度营收增长125%
- ✅ **亏损持续收窄**：盈利拐点可期
- ✅ **政策支持**：AI芯片国产化政策红利

### 成长驱动因素

1. **AI算力需求爆发**：大模型训练和推理需求大幅增长
2. **国产替代加速**：AI芯片国产化率低，替代空间大
3. **产品迭代升级**：新一代训练芯片性能提升

### 催化剂

- 大型AI训练集群订单
- 新一代训练芯片发布
- 政府采购项目落地
- 盈利拐点确认
