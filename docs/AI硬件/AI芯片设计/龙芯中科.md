## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 龙芯中科技术股份有限公司 |
| 证券代码 | 688047 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立日期 | 2010年04月 |
| 上市时间 | 2022年06月 |
| 所属行业 | CPU芯片设计 |
| 市场地位 | 国产自主CPU龙头，完全自主指令集 |

**公司概况**：
龙芯中科是国内领先的自主可控处理器芯片企业，专注于LoongArch（龙架构）指令集处理器研发。公司是国内少数具备完全自主指令集的CPU企业，产品应用于政务、工控、教育等领域，是国产自主可控的核心力量。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **龙芯CPU** | 龙芯3号系列桌面/服务器CPU | 政务办公、服务器、工控 |
| **龙芯MCU** | 龙芯1号/2号系列MCU芯片 | 工业控制、物联网 |
| **基础软件** | 龙芯操作系统、开发工具 | 软件生态建设 |
| **解决方案** | 龙芯整机解决方案 | 政务、教育、企业 |

### 业务模式

公司采用"芯片+软件+解决方案"一体化模式，以芯片销售为主，软件和解决方案为辅。客户涵盖政府机构、军工单位、教育机构、工控企业等。

### 技术路线/核心技术

- **LoongArch指令集**：完全自主知识产权指令集，不依赖国外授权
- **CPU架构**：自研高性能CPU核心，支持多核并行
- **软件生态**：自研操作系统和开发工具，软件生态建设

**技术优势**：完全自主指令集，自主可控程度最高；软件生态持续完善；军工和政务市场认可度高。

## 四、行业地位与产业链定位

### 产业链定位

```
自主CPU产业链：
指令集 → 设计 → 代工 → 【龙芯中科】→ 整机厂商 → 终端用户
(LoongArch) (芯片设计) (国内代工) (CPU)  (整机厂)  (政务/军工)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国产自主CPU市场份额领先 |
| **竞争优势** | 完全自主指令集，自主可控程度最高 |
| **客户结构** | 政府机构、军工单位、教育机构 |
| **供应商** | 晶圆代工国内为主，封装测试国内 |

### 竞争格局

**主要竞争对手**：海光信息（x86授权）、华为鲲鹏（ARM授权）、飞腾（ARM授权）

**公司优势**：完全自主指令集，自主可控；政务和军工市场认可度高；软件生态自主可控。
