## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 长沙景嘉微电子股份有限公司 |
| 证券代码 | 300474 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2006年04月 |
| 上市时间 | 2016年03月 |
| 所属行业 | GPU芯片设计 |
| 市场地位 | 国产GPU龙头，军用GPU领先 |

**公司概况**：
景嘉微是国内领先的GPU芯片设计企业，专注于图形处理器和特种芯片的研发。公司是国内少数能够自主研发GPU的企业，产品应用于军用和民用领域。在AI算力领域，公司布局AI加速GPU，是国产GPU的重要力量。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **GPU芯片** | JM系列图形处理器 | 军用、政务、办公 |
| **特种芯片** | 军用特种集成电路 | 航空航天、军工 |
| **小型雷达系统** | 机载雷达模块 | 军用航空 |
| **AI加速芯片** | 新一代AI GPU（研发中） | AI推理、边缘计算 |

### 业务模式

公司采用"芯片+模块+系统"一体化模式，以芯片和模块销售为主。军品业务占比高，民用业务持续拓展。客户涵盖军工单位、政府机构、企业客户等。

### 技术路线/核心技术

- **GPU架构**：自研GPU架构，支持OpenGL、DirectX等图形标准
- **军用GPU技术**：军用抗辐照、高可靠性GPU技术
- **AI加速技术**：新一代GPU集成AI加速功能

**技术优势**：国产GPU技术领先；军用GPU市场占有率高；自主研发能力强；军工客户认可度高。

## 四、行业地位与产业链定位

### 产业链定位

```
GPU芯片产业链：
设计 → 代工 → 【景嘉微】→ 整机厂商 → 终端用户
(芯片设计) (国内代工) (GPU)  (军工/政务)  (应用)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国产GPU市场份额领先，军用GPU市占率高 |
| **竞争优势** | 国产GPU技术领先，军用市场认可度高 |
| **客户结构** | 军工单位、政府机构、企业客户 |

### 竞争格局

**主要竞争对手**：英伟达（全球GPU龙头）、AMD（国际领先）、海光信息（国产竞争）

**公司优势**：国产GPU技术领先；军用市场占有率高；自主研发能力强；军工客户稳定。
