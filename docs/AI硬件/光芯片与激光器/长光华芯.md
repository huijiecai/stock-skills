## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 苏州长光华芯光电技术股份有限公司 |
| 证券代码 | 688048 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2012年03月 |
| 上市时间 | 2022年04月 |
| 所属行业 | 半导体激光器芯片 |
| 市场地位 | 国内高功率半导体激光器芯片龙头 |

**公司概况**：
长光华芯是国内领先的高功率半导体激光器芯片企业，专注于高功率激光器芯片的研发、设计和制造。公司产品广泛应用于工业加工、光通信、激光雷达、医疗美容等领域。在AI数据中心领域，公司布局泵浦激光器芯片，用于光纤放大器和光模块。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **高功率激光器芯片** | 单管芯片、Bar条芯片 | 工业激光加工、激光泵浦 |
| **光纤激光器泵浦源** | 915nm/976nm泵浦激光器 | 光纤激光器、光通信 |
| **VCSEL芯片** | 垂直腔面发射激光器 | 3D传感、激光雷达、数据中心 |
| **模块产品** | 激光器模块、泵浦模块 | 工业应用、光通信 |

### 业务模式

公司采用IDM模式，具备从芯片设计、外延生长、晶圆制造到器件封装的完整能力。产品以直销为主（60%），经销为辅（40%）。客户涵盖激光器厂商、设备商和终端用户。

### 技术路线/核心技术

- **高功率芯片技术**：自主开发高功率单管和Bar条芯片，功率密度行业领先
- **VCSEL技术**：多结VCSEL芯片，支持高速光通信和3D传感
- **外延生长技术**：MOCVD外延技术，材料质量优异

**技术优势**：国内高功率半导体激光器芯片领先企业；IDM模式技术和成本可控；VCSEL芯片布局领先。

## 四、行业地位与产业链定位

### 产业链定位

```
半导体激光器产业链：
激光器芯片 → 【长光华芯】→ 激光器/模块 → 终端应用
(高功率芯片)  (IDM模式)  (锐科、杰普特)  (工业/通信/3D传感)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内高功率半导体激光器芯片前三 |
| **竞争优势** | IDM模式，技术自主可控，产品线完整 |
| **客户结构** | 锐科激光、杰普特、创鑫激光等激光器厂商 |
| **供应商** | 设备进口为主，原材料部分国产 |

### 竞争格局

**主要竞争对手**：nLight（国际龙头）、Coherent（国际领先）、II-VI（国际大厂）

**公司优势**：高功率芯片国产替代领先；IDM模式成本和品质可控；产品线覆盖高功率和VCSEL；客户资源丰富。

## 五、投资亮点

### 核心亮点

- ✅ **高功率激光器芯片龙头**：国内领先的高功率半导体激光器芯片企业
- ✅ **IDM模式优势**：芯片设计到制造一体化，技术和成本可控
- ✅ **VCSEL布局领先**：多结VCSEL芯片，受益光通信和3D传感
- ✅ **光通信业务增长**：泵浦激光器受益AI数据中心建设
- ✅ **国产替代空间大**：高端激光器芯片国产化率低

### 成长驱动因素

1. **AI数据中心建设**：光放大器泵浦源需求增加
2. **工业激光复苏**：制造业复苏带动激光加工需求
3. **VCSEL应用拓展**：激光雷达、3D传感、数据中心需求增长

### 催化剂

- 大型数据中心泵浦激光器订单
- VCSEL芯片新客户认证
- 工业激光需求复苏
- 激光雷达业务突破
