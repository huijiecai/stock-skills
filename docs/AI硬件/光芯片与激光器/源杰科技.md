## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 陕西源杰半导体科技股份有限公司 |
| 证券代码 | 688498 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2013年01月 |
| 上市时间 | 2022年12月 |
| 所属行业 | 光芯片、激光器芯片 |
| 市场地位 | 国内DFB激光器芯片龙头，高速光芯片领先 |

**公司概况**：
源杰科技是国内领先的光芯片企业，专注于激光器芯片的研发、设计和制造。公司是DFB（分布式反馈）激光器芯片的核心供应商，产品广泛应用于数据中心、电信网络、光纤接入等领域。在AI数据中心高速光模块领域，公司是核心光芯片供应商。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **DFB激光器芯片** | 2.5G/10G/25G/50G DFB芯片 | 数据中心、电信传输、光纤接入 |
| **EML激光器芯片** | 电吸收调制激光器芯片 | 100G/200G/400G光模块 |
| **CW DFB芯片** | 连续波激光器芯片 | 硅光模块、相干通信 |
| **探测器芯片** | PD/APD探测器芯片 | 光模块接收端 |

### 业务模式

公司采用IDM模式，具备从芯片设计、外延生长、晶圆制造到芯片测试的完整能力。产品以直销为主（65%），经销为辅（35%）。客户涵盖国内外光模块厂商，出口业务占比约30%。

### 技术路线/核心技术

- **DFB激光器技术**：自主开发高速DFB芯片，支持25G/50G/100G速率
- **EML芯片技术**：电吸收调制激光器，支持400G/800G光模块
- **外延生长技术**：MOCVD外延生长技术，材料质量行业领先

**技术优势**：国内少数具备IDM能力的激光器芯片企业；高速DFB芯片国产替代领先；外延生长技术自主可控。

## 四、行业地位与产业链定位

### 产业链定位

```
光模块产业链：
光芯片 → 【源杰科技】→ 光模块 → 数据中心
(激光器芯片) (DFB/EML芯片) (中际旭创等) (AI算力)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内DFB激光器芯片前三，高速光芯片国产领先 |
| **竞争优势** | IDM模式，技术自主可控，产品性能优异 |
| **客户结构** | 中际旭创、新易盛、华工科技、光迅科技等 |
| **供应商** | 设备进口为主，原材料部分国产 |

### 竞争格局

**主要竞争对手**：Lumentum（国际龙头）、Finisar（已被收购）、光迅科技（光芯片自研）

**公司优势**：高速DFB/EML芯片国产替代领先；IDM模式成本和品质可控；绑定头部光模块厂商；盈利能力优异。

## 五、投资亮点

### 核心亮点

- ✅ **高速光芯片龙头**：25G/50G/100G DFB/EML芯片国产替代领先
- ✅ **IDM模式优势**：芯片设计到制造一体化，技术和成本可控
- ✅ **AI数据中心核心供应商**：800G光模块光芯片需求爆发
- ✅ **毛利率行业领先**：毛利率47%，盈利能力优异
- ✅ **客户资源优质**：绑定中际旭创、新易盛等头部光模块厂商

### 成长驱动因素

1. **AI数据中心建设**：800G/1.6T光模块需求增加，高速光芯片需求增长
2. **国产替代加速**：高端光芯片国产化率低，替代空间大
3. **技术升级**：向更高速率EML芯片演进，ASP提升

### 催化剂

- 800G/1.6T光模块量产
- 新客户认证突破
- EML芯片放量
- 海外市场拓展
