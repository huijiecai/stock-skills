## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 河南仕佳光子科技股份有限公司 |
| 证券代码 | 688313 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2010年03月 |
| 上市时间 | 2020年08月 |
| 所属行业 | 光通信器件、光芯片 |
| 市场地位 | 国内PLC光分路器龙头，AWG芯片领先 |

**公司概况**：
仕佳光子是国内领先的光通信器件和光芯片企业，专注于PLC光分路器、AWG芯片、数据中心光器件等产品。公司是光通信无源器件的核心供应商，在AWG（阵列波导光栅）芯片领域实现国产替代，是AI数据中心光互联产业链的重要一环。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **PLC光分路器** | 平面光波导分路器芯片及器件 | FTTx、PON网络、光纤接入 |
| **AWG芯片** | 阵列波导光栅芯片、DWDM模块 | 数据中心互联、骨干传输 |
| **数据中心光器件** | MPO连接器、光收发接口组件 | 数据中心、云计算 |
| **有源光器件** | DFB激光器芯片、探测器芯片 | 光模块、光纤通信 |

### 业务模式

公司采用"IDM模式"，具备从芯片设计、晶圆制造到器件封装的完整能力。产品以直销为主（70%），经销为辅（30%）。客户涵盖光模块厂商、通信设备商和数据中心运营商。

### 技术路线/核心技术

- **AWG芯片技术**：自主研发高通道数AWG芯片，支持100G/400G/800G模块
- **PLC芯片技术**：平面光波导芯片设计与制造，良率达行业领先水平
- **硅光技术**：布局硅光子技术平台，支持高速光模块集成

**技术优势**：国内少数具备IDM能力的光芯片企业；AWG芯片国产替代领先；拥有完整的光器件产品线。

## 四、行业地位与产业链定位

### 产业链定位

```
光通信产业链：
光芯片 → 光器件 → 【仕佳光子】→ 光模块 → 数据中心
(AWG/PLC)  (无源/有源) (芯片+器件)  (中际旭创等)  (AI算力)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | PLC光分路器国内前三，AWG芯片国产替代领先 |
| **竞争优势** | IDM模式，芯片自给率高，成本控制能力强 |
| **客户结构** | 中际旭创、新易盛、光迅科技、华为等 |
| **供应商** | 晶圆代工（国内代工厂）、原材料进口 |

### 竞争格局

**主要竞争对手**：光迅科技（光器件龙头）、天孚通信（光器件领先）、博创科技（无源器件）

**公司优势**：AWG芯片国产替代领先；IDM模式成本优势；产品线完整；客户覆盖头部光模块厂商。

## 五、投资亮点

### 核心亮点

- ✅ **AWG芯片国产替代**：高通道数AWG芯片实现国产化，打破国外垄断
- ✅ **IDM模式优势**：芯片设计到器件封装一体化，成本和品质可控
- ✅ **数据中心业务高增长**：AI驱动数据中心光互联需求爆发
- ✅ **硅光技术布局**：布局硅光子平台，面向下一代光模块
- ✅ **客户资源优质**：绑定中际旭创、新易盛等头部光模块厂商

### 成长驱动因素

1. **AI数据中心建设**：800G/1.6T光模块需求增加，AWG芯片需求增长
2. **国产替代加速**：光芯片国产化率提升，替代空间大
3. **硅光技术趋势**：硅光模块渗透率提升，公司技术布局领先

### 催化剂

- 大型数据中心AWG芯片订单
- 800G/1.6T光模块量产
- 硅光产品商用落地
- 新客户认证突破
