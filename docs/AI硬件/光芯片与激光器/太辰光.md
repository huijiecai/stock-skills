## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 深圳太辰光通信股份有限公司 |
| 证券代码 | 300570 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2000年04月 |
| 上市时间 | 2016年12月 |
| 所属行业 | 光通信器件、光纤连接器 |
| 市场地位 | 国内光纤连接器领先企业，MPO连接器龙头 |

**公司概况**：
太辰光是国内领先的光通信器件企业，专注于光纤连接器、光器件和光模块的研发、生产和销售。公司是MPO（多光纤推拉式）连接器的核心供应商，产品广泛应用于数据中心、电信网络、光纤接入等领域。在AI数据中心光互联领域，公司是核心连接器供应商。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **MPO连接器** | 多光纤连接器、预端接光缆 | 数据中心、云计算 |
| **光纤跳线** | 单模/多模光纤跳线、适配器 | 光纤通信、机房布线 |
| **光器件** | 光分路器、波分复用器 | FTTx、PON网络 |
| **光模块** | 低速光模块、AOC有源光缆 | 短距互联、数据中心 |

### 业务模式

公司采用"自主研发+生产+销售"模式，以ODM/OEM为主（60%），自主品牌为辅（40%）。产品以出口为主（55%），国内销售为辅（45%）。客户涵盖国内外光模块厂商、设备商和数据中心运营商。

### 技术路线/核心技术

- **MPO连接器技术**：高密度多光纤连接器，支持64芯/128芯高密度连接
- **预端接光缆技术**：工厂预端接光缆组件，支持快速部署
- **AOC有源光缆**：有源光缆产品，支持短距高速互联

**技术优势**：MPO连接器技术领先，产品覆盖高密度数据中心需求；ODM/OEM能力强；海外市场布局完善。

## 四、行业地位与产业链定位

### 产业链定位

```
数据中心光互联产业链：
光纤/光缆 → 连接器 → 【太辰光】→ 光模块 → 数据中心
(预制棒)   (MPO等)   (MPO组件)  (中际旭创等) (AI算力)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内MPO连接器前三，海外市场拓展顺利 |
| **竞争优势** | MPO技术领先，产品线完整，ODM能力强 |
| **客户结构** | 国内外光模块厂商、数据中心运营商 |
| **供应商** | 光纤来自长飞、亨通等，塑料件国产为主 |

### 竞争格局

**主要竞争对手**：康宁（国际龙头）、TE Connectivity（国际领先）、中航光电（连接器龙头）

**公司优势**：MPO连接器技术领先；ODM/OEM能力强；海外市场布局完善；产品性价比高。

## 五、投资亮点

### 核心亮点

- ✅ **MPO连接器龙头**：高密度多光纤连接器，受益AI数据中心建设
- ✅ **海外市场领先**：出口占比55%，海外客户资源丰富
- ✅ **盈利能力良好**：净利率17%，毛利率33%
- ✅ **数据中心业务高增长**：AI驱动数据中心光互联需求爆发
- ✅ **ODM能力强**：为国内外厂商提供ODM/OEM服务

### 成长驱动因素

1. **AI数据中心建设**：800G/1.6T光模块需求增加，MPO连接器需求增长
2. **海外市场拓展**：北美、欧洲数据中心建设加速
3. **产品升级**：高密度MPO连接器渗透率提升

### 催化剂

- 大型数据中心MPO连接器订单
- 海外大客户突破
- 高密度产品渗透率提升
- 新产品线拓展
