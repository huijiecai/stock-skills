## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广东风华高新科技股份有限公司 |
| 证券代码 | 000636 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1984年 |
| 上市时间 | 1996年11月29日 |
| 所属行业 | 电子元器件 |
| 市场地位 | 国内MLCC（多层陶瓷电容器）龙头 |

**公司概况**：
风华高科是国内领先的被动元件制造商，主营业务为MLCC、片式电阻器、电感器等电子元器件的研发、生产和销售。公司产品广泛应用于消费电子、汽车电子、通信设备、工业控制、AI服务器等领域。

## 二、主营业务

### 核心产品

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **MLCC（多层陶瓷电容器）** | 核心产品，收入占比最大 | 消费电子、汽车、AI服务器 |
| **片式电阻器** | 第二大产品线 | 消费电子、工控 |
| **电感器** | 配套产品 | 电源管理、通信 |
| **压敏/热敏电阻** | 保护器件 | 电源、家电 |
| **铝电解电容器** | 大容量电容 | 电源、工业 |

### 业务模式

公司采用"研发-生产-销售"一体化模式：
- **上游**：采购陶瓷粉体、金属电极材料等原材料
- **中游**：自主生产MLCC、电阻、电感等被动元件
- **下游**：销售给消费电子、汽车电子、通信设备等客户

### 应用领域

| 应用领域 | 占比趋势 | 说明 |
|----------|----------|------|
| 消费电子 | 约50% | 手机、电脑等，占比下降 |
| 汽车电子 | 约20% | 新能源车，占比提升 |
| 工业控制 | 约15% | 稳定 |
| 通信/AI服务器 | 约15% | 增长最快 |

## 五、投资亮点

### 核心亮点

- ✅ **MLCC涨价周期开启**：2025年底开始，行业进入涨价周期
- ✅ **AI算力需求爆发**：AI服务器MLCC用量是普通服务器的3-4倍
- ✅ **国产替代空间大**：高端MLCC国产化率不足10%
- ✅ **行业龙头地位**：国内MLCC产能最大，受益行业复苏
