## 三、验证标准和流程

### 3.1 股票归属验证标准

#### 验证维度

1. **主营业务匹配度** ⭐⭐⭐⭐⭐
   - [ ] 该概念业务营收占比 > 30%
   - [ ] 在年报中作为主营业务披露
   - [ ] 官网/公告明确提及相关业务

2. **行业地位** ⭐⭐⭐⭐⭐
   - [ ] 市占率排名（前3/前5/其他）
   - [ ] 技术专利数量
   - [ ] 行业研报提及频率

3. **产品/技术能力** ⭐⭐⭐⭐
   - [ ] 核心产品技术参数领先
   - [ ] 独家或领先技术
   - [ ] 客户结构（是否覆盖头部客户）

4. **财务指标** ⭐⭐⭐
   - [ ] 毛利率水平（行业对比）
   - [ ] ROE水平
   - [ ] 营收增速

#### 核心标的 vs 相关标的判定标准

**核心标的**（满足以下至少3项）：
- ✅ 该细分领域市占率前3
- ✅ 主营业务营收占比 > 50%
- ✅ 拥有核心技术/专利壁垒
- ✅ 行业研报频繁提及为龙头

**相关标的**：
- 有该业务但不是主营
- 市占率排名靠后
- 转型中或布局中

### 3.2 信息来源优先级

1. **官方信息**（最高权重）
   - [ ] 公司官网 - 产品/业务介绍
   - [ ] 定期报告 - 年报/半年报
   - [ ] 公司公告 - 重大合同/技术突破
   - [ ] 投资者关系 - 互动易问答

2. **行业研究**（高权重）
   - [ ] 券商研报 - 行业地位分析
   - [ ] 行业协会 - 市占率数据
   - [ ] 专业媒体 - 36氪、钛媒体、第一财经

3. **市场数据**（参考）
   - [ ] 东方财富/同花顺 - 概念归属
   - [ ] Wind/Choice - 行业分类
   - [ ] 雪球/股吧 - 市场认知

### 4.3 验证流程

#### 标准工作流（每只股票）

```
1. 信息收集（30分钟）
   ├─ 查看公司官网 → 主营业务
   ├─ 阅读最新年报 → 营收结构
   ├─ 搜索研报 → 行业地位
   └─ 查看公告 → 重大合同/技术

2. 逻辑验证（15分钟）
   ├─ 主营业务是否匹配？
   ├─ 营收占比是否足够？
   ├─ 行业排名如何？
   └─ 技术/产品是否领先？

3. 结论记录（5分钟）
   ├─ 归属概念是否合理？（是/否/待调整）
   ├─ 核心/相关标记是否准确？
   ├─ 信息来源记录
   └─ 置信度评分（1-5星）

总计：约50分钟/只股票
```

#### 持续更新机制

**定期审核**（每季度）：
- [ ] 删除退市/ST股票
- [ ] 补充新上市龙头
- [ ] 复核财报变化（营收结构调整）
- [ ] 跟踪重大技术突破/合同

- [ ] 概念热度变化 → 调整池子
