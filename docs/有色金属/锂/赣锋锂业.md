## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江西赣锋锂业集团股份有限公司 |
| 证券代码 | 002460（A股）/ 01772（H股） |
| 上市板块 | 深圳证券交易所（主板）、香港联交所 |
| 成立时间 | 2000年3月2日 |
| 上市时间 | 2010年8月10日（A股）、2018年10月11日（H股） |
| 所属行业 | 有色金属-锂资源开发与锂化合物深加工 |
| 市场地位 | 锂化工行业龙头，全球领先的锂资源开发和锂化合物深加工企业 |

**公司概况**：
赣锋锂业是全球领先的锂资源开发和锂化合物深加工企业，拥有从盐湖提锂、矿石提锂到退役电池回收的完整锂产业链布局，是唯一建立完整产品链"盐卤/含锂回收物—碳酸锂/氯化锂—锂—丁基锂/锂电池级"的企业。产品广泛应用于电动汽车、储能、3C产品、化学品及制药等领域。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **锂化工产品** | 碳酸锂、氯化锂、氟化锂、丁基锂等20多种锂盐产品 | 动力电池、储能、3C电池、化学品 |
| **锂电池** | 动力电池、消费电池、储能电池、固态锂电池及PACK系统 | 新能源汽车、储能电站、消费电子 |
| **锂电池回收** | 拆解和再生，处理能力20万吨/年，锂回收率超90% | 循环经济、资源再利用 |

### 业务模式

公司采用"资源+加工+电池+回收"一体化模式，通过自有锂矿/盐湖开采、锂化合物深加工、锂电池制造及退役电池回收获取收入。盈利模式以锂盐销售为主，享受锂价上行带来的利润弹性；锂电池及储能业务贡献增量；回收业务提升资源自供率、降低成本。

### 技术路线/核心技术

- **盐湖提锂**：阿根廷Cauchari-Olaroz等盐湖项目工业化技术
- **矿石提锂**：澳大利亚、马里等锂辉石矿开发
- **回收提锂**：退役电池拆解再生，锂回收率超90%
- **固态电池**：固态锂电池研发及产业化布局

**技术优势**：
公司拥有"卤水提锂"、"矿石提锂"和"回收提锂"三大产业化技术，是全球少数实现全技术路线布局的锂业企业。锂资源全球布局覆盖澳大利亚、阿根廷、马里、中国等地。

## 四、行业地位与产业链定位

### 产业链定位

```
锂产业链：
锂矿/盐湖 → 锂化合物冶炼 → 【赣锋锂业】→ 锂电池制造 → 新能源汽车/储能
                    ↑
            锂盐+锂电池+回收
            全产业链布局
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 锂化工行业龙头，2025年自有矿权益产量预计达10万吨LCE，2026年达13万吨；锂盐销量预计全年17-18万吨 |
| **竞争优势** | 全产业链布局、三大提锂技术、全球资源布局、锂电池及储能业务协同 |
| **客户结构** | 覆盖全球主流动力电池、储能及3C客户 |
| **供应商** | 自有矿+盐湖+回收，自供比例55%以上，2025年有望提升至60% |

### 竞争格局

**主要竞争对手**：
- **天齐锂业**：锂矿龙头，格林布什矿山资源优质，2025年自有矿权益产量领先
- **盐湖股份**：盐湖提锂龙头，成本优势明显
- **雅化集团**：锂盐加工企业

**公司优势**：
赣锋锂业为锂化工业务龙头，拥有完整产业链和多元化技术路线。2025-2026年自有矿增速较快，Mariana盐湖、Goulamina项目等投产将显著提升自供率。锂电池及储能业务贡献利润增量，差异化竞争优势明显。

## 五、投资亮点

### 核心亮点

- ✅ **锂盐价格弹性可观**：2025年Q3锂盐含税均价7.5万元/吨以上，从微亏大幅减亏；公司预期锂价中枢维持8-9万元/吨，供需格局大幅改善

- ✅ **资源布局步入收获期**：Mariana盐湖一期2万吨氯化锂2025年2月投产；Goulamina项目2025年产量超30万吨（折4万吨LCE）；Cauchari-Olaroz盐湖投产；2025年自有矿权益产量达10万吨，自供比例60%左右

- ✅ **锂电池业务贡献利润增量**：2025年前三季度锂电池出货20-25GWh，Q3约10GWh，贡献利润1亿以上；全年出货预期35GWh，有望贡献3亿以上利润；储能电站全年交付近10GWh

- ✅ **完整产业链与技术壁垒**：盐湖+矿石+回收三大提锂技术，唯一建立完整产品链的企业；回收处理能力20万吨/年，锂回收率超90%

- ✅ **锂价企稳预期**：机构预计2027年全球锂供应或出现14.4万吨紧缺，锂价有望迎来大幅上涨

### 成长驱动因素

1. **锂价企稳回升**
   - 碳酸锂价格从2024年7万/吨低位企稳，供需格局改善
   - 新能源车、储能需求持续增长

2. **自有矿产能释放**
   - Goulamina、Cauchari-Olaroz、Mariana等项目陆续投产
   - 自供率提升至60%，成本端压力缓解

3. **锂电池及储能放量**
   - 储能领域电网侧独立共享储能、用户侧工商业储能取得进展
   - 锂电池出货量快速增长，贡献利润增量

### 催化剂

- 锂价企稳或进一步上涨
- 新矿山/盐湖项目投产
- 储能订单集中交付
- 固态电池产业化进展

## 七、业绩预期与估值

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 锂价维持8-9万/吨，锂电池放量超预期 | 归母净利8亿+ | 归母净利16亿+ |
| **中性** | 锂价企稳，自有矿顺利投产 | 归母净利8.1亿元 | 归母净利15.8亿元 |
| **悲观** | 锂价再度下跌，产能释放不及预期 | 归母净利<5亿 | 归母净利<10亿 |

*机构预测参考：东吴证券预计2025-2027年归母净利润分别为8.1/15.8/24.5亿元*

### 估值水平

| 指标 | 当前水平 | 行业对比 | 评估 |
|------|----------|----------|------|
| PE(TTM) | 亏损（2024年） | - | 不适用 |
| 2025E PE | 约75倍（东吴证券） | 天齐锂业72倍 | 估值偏高 |
| 2026E PE | 约38倍 | 天齐锂业43倍 | 合理 |
| 2027E PE | 约25倍 | 天齐锂业32倍 | 合理 |

**估值分析**：
当前公司处于业绩修复期，2024年亏损导致PE(TTM)不适用。机构预计2025年扭亏后归母净利约8亿元，对应PE约75倍，估值偏高。2026-2027年随锂价企稳、产能释放，PE有望回落至合理区间。与天齐锂业相比，赣锋锂业锂电池业务贡献增量，成长性更优。
