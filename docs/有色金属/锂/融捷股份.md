## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 融捷股份有限公司 |
| 证券代码 | 002192 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2007年12月5日 |
| 所属行业 | 小金属（锂矿） |
| 市场地位 | 国内锂矿采选龙头之一 |

**公司概况**：
融捷股份是一家以锂矿采选、锂盐加工及冶炼、锂电池设备制造为主营业务的公司。公司核心资产为四川康定甲基卡锂辉石矿134号脉，是国内品位最高、开采条件最好的锂矿之一。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **锂精矿** | 6%品位锂精矿，主要产品 | 锂盐厂原料 |
| **锂盐** | 碳酸锂、氢氧化锂 | 动力电池、储能电池 |
| **锂电池设备** | 锂电池生产设备 | 电池制造商 |

### 业务模式

公司采用"采矿-选矿-锂盐-设备"的一体化布局：
- **上游**：自有锂矿采选，锂精矿对外销售或自用
- **中游**：锂盐加工冶炼
- **下游**：锂电池设备制造

### 核心资产：甲基卡锂辉石矿

| 项目 | 数据 |
|------|------|
| 矿区位置 | 四川康定甲基卡 |
| 矿脉 | 134号脉 |

| 平均品位 | **1.42%**（国内最高） |
| 开采方式 | 露天开采 |
| 采矿产能 | 105万吨/年 |
| 选矿产能 | 45万吨/年 |
| 2025年锂精矿产量 | 18.56万吨（+174.83%） |

## 四、行业地位与产业链定位

### 产业链定位

```
锂电产业链：
上游（锂矿）→ 中游（锂盐）→ 下游（正极材料）→ 终端（电池）
    ↑
 融捷股份
 （核心优势）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **资源禀赋** | 甲基卡锂矿品位1.42%，国内最高 |
| **成本优势** | 露天开采，成本约2-3万元/吨，行业领先 |
| **产能规模** | 105万吨采矿产能，国内前列 |
| **毛利率** | 约38.6%，行业较高水平 |

### 竞争格局

**主要竞争对手**：

| 公司 | 资源类型 | 品位 | 成本 |
|------|----------|------|------|
| **融捷股份** | 锂辉石（甲基卡） | 1.42% | 低 |
| **天齐锂业** | 锂辉石+盐湖 | 1.0-1.5% | 中 |
| **赣锋锂业** | 锂辉石+盐湖+云母 | 0.8-1.2% | 中 |
| **盛新锂能** | 锂辉石+盐湖 | 0.8-1.0% | 中高 |

**公司优势**：
- 资源品位最高，采矿成本最低
- 露天开采，安全性和效率高
- 位于国内，不受海外政策影响

## 五、投资亮点

### 核心亮点

- ✅ **资源禀赋极佳**：甲基卡锂矿品位1.42%，国内最高，露天开采成本低
- ✅ **产能释放加速**：2025年锂精矿产量增长175%，采矿产能满产
- ✅ **成本优势明显**：锂精矿成本约2-3万元/吨，行业领先，在锂价低位仍有盈利
- ✅ **锂价触底反弹**：碳酸锂价格连续18周上涨至15.79万元/吨，行业拐点或已至
- ✅ **产能扩张空间**：选矿产能有望扩产，业绩弹性大

### 成长驱动因素

1. **锂价反弹**
   - 碳酸锂价格从低位反弹，连续18周上涨
   - 2026年供需基本平衡，价格有支撑
   - 若锂价回升至15万元/吨以上，业绩弹性巨大

2. **产能释放**
   - 采矿产能已满产（105万吨/年）
   - 选矿产能有望从45万吨扩至更高
   - 委外加工模式突破选矿瓶颈

3. **行业出清**
   - 高成本产能逐步退出
   - 优质企业市场份额提升
   - 供需格局改善

### 催化剂

- 锂价持续上涨
- 选矿产能扩产落地
- 锂盐加工业务放量
- 新能源车/储能需求超预期
