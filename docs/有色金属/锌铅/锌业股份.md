## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 葫芦岛锌业股份有限公司 |
| 证券代码 | 000751 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1992年8月批准设立，1993年7月15日登记注册 |
| 上市时间 | 1997年6月 |
| 所属行业 | 有色金属-锌铅铜冶炼 |
| 市场地位 | 东北华北地区主要大型重有色金属冶炼企业 |

**公司概况**：
锌业股份是东北和华北地区主要的大型重有色金属冶炼企业，主要从事锌、铜、铅冶炼及深加工产品，同时综合回收镉、铟、金、银、铋等有价金属，副产硫酸、硫酸锌等。公司锌年产量约29万吨，铜年产量约10万吨，2024年锌产量占国内产量4.93%。

*数据来源：公司官网www.hldxygf.com、中财网、证券之星，2024-2025年*

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **锌及锌合金** | 年产量约29万吨，2024年实际27.91万吨 | 镀锌、压铸合金、电池、黄铜 |
| **阴极铜** | 全资子公司葫芦岛宏跃北方铜业，年产量约10万吨，2024年8.57万吨 | 电力、建筑、电子 |
| **精铅** | 2024年产量2.69万吨 | 蓄电池、合金 |
| **硫酸** | 冶炼副产品，2024年产量58.19万吨 | 化工、化肥 |
| **有价金属** | 综合回收镉、铟、金、银、铋等 | 电子、光伏、贵金属 |

### 业务模式

公司采用纯冶炼模式，无自有矿山，主要依靠外购锌精矿、铜精矿进行冶炼加工。盈利来源以冶炼加工费为主，受锌铜市场加工费波动影响较大。2024年有色金属贸易业务同比增长67.93%，业务结构有所调整。

### 技术路线/核心技术

- **锌冶炼**：湿法炼锌工艺，产能约29万吨/年
- **铜冶炼**：火法炼铜，葫芦岛宏跃北方铜业负责
- **综合回收**：镉、铟、金、银、铋等伴生金属回收技术
- **环保冶炼**：硫酸等副产品综合利用

**技术优势**：
公司为老牌冶炼企业，工艺成熟，综合回收能力强。但无矿山资源，成本受原料采购价格和加工费双重影响，在加工费低迷时盈利压力较大。

*数据来源：公司年报、中金在线、证券之星，2024-2025年*

## 四、行业地位与产业链定位

### 产业链定位

```
锌产业链：
锌矿采选 → 锌精矿 → 【锌业股份】→ 锌锭/锌合金 → 镀锌板/电池/合金 → 房地产/汽车/基建
                    ↑
              纯冶炼环节
              无自有矿山
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 锌产量27.91万吨（2024年），占国内产量约4.93% |
| **竞争优势** | 东北华北区域龙头，老牌冶炼企业，综合回收能力强 |
| **客户结构** | 面向镀锌、电池、合金等下游加工企业 |
| **供应商** | 外购锌精矿、铜精矿，无自有矿山，原料依赖外部采购 |

### 竞争格局

**主要竞争对手**：
- **驰宏锌锗**：矿山+冶炼一体化，铅锌锗龙头，央企中铝控股
- **株冶集团**：湖南锌冶炼龙头
- **中金岭南**：铅锌精矿产能全国首位
- **宏达股份**：四川铅锌冶炼
- **葫芦岛有色集团**：同区域冶炼企业

**公司劣势**：
锌业股份为纯冶炼企业，无自有矿山，在加工费低迷时抗风险能力弱于矿山+冶炼一体化企业。2024年锌铜加工费维持低位，铜系统检修影响产能释放，业绩大幅下滑。

## 五、投资亮点

### 核心亮点

- ✅ **收现能力强**：主营业务收现比率平均达113.52%，应收账款回款良好
- ✅ **2025年业绩修复**：前三季度净利润同比+1110.26%，经营现金流转正
- ✅ **贸易业务增长**：有色金属贸易收入+67.93%，业务多元化
- ✅ **成本管控**：2024年管理费用-2.69%，财务费用-9.01%
- ✅ **区域地位**：东北华北地区主要锌铅铜冶炼企业，区位优势

### 成长驱动因素

1. **加工费修复**
   - 锌铜精矿供应紧张有望推动加工费从历史低位回升

2. **贸易业务放量**
   - 有色金属贸易业务快速增长，贡献收入增量

3. **产能利用率提升**
   - 铜系统检修完成后产能释放

### 催化剂

- 锌铜加工费从低位回升
- 锌铜价格上行
- 冶炼行业供给侧优化

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **加工费低迷** | 锌铜市场加工费维持低位，冶炼环节盈利空间收窄，2024年净利率仅0.14% |
| ⚠️ **无自有矿山** | 纯冶炼模式，原料完全外购，成本控制能力弱于一体化企业 |
| ⚠️ **现金流风险** | 2024年经营现金流-5.28亿元，连续三年下滑；存贷双高（货币资金16亿、短期借款20.83亿） |
| ⚠️ **盈利质量** | 净利润与经营活动净现金流持续背离，盈利质量有待改善 |
| ⚠️ **产能检修** | 铜系统装备集中检修影响产能释放，2024年阴极铜收入-21.80% |
| ⚠️ **研发投入缩减** | 2024年研发费用同比-34.57%，长期竞争力需关注 |

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 加工费显著回升、锌铜价上涨 | 净利润恢复至1亿元以上 | 净利润增速30%+ |

| 指标 | 当前水平 | 行业平均 | 评估 |

**估值分析**：
公司2024年净利润仅2128万元，PE估值失真。EV/EBITDA为18.07，处于历史高位（历史均值11.25）。纯冶炼企业业绩对加工费敏感度高，若加工费修复，业绩弹性较大，但当前估值性价比一般。
