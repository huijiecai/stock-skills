## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 深圳市中金岭南有色金属股份有限公司 |
| 证券代码 | 000060 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1984年9月 |
| 上市时间 | 1997年1月23日 |
| 所属行业 | 有色金属-铅锌铜 |
| 市场地位 | 国内铅锌龙头，铅锌精矿产能全国首位 |

**公司概况**：
中金岭南是以铅、锌、铜为主业的多金属国际化全产业链资源公司，主要从事铅锌矿采选、冶炼及深加工。公司拥有国内最大的凡口铅锌矿，铅锌精矿产能居全国首位；拥有国内第三大铅锌冶炼企业韶关冶炼厂。电铅、精锌产品获伦敦金属交易所(LME)注册认证。广东省属国有控股企业，控股股东为广晟资产经营有限公司。位列2024年《财富》中国500强第283位。

*数据来源：公司官网www.nonfemet.com、百度百科、理杏仁，2025年*

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **铅锌精矿** | 2024年精矿铅锌金属量26.5万吨，铜1.2万吨 | 冶炼原料 |
| **铅锭** | 2025年计划14.07万吨 | 电池、合金 |
| **锌锭及锌制品** | 2025年计划31.49万吨 | 镀锌、电池、合金 |
| **阴极铜** | 2025年计划46.98万吨 | 电力、电子 |
| **工业硫酸** | 2024年产量173.77万吨 | 化工 |
| **稀贵金属** | 白银153吨、黄金267公斤、镓17吨、锗18吨 | 电子、光伏、半导体 |
| **铝型材/电池锌粉** | 铝型材1.81万吨、电池锌粉1.97万吨 | 建筑、电池 |

### 业务模式

公司构建"以资源为主体，以三稀金属回收为重点的绿色矿冶和有色金属材料精深加工为两翼"的战略布局。盈利来自矿山采选（铅锌精矿销售）和冶炼加工（铅锭、锌锭、阴极铜等）。聚焦镓、锗、金、银等稀有金属回收，形成多元化产品结构。2024年铅精矿加工费创历史新低（550元/金属吨），冶炼环节利润承压。

### 技术路线/核心技术

- **凡口铅锌矿**：国内最大铅锌矿，高品位资源，采选技术领先
- **韶关冶炼厂**：国内第三大铅锌冶炼企业，LME注册认证
- **三稀金属回收**：镓、锗、金、银等综合回收，绿色矿冶
- **国家级技术中心**：研发实力强

**技术优势**：
公司拥有从矿山到冶炼的完整铅锌铜产业链，凡口矿资源禀赋优异。电铅、精锌获LME注册，产品国际认可度高。三稀金属回收布局符合战略资源发展方向。

*数据来源：公司年报、华创证券调研、上海有色网，2024-2025年*

## 四、行业地位与产业链定位

### 产业链定位

```
铅锌铜产业链：
矿产资源勘探 → 矿产开采采选 → 【中金岭南】→ 冶炼加工 → 下游制造 → 终端应用
                    ↑
              铅锌铜+稀贵金属
              凡口矿+韶关冶炼厂
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 铅锌精矿产能全国首位，凡口铅锌矿国内最大；韶关冶炼厂国内第三大铅锌冶炼企业 |
| **竞争优势** | 资源储备丰富（锌713万吨、铅366万吨、铜143万吨）、LME注册、三稀金属布局 |
| **客户结构** | 面向全球金属贸易商、下游制造商，产品出口 |
| **供应商** | 自有矿山为主，凡口矿为核心资源 |

### 竞争格局

**主要竞争对手**：
- **驰宏锌锗**：铅锌锗一体化，中铝控股
- **西部矿业**：西部铅锌铜龙头
- **紫金矿业**：多金属矿业龙头

**公司优势**：
中金岭南拥有国内最大凡口铅锌矿，资源禀赋优异。铅锌精矿产能全国第一，矿山端利润弹性大。三稀金属（镓、锗、金、银）回收布局符合战略资源方向，具备多元化增长点。

## 五、投资亮点

### 核心亮点

- ✅ **资源储备丰富**：截至2024年底保有锌713万吨、铅366万吨、铜143万吨、银6607吨、金90吨、镍9.24万吨、镓717吨、锗240吨、钨1.65万吨；2024年新增铅锌铜金属资源量151.45万吨

- ✅ **凡口矿+韶关冶炼厂**：国内最大铅锌矿+国内第三大铅锌冶炼企业，产业链一体化

- ✅ **三稀金属布局**：镓、锗、金、银等稀有金属回收，战略资源价值凸显

- ✅ **2025年产量计划明确**：精矿铅锌25.17万吨、铜1.2万吨；冶炼铅锭14.07万、锌锭31.49万、阴极铜46.98万吨

- ✅ **业绩稳健增长**：2024年净利+57.30%，2025年前三季度+5.18%，经营现金流大幅改善

### 成长驱动因素

1. **资源内生外延**
   - 2024年新增资源量151.45万吨，持续夯实资源基础
   - 华创证券调研称公司积极进行资源内生外延

2. **三稀金属价值**
   - 镓、锗等战略金属出口管制背景下，回收业务价值提升

3. **加工费修复**
   - 铅锌精矿供应紧张有望推动加工费从历史低位回升

### 催化剂

- 铅锌加工费从低位回升
- 铜、铅锌价格上行
- 三稀金属业务放量
