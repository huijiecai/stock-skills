## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江西铜业股份有限公司 |
| 证券代码 | 600362（A股）/ 00358（H股） |
| 上市板块 | 上海证券交易所（主板）、香港联交所 |
| 成立时间 | 1997年1月24日 |
| 上市时间 | 1997年6月（H股、伦敦）、2002年1月（A股） |
| 所属行业 | 有色金属-铜冶炼与加工 |
| 市场地位 | 国内铜冶炼产能第一、全球前三铜冶炼生产商 |

**公司概况**：
江西铜业是中国最大的铜金属生产商，铜金属储量居全国首位，阴极铜产量约占全国18.5%，为全球前三的铜冶炼生产商。公司构建了从勘探、采矿、选矿、冶炼到加工的完整产业链，旗下贵溪冶炼厂为全球单体规模最大的铜冶炼厂。江铜集团自2013年以来连续13年上榜《财富》世界500强，2025年位列第165位。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **阴极铜** | 精炼铜，2024年产量229万吨，营收占比约50% | 电力电缆、铜材加工、电子 |
| **铜杆线** | 铜加工产品，营收占比约20% | 电力、建筑、家电 |
| **黄金** | 冶炼副产品及贸易，营收占比约14.5% | 珠宝、投资、电子 |
| **稀散及其他有色金属** | 钼、硒、碲、铼、铂、钯、铋等 | 特种材料、化工、电子 |
| **白银** | 冶炼副产品 | 光伏、电子、珠宝 |
| **铜加工产品** | 铜箔、铜管等 | 新能源、电子、建筑 |
| **化工产品** | 硫酸等 | 化工、化肥 |

### 业务模式

公司采用"采选冶加"一体化模式，以铜冶炼为核心，通过自有矿山（德兴铜矿、永平铜矿等）提供部分铜精矿，同时外购铜精矿进行冶炼加工。盈利主要来自冶炼加工费（TC/RC）、副产品（硫酸、金银）回收及铜加工业务。2025年正从"冶炼加工主导"向"全球资源配置主导"战略转型。

### 技术路线/核心技术

- **闪速炼铜技术**：贵溪冶炼厂闪速炉作业率、铜冶炼总回收率、综合能耗、精矿处理能力、总硫利用率等5项核心指标位居全球第一
- **多金属综合回收**：德兴铜矿年产铜15-16万吨，伴生金5吨，稀贵稀散金属综合回收
- **全球最大单体冶炼产能**：贵溪冶炼厂年产能超100万吨
- **产品认证**："江铜牌""贵冶牌"阴极铜为LME注册产品；黄金、白银为LBMA注册产品

**技术优势**：
贵溪冶炼厂为全球单体规模最大的铜冶炼厂，技术指标行业领先。公司拥有完整的铜产业链技术积累，在冶炼效率、环保、多金属回收方面具有显著优势。

## 四、行业地位与产业链定位

### 产业链定位

```
有色金属产业链：
矿产资源 → 铜精矿 → 【江西铜业】冶炼加工 → 阴极铜/铜材 → 下游制造 → 终端应用
                    ↑
              采选+冶炼+加工一体化
              国内冶炼产能第一
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 铜冶炼产能238万吨全国第一；阴极铜产量占全国约18.5%；全球前三铜冶炼生产商 |
| **竞争优势** | 全球最大单体冶炼厂、技术指标全球第一、采选冶加一体化、多金属综合回收 |
| **客户结构** | 面向电力、建筑、电子、新能源等下游行业，产品获LME、LBMA认证 |
| **供应商** | 自有矿山+外购铜精矿，德兴铜矿年产铜15-16万吨 |

### 竞争格局

**主要竞争对手**：
- **铜陵有色**：铜冶炼产能170万吨，全国第二
- **金川集团**：铜冶炼产能149.5万吨，全国第三
- **紫金矿业**：矿产端龙头，产业链上游
- **云南铜业**：西南地区铜冶炼龙头

**公司优势**：
江西铜业铜冶炼产能位居全国榜首，贵溪冶炼厂5项核心指标全球第一。通过副产品（硫酸、金银）价值回收、提升冶炼回收率应对低加工费压力。2025年收购SolGold获得厄瓜多尔Cascabel铜金矿，向产业链上游资源端转型。

## 五、投资亮点

### 核心亮点

- ✅ **冶炼产能绝对领先**：铜冶炼产能238万吨全国第一，贵溪冶炼厂全球单体最大，技术指标全球第一

- ✅ **战略转型进行时**：从冶炼加工向全球资源配置主导转型，2025年12月完成SolGold收购，获得厄瓜多尔Cascabel矿（1090万吨铜、2300万盎司金）

- ✅ **多金属综合回收**：黄金、白银、稀散金属等副产品增强抗加工费波动能力，2025年计划生产黄金139吨

- ✅ **内部扩产**：武山铜矿三期扩建（1万吨/日）2025年Q3进入试生产；哈萨克斯坦巴库塔钨矿2024年试产

- ✅ **铜价高位受益**：2025年LME铜价触及12000美元/吨历史高位，铜价上涨部分对冲加工费下降

### 成长驱动因素

1. **铜供给结构性紧张**
   - 全球铜精矿供应面临印尼Grasberg、巴拿马Cobre Panama、刚果金Kamoa-Kakula等项目调整
   - 利润向上游资源端转移，公司通过并购获取资源

2. **资源端布局**
   - SolGold收购获得Cascabel矿，计划2028年投产
   - 与全球铜供给缺口扩大时间窗口对齐

3. **副产品价值**
   - 硫酸、金银等副产品价值回收
   - 贵金属对冲加工费下降冲击

### 催化剂

- 铜价维持高位
- SolGold/Cascabel矿开发进展
- 武山铜矿三期、巴库塔钨矿产能释放
- 加工费触底反弹

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **加工费大幅下滑** | 2025年铜精矿长单TC/RC降至21.25美元/吨，同比降73.4%；中国进口现货TC出现负值（-40美元/干吨），创历史新低，冶炼厂面临亏损压力 |
| ⚠️ **产品价格波动** | 铜、金价格受供需和宏观影响，价格下跌将影响业绩；2024年LME铜价波动幅度明显放大 |
| ⚠️ **安全生产风险** | 采矿、冶炼环节存在安全风险 |
| ⚠️ **汇率波动风险** | 进口铜精矿、设备以美元计价，汇率波动影响成本 |
| ⚠️ **环保风险** | 冶炼行业环保要求趋严，环保投入增加 |
| ⚠️ **并购整合风险** | SolGold收购后的整合、Cascabel矿开发存在不确定性 |

*数据来源：江西铜业2024年年报、铜行业快评（加工费），2025年*
