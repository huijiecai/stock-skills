## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 紫金矿业集团股份有限公司 |
| 证券代码 | 601899（A股）/ 2899（H股） |
| 上市板块 | 上海证券交易所（主板）、香港联交所 |
| 成立时间 | 1993年8月（前身为1986年上杭县矿产公司） |
| 上市时间 | 2003年12月（H股）、2008年4月（A股） |
| 所属行业 | 有色金属-贵金属与采矿 |
| 市场地位 | 中国及亚洲有色金属矿业龙头，全球贵金属与采矿第4位 |

**公司概况**：
紫金矿业是中国最大的黄金、铜和锌生产商之一，2024年矿产铜产量107万吨，是中国及亚洲唯一破百万吨大关的铜矿企业；矿产金产量73吨。公司在全球17个国家拥有矿山资产，覆盖亚洲、非洲、欧洲和美洲，在Forbes全球2000强中跻身贵金属与采矿公司第4位。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **矿产铜** | 自有矿山开采的铜精矿，2024年产量107万吨 | 电力电缆、新能源、建筑、电子 |
| **矿产金** | 自有矿山开采的黄金，2024年产量73吨 | 珠宝首饰、金融投资、电子元件 |
| **矿产锌(铅)** | 锌铅精矿，2024年产量45万吨 | 镀锌、电池、合金 |
| **矿产银** | 白银，2024年产量436吨 | 珠宝、光伏、电子 |
| **冶炼产铜** | 铜冶炼加工产品 | 铜材加工、下游制造 |
| **冶炼加工及贸易金** | 黄金冶炼加工及贸易 | 黄金饰品、投资 |
| **锂、钼** | 战略性矿产品 | 新能源电池、特种钢 |

### 业务模式

公司采用"勘探+并购"双轮驱动战略，通过自有矿山开采、冶炼加工及贸易获取收入。盈利模式以矿产金属销售为主，享受金属价格上涨带来的利润弹性；冶炼加工业务作为补充。通过逆周期并购、技改扩产提升产能，精准把握金属价格上行窗口实现尽产尽销。

### 技术路线/核心技术

- **低成本采矿技术**：矿产铜、金成本均处于全球前20%分位，成本管控优异
- **难选矿处理技术**：紫金山金矿等低品位矿山的开发经验
- **海外矿山运营**：非洲员工本地化比例超90%，本地化运营模式成熟
- **资源增储**：2025年上半年新增铜资源量204.9万吨、金资源量88.8吨

**技术优势**：
公司拥有从勘探、采矿、选矿到冶炼的完整技术链条，在低品位矿开发、成本管控方面具有显著优势。2024年研发投入15.82亿元，近5年同期排名第1。

## 四、行业地位与产业链定位

### 产业链定位

```
有色金属产业链：
矿产资源勘探 → 矿产开采采选 → 【紫金矿业】→ 冶炼加工 → 下游制造 → 终端应用
                    ↑
              矿产铜、金、锌、银
              自有矿山+冶炼
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 矿产铜产量中国及亚洲第一（107万吨）、全球前列；矿产金73吨；Forbes全球2000强贵金属与采矿第4位 |
| **竞争优势** | 资源储量丰富、成本管控优异、全球布局、逆周期并购能力 |
| **客户结构** | 面向全球金属贸易商、冶炼厂、下游制造商 |
| **供应商** | 自有矿山为主，资源自给率高 |

### 竞争格局

**主要竞争对手**：
- **洛阳钼业**：多金属矿业，铜钴产量领先
- **西部矿业**：国内铅锌铜龙头
- **江西铜业**：铜冶炼龙头，产业链不同环节
- **中国黄金**：黄金开采为主

**公司优势**：
紫金矿业为矿产端龙头，拥有自有矿山资源，享受金属价格上涨带来的利润弹性；相比冶炼企业，抗加工费波动能力更强。矿产铜、金成本处于全球前20%分位，成本优势明显。

## 五、投资亮点

### 核心亮点

- ✅ **铜金业务量价齐升**：2025年黄金价格涨幅39.2%，铜价上涨2.6%；矿产金、铜产量持续增长

- ✅ **资源增储与产能扩张**：2025年上半年新增铜资源量204.9万吨、金资源量88.8吨；远期规划2028年金产量100-110吨、铜产量150-160万吨

- ✅ **黄金板块价值重估**：紫金黄金国际分拆上市，2025年前三季度实现矿产金32吨、归母净利润64.84亿元；黄金板块估值有望提升

- ✅ **成本管控优异**：矿产铜、金成本均处于全球前20%分位；2024年毛利率20.37%、净利率12.97%

- ✅ **成长性强**：2020-2024年矿产铜产量CAGR为24%，金产量CAGR为12%

### 成长驱动因素

1. **金属价格上行**
   - 降息周期开启有助于黄金、铜等金属价格上涨
   - 2025年黄金价格涨幅领先，铜价维持高位

2. **产能释放**
   - 技改扩产和运营优化快速释放产能
   - 收购加纳阿基姆金矿、哈萨克斯坦Raygorodok金矿等项目

3. **分拆上市**
   - 紫金黄金国际分拆有助于海外业务拓展和资本市场价值重估

### 催化剂

- 黄金价格维持高位或继续上涨
- 紫金黄金国际分拆上市进展
- 新矿山投产、产能释放
- 降息周期深化

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **地缘政治风险** | 塞尔维亚项目受俄乌战争影响；刚果(金)政治稳定性较差；2025年卡莫阿铜矿因地质灾害临时停产 |
| ⚠️ **政策与成本压力** | 刚果(金)拟提高铜权益金从3.5%至5%；阿根廷要求锂矿50%当地加工；2025年上半年经营成本增长15% |
| ⚠️ **环保与社区风险** | 阿根廷3Q盐湖需环保投资；苏里南罗斯贝尔金矿因原住民土地问题延迟投产；哥伦比亚武里蒂卡存在非法开采干扰 |
| ⚠️ **汇率与供应链风险** | 人民币贬值影响进口设备成本；苏伊士运河堵塞导致塞尔维亚设备运输延误；阿根廷提高采矿设备进口关税至15% |
| ⚠️ **金属价格波动** | 铜、金价格受供需和宏观影响，价格下跌将影响业绩 |

*数据来源：紫金矿业海外运营风险分析、公司公告，2025年*
