## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国铝业股份有限公司 |
| 证券代码 | 601600 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2001年9月10日 |
| 上市时间 | 2007年4月30日 |
| 所属行业 | 有色金属-铝冶炼 |
| 市场地位 | 国内铝业龙头、全球铝业第一梯队 |

**公司概况**：
中国铝业是全球最大的氧化铝、电解铝、精细氧化铝、高纯铝生产供应商，也是全球铝行业唯一拥有完整产业链的国际化大型铝业公司。公司氧化铝产能全球第一，电解铝产能全球第一，2024年创造历史最佳业绩，归母净利润124亿元，同比增长85.38%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **冶金级氧化铝** | 电解铝生产的主要原料 | 铝冶炼、铝加工 |
| **原铝（电解铝）** | 铝锭、铝合金等初级铝产品 | 建筑、交通、包装、电力电缆 |
| **精细氧化铝** | 高纯度氧化铝产品 | 陶瓷、催化剂、电子材料 |
| **炭素材料** | 预焙阳极等 | 电解铝生产 |
| **煤炭/电力** | 能源板块自供 | 铝冶炼能源配套 |

### 业务模式

公司采用"铝土矿-氧化铝-电解铝-铝加工"全产业链一体化经营模式，通过规模效应和产业链协同降低成本。氧化铝板块以自产+外销为主，电解铝板块覆盖从原料到成品的完整链条，能源板块为冶炼提供煤炭和电力配套，形成煤电铝联营优势。

### 技术路线/核心技术

- **氧化铝生产技术**：拜耳法工艺，低品位铝土矿利用技术
- **电解铝技术**：大型预焙阳极电解槽，电流效率行业领先
- **清洁能源消纳**：电解铝清洁能源消纳占比行业领先
- **精细氧化铝**：高纯氧化铝、活性氧化铝等特种产品

**技术优势**：
公司产能利用率创历史最高水平，拥有完整的产业链和行业领先的能效控制能力，在氧化铝、电解铝、精细氧化铝、高纯铝、炭素、金属镓等多个细分领域产能全球第一。

## 四、行业地位与产业链定位

### 产业链定位

```
铝产业链：
铝土矿开采 → 氧化铝冶炼 → 【中国铝业】→ 铝加工 → 终端应用
                              ↑
                    氧化铝+电解铝双龙头
                    （电力、建筑、交通、包装、新能源）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 氧化铝全球市占率约20.3%，电解铝全球市占率约15%，国内铝业绝对龙头 |
| **竞争优势** | 全球唯一完整铝产业链企业，氧化铝、电解铝、精细氧化铝、高纯铝、炭素、金属镓产能全球第一 |
| **客户结构** | 覆盖建筑、交通、电力、包装、新能源等下游头部客户 |
| **供应商** | 铝土矿自给+外采，能源板块煤电自供，产业链自主可控 |

### 竞争格局

**主要竞争对手**：
- 中国宏桥：民营铝业龙头，成本控制能力强
- 云铝股份：水电铝龙头，绿色能源优势
- 神火股份：煤铝联营，区域龙头
- 南山铝业：铝加工延伸，高端产品

**公司优势**：
央企背景，规模最大，产业链最完整，氧化铝产能3000万吨/年、电解铝746万吨/年，产能利用率创历史最高，清洁能源消纳占比行业领先。

## 五、投资亮点

### 核心亮点

- ✅ **全球铝业龙头地位**：氧化铝、电解铝、精细氧化铝等六大品类产能全球第一，规模优势无可比拟
- ✅ **全产业链一体化**：铝土矿-氧化铝-电解铝-铝加工完整布局，成本协同效应显著
- ✅ **业绩创历史新高**：2024年归母净利润124亿元（+85.38%），利润总额223亿元创历史纪录
- ✅ **氧化铝盈利爆发**：2024年氧化铝板块税前盈利117亿元，同比增106亿元，量价齐升
- ✅ **分红持续提升**：2024年派息37.23亿元，占净利润30.02%，股息率具备吸引力

### 成长驱动因素

1. **铝价中枢上移**
   - 国内电解铝接近产能天花板（4483万吨），2026年新增产能仅59万吨
   - "铝代铜"需求驱动，铝价中枢21,500-22,000元/吨
   - 新能源、AI算力对铝需求增长

2. **产能利用率提升**
   - 2024年产能利用率创历史最高水平
   - 氧化铝产量1687万吨、电解铝761万吨，同比双增

3. **成本管控优化**
   - 最优成本管控、极致经营
   - 煤电铝联营降低能源成本

### 催化剂

- 铝价持续上涨
- 巴西铝业股权收购推进（2026年1月公告）
- 有色金属行业稳增长政策（工信部等九部门2025-2026年工作方案）

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **原材料成本波动** | 铝土矿、煤炭、预焙阳极等原燃材料价格上升，2024年原铝板块已因成本上升减利23亿元 |
| ⚠️ **铝价下跌风险** | 全球经济复苏不及预期、下游需求萎缩可能导致铝价下跌，影响盈利 |
| ⚠️ **能源板块波动** | 煤炭业务受市场价格影响，2024年能源板块减利11亿元 |
| ⚠️ **产能天花板** | 国内电解铝产能指标受限，扩张空间有限 |
| ⚠️ **环保政策** | 冶炼环节环保成本上升，碳达峰碳中和约束 |
