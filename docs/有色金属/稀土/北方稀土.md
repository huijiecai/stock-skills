## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国北方稀土（集团）高科技股份有限公司 |
| 证券代码 | 600111 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997年9月12日（上市法人实体） |
| 上市时间 | 1997年9月24日 |
| 所属行业 | 有色金属-稀土 |
| 市场地位 | 全球稀土龙头 |

**公司概况**：
北方稀土是全球最大的稀土企业，依托包钢集团白云鄂博矿独家开采权，稀土开采配额占全国总量控制指标的70%左右，冶炼分离指标占67%左右。公司形成稀土原料-功能材料-终端应用的全产业链布局，磁材产能9.5万吨，市占率行业领先，被誉为"中华稀土第一股"。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **稀土原料** | 稀土盐类、稀土氧化物、稀土金属 | 下游稀土功能材料及新材料生产原料 |
| **磁性材料** | 钕铁硼永磁材料 | 新能源汽车电机、风电、工业电机、消费电子 |
| **储氢材料** | 镍氢动力电池材料 | 混合动力汽车、储能 |
| **抛光材料** | 稀土抛光粉 | 显示面板、光学玻璃 |
| **催化材料** | 稀土催化剂 | 石油化工、汽车尾气净化 |

### 业务模式

公司采用"上游资源+中游冶炼+下游深加工"一体化模式。控股股东包钢集团拥有白云鄂博矿独家开采权，为公司提供稳定稀土精矿供应；公司负责冶炼分离和功能材料生产，向下游永磁电机等终端应用延伸，实现全产业链价值最大化。

### 技术路线/核心技术

- **冶炼分离技术**：稀土绿色冶炼升级改造项目一期于2024年10月建成投产，提升原料处理能力和环保水平
- **磁材制备技术**：钕铁硼永磁材料生产技术成熟，2023年销量5.1万吨，市占率行业领先
- **资源综合利用**：白云鄂博矿伴生稀土综合利用，资源禀赋独特

**技术优势**：
依托白云鄂博矿全球最大稀土矿床（保有资源总储量1.59亿吨），公司拥有稳定的原料供应保障。稀土冶炼分离产能17万吨/年，占国家总量控制指标的2/3以上，具备显著的规模壁垒和成本优势。

## 四、行业地位与产业链定位

### 产业链定位

```
稀土矿开采（白云鄂博矿）
    ↓
【稀土冶炼分离】→ 【北方稀土】→ 稀土功能材料 → 终端应用
         ↑
   北方稀土（占全国67%冶炼配额）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 稀土开采配额占全国69.9%，冶炼分离占67%，全球稀土龙头 |
| **竞争优势** | 白云鄂博矿独家开采权、全产业链布局、规模效应显著 |
| **客户结构** | 覆盖永磁材料、新能源、风电、消费电子等下游头部企业 |
| **供应商** | 包钢集团排他性供应稀土精矿，原料保障稳定 |

### 竞争格局

**主要竞争对手**：
- **盛和资源**：六大稀土集团之一，冶炼分离产能扩张中
- **五矿稀土/中国稀土**：央企背景，稀土冶炼分离
- **广晟有色**：南方稀土代表，重稀土资源

**公司优势**：
北方稀土在轻稀土领域占据绝对垄断地位，开采配额占比近70%，远超其他五大集团。依托白云鄂博矿资源禀赋和包钢集团精矿供应，成本优势明显，抗风险能力强。

## 五、投资亮点

### 核心亮点

- ✅ **全球稀土垄断地位**：开采配额占全国69.9%，冶炼分离占67%，战略金属定价权突出
- ✅ **资源禀赋独特**：白云鄂博矿全球最大稀土矿床，包钢集团独家开采权保障原料供应
- ✅ **全产业链布局**：从稀土原料到磁性材料、永磁电机，产业链完整，抗周期能力强
- ✅ **磁材产能领先**：磁材产能9.5万吨，市占率行业领先，受益新能源、风电需求
- ✅ **绿色冶炼升级**：2024年10月绿色冶炼项目一期投产，成本竞争力增强

### 成长驱动因素

1. **稀土战略价值提升**
   - 国家加强行业管控打击黑稀土，供需缺口预计缩小
   - 2025-2026年全球氧化镨钕需求增速约9.7%、8.4%，供需紧平衡

2. **下游需求复苏**
   - 新能源车、风电、工业电机对稀土永磁需求持续增长
   - "设备更新"和"以旧换新"政策带来工业电机需求增量

3. **量价齐升**
   - 2025年氧化镨钕均价同比提升，公司主要产品量价齐升
   - 绿色冶炼项目投产提升产能和成本竞争力

### 催化剂

- 稀土价格底部反转，氧化镨钕价格中枢上移
- 国家稀土配额政策调整
- 新能源、风电装机超预期

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **稀土价格波动** | 稀土产品价格受供需、政策影响大，价格下跌将直接冲击盈利，2024年氧化镨钕均价同比降26%导致净利下滑57.64% |
| ⚠️ **下游需求不及预期** | 新能源车渗透率放缓、风电装机下降、磁材需求增长不及预期将影响销量 |
| ⚠️ **行业供给过快释放** | 海外稀土项目投产、再生稀土产量增加可能打破供需平衡 |
| ⚠️ **应收账款风险** | 应收账款占归母净利润比达340.39%，回款压力需关注 |
| ⚠️ **现金流压力** | 2024年经营活动现金流净额同比降57.76%，货币资金与流动负债比79.56% |

## 七、业绩预期与估值

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 稀土价格持续上涨，需求超预期 | 净利润增速100%+ | 净利润增速50%+ |
| **中性** | 稀土价格企稳回升，供需紧平衡 | 净利润增速50-80% | 净利润增速30-50% |
| **悲观** | 稀土价格继续低迷，需求不及预期 | 净利润增速<30% | 净利润增速<20% |

*参考：国金证券预测2025-2027年归母净利润32、44、53亿元；国信证券预测25.96、34.36、43.00亿元*

### 估值水平

| 指标 | 当前水平 | 行业平均 | 评估 |
|------|----------|----------|------|
| PE(TTM) | 约55-79倍（2025年预测） | 有色金属龙头约15-30倍 | 偏高，反映稀土战略溢价 |
| PB | - | - | - |
| PS | - | - | - |

**估值分析**：
北方稀土估值显著高于一般有色金属企业，反映其稀土战略资源垄断地位和稀缺性溢价。多家券商给予2025年目标价25-57元区间，对应PE 40-59倍。随着业绩复苏，PE有望逐年下降，但需关注稀土价格走势。
