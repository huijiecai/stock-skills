## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广晟有色金属股份有限公司 |
| 证券代码 | 600259 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2009年1月（借壳上市，前身可追溯至1953年广东省冶金厅） |
| 上市时间 | 2000年5月25日（2009年借壳海南兴业聚酯上市） |
| 所属行业 | 有色金属-稀土 |
| 市场地位 | 华南稀土龙头，六大稀土集团之一广东稀土唯一上市平台 |

**公司概况**：
广晟有色是广东稀土集团的唯一上市平台，中国稀土集团旗下"双星"之一。公司拥有广东省内全部3本稀土采矿证，控股新丰左坑矿（全国储量最大的离子型稀土矿山，离子相资源量11.14万吨），建成全国最大的中重稀土分离绿色示范工厂（富远分离厂）。2024年1月，广东稀土集团100%股权无偿划转至中国稀土集团，公司实际控制人由广东省国资委变更为中国稀土集团，央企整合带来资产注入预期。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **稀土精矿** | 离子吸附型稀土矿采选产品 | 冶炼分离原料 |
| **稀土氧化物** | 氧化镨钕、氧化镝、氧化铽等中重稀土氧化物 | 永磁材料、催化、抛光 |
| **稀土化合物** | 混合稀土、稀土盐类 | 功能材料、催化剂 |
| **稀土永磁材料** | 钕铁硼永磁材料（与日本TDK合资） | 新能源汽车电机、风电、工业电机 |
| **有色金属贸易** | 铜、钨等有色金属贸易 | 贸易流通 |

### 业务模式

公司采用"稀土+铜+钨"三大产业横向布局，纵向打造"矿山开采、冶炼分离、精深加工、贸易流通与进出口"完整稀土产业链。盈利模式以稀土矿开采、冶炼分离及深加工产品销售为主，享受稀土价格上涨带来的利润弹性；参股大宝山铜矿（持股40%）贡献稳定投资收益；有色金属贸易业务补充现金流。

### 技术路线/核心技术

- **离子吸附型稀土矿冶炼分离**：掌握国内最先进的离子吸附型稀土矿冶炼分离技术，富远分离厂年处理5,000吨中钇富铕稀土矿，为全国最大的中重稀土分离绿色示范工厂
- **磁材制备技术**：与日本TDK合资建设高新材料企业，具备国际先进的烧结钕铁硼生产技术，晟源公司毛坯产量2025年上半年同比增55%，一次性能直通率达97%以上
- **资源综合利用**：新丰左坑矿为国内最大单体南方离子吸附型稀土矿山，离子相资源量11.14万吨

**技术优势**：
公司拥有广东省全部稀土采矿证，控制中重稀土资源超12万吨。中重稀土分离能力14,000吨/年，冶炼分离配额5,594吨。离子吸附型矿床以重稀土元素富集，战略价值极高，是全球中重稀土最主要来源。

## 四、行业地位与产业链定位

### 产业链定位

```
稀土矿开采（新丰、仁居、五丰等）
    ↓
【稀土冶炼分离】→ 【广晟有色】→ 稀土功能材料 → 终端应用
         ↑                    ↑
   富远分离厂（全国最大中重稀土分离工厂）  磁材（与TDK合资）
   冶炼配额5,594吨
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 六大稀土集团之一，广东稀土唯一上市平台；拥有广东省全部3本稀土采矿证；稀土矿产量占中国稀土集团配额约3%，冶炼品产量占比低于10%，资产注入空间大 |
| **竞争优势** | 中重稀土资源禀赋独特，新丰左坑矿离子相储量11.14万吨；全国最大中重稀土分离工厂；央企整合后资产注入预期 |
| **客户结构** | 覆盖永磁材料、新能源、风电等下游企业；与日本TDK合资拓展高端磁材市场 |
| **供应商** | 自有稀土矿为主，新丰矿投产后原料自给率将显著提升 |

### 竞争格局

**主要竞争对手**：
- **北方稀土**：轻稀土绝对龙头，开采配额占全国近70%
- **盛和资源**：六大稀土集团之一，国内外双布局
- **五矿稀土/中国稀土**：央企背景，稀土冶炼分离
- **厦门钨业**：福建稀土资源，钨稀土双主业

**公司优势**：
广晟有色在**中重稀土**领域具备独特优势。离子吸附型稀土矿主要分布在南方（广东、江西、福建），以重稀土元素（镝、铽等）富集，战略价值高于轻稀土。公司拥有全国储量最大的离子型稀土矿山新丰左坑矿，若左坑矿全面投产，稀土矿产量有望从2,000余吨增至5,000余吨，实现倍增。2024年划入中国稀土集团后，作为集团"双星"上市平台之一，同业竞争解决及资产注入预期强烈。

## 五、投资亮点

### 核心亮点

- ✅ **六大稀土集团唯一广东上市平台**：广东稀土集团唯一上市主体，2024年划入中国稀土集团，央企整合带来资源协同与资产注入预期
- ✅ **中重稀土资源稀缺**：新丰左坑矿为全国储量最大离子型稀土矿山，离子相资源量11.14万吨，相当于原有储量8倍；中重稀土（镝、铽）战略价值高于轻稀土
- ✅ **全国最大中重稀土分离工厂**：富远分离厂年处理5,000吨，中重稀土分离能力14,000吨/年，冶炼配额5,594吨
- ✅ **盈利拐点向上**：2025年稀土价格回升，公司扭亏为盈，氧化镨钕/氧化铽均价预计同比+15%/+30%
- ✅ **多元产业增厚业绩**：大宝山铜矿（持股40%）2025-2027年预计年贡献投资收益1.8-2.0亿元；钨矿、磁材业务协同

### 成长驱动因素

1. **新丰左坑矿产能释放**
   - 左坑矿采矿权已取得，许可期限至2047年
   - 投产后稀土矿产量有望从2,000余吨增至5,000余吨，实现倍增
   - 原料自给率提升，成本优势增强

2. **稀土战略价值提升**
   - 2025年4月商务部对中重稀土实施出口管制，海外镝、铽价格较国内溢价率超200%
   - 工信部发布稀土开采和冶炼分离总量调控管理办法，行业管控趋严
   - 供需紧平衡下，内盘稀土价格弹性空间大

3. **中国稀土集团资产整合**
   - 公司稀土矿产量占集团配额不超3%，冶炼品占比低于10%
   - 中稀集团承诺解决同业竞争，资产注入空间较大
   - 央企平台资源协同、管理赋能

### 催化剂

- 新丰左坑矿投产进度超预期
- 中国稀土集团资产注入方案落地
- 稀土出口管制政策持续，中重稀土价格强势
- 新能源、风电需求超预期

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **稀土价格波动** | 稀土产品价格受供需、政策影响大，2024年镨钕、镝、铽均价同比降26%、21%、37%导致亏损2.99亿元；价格下跌将直接冲击盈利 |

| ⚠️ **资产注入不及预期** | 中国稀土集团资产注入时间、规模存在不确定性，同业竞争解决进度需跟踪 |
| ⚠️ **估值偏高** | 当前PE(TTM)约274倍（扭亏后分母较小），静态PE波动大，需关注业绩兑现与估值匹配度 |

## 七、业绩预期与估值

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 稀土价格持续上涨，新丰矿投产，资产注入落地 | 归母净利润3亿元+ | 归母净利润4-5亿元 |
| **中性** | 稀土价格企稳，产销量稳步提升 | 归母净利润2.5-3亿元 | 归母净利润3.5-4亿元 |
| **悲观** | 稀土价格再度下行，矿山投产延迟 | 归母净利润1.5-2亿元 | 归母净利润2-3亿元 |

*参考：国金证券预测2025-2027年归母净利润2.62/3.84/4.84亿元，对应PE 67/46/36倍*

### 估值水平

| 指标 | 当前水平 | 行业平均 | 评估 |
|------|----------|----------|------|
| PE(TTM) | 约274倍（扭亏后基数小） | 小金属行业平均约70倍 | 偏高，反映中重稀土稀缺性及资产注入预期 |
| PB | 约5.5-6.9倍 | - | 反映资源价值 |
| 预测PE(2025) | 约67倍（国金预测） | 可比公司均值更高 | 显著低于可比公司，国金给予"买入"目标价66.93元 |

**估值分析**：
广晟有色估值反映其中重稀土资源稀缺性、六大稀土集团地位及中国稀土集团资产注入预期。国金证券研报认为公司"低估、高成长"，2025年预测PE 67倍显著低于可比公司均值，首次覆盖给予"买入"评级。随着新丰矿投产及业绩释放，PE有望逐年下降，但需关注稀土价格走势及资产注入进度。
