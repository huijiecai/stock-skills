## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中金黄金股份有限公司 |
| 证券代码 | 600489 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2000年6月 |
| 上市时间 | 2003年8月 |
| 所属行业 | 有色金属-贵金属与采矿 |
| 市场地位 | 黄金行业唯一上市央企，第三梯队潜力企业 |

**公司概况**：
中金黄金是中国黄金集团有限公司（国资委管理）控股的黄金矿业上市公司，是黄金行业唯一的央企平台。公司以黄金、铜采选冶炼为主营业务，拥有金铜双轮驱动格局，资源储量丰富，2024年保有黄金资源量约895吨、铜资源量约228万吨。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **矿产金** | 自有矿山开采的黄金，2024年产量18.35吨 | 珠宝首饰、金融投资、电子元件 |
| **冶炼金** | 黄金冶炼加工产品，2024年产量37.95吨 | 黄金饰品、投资金条 |
| **矿山铜** | 自有矿山铜精矿，2024年产量8.20万吨 | 电力电缆、新能源、建筑 |
| **电解铜** | 铜冶炼产品，2024年产量39.69万吨 | 铜材加工、下游制造 |
| **钼** | 钼精矿，保有资源59.46万吨 | 特种钢、合金 |

### 业务模式

公司采用"采选+冶炼"一体化模式，以自有矿山开采为主、冶炼加工为辅。盈利模式以矿产金属销售为主，享受金属价格上涨带来的利润弹性；冶炼业务贡献加工费收入。通过集团资产注入、新矿建设持续提升矿产金产能。

### 技术路线/核心技术

- **低品位矿开发技术**：具备难选矿处理能力
- **绿色矿山建设**：安全环保标准符合央企要求
- **资源增储**：近5年年均增储黄金约30吨
- **技改扩产**：内蒙古深部资源开采项目一期已投产

**技术优势**：
公司拥有从勘探、采矿、选矿到冶炼的完整技术链条，央企背景在资源获取、政策支持方面具有优势。2024年在建项目累计完成投资10.24亿元。

## 四、行业地位与产业链定位

### 产业链定位

```
有色金属产业链：
矿产资源勘探 → 矿产开采采选 → 【中金黄金】→ 冶炼加工 → 下游制造 → 终端应用
                    ↑
              矿产金、铜+冶炼金、电解铜
              自有矿山+冶炼一体化
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 黄金行业唯一上市央企；矿产金18.35吨，国内黄金企业前列 |
| **竞争优势** | 央企背景、资源储量丰富（金895吨、铜228万吨）、集团资产注入预期 |
| **客户结构** | 面向金属贸易商、冶炼厂、下游制造商 |
| **供应商** | 自有矿山为主，资源自给率较高 |

### 竞争格局

**主要竞争对手**：
- **紫金矿业**：矿产金73吨，全球龙头
- **山东黄金**：国内黄金龙头，PE约41倍
- **赤峰黄金**：民营黄金矿企，海外布局领先
- **招金矿业**：港股黄金企业

**公司优势**：
央企平台唯一性，集团资产注入空间大；金铜双轮驱动分散单一金属风险；相对山东黄金存在估值折价（PE约22倍 vs 41倍），估值修复取决于纱岭投产和集团注入推进。

## 五、投资亮点

### 核心亮点

- ✅ **央企唯一平台**：黄金矿业上市公司中唯一央企，资源获取和政策支持优势明显

- ✅ **金铜双轮驱动**：黄金、铜业务毛利占比分别约41%、39%，分散单一金属价格风险

- ✅ **集团资产注入预期**：纱岭金矿已注入（储量372吨，达产后年产金约10吨）；待注入资产（内蒙古金陶、贵州锦丰、云南黄金、河北大白阳）年产金量合计约7.5吨

- ✅ **纱岭金矿投产在即**：莱州汇金纱岭金矿预计2025年底投产，完全达产后年产量约10吨，为全国十大金矿

- ✅ **分红能力突出**：2024年每10股派现3.88元，股息率持续提升

### 成长驱动因素

1. **金价上行**
   - 2025年黄金价格同比上涨37%以上，降息周期支撑金价
   - 央行购金需求强劲

2. **产能释放**
   - 纱岭金矿2025年底投产
   - 内蒙古深部资源开采项目一期已投产，累计实现收益8,596万元

3. **集团资产注入**
   - 4家金企待注入，预计增厚矿产金权益产量约1.09吨

### 催化剂

- 纱岭金矿正式投产
- 集团资产注入方案落地
- 金价维持高位或继续上涨
- 降息周期深化

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **金价波动风险** | 金价是业绩最主要驱动因素，大幅回调将显著影响利润；预期波动范围3000-3600美元/盎司 |
| ⚠️ **产量不及预期** | 纱岭金矿投产进度存在延期风险；自产矿产量增长受矿山条件制约 |
| ⚠️ **集团注入不确定性** | 资产注入方案落地时间、对价存在不确定性 |
| ⚠️ **成本上升风险** | 安全、环保投入增加，人工及原材料成本上升 |
| ⚠️ **冶炼业务占比高** | 冶炼利润主要为加工费，与金价弹性弱，估值折价有结构性原因 |

*数据来源：中金黄金估值分析、公司公告，2025年*

## 七、业绩预期与估值

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 金价高位维持，纱岭顺利投产 | 净利润增速50%+ | 净利润增速30%+ |
| **中性** | 金价温和上涨，产能按计划释放 | 净利润增速40-50% | 净利润增速20-30% |
| **悲观** | 金价回调，项目延期 | 净利润增速<30% | 净利润增速<15% |

*机构预测参考：2025年归母净利52.76-57.58亿元；2025年PE约13-15.6倍，2026年约10-14.2倍*

### 估值水平

| 指标 | 当前水平 | 行业对比 | 评估 |
|------|----------|----------|------|
| PE(TTM) | 约22倍 | 山东黄金41倍 | 偏低 |
| PB | 约3.8倍 | 山东黄金6.8倍 | 偏低 |
| PS | - | - | - |

**估值分析**：
相对同业龙头山东黄金存在明显估值折价，主要因冶炼业务占比高、矿山端规模较小。估值修复取决于集团注入推进、纱岭金矿投产、金价周期与产量释放的共振。多家机构维持"买入"或"推荐"评级，目标价约16.25元。
