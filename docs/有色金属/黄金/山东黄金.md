## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 山东黄金矿业股份有限公司 |
| 证券代码 | 600547.SH / 1787.HK |
| 上市板块 | 上海证券交易所（主板）、香港联交所 |
| 成立时间 | 2000年 |
| 上市时间 | 2003年8月28日（上交所） |
| 所属行业 | 有色金属-贵金属与采矿 |
| 市场地位 | 国内黄金行业龙头，全球黄金企业前十 |

**公司概况**：
山东黄金是山东省属国有重点骨干企业山东黄金集团控股的黄金矿业上市公司，是国内黄金行业绝对龙头。公司拥有黄金勘探、开采、选冶、精炼和销售于一体的完整产业链，2024年矿产金产量46.17吨、保有黄金资源量超3000吨，境内矿产金产量连续多年保持全国第一，2025年集团荣登世界500强榜单。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **矿产金** | 自有矿山开采的黄金，2024年产量46.17吨 | 珠宝首饰、金融投资、电子元件 |
| **冶炼金/外购金** | 黄金冶炼加工及贸易，标准金锭、投资金条 | 黄金饰品、投资金条 |
| **银锭** | 白银冶炼产品 | 工业、饰品 |
| **黄金贸易** | 黄金贸易业务 | 贸易流通 |

### 业务模式

公司采用"采选+冶炼+贸易"一体化模式，以自有矿山开采为核心、冶炼加工和黄金贸易为辅。盈利模式以矿产金销售为主，享受金价上涨带来的利润弹性；外购金及贸易业务贡献收入规模。通过集团资产注入、矿山技改扩产持续提升矿产金产能。

### 技术路线/核心技术

- **深井采矿技术**：三山岛金矿开采深度超2000米，副井井筒掘进突破-1950米，刷新亚洲最深竖井纪录
- **智能采矿**：打造"采掘支铲装运"一体机械化智能采区，日采矿能力增长1050吨
- **海底采矿**：胶西北地区海底采矿技术
- **深部勘查**：成功探获西岭、新立、东风、南吕—欣木等4个过百吨超大型金矿床

**技术优势**：
公司深部勘查、海底采矿、深井采矿、智能采矿等黄金采选冶关键核心技术处于行业领先水平，在山东胶西北地区建成世界级黄金生产与资源储备基地，形成完整产业链。

## 四、行业地位与产业链定位

### 产业链定位

```
有色金属产业链：
矿产资源勘探 → 矿产开采采选 → 【山东黄金】→ 冶炼加工 → 下游制造 → 终端应用
                    ↑
              矿产金+冶炼金+黄金贸易
              自有矿山+冶炼+贸易一体化
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 境内矿产金产量全国第一，2022年国内占比13.09%；2025年三季度营收、净利润行业居首 |
| **竞争优势** | 资源储量丰富（金超3000吨）、四大百吨金矿、西岭592吨世界级矿床、技术领先 |
| **客户结构** | 面向金属贸易商、冶炼厂、下游制造商、珠宝企业 |
| **供应商** | 自有矿山为主，胶西北世界级黄金基地 |

### 竞争格局

**主要竞争对手**：
- **紫金矿业**：矿产金73吨，全球龙头
- **中金黄金**：黄金行业唯一央企，矿产金18.35吨、冶炼占比高
- **赤峰黄金**：民营黄金矿企，海外布局领先
- **招金矿业**：港股黄金企业

**公司优势**：
国内黄金绝对龙头，营收规模、净利润、矿产金产量均居行业前列；西岭金矿（592吨）矿权注入完成，焦家、三山岛、新城等核心矿山产能持续提升；山金国际矿产金完全成本仅160元/克，显著低于中金黄金310元/克，主要源于西岭等资产品位更高（5.56克/吨）。

## 五、投资亮点

### 核心亮点

- ✅ **国内黄金龙头**：境内矿产金产量连续多年全国第一，营收、净利润行业居首，2025年集团荣登世界500强

- ✅ **资源储量丰厚**：集团保有黄金资源量超3000吨，拥有4座百吨金矿、16座吨金矿山、10座保有资源量超百吨金矿；西岭金矿探明592吨，为世界级全国最大单体金矿床，矿权已注入

- ✅ **核心矿山产能释放**：焦家金矿产能12吨/年、计划提升至15吨/年；三山岛金矿产能15吨/年以上，副井井筒掘进突破-1950米；三山岛、焦家、新城三大核心矿山合计产能占公司总产能60%以上

- ✅ **产能扩张规划**：预计"十五五"末（2030年）总产能从约60吨/年提升至80-85吨/年

- ✅ **金价上行受益**：2024年国际金价年末较上年末上涨约27%，2025年黄金价格维持高位，公司业绩弹性显著

### 成长驱动因素

1. **金价上行**
   - 2024-2025年黄金价格大幅上涨，降息周期支撑金价
   - 央行购金需求强劲

2. **产能释放**
   - 西岭金矿矿权注入完成

3. **生产效率提升**
   - 加大井巷开掘和探矿投入，推进外委转自营和智能化建设
   - 提升安全水平，优化资源利用

### 催化剂

- 三山岛金矿深部工程投产
- 焦家金矿产能提升至15吨/年
- 金价维持高位或继续上涨
- 降息周期深化

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **金价波动风险** | 金价是业绩最主要驱动因素，大幅回调将显著影响利润 |
