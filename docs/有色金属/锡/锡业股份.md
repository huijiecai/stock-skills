## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 云南锡业股份有限公司 |
| 证券代码 | 000960 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年11月23日（前身可追溯至1883年个旧厂务招商局） |
| 上市时间 | 2000年2月 |
| 所属行业 | 有色金属-锡 |
| 市场地位 | 全球锡业龙头，锡产销量全球第一 |

**公司概况**：
锡业股份是全球最大的精锡生产企业，自2005年以来锡产销量稳居全球第一。公司全球锡金属市占率约25%，国内市占率约48%。锡、铟资源储量均位居全球第一，拥有完整的锡全产业链布局，是A股唯一的锡全产业链上市公司。连续多年上榜《财富》中国500强，位列2023云南高新技术企业100强第1位。

*数据来源：公司官网www.ytl.com.cn、新浪财经、百度百科，2025年*

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **锡** | 2024年产量8.48万吨，同比+6%；精锡全球第一 | 焊料、镀锡、化工、电子 |
| **铜** | 2024年产量13.03万吨，同比+1% | 电力、电子、建筑 |
| **锌** | 2024年产量14.4万吨，同比+7% | 镀锌、电池、合金 |
| **铟** | 资源储量全球第一 | 显示面板、半导体、光伏 |
| **钨** | 2025年获钨矿开采配额688吨 | 硬质合金、特种钢 |

### 业务模式

公司具有集锡、铟、锌、铜等有色金属资源探采、选冶、深加工以及新材料研发、贸易等全产业链供应链一体化的产业格局。盈利主要来自锡、铜、锌等金属产品销售，锡业务贡献核心利润。锡价上涨时业绩弹性显著，公司持续推进"内增外拓"资源战略。

### 技术路线/核心技术

- **锡冶炼技术**：全球最大精锡生产基地，冶炼技术成熟
- **地质找矿**：2024年研发费用增长61.92%，地质找矿和冶炼技术研究投入增加
- **多金属综合回收**：锡、铟、锌、铜多金属协同开发
- **资源增储**：获得2025年度钨矿开采配额688吨

**技术优势**：
公司拥有从矿山勘探、采选到冶炼、深加工的完整锡产业链，锡和铟资源储量均居全球第一。2024年位列十大精锡生产商之首，冶炼技术处于行业领先水平。

*数据来源：公司年报、新浪财经、时达科技财务分析，2024-2025年*

## 四、行业地位与产业链定位

### 产业链定位

```
锡产业链：
锡矿勘探开采 → 选矿 → 【锡业股份】→ 精锡/锡材 → 下游应用
                        ↑
                  锡+铜+锌+铟
                  全球锡业龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球锡金属市占率25.03%，国内市占率47.98%，自2005年以来产销量全球第一 |
| **竞争优势** | 锡铟资源储量全球第一、全产业链布局、A股唯一锡全产业链上市公司 |
| **客户结构** | 面向电子、焊料、镀锡、化工等下游，覆盖全球市场 |
| **供应商** | 自有矿山为主，持续推进"内增外拓"资源战略 |

### 竞争格局

**主要竞争对手**：
- **全球**：印尼天马公司、马来西亚冶炼集团等
- **国内**：锡业股份一家独大，国内无直接竞争对手

**公司优势**：
锡业股份为全球锡业绝对龙头，市占率超25%，资源储量、冶炼产能、产业链完整度均领先。锡价上涨时业绩弹性最大，AI电子需求增长有望进一步支撑锡价。

## 五、投资亮点

### 核心亮点

- ✅ **全球锡业龙头**：全球市占率25%、国内48%，自2005年以来锡产销量稳居全球第一，锡和铟资源储量全球第一

- ✅ **锡价慢牛趋势**：2025年Q3锡期货均价26.91万元/吨环比持续上涨；缅甸复产缓慢、印尼RKAB政策变化加剧供应扰动，矿端紧缺支撑锡价

- ✅ **AI电子需求**：AI发展助力电子需求持续向好，锡作为焊料核心材料受益

- ✅ **降本增效显著**：2025Q1-3期间费用占营收比重3.29%同比-1.21pct；资产负债率37.03%同比-3.16pct

- ✅ **资源储备充足**：获得2025年度钨矿开采配额688吨，持续推进"内增外拓"资源战略

### 成长驱动因素

1. **锡价上行**
   - 供应紧张（缅甸、印尼政策）支撑锡价慢牛
   - 矿端紧缺局势加剧

2. **需求增长**
   - AI电子、光伏、新能源汽车等新兴领域锡需求增长

3. **资源价值提升**
   - 钨矿配额、铟等稀贵金属价值重估

### 催化剂

- 锡价维持高位或继续上涨
- 缅甸、印尼供应扰动持续
- AI电子需求超预期

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **锡价波动风险** | 锡、锌、铜价格受供需和宏观影响，价格大幅下跌将显著影响业绩 |
| ⚠️ **供应端变化** | 缅甸复产节奏、印尼RKAB政策变化存在不确定性 |
| ⚠️ **下游需求不及预期** | 电子、焊料等下游需求疲软影响锡消费 |
| ⚠️ **安全环保风险** | 矿山开采、冶炼环节面临环保合规压力 |
| ⚠️ **成本压力** | 2024年Q4成本集中释放导致单季利润下滑 |
