## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江华友钴业股份有限公司 |
| 证券代码 | 603799 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2002年5月22日 |
| 上市时间 | 2015年1月29日 |
| 所属行业 | 有色金属-钴镍锂新材料、新能源锂电材料 |
| 市场地位 | 国内钴产品龙头，镍钴锂铜矿业及新能源材料一体化龙头 |

**公司概况**：
华友钴业是一家从事新能源锂电材料和钴新材料研发、制造的高新技术企业，拥有从钴镍资源开发到锂电材料制造的一体化产业链。公司通过"刚果（金）开采-中国冶炼-全球销售"的闭环体系布局全球供应链，是国内最大的钴产品供应商，钴酸锂市占率25%（国内第二），镍产品及中间品已成为第一大利润来源。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **钴产品** | 金属钴、硫酸钴、四氧化三钴等，2024年出货4.7万吨 | 钴酸锂、三元正极、硬质合金 |
| **镍产品** | 镍中间品、硫酸镍等，2024年出货18.4万吨（含内部自供） | 三元正极前驱体、不锈钢 |
| **锂产品** | 碳酸锂等，2024年约3.9万吨LCE | 动力电池、储能 |
| **正极前驱体** | 三元前驱体，2024年超12万吨 | 动力电池正极材料 |
| **正极材料** | 三元正极、钴酸锂，2024年超9万吨 | 动力电池、消费电池 |

### 业务模式

公司采用"资源+冶炼+材料"一体化经营模式，从非洲刚果（金）钴矿、印尼镍矿开采，到国内冶炼加工，再到正极前驱体、正极材料制造，形成完整产业链。盈利模式以钴镍锂金属及中间品销售为主，享受金属价格上涨带来的利润弹性；正极材料业务贡献稳定收入；产业协同降低综合成本。

### 技术路线/核心技术

- **钴冶炼技术**：国内钴冶炼龙头，硫酸钴市占率17%（2022年国内第一）
- **镍湿法冶炼**：印尼红土镍矿湿法冶炼，Pomalaa等项目
- **正极材料**：三元前驱体、钴酸锂，钴酸锂市占率25%
- **循环回收**：锂电池回收及再生利用

**技术优势**：
公司产销规模和技术开发能力均处于行业前列，具有高效的产业协同、领先的技术创新和显著的行业地位。印尼拥有40.5万吨镍产能，镍产品毛利占比已升至52%。

## 四、行业地位与产业链定位

### 产业链定位

```
新能源锂电材料产业链：
钴镍矿开采 → 冶炼加工 → 【华友钴业】→ 正极前驱体/正极材料 → 动力电池 → 新能源汽车
                    ↑
            钴+镍+锂+前驱体+正极
            一体化布局
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内钴产品龙头，2022年金属钴产销量行业第一；硫酸钴市占率17%国内第一；钴酸锂市占率25%国内第二 |
| **竞争优势** | 一体化产业链、海外资源布局、镍产品已成为第一大利润来源、钴价上行深度受益 |
| **客户结构** | 覆盖全球主流动力电池、消费电池客户 |
| **供应商** | 刚果（金）钴矿、印尼镍矿，海外资源自给 |

### 竞争格局

**主要竞争对手**：
- **寒锐钴业**：钴产品企业
- **腾远钴业**：钴产品企业
- **厦门钨业**：钴酸锂市占率40%，国内第一
- **中伟股份**：正极前驱体龙头

**公司优势**：
华友钴业为钴镍锂一体化龙头，镍产品毛利占比52%超越钴产品，周期成长属性凸显。钴价上行趋势确定（刚果金配额政策），公司深度受益。印尼镍产能40.5万吨，Pomalaa等项目稳步推进，业绩弹性大。

## 五、投资亮点

### 核心亮点

- ✅ **钴价上行深度受益**：钴价由年初17万元/吨涨至41万元/吨，刚果金配额政策影响下钴价上行趋势确定；公司为国内钴产品龙头，业绩弹性大

- ✅ **镍产品成为第一大利润来源**：2024年镍产品毛利占比升至52%，出货18.4万吨同比+46%；印尼拥有40.5万吨镍产能，Pomalaa等项目稳步推进

- ✅ **一体化产业链优势**：从钴镍矿开采到正极材料制造全产业链布局，产业协同降低综合成本，逆周期抗风险能力强

- ✅ **业绩持续高增长**：2025年前三季度归母净利42.16亿元同比+39.59%，2025年Q1同比+140%；机构预测2025年归母净利49.6-60.5亿元

- ✅ **产能释放驱动**：印尼华能5万吨前驱体、韩国正极材料项目等产能释放，Pomalaa镍项目推进，成长性确定

### 成长驱动因素

1. **钴价上行**
   - 刚果金配额政策影响，钴供应收紧
   - 钴价从17万/吨涨至41万/吨，公司深度受益

2. **镍业务周期底部反转**
   - 镍价处于周期底部，全产业链布局优势明显
   - Pomalaa等项目投产，镍产品放量

3. **锂电材料产能释放**
   - 印尼华能5万吨前驱体、韩国正极材料项目
   - 一体化优势持续释放

### 催化剂

- 钴价维持高位或继续上涨
- Pomalaa镍项目投产
- 前驱体、正极材料新产能释放
- 机构普遍给予"买入"评级，目标价78.98-95元

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **金属价格波动风险** | 钴、镍、锂价格受供需和宏观影响，价格下跌将直接影响业绩 |
| ⚠️ **应收账款风险** | 2025年三季度应收账款88.39亿元，同比+45.30%，应收账款/利润比率达212.73%，回款压力需关注 |
| ⚠️ **资产负债率较高** | 资产负债率59.44%，高于行业平均43.99%，有息负债554.01亿元，流动性管理面临挑战 |
| ⚠️ **海外经营风险** | 刚果（金）、印尼项目存在地缘政治、政策变动风险 |
| ⚠️ **毛利率压力** | 毛利率16.18%低于行业平均18.62%，盈利能力偏弱 |
