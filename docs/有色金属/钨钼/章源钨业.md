## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 崇义章源钨业股份有限公司 |
| 证券代码 | 002378 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2000年2月28日 |
| 上市时间 | 2010年3月31日 |
| 所属行业 | 有色金属-钨 |
| 市场地位 | 钨行业骨干企业、产业链完整 |

**公司概况**：
章源钨业是集钨的采选、冶炼、制粉、硬质合金和深加工、贸易为一体的大型钨行业骨干企业，位于江西省赣州市崇义县。公司建立了从上游到下游的完整一体化钨产业链，钨粉产量排名国内行业第一，碳化钨粉产量排名第二，是国内钨行业产业链最完整的厂商之一。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **碳化钨粉** | 钨冶炼核心产品，营收占比约34% | 硬质合金、切削工具、耐磨材料 |
| **钨粉** | 钨粉体产品，营收占比约31% | 硬质合金原料、金属添加剂 |
| **硬质合金** | 高附加值深加工产品，毛利率约17% | 切削刀片、棒材、矿用工具 |
| **仲钨酸铵(APT)** | 钨冶炼中间产品 | 钨粉、氧化钨生产原料 |
| **氧化钨** | 钨化工产品 | 催化剂、电子材料 |

### 业务模式

公司采用"采选-冶炼-制粉-深加工"纵向一体化模式，上游拥有自有钨矿保障部分原料供应，中游钨粉/碳化钨粉产能国内领先，下游通过全资子公司赣州澳克泰生产硬质合金涂层刀片和棒材。主要原料外购占比约80%，产品以内销为主（2024年国内营收占比约81%）。

### 技术路线/核心技术

- **超高性能钨粉体全自动数字化生产线**：提升制粉环节效率与品质
- **硬质合金涂层技术**：子公司赣州澳克泰为国家级专精特新"小巨人"企业
- **表面喷涂服务**：章源喷涂提供热喷涂等表面处理服务

**技术优势**：
在钨冶炼环节形成国内领先的技术优势，钨粉、碳化钨粉产量位居国内前列，硬质合金产品向高附加值方向延伸。

## 四、行业地位与产业链定位

### 产业链定位

```
钨产业链：
上游采选 → 中游冶炼制粉 → 【章源钨业】→ 下游深加工 → 终端应用
                              ↑
                    (采选-冶炼-制粉-硬质合金)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 钨粉产量国内第一，碳化钨粉产量国内第二 |
| **竞争优势** | 产业链完整、冶炼环节技术领先、硬质合金向高附加值延伸 |
| **客户结构** | 以内销为主，国内营收占比约81% |
| **供应商** | 自有矿占比约20%，外购原料占比约80% |

### 竞争格局

**主要竞争对手**：
- **厦门钨业**：钨钼稀土多赛道龙头，产业链更完整，规模更大
- **中钨高新**：五矿集团旗下，硬质合金龙头
- **翔鹭钨业**：钨冶炼及深加工企业
- **江西耀升钨业**：区域性钨企业

**公司优势**：
钨粉/碳化钨粉产能国内领先，产业链完整度较高，硬质合金子公司赣州澳克泰为专精特新"小巨人"，具备一定技术壁垒。

## 五、投资亮点

### 核心亮点

- ✅ **产业链一体化**：从采选到硬质合金完整布局，成本控制与协同效应明显
- ✅ **冶炼环节领先**：钨粉、碳化钨粉产量国内前列，技术优势突出
- ✅ **高附加值产品放量**：硬质合金产品毛利率约17%，高于粉体产品，销量增长带动盈利改善
- ✅ **战略金属属性**：钨为国家战略资源，开采受总量控制，供给刚性约束
- ✅ **税收优惠**：先进制造业企业享受进项税额加计5%抵减

### 成长驱动因素

1. **钨价中枢上移**
   - 供应刚性、需求增长，钨价有望维持高位
   - 2025年出口管制政策强化战略属性定价

2. **产品结构优化**
   - 硬质合金等高附加值产品占比提升
   - 赣州澳克泰刀片、棒材产能释放

3. **制造业复苏**
   - 硬质合金需求与制造业景气度相关
   - 光伏钨丝渗透率提升带来增量需求

### 催化剂

- 钨价持续上涨
- 新产品/新产能投产
- 出口管制政策落地
