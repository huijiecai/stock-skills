## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 厦门钨业股份有限公司 |
| 证券代码 | 600549 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997年12月30日（前身1958年厦门氧化铝厂） |
| 上市时间 | 2002年11月7日 |
| 所属行业 | 有色金属-钨钼、能源新材料、稀土 |
| 市场地位 | 钨钼、正极材料、稀土多赛道龙头 |

**公司概况**：
厦门钨业是国内少有的钨钼、能源新材料、稀土三大主业并进的综合性龙头企业。钨钼业务拥有从矿山到硬质合金、钨丝的完整产业链；能源新材料业务钴酸锂市占率近50%，三元材料销量业界领先；稀土业务子公司金龙稀土为创建世界一流专业领军示范企业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **钨钼产品** | 钨精矿、APT、钨粉、硬质合金、切削工具、细钨丝 | 硬质合金、光伏切割、机械加工 |
| **能源新材料** | 钴酸锂、三元正极材料 | 消费电子、动力电池 |
| **稀土产品** | 钕铁硼永磁材料 | 新能源汽车、风电、消费电子 |
| **房地产业务** | 存量地产 | 非核心业务，逐步收缩 |

### 业务模式

公司采用"钨钼+能源新材料+稀土"三大主业并行模式，各板块独立运营、协同发展。钨钼板块纵向一体化从矿山到深加工；能源新材料依托技术优势服务头部电池客户；稀土板块聚焦永磁材料高端应用。

### 技术路线/核心技术

- **钨丝技术**：高强度钨丝在光伏切割金刚线中渗透率已超50%
- **正极材料**：钴酸锂龙头，三元材料高镍、高电压技术领先
- **稀土永磁**：钕铁硼产能扩张，高端应用占比提升

**技术优势**：
多赛道技术积累深厚，钨丝、钴酸锂、稀土永磁均具备行业领先地位，研发投入持续加大。

## 四、行业地位与产业链定位

### 产业链定位

```
钨钼产业链：矿山 → 冶炼 → 硬质合金/钨丝 → 终端
能源新材料：正极材料 → 动力/消费电池
稀土产业链：稀土分离 → 钕铁硼永磁 → 电机/风电等
                    ↑
              【厦门钨业】三大主业全覆盖
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 钨矿年产量约8000金属吨占全国15%；硬质合金出口占全国30%+；钴酸锂市占率近50%；钨丝光伏渗透率超50% |
| **竞争优势** | 多赛道龙头、产业链完整、技术领先、客户资源优质 |
| **客户结构** | 覆盖动力电池、消费电子、光伏、机械加工等头部客户 |
| **供应商** | 自有四大钨矿山，储量36.53万吨；APT产能3.5万吨、钨粉2.5万吨 |

### 竞争格局

**主要竞争对手**：
- **钨钼**：中钨高新、章源钨业、翔鹭钨业
- **正极材料**：容百科技、当升科技、德方纳米
- **稀土永磁**：金力永磁、中科三环、宁波韵升

**公司优势**：
钨钼、钴酸锂、钨丝细分领域市占率领先，多主业分散单一行业风险，产业链纵向一体化成本优势明显。

## 五、投资亮点

### 核心亮点

- ✅ **多赛道龙头**：钨钼、正极材料、稀土三大主业均居行业前列
- ✅ **钨钼产业链完整**：从矿山到硬质合金、钨丝全链条，自有矿保障原料
- ✅ **钨丝高景气**：光伏切割金刚线钨丝渗透率超50%，细钨丝销量高增长
- ✅ **钴酸锂龙头**：市占率近50%，消费电子需求稳定
- ✅ **稀土业务成长**：金龙稀土为世界一流示范企业，钕铁硼扩产至1.2万吨

### 成长驱动因素

1. **钨钼战略价值提升**
   - 钨出口管制强化战略属性
   - 硬质合金、切削工具需求与制造业复苏相关

2. **能源新材料结构优化**
   - 钴酸锂份额稳固
   - 三元材料高镍、高电压产品放量

3. **稀土永磁需求增长**
   - 新能源汽车、风电驱动钕铁硼需求
   - 产能扩张+高端应用占比提升

### 催化剂

- 钨价、稀土价格上行
- 光伏钨丝需求回暖
- 动力电池需求超预期

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **能源新材料下滑** | 三元材料受动力电池竞争加剧影响，营收利润承压 |
| ⚠️ **细钨丝需求波动** | 光伏行业景气度下行导致钨丝需求减少 |
| ⚠️ **原材料价格波动** | 锂、钴、稀土等价格波动影响正极材料、稀土业务盈利 |
| ⚠️ **多主业管理复杂度** | 三大主业跨度大，协同与资源配置难度较高 |
| ⚠️ **地产拖累** | 房地产业务虽收缩但仍有存量，可能产生减值 |
