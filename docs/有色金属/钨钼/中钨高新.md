## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中钨高新材料股份有限公司 |
| 证券代码 | 000657 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1993年3月18日 |
| 上市时间 | 1996年12月（2000年6月更名为中钨高新） |
| 所属行业 | 有色金属冶炼和压延加工业 |
| 市场地位 | 钨业三大龙头之一、全球第一大硬质合金生产商 |

**公司概况**：
中钨高新是世界500强中国五矿集团下属专业钨产业企业，主营钨精矿、硬质合金及钨钼钽铌等有色金属的研制、生产与销售。公司钨精矿、APT、硬质合金产量稳居全球第一，数控刀片产量国内第一，PCB微钻全球市占率超20%，是国内钨业三大龙头（中钨高新、章源钨业、厦门钨业）之一。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **切削刀具及工具** | 数控刀片、涂层刀片，2023年产量约1.4亿片 | 机械加工、汽车、航空航天 |
| **硬质合金** | 产量超0.75万吨，占国内市场25%以上 | 切削工具、矿用工具、耐磨件 |
| **PCB微钻** | 子公司金洲精工全球市占率第一，最细0.01mm超细钻头 | 印制电路板钻孔 |
| **精矿及粉末产品** | 钨精矿、APT、钨粉、碳化钨粉 | 硬质合金原料、金属添加剂 |
| **难熔金属** | 钨、钼、钽、铌及其深加工产品 | 电子、军工、新能源 |
| **贸易及装备** | 有色金属贸易、装备制造 | 产业链配套 |

### 业务模式

公司采用"钨矿-冶炼-硬质合金-刀具"纵向一体化模式，2024年完成柿竹园100%股权收购后，实现从矿山到深加工的全产业链布局。上游钨矿保障原料供应，中游硬质合金产能国内第一，下游株洲钻石（数控刀片）和金洲精工（PCB微钻）为高毛利核心利润来源，二者营收占比约21%但盈利贡献达75%。

### 技术路线/核心技术

- **硬质合金国家重点实验室**：国内唯一硬质合金领域国家重点实验室
- **数控刀片涂层技术**：株洲钻石为国内数控刀片龙头，刀片毛利率35%+
- **超细微型钻头**：金洲精工拥有业界最细0.01mm PCB微钻技术
- **光伏钨丝**：新建100亿米产能逐步投产，替代高碳钢丝用于光伏切割

**技术优势**：
拥有4个国家级科技创新平台，累计有效专利1401件（含565项发明专利），研发投入保持高位。硬质合金领域CR2/CR8分别达39.05%和58.44%，行业集中度较高。

## 四、行业地位与产业链定位

### 产业链定位

```
钨产业链：
钨矿采选 → 钨冶炼(APT) → 钨粉/碳化钨粉 → 【硬质合金】→ 切削刀具/PCB微钻 → 机械/电子/光伏
                              ↑
                        中钨高新
        （柿竹园矿、远景钨业）  （株洲钻石、金洲精工）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 硬质合金占国内市场25%+，数控刀片约占国内20%，PCB微钻全球市占率超20% |
| **产能规模** | 钨精矿0.42万吨、APT 0.85万吨、硬质合金超0.75万吨，均居全球第一 |
| **竞争优势** | 全产业链布局、五矿集团资源背书、国内唯一硬质合金国家重点实验室 |
| **客户结构** | 覆盖机械加工、汽车、电子、光伏等下游头部客户 |
| **供应商** | 自有钨矿（柿竹园、远景钨业）提升自给率，降低原料波动风险 |

### 竞争格局

**主要竞争对手**：
- **厦门钨业**：钨钼+能源新材料+稀土多赛道龙头，钨丝、钴酸锂、稀土永磁领先
- **章源钨业**：钨粉、碳化钨粉产量国内前列，硬质合金刀片布局
- **翔鹭钨业**：钨冶炼及硬质合金企业

**公司优势**：
中钨高新在硬质合金及数控刀具领域优势最为突出，背靠五矿集团资源保障能力强，柿竹园、远景钨业注入后钨矿自给率显著提升。PCB微钻全球第一、数控刀片国内第一，高毛利产品占比高，盈利质量优于同业。

## 五、投资亮点

### 核心亮点

- ✅ **全产业链龙头**：钨矿-冶炼-硬质合金-刀具一体化，柿竹园+远景钨业注入强化资源端
- ✅ **硬质合金全球第一**：产量占国内25%+，连续多年国内首位，规模优势显著
- ✅ **高毛利产品占比高**：株洲钻石+金洲精工营收占21%但盈利贡献75%，刀具毛利率35%+
- ✅ **PCB微钻全球市占率第一**：金洲精工0.01mm超细钻头技术领先，受益AI算力建设拉动PCB需求
- ✅ **五矿集团背书**：世界500强控股，资源整合与资本运作能力强

### 成长驱动因素

1. **钨价上行周期**
   - 2025年钨价大幅上涨，65%黑钨精矿从170万/吨升至288万/吨
   - 柿竹园并表后直接受益钨价上涨，Q3毛利率环比提升1.3pct

2. **资产注入持续推进**
   - 柿竹园2024年并表贡献超5亿元利润增量
   - 远景钨业收购推进中（8.2亿元，年产钨约2600吨），进一步提升自给率

3. **新业务拓展**
   - 光伏钨丝100亿米产能逐步投产，替代高碳钢丝，预计达产带来约0.59亿元盈利增量
   - AI服务器需求拉动PCB钻针业务增长

### 催化剂

- 远景钨业收购完成并表
- 钨价维持高位或继续上行
- 光伏钨丝产能放量
- PCB/算力需求超预期

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **钨价波动风险** | 钨精矿、APT价格大幅波动影响冶炼及刀具成本，下游价格传导存在滞后 |
| ⚠️ **宏观经济风险** | 机械加工、汽车等下游需求与宏观景气度相关，复苏不及预期影响刀具销量 |
| ⚠️ **竞争加剧风险** | 中高端硬质合金市场竞争加剧，产品溢价能力可能下降 |
| ⚠️ **PCB/算力需求风险** | PCB钻针业务依赖电子、算力服务器需求，工艺进展或需求不及预期影响业绩 |
| ⚠️ **技术研发风险** | 高端硬质合金、超细钻头等技术迭代快，研发投入不足可能落后 |
| ⚠️ **估值风险** | 当前PE约21-24倍，机构评级显示股价偏高，需关注业绩兑现与估值匹配 |

### 业绩预期
