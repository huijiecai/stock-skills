## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广州海格通信集团股份有限公司 |
| 证券代码 | 002465 |
| 上市板块 | 深圳证券交易所（中小板） |
| 所属行业 | 通信设备 |
| 市场地位 | 军用通信和北斗导航领先企业 |

**公司概况**：
海格通信是国内军用通信和北斗导航领先企业，主营无线通信、北斗导航、卫星通信、智能无人系统四大业务。公司是北斗三号芯片主要供应商，天通卫星通信终端主流供应商，实现卫星通信全产业链布局。
