## 一、公司简介

| 项目 | 内容 |
|------|------|

| 证券代码 | 301005 |
| 上市板块 | 深圳证券交易所（创业板） |
| 所属行业 | 紧固件制造 |
| 市场地位 | 可复用火箭结构件供应商 |

**公司概况**：
超捷股份主营紧固件产品，近年来拓展至航天领域，成为可复用火箭结构件供应商。公司为蓝箭航天等民营火箭企业提供箭体结构产品。
