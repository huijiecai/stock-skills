## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 陕西航天动力高科技股份有限公司 |
| 证券代码 | 600343 |
| 上市板块 | 上海证券交易所（主板） |
| 所属行业 | 航天装备 |
| 市场地位 | 国内液体火箭发动机龙头 |

**公司概况**：
航天动力是中国航天科技集团旗下液体火箭发动机研制企业，是国内唯一集运载火箭主动力系统研究与生产的机构。公司主营液体火箭发动机、液力变矩器、特种泵等产品，液体火箭发动机市占率超80%。
