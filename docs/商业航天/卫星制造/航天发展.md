| 项目 | 内容 |

| 证券代码 | 000547 |
| 上市板块 | 深圳证券交易所（主板） |
| 所属行业 | 航天军工 |
| 市场地位 | 卫星星座运营与火箭发射 |

**公司概况**：
航天发展是中国航天科工集团旗下上市公司，主营电子蓝军、卫星应用、雷达探测等业务。公司布局天目一号卫星星座，参与商业火箭发射，是商业航天核心标的。
