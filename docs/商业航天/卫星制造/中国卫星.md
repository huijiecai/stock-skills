## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国东方红卫星股份有限公司 |
| 证券代码 | 600118 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997年8月21日 |
| 上市时间 | 1997年9月8日 |
| 所属行业 | 航天国防 |
| 市场地位 | 国内小卫星研制龙头 |

**公司概况**：
中国卫星是中国航天科技集团旗下唯一的小卫星研制上市公司，是国内唯一覆盖卫星全生命周期的企业。公司主营宇航制造和卫星应用两大业务，具备CAST系列卫星平台研发能力，广泛应用于通信、导航、遥感等领域。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **小卫星/微卫星研制** | CAST系列卫星平台，覆盖50kg-1000kg级卫星 | 通信卫星、导航卫星、遥感卫星、科学卫星 |
| **卫星地面应用系统** | 卫星测控、数据接收、处理与应用系统 | 卫星运营服务、数据服务 |
| **卫星应用服务** | 北斗导航应用、遥感数据服务、卫星通信服务 | 政府、企业、国防 |

### 业务模式

公司采用"研制+运营"一体化模式：
- **宇航制造**：卫星系统研制、AIT（总装集成测试）、在轨交付
- **卫星应用**：卫星运营服务、数据产品销售、系统集成服务

### 技术路线/核心技术

- **CAST系列卫星平台**：CAST100、CAST2000、CAST3000等平台覆盖不同重量级卫星
- **AIT技术**：自主可控的卫星总装集成测试能力
- **通导遥一体化**：卫星通信、导航、遥感融合发展

**技术优势**：
- 国内小卫星研制资质最全的企业
- 完成北斗导航卫星、高分遥感卫星等重大专项
- 具备卫星星座组网设计与运营能力

## 四、行业地位与产业链定位

### 产业链定位

```
商业航天产业链：
上游（材料/元器件）→ 中游（卫星制造）→ 【中国卫星】→ 下游（卫星运营）→ 终端应用
                                              ↑
                                        小卫星研制龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内小卫星市占率超60%，行业排名第一 |
| **竞争优势** | 国内唯一覆盖卫星全生命周期的企业，CAST平台成熟 |
| **客户结构** | 政府、军方、科研院所为主，商业客户占比提升 |
| **供应商** | 航天电子、国博电子等核心配套商 |

### 竞争格局

**主要竞争对手**：
- 长光卫星（商业遥感卫星）
- 银河航天（低轨通信卫星）
- 九天微星（物联网卫星）

**公司优势**：
- 国家队背景，资质最全
- 技术积累深厚，平台成熟
- 深度参与北斗、高分等重大专项

## 五、投资亮点

### 核心亮点

- ✅ **小卫星龙头地位稳固**：国内小卫星市占率超60%，唯一覆盖卫星全生命周期
- ✅ **深度绑定国家重大工程**：北斗导航、高分遥感、星网星座核心供应商
- ✅ **商业航天爆发受益**：低轨卫星星座建设带来订单高增长
- ✅ **技术壁垒高**：CAST平台技术积累深厚，新进入者难以复制
- ✅ **业绩拐点显现**：2025年营收大增85%，净利润扭亏为盈

### 成长驱动因素

1. **低轨卫星互联网建设**
   - 中国星网GW星座计划部署超1.3万颗卫星
   - 公司作为核心供应商将显著受益

2. **北斗三号规模化应用**
   - 北斗三号全球组网完成，应用市场爆发
   - 公司在北斗导航卫星研制领域地位领先

3. **商业遥感市场增长**
   - 高分辨率遥感需求持续增长
   - 公司高分遥感卫星研制能力领先

### 催化剂

- 星网星座订单落地
- 商业航天政策持续加码
- 军用卫星订单增长
