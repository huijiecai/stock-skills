# 中国卫通（601698）投资分析报告

> 更新日期：2026年3月1日

---

## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国卫通集团股份有限公司 |
| 证券代码 | 601698 |
| 上市板块 | 上海证券交易所（主板） |
| 所属行业 | 卫星通信运营 |
| 市场地位 | 国内唯一商用卫星通信运营商 |

**公司概况**：
中国卫通是国内唯一商用卫星通信运营商，运营17颗通信广播卫星，拥有稀缺的轨道资源。公司主营卫星运营服务、网络系统集成，市占率超80%，是卫星互联网核心标的。

---

## 二、主营业务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **卫星运营服务** | 通信卫星运营、广播电视传输 | 媒体、政府、企业 |
| **卫星通信服务** | 卫星互联网、海事通信、应急通信 | 偏远地区、航海、航空 |
| **网络系统集成** | 卫星地面站建设、系统集成 | 政企客户 |

**核心优势**：
- 拥有17颗商用通信卫星
- 国内唯一商用卫星通信运营商
- 稀缺轨道资源，竞争壁垒高
