## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 南京国博电子股份有限公司 |
| 证券代码 | 688375 |
| 上市板块 | 上海证券交易所（科创板） |
| 所属行业 | 半导体/军工电子 |
| 市场地位 | T/R组件国内龙头 |

**公司概况**：
国博电子是中国电科旗下有源相控阵T/R组件和射频集成电路龙头企业，T/R组件市占率超60%，是国内唯一能批量供应军用T/R组件的上市公司。产品广泛应用于雷达、卫星、5G基站等领域。

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **T/R组件** | 有源相控阵雷达核心组件 | 军用雷达、卫星通信 |
| **射频芯片** | GaN射频模块、射频集成电路 | 5G基站、卫星通信 |
| **射频模块** | 多通道射频模块 | 通信、雷达 |

**核心优势**：
- T/R组件市占率超60%
- GaN技术国内领先
- 深度参与商业航天配套
