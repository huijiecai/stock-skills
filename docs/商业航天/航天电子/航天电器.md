## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 贵州航天电器股份有限公司 |
| 证券代码 | 002025 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001年12月30日 |
| 上市时间 | 2004年7月26日 |
| 所属行业 | 元器件（军工电子） |
| 市场地位 | 航天连接器龙头 |

**公司概况**：
航天电器是中国航天科工集团旗下高端电子元器件龙头企业，专注于高端连接器、微特电机、继电器及光电器件等产品的研制与销售，产品广泛应用于航天、航空、兵器、船舶等国防军工领域，以及通信、数据中心等民用领域。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **连接器及互连一体化产品** | 高端连接器、高速背板连接器、液冷互连产品 | 航天航空、通信设备、数据中心（占比70%） |
| **电机与控制组件** | 微特电机、伺服系统、控制组件 | 航天器姿态控制、武器制导系统（占比22%） |
| **继电器** | 军用继电器、密封继电器 | 航天、航空电子设备（占比3.4%） |
| **光通信器件** | CPO光源模块、光电器件 | 数据中心、AI算力基础设施（占比2%） |

### 业务模式

公司采用"军品为主、民品为辅"的双轮驱动模式：
- **军品业务**：依托航天科工集团背景，为国防军工提供核心电子元器件，具有较高壁垒和稳定订单
- **民品业务**：拓展通信、数据中心等民用市场，布局CPO光模块、液冷散热等AI算力基础设施领域

### 技术路线/核心技术

- **宇航级连接器技术**：具备抗辐射、耐高低温、高可靠性等特性，满足航天级应用需求
- **CPO光模块技术**：子公司研发的CPO光源模块产品已给部分客户送样，提供高功率高效率方案
- **液冷互连技术**：研制生产的液冷互连产品主要应用于通信领域，2025年订货占比约2%
- **高速背板连接器**：支持800G硅光模块配套AI数据中心

**技术优势**：
公司拥有多项核心技术专利，在宇航级连接器领域具备领先优势。CPO光源模块和液冷互连产品是公司切入AI算力基础设施的重要技术储备。

## 四、行业地位与产业链定位

### 产业链定位

```
军工电子产业链：
原材料 → 零部件 → 【电子元器件】 → 系统集成 → 终端装备
                         ↑
                    航天电器
```

```
AI算力产业链：
芯片 → 服务器 → 【互连/光模块/液冷】 → 数据中心 → AI应用
                    ↑
               航天电器（新兴布局）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 特种连接器市场主要厂商之一，与中航光电为行业双龙头 |
| **竞争优势** | 航天科工集团背景，宇航级连接器技术领先，军品壁垒高 |
| **客户结构** | 航天科工集团、中航工业等军工集团，以及通信设备厂商 |
| **供应商** | 金属材料、塑料、电子元器件等上游供应商 |

### 竞争格局

**主要竞争对手**：
- **中航光电（002179）**：特种连接器市场份额最大，军工连接器龙头
- **永贵电器（300351）**：轨道交通连接器领先企业
- **陕西华达**：军用连接器专业厂商

**公司优势**：
- 航天科工集团旗下，航天领域连接器具有独特优势
- 宇航级产品技术壁垒高，竞争对手难以替代
- 积极布局CPO、液冷等AI算力基础设施新赛道

## 五、投资亮点

### 核心亮点

- ✅ **军工电子龙头**：航天科工集团旗下核心电子元器件平台，军品业务稳定，壁垒高
- ✅ **商业航天受益标的**：随着中国商业航天加速发展，卫星制造、火箭发射等对连接器需求持续增长
- ⚠️ **AI算力基础设施布局**：CPO光模块进入华为批产应用，液冷互连产品切入数据中心赛道，但业务占比小（约2%）
- ✅ **技术储备丰富**：宇航级连接器、CPO光源模块、液冷互连等技术储备，具备技术迁移能力
- ✅ **订单稳步增长**：战新产业订货同比增长超30%，数据中心订单占比稳步提升

### 成长驱动因素

1. **商业航天加速发展**
   - 中国商业航天进入快速发展期，卫星互联网、火箭发射等带动连接器需求

2. **AI算力基础设施扩张**
   - 数据中心高速扩张，高速光模块、液冷散热等需求爆发
   - 公司CPO光源模块、液冷互连产品已送样，有望切入供应链

3. **军工订单稳定增长**
   - 国防现代化持续推进，军工电子需求稳定

### 催化剂

- CPO光模块在华为等国产算力客户处批量出货
- 液冷互连产品在数据中心领域订单放量
- 商业航天重大发射任务带动市场关注度

**重要提示**：
- ❌ 英伟达认证为假消息（2026年3月6日公司官方辟谣）
- ✅ CPO产品已进入华为等头部企业批产应用
- ⚠️ AI业务目前占比约2%，不要过度期待
