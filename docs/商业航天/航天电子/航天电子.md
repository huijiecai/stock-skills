## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 航天时代电子技术股份有限公司 |
| 证券代码 | 600879 |
| 上市板块 | 上海证券交易所（主板） |
| 所属行业 | 航天电子 |
| 市场地位 | 国内航天电子设备龙头 |

**公司概况**：
航天电子是中国航天科技集团旗下高科技上市公司，专注于航天电子测控、惯性导航、制导系统及元器件。公司在航天电子领域市占率超80%，是卫星、火箭核心配套商，深度参与北斗、载人航天、探月等重大工程。
