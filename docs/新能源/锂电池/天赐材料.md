## 二、主营业务

### 1. 核心业务

天赐材料是全球锂电池电解液龙头企业：

- **电解液**：锂电池电解液，全球份额第一
- **六氟磷酸锂**：电解液核心原材料，自供
- **正极材料**：磷酸铁锂正极
- **日化材料**：表面活性剂等

### 2. 业务结构

| 业务板块 | 营收占比 | 说明 |
|----------|----------|------|
| 电解液 | ~70% | 全球份额超30% |
| 正极材料 | ~15% | 磷酸铁锂 |
| 日化材料 | ~10% | 稳定贡献利润 |
| 其他 | ~5% | - |

### 3. 技术路线

- 液态电解液：主流产品
- 固态电解质：固态电池配套
- 六氟磷酸锂：核心原材料自供

## 四、行业地位

### 产业链定位

```text
锂电池产业链：
上游材料（六氟磷酸锂）→ 电解液（天赐材料）→ 电池制造
```

### 市场地位

- **电解液**：全球份额超30%，龙头地位稳固
- **六氟磷酸锂**：自供比例高，成本优势明显
- **客户覆盖**：宁德时代、比亚迪、LG等全球主流电池厂

### 核心客户

- **国内客户**：宁德时代、比亚迪、国轩高科、亿纬锂能
- **海外客户**：LG新能源、三星SDI、松下
