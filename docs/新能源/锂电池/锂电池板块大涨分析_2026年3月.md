## 二、核心驱动因素

### 2.1 主驱动：六氟磷酸锂价格暴涨 🔥🔥🔥

**价格走势**：

| 时间 | 价格（万元/吨） | 涨幅 |
|------|----------------|------|
| 2025年7月 | 4.93 | - |
| 2025年12月 | 18.0 | +265% |
| 2026年3月 | 17-19 | 维持高位 |

**为什么暴涨？**

```
供给端：
  • 中小产能因价格战亏损而退出
  • 四部委联合整治非理性竞争，中小企业加速出清
  • 行业CR5集中度从60% → 80%

需求端：
  • 新能源车产销量持续增长（2026年预期+30%）
  • 储能装机爆发（2026年预期+40%）
  • 全球锂电池需求增速35-45%

结果：
  2026年全球六氟磷酸锂缺口3-8万吨
  → 供需紧张 → 价格暴涨
```

**产业链传导**：

```
六氟磷酸锂价格↑ 
  ↓
电解液成本↑ 但利润率↑（龙头定价权强）
  ↓
天赐材料等电解液龙头利润暴增
  ↓
股价暴涨
```

### 2.2 政策催化：储能首次写入政府工作报告 🔥🔥

**2026年政府工作报告**：

```
✅ "发展新型储能"正式写入政府工作报告
✅ 构建新型电力系统，加快智能电网建设
✅ 扩大绿电应用
```

**具体政策**：

1. **储能补贴细则落地**
   - 钠离子电池、液流电池等新型储能：30%建设成本补贴
   - 强制要求储能时长≥4小时
   - 精准契合储能技术迭代与电网调峰需求

2. **锂电原材料零关税延续**
   - 锂辉石、氯化锂、碳酸锂：暂定税率0%
   - 锂离子电池用再生黑粉：3%暂定税率
   - 大幅降低进口成本

3. **锂亚电池出口便利化**
   - 通关效率提升80%+
   - 出口成本降低约2%

**影响**：
```
储能政策支持 
  ↓
储能电池需求爆发
  ↓
宁德时代、亿纬锂能等储能电池龙头受益
  ↓
股价上涨
```
