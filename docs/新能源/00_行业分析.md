## 一、行业定义与范围

### 1. 是什么？

> **新能源 = 以清洁能源为核心的能源体系，涵盖发电、储能、应用全产业链**

新能源行业是能源转型的核心产业，涵盖从能源生产（光伏、风电、氢能）到能源存储（锂电池、储能系统）再到能源应用（新能源汽车）的完整产业链。

| 细分领域 | 说明 | 市场规模占比 |
|----------|------|--------------|
| **锂电池** | 动力电池+储能电池 | ~35% |
| **光伏** | 硅料、硅片、电池片、组件 | ~25% |
| **新能源车** | 整车制造+零部件 | ~20% |
| **风电** | 整机+零部件+运营 | ~8% |
| **储能** | 储能系统+逆变器 | ~7% |
| **锂矿** | 锂资源开发 | ~3% |
| **氢能/充电桩等** | 配套产业 | ~2% |

### 2. 产业链定位

```text
上游：资源与材料
    ├─ 锂矿：天齐锂业、赣锋锂业、盐湖股份
    ├─ 硅料：通威股份、大全能源
    └─ 稀土：北方稀土、盛和资源
    ↓
中游：制造与设备
    ├─ 锂电池：宁德时代、比亚迪、亿纬锂能
    ├─ 光伏组件：隆基绿能、晶澳科技、天合光能
    ├─ 风机整机：金风科技、明阳智能
    └─ 储能系统：阳光电源、上能电气
    ↓
下游：应用与服务
    ├─ 新能源车：比亚迪、理想、蔚来产业链
    ├─ 充电桩：特锐德、盛弘股份
    └─ 电站运营：三峡能源、龙源电力
```

**说明**：
- 新能源是全球能源转型的核心产业
- 中国在光伏、锂电池、风电等领域全球领先
- 产业链完整，国产化率高
- 政策驱动与市场驱动双重动力

## 二、市场规模与增长趋势

### 1. 全球市场

```text
全球新能源市场规模：
- 2024年：约 2.5万亿美元
- 2025年：约 3.0万亿美元（+20%）
- 2026年：预计 3.5万亿美元（+17%）
- 2030年：预计突破5万亿美元

细分市场：
- 新能源车：约1.5万亿美元
- 光伏：约0.8万亿美元
- 储能：约0.5万亿美元
- 风电：约0.4万亿美元
```

**结构性特征**：

- **新能源车爆发**：2026年全球销量预计2600万辆，同比增长15%
- **储能高速增长**：全球储能装机量持续高增长
- **光伏周期调整**：产能过剩，价格触底
- **风电稳步增长**：海上风电成为新增长点

### 2. 中国市场

- **市场规模**
  - 2025年：约 15万亿元人民币
  - 占全球市场份额约50%
  - 是全球最大的新能源市场

- **市场地位**
  - 光伏：全球份额超80%
  - 锂电池：全球份额超70%
  - 新能源车：全球份额超60%
  - 风电：全球份额超50%

### 3. 增长驱动因素

- **政策支持**
  - 碳中和目标驱动
  - 新能源车购置税减免延续
  - 储能政策密集出台

- **技术进步**
  - 电池能量密度持续提升
  - 光伏转换效率突破
  - 固态电池量产在即

- **成本下降**
  - 光伏组件价格持续下降
  - 电池成本逐年降低
  - 规模效应显现

## 四、产业链结构

### 1. 产业链全景

```text
上游：资源与原材料
  │
  ├─ 锂矿（天齐锂业、赣锋锂业、盐湖股份）
  │   └─ 锂辉石、盐湖锂、锂云母
  │
  ├─ 硅料（通威股份、大全能源、协鑫科技）
  │   └─ 多晶硅、单晶硅
  │
  ├─ 稀土（北方稀土、盛和资源）
  │   └─ 永磁材料（风电、电机）
  │
  └─ 正极材料（容百科技、当升科技、德方纳米）
      └─ 三元、磷酸铁锂
  │
  ↓
中游：核心制造
  │
  ├─ 锂电池（宁德时代、比亚迪、亿纬锂能、国轩高科）
  │   ├─ 动力电池：新能源汽车
  │   └─ 储能电池：储能系统
  │
  ├─ 光伏（隆基绿能、通威股份、晶澳科技、天合光能）
  │   ├─ 硅片：隆基、中环
  │   ├─ 电池片：通威、爱旭
  │   └─ 组件：晶澳、天合
  │
  ├─ 风电（金风科技、明阳智能、运达股份）
  │   ├─ 整机：金风、明阳
  │   └─ 零部件：新强联、大金重工
  │
  └─ 储能系统（阳光电源、上能电气）
      ├─ 逆变器
      └─ 储能系统集成
  │
  ↓
下游：应用与服务
  │
  ├─ 新能源车（比亚迪、赛力斯、理想产业链）
  │   ├─ 整车制造
  │   └─ 零部件
  │
  ├─ 充电桩（特锐德、盛弘股份、科士达）
  │   ├─ 充电设备
  │   └─ 充电运营
  │
  └─ 电站运营（三峡能源、龙源电力、太阳能）
      └─ 光伏/风电电站
```

### 2. 关键环节分析

#### （1）锂电池（核心环节）

- **作用**：新能源汽车和储能的核心部件
- **技术门槛**：高（材料、工艺、设备）
- **价值占比**：新能源车成本30-40%
- **特点**：规模效应明显、技术迭代快

#### （2）光伏（调整周期）

- **作用**：清洁能源发电的核心
- **技术门槛**：中等（成本竞争为主）
- **价值分布**：硅料>硅片>电池片>组件
- **特点**：产能过剩、价格竞争激烈

#### （3）储能（增长最快）

- **作用**：新能源消纳的关键
- **技术门槛**：中等
- **增长驱动**：新能源配储、独立储能
- **特点**：市场空间大、竞争格局未定

## 七、投资逻辑与风险

### 1. 投资逻辑（行业层面）

1. **碳中和长期趋势**
   - 全球能源转型不可逆转
   - 政策支持力度持续
   - 市场空间巨大

2. **中国产业链优势**
   - 光伏、锂电池、新能源车全球领先
   - 产业链完整，成本优势显著
   - 出海加速，全球份额提升

3. **技术迭代机遇**
   - 固态电池2026-2027年量产
   - 钠电池低成本替代
   - 光伏新技术（HJT、钙钛矿）

### 2. 投资逻辑（公司/选股层面）

**选股思路**：

- **环节选择**：
  - 锂电池：格局稳定，龙头优势明显
  - 储能：增长最快，空间最大
  - 光伏：关注周期拐点

- **重点关注**：
  - 宁德时代、比亚迪：锂电池双雄
  - 阳光电源：储能龙头
  - 天齐锂业、赣锋锂业：锂矿龙头（关注价格拐点）

### 3. 主要风险因素

| 风险类型 | 具体描述 | 影响程度 |
|----------|----------|----------|
| ⚠️ 产能过剩 | 光伏、部分锂电池环节产能过剩 | 高 |
| ⚠️ 价格波动 | 锂价、硅料价格大幅波动 | 高 |
| ⚠️ 地缘政治 | 海外市场贸易壁垒 | 中-高 |
| ⚠️ 技术迭代 | 固态电池等新技术替代风险 | 中 |
| ⚠️ 政策变化 | 补贴退坡、政策调整 | 中 |
