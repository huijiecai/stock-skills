## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 宁波德业科技股份有限公司 |
| 证券代码 | 605117 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2004年 |
| 上市时间 | 2020年11月 |
| 所属行业 | 电气设备 / 户用储能 |
| 市场地位 | 全球户用储能逆变器第一 |

**公司概况**：
德业股份是全球领先的户用储能逆变器制造商，以销售金额计已登顶2024年全球户用储能逆变器市场冠军。公司产品涵盖储能逆变器、储能电池包、光伏逆变器等，形成"逆变器+电池包"的一体化储能解决方案。海外收入占比高达76%，深耕欧洲、美国、澳洲等主流市场。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **储能逆变器** | 户用储能逆变器、混合逆变器 | 家庭储能系统 |
| **储能电池包** | 磷酸铁锂电池包、模块化设计 | 户用储能配套 |
| **光伏逆变器** | 户用光伏逆变器 | 家庭光伏发电 |
| **家电** | 除湿机、空调等传统业务 | 家用电器 |

### 业务模式

公司采用"研发+制造+销售"一体化模式，以ODM/OEM为主，同时发展自主品牌。产品主要销往欧洲、美国、澳洲等海外市场，通过经销商和直销相结合的方式覆盖全球客户。

### 技术路线/核心技术

- **高效逆变技术**：转换效率达98%+，行业领先
- **模块化设计**：电池包可扩展，适配不同用户需求
- **一体化方案**：逆变器+电池包+BMS+EMS完整解决方案
- **智能管理**：APP远程监控、AI优化充放电策略

**技术优势**：
公司拥有超300项专利，在户用储能逆变器领域技术积累深厚。产品通过欧洲、美国、澳洲等全球主流市场认证，品质获得国际认可。

## 五、投资亮点

### 核心亮点

- ✅ **全球户用储能逆变器第一**：2024年销售额市占率冠军
- ✅ **毛利率行业最高**：40%+，远超同行（阳光电源30%、固德威38%）
- ✅ **一体化布局完善**：逆变器+电池包，价值量提升5倍
- ✅ **海外市场强势**：欧洲市占率20%+，品牌认可度高
- ✅ **估值极低**：PE(2026E)仅7.8倍，PEG 0.27

### 成长驱动因素

1. **欧洲户储市场持续增长**
   ```
   欧洲户储装机量：
   2024年：35GWh
   2025年：40GWh（+14%）
   2026年预期：50GWh（+25%）
   
   德业股份出货量：
   2024年：约7GWh（市占率20%）
   2026年预期：10-12GWh
   ```

2. **一体化方案价值量提升**
   ```
   单户价值量：
   仅逆变器：5000-8000元
   逆变器+电池包：3-5万元（提升5倍）
   
   毛利率提升：
   逆变器：35-38%
   一体化方案：40-43%
   ```

3. **美国市场快速增长**
   ```
   美国IRA法案：
   户储系统补贴30%（税收抵免）
   预计2026年美国户储装机15GWh
   德业股份在美国市场快速渗透
   ```

4. **电池包业务放量**
   ```
   2024年电池包营收：约20亿元
   2025年预期：30亿元（+50%）
   2026年预期：45亿元（+50%）
   ```

### 催化剂

- 欧洲户储需求回暖（电价上涨）
- 美国市场开拓加速（IRA法案支持）
- 电池包业务持续放量
- 新产品发布（大功率逆变器、商用储能）
