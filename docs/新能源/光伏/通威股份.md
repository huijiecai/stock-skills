## 二、主营业务

### 1. 核心业务

通威股份是光伏产业链一体化龙头：

- **高纯晶硅**：多晶硅、单晶硅，全球份额第一
- **太阳能电池**：PERC、TOPCon、HJT电池，全球份额第一
- **组件**：光伏组件产品
- **农牧业务**：水产饲料

### 2. 业务结构

| 业务板块 | 营收占比 | 说明 |
|----------|----------|------|
| 高纯晶硅 | ~35% | 市占率30%，全球第一 |
| 太阳能电池 | ~45% | 连续八年全球第一 |
| 组件 | ~15% | 一体化延伸 |
| 农牧业务 | ~5% | 传统业务 |

### 3. 技术路线

- 硅料：N型硅料占比超90%
- 电池：TOPCon、HJT技术领先
- 组件：HJT组件功率达790.8W
