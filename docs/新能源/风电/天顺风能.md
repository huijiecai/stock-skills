## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 天顺风能（苏州）股份有限公司 |
| 证券代码 | 002531 |

| 成立时间 | 2005年 |
| 上市时间 | 2010年12月31日 |
| 所属行业 | 新能源 / 风电装备制造 |
| 市场地位 | 全球领先的风力发电塔架和海工装备专业制造企业 |

**公司概况**：

天顺风能是集风塔制造、海工装备、叶片生产、风电场开发运营于一体的综合性风电企业。公司在太仓、包头、珠海、丹麦等地设有生产基地，是全球最具规模的风力发电塔架专业制造企业之一。

**核心优势**：
- 风塔市场：国内领先地位，全球产能超250万吨
- 海工装备：积极布局国内七大沿海区域和欧洲德国基地
- 欧洲本地化：在丹麦有资产，德国有生产基地，享受欧洲市场准入优势
- 战略转型：2025年停产6家亏损子公司，聚焦海工核心业务

### 技术优势


**天顺风能劣势**：

- ⚠️ 海工业务起步较晚，市占率待提升
- ⚠️ 2025年业绩亏损，短期承压

---

**与风机厂商的关系**：

天顺风能**不是**风机整机制造商，而是风机厂商的**上游供应商**：

```
产业链位置：

上游零部件 → 风机整机厂 → 风电场运营商
  ↓              ↓
天顺风能      明阳智能
大金重工      金风科技
(风塔供应)    (整机制造)
```

**为什么这个关系很重要？**

1. **客户集中度风险**：天顺的订单依赖风机厂商（金风、明阳等），如果风机厂商订单下滑，天顺会受影响

2. **议价能力**：风机厂商是甲方，天顺是乙方，价格谈判中天顺处于弱势（这也是2024-2025年风塔价格战的原因）

3. **成长空间联动**：
   - 如果明阳智能、金风科技海风订单爆发 → 天顺风塔/海工订单也会增长
   - 反之，如果风机厂商竞争加剧、价格战 → 天顺也会被压价

4. **投资逻辑差异**：
   - **投明阳智能/金风科技**：赌风机整机需求（直接受益海风装机增长）
   - **投天顺风能/大金重工**：赌风塔/海工需求（间接受益，但估值更低）

### 核心预期是什么

**一句话总结**：
天顺风能投的是**"陆风周期拐点+海风成长爆发"的叠加预期**，2025年业绩出清后，2026年进入业绩兑现期，海风装机翻倍+订单加速落地，预计净利润从亏损2亿到盈利6-10亿。

**市场当前预期**：
- 悲观预期：2025年亏损2亿，陆风继续内卷，海风项目延期
- 机构预期：2026年拐点，但对幅度存疑（净利润6-10亿）

**实际情况（预期差）**：
1. **行业拐点已确认**：
   - 陆风：风塔价格从低点反弹15%+，突破2000元/kW（供需改善）
   - 海风：2025年装机翻倍到8-10GW，2026年+30%到11-13GW（渗透率加速）
   
2. **订单加速落地**：
   - 3个月新签8.7亿海工订单（金风科技、中广核等大客户）
   - 海风项目延期问题解决（2025年1月政策调整期结束）
   
3. **战略转型见效**：
   - 壮士断腕：停产6家亏损子公司，聚焦海工
   - 欧洲布局：丹麦收购+德国基地，海外市场打开
   
4. **宏观催化剂**（市场忽视）：
   - 欧洲能源危机加剧（美伊战争+俄乌冲突）
   - 欧洲加速发展海上风电（能源安全战略）
   - 天顺欧洲本土化优势凸显（稀缺性）

**为什么有预期差**：
- **信息不对称**：市场还在关注2025年亏损，忽略了2026年拐点
- **认知偏差**：市场认为海风延期=需求不足，实际是政策调整期后集中释放
- **时间差**：行业拐点已现（价格+订单），但业绩拐点要到2026年Q2才能验证
