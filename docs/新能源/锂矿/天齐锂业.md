## 一、公司简介

### 基本信息
- **公司名称**：天齐锂业股份有限公司
- **股票代码**：002466.SZ / 09696.HK
- **所属行业**：有色金属 / 锂矿
- **上市时间**：2010年
- **总部位置**：四川成都
- **市值**：约930亿元

### 公司定位
天齐锂业是全球领先的锂矿企业，拥有全球最优质的锂资源（格林布什锂辉石矿），是新能源汽车产业链上游的核心标的。

| 锂精矿 | ~35% | 锂辉石精矿销售 |
| 其他 | ~5% | 锂金属、衍生品等 |

### 核心资产

- **雅江措拉锂矿**：国内优质锂矿资源
- **盐湖布局**：西藏盐湖资源

### 产品矩阵
- 锂精矿：销售给锂盐厂
- 碳酸锂：动力电池正极材料
- 氢氧化锂：高镍三元电池
- 金属锂：固态电池布局
