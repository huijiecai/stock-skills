# 07 存储芯片

> 数据的"仓库"，AI时代的核心瓶颈之一

---

## 一、存储芯片总览

### 1️⃣ 是什么？

> **存储芯片 = 电子产品的"仓库"**
>
> 用来存放数据，不同类型的存储芯片就像不同用途的仓库。

```
┌─────────────────────────────────────────────────────────────┐
│                    计算机存储层次                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   速度快                                                     │
│     ↑      ┌─────────┐                                      │
│     │      │ 寄存器  │   ← CPU内部，最快                     │
│     │      └────┬────┘                                      │
│     │      ┌────┴────┐                                      │
│     │      │ 缓存L1/2│   ← SRAM，很快很贵                   │
│     │      └────┬────┘                                      │
│     │      ┌────┴────┐                                      │
│     │      │  内存   │   ← DRAM，快且较贵                   │
│     │      └────┬────┘                                      │
│     │      ┌────┴────┐                                      │
│     ↓      │ 固态硬盘│   ← NAND Flash，较快较便宜           │
│   速度慢   └────┬────┘                                      │
│            ┌────┴────┐                                      │
│            │ 机械硬盘│   ← HDD，慢但便宜                    │
│            └─────────┘                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2️⃣ 市场规模

```
全球存储芯片市场规模（2025年）：

市场规模：约 1500亿美元
年增速：约 15-20%（AI带动）

按类型分：
DRAM:    ████████████████████████████████████████████████ 55%
NAND:    ████████████████████████████████████ 40%
其他:    ████ 5%
```

### 3️⃣ 存储芯片分类

| 类型 | 全称 | 特点 | 速度 | 断电 | 用途 |
|------|------|------|------|------|------|
| **SRAM** | 静态随机存储器 | 最快、最贵 | ⚡⚡⚡⚡⚡ | 丢失 | CPU缓存 |
| **DRAM** | 动态随机存储器 | 快、需刷新 | ⚡⚡⚡⚡ | 丢失 | 内存条 |
| **HBM** | 高带宽存储器 | 超高带宽 | ⚡⚡⚡⚡⚡ | 丢失 | **AI芯片** |
| **NAND** | 闪存 | 断电保存 | ⚡⚡ | 保存 | SSD |
| **NOR** | 闪存 | 可执行代码 | ⚡⚡⚡ | 保存 | 启动程序 |

### 1️⃣ 是什么？

> **DRAM = 电脑/手机的"内存"**

> 用于临时存储CPU正在处理的数据，断电丢失。

## 五、HBM详解（重点！）

### 1️⃣ 是什么？

> **HBM = 高带宽存储器 = AI芯片的"专用内存"**
>
> 把多层DRAM垂直堆叠，通过3D封装实现超高带宽。

### 2️⃣ 市场规模

```
HBM市场规模（2025年）：

市场规模：约 150亿美元
年增速：约 50-80%（最快增长的存储领域！）

驱动因素：
• AI芯片需求爆发
• 每颗AI芯片需要多颗HBM
```

### 3️⃣ 为什么AI需要HBM？

```
AI模型训练/推理的瓶颈：

GPU算力 ████████████████████████████████ 足够
内存带宽 ████████ 不够！

传统GDDR带宽：~512GB/s
HBM3e带宽：~1TB/s+ （翻倍！）

结论：没有HBM，GPU性能发挥不出来
```

### 4️⃣ 竞争格局

```
HBM市场份额（2025年）：

SK海力士:    ████████████████████████████████████████████ 53%
三星:        ████████████████████████████████████ 38%
美光:        ██████ 9%

⚠️ SK海力士是英伟达独家供应商！
```

| 公司 | 市占率 | 优势 | 主要客户 |
|------|--------|------|----------|
| **SK海力士** | 53% | 技术最领先 | **英伟达独家** |
| **三星** | 38% | 产能大 | AMD、谷歌 |
| **美光** | 9% | 追赶中 | 部分客户 |

### 5️⃣ 国产替代

## 六、存储模组

### 1️⃣ 是什么？

> **存储模组 = 把存储颗粒"组装"成可直接使用的产品**
>
> 像组装电脑一样，把颗粒+主控+PCB组装成SSD、内存条等。

```
存储产业链分工：

【上游】颗粒厂商              【中游】模组厂商              【下游】终端
三星、美光、长江存储          江波龙、德明利等              手机、电脑、服务器
        │                           │                           │
        ▼                           ▼                           ▼
  NAND/DRAM颗粒    →    组装成SSD/内存条/U盘    →    装进设备使用
  （芯片本身）            （可直接使用的产品）
```

### 2️⃣ 模组的组成

```
┌─────────────────────────────────────────┐
│           SSD固态硬盘（模组产品）         │
├─────────────────────────────────────────┤
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐  │
│  │NAND  │ │NAND  │ │NAND  │ │NAND  │  │ ← 存储颗粒（采购）
│  │颗粒  │ │颗粒  │ │颗粒  │ │颗粒  │  │
│  └──────┘ └──────┘ └──────┘ └──────┘  │
│           ┌──────────────┐              │
│           │   主控芯片   │              │ ← 控制读写/纠错
│           └──────────────┘              │
│  ═══════════════════════════════════   │ ← PCB板 + 固件
└─────────────────────────────────────────┘

模组 = 存储颗粒 + 主控芯片 + 固件 + PCB
```

### 3️⃣ 颗粒厂商 vs 模组厂商

| 对比 | 颗粒厂商 | 模组厂商 |
|------|----------|----------|
| 代表 | 三星、美光、长江存储 | 江波龙、德明利 |
| 做什么 | 造芯片颗粒 | 组装成产品 |
| 技术壁垒 | **极高**（晶圆制造） | 中等（组装+主控） |
| 资金门槛 | 几百亿美元 | 几亿~几十亿 |
| 毛利率 | 高（30-50%） | 中低（15-25%） |

## 七、存储周期

### 周期特点

```
存储芯片是强周期行业：

价格上涨 → 厂商扩产 → 供过于求 → 价格下跌 → 减产 → 供不应求 → 价格上涨
    ↑                                                              │
    └──────────────────────────────────────────────────────────────┘

周期通常3-4年一轮
```

### 当前周期位置（2026年初）

```
2022-2023: 下行周期（价格暴跌、库存高企）
2024: 周期底部（去库存完成）
2025-2026: 上行周期（AI需求拉动、价格回升）

当前位置：上行周期中
```

## 八、投资逻辑总结

### 核心逻辑

```
1. AI带动HBM需求爆发
   • AI芯片必须配HBM
   • HBM供不应求，价格上涨
   • 利好：SK海力士、三星、美光

2. 存储周期上行
   • 价格见底回升

   • 利好：全行业

3. 国产替代
   • 长江存储、长鑫存储技术进步
```
