## 一、行业定义与范围

### 是什么？

> **有色金属 = 除黑色金属（铁、铬、锰）之外的所有金属**
>
> 有色金属是指除铁、铬、锰以外的所有金属及其合金，包括基本金属（铜、铝、铅、锌、镍、锡等）、贵金属（黄金、白银、铂等）、稀有金属（稀土、钨、锂、钼、钴等）。广泛应用于电力、建筑、交通、电子、新能源等领域。

| 细分领域 | 说明 | 典型应用 |
|----------|------|----------|
| **基本金属** | 铜、铝、铅、锌、镍、锡等常用工业金属 | 电力电缆、建筑材料、交通运输、包装材料 |
| **贵金属** | 黄金、白银、铂、钯等稀贵金属 | 珠宝首饰、电子元件、金融投资、化工催化 |
| **稀有金属** | 稀土、钨、锂、钼、钴、锑等战略金属 | 新能源电池、永磁材料、国防军工、半导体 |

### 产业链定位

```
矿产资源勘探开采
    ↓
【矿石采选与粗加工】
    ↓
【有色金属冶炼】（本行业核心）
    ↓
【压延加工与深加工】
    ↓
终端应用（电力、建筑、交通、电子、新能源等）
```

**说明**：
有色金属行业是连接资源开采与终端制造的关键中游环节，承担着将原矿资源转化为可用金属材料的重任。中国已构建全球最完备的有色金属产业链，精炼金属产量占据全球主导地位。

### 增长驱动因素

1. **新能源产业爆发式增长**
   - 新能源汽车、光伏、风电对铜、铝、锂、稀土等金属需求激增

2. **AI与数据中心建设提速**
   - AI算力中心建设带动铜、铝等导电材料需求
   - 单座数据中心铜用量约1,000-2,000吨

3. **战略资源争夺加剧**
   - 中美战略竞争推动各国增加关键金属储备
   - 稀土、钨、锂等战略小金属地位提升

4. **传统基建与制造业升级**
   - 中国"双碳"目标下电网改造、新基建持续推进
   - 高端制造对金属材料性能要求提升

### 竞争态势

- 资源为王：上游矿产资源成为核心竞争力，全球铜矿资源集中于4大矿企
- 技术壁垒：冶炼环节对能效、环保、智能化有严格要求，新进入壁垒极高
- 规模效应：头部企业产能规模优势明显，成本控制能力突出
- 地缘政治：战略金属受国家管控，行业竞争与国际关系深度绑定

## 四、产业链结构

### 产业链全景

```
上游：矿产资源勘探与开采
  │
  ├─ 铜矿开采 → 紫金矿业、江西铜业、中国有色矿业
  ├─ 铝土矿开采 → 中国铝业、中国宏桥
  ├─ 稀土矿开采 → 北方稀土、中国稀土、五矿稀土
  └─ 锂矿开采 → 天齐锂业、赣锋锂业
  │
  ↓
中游：有色金属冶炼（本行业）
  │
  ├─ 铜冶炼 → 江西铜业、铜陵有色、云南铜业
  ├─ 铝冶炼 → 中国铝业、云铝股份、神火股份
  ├─ 稀土冶炼分离 → 盛和资源、广晟有色
  └─ 锂盐加工 → 天齐锂业、赣锋锂业、盛新锂能
  │
  ↓
下游：金属压延加工与终端应用
  │
  ├─ 电力电缆 → 远东电缆、宝胜股份
  ├─ 新能源汽车 → 比亚迪、宁德时代（用铜、铝、锂）
  ├─ 永磁材料 → 金力永磁、中科三环（用稀土）
  └─ 半导体芯片 → 中芯国际、华虹半导体（用高纯金属）
```

### 关键环节分析

#### 矿产资源开采

- **作用**：提供原料金属矿石，决定产业链成本与供应稳定性
- **技术门槛**：高（地质勘探、采矿技术、环保要求）
- **价值占比**：约40-60%（占总成本）
- **龙头企业**：紫金矿业、洛阳钼业、西部矿业

#### 金属冶炼

- **作用**：将矿石提炼为精炼金属，是产业链核心环节
- **技术门槛**：高（冶炼工艺、能效控制、环保排放）
- **价值占比**：约20-30%（加工增值）
- **龙头企业**：江西铜业、中国铝业、北方稀土

#### 深加工与应用

- **作用**：将精炼金属加工为终端产品，满足下游需求
- **技术门槛**：中-高（精密加工、材料科学）
- **价值占比**：约20-30%（终端溢价）
- **龙头企业**：宁德时代（锂电池）、金力永磁（稀土磁材）

## 六、投资逻辑与风险

### 投资逻辑

#### 行业层面

1. **供给约束下的资源稀缺性价值重估**
   - 全球铜矿供给2026年预计零增长甚至负增长，供需缺口约83万吨
   - 矿山开发周期长（10-15年），短期供给刚性
   - 铜价中枢有望上移至1.3万美元/吨，支撑铜企盈利

2. **新能源与AI双轮驱动的需求爆发**
   - 新能源车、光伏、风电对铜、铝、锂、稀土需求激增
   - AI算力中心建设带动铜、铝用量大幅增长
   - 2026年新能源产业对铜需求增量预计超30万吨

3. **战略金属地位提升，地缘溢价凸显**
   - 中美战略竞争推动稀土、钨、锂、钴等金属战略地位提升
   - 各国央行增加战略金属储备，稀土、钨价格受政策支撑
   - 关税壁垒与出口管控强化资源稀缺性

#### 公司层面

**选股逻辑**：
- 资源优势：海外矿产资源丰富、成本控制能力强（紫金矿业、洛阳钼业）
- 产业链完整：上游资源+中游冶炼+下游加工一体化（江西铜业、北方稀土）
- 战略定位：国家战略金属龙头，政策受益明显（北方稀土、中国稀土）
- 成长性：新能源材料布局、产能扩张快（天齐锂业、华友钴业）

### 风险因素

| 风险类型 | 具体描述 | 影响程度 |
|----------|----------|----------|
| ⚠️ **宏观经济波动** | 全球经济复苏不及预期，下游需求萎缩，金属价格下跌 | 高 |
| ⚠️ **供给超预期释放** | 矿山投产进度加快、再生金属产量大增，供需平衡被打破 | 中 |
| ⚠️ **地缘政治风险** | 中美贸易摩擦升级、关税加征、资源出口管制 | 高 |
| ⚠️ **新能源需求不及预期** | 新能源车渗透率增速放缓、光伏装机下降 | 中 |
| ⚠️ **技术替代风险** | 新材料替代传统金属（如石墨烯替代铜、固态电池减少锂用量） | 低 |
| ⚠️ **环保政策收紧** | 矿山开采、冶炼环节环保成本上升，盈利承压 | 中 |
