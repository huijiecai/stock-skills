## 二、算电协同三大受益主线

### 主线一：电网设备升级（供给侧）

**核心逻辑**：
```
西部绿电 → 特高压输送 → 东部算力中心 → 末端配电升级
```

**受益环节**：
1. **特高压输变电**：远距离大容量输电，西电东送加速
2. **数据中心配电**：单机柜功耗从10-20kW升至50-100kW，传统配电系统无法支撑
3. **智能调度系统**：AI优化电网调度，提升运行效率

**核心标的**：
- 特变电工（600089）：特高压变压器龙头
- 平高电气（600312）：GIS龙头
- 许继电气（000400）：换流阀龙头
- 金盘科技（688676）：液冷变压器+SST
- 国电南瑞（600406）：电网AI调度

