## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 思源电气股份有限公司 |
| 证券代码 | 002028 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 2004年 |
| 所属行业 | 输配电设备、电力电子 |
| 市场地位 | 电力设备出海领军企业 |

**公司概况**：
思源电气是一家专注于输配电设备的中国高新技术企业，产品涵盖开关、线圈、无功补偿及智能设备等领域。公司是国内电力设备出海领军企业，海外订单占比超33%，业务覆盖100多个国家和地区。2025年营收212亿元，净利润32亿元，同比增速超50%。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **开关设备** | 高压开关、GIS组合电器 | 变电站、电网 |
| **变压器** | 电力变压器、特种变压器 | 输配电系统 |
| **电力电子** | 无功补偿、SVG | 电网稳定 |
| **储能系统** | 储能设备、微电网 | 新能源配套 |

### 业务模式

公司采用"产品+工程+服务"模式，为国内外电网公司提供输配电设备和解决方案。海外订单占比超33%，国际化程度高。

### 技术路线/核心技术

- **开关技术**：高压开关技术领先
- **变压器技术**：高端变压器工厂投产
- **电力电子**：无功补偿技术成熟
- **储能技术**：与宁德时代合作50GWh储能系统

**技术优势**：
公司产品线多元化，技术积累深厚。海外布局领先，在泰国、美国、墨西哥等地设有生产基地。

## 四、行业地位与产业链定位

### 产业链定位

```
输配电产业链：
原材料 → 设备制造 → 【思源电气】→ 电网公司 → 终端用户
                              ↑
                        出海领军企业
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内市场份额约3.5%，出海领先 |
| **竞争优势** | 产品多元化+出海领先+成本优势 |
| **客户结构** | 国家电网、海外电网公司 |
| **海外布局** | 泰国、美国、墨西哥等生产基地 |

### 竞争格局

**主要竞争对手**：
- 特变电工：变压器龙头
- 中国西电：特高压龙头
- 国电南瑞：电网自动化龙头

**公司优势**：
出海领先（海外订单33%+）、产品多元化、成本控制能力强、全球化布局领先。

## 五、投资亮点

### 核心亮点

- ✅ **出海领军企业**：海外订单占比超33%，业务覆盖100+国家
- ✅ **业绩高增长**：2025年营收+37%，净利润+54%
- ✅ **储能合作**：与宁德时代签订50GWh储能系统供应合同
- ✅ **港股IPO**：冲击港股IPO，国际化加速
- ✅ **盈利能力强**：毛利率30%，净利率15%，ROE 20%

### 成长驱动因素

1. **出海加速**
   - 欧美电网老旧升级需求旺盛
   - 中国电力设备性价比优势明显

2. **储能业务增长**
   - 与宁德时代合作50GWh
   - 储能装机快速增长

3. **产品多元化**
   - 开关、变压器、电力电子全覆盖
   - 抗风险能力强

### 催化剂

- 港股IPO进展
- 海外大单签约
- 储能业务放量
