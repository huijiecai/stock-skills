## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 伊戈尔电气股份有限公司 |
| 证券代码 | 002922 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1996年 |
| 上市时间 | 2017年 |
| 所属行业 | 电力电子、电源 |
| 市场地位 | 数据中心电源+变压器出海成长股 |

**公司概况**：
伊戈尔是一家专注于电源及电源组件的企业，产品涵盖数据中心电源、变压器等。公司积极布局出海市场，在泰国、美国、墨西哥等地设有生产基地，是数据中心电源和变压器出海的成长股。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **数据中心电源** | 模块化电源、UPS | AI数据中心 |
| **变压器** | 电力变压器、配电变压器 | 电网、工业 |
| **照明电源** | LED驱动电源 | 照明行业 |

### 业务模式

公司采用"产品+定制"模式，为数据中心、电网、照明行业提供电源和变压器产品。

### 技术优势

公司在数据中心电源、变压器领域技术积累深厚，出海布局领先。
