## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国能源建设股份有限公司 |
| 证券代码 | 601868 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2014年 |
| 上市时间 | 2021年 |
| 所属行业 | 电力工程、能源建设 |
| 市场地位 | 能源电力建设国家队 |

**公司概况**：
中国能建是国务院国资委监管的中央企业，是中国乃至全球领先的电力行业综合解决方案提供商。公司深度参与"东数西算"战略，是全国算力枢纽和数据中心项目的核心建设者，发挥"数能融合"优势，推动电力基础设施升级。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **电力工程总承包** | 火电、水电、核电、新能源电站建设 | 电力基础设施建设 |
| **勘测设计** | 电力工程勘测、设计、咨询 | 工程前期服务 |
| **装备制造** | 电力装备、建材装备 | 工程配套 |
| **投资运营** | 电站投资、运营 | 长期收益 |
| **算力枢纽建设** | 数据中心、算力基础设施 | "东数西算" |

### 业务模式

公司采用"勘测设计+工程总承包+装备制造+投资运营"一体化模式，提供电力工程全生命周期服务。

### 技术路线/核心技术

- **电力工程技术**：国内领先，参与国家重大工程
- **新能源技术**：风光储一体化解决方案
- **氢能技术**：氢能投资超76亿元，布局前沿
- **储能技术**：压缩空气储能装备

**技术优势**：
公司拥有完整的电力工程技术体系，参与国家重大能源项目建设，在新能源、氢能、储能等前沿领域积极布局。

## 四、行业地位与产业链定位

### 产业链定位

```
电力工程产业链：
勘测设计 → 工程总承包 → 【中国能建】→ 电站运营 → 电力供应
                              ↑
                        能源电力建设国家队
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内电力工程市场份额领先 |
| **竞争优势** | 央企背景+全产业链+技术领先 |
| **客户结构** | 国家电网、发电集团、政府 |
| **订单情况** | 订单饱满，排期至2027年 |

### 竞争格局

**主要竞争对手**：
- 中国电建：水电工程优势
- 葛洲坝：水利工程优势（已并入能建）

**公司优势**：
央企背景、全产业链能力、参与国家重大工程、订单饱满、新兴业务布局领先。

## 五、投资亮点

### 核心亮点

- ✅ **能源电力建设国家队**：央企背景，订单稳定
- ✅ **算力枢纽核心建设者**：深度参与"东数西算"战略
- ✅ **氢能布局领先**：氢能投资超76亿元
- ✅ **储能业务增长**：压缩空气储能装备，装机0.5GW
- ✅ **订单饱满**：订单排期至2027年，收入可见性强

### 成长驱动因素

1. **"十五五"电网投资**
   - 4万亿电网投资带来工程需求
   - 公司作为总承包商直接受益

2. **算力枢纽建设**
   - "东数西算"战略推进
   - 数据中心电力基础设施需求旺盛

3. **新能源转型**
   - 氢能、储能等新兴业务快速增长
   - 新能源工程占比提升

### 催化剂

- 特高压项目核准
- 算力枢纽大单签约
- 氢能产业化突破
