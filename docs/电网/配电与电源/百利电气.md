| 项目 | 内容 |

| 公司全称 | 天津百利特精电气股份有限公司 |
| 证券代码 | 600468 |

| 上市时间 | 2001年 |
| 所属行业 | 低压电器 |

百利电气是一家专注于低压电器的企业，产品涵盖低压开关、配电设备等。公司正在布局超导材料领域，具有题材潜力。
