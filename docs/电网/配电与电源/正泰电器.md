## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江正泰电器股份有限公司 |
| 证券代码 | 601877 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997年 |
| 上市时间 | 2010年 |
| 所属行业 | 低压电器、户用光伏 |
| 市场地位 | 低压电器龙头+户用光伏第一 |

**公司概况**：
正泰电器是中国低压电器行业的龙头企业，通过"智慧电器+绿色能源"双轮驱动战略，在传统低压电器保持行业领先的同时，在户用光伏领域取得突破性进展。子公司正泰安能是中国户用光伏市场的领导者，市场份额连续多年第一。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **低压电器** | 断路器、接触器、配电设备 | 建筑配电、工业用电 |
| **户用光伏** | 光伏组件、逆变器、储能 | 家庭分布式光伏 |
| **智能配电** | 智能终端、数字化配电系统 | 智能电网、数据中心 |
| **AIDC供电** | 固态变压器(SST)、模块化供电 | AI数据中心（布局中） |

### 业务模式

公司采用"产品+服务+资产运营"模式，低压电器以产品销售为主，户用光伏采用"开发-出售-运维"滚动模式，实现电站资产证券化。

### 技术路线/核心技术

- **低压电器技术**：行业领先，品牌影响力强
- **户用光伏技术**：系统集成能力强，运维效率高
- **AIDC供电**：固态变压器(SST)研发中，2026年有望推出

**技术优势**：
低压电器产品线齐全，技术领先；户用光伏系统集成能力强，运维效率高；正在布局AI数据中心供电新赛道。

## 四、行业地位与产业链定位

### 产业链定位

```
能源产业链：
原材料 → 设备制造 → 【正泰电器】→ 终端用户
                          ↑
                  低压电器+户用光伏双龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 低压电器国内第一，户用光伏市占率35-40% |
| **竞争优势** | 品牌+渠道+成本控制+光伏协同 |
| **客户结构** | 建筑配电、工业用户、家庭用户 |
| **电站资产** | 自持光伏电站27.5GW+，行业最大 |

### 竞争格局

**主要竞争对手**：
- 施耐德电气：国际低压电器龙头
- 常熟开关：国内低压电器企业
- 天合光能、晶科能源：户用光伏竞争者

**公司优势**：
低压电器市占率第一、户用光伏市占率第一、自持电站规模最大、渠道网络完善、品牌影响力强。

## 五、投资亮点

### 核心亮点

- ✅ **低压电器龙头**：市占率第一，品牌+渠道优势明显
- ✅ **户用光伏第一**：市占率35-40%，自持电站27.5GW+
- ✅ **利润增速远超营收**：2025年净利润增速19% vs 营收增速0%
- ✅ **AIDC供电新赛道**：固态变压器研发中，千亿级市场
- ✅ **国际化拓展**：北美市场突破，海外收入增长

### 成长驱动因素

1. **户用光伏持续增长**
   - 国内户用光伏装机快速增长
   - 公司市占率有望提升至40%+

2. **AIDC供电蓝海市场**
   - AI数据中心用电需求激增
   - 固态变压器等新产品2026年推出

3. **国际化加速**
   - 北美市场突破
   - 海外低压电器市场空间广阔

### 催化剂

- AIDC供电产品发布
- 户用光伏装机超预期
- 海外市场大单
