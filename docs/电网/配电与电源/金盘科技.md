## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 金盘科技股份有限公司 |
| 证券代码 | 688676 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2001年 |
| 上市时间 | 2021年8月 |
| 所属行业 | 电力设备 / 变压器制造 |
| 市场地位 | 干式变压器龙头、液冷变压器领先者 |

**公司概况**：
金盘科技是国内领先的干式变压器制造商，专注于环保节能型变压器研发、生产、销售。公司是国内首家实现干式变压器自主化、产业化的企业，产品广泛应用于新能源、数据中心、轨道交通、石油化工等领域。近年来，公司抓住AI数据中心爆发机遇，在液冷变压器、固态变压器（SST）等前沿技术领域取得突破，成为AI数据中心配电的核心受益标的。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **液冷干式变压器** | AI数据中心专用，单台≥2000kVA，效率≥99% | AI超算中心、大型数据中心 |
| **固态变压器（SST）** | 10kV/2.4-3MW，效率98-98.5%，适配HVDC 800V | 英伟达GB200、xAI等高功率AI服务器 |
| **传统干式变压器** | 6-35kV系列产品 | 新能源电站、轨道交通、工业 |
| **非晶合金变压器** | 低损耗、环保型变压器 | 绿色建筑、节能项目 |

### 业务模式

**产品销售+定制化方案**：
```
客户需求分析 → 产品设计定制 → 生产制造 → 交付安装 → 售后服务
```

**收入来源**：
- AI数据中心业务（AIDC）：19%（2025年前三季度）
- 传统业务（新能源、工业等）：81%

**vs 传统变压器企业**：

| 维度 | 金盘科技（液冷变压器） | 传统企业 |
|------|---------------------|---------|
| 技术路线 | 液冷+固态变压器 | 油浸式/干式变压器 |
| 单台价值 | 液冷200-300万，SST近千万 | 传统50-100万 |
| 毛利率 | 30%+ | 15-20% |
| 成长性 | AI驱动，爆发式增长 | 稳定增长 |
| 客户 | 英伟达、xAI、亚马逊等 | 电网公司、工业企业 |

### 技术路线/核心技术

**核心技术突破**：

1. **液冷干式变压器**
   - 已批量供货至xAI孟菲斯超算中心
   - 单台容量≥2000kVA，转换效率≥99%
   - 冷却效率高，适合高功率密度AI服务器

2. **固态变压器（SST）**
   - 10kV/2.4MW样机已完成设计和生产
   - 转换效率98-98.5%（行业先进水平）
   - 适配HVDC 800V供电架构
   - 首台3MW样机2025年12月交付英伟达
   - 预计2026年Q2通过认证并小批量供货

3. **非晶合金变压器**
   - 空载损耗降低70-80%
   - 节能环保优势明显

**技术优势**：
- 国内首家实现干式变压器自主化、产业化
- 液冷变压器技术国内领先，与国际巨头同步
- 固态变压器技术突破，单台价值近千万

## 五、投资亮点

### 核心亮点

- ✅ **AI数据中心配电核心受益标的**：液冷变压器+SST双轮驱动
- ✅ **固态变压器技术突破**：单台价值近千万，2026年Q2量产
- ✅ **全球顶级客户背书**：英伟达、xAI、亚马逊深度合作
- ✅ **业绩爆发式增长**：AIDC业务同比+337%，占比快速提升
- ✅ **产能扩张降本增效**：墨西哥工厂2026年Q3投产，成本降30%
- ✅ **在手订单充足**：75.40亿元，同比+14.89%，订单可见性强

### 成长驱动因素

1. **AI算力爆发驱动配电升级**
   ```
   AI服务器功耗暴增：
   传统服务器：5-10kW/机柜
   AI服务器（H100/H200）：50-80kW/机柜
   AI服务器（GB200）：120kW/机柜
   
   配电需求升级：
   传统UPS+变压器 → 液冷变压器+HVDC
   单机柜配电价值量：3-5万 → 15-25万（提升5倍）
   
   金盘科技受益：
   液冷变压器市场规模：2026年约200亿元
   固态变压器市场规模：2027年开始放量，2030年约500亿元
   ```

2. **固态变压器（SST）量产在即**
   ```
   SST技术优势：
   转换效率98-98.5%（传统95-96%）
   体积缩小50%+
   支持双向功率流动（配合储能）
   支持HVDC 800V架构（英伟达GB200标配）
   
   金盘科技进展：
   2025年12月：首台3MW样机交付英伟达
   2026年Q2：英伟达认证预计完成
   2026年Q4：规模化量产启动
   
   单台价值：
   液冷变压器：200-300万
   固态变压器：近千万
   价值量提升3-5倍
   ```

3. **海外产能布局降本增效**
   ```
   墨西哥工厂：
   投产时间：2026年Q3
   产能规划：满足北美市场需求
   成本优势：比国内生产成本低30%
   
   受益逻辑：
   贴近美国市场（xAI、亚马逊、谷歌等）
   规避关税风险
   毛利率有望从30%→35%+
   ```

4. **全球顶级客户深度绑定**
   ```
   客户质量：
   英伟达：首台SST样机客户，认证通过后大批量采购
   xAI（马斯克）：孟菲斯超算中心液冷变压器供应商
   亚马逊AWS：海外数据中心合作伙伴
   
   客户粘性：
   技术认证周期长（6-12个月）
   认证通过后形成长期合作关系
   英伟达认证相当于全球通行证
   ```

5. **AIDC业务占比快速提升**
   ```
   业务占比变化：
   2024年：<5%
   2025年1-9月：19%
   2026E：30-40%
   2027E：50%+
   
   驱动因素：
   AI数据中心建设高峰期（2026-2028年）
   液冷变压器从0到1渗透
   固态变压器从1到10放量
   ```
