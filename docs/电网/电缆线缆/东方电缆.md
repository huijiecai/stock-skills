## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 宁波东方电缆股份有限公司 |
| 证券代码 | 603606.SH |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2014年10月15日 |
| 所属行业 | 电气设备 - 电缆线缆 |
| 市场地位 | 国内海缆行业龙头 |

**公司概况**：
东方电缆是中国海底电缆行业的龙头企业，专注于海底电缆、高压电缆、海洋装备及工程运维的研发、生产与销售。公司坚持"海陆并进"和"国际化"战略，产品技术达到国际先进水平，是国内少数具备500kV超高压海缆和±525kV直流海缆量产能力的企业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **海缆系统** | 海底电缆（交流/直流）、脐带缆、动态缆 | 海上风电、油气平台、跨洋输电 |
| **陆缆系统** | 高压电力电缆、特种电缆 | 电网建设、城市配电、新能源 |
| **海洋工程** | 海缆敷设、安装、运维服务 | 海上风电场建设、海底电缆工程 |

### 业务模式

公司采用"产品+服务"一体化模式：
- **产品销售**：海缆、陆缆的生产销售，以项目制为主
- **工程服务**：提供海缆敷设、安装、运维等全生命周期服务
- **技术授权**：高端海缆技术输出（海外项目）

### 技术路线/核心技术

- **500kV三芯交流海底电缆**：全球首次应用，技术国际领先
- **±525kV直流海缆**：实现国产化突破，打破国外垄断
- **深远海风电用环保海缆**：满足深远海风电需求
- **超深水海缆**：具备1500米水深敷设能力

**技术优势**：
累计交付超过1.2万公里海底电缆，拥有完整的深海电缆设计、制造、安装及运维能力，多项技术填补国内空白。

## 四、行业地位与产业链定位

### 产业链定位

```
海上风电产业链：
风机整机 → 海缆输电 → 【东方电缆】→ 海上风电场 → 电网并网
                  ↑
            海缆设计/制造/敷设/运维
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内海缆市场前三，高端海缆市占率领先 |
| **竞争优势** | 500kV超高压海缆技术领先，具备EPC总包能力 |
| **客户结构** | 覆盖三峡集团、中广核、国家电投等头部能源企业 |
| **供应商** | 铜材、绝缘材料等上游依赖度适中 |

### 竞争格局

**主要竞争对手**：
- **中天科技（600522）**：海缆+光通信双主业，工程交付能力强
- **亨通光电（600487）**：全球光纤通信龙头，海缆业务快速追赶
- **宝胜股份**：电力电缆为主，海缆布局较晚

**公司优势**：
- 高端海缆技术领先（500kV/±525kV）
- 海缆+海工一体化服务能力
- 海外市场拓展领先（欧洲子公司）

## 五、核心投资逻辑

### 投资类型判断

| 类型 | 判断 | 说明 |
|------|------|------|
| **成长股** | ✅ 是 | 海上风电渗透率提升，深远海趋势明确 |
| **周期股** | ⚠️ 部分 | 受海上风电装机周期影响 |
| **价值股** | ❌ 否 | 估值不低，非典型价值标的 |

### 核心预期差分析

| 维度 | 市场预期（A） | 实际情况（B） | 预期差（B-A） |
|-----|-------------|-------------|--------------|
| 行业趋势 | 海上风电增速放缓 | 2025年国内装机预计超10GW，欧洲需求爆发 | ⭐⭐⭐ |
| 公司业绩 | 2025年净利润持平或微增 | 在手订单196亿，交付加速，业绩有望超预期 | ⭐⭐⭐ |
| 海外拓展 | 海外收入占比低 | 2024年海外收入增长480%，欧洲市场突破 | ⭐⭐⭐ |
| 毛利率 | 持续承压 | 高规格海缆占比提升，毛利率有望修复 | ⭐⭐ |
| 催化剂 | 政策不确定 | 国内海风政策拐点已现，欧洲《汉堡宣言》 | ⭐⭐⭐ |

**预期差总分**：14个⭐（大幅超预期）

### 核心投资逻辑

**一句话总结**：
投的是海上风电深远海化+出海双轮驱动，国内海风装机迎来拐点，欧洲市场需求爆发，公司在手订单创历史新高，业绩弹性有望持续释放。

**详细逻辑**：

1. **行业景气度拐点确立**
   - 2024年国内海风招标11.3GW，预计2025年装机超10GW
   - 欧洲多国签署《汉堡宣言》，目标2050年北海100GW海上风电
   - 深远海趋势明确，高规格海缆需求占比提升

2. **订单储备充足，业绩确定性强**
   - 在手订单约196亿元（2025年初），同比增长130%
   - 订单结构优化，高规格海缆占比提升
   - 产能饱满，2025年产能利用率预计维持高位

3. **海外业务爆发，第二增长曲线**
   - 2024年海外收入7.33亿元，同比增长480.54%
   - 已在欧洲、英国设立子公司，中标多个海外项目
   - 欧洲海缆产能短缺，中国厂商迎来出海窗口期

4. **技术壁垒深厚，竞争格局优**
   - 国内少数具备500kV/±525kV海缆量产能力
   - 海缆行业CR3超80%，寡头垄断格局稳定
   - 技术迭代加速，落后产能逐步出清

## 十一、最终判断

### 核心结论

东方电缆是海上风电产业链的核心标的，受益于国内海风装机拐点和欧洲市场需求爆发，公司在手订单创历史新高，业绩弹性有望持续释放。虽然当前估值偏高，但中期业绩增长确定性强，具备较好的配置价值。

| 时间维度 | 评级 | 操作建议 |

| 短期（1-3月） | 🟡 中性偏多 | 逢低关注，控制仓位 |

| 长期（1-2年） | 🟢 看多 | 核心配置，长期持有 |
