## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏通光电子线缆股份有限公司 |
| 证券代码 | 300265 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2002年 |
| 上市时间 | 2011年 |
| 所属行业 | 特种线缆 |
| 市场地位 | 特种线缆细分龙头 |

**公司概况**：
通光线缆是一家专注于特种线缆的企业，产品涵盖光纤光缆、特种电缆等。公司具有军工+特高压双概念，是特种线缆领域的细分龙头。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **光纤光缆** | 通信光缆 | 通信网络 |
| **特种电缆** | 军工电缆、特高压电缆 | 军工、电网 |
| **输电线路** | 输电导线 | 电网 |

### 业务模式

公司主要采用"产品+定制"模式，为通信、军工、电网客户提供特种线缆产品。

### 技术优势

公司在特种线缆领域技术积累深厚，具有军工资质。
