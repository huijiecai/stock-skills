## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 青岛汉缆股份有限公司 |
| 证券代码 | 002498 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1982年 |
| 上市时间 | 2010年 |
| 所属行业 | 电缆线缆 |
| 市场地位 | 高压电缆龙头 |

**公司概况**：
汉缆股份是中国电线电缆行业的龙头企业，专注于高压、超高压电缆的研发、生产和销售。公司在高压电缆技术领域领先，受益于特高压建设和电网升级。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **高压电缆** | 110kV-500kV高压电缆 | 输电线路 |
| **超高压电缆** | 500kV以上超高压电缆 | 特高压输电 |
| **特种电缆** | 海底电缆、核电电缆 | 特殊应用 |

### 业务模式

公司主要采用"产品+工程"模式，为电网公司提供电缆产品和工程服务。

### 技术优势

公司在高压、超高压电缆技术领域领先，是特高压建设的重要供应商。
