## 一、公司概述

### 业务结构

豫能控股是河南省最大的火电运营企业，同时积极布局新能源和算力业务。

**主营业务**：
1. **火力发电**（核心业务）
   - 装机容量：约1200万千瓦
   - 覆盖河南省主要地市
   - 承担河南电网基荷电源任务

2. **新能源发电**（成长业务）
   - 光伏、风电项目逐步投产
   - 目标：清洁能源占比达30%

3. **算力业务**（战略业务）
   - 布局数据中心建设
   - 利用自有电力优势降低算力成本
   - "电力+算力"协同模式探索

### 竞争优势

1. **电价成本优势**
   - 自有火电厂，电力成本可控
   - 可为数据中心提供低价绿电直供

2. **区位优势**
   - 河南是人口大省、数据流量大省
   - 靠近京津冀算力枢纽节点

3. **国资背景**
   - 河南省属国企，资源整合能力强
   - 土地、审批等方面有政策支持

## 三、投资逻辑

### 核心看点

#### 1. "电力+算力"协同模式（最核心）

**商业逻辑**：
```
自有电厂 → 低成本电力 → 数据中心直供 → 降低算力成本30-50%
```

**进展**：
- 已布局数据中心项目
- 与算力企业签约合作
- 探索虚拟电厂参与电网调度

**价值**：
- 传统火电企业转型突破口
- "算电协同"政策直接受益标的

#### 2. 河南省算力枢纽建设

**政策支持**：
- 河南省"十五五"规划明确支持算力基础设施
- 郑州、洛阳等地布局智算中心

**公司优势**：
- 省属国企，政策资源丰富
- 已有电力基础设施，改造成本低

#### 3. 传统火电业务盈利改善

**驱动因素**：
- 煤价回落，成本压力缓解
- 电力市场化改革，电价有上调空间
- 冬季供暖+夏季用电高峰，火电利用小时数提升

---

## 四、风险因素

### 主要风险

1. **煤价波动风险**（高）
   - 火电成本高度依赖煤价
   - 煤电价格联动机制不完善

2. **算力业务不确定性**（中）
   - 数据中心建设进度慢于预期
   - 算力市场竞争激烈，价格战风险

3. **政策风险**（低）
   - 碳中和压力，火电装机受限
   - 新能源替代趋势

4. **财务风险**（中）
   - 负债率高，财务费用压力大
   - 资本开支需求大（数据中心投资）

## 八、研究结论

### 投资评级

**⭐⭐⭐ 谨慎推荐（3星）**

**核心观点**：
1. **传统业务稳健**：火电基本盘稳定，为算力业务提供试错空间
2. **转型方向正确**："电力+算力"是火电企业最佳转型路径之一
3. **执行风险较大**：算力业务尚在起步期，能否兑现存在不确定性
4. **估值略贵**：当前PE 25x，需要业绩快速增长来消化

### 适合场景

✅ **适合**：
- 看好"算电协同"主题的投资者
- 能承受较大波动的投资者
- 小仓位试探性配置（5-10%）

❌ **不适合**：
- 寻求稳定分红的投资者
- 不能承受高波动的投资者
- 希望短期快速获利的投资者
