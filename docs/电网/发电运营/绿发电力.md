## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 天津绿发电力集团股份有限公司 |
| 证券代码 | 000537 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1986年3月5日 |
| 上市时间 | 1993年12月10日 |
| 所属行业 | 电力 / 新能源发电 |
| 市场地位 | 千万千瓦级绿电运营商、央企平台 |

**公司概况**：
绿发电力是中国绿发投资集团有限公司旗下唯一新能源开发运营主体，国内最早从事新能源开发的企业之一。公司于2021年完成从房地产向新能源业务的战略转型，2026年1月正式更名为"天津绿发电力集团股份有限公司"。截至2024年底，公司在运装机容量达1845万千瓦，实现从"百万千瓦"向"千万千瓦"级的历史性跨越。

## 二、主营业务

### 核心产品/服务

| 业务类型 | 说明 | 应用领域 |
|----------|------|----------|
| **光伏发电** | 集中式光伏电站（占比79%） | 清洁能源供应 |
| **风力发电** | 陆上风电、海上风电（占比21%） | 清洁能源供应 |
| **光热发电** | 光热储能一体化项目 | 清洁能源供应 |
| **储能** | 配套储能系统 | 电网调峰调频 |

### 业务模式

**新能源运营商模式**：
```
投资建设新能源电站 → 运营发电 → 卖电给电网 → 获得电费收入
                      ↓
            规模化扩张 → 装机容量快速增长
```

**收入来源**：
- 电费收入：95%+（主要）
- 补贴收入：逐步退坡（存量项目）
- 绿证收入：约5%

**业务模式特点**：
```
绿发电力（新能源）vs 节能风电（纯风电）

| 维度 | 绿发电力 | 节能风电 |
|------|---------|---------|
| 装机规模 | 1845万千瓦 | 900万千瓦 |
| 技术路线 | 光伏79%+风电21% | 纯风电 |
| 增速 | +228%（2024年） | +27%稳定增长 |
| 分红率 | 20-30% | 50%（高分红） |
| 定位 | 高成长型 | 稳定分红型 |
```

### 技术路线/核心技术

**开发模式创新**：
- **"旅游+新能源基地"模式**：独特开发模式，在新疆等地获取大基地项目指标
- **大基地项目开发**：截至2024年底累计获取新疆大基地项目指标30.3GW
- **多能互补布局**：海陆风电、光伏、光热多能互补格局

**区域布局**：
- 业务覆盖16个省份
- 资产主要集中在新疆（最大）、青海、江苏等西北和沿海地区
- 新疆一、二期大基地项目为核心

**核心能力**：
- 项目获取能力：依托央企背景，获取大基地项目指标
- 成本控制能力：新建项目坚守7%资本金内部收益率
- 集团协同优势：大股东定增注资、收购非核心资产提供现金流支持

## 四、行业地位与产业链定位

### 产业链定位

```
新能源发电产业链：
上游：光伏组件、风机制造商（隆基绿能、金风科技）
  ↓
中游：【绿发电力】新能源电站投资建设运营
  ↓
下游：电网公司（国家电网、南方电网）
  ↓

```

## 五、投资亮点

### 核心亮点

- ✅ **高成长爆发期**：2024年装机同比+228.52%，从"百万千瓦"跨向"千万千瓦"级
- ✅ **在建项目业绩释放潜力大**：在建918万千瓦，占资产总额46.20%，陆续投产将驱动业绩增长
- ✅ **大基地项目优势明显**：累计获取新疆大基地指标30.3GW，未来3-5年增长可见性强
- ✅ **成本竞争力突出**：在西北地区电价承压情况下仍能坚守7%的资本金内部收益率
- ✅ **集团协同支持**：大股东定增注资、收购非核心资产，提供现金流支持
- ✅ **消纳条件改善**：特高压"疆电外送"第三、四通道有望2025年投产，新疆新能源外送能力提升至33GW+

### 成长驱动因素

1. **"十五五"新能源装机高峰**
   ```
   "十五五"新能源装机目标（2026-2030）：
   全国年均新增：200-250GW
   其中：
     光伏：150-180GW
     风电：50-70GW
   
   绿发电力受益：
   依托大基地项目，年均新增装机有望达到300-500万千瓦
   2030年装机目标：4000-5000万千瓦
   装机规模从1845万千瓦 → 4000-5000万千瓦
   ```

2. **大基地项目陆续投产**
   ```
   大基地项目特点：
   规模大：单个项目100-200万千瓦
   成本低：资本金IRR 7%+
   消纳好：配套特高压外送通道
   
   绿发电力大基地布局：
   已获取指标：30.3GW（新疆为主）
   在建项目：918万千瓦
   2026-2027年投产：约500-600万千瓦
   ```

3. **特高压消纳改善**
   ```
   "疆电外送"通道建设：
   第三、四通道2025年投产
   外送能力提升至33GW+
   新疆新能源消纳能力大幅改善
   
   对绿发电力影响：
   利用小时数提升
   弃光弃风率下降
   发电量和收入增长
   ```

4. **"算电协同"新机遇**
   ```
   绿电+算力融合：
   甘肃、青海、新疆设立区域公司
   对接数据中心绿电需求
   绿电直供降低数据中心成本30-50%
   
   绿发电力布局：
   西北地区低电价绿电资源丰富
   推动与数据中心的绿电交易合作
   新增收入来源
   ```

5. **转型红利释放**
   ```
   转型时间线：
   2021年：完成从房地产向新能源转型
   2022-2023年：项目储备期
   2024年：装机爆发元年（+228.52%）
   2025-2027年：在建项目陆续投产，业绩持续释放
   
   转型优势：
   央企背景融资成本低
   集团协同支持力度大
   轻装上阵无历史包袱
   ```

### 催化剂

- 在建项目投产超预期
- 新疆大基地项目加速落地
- 特高压消纳通道提前投运
- 绿电交易价格上涨
- 算电协同业务签约

## 六、风险提示

| 风险类型 | 具体内容 | 影响程度 |
|----------|----------|----------|
| ⚠️ **市场电价下行风险** | 西北地区电力市场竞争激烈，电价可能下降，影响收益率 | 中 |
| ⚠️ **风光消纳不足风险** | 特高压建设进度不及预期，弃光弃风率上升 | 中 |
| ⚠️ **电费补贴拖欠风险** | 存量项目补贴拖欠，影响现金流 | 低 |
| ⚠️ **在建项目延期风险** | 大基地项目建设周期长，可能延期投产 | 中 |
| ⚠️ **资产周转率低** | 在建工程占比46.20%，短期内ROE偏低 | 中 |
| ⚠️ **融资成本上升** | 大规模扩张需要融资，利率上行影响盈利 | 低 |
| ⚠️ **分红较低** | 分红率20-30%，股息率2-3%，不适合高分红偏好投资者 | - |

**应对措施**：
- 特高压"疆电外送"通道建设加速，改善消纳
- 大基地项目成本优势明显，IRR 7%+具备竞争力
- 央企背景融资成本低（约3-4%）
- 集团协同支持，现金流有保障
