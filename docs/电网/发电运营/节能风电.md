## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中节能风力发电股份有限公司 |
| 证券代码 | 601016 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2006年 |
| 上市时间 | 2010年12月 |
| 所属行业 | 电力 / 风力发电 |
| 市场地位 | 纯风电运营商，央企背景 |

**公司概况**：
节能风电是中国节能环保集团旗下专业从事风力发电项目投资、建设及运营的企业，是A股为数不多的纯风电运营商。公司控股股东为中国节能环保集团（央企），截至2025年底，公司风电装机规模约9GW，项目分布在内蒙古、甘肃、河北、云南、贵州等风资源丰富地区。

## 二、主营业务

### 核心产品/服务

| 业务类型 | 说明 | 应用领域 |
|----------|------|----------|
| **风力发电** | 陆上风电场运营（主要） | 清洁能源供应 |
| **海上风电** | 海上风电场投资建设 | 清洁能源供应 |
| **电力销售** | 向电网公司销售电力 | 电力市场 |
| **绿证交易** | 绿色电力证书交易 | 碳交易市场 |

### 业务模式

**纯运营模式**：
```
投资建设风电场 → 运营发电 → 卖电给电网 → 获得电费收入
                  ↓
            稳定现金流 → 高分红
```

**收入来源**：
- 电费收入：95%+（主要）
- 补贴收入：逐步退坡（存量项目）
- 绿证收入：约5%

**vs 设备制造商**：

| 维度 | 节能风电（运营） | 金风科技（制造） |
|------|----------------|----------------|
| 业务模式 | 运营风电场 | 卖风机设备 |
| 收入来源 | 电费收入 | 设备销售 |
| 毛利率 | 60%+ | 15-20% |
| 现金流 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 成长性 | 稳定（+27%） | 波动大 |
| 分红 | 高分红（50%） | 分红率低 |

### 技术路线/核心技术

- **风电场选址**：精准风资源评估，确保发电量
- **设备选型**：选择金风、明阳等头部品牌风机
- **运维管理**：智能化运维，降低故障率
- **电力交易**：参与电力市场化交易，优化收益

**核心能力**：
项目选址能力、建设管理能力、运维管理能力、融资能力（央企背景融资成本低）

## 五、投资亮点

### 核心亮点

- ✅ **高分红稳定收益**：分红率50%，股息率4-5%，类似"高股息银行股"
- ✅ **央企背景安全性高**：中国节能集团100%控股
- ✅ **现金流充沛**：经营性现金流覆盖投资，资本支出可控
- ✅ **受益风电装机高峰**："十五五"年均新增装机120-130GW
- ✅ **估值合理**：PE(2026E) 13倍，PEG 0.48

### 成长驱动因素

1. **"十五五"风电装机高峰**
   ```
   "十五五"风电装机目标（2026-2030）：
   年均新增装机：120-130GW
   其中：
     陆风：110-120GW
     海风：10-15GW
   
   节能风电受益：
   年均新增装机：1.5-2GW
   2026-2030年总新增：7.5-10GW
   装机规模从9GW → 16-19GW
   ```

2. **海风项目贡献增量**
   ```
   海风特点：
   发电量更高：利用小时数4000+（陆风2500）
   电价更高：约0.5-0.6元/kWh（陆风0.4-0.45元）
   
   节能风电海风布局：
   储备项目：约2GW
   2026-2027年投产：约1GW
   海风占比提升至10%+
   ```

3. **电力市场化改革**
   ```
   现货市场试点：
   参与电力现货交易
   峰谷价差套利
   预计电价提升5-10%
   ```

4. **绿证收益增加**
   ```
   绿证价格上涨：
   2024年：50-80元/张
   2026年预期：80-120元/张
   
   绿证收入占比：
   2024年：2%
   2026年预期：5%
   ```

### 催化剂

- 风电装机目标上调
- 海风项目并网投产
- 电力市场化改革加速
- 绿证价格上涨
- 分红比例提升

### 目标价测算

**基于股息贴现模型（DDM）**：

```

股息率要求：4-5%

合理价格区间：
股息率5%：0.17 ÷ 5% = 3.4元
股息率4.5%：0.17 ÷ 4.5% = 3.8元
股息率4%：0.17 ÷ 4% = 4.25元

当前股价约4.0元，处于合理区间中枢
```
