## 一、投资主题概述

### 为什么是"算电协同"？

> **2026年3月政策催化**："算电协同"首次写入《政府工作报告》，上升为国家战略

**核心逻辑**：
```
AI算力需求爆发 → 数据中心能耗激增 → 电力基础设施瓶颈
                  ↓
            算电协同（政策解决方案）
                  ↓
    三大投资主线：
    1. 电网设备升级（远距离输电+末端配电）
    2. 电力运营服务（虚拟电厂+绿电直供）
    3. 数据中心基础设施（IDC+液冷+储能）
```

**市场规模**：
- 十五五电网投资：4万亿元
- 数据中心市场（2030年）：1.2万亿元
- 虚拟电厂调节能力（2030年）：5000万千瓦

**时间窗口**：
- 短期催化（6-12个月）：订单签约、项目落地
- 中期兑现（1-3年）：业绩增长、市占率提升
- 长期空间（3-5年）：新业态成熟、商业模式验证

## 六、风险提示与应对

### 系统性风险

**1. 政策落地不及预期**
- **风险**：虚拟电厂商业化进度慢、智算中心建设放缓
- **影响**：协鑫能科、科华数据、南网数字
- **应对**：
  - 降低虚拟电厂标的仓位（协鑫能科20% → 15%）
  - 增加传统电网设备仓位（特变电工15% → 25%）
  - 密切跟踪各地试点政策出台情况

**2. 宏观经济下行**
- **风险**：电网投资不及预期、数据中心建设延期
- **影响**：全部标的
- **应对**：
  - 整体降低仓位至50-60%
  - 优先持有在手订单饱满的标的（特变电工、国电南瑞）
  - 清仓弹性组合（伊戈尔、南网数字）

**3. 估值系统性回调**
- **风险**：市场风格切换、资金流出成长股
- **影响**：高估值标的（伊戈尔、南网数字）
- **应对**：
  - 及时止盈高估值标的
  - 转向低估值标的（特变电工）
  - 保持40-50%仓位等待机会

### 个股风险

**协鑫能科**：
- ⚠️ 虚拟电厂商业化不及预期 → 降低仓位至20%
- ⚠️ 辅助服务市场规则变化 → 等待规则明确后加仓

**科华数据**：
- ⚠️ 数据中心建设放缓 → 降低仓位至20%
- ⚠️ 毛利率下降 → 及时止盈
- ⚠️ 竞争加剧 → 关注订单价格变化

**特变电工**：
- ⚠️ 多晶硅持续低迷 → 已充分反映在估值中，无需担心
- ⚠️ 海外订单交付风险 → 跟踪海外收入确认进度

**国电南瑞**：
- ⚠️ 估值偏高 → 等待回调至PE 22x以下
- ⚠️ 定价权有限 → 关注毛利率变化
