## 📊 投资价值综合评分模型

### 评分维度（5个维度）

1. **PEG估值** (25分)：PE ÷ 业绩增速，PEG<0.5得25分，>2得0分
2. **订单可见性** (25分)：在手订单/年营收比例，>2年得25分
3. **竞争力** (20分)：市占率、技术壁垒、客户粘性
4. **成长空间** (20分)：行业天花板、公司成长潜力
5. **安全边际** (10分)：估值vs历史分位、下跌空间

## 💡 深度分析：为什么这样排序？

### 🥇 NO.1: 阳光电源（95分）

- **PEG 0.40**：PE 22.65 ÷ 增速56% = 极致性价比
- **双龙头地位**：储能逆变器全球第一（30%）+ 光伏逆变器全球第一
- **订单爆发**：储能2025年增速127%，2026年全球储能市场40-50%增长
- **盈利能力**：毛利率34.88%，ROE 35.52%
- **确定性**：欧洲市占率30%，地缘政治受益明确

**投资逻辑**：储能是未来5-10年确定性最高的赛道，阳光电源是绝对龙头

## 🚨 被高估的陷阱（看起来PE低，实则有坑）

### 三一重工（PE 20.79，排名倒数）

**为什么PE最低，我却不推荐？**

1. **主业是工程机械**：风电只是子公司三一重能
2. **工程机械周期性强**：2025年挖掘机销量下滑，主业承压
3. **风电业务占比小**：三一重能2025年营收274亿，三一重工总营收1000亿+，占比不到30%
4. **估值陷阱**：PE 20.79是整体估值，工程机械拖累，风电业务估值被掩盖
5. **不纯粹**：想投风电，为什么要买个工程机械公司？

**投资结论**：如果看好风电，直接买明阳智能、大金重工、东方电缆，不要绕弯子

---

### 南网能源（PE 87.96，排名倒数）

**为什么PE这么高？**

1. **2024年亏损**：基数低导致PE失真
2. **虚拟电厂业务不成熟**：商业模式还在探索期
3. **盈利能力弱**：ROE低，现金流差
4. **不如协鑫能科**：同样是虚拟电厂，协鑫能科更纯粹

---

### 保变电气（PE 169.03，排名倒数）

**极高PE的背后：**

1. **业绩基数极低**：2024年净利润不到2亿
2. **盈利能力弱**：毛利率低，竞争力一般
3. **没有技术壁垒**：变压器行业，特变电工、中国西电都比它强
4. **估值修复空间有限**：即使业绩翻倍，PE仍在80+
