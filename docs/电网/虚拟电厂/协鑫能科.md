## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 协鑫能源科技股份有限公司 |
| 证券代码 | 002015 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2004年 |
| 所属行业 | 能源服务、虚拟电厂 |
| 市场地位 | 虚拟电厂龙头、算电协同先行者 |

**公司概况**：
协鑫能科是清洁能源综合服务商，采用"能源资产+能源服务"双轮驱动模式。公司积极布局虚拟电厂、算电协同等新赛道，已投运超千P算力，推动能源资产整合与智能调度。公司是AI用电需求的受益标的。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **清洁能源发电** | 分布式光伏、风电、储能 | 清洁能源供应 |
| **虚拟电厂** | 需求响应、负荷聚合 | 电网调峰 |
| **算电协同** | 智能算力中心 | AI算力服务 |
| **能源服务** | 节能、售电、绿电交易 | 企业能源管理 |

### 业务模式

公司采用"能源资产+能源服务"模式，运营清洁能源资产，同时提供虚拟电厂、算电协同等能源服务。

### 技术路线/核心技术

- **虚拟电厂技术**：需求响应、负荷聚合
- **算电协同**：智能算力中心运营
- **储能技术**：电网侧储能项目
- **AI调度**：与蚂蚁数科合作开发智能调度系统

**技术优势**：
公司在虚拟电厂、算电协同等新兴领域布局领先，与蚂蚁数科合作开发AI智能调度系统，技术壁垒逐步建立。

## 四、行业地位与产业链定位

### 产业链定位

```
能源服务产业链：
能源资产 → 能源服务 → 【协鑫能科】→ 电网/用户 → 终端应用
                              ↑
                        虚拟电厂龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 虚拟电厂细分领域领先 |
| **竞争优势** | 能源资产+服务双轮驱动，算电协同先行 |
| **客户结构** | 电网公司、企业用户、AI算力客户 |
| **技术合作** | 与蚂蚁数科合作AI调度 |

### 竞争格局

**主要竞争对手**：
- 国电南瑞：电网自动化龙头
- 中国能建：电力工程龙头
- 其他虚拟电厂运营商

**公司优势**：
虚拟电厂布局领先、算电协同先行、与蚂蚁数科合作、能源资产规模大。

## 五、投资亮点

### 核心亮点

- ✅ **虚拟电厂龙头**：需求响应、负荷聚合领先
- ✅ **算电协同先行**：已投运超千P算力，AI用电受益
- ✅ **蚂蚁数科合作**：AI智能调度系统开发
- ✅ **业绩高增长**：2025年净利润+26%
- ✅ **储能布局**：电网侧储能项目规模达70万千瓦

### 成长驱动因素

1. **AI用电需求爆发**
   - AI数据中心用电需求激增
   - 算电协同业务快速增长

2. **虚拟电厂发展**
   - 电网调峰需求增加
   - 虚拟电厂政策支持

3. **储能业务增长**
   - 储能装机快速增长
   - 公司储能项目规模大

### 催化剂

- 算电协同大单签约
- 虚拟电厂政策出台
- AI调度系统落地
