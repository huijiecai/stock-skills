**公司概况**：
南网能源是中国南方电网有限责任公司的控股国有上市企业，国内领先的综合能源供应商和综合能源服务商。公司主要从事节能服务和综合能源服务，为客户提供诊断、设计、改造、项目投资及运营维护等一站式综合节能服务。业务覆盖工业节能、建筑节能、光伏发电、风电、余热余压利用、燃气分布式能源等多个领域。

## 二、主营业务

### 核心产品/服务

| 业务类型 | 说明 | 应用领域 |
|----------|------|----------|
| **工业节能** | 为工业企业提供能效提升、余热余压利用等服务 | 钢铁、化工、建材等高耗能行业 |
| **建筑节能** | 楼宇节能改造、中央空调系统优化、绿色建筑 | 商业综合体、办公楼、医院、学校 |
| **光伏发电** | 分布式光伏电站投资建设运营 | 工业厂房、商业屋顶 |
| **生物质综合利用** | 生物质发电、农林废弃物资源化利用 | 农业、林业 |
| **综合能源服务** | 冷热电气蒸汽一体化供应、虚拟电厂、能源托管 | 园区、数据中心、工业企业 |

### 业务模式

**合同能源管理（EMC）模式**：
```
诊断客户能耗 → 设计节能方案 → 投资建设改造 → 运营维护 → 
按节能效益分成收费（客户无需前期投资）
```

**收入来源**：
- 节能服务收入：85.27%（主要）
- 综合资源利用：14.70%
- 按节能量或固定电费差价收费

**vs 传统电力公司**：

| 维度 | 南网能源（综合能源服务） | 传统电力公司 |
|------|---------------------|------------|
| 业务模式 | 节能服务+能源托管 | 发电+输配电 |
| 收入来源 | 节能分成+能源供应 | 卖电 |
| 毛利率 | 36.71% | 20-30% |
| 客户关系 | 深度绑定（长期合同） | 浅层（用多少卖多少） |
| 成长驱动 | "双碳"政策+数字化 | 装机规模增长 |

### 技术路线/核心技术

**核心能力**：
- **节能诊断能力**：国家发改委备案、工信部推荐的节能服务公司
- **能源系统集成能力**：冷热电气蒸汽一体化供应
- **虚拟电厂技术**：2025年组建数字服务公司，聚焦虚拟电厂、电力交易
- **分布式能源运营**：光伏、风电、燃气分布式能源站

**技术优势**：
- 南方电网技术支持，能源管理经验丰富
- 国家"双百企业"，政策支持力度大
- 装机规模308.46万千瓦（2024年底），同比增长32%

## 四、行业地位与产业链定位

### 产业链定位

```
综合能源服务产业链：
上游：能源设备供应商（光伏组件、空调系统、余热设备）
  ↓
中游：【南网能源】综合能源服务商
  ├─ 节能诊断与方案设计
  ├─ 能源系统投资建设
  ├─ 运营维护与能源托管
  └─ 虚拟电厂调度
  ↓
下游：终端用户（工业企业、商业建筑、园区）
  ↓
终端价值：降低能耗成本、实现碳减排目标
```

## 五、投资亮点

### 核心亮点

- ✅ **业绩扭亏反转**：2025年前三季度净利润3.4亿元（同比+125.1%），消除2024年减值包袱
- ✅ **虚拟电厂+算电协同双驱动**：2025年组建数字服务公司，虚拟电厂业务实现突破
- ✅ **央企背景支持**：南方电网拟增持1.5-3亿元，彰显长期投资价值
- ✅ **政策红利显著**："算电协同"首次写入2026年政府工作报告，综合能源服务迎来黄金期
- ✅ **现金流优秀**：2024年经营性现金流14.28亿元，同比+38.62%，业务造血能力强
- ✅ **毛利率持续提升**：36.71%，5年连续上涨，盈利能力改善

### 成长驱动因素

1. **"双碳"政策长期驱动**
   ```
   中国碳达峰碳中和目标：
   2030年：碳达峰
   2060年：碳中和
   
   节能服务市场规模：
   2024年：约6000亿元
   2030年目标：1万亿元+
   
   南网能源受益：
   工业节能需求爆发（高耗能行业节能改造）
   建筑节能市场扩容（绿色建筑标准提升）
   分布式能源渗透率提升
   ```

2. **虚拟电厂业务突破**
   ```
   虚拟电厂市场前景：
   国务院目标：2027年调节能力2000万千瓦，2030年5000万千瓦
   南网区域市场潜力：约占全国1/3
   
   南网能源布局：
   2025年组建数字服务公司
   聚焦虚拟电厂、电力交易等数字化服务
   虚拟电厂业务已实现突破
   
   商业模式：
   类似协鑫能科，通过调度用户侧资源参与辅助服务市场
   获得调峰调频收益+电力交易收益
   ```

3. **算电协同新机遇**
   ```
   "算电协同"写入2026年政府工作报告：
   推动算力基础设施与电力基础设施协同发展
   
   南网能源优势：
   综合能源服务经验丰富
   可为数据中心提供：
     - 绿电直供（降低电费30-50%）
     - 余热余压利用（提高能效）
     - 冷热电气蒸汽一体化供应
     - 虚拟电厂调度（参与电网调节）
   
   市场空间：
   南方五省区数据中心建设加速
   AI算力需求驱动配套能源服务
   ```

4. **装机规模快速增长**
   ```
   装机规模增长：
   2024年底：308.46万千瓦（同比+32%）
   在建项目：储备充足
   
   增长驱动：
   分布式光伏爆发
   工业企业自建光伏需求
   建筑光伏一体化（BIPV）
   ```

5. **业务结构优化**
   ```
   转型方向：
   从单一投资向"投资+服务"模式升级
   业务结构从"粗放投资"向"精益投资+高端服务"转变
   
   盈利能力提升：
   毛利率从2019年30%+ → 2024年36.71%
   净利率（剔除减值）提升至17%+
   轻资产服务占比提升
   ```

### 催化剂

- 2025年业绩预告超预期（净利润3.0-3.6亿元）
- 虚拟电厂项目落地签约
- 算电协同业务订单公告
- 南方电网完成增持
- 南方五省区综合能源服务政策出台
