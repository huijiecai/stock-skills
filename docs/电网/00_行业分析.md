```

【环节1：发电侧】 产生电能

    ↓
  [远距离输送几百上千公里]
    ↓
  [电压：500kV/1000kV]
    ↓

【环节3：配电】 降压→分配到用户
    ↓
  • 降压设备：伊戈尔（变压器：500kV→380V）
  • 末端配电：正泰电器（低压开关）、百利电气（配电柜）
    ↓
  [配电到工厂、数据中心、小区...]
    ↓
  [电压：220V/380V]
    ↓

【环节4：电网调度】 全程监控与智能控制
    ↓
  • 省级调度：国电南瑞（电网调度大脑）
  • 区域平台：南网数字（南网数字化）、国电南自（区域调度）
  • 配网自动化：积成电子（末端控制）
  • 用电计量：三星医疗（智能电表）

【环节5：工程建设】 建设所有基础设施
    ↓
  • 总承包：中国能建（电站+线路建设）
```

**⚡ 什么是虚拟电厂？**

- **传统电厂**：电网缺电 → 电厂多发电（烧煤、烧气）
- **虚拟电厂**：电网缺电 → 让用户少用电（如暂停数据中心AI训练）
- **效果相同**：都能缓解电网供需矛盾，但虚拟电厂更灵活、成本更低
- **投资热点**：2027年目标2000万千瓦，2030年目标5000万千瓦

### "算电协同"如何串联产业链？

```
西部绿电（发电运营）→ 特高压输送（输变电设备）→ 
东部数据中心配电（配电与电源）→ AI智能调度（电网自动化+虚拟电厂）
```

**投资主线**：

- 主线1：**远距离电力调配** → 输变电设备（特变电工、中国西电）
- 主线2：**数据中心配电升级** → 配电与电源（伊戈尔、正泰电器）
- 主线3：**虚拟电厂调度** → 虚拟电厂（协鑫能科）+ 电网自动化（国电南瑞）
- 主线4：**绿电直供** → 发电运营（豫能控股、内蒙华电）

## 一、行业定义与范围

### 是什么？

> **电网设备 = 电力系统的"血管与神经"**
>
> 电网设备涵盖发电、输电、变电、配电、用电全环节的设备制造，是电力系统的核心基础设施，支撑能源转型与智能电网建设。


| 细分领域      | 说明              | 典型应用            |
| --------- | --------------- | --------------- |
| **电网自动化** | 电网调度、保护、监控系统    | 智能调度、继电保护、配网自动化 |
| **输变电设备** | 变压器、开关、GIS等一次设备 | 特高压输电、变电站建设     |
| **电缆线缆**  | 电力电缆、特种线缆       | 输电线路、海底电缆       |
| **低压电器**  | 低压开关、配电设备       | 建筑配电、工业用电       |
| **电力工程**  | 电力设计、施工、运营服务    | 电站建设、能源服务       |
| **电力电子**  | 变频器、电源、储能设备     | 数据中心供电、新能源配套    |


### 产业链定位

```
上游：原材料（铜、铝、硅钢、绝缘材料）
    ↓
中游：电网设备制造（本行业）
    ├─ 一次设备：变压器、开关、电缆
    ├─ 二次设备：自动化、保护、监控
    └─ 系统集成：电力工程、运维服务
    ↓
下游：电网公司（国家电网、南方电网）
    ↓
终端应用：工业用电、居民用电、新能源并网
```

**说明**：电网设备行业处于电力产业链中游，是连接上游原材料与下游电网公司的核心环节，技术壁垒高、客户粘性强。

## 三、🔥 算电协同：电网行业新增长引擎

> **重大政策催化**：2026年3月，"算电协同"首次写入《政府工作报告》，作为新基建工程的重点方向，上升为国家战略。

### 什么是算电协同？

> **算电协同 = 算力基础设施与电力基础设施的深度融合**

```
算电协同的两大维度：

1. 以电优算（Supply Side）
   - 绿电直供：低电价区域建设智算中心
   - 动态调度：根据电价波动灵活调配算力
   - 能效优化：提升数据中心PUE，降低能耗

2. 以算调电（Demand Side）
   - 削峰填谷：算力负载作为"虚拟电厂"调节电网
   - 需求响应：参与辅助服务市场获取收益
   - 储能协同：算力中心+储能实现能源自治
```

### 投资机遇分析

#### 主线一：电网设备升级

**1. 特高压输电（远距离电力调配）**

**受益逻辑**：

- 西部绿电（风光）输送至东部算力中心
- 跨省区电力调度需求激增
- "十五五"特高压投资加码

**核心标的**：

- **特变电工（600089）**：在手订单800亿+，排产至2028年
- **中国西电（601179）**：在手订单420亿，特高压全产业链
- **平高电气（600312）**：开关设备龙头

**2. 数据中心配电设备（末端配电）**

**受益逻辑**：

- AI服务器功耗暴增（单机柜10-20kW → 50-100kW）
- 传统UPS向高压直流（HVDC）升级
- 液冷、储能系统配套需求

**核心标的**：

- **科华数据（002335）**：UPS+液冷+预制舱龙头，市占率第一
- **科士达（002518）**：光储充一体化，数据中心电源前三
- **伊戈尔（002922）**：数据中心变压器+电源双轮驱动

**3. 智能配电网（调度与自动化）**

**受益逻辑**：

- 虚拟电厂技术应用，算力负载参与电网调度
- 电网数字化、AI调度需求
- 配网自动化率提升

**核心标的**：

- **国电南瑞（600406）**：电网自动化绝对龙头，市占率75%+
- **南网数字（301638）**：南方电网数字化平台，4天涨近50%
- **四方股份（601126）**：虚拟电厂调度系统

#### 主线二：电力运营与服务

**1. 虚拟电厂运营商**

**受益逻辑**：

- 国务院目标：2027年调节能力2000万千瓦，2030年5000万千瓦
- 数据中心作为"可控负荷"参与辅助服务市场
- 商业模式清晰，政策补贴支持

**核心标的**：

- **协鑫能科（002015）**：可调负荷100万千瓦，"聚星"平台+AI大模型
- **金开新能（600821）**：绿电直供+虚拟电厂双驱动
- **同力天启（未上市）**：储能+虚拟电厂解决方案

**2. 绿电运营商（低成本电力）**

**受益逻辑**：

- 数据中心向西部、北部低电价区域迁移
- 绿电直供降低数据中心电费30-50%
- "东数西算"枢纽节点布局

**核心标的**：

- **豫能控股（001896）**：火电+算力双轮，参与数据中心建设
- **内蒙华电（600863）**：蒙西电网，绿电资源丰富
- **大唐发电（601991）**：央企新能源平台

#### 主线三：数据中心基础设施

**受益逻辑**：

- AI算力需求驱动IDC扩建
- 液冷、高密度机柜渗透率提升
- REITs模式加速资产轮动

**核心标的**：

- **润泽科技（300442）**：园区级数据中心龙头，液冷技术领先
- **光环新网（300383）**：老牌IDC运营商，8.2万机柜
- **数据港（603881）**：深度绑定阿里云

### 全球对比：美国电网升级的启示

**美国困境**：

- 数据中心用电占全球45%（中国2倍）

- 产业链完整，成本低30%

**出口机遇**：

- 美国变压器进口依赖度80%
- 韩国晓星HICO获5.41亿美元订单
- 中国企业凭借性价比优势加速出海

### 关键环节分析

#### 电网自动化

- **作用**：电网"大脑"，实现智能调度、故障隔离、自愈控制
- **技术门槛**：极高（软件+硬件+算法）
- **价值占比**：15-20%
- **龙头企业**：国电南瑞

#### 特高压输变电

- **作用**：能源"高速公路"，实现远距离大容量输电
- **技术门槛**：极高（世界领先）
- **价值占比**：25-30%

- ⭐⭐⭐ 电网自动化（调度系统）：国电南瑞、南网数字
- ⭐⭐ 虚拟电厂运营（新模式）：协鑫能科、金开新能
- ⭐ 绿电运营（电价优势）：豫能控股、内蒙华电

**维度二：公司特征**

- ✅ 在手订单饱满（排产至2027-2028年）
- ✅ 市占率前三或细分龙头
- ✅ 技术壁垒高（特高压、液冷、AI调度等）
- ✅ 客户绑定度强（国网/南网/云厂商深度合作）
- ✅ 出海加速（海外收入占比提升）

**维度三：业绩弹性**

- 订单确认周期：特高压12-18个月，配电设备3-6个月
- 毛利率趋势：规模效应+技术溢价能否对冲原材料涨价
- ROE改善：重资产模式下的运营效率提升

### 风险因素


| 风险类型            | 具体描述             | 影响程度 |
| --------------- | ---------------- | ---- |
| ⚠️ **电网投资不及预期** | 宏观经济下行，电网投资增速放缓  | 中    |
| ⚠️ **原材料价格波动**  | 铜、铝、硅钢等原材料价格大幅波动 | 中    |

| ⚠️ **竞争加剧**     | 行业产能扩张，价格战风险     | 低    |
