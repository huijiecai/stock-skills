## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 宁波三星医疗电气股份有限公司 |
| 证券代码 | 601567 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1986年 |
| 上市时间 | 2011年 |
| 所属行业 | 智能电表、医疗 |
| 市场地位 | 智能电表龙头 |

**公司概况**：
三星医疗是中国智能电表领域的龙头企业，同时布局医疗业务，形成"电表+医疗"双主业格局。公司智能电表出口海外市场，国际化程度高。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **智能电表** | 智能电能表、用电采集系统 | 电网公司 |
| **医疗设备** | 医疗诊断设备 | 医疗机构 |

### 业务模式

公司采用"产品+服务"模式，为电网公司提供智能电表和用电采集系统，同时为医疗机构提供医疗设备。

### 技术优势

公司在智能电表领域技术领先，市场份额高，同时医疗业务提供多元化收入来源。
