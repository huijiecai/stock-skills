## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 国电南京自动化股份有限公司 |
| 证券代码 | 600268 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 1999年 |
| 所属行业 | 电网自动化 |
| 市场地位 | 电网自动化老牌企业 |

**公司概况**：
国电南自是中国电力系统自动化领域的老牌企业，是国家电网旗下重要的自动化设备供应商。公司专注于电网保护、自动化设备，技术积淀深厚，在继电保护等领域具有竞争优势。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **继电保护** | 电力系统保护设备 | 变电站、发电厂 |
| **电网自动化** | 调度自动化、配网自动化 | 电网公司 |
| **工业自动化** | 工业控制系统 | 工业企业 |

### 业务模式

公司主要采用"产品+服务"模式，为国家电网、南方电网提供自动化设备和系统集成服务。

### 技术路线/核心技术

- **继电保护技术**：技术积淀深厚
- **电网自动化**：调度、配网自动化
- **工业自动化**：工业控制系统

**技术优势**：
公司在继电保护领域技术积淀深厚，是电网自动化的老牌供应商。
