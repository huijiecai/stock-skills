## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 积成电子股份有限公司 |
| 证券代码 | 002339 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1994年 |
| 上市时间 | 2010年 |
| 所属行业 | 电网自动化 |
| 市场地位 | 配网自动化细分龙头 |

**公司概况**：
积成电子是中国电力系统自动化领域的企业，专注于配网自动化、智能电网等业务。公司在配网自动化细分领域具有竞争优势。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **配网自动化** | 配网调度、故障隔离 | 配电网 |
| **智能电网** | 智能电表、用电采集 | 电网公司 |
| **工业自动化** | 工业控制系统 | 工业企业 |

### 业务模式

公司主要采用"产品+服务"模式，为电网公司提供配网自动化设备和系统集成服务。

### 技术优势

公司在配网自动化领域技术积累深厚，是配网细分领域的领先企业。
