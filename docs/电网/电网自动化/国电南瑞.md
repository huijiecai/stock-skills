## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 国电南瑞科技股份有限公司 |
| 证券代码 | 600406 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年11月 |
| 上市时间 | 2011年12月 |
| 所属行业 | 电网自动化设备 |
| 市场地位 | 电网二次设备绝对龙头 |

**公司概况**：
国电南瑞是国家电网旗下南瑞集团的核心上市平台，源自国家电力主管部门的直属科研机构，具有鲜明的"科研院所改制"基因。公司是中国电网调度自动化、继电保护及柔性输电的技术引领者，被誉为国家电网的"大脑与神经系统"承包商。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **电网调度自动化** | 电网全局监控、智能分析、自动控制 | 国家电网、南方电网调度中心 |
| **继电保护系统** | 毫秒级故障检测和隔离 | 变电站、发电厂、配电网 |
| **柔性直流输电** | ±800kV-±1100kV换流站系统 | 特高压输电、新能源汇集 |
| **储能EMS系统** | 储能电池充放电优化控制 | 集中式储能、虚拟电厂 |
| **配电网自动化** | 配网故障自愈、自适应控制 | 中压配电、微电网 |

### 业务模式

公司主要采用"产品+服务"模式，为国家电网、南方电网提供电网自动化设备和系统集成服务。客户高度集中，国网、南网占比超90%。

### 技术路线/核心技术

- **电网调度技术**：国内领先，参与行业标准制定
- **继电保护技术**：市占率35%+，技术壁垒极高
- **柔性直流输电**：±1100kV技术全球首创，市占率100%
- **IGBT芯片**：自主研发进行中，向产业链上游延伸

**技术优势**：
公司拥有超1200项专利，参与制定80%以上的电网自动化行业标准。特高压柔直技术全球领先，是新型电力系统建设的核心参与者。

## 四、行业地位与产业链定位

### 产业链定位

```
电网产业链：
原材料（芯片、传感器） → 设备制造 → 【国电南瑞】→ 电网公司 → 终端用户
                                        ↑
                                  电网二次设备龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 电网自动化市占率35%+，行业排名第一 |
| **竞争优势** | 国网背景+技术领先+客户粘性强 |
| **客户结构** | 国家电网（80%+）、南方电网（10%+） |
| **供应商** | 芯片依赖进口，正在推进国产替代 |

### 竞争格局

**主要竞争对手**：
- 国电南自：保护类产品优势，规模较小
- 南网数字：南网系龙头，成长快速
- 积成电子：配网细分领先，综合竞争力弱

**公司优势**：
国网控股背景（56.9%）、技术迁移成本极高（调度系统切换风险大）、验证周期长（3-5年无故障纪录才可挂网）。

## 五、投资亮点

### 核心亮点

- ✅ **电网智能化最大受益者**：新能源渗透率提升→调度系统升级需求→公司订单增加
- ✅ **"十五五"4万亿投资确定性**：电网二次设备投资占比15-20%，对应市场空间800-1000亿元/年
- ✅ **海外市场加速拓展**：2025年海外收入同比增长135%，国际化程度提升
- ✅ **储能EMS快速成长**：储能装机增速40%+，该业务增速超20%
- ✅ **IGBT芯片国产化**：自主研发进行中，降低进口依赖

### 成长驱动因素

1. **电网投资加速**
   - "十五五"期间国网、南网计划投资超4万亿元
   - 特高压、智能配网、新能源接入为重点

2. **AI算力驱动电力需求**
   - AI数据中心用电需求激增
   - 推动智能配电网升级

3. **出海加速**
   - 欧美电网老旧升级需求旺盛
   - 一带一路沿线国家电网建设加速

### 催化剂

- 特高压项目核准加速
- 海外大单签约
- IGBT芯片产业化突破

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **增收不增利** | 营收增速高于利润增速，毛利率承压，成本传导能力受限 |
| ⚠️ **客户集中度高** | 国网、南网占比超90%，议价权不对等 |
| ⚠️ **电网投资政策风险** | 宏观经济下行可能导致电网投资调整 |
| ⚠️ **海外政策风险** | 地缘政治可能影响海外项目执行 |
| ⚠️ **技术迭代风险** | 云计算、AI对调度系统的重构需要持续跟进 |
