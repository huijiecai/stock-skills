## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国西电电气股份有限公司 |
| 证券代码 | 601179 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 央企背景国有控股企业 |
| 上市时间 | 2010年 |
| 所属行业 | 输变电设备制造 |
| 市场地位 | 特高压全产业链央企龙头 |

**公司概况**：
中国西电是中国最大的输配电设备制造企业和特高压输电技术的先行者，是国内唯一具备全电压等级输变电设备成套生产能力的企业。作为"国家队"成员，公司参与超过80%的国内特高压工程建设，主导中国80%的特高压标准制定。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **特高压变压器** | ±800kV-±1100kV换流变、升压变 | 特高压直流输电 |
| **GIS组合电器** | 气体绝缘开关设备 | 变电站、新能源汇集站 |
| **高压开关** | 断路器、隔离开关 | 输配电系统 |
| **换流阀** | 直流输电核心设备 | 柔性直流输电 |
| **新能源配套** | 直流变压器、柔直设备 | 新能源并网 |

### 业务模式

公司主要采用"产品+工程成套"模式，为国家电网、南方电网提供特高压全套设备和技术解决方案。央企背景确保订单稳定。

### 技术路线/核心技术

- **特高压技术**：全球唯一的特高压成套设备检测平台
- **柔性直流**：国产化突破，市场空间巨大
- **全电压等级**：从低压到±1100kV全覆盖

**技术优势**：
公司主导中国80%的特高压标准制定，拥有全球唯一的特高压成套设备检测平台，技术壁垒极高。

## 四、行业地位与产业链定位

### 产业链定位

```
特高压产业链：
原材料 → 设备制造 → 【中国西电】→ 电网公司 → 终端用户
                          ↑
                    特高压全产业链龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 特高压市场份额70%+，行业绝对龙头 |
| **竞争优势** | 央企背景+技术领先+全产业链能力 |
| **客户结构** | 国家电网（主要）、南方电网 |
| **供应商** | 钢材、铜材、绝缘材料等 |

### 竞争格局

**主要竞争对手**：
- 特变电工：变压器领域，成本优势明显
- 国电南瑞：柔直系统，技术领先
- 保变电气：变压器细分领域

**公司优势**：
特高压市场份额70%+、参与80%+国内特高压工程、央企背景订单稳定、全产业链成套能力业界领先。

## 五、投资亮点

### 核心亮点

- ✅ **特高压绝对龙头**：市场份额70%+，参与80%+国内特高压工程
- ✅ **4万亿电网投资受益**："十五五"期间特高压项目优先级最高
- ✅ **新能源并网需求**：风光储接入需要特高压远距离输送
- ✅ **央企平台价值重估**：国企改革深化，估值有望提升
- ✅ **盈利能力改善**：2025年净利润增速(19%)快于营收增速(11%)

### 成长驱动因素

1. **特高压建设加速**
   - "十五五"期间特高压项目加速核准
   - 公司特高压设备市占率70%+

2. **新能源并网刚需**
   - 风光储装机快速增长
   - 需要特高压远距离输送

3. **央企改革红利**
   - 国企改革深化
   - 估值体系有望重塑

### 催化剂

- 特高压项目核准加速
- 新能源大基地建设
- 央企改革政策出台
