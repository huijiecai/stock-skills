## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 特变电工股份有限公司 |
| 证券代码 | 600089 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1988年 |
| 上市时间 | 1997年 |
| 所属行业 | 输变电设备、多晶硅、新能源 |
| 市场地位 | 全球变压器龙头 |

**公司概况**：
特变电工是中国乃至全球能源装备行业的龙头企业，业务涵盖输变电、新能源、多晶硅和新材料四大产业，形成完整的能源产业链。公司是全球最大的变压器制造商，年产2.4亿kVA，远超ABB、西门子等国际巨头。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **变压器** | 电力变压器、换流变、特种变压器 | 特高压输电、数据中心、工业应用 |
| **开关设备** | 高压开关、GIS组合电器 | 变电站、风电场、水电站 |
| **多晶硅** | 高纯多晶硅、硅材料 | 光伏产业链上游 |
| **新能源** | 风光电项目开发、储能系统 | 清洁能源发电 |

### 业务模式

公司采用"纵向一体化"模式，从煤炭→火电→多晶硅→光伏→变压器→电网形成完整产业链，实现成本领先和协同效应。

### 技术路线/核心技术

- **特高压变压器**：±1100kV直流换流变全球首创，市占率100%
- **多晶硅工艺**：现金成本3万元/吨，全球最低
- **新能源技术**：风光储一体化解决方案

**技术优势**：
公司拥有超1200项专利，特高压变压器技术全球领先。多晶硅成本优势源于自有煤矿（120亿吨储量）和电力资源。

## 四、行业地位与产业链定位

### 产业链定位

```
能源产业链：
煤炭 → 火电 → 多晶硅 → 光伏 → 【特变电工】→ 电网 → 终端用户
                              ↑
                        变压器+新能源综合服务商
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 变压器全球市占率12-15%，排名第一 |
| **竞争优势** | 成本领先+产能最大+产业链一体化 |
| **客户结构** | 国家电网、南方电网、海外电网公司 |
| **供应商** | 自有资源（煤矿、电力），成本优势明显 |

### 竞争格局

**主要竞争对手**：
- 中国西电：特高压设备，央企背景
- ABB、西门子：国际巨头，技术领先但成本高
- 保变电气：变压器细分领域

**公司优势**：
产能全球最大（2.4亿kVA）、成本最低（多晶硅3万元/吨）、产业链最完整、国际化程度最高（海外收入35-40%）。

## 五、投资亮点

### 核心亮点

- ✅ **全球变压器龙头**：年产2.4亿kVA，远超ABB、西门子
- ✅ **多晶硅成本优势**：现金成本3万元/吨，全球最低，周期底部已过
- ✅ **特高压建设加速**：±1100kV换流变市占率100%，受益4万亿投资
- ✅ **AI数据中心新增长点**：专用变压器需求增长30%+
- ✅ **出海加速**：2025年海外签约增长65.91%，国际订单饱满

### 成长驱动因素

1. **特高压投资加速**
   - "十五五"期间特高压项目优先级最高
   - 公司特高压变压器市占率45%+

2. **多晶硅周期反转**
   - 2024年价格触底，2025-2026年供需改善
   - 成本优势公司盈利能力最强

3. **AI数据中心变压器**
   - 新兴细分市场，年增速30%+
   - 已进入主流数据中心供应链

### 催化剂

- 多晶硅价格回升
- 特高压大单签约
- 海外市场突破
