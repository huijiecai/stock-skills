## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广东顺钠电气股份有限公司 |
| 证券代码 | 000533 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1992年 |
| 上市时间 | 1994年 |
| 所属行业 | 输配电设备 |
| 市场地位 | 输配电设备成长企业 |

**公司概况**：
顺钠股份是一家专注于输配电设备的企业，产品涵盖变压器、开关设备等。公司业绩弹性大，订单增长快，是电网设备领域的成长型企业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **变压器** | 电力变压器、配电变压器 | 电网、工业 |
| **开关设备** | 高低压开关 | 变电站 |

### 业务模式

公司主要采用"产品+工程"模式，为电网公司提供输配电设备。

### 技术优势

公司在输配电设备领域有一定技术积累，订单增长快。
