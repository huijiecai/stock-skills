## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 大金重工股份有限公司 |
| 证券代码 | 002487.SZ |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001年 |
| 上市时间 | 2010年10月15日 |
| 所属行业 | 电气设备 - 风电设备 |
| 市场地位 | 全球海上风电基础结构龙头 |

**公司概况**：
大金重工是全球领先的海上风电基础结构供应商，专注于风电塔筒、海上风电单桩基础、导管架、浮式基础等产品的研发、生产与销售。公司是亚太区唯一向欧洲批量交付海上风电基础结构的供应商，海外收入占比超75%，是风电出海的核心标的。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **海上风电基础** | 单桩基础、导管架、浮式基础、过渡段 | 海上风电场建设 |
| **风电塔筒** | 陆上/海上风电塔筒 | 风力发电机组配套 |
| **海工装备** | 海工特种运输船、船舶设计建造 | 海上风电运输与施工 |
| **新能源发电** | 自持风电场运营 | 风力发电 |

### 业务模式

公司采用"产品+服务"一体化模式，向系统服务商转型：
- **装备制造**：海上风电基础结构、塔筒的生产销售
- **建造+运输+交付**：一站式海上风电装备解决方案
- **海工服务**：海工船舶运营、风电母港运营
- **新能源运营**：自持风电场发电收入

### 技术路线/核心技术

- **超大型单桩制造**：具备14.7MW+超大型单桩供应能力
- **浮式基础技术**：深远海浮式风电基础研发与制造
- **海工特种船舶**：自建特种运输船队，降低物流成本
- **欧洲标准认证**：产品通过欧洲多项认证，符合国际标准

**技术优势**：
亚太区唯一向欧洲批量交付海上风电基础结构的供应商，拥有完整的海外项目交付经验和本地化服务能力。

## 四、行业地位与产业链定位

### 产业链定位

```
海上风电产业链：
风机整机 → 基础结构 → 【大金重工】→ 海上风电场 → 电网并网
                  ↑
            单桩/导管架/浮式基础
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 欧洲海上风电基础结构市场约25-30% |
| **竞争优势** | 亚太区唯一向欧洲批量交付的供应商 |
| **客户结构** | 欧洲主流风电开发商（Ørsted、Vattenfall等） |
| **供应商** | 钢材等原材料成本占比高 |

### 竞争格局

**主要竞争对手**：
- **天顺风能（002531）**：国内塔筒龙头，海工业务快速追赶
- **泰胜风能**：海上风电塔筒和基础结构
- **欧洲本土企业**：Sif、Bladt等欧洲海工基础制造商

**公司优势**：
- 欧洲市场准入壁垒高，先发优势明显
- 自主船队降低物流成本
- 从制造向系统服务转型，附加值提升

## 五、核心投资逻辑

### 投资类型判断

| 类型 | 判断 | 说明 |
|------|------|------|
| **成长股** | ✅ 是 | 海外海上风电市场爆发，订单高增长 |
| **周期股** | ⚠️ 部分 | 受海上风电装机周期影响 |
| **出口股** | ✅ 是 | 海外收入占比超75%，出口导向 |

### 核心预期差分析

| 维度 | 市场预期（A） | 实际情况（B） | 预期差（B-A） |
|-----|-------------|-------------|--------------|
| 海外订单 | 增长放缓 | 在手订单超100亿，欧洲需求爆发 | ⭐⭐⭐ |
| 盈利能力 | 毛利率承压 | 出口毛利率81%，净利率17.87% | ⭐⭐⭐ |
| 业绩增速 | 2025年净利润约8亿 | 实际11.03亿，超预期 | ⭐⭐⭐ |
| 市场拓展 | 仅限欧洲 | 向系统服务商转型，业务多元化 | ⭐⭐ |

**预期差总分**：11个⭐（大幅超预期）

### 核心投资逻辑

**一句话总结**：
投的是海上风电出海龙头，欧洲市场需求爆发，公司在手订单充足，盈利能力超预期，是风电设备出海最纯标的。

**详细逻辑**：

1. **欧洲海风市场爆发**
   - 欧洲多国签署《汉堡宣言》，目标2050年北海100GW海上风电
   - 欧洲本土产能不足，中国厂商迎来出海窗口期
   - 英国零关税政策落地，利好中国供应商

2. **订单储备充足，业绩确定性强**
   - 在手订单超100亿元，集中在欧洲北海、波罗的海
   - 订单交付周期2-3年，业绩可见度高
   - 2026-2027年净利润预计16亿/20亿+

3. **盈利能力持续提升**
   - 出口业务毛利率81%，显著高于国内
   - 规模效应显现，净利率提升至17.87%
   - 向系统服务商转型，附加值提升

4. **竞争格局优，壁垒深厚**
   - 欧洲市场准入壁垒高，认证周期长
   - 亚太区唯一批量交付供应商，先发优势明显
   - 自主船队降低物流成本，形成差异化竞争

## 十一、最终判断
