## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 平高集团有限公司 |
| 证券代码 | 600312 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1970年 |
| 所属行业 | 电力设备 / 高压开关 |
| 市场地位 | 特高压GIS龙头，市占率36.4% |

**公司概况**：平高电气是中国高压开关设备市占率第一的企业，专注于高压、超高压、特高压输配电设备制造。公司在特高压GIS（气体绝缘封闭开关）领域占有绝对优势，产品广泛应用于特高压工程、电网升级改造、新能源并网等领域。

- ✅ **特高压GIS龙头地位**：市占率36.4%，国网、南网市场占有率双提升

- ✅ **订单饱满**：2024年确认14间隔1000kV GIS、97间隔750kV GIS
- ✅ **业绩高增长**：2025年Q1净利润同比+55.94%
- ✅ **出海加速**：中东订单增长200%，墨西哥等拉美市场增长285%

### 成长驱动

1. **"十五五"特高压建设加速**：2026-2030年15项特高压直流工程，GIS需求旺盛
2. **配网智能化**：配网自动化率从50%→90%，GIS向配网延伸
3. **新能源接入**：大型风光基地配套升压站，GIS需求增长
4. **出海空间广阔**：中东、拉美、东南亚市场快速增长
