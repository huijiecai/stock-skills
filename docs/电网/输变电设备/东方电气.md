## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 东方电气股份有限公司 |
| 证券代码 | 600875 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 1995年 |
| 所属行业 | 发电设备、输变电设备 |
| 市场地位 | 全球发电设备龙头 |

**公司概况**：
东方电气是中国最大的发电设备制造企业之一，是全球能源装备"国家队"成员。公司业务涵盖"六电六业"：水电、火电、核电、气电、风电、太阳能发电，以及工程贸易、现代制造服务、金融等。近年来积极拓展输变电设备业务，实现协同发展。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **水轮发电机组** | 大型水电设备 | 水力发电站 |
| **汽轮发电机组** | 火电、核电设备 | 火电厂、核电站 |
| **燃气轮机** | 燃气发电设备 | 燃气电站（国产化突破） |
| **风电机组** | 风力发电设备 | 陆上、海上风电场 |
| **输变电设备** | 变压器、开关设备 | 电网建设 |

### 业务模式

公司采用"设备制造+工程服务+投资运营"模式，为发电企业提供全套发电设备及解决方案。

### 技术路线/核心技术

- **水电技术**：国内领先，参与三峡等重大项目
- **燃气轮机**：实现国产化突破，出口哈萨克斯坦
- **核电技术**：核电设备制造能力强
- **输变电技术**：协同发电设备，拓展电网市场

**技术优势**：
公司在发电设备领域技术积累深厚，燃气轮机实现国产化突破，毛利率超30%。输变电业务与发电设备协同发展。

## 四、行业地位与产业链定位

### 产业链定位

```
发电产业链：
原材料 → 设备制造 → 【东方电气】→ 发电集团 → 电力供应
                              ↑
                        发电设备龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 发电设备国内市占率领先 |
| **竞争优势** | 技术领先+全系列产品+央企背景 |
| **客户结构** | 五大发电集团、国家电网 |
| **订单情况** | 订单饱满，排期至2027年 |

### 竞争格局

**主要竞争对手**：
- 上海电气：发电设备竞争者
- 哈尔滨电气：发电设备竞争者
- 国电南瑞：输变电设备竞争者

**公司优势**：
发电设备全系列覆盖、燃气轮机国产化突破、央企背景、订单饱满、输变电业务协同发展。

## 五、投资亮点

### 核心亮点

- ✅ **发电设备龙头**：水电、火电、核电、燃气轮机全覆盖
- ✅ **燃气轮机国产化**：实现突破，出口哈萨克斯坦，毛利率>30%
- ✅ **风电业务增长**：风电装机快速增长，公司受益
- ✅ **输变电协同**：拓展输变电业务，与发电设备协同
- ✅ **订单饱满**：订单排期至2027年，收入可见性强

### 成长驱动因素

1. **发电设备需求旺盛**
   - 新能源配套火电调峰需求
   - 燃气轮机国产替代

2. **风电业务增长**
   - 海上风电加速发展
   - 风电设备需求旺盛

3. **输变电业务拓展**
   - 电网投资加速
   - 与发电设备协同

### 催化剂

- 燃气轮机大单签约
- 风电装机超预期
- 输变电业务突破
