- ✅ **海外业务突破**：沙特、巴西等市场订单增长

1. **"十五五"特高压直流大建设**：预计"5直3交"开工，换流阀需求集中释放
2. **柔性直流技术爆发**：新能源并网、海上风电、城市背靠背互联需求旺盛
3. **自主可控突破**：IGBT、IGCT等核心器件国产化
4. **海外市场拓展**：中东、南美、东南亚市场空间广阔
