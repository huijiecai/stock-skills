## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 天顺风能（苏州）股份有限公司 |
| 证券代码 | 002531.SZ |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2005年 |
| 上市时间 | 2010年12月31日 |
| 所属行业 | 电气设备 - 风电设备 |
| 市场地位 | 国内风电塔筒龙头，海工新秀 |

**公司概况**：
天顺风能是国内风电塔筒行业龙头企业，正积极向海上风电领域转型。公司通过"新能源装备制造+零碳实业发展"双轮驱动战略，在国内七大沿海基地与德国基地建立了约80万吨的海工产能网络，是风电设备领域的重要参与者。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **风电塔筒** | 陆上/海上风电塔筒 | 风力发电机组配套 |
| **风电叶片** | 风电叶片及模具制造 | 风力发电机组配套 |
| **海工装备** | 导管架、单管桩等海工基础 | 海上风电场建设 |
| **零碳运营** | 自持新能源电站运营 | 风力/光伏发电 |

### 业务模式

公司采用"制造+运营"双轮驱动模式：
- **装备制造**：风电塔筒、叶片、海工基础的生产销售
- **零碳运营**：自持风电场、光伏电站的投资运营
- **海工服务**：海上风电基础结构制造与交付

### 技术路线/核心技术

- **大型塔筒制造**：具备大兆瓦风机塔筒制造能力
- **叶片技术**：风电叶片设计与制造
- **海工基础**：导管架、单管桩制造技术
- **零碳技术**：新能源电站开发运营经验

**技术优势**：
国内塔筒行业龙头，拥有丰富的风电设备制造经验和客户资源，正快速拓展海工业务。

## 四、行业地位与产业链定位

### 产业链定位

```
海上风电产业链：
风机整机 → 塔筒/基础 → 【天顺风能】→ 海上风电场 → 电网并网
                  ↑
            塔筒/叶片/海工基础
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内塔筒市场前三 |
| **竞争优势** | 国内塔筒龙头，客户资源丰富 |
| **客户结构** | 金风科技、中广核、华能等头部企业 |
| **产能布局** | 七大沿海基地+德国基地，约80万吨海工产能 |

### 竞争格局

**主要竞争对手**：
- **大金重工（002487）**：海上风电基础龙头，海外布局领先
- **泰胜风能**：海上风电塔筒和基础结构
- **天能重工**：风电塔筒制造商

**公司优势**：
- 国内塔筒行业龙头，品牌知名度高
- 客户资源丰富，与头部主机厂合作紧密
- 产能布局完善，沿海基地覆盖主要市场

**公司劣势**：
- 海外业务占比低，出海进度落后
- 海工业务起步晚，竞争力待验证
- 2024年业绩大幅下滑，盈利能力承压

## 五、核心投资逻辑

### 投资类型判断

| 类型 | 判断 | 说明 |
|------|------|------|
| **周期股** | ✅ 是 | 风电塔筒行业周期性强，2024年处于周期底部 |
| **困境反转** | ✅ 是 | 业绩触底，海工业务带来新增长点 |

### 核心预期差分析

| 维度 | 市场预期（A） | 实际情况（B） | 预期差（B-A） |
|-----|-------------|-------------|--------------|
| 业绩预期 | 2025年继续低迷 | 海工订单突破，业绩有望触底回升 | ⭐⭐ |
| 海工业务 | 竞争力不足 | 新签8.7亿订单，产能布局完善 | ⭐⭐⭐ |
| 估值水平 | 估值偏高 | PB仅2.5倍，处于历史低位 | ⭐⭐⭐ |

**预期差总分**：10个⭐（略超预期）

### 核心投资逻辑

**一句话总结**：
投的是风电塔筒周期底部反转+海工业务第二增长曲线，当前估值处于历史低位，海工订单突破验证转型成效。

**详细逻辑**：

1. **塔筒业务周期底部**
   - 2024年行业周期调整，价格竞争激烈
   - 2025年海上风电装机回暖，塔筒需求回升
   - 公司塔筒产能充足，有望受益行业复苏

2. **海工业务突破**
   - 新签8.7亿元海工订单，客户包括金风科技、中广核
   - 七大沿海基地+德国基地，产能布局完善
   - 产品结构升级，专注高附加值导管架和单管桩

3. **零碳运营稳健**
   - 自持新能源电站提供稳定现金流
   - 对冲装备制造业务周期波动
   - 双轮驱动战略初见成效

4. **估值处于低位**
   - PB仅2.5倍，处于历史低位
   - 市值223亿，低于大金重工（514亿）
   - 困境反转标的，安全边际较高

## 十一、最终判断

### 核心结论

天顺风能是风电塔筒行业龙头，正积极向海上风电领域转型。2024年业绩触底，但海工业务获得突破，估值处于历史低位。作为困境反转标的，具备一定配置价值，但需关注海工业务进展和盈利能力恢复情况。

| 长期（1-2年） | 🟡 中性偏多 | 卫星配置，跟踪验证 |

1. **海工订单**：季度跟踪新增海工订单金额
