## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广东奥飞数据科技股份有限公司 |
| 证券代码 | 300738 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2004年9月27日 |
| 上市时间 | 2018年1月19日 |
| 所属行业 | 互联网数据中心（IDC） |
| 市场地位 | 华南地区领先的IDC服务商 |

**公司概况**：
奥飞数据是华南地区领先的IDC服务商，专注于数据中心服务，采取激进扩张策略，快速增加机柜规模。已拥有14座自建数据中心，机柜规模超过4.3万架，布局主要在北京、广州、深圳等核心城市。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **数据中心服务** | 服务器托管、租用、运维服务 | 互联网企业、金融机构 |
| **带宽租赁** | 提供网络带宽接入服务 | 企业客户 |
| **安全服务** | 网络安全防护、数据备份 | 金融、政府客户 |

### 业务模式

- **自建数据中心**：自建机房，提供机柜租赁服务
- **批发+零售**：同时服务大客户和中小客户
- **激进扩张**：快速投资建设新数据中心，追求规模增长

### 技术路线/核心技术

- **高密度机柜**：支持AI算力部署的高功率机柜
- **多线接入**：接入多家运营商网络，网络优势明显
- **绿色节能**：采用节能技术降低运营成本

**技术优势**：
- 自建数据中心，资产质量高
- 多运营商接入，网络稳定性强
- 快速扩张能力，规模增长快

## 四、行业地位与产业链定位

### 产业链定位

```
算力租赁产业链：
上游（芯片/服务器）→ 中游（IDC服务商）→ 【奥飞数据】→ 下游（互联网企业）→ 终端应用
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 华南地区IDC市场领先地位 |
| **竞争优势** | 自建数据中心，网络资源丰富 |
| **客户结构** | 互联网企业、金融机构 |
| **供应商** | 服务器厂商、网络设备供应商 |

### 竞争格局

**主要竞争对手**：
- 光环新网（300383）：北京地区领先，规模更大
- 数据港（603881）：阿里云深度绑定，定制化服务
- 润泽科技（300442）：园区级数据中心龙头

**公司优势**：
- 华南地区网络资源丰富
- 自建数据中心，资产质量高
- 扩张速度快，规模增长潜力大

## 五、投资亮点

### 核心亮点

- ✅ **激进扩张策略**：快速增加机柜规模，追求市场份额
- ✅ **自建数据中心**：资产质量高，长期价值明显
- ✅ **华南网络优势**：接入多家运营商，网络稳定性强
- ✅ **AI算力布局**：支持AI算力部署，受益大模型需求
- ✅ **新项目投产**：多个新数据中心项目即将交付

### 成长驱动因素

1. **数据中心扩产**
   - 多个新数据中心项目即将投产
   - 机柜规模持续增长

2. **AI算力需求**
   - 大模型训练和推理需求增长
   - 公司高功率机柜支持AI部署

3. **市场份额提升**
   - 激进扩张策略抢占市场
   - 规模效应逐步释放

### 催化剂

- 新数据中心项目投产
- 大客户合同签订
- AI算力业务突破
