## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 北京光环新网科技股份有限公司 |
| 证券代码 | 300383 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1999年1月27日 |
| 上市时间 | 2014年1月29日 |
| 所属行业 | 互联网数据中心（IDC）、云计算 |
| 市场地位 | 国内领先的IDC服务商，老牌数据中心运营商 |

**公司概况**：
光环新网是国内老牌IDC服务商，主营互联网数据中心业务和云计算服务，在全国范围内布局数据中心，机柜规模超过7.2万个，规划总规模超23万个，是北京地区重要的数据中心运营商。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **IDC及增值服务** | 服务器托管、租用、运维及网络接入服务 | 大型企业、互联网公司 |
| **云计算服务** | AWS云服务运营、公有云、私有云解决方案 | 企业数字化转型 |
| **算力租赁** | GPU算力资源租赁，规模超4000P | AI训练、推理部署 |

### 业务模式

- **IDC业务**：提供机房、机柜、电力、冷却等基础设施，客户自备服务器，按机柜月租收费
- **云计算业务**：与AWS合作运营云服务，提供公有云、私有云解决方案
- **算力租赁**：提供GPU算力资源，按算力规模或时间收费

### 技术路线/核心技术

- **数据中心技术**：高功率密度机柜设计，支持AI算力需求
- **绿色节能技术**：液冷技术应用，降低PUE值
- **智能运维平台**：自动化运维管理系统

**技术优势**：
- 北京地区稀缺的数据中心资源
- 多年积累的运维经验和客户资源
- 与AWS深度合作，云服务能力领先

## 四、行业地位与产业链定位

### 产业链定位

```
算力租赁产业链：
上游（芯片/服务器）→ 中游（IDC服务商）→ 【光环新网】→ 下游（互联网企业）→ 终端应用
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 北京地区IDC市场领先地位 |
| **竞争优势** | 稀缺的北京数据中心资源、AWS合作、规模优势 |
| **客户结构** | 互联网大厂、金融机构、政府客户 |
| **供应商** | 服务器厂商、网络设备供应商 |

### 竞争格局

**主要竞争对手**：
- 数据港（603881）：阿里云深度绑定，定制化服务
- 奥飞数据（300738）：激进扩张，快速增加机柜
- 润泽科技（300442）：园区级数据中心龙头

**公司优势**：
- 北京地区稀缺资源，牌照优势
- 与AWS合作，云服务能力领先
- 规模优势明显，机柜数量行业前列

## 五、投资亮点

### 核心亮点

- ✅ **稀缺资源优势**：北京地区数据中心资源稀缺，牌照壁垒高
- ✅ **规模领先**：已投产机柜超7.2万个，规划总规模超23万个
- ✅ **AWS合作**：与AWS深度合作，云服务能力领先
- ✅ **AI算力布局**：算力租赁规模超4000P，向智算转型
- ✅ **区位优势**：京津冀、海南、马来西亚等多地布局

### 成长驱动因素

1. **AI算力需求爆发**
   - 大模型训练和推理需求增长
   - 公司向智算中心转型，提供GPU算力服务

2. **数据中心扩产**
   - 天津宝坻等大型数据中心项目推进
   - 机柜规模持续增长

3. **云计算业务稳定**
   - AWS合作持续深化
   - 企业上云需求稳定

### 催化剂

- 天津宝坻数据中心项目交付
- AI算力租赁业务突破
- 与大客户签订长期合同
