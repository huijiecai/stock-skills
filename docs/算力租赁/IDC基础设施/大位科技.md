## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 大位数据科技（广东）股份有限公司 |
| 证券代码 | 600589 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997年 |
| 上市时间 | 2002年 |
| 所属行业 | 互联网数据中心（IDC）、算力租赁 |
| 市场地位 | 头部互联网IDC服务商，盈利修复中 |

**公司概况**：
大位科技是头部互联网IDC服务商，主营互联网数据中心与算力服务，重点布局京津冀地区的绿色高性能数据中心项目。2025年前三季度净利润3803万元，同比增长236.78%，盈利能力持续修复。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **IDC服务** | 服务器托管、租用、运维服务 | 互联网企业 |
| **算力租赁** | GPU算力资源租赁 | AI训练、推理部署 |
| **定制化数据中心** | 根据客户需求定制建设数据中心 | 大型互联网企业 |

### 业务模式

- **IDC服务**：提供机房、机柜、电力、冷却等基础设施
- **算力租赁**：子公司森华易腾提供高性能GPU算力资源
- **定制化服务**：根据客户需求定制数据中心解决方案

### 技术路线/核心技术

- **绿色数据中心**：张北榕泰云计算数据中心，绿色高标准
- **源网荷储一体化**：新能源与智能算力融合
- **高性能GPU算力**：满足AI和高性能计算需求

**技术优势**：
- 张北地区电力资源丰富，成本低
- 绿色数据中心技术，PUE优势
- 高性能GPU算力资源

## 四、行业地位与产业链定位

### 产业链定位

```
算力租赁产业链：
上游（芯片/服务器）→ 中游（IDC服务商）→ 【大位科技】→ 下游（互联网企业）→ 终端应用
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 头部互联网IDC服务商 |
| **竞争优势** | 张北地区资源优势、绿色数据中心 |
| **客户结构** | 互联网企业 |
| **供应商** | 服务器厂商、网络设备供应商 |

### 竞争格局

**主要竞争对手**：
- 光环新网（300383）：规模更大，北京地区领先
- 数据港（603881）：阿里云深度绑定
- 奥飞数据（300738）：华南地区领先

**公司优势**：
- 张北地区电力资源丰富，成本低
- 盈利能力修复，经营改善明显
- 绿色数据中心技术领先

## 五、投资亮点

### 核心亮点

- ✅ **盈利修复**：2025年净利润大幅增长，扭亏为盈
- ✅ **张北资源优势**：电力资源丰富，成本低
- ✅ **绿色数据中心**：符合碳中和政策方向
- ✅ **AI算力布局**：子公司提供GPU算力服务
- ✅ **东数西算受益**：张北项目符合国家战略

### 成长驱动因素

1. **张北数据中心投产**

   - 预计每年贡献约1.63亿元收入

2. **AI算力需求**
   - 子公司森华易腾提供GPU算力
   - AI训练和推理需求增长

3. **盈利能力修复**
   - 毛利率提升
   - 成本控制有效

### 催化剂

- 张北数据中心全面投产
- AI算力业务突破
- 新客户签约
