## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海数据港股份有限公司 |
| 证券代码 | 603881 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2009年11月18日 |
| 上市时间 | 2017年2月8日 |
| 所属行业 | 互联网数据中心（IDC） |
| 市场地位 | 国内领先的第三方数据中心运营商 |

**公司概况**：
数据港是国内领先的第三方数据中心运营商，专注于数据中心服务，深度绑定阿里云等互联网巨头，全国已建成35座数据中心，覆盖主要经济区域，平均PUE达1.21，绿色节能水平行业领先。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **数据中心服务** | 服务器托管、租用、运维服务 | 阿里云、互联网大厂 |
| **IDC增值服务** | 网络接入、安全防护、运维管理 | 企业客户 |
| **定制化数据中心** | 根据客户需求定制建设数据中心 | 大型互联网企业 |

### 业务模式

- **批发型IDC**：与大客户签订长期合同，定制化建设数据中心
- **零售型IDC**：提供标准化机柜租赁服务
- **深度绑定阿里云**：阿里云是公司最大客户，收入高度集中

### 技术路线/核心技术

- **绿色数据中心技术**：平均PUE达1.21，行业领先
- **液冷技术**：早期布局液冷技术，提升散热效率
- **智能运维平台**：自动化运维管理，降低运营成本

**技术优势**：
- 绿色节能技术领先，PUE行业最低之一
- 定制化服务能力强，深度绑定大客户
- 运维效率高，成本控制能力强

## 四、行业地位与产业链定位

### 产业链定位

```
算力租赁产业链：
上游（芯片/服务器）→ 中游（IDC服务商）→ 【数据港】→ 下游（阿里云等）→ 终端应用
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 第三方IDC运营商前列 |
| **竞争优势** | 深度绑定阿里云，绿色技术领先 |
| **客户结构** | 阿里云为主，高度集中 |
| **供应商** | 服务器厂商、网络设备供应商 |

### 竞争格局

**主要竞争对手**：
- 光环新网（300383）：北京地区领先，规模更大
- 奥飞数据（300738）：激进扩张，快速增加机柜
- 润泽科技（300442）：园区级数据中心龙头

**公司优势**：
- 深度绑定阿里云，订单稳定
- 绿色节能技术领先，PUE行业最低
- 定制化服务能力强，客户粘性高

## 五、投资亮点

### 核心亮点

- ✅ **阿里云深度绑定**：阿里云是最大客户，订单稳定，合同期长
- ✅ **绿色技术领先**：平均PUE 1.21，行业最低之一
- ✅ **盈利能力提升**：毛利率、净利率持续改善
- ✅ **规模扩张**：全国35座数据中心，持续扩产
- ✅ **东数西算受益**：数据中心布局符合国家战略

### 成长驱动因素

1. **阿里云业务增长**
   - 阿里云持续扩张，数据中心需求增加
   - 公司作为阿里云重要合作伙伴，受益明显

2. **AI算力需求**
   - 大模型训练和推理需求增长
   - 公司数据中心支持AI算力部署

3. **绿色节能优势**
   - 碳中和政策推动绿色数据中心发展
   - 公司PUE优势明显，政策受益

### 催化剂

- 与阿里云签订新的大额合同
- 新数据中心项目投产
- AI算力业务突破
