## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏利通电子股份有限公司 |
| 证券代码 | 603629 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2006年 |
| 上市时间 | 2018年12月 |
| 注册资本 | 2.5883亿股 |
| 所属行业 | 电子元器件、算力服务（转型中） |
| 市场地位 | 算力租赁行业龙头（33,000P规模第一） |

**公司概况**：
利通电子原为液晶显示领域精密金属结构件制造企业，2023年起战略转型AI算力租赁业务。通过与NVIDIA、超聚变等GPU服务器厂商合作，快速建立算力规模，目前已跃升至算力租赁行业第一梯队。公司呈现"传统制造+算力服务"双轮驱动格局。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **精密金属冲压结构件** | 液晶电视/显示器金属结构件，营收占比约60% | 液晶显示、消费电子 |
| **AI算力租赁服务** | GPU算力云服务，营收占比约32%，毛利率67%+ | AI训练、大模型开发 |
| **电子元器件** | 电子元器件制造销售，营收占比约4% | 消费电子 |
| **模具及底座** | 模具设计制造、底座产品 | 工业应用 |

### 业务模式

**传统制造业务**：
- 为海信、TCL等电视厂商提供精密金属结构件
- 采用订单生产模式，毛利率受行业竞争影响较大
- 2024年制造端业务亏损，但亏损幅度收窄

**算力租赁业务**：
- 通过"硬件+服务"模式提供AI算力云服务
- 与世纪珑腾合作，依托腾讯长三角人工智能超算中心
- 采用直租和转租两种模式，配备专业技术团队
- 算力设备销售毛利率高达67.67%

### 技术路线/核心技术

- **GPU服务器采购**：与NVIDIA、超聚变等建立战略合作
- **算力云平台**：云交付的算力租赁和技术支持
- **海外布局**：新加坡子公司布局海外算力中心

**技术优势**：
- 先发资源整合优势，快速建立大规模算力集群
- 完整的"硬件+服务"生态系统
- 专业技术团队提供持续支持

## 五、投资亮点

### 核心亮点

- ✅ **算力规模行业第一**：33,000P已运营算力规模，超越润泽科技、中贝通信等竞争对手，跃居行业龙头

- ✅ **业绩爆发式增长**：2025年前三季度净利润同比增长309.76%，算力业务成为核心增长引擎

- ✅ **高毛利率业务**：算力设备销售毛利率达67.67%，远高于传统制造业务，显著提升整体盈利能力

- ✅ **GPU战略合作**：与NVIDIA、超聚变等GPU服务器厂商建立战略合作，算力资源获取能力强

- ✅ **算力涨价红利**：2026年算力服务涨价趋势明确，直接增厚盈利空间

### 成长驱动因素

1. **AI算力需求持续爆发**
   - 大模型训练需求持续增长
   - 算力租赁市场渗透率提升
   - 公司作为行业龙头直接受益

2. **算力规模持续扩张**
   - 子公司昊利智算（上海）成立，注册资本4亿元
   - 海外算力中心布局（新加坡）
   - 预计算力规模将持续增长

3. **算力涨价周期**
   - 2026年算力服务涨价趋势明确
   - 高端算力供需紧张
   - 涨价直接转化为利润增长

### 催化剂

- 算力服务价格上调公告
- 新增算力规模投入运营
- 与头部AI企业签订大额合同
- 海外算力中心投产

### 拐点逻辑

**行业周期拐点**：AI算力需求爆发期，公司处于行业景气度上升周期

**业务转型拐点**：从传统制造亏损转向算力高增长，业绩拐点已现

**市场地位拐点**：从第二梯队跃升至行业第一，市场地位显著提升

#### 价格走势预测

| 时间段 | 价格变动 | 驱动因素 |
|--------|----------|----------|
| **2026年上半年** | **再涨10%-15%** | 供需紧张、交付周期延长 |
| **2026年全年** | **累计涨20%-30%** | 硬件成本上升、GPU紧缺 |

**当前价格水平**：
- 高端GPU（H200/H100）租赁价格已上涨 **15%-30%**
- 订单排期延长，部分交付期延至2027年Q2
- 硬件成本（GPU、存储芯片）上涨约30%

**关键供给约束**：
- 美国对英伟达H200/H100对华出口设限，每家企业限购7.5万枚
- 中国海关对部分芯片暂停入境，加征25%进口关税
- 英伟达已停止H200生产，转向新一代Vera Rubin平台
