## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 鸿博股份有限公司 |
| 证券代码 | 002229 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1999年 |
| 上市时间 | 2008年 |
| 所属行业 | AI算力服务、印刷业务 |
| 市场地位 | AI算力租赁领先企业，英伟达合作伙伴 |

**公司概况**：
鸿博股份原主营印刷业务，2023年起转型AI算力服务，通过子公司英博数科与英伟达合作，提供基于英伟达DGX云的全栈AI算力基础设施解决方案，成为A股AI算力租赁领域的领先企业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **AI算力服务** | 基于英伟达GPU的算力租赁服务 | 大模型训练、AI应用 |
| **智算中心运营** | 建设运营AI智算中心 | AI企业、金融机构 |
| **印刷业务** | 传统安全印务（逐步收缩） | 票据、证书等 |

### 业务模式

- **算力租赁**：提供GPU算力资源，按算力规模或时间收费
- **智算中心运营**：建设AI智算中心，提供算力服务
- **英伟达合作**：通过英博数科与英伟达深度合作

### 技术路线/核心技术

- **英伟达DGX云**：基于英伟达高端GPU的算力平台
- **液冷技术**：自主研发液冷方案
- **异构算力调度**：多型号GPU统一调度平台

**技术优势**：
- 英伟达生态深度绑定，高端GPU优先采购权
- 液冷技术降低能耗
- 异构调度平台提升资源利用率

## 五、投资亮点

### 核心亮点

- ✅ **英伟达深度合作**：享有高端GPU芯片优先采购权
- ✅ **AI算力领先**：AI算力业务占比超75%，核心增长引擎
- ✅ **技术壁垒**：液冷技术、异构调度平台
- ✅ **转型成功**：从传统印刷转型AI算力，成效显著
- ✅ **目标3000P**：计划将自有算力规模扩展至3000P

### 成长驱动因素

1. **AI算力需求爆发**
   - 大模型训练和推理需求增长
   - 公司算力规模持续扩张

2. **英伟达生态优势**
   - 高端GPU资源稀缺
   - 优先采购权保障供应

3. **智算中心建设**
   - 合肥和长三角智算中心布局
   - 郑新鸿博数智港项目推进

### 催化剂

- 新智算中心项目投产
- 大客户签订长期合同
- 算力规模突破3000P
