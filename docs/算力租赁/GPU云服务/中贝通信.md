## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中贝通信集团股份有限公司 |
| 证券代码 | 603220 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1996年 |
| 上市时间 | 2017年 |
| 所属行业 | 通信工程、算力租赁 |
| 市场地位 | **算力规模A股第二（17,000P）** |

**公司概况**：
中贝通信原主营通信工程服务，2024年起大力转型AI算力租赁业务。目前已运营算力规模超过17,000P，位居A股第二（仅次于利通电子33,000P），已完成智算中心全国业务布局，规划建设11,000P新集群，2026年二季度将有万卡级智算中心逐步投产。

- **全国智算布局**：武当智算集群、三江源智算集群、上海松江智算集群、上海临港智算集群
- **融资策略**：银行贷款+融资租赁（年化成本约5%）
- **规模扩张**：2026年Q2将投产11,000P新集群

**技术优势**：
- 算力规模A股第二，仅次于利通电子
- 全国智算中心网络布局完善
- 大客户资源丰富（金山云、阶跃星辰等）

## 五、投资亮点（深度分析）

### 核心亮点

- ✅ **算力规模第二**：17,000P已运营，A股仅次于利通电子
- ✅ **规模爆发在即**：2026年Q2投产11,000P，规模跃升至28,000P
- ✅ **大客户资源**：金山云、阶跃星辰、临港算力等
- ✅ **全国布局**：武当、三江源、上海松江/临港全覆盖
- ✅ **高毛利率**：算力租赁毛利率30%+

### OpenClaw受益逻辑 ⭐⭐⭐⭐⭐

**直接受益路径**：
1. **推理需求爆发**：OpenClaw推动推理需求超过训练4.5倍
2. **算力规模优势**：17,000P→28,000P（2026年Q2），供给能力强
3. **涨价受益**：2026年算力价格预期上涨20-30%
4. **客户扩张**：金山云、阶跃星辰等AI公司需求增长

**量化测算**：
- 当前17,000P算力，若按单P月租3.5万元计算
- 月度营收：17,000P × 3.5万 = 5.95亿元
- 年度营收：71.4亿元（算力业务）
- 若涨价20%：增加营收14.3亿元
- 按毛利率30%，净利率10%：增加净利润1.4亿元

### 成长驱动因素

1. **新产能投放**
   - 2026年Q2投产11,000P
   - 规模从17,000P→28,000P（+65%）
   - 营收预期翻倍

2. **算力涨价**
   - 2026年GPU租赁价格预期上涨20-30%
   - 高毛利率业务占比提升
   - 净利润弹性巨大

3. **客户扩张**
   - 金山云、阶跃星辰等AI公司需求增长
   - 临港算力、青海联通等政府客户稳定

4. **业务转型**
   - 算力业务占比从35%→50%+（2026年）
   - 从低毛利通信工程→高毛利算力租赁

### 催化剂

- 2026年Q2：11,000P新产能投产
- 算力涨价公告
- 新大客户签约
- 算力业务营收占比突破50%

### 拐点逻辑

**行业周期拐点**：AI算力需求爆发期 ✅

**业务转型拐点**：从通信工程→算力租赁，2026年占比超50% ✅

**规模拐点**：2026年Q2产能翻倍（17,000P→28,000P） ⭐⭐⭐⭐⭐

**盈利拐点**：算力业务高毛利率+涨价，2026年净利润预期大幅增长 ✅
