## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 曙光信息产业股份有限公司 |
| 证券代码 | 603019 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1995年 |
| 上市时间 | 2014年 |
| 所属行业 | 国产算力、超级计算机 |
| 市场地位 | **国产算力国家队，海光/超算双轮驱动** |
| **重组状态** | **与海光信息换股吸收合并中，将形成4000亿"算力航母"** |

**公司概况**：
中科曙光是国产算力龙头企业，"中科院系"背景，主营高性能计算机、服务器、存储等业务。拥有**2.5-3.0万P海光/超算算力**，采用国产芯片。2025年Q3算力租赁营收同比增长85%，毛利率达42%。2025年6月，海光信息发布换股吸收合并中科曙光预案，合并后将形成总市值超4000亿元的"算力航母"。

## 五、投资亮点（深度分析）

### 核心亮点

- ✅ **国产算力国家队**：中科院背景，政策支持力度大
- ✅ **海光芯片生态**：自主可控，信创市场受益
- ✅ **超算能力领先**：中国超算排名前列
- ✅ **高毛利率**：42%，远高于行业平均
- ✅ **80亿扩产计划**：募资建设AI算力集群
- ✅ **4000亿航母级重组**：与海光信息合并，强强联合

### OpenClaw受益逻辑 ⭐⭐⭐⭐

**国产算力受益路径**：
1. **信创政策加速**：OpenClaw推动AI普及，政府/金融加速国产化替代
2. **海光芯片需求**：国产算力需求增长，海光芯片受益
3. **超算服务需求**：大模型训练需要超算资源
4. **全国调度能力**：分散算力资源统一调度，效率提升

**量化测算**：
- 当前2.5-3.0万P算力
- 80亿扩产后预计达到5-6万P
- 若按单P月租2.5万元（国产算力价格低于英伟达）
- 年度营收：3万P × 2.5万 × 12月 = 90亿元
- 按毛利率42%：毛利润37.8亿元
- 按净利率15%：净利润13.5亿元

### 成长驱动因素

1. **信创政策加速**
   - 政府、金融等领域国产化替代
   - 国产算力需求爆发
   - 中科曙光是最大受益者

2. **80亿扩产计划**
   - 募资建设AI算力集群
   - 算力规模从3万P→5-6万P
   - 营收翻倍

3. **海光信息合并**
   - 形成4000亿"算力航母"
   - 上下游一体化，成本控制
   - 市值重估

4. **超算服务需求**
   - AI大模型训练需要超算资源
   - 科研机构、高校需求增长

### 催化剂

- 80亿募资获批
- 新算力集群投产
- 信创政策出台
