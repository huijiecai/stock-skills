## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 北京首都在线科技股份有限公司 |
| 证券代码 | 300846 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2005年 |
| 上市时间 | 2020年 |
| 所属行业 | IDC、云计算、GPU云服务 |
| 市场地位 | IDC+云计算双核驱动，全球布局 |

**公司概况**：
首都在线是国内唯一成功实现"IDC+云计算"双核驱动的企业，拥有自主云服务能力，覆盖全球多个数据中心节点。公司提供稳定、安全的云和IDC服务，客户涵盖游戏、电商、视频等行业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **GPU云服务** | 基于GPU的云计算服务 | AI训练、推理部署 |
| **IDC服务** | 数据中心服务，服务器托管 | 企业IT基础设施 |
| **云主机** | 虚拟化云主机服务 | 企业云计算 |

### 业务模式

- **IDC+云计算双核驱动**：同时提供IDC和云计算服务
- **全球节点布局**：覆盖全球多个数据中心节点
- **按需付费**：弹性计算，按使用量收费

### 技术路线/核心技术

- **GPU云平台**：支持AI训练和推理的GPU云服务
- **全球网络**：覆盖全球的数据中心网络
- **弹性计算**：按需扩展的计算资源

**技术优势**：
- IDC+云计算双核驱动，业务协同
- 全球化布局，服务能力强
- 自主云服务能力，技术独立

## 四、行业地位与产业链定位

### 产业链定位

```
算力租赁产业链：
上游（芯片/服务器）→ 中游（IDC+云服务商）→ 【首都在线】→ 下游（企业客户）→ 终端应用
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | IDC+云计算双核驱动企业 |
| **竞争优势** | 全球布局、双核驱动、自主云服务 |
| **客户结构** | 游戏、电商、视频等行业 |
| **供应商** | 服务器厂商、网络设备供应商 |

**公司优势**：

- 全球化布局
- 自主云服务能力

## 五、投资亮点

### 核心亮点

- ✅ **双核驱动**：IDC+云计算双核驱动，业务协同
- ✅ **全球布局**：覆盖全球多个数据中心节点

   - 公司GPU云服务受益

2. **全球化布局**
   - 海外市场需求增长
   - 全球节点优势

3. **经营改善**
   - 亏损收窄
   - 盈利在望

### 催化剂

- 实现盈利
- 大客户签约
- 海外业务突破
