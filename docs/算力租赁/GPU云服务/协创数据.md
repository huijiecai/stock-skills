## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 协创数据技术股份有限公司 |
| 证券代码 | 300857 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2005年 |
| 上市时间 | 2020年 |
| 所属行业 | 数据存储、算力服务、物联网终端 |
| 市场地位 | 全球化数据中心网络，算力租赁快速增长 |

**公司概况**：
协创数据是一家全球化数据中心网络运营商，主营数据存储设备、智能算力产品、物联网终端及服务器再制造。2025年净利润预计增长52%-81%，达10.5-12.5亿元，算力相关收入实现高速增长。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **算力服务** | GPU算力租赁，AI算力集群 | AI训练、云游戏、AI应用 |
| **数据存储** | 数据存储设备、服务器再制造 | 数据中心、企业存储 |
| **物联网终端** | AIoT终端设备 | 智能家居、智能城市 |
| **服务器再制造** | 服务器回收再制造 | 数据中心 |

### 业务模式

- **算力租赁**：大规模服务器采购，提供算力服务
- **数据存储**：提供存储设备和解决方案
- **物联网终端**：AIoT产品销售
- **服务器再制造**：服务器回收、翻新、销售

### 技术路线/核心技术

- **大规模算力集群**：累计采购服务器超82亿元
- **英伟达合作资质**：取得英伟达合作资质
- **全球化布局**：香港、海外数据中心网络

**技术优势**：
- 大规模算力资源
- 英伟达合作资质
- 全球化数据中心网络

## 五、投资亮点

### 核心亮点

- ✅ **业绩高增长**：2025年净利润预计增长52%-81%
- ✅ **算力业务爆发**：算力相关收入高速增长
- ✅ **英伟达合作**：取得英伟达合作资质
- ✅ **全球化布局**：香港、海外数据中心网络
- ✅ **回报率高**：算力业务回报率28%-35%

### 成长驱动因素

1. **算力需求爆发**
   - 大模型训练和推理需求增长
   - 公司大规模采购服务器，算力资源丰富

2. **物联网终端增长**
   - AIoT业务增长100%
   - 智能家居、智能城市需求旺盛

3. **服务器再制造**
   - 数据中心设备更新需求
   - 成本优势明显

### 催化剂

- 新算力合同签订
- 海外业务突破
- 算力规模扩张
