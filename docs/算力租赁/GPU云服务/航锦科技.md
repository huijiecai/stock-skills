## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 航锦科技股份有限公司 |
| 证券代码 | 000818 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1997年 |
| 上市时间 | 2010年 |
| 所属行业 | 化工、电子、智算算力 |
| 市场地位 | 算力租赁合同额领先企业 |

**公司概况**：
航锦科技原主营氯碱化工和军工电子，近年来积极转型智算算力业务，2024年上半年签订算力租赁合同约41.93亿元，自持约1400台服务器，运算能力约1.5万P，成为算力租赁合同额领先企业。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **算力租赁** | GPU服务器租赁，算力服务 | AI训练、推理部署 |
| **化工产品** | 氯碱化工产品（传统业务） | 工业、建筑 |
| **军工电子** | 军用电子产品 | 国防军工 |

### 业务模式

- **算力租赁**：自持GPU服务器，提供算力服务
- **化工+电子+算力**：三大板块协同发展
- **合同制服务**：与大客户签订长期算力服务合同

### 技术路线/核心技术

- **GPU服务器集群**：自持约1400台服务器
- **英伟达GPU**：部署英伟达GPU服务器
- **智算中心建设**：建设智算中心提供算力服务

**技术优势**：
- 自持服务器规模大，算力资源丰富
- 算力租赁合同额行业领先
- 化工+电子+算力协同发展

## 四、行业地位与产业链定位

### 产业链定位

```
算力租赁产业链：
上游（英伟达GPU）→ 中游（算力服务商）→ 【航锦科技】→ 下游（AI企业）→ 终端应用
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 算力租赁合同额领先 |
| **竞争优势** | 自持服务器规模大，合同额高 |
| **客户结构** | AI企业、金融机构 |
| **供应商** | 英伟达等GPU供应商 |

**公司优势**：
- 算力租赁合同额41.93亿元，行业领先
- 自持1400台服务器，算力规模约1.5万P
- 出租率100%，客户需求旺盛

## 五、投资亮点

### 核心亮点

- ✅ **合同额领先**：算力租赁合同约41.93亿元，行业领先
- ✅ **算力规模大**：自持1400台服务器，运算能力1.5万P
- ✅ **出租率100%**：算力资源供不应求
- ✅ **毛利率高**：算力业务毛利率34.17%
- ✅ **转型成功**：预计2025年算力业务占比超50%

### 成长驱动因素

1. **算力需求爆发**
   - 大模型训练和推理需求增长
   - 公司算力规模持续扩张

2. **合同收入兑现**
   - 大额合同逐步兑现
   - 收入确定性高

3. **业务结构优化**
   - 算力业务占比提升
   - 毛利率持续改善

### 催化剂

- 新算力合同签订
- 算力规模扩张
- 算力业务收入兑现

## 七、业绩预期与估值

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 合同快速兑现，算力扩张 | 净利润3亿+ | 净利润5亿+ |
| **中性** | 合同稳定兑现，毛利率稳定 | 净利润1.5-2亿 | 净利润2.5-3.5亿 |
| **悲观** | 合同兑现慢，竞争加剧 | 净利润<1亿 | 净利润<2亿 |

### 估值水平

| 指标 | 当前水平 | 行业平均 | 评估 |

| PE(TTM) | 约75倍 | 约40倍 | 偏高 |
