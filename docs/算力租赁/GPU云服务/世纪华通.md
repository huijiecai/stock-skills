## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 浙江世纪华通集团股份有限公司 |
| 证券代码 | 002602 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2011年 |
| 所属行业 | 游戏、算力租赁 |
| 市场地位 | **腾讯深度绑定，50亿订单锁定** |

**公司概况**：
世纪华通原为游戏龙头企业（《Whiteout Survival》全球爆款），2024年起通过子公司世纪利通大力布局AI算力租赁业务。目前已累计出租10,000P算力，其中80%服务腾讯，与腾讯锁定2025-2027年期间50亿元算力订单。2025年净利润预增357%-475%，创历史新高。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **GPU算力租赁** | 10,000P已出租，80%服务腾讯 | AI训练、大模型推理 |
| **移动游戏** | 全球爆款《Whiteout Survival》等 | 游戏娱乐 |
| **AI技术应用** | AI工具赋能游戏研发 | 游戏美术、智能NPC |

### 业务模式

- **腾讯深度绑定**：与腾讯锁定2025-2027年50亿元算力订单
- **H20服务器租赁**：提供H20服务器租赁及液冷改造
- **数据中心运营**：腾讯长三角人工智能先进计算中心、深圳数据中心

### 技术路线/核心技术

- **英伟达DGX优先级合作**：上游智算力基础设施优势
- **液冷改造技术**：提供液冷数据中心解决方案
- **AI赋能游戏**：AI工具使美术批量生产效率提升60%-80%

**技术优势**：
- 腾讯深度绑定，订单确定性强
- 英伟达生态优先级合作
- 游戏+算力双轮驱动
