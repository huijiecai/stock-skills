## 一、行业定义与范围

### 是什么？

> **算力租赁 = "算力版的共享经济"**
>
> 企业无需自建数据中心或购买昂贵GPU服务器，通过租赁方式按需获取计算资源，按使用量或时间付费。

| 细分领域 | 说明 | 典型应用 |
|----------|------|----------|
| **IDC运营商** | 机柜租赁、重资产、长期合同 | 大型企业自建IT基础设施 |
| **GPU算力服务商** | 自持GPU服务器、按算力收费 | AI模型训练、推理部署 |
| **云服务商** | 虚拟化云主机、按量付费 | 中小企业弹性计算需求 |
| **硬件销售/集成** | 卖服务器、做集成 | 政企客户IT采购 |
| **跨界布局** | 传统业务+少量算力投资 | 多元化转型 |

### 产业链定位

```
[上游：芯片/服务器/网络设备]
    ↓
【算力租赁服务商】（本行业）
    ↓
[下游：AI公司/互联网企业/政府/金融]
    ↓
[终端应用：大模型训练、AI推理、云计算、科学计算]
```

**说明**：算力租赁服务商位于产业链中游，整合上游硬件资源，为下游客户提供弹性计算服务。

## 四、产业链结构

### 产业链全景

```
上游：硬件与基础设施
  │
  ├─ AI芯片 → NVIDIA、华为昇腾、寒武纪
  ├─ GPU服务器 → 浪潮信息、中科曙光、华为
  ├─ 网络设备 → 中兴通讯、紫光股份
  └─ 数据中心建设 → 润泽科技、光环新网
  │
  ↓
中游：算力租赁服务商（本行业）
  │
  ├─ IDC基础设施 → 光环新网、数据港、奥飞数据、大位科技
  ├─ GPU云服务 → 鸿博股份、航锦科技、协创数据、首都在线
  ├─ 云计算IaaS → 优刻得、云赛智联、华胜天成、网宿科技
  ├─ 智算中心 → 润泽科技、宏景科技、润建股份、拓维信息
  └─ 跨界转型 → 豫能控股、海南华铁、杭钢股份、宁波建工、利通电子
  │
  ↓
下游：应用场景
  │
  ├─ AI大模型训练 → 百度、阿里、腾讯、字节跳动
  ├─ AI推理部署 → 各行业AI应用企业
  ├─ 云计算服务 → 中小企业、开发者
  └─ 科学计算 → 高校、科研机构
```

### 关键环节分析

#### AI芯片（上游核心）

- **作用**：算力的"心脏"，决定算力性能上限
- **技术门槛**：极高
- **价值占比**：40-50%
- **龙头企业**：NVIDIA（全球）、华为昇腾（国产）
- **现状**：高端GPU供应紧张，国产替代加速

#### 数据中心基础设施（中游基础）

- **作用**：算力的"房子"，提供稳定运行环境
- **技术门槛**：中等（重资产）
- **价值占比**：20-30%
- **龙头企业**：润泽科技、光环新网、数据港
- **现状**：区域布局不均，一线城市资源稀缺

#### 算力调度平台（中游增值）

- **作用**：算力的"管家"，提升资源利用效率
- **技术门槛**：高（软件能力）
- **价值占比**：10-15%
- **龙头企业**：优刻得、首都在线
- **现状**：异构调度成为竞争焦点

## 五、国产替代进展

### 国产化现状

| 细分领域 | 国产化率 | 国内龙头 | 技术差距 | 突破难度 |
|----------|----------|----------|----------|----------|
| **AI训练芯片** | 15% | 华为昇腾 | 2-3代 | ⭐⭐⭐⭐⭐ |
| **AI推理芯片** | 35% | 华为昇腾、寒武纪 | 1-2代 | ⭐⭐⭐⭐ |
| **GPU服务器** | 60% | 浪潮信息、华为 | 较小 | ⭐⭐⭐ |
| **数据中心** | 95% | 润泽科技、光环新网 | 基本持平 | ⭐ |
| **云平台软件** | 80% | 阿里云、优刻得 | 较小 | ⭐⭐ |

**难度说明**：
- ⭐⭐⭐⭐⭐：极高难度，长期攻关
- ⭐⭐⭐⭐：高难度，5-10年
- ⭐⭐⭐：中等难度，3-5年
- ⭐⭐：较低难度，1-3年
- ⭐：基本实现

### 突破方向

1. **华为昇腾生态建设**
   - 当前进展：昇腾910B性能接近A100，M300系列迭代中
   - 关键企业：华为、拓维信息、神州数码
   - 预计突破时间：2026-2027年

2. **国产GPU芯片研发**
   - 当前进展：寒武纪、海光信息、壁仞等持续迭代
   - 关键企业：寒武纪、海光信息、壁仞科技、沐曦
   - 预计突破时间：2027-2028年

### 政策支持

- **"东数西算"工程**（2022年启动）- 建设国家算力枢纽节点
- **算力基础设施高质量发展行动计划**（2023年）- 2025年算力规模超300 EFLOPS
- **各地方智算中心规划**（2024-2025年）- 北京、上海、深圳等地密集出台

### 6.4 投资标的选择建议

#### 稳健配置

优先选择第一梯队龙头企业，具有：
- 稳定的市场地位和盈利能力
- 完善的产业链布局
- 较强的抗风险能力
- 适合长期持有

**推荐标的**：润泽科技、光环新网、数据港

#### 成长配置

关注第二梯队中快速成长的企业，特征：
- 细分领域领先
- 业绩增速快于行业平均
- 有技术或模式创新
- 适合中期布局

**推荐标的**：鸿博股份、航锦科技、协创数据、首都在线

#### 弹性配置

适度配置第三梯队潜力企业，特点：
- 估值相对较低
- 转型或突破带来的业绩弹性大
- 风险较高，需密切跟踪
- 适合小仓位博弈

**推荐标的**：海南华铁、豫能控股

## 七、投资逻辑与风险

### 投资逻辑

#### 行业层面

1. **AI浪潮驱动需求爆发**
   - 大模型训练和推理需求持续增长
   - 2025年智能算力需求增长超50%
   - 预计2026年市场规模达2600亿元

2. **政策强力支持**
   - "东数西算"国家战略持续推进
   - 各地智算中心建设规划密集出台
   - 算力券等补贴政策降低使用成本

3. **国产替代加速**
   - 华为昇腾等国产芯片逐步成熟
   - 信创、政务领域国产化需求强烈
   - 国产算力生态逐步完善

#### 公司层面

**选股逻辑**：
- 第一梯队：核心业务占比高、盈利能力强
- 第二梯队：细分领域领先、成长性好
- 第三梯队：转型逻辑清晰、估值有安全边际

### 风险因素

| 风险类型 | 具体描述 | 影响程度 |
|----------|----------|----------|
| ⚠️ **价格战风险** | GPU租赁价格已下跌超70%，利润空间被压缩 | 高 |
| ⚠️ **供应链风险** | 高端GPU芯片供应紧张，依赖进口 | 高 |
| ⚠️ **竞争加剧** | 行业门槛降低，新进入者增多 | 中 |
| ⚠️ **技术迭代风险** | AI芯片技术快速迭代，设备贬值快 | 中 |
| ⚠️ **政策风险** | 数据安全监管趋严，合规成本上升 | 低 |
