## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 润建股份有限公司 |
| 证券代码 | 002929 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2003年 |
| 上市时间 | 2019年 |
| 所属行业 | 通信基础设施、算力中心 |
| 市场地位 | 通信基础设施服务商，算力中心布局中 |

**公司概况**：
润建股份是一家通信基础设施服务商，主营通信网络建设、维护和优化服务。公司积极布局算力业务，依托五象云谷数据中心，与阿里云、百度、华为等合作，推动"智算云"项目和大模型部署。
