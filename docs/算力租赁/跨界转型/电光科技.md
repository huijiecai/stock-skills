## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 电光防爆科技股份有限公司 |
| 证券代码 | 002730 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1993年 |
| 上市时间 | 2014年10月9日 |
| 所属行业 | 专用机械 / 算力租赁 |
| 市场地位 | 国内矿用防爆电器龙头 |

**公司概况**：
电光科技是国内矿用防爆电器领域的龙头企业，主要产品包括矿用防爆开关、真空接触器、软起动器等。2024年起公司跨界进入算力租赁领域，成立子公司"电光云"开展GPU算力服务业务。

## 二、主营业务

### 业务构成

| 业务板块 | 收入占比 | 说明 |
|----------|----------|------|
| **矿用防爆开关** | 58.00% | 核心传统业务 |
| **智能化装备及系统** | 13.25% | 智能矿山配套 |
| **驱动设备** | 12.80% | 电机驱动系统 |
| **算力服务** | 9.35% | 新增业务，增长最快 |
| **电力设备** | 3.92% | 配套产品 |
| **其他** | 2.68% | - |

### 业务模式

**传统业务（矿用防爆设备）**：
- 客户：大型煤炭集团（国家能源集团、中煤等）
- 模式：研发-生产-销售
- 特点：周期性强，与煤炭行业景气度相关

**新业务（算力租赁）**：
- 客户：AI企业、互联网公司
- 模式：建设智算中心，出租GPU算力
- 特点：高投入、高毛利、回收周期长

## 四、行业地位与产业链定位

### 传统业务：矿用防爆设备

```
产业链定位：
上游（钢材、电子元器件）→ 中游（防爆设备制造）→ 下游（煤矿、石化）
                                    ↑
                                电光科技
```

**市场地位**：

| 维度 | 说明 |
|------|------|
| **国内地位** | 矿用防爆电器领域龙头，产品系列最全 |
| **主要客户** | 国家能源集团、中煤集团等大型煤炭企业 |
| **竞争优势** | 产品线完整、客户资源稳定、技术积累深厚 |
| **行业趋势** | 煤炭行业下行，需求承压 |

**主要竞争对手**：
- 华荣股份（厂用防爆龙头）
- 新黎明（防爆电器）

### 新业务：算力租赁

```
产业链定位：
上游（GPU芯片、服务器）→ 中游（算力中心建设运营）→ 下游（AI企业）
                                    ↑
                                电光云
```

**业务进展**：

| 项目 | 详情 |
|------|------|
| **子公司** | 浙江电光云科技有限公司 |
| **成立时间** | 2024年4月 |
| **重大合同** | 2024年6月签署5.54亿元算力服务合同 |
| **客户** | 上海无问芯穹智能科技有限公司 |
| **服务期限** | 5年 |
| **年收入贡献** | 约1.1亿元/年 |

## 五、投资亮点

### 核心亮点

- ✅ **5.54亿元算力合同**：签署5年期算力服务合同，年均贡献约1.1亿元收入
- ✅ **算力租赁进入涨价周期**：2026年算力市场供不应求，进入"卖方市场"
- ✅ **跨界转型**：从传统防爆设备跨界算力，打开新增长空间
- ✅ **传统业务稳健**：矿用防爆业务提供基本盘
