## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 拓维信息系统股份有限公司 |
| 证券代码 | 002261 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1996年 |
| 上市时间 | 2008年 |
| 所属行业 | 软件服务、AI算力、鸿蒙生态 |
| 市场地位 | 华为生态深度合作伙伴 |

**公司概况**：
拓维信息是一家软件服务企业，以AI与鸿蒙生态为核心战略，深度绑定华为生态。公司专注于国产AI算力基础设施和大模型一体机的研发与应用，推出多款AI服务器、一体机和边缘计算设备。
