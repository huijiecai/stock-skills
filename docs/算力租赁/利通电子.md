## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 江苏利通电子股份有限公司 |
| 证券代码 | 603629.SH |

| 成立时间 | 2001年 |
| 上市时间 | 2018年12月24日 |
| 所属行业 | 元器件 - 消费电子 |
| 市场地位 | 算力租赁新兴龙头 |

**公司概况**：
利通电子原主营业务为液晶电视金属结构件制造，2023年起战略转型AI算力业务。目前公司算力规模达33,000P（自有15,000P+转租18,000P），已成为算力租赁领域的重要参与者，是A股稀缺的算力租赁标的。

## 二、主营业务

### 核心产品/服务

| 业务板块 | 说明 | 应用领域 |
|----------|------|----------|
| **AI算力租赁** | GPU集群算力租赁服务（直租+转租） | AI训练、大模型推理 |
| **算力云服务** | 云交付的算力服务 | 生成式AI客户 |
| **设备经销** | AI服务器及相关设备销售 | 算力基础设施建设 |
| **电视结构件** | 液晶电视金属结构件制造 | 消费电子 |

### 业务模式

公司采用"自有+转租"双模式运营：
- **直租模式**：自有AI设备算力云直租，毛利率高
- **转租模式**：租入算力后转租，快速扩张规模
- **云交付**：提供云化算力服务，客户粘性高

### 技术路线/核心技术

- **液冷技术**：自主研发液冷技术，降低运营成本
- **算力调度**：智能算力调度系统，提升利用率
- **云原生架构**：支持云交付，灵活扩展

**技术优势**：
算力规模33,000P，合同期限均在36个月以上，客户包括腾讯、阿里云等头部企业。
