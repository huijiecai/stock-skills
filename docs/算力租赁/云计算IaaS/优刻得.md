## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 优刻得科技股份有限公司 |
| 证券代码 | 688158 |
| 上市板块 | 上海证券交易所（科创板） |
| 成立时间 | 2012年 |
| 上市时间 | 2020年1月 |
| 所属行业 | 云计算IaaS服务 |
| 市场地位 | "云计算第一股"，中立云服务商 |

**公司概况**：
优刻得是中国首家云计算科创板上市企业，定位中立云服务商，不与客户竞争底层服务。公司在全球部署30个可用区，服务超过5万家企业客户，提供云主机、存储、网络、安全、人工智能和大数据等全栈云服务。

### 核心产品/服务

|----------|------|----------|
| **公有云** | 云主机、云存储、云网络 | 企业云计算 |

### 业务模式

- **中立云服务**：不与客户竞争，专注提供底层云服务

- **全球化布局**：全球30个可用区

### 技术路线/核心技术

- **绿色算力**：上海青浦、内蒙古乌兰察布智算中心
- **AI产品**："孔明"AI平台、UModelverse
- **云网融合**：云网一体化解决方案

**技术优势**：
- 中立云服务商，客户信任度高
- 绿色算力布局，成本优势
- 全栈云服务能力
