## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 云赛智联股份有限公司 |
| 证券代码 | 600602 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1986年 |
| 上市时间 | 1990年 |
| 所属行业 | 云计算、大数据、信息化服务 |
| 市场地位 | 上海国资背景，上海算力建设主力 |

**公司概况**：
云赛智联是上海国资控股的信息化服务企业，专注于云计算、大数据、行业解决方案和智能产品。公司是上海市大数据中心资源平台总集成商，深度参与上海算力建设，与微软、阿里、华为等云厂商合作。

## 五、投资亮点

### 核心亮点

- ✅ **上海国资背景**：资源优势明显，项目获取能力强
- ✅ **政务云领先**：上海市大数据中心资源平台总集成商
- ✅ **算力建设主力**：参与上海超级计算中心项目
- ✅ **多云合作**：与微软、阿里、华为合作
- ✅ **松江大数据中心**：投资建设二期项目

### 成长驱动因素

1. **政务云需求**
   - 政府数字化转型推动
   - 公司政务云业务受益

2. **AI算力建设**
   - 上海算力建设需求
   - 公司参与超级计算中心项目

3. **国资资源优势**
   - 上海国资背景
   - 项目获取能力强

### 催化剂

- 上海算力项目中标
- 松江大数据中心投产
- 新政务云合同签订
