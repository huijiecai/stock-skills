## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 北京华胜天成科技股份有限公司 |
| 证券代码 | 600410 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年 |
| 上市时间 | 2004年 |
| 所属行业 | IT解决方案、云计算服务 |
| 市场地位 | IT解决方案提供商，华为生态合作 |

**公司概况**：
华胜天成是一家领先的云计算和IT解决方案提供商，主营业务涵盖通讯软件技术开发、系统集成、咨询、培训及电子商务服务。公司深度合作华为，推动"算力基建+AI场景+华为生态"战略。
