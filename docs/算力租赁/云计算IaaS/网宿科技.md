## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 网宿科技股份有限公司 |
| 证券代码 | 300017 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2000年 |
| 上市时间 | 2009年 |
| 所属行业 | CDN、边缘计算、云计算 |
| 市场地位 | CDN龙头，边缘计算领先 |

**公司概况**：
网宿科技是中国领先的CDN和边缘计算服务提供商，拥有2800+全球节点，覆盖70多个国家。公司核心业务包括CDN及边缘计算、安全及增值服务，并在液冷技术和绿色云基础设施方面取得突破。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **CDN服务** | 内容分发网络服务 | 互联网企业 |
| **边缘计算** | 边缘云平台（ECP） | 边缘AI、IoT |
| **安全服务** | 网络安全、大模型安全评估 | 企业客户 |

### 业务模式

- **CDN服务**：提供内容分发网络服务，按流量收费
- **边缘计算**：提供边缘云平台服务
- **安全服务**：提供网络安全和大模型安全评估服务

### 技术路线/核心技术

- **边缘AI平台**：支持边缘AI应用部署
- **液冷技术**：绿色云基础设施
- **大模型安全评估**：AI安全服务

**技术优势**：
- CDN龙头，技术积累深厚
- 边缘计算平台领先
- 全球化布局，2800+节点

## 四、行业地位与产业链定位

### 产业链定位

```
算力租赁产业链：
上游（芯片/服务器）→ 中游（CDN/云服务商）→ 【网宿科技】→ 下游（互联网企业）→ 终端应用
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | CDN龙头，边缘计算领先 |
| **竞争优势** | 全球节点、技术积累、独立第三方 |
| **客户结构** | 字节跳动、快手等互联网大厂 |
| **供应商** | 服务器厂商、网络设备供应商 |

### 竞争格局

**主要竞争对手**：
- 阿里云CDN：市场份额最大
- 腾讯云CDN：企业客户资源丰富
- 优刻得（688158）：中立云服务商

**公司优势**：
- CDN龙头，技术积累深厚
- 独立第三方，客户信任度高
- 全球化布局，2800+节点

## 五、投资亮点

### 核心亮点

- ✅ **CDN龙头**：技术积累深厚，市场份额领先
- ✅ **边缘计算领先**：边缘AI平台布局
- ✅ **盈利能力强**：毛利率33%，净利率16%
- ✅ **独立第三方**：客户信任度高
- ✅ **全球化布局**：2800+全球节点

### 成长驱动因素

1. **边缘AI需求**
   - 边缘AI应用需求增长
   - 公司边缘计算平台受益

2. **安全服务增长**
   - 大模型安全评估需求
   - 公司安全服务增长

3. **CDN需求稳定**
   - 视频流量持续增长
   - CDN需求稳定

### 催化剂

- 边缘AI平台突破
- 大模型安全评估业务增长
- 海外市场拓展
