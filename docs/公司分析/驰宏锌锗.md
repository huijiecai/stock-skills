## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 云南驰宏锌锗股份有限公司 |
| 证券代码 | 600497 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2000年7月（前身为云南会泽铅锌矿） |
| 上市时间 | 2004年4月 |
| 所属行业 | 有色金属-铅锌锗 |
| 市场地位 | 国内铅锌锗龙头，央企中铝集团控股 |

**公司概况**：
驰宏锌锗是国内重要的铅锌锗生产企业，主要从事铅锌矿采选、冶炼及新材料研发。公司为中国铝业集团有限公司控股子公司，拥有铅锌精矿产能42万吨/年、铅锌精炼产品产能63万吨/年。锗产品含锗34.29吨（2024年），锗价2024年6月起大幅上涨。上市以来累计分红19次，分红总额66.86亿元，2024-2025年度获上交所信息披露A级评价。

*数据来源：公司官网chxz.chinalco.com.cn、金投网、新浪财经，2025年*

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **铅锌精矿** | 产能42万吨/年，矿山包括会泽、彝良、西藏鑫湖等7座 | 冶炼原料 |
| **铅锌冶炼产品** | 产能63万吨/年，曲靖、会泽、呼伦贝尔、永昌4座冶炼基地 | 镀锌、电池、合金 |
| **锗产品** | 2024年含锗34.29吨，6月后锗价大幅上涨至14457元/公斤 | 红外、光纤、半导体、光伏 |
| **白银** | 伴生矿产 | 工业、光伏 |
| **硫酸** | 冶炼副产品 | 化工 |

### 业务模式

公司采用"矿山+冶炼"一体化模式，铅锌精矿自采自炼，冶炼加工费为主要盈利来源之一。2024年铅锌加工费大幅下跌至历史低位（8月国产锌精矿加工费降至1450元/吨），国内冶炼厂大面积亏损，公司业绩承压。锗产品作为特色业务，受益战略金属出口管制及价格上涨。

### 技术路线/核心技术

- **铅锌采选冶炼**：会泽、彝良等矿山高品位资源，采选技术成熟
- **锗回收技术**：锗作为铅锌伴生矿产，综合回收能力强
- **绿色冶炼**：推进绿色开采、循环经济转型
- **研发创新**：2024年获云南省科学技术进步特等奖

**技术优势**：
公司拥有从矿山采选到冶炼的完整铅锌产业链，锗产品具备战略稀缺性。在成本管控、技术创新方面具备优势，反内卷政策框架下具备成本优势的企业更具配置价值。

*数据来源：公司年报、新浪财经、东方财富研报，2024-2025年*

## 四、行业地位与产业链定位

### 产业链定位

```
铅锌产业链：
铅锌矿勘探开采 → 选矿 → 【驰宏锌锗】→ 冶炼 → 下游应用
                        ↑
                  铅锌精矿+冶炼+锗
                  国内铅锌锗龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内铅锌锗行业龙头之一，铅锌精矿产能42万吨/年、冶炼63万吨/年 |
| **竞争优势** | 矿山+冶炼一体化、锗产品战略稀缺、央企中铝控股 |
| **客户结构** | 面向镀锌、电池、合金等下游 |
| **供应商** | 自有矿山为主，会泽、彝良、西藏鑫湖等7座矿山 |

### 竞争格局

**主要竞争对手**：
- **中金岭南**：铅锌精矿产能全国首位，凡口铅锌矿
- **西部矿业**：西部铅锌铜龙头
- **紫金矿业**：多金属矿业，铅锌为补充

**公司优势**：
驰宏锌锗具备铅锌锗一体化优势，锗产品在战略金属管制背景下价值凸显。西藏鑫湖2025年4月恢复生产，资源端有增量。但铅锌加工费低迷阶段，冶炼环节利润承压。

## 五、投资亮点

### 核心亮点

- ✅ **锗产品战略价值**：锗、钨、镓等战略性关键矿产面临出口管制，国家将其列入优势资源规则输出范畴；2024年6月后锗价大幅上涨至14457元/公斤，量价齐升

- ✅ **铅锌锗一体化**：国内少数具备铅锌锗完整产业链的企业，矿山+冶炼协同

- ✅ **西藏鑫湖复产**：2025年4月恢复生产，前三季度实现原矿含锌3490吨、含铅2832吨，资源端增量

- ✅ **现金流改善**：2025年前三季度经营现金流34.14亿元同比+32.0%

- ✅ **分红稳定**：上市以来累计分红19次、66.86亿元，拟每10股派1.30元

### 成长驱动因素

1. **锗价上行**
   - 战略金属出口管制、AI/光伏需求支撑锗价

2. **加工费修复**
   - 铅锌精矿供应紧张有望推动加工费从历史低位回升

3. **产能释放**
   - 西藏鑫湖复产、12个重点项目建成

### 催化剂

- 铅锌加工费从低位回升
- 锗价维持高位或继续上涨
- 战略金属政策利好
