## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 赣州腾远钴业新材料股份有限公司 |
| 证券代码 | 301219 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2004年 |
| 上市时间 | 2022年3月17日 |
| 所属行业 | 有色金属-钴镍锂新材料、新能源电池材料 |
| 市场地位 | 国内领先钴盐生产企业，第三梯队潜力企业 |

**公司概况**：
腾远钴业是一家集科研、生产、贸易于一体的跨国现代化企业，主要从事钴、铜、镍、锂、电池材料前驱体等产品的研发、生产与销售。公司聚焦新能源金属资源开发利用，为全球领先的钴、铜、电池材料前驱体产品生产企业之一，2024年新增四氧化三钴、三元前驱体产品，向下游延伸布局。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **钴产品** | 氯化钴、硫酸钴等钴盐，电积钴 | 锂电池正极材料、合金、磁性材料、电镀 |
| **铜产品** | 电积铜 | 电子电气、机械制造、建筑材料 |
| **镍产品** | 硫酸镍 | 锂电池正极材料、不锈钢、电镀 |
| **锂产品** | 电池级碳酸锂 | 锂电池正极材料及电解液 |
| **电池材料** | 四氧化三钴、三元前驱体（2024年新增） | 消费电池、动力电池 |

### 业务模式

公司拥有独立完整的采购、生产和销售体系，采用直销模式。采购主要原料为铜钴矿、钴中间品、MHP及电池黑粉等，与全球著名钴镍原料供应商、国内电池厂及材料厂建立长期合作。销售定价以伦敦金属导报（MB）钴金属报价、上海有色网（SMM）镍锂金属报价为基准，结合市场供需确定。刚果腾远在刚果（金）当地加工电积铜，通过维克托全球销售。

### 技术路线/核心技术

- **钴湿法冶炼技术**：国内领先的钴湿法冶炼企业，低品位复杂金属处理能力强，2021年钴产品加工成本22,182元/吨，明显低于同行
- **铜钴协同冶炼**：刚果（金）铜钴矿综合利用，电积铜毛利率高
- **产品延伸**：赣州腾驰2024年新增四氧化三钴、三元前驱体，切入3C消费电池与动力电池赛道

**技术优势**：
公司在钴产品湿法冶炼上具有较强的技术优势、工艺优势和成本优势，2021年钴产品毛利率达43.22%，远高于行业平均水平。铜产品毛利率33.6%-37.65%，盈利能力强劲。

## 四、行业地位与产业链定位

### 产业链定位

```
新能源电池材料产业链：
铜钴矿/钴中间品 → 冶炼加工 → 【腾远钴业】→ 钴盐/四氧化三钴/三元前驱体 → 正极材料 → 动力电池
                    ↑
            钴+铜+镍+锂+电池材料
            刚果金+国内双基地
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 硫酸钴市占率约7%（2022年国内第三，仅次于华友17%、格林美13%）；氯化钴市占率约10%（与华友并列第三） |
| **竞争优势** | 钴湿法冶炼成本优势、铜钴协同、刚果金原料布局 |
| **客户结构** | 与行业内多家知名电池厂、材料厂建立长期合作 |
| **供应商** | 全球钴镍原料供应商、刚果腾远当地铜钴矿 |

### 竞争格局

**主要竞争对手**：
- **华友钴业**：钴镍锂一体化龙头，硫酸钴市占率17%
- **格林美**：氯化钴市占率26%，硫酸钴13%
- **寒锐钴业**：钴粉全球市占率前三
- **中金格派、新时代中能**：其他主要竞争者

**公司优势**：
腾远钴业为国内领先钴盐生产企业，钴湿法冶炼成本优势明显，铜产品毛利率高贡献稳定利润。2024年新增四氧化三钴、三元前驱体，向下游延伸。刚果金年产3万吨铜、2000吨钴湿法冶炼项目已获投资证书，预计2026年四季度投产，铜冶炼产能将达9万吨。

## 五、投资亮点

### 核心亮点

- ✅ **钴价上行受益**：氯化钴均价5.5万元/吨同比+45%，刚果金出口政策支撑钴价，公司钴产品弹性大

- ✅ **铜钴产能扩张**：已具备铜6万吨、钴2.65万金属吨产能，规划扩产至铜9万吨、钴4.6万吨；刚果金3万吨铜、2000吨钴项目预计2026年Q4投产

- ✅ **技术成本优势**：钴湿法冶炼成本低于同行，铜产品毛利率33.6%远高于国内同行

- ✅ **产品结构升级**：2024年新增四氧化三钴、三元前驱体，切入消费电池与动力电池赛道，提升附加值

- ✅ **业绩高增长**：2024年归母净利+81.2%，2025年前三季度+21.65%，第三季度单季+47.84%

### 成长驱动因素

1. **钴价上行周期**
   - 刚果金配额政策影响，钴供应收紧
   - AI、低空经济等新兴领域催生钴需求

2. **产能释放**
   - 刚果金项目2026年Q4投产
   - 国内电解钴、四氧化三钴项目推进

3. **产品延伸**
   - 四氧化三钴、三元前驱体放量
   - 提升产业链附加值

### 催化剂

- 钴价维持高位或继续上涨
- 刚果金铜钴项目投产
- 四氧化三钴、三元前驱体产能释放

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **刚果金政策风险** | 资源国政策变动影响原料供应和项目推进 |
