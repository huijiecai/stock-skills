## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 赤峰吉隆黄金矿业股份有限公司 |
| 证券代码 | 600988（A股）/ 06693（H股） |
| 上市板块 | 上海证券交易所（主板）、香港联交所 |
| 成立时间 | 1998年（前身赤峰吉隆矿业） |
| 上市时间 | 2012年（A股）、2025年（H股） |
| 所属行业 | 有色金属-贵金属与采矿 |
| 市场地位 | 中国最大民营黄金生产商，第三梯队潜力企业 |

**公司概况**：
赤峰黄金是中国最大的民营黄金生产商，国内唯一拥有两座海外金矿的民营黄金矿企。公司业务覆盖中国、东南亚、西非三大区域，运营7座黄金与多金属矿山及1个资源综合回收项目，2024年矿产金产量15.16吨，以高成长、高效率、低成本定义黄金行业新标杆。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **合质金/金精粉** | 黄金采选产品，2024年产量15.16吨 | 珠宝首饰、金融投资、电子元件 |
| **电解铜** | 万象矿业伴生铜，塞班金铜矿 | 电力电缆、新能源、建筑 |
| **锌铅锌钼** | 瀚丰矿业多金属矿 | 镀锌、合金、特种钢 |
| **资源回收** | 广源科技废弃电器电子产品拆解 | 循环经济 |

### 业务模式

公司采用"以金为主"战略，聚焦黄金采选核心业务。盈利模式以矿产金销售为主，充分享受金价上涨弹性。通过国内外矿山并购、技改扩产提升产能，成本管控优异，全维持成本处于全球较低水平。

### 技术路线/核心技术

- **低成本采矿技术**：2024年上半年矿产金维持成本264.67元/克（约1158.6美元/盎司），同比下降0.9%，显著低于全球平均1388美元/盎司
- **海外矿山运营**：老挝万象矿业、加纳金星瓦萨成熟运营
- **产能扩张**：塞班铜金矿地下采矿产能从53.6万吨提升至80.6万吨（2025年）
- **资源增储**：2023年保有黄金资源量454吨，占国内总量约18%

**技术优势**：
公司运营效率提升领先全球黄金行业平均水平，黄金全维持成本(AISC)低于全球平均水平，持续彰显"高效率低成本"的竞争优势。

## 四、行业地位与产业链定位

### 产业链定位

```
有色金属产业链：
矿产资源勘探 → 矿产开采采选 → 【赤峰黄金】→ 冶炼加工 → 下游制造 → 终端应用
                    ↑
              矿产金、铜、锌铅钼
              国内外7座矿山
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 中国最大民营黄金生产商；2023年保有黄金资源量454吨，占国内总量约18%；国内唯一拥有两座海外金矿的民营黄金矿企 |
| **竞争优势** | 成本优势突出、国际化布局领先、资源储量丰富 |
| **客户结构** | 面向全球金属贸易商、冶炼厂 |
| **供应商** | 自有矿山为主，海外矿山贡献主要产量 |

### 竞争格局

**主要竞争对手**：
- **紫金矿业**：全球龙头，矿产金73吨
- **山东黄金**：国内黄金龙头
- **中金黄金**：央企黄金平台
- **招金矿业**：港股黄金企业

**公司优势**：
民营黄金矿企中国际化布局最领先；成本管控优异，全维持成本低于全球平均；纯金矿属性，金价弹性最强；2025年实现H股上市，融资渠道拓宽。

## 五、投资亮点

### 核心亮点

- ✅ **成本优势突出**：2024年上半年矿产金维持成本264.67元/克，同比降0.9%，显著低于全球生产成本1388美元/盎司

- ✅ **国际化布局领先**：国内唯一拥有两座海外金矿的民营黄金矿企，境外矿山产量占比约74%

- ✅ **资源储量丰富**：国内外共拥有7个矿业项目，海外矿山保有黄金资源量374吨

- ✅ **海外扩产空间大**：塞班铜金矿地下采矿产能2025年提升至80.6万吨；瓦萨金矿规划2028年年产黄金约12.5吨

- ✅ **金价弹性强**：矿产金业务毛利占比约96%，纯金矿属性享受金价上涨最大弹性

### 成长驱动因素

1. **金价上行**
   - 2025年黄金价格持续上涨，央行购金需求强劲（2024年全球央行购金1180吨，同比+12.3%）
   - 美元信用体系弱化，金价有望延续上行趋势

2. **产能释放**
   - 塞班铜金矿产能提升
   - 瓦萨金矿扩产规划

3. **财务结构优化**
   - 资产负债率降至33.85%，财务风险显著降低
   - H股上市拓宽融资渠道

### 催化剂

- 黄金价格维持高位或继续上涨
- 海外矿山产能释放
- 降息周期深化
- 美元走弱
