## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 河南神火煤电股份有限公司 |
| 证券代码 | 000933 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年8月31日 |
| 上市时间 | 1999年8月31日 |
| 所属行业 | 有色金属-电解铝、采掘业-煤炭 |
| 市场地位 | 煤铝双核龙头，电解铝全国前十、煤炭河南第四 |

**公司概况**：
神火股份是煤电铝材一体化企业，主营业务包括煤炭采选、火力发电、电解铝及铝材加工。公司拥有电解铝产能170万吨/年（分布于云南、新疆两大电价洼地）、煤炭产能855万吨/年、铝箔产能14万吨/年。电解铝产量位列全国前十，煤炭产量位列河南省第四，成本优势突出。

*数据来源：公司官网www.shenhuo.com、证券之星、百度百科，2024-2025年*

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **电解铝** | 产能170万吨/年，云南+新疆双基地 | 建筑、交通、包装、电力 |
| **煤炭** | 无烟煤、贫瘦煤，设计产能255万吨，实际年产超320万吨 | 冶金、化工、电力 |
| **铝箔** | 产能14万吨/年（神火新材11.5万+上海铝箔2.5万） | 包装、电池箔、电子 |
| **铝箔坯料** | 云南铝箔一期已投产 | 铝箔加工原料 |
| **冷轧卷** | 2024年收入13.71亿元，同比+2415.50% | 建材、家电 |

### 业务模式

公司采用"煤-电-铝-材"一体化模式，煤炭自给降低电解铝能源成本，电解铝深加工延伸产业链。新疆神火成本稳居行业第一梯队，云南神火电价处于全行业偏低水平。2025年计划生产铝产品170万吨、商品煤720万吨。

### 技术路线/核心技术

- **电解铝冶炼**：云南、新疆两大基地，利用当地低电价优势
- **煤炭采选**：新庄煤矿、葛店煤矿等，无烟煤、贫瘦煤低硫低磷
- **铝箔加工**：电池铝箔（AA1070、AA1235D合金）技术研发，成品率>75%
- **绿电项目**：新疆80万千瓦绿电项目年内并网，进一步降本

**技术优势**：
煤电铝一体化集成优势明显，新疆、云南双基地布局享受电价洼地红利。电池铝箔已完成技术研发，正在推进认证。参股五彩湾露天矿等远期煤炭产能储备。

*数据来源：公司年报、证券之星、开源证券研报，2024-2025年*

## 四、行业地位与产业链定位

### 产业链定位

```
铝产业链：
铝土矿→氧化铝→【神火股份】→电解铝→铝材/铝箔→建筑/交通/包装/电池
                    ↑
            煤-电-铝-材一体化
            云南+新疆双基地
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 电解铝产量全国前十（2025年上半年全国产量2016万吨）；煤炭河南第四 |
| **竞争优势** | 新疆成本行业第一梯队、云南电价偏低；煤电铝一体化降本 |
| **客户结构** | 电解铝面向建筑、交通、包装等；铝箔面向电池、包装 |
| **供应商** | 煤炭自给，氧化铝外购；参股五彩湾露天矿等远期储备 |

### 竞争格局

**主要竞争对手**：
- **中国铝业**：央企铝业龙头，产业链完整
- **云铝股份**：云南电解铝龙头，水电铝
- **南山铝业**：铝加工龙头，高端铝材
- **天山铝业**：新疆电解铝，成本优势

**公司优势**：
神火股份煤铝双核驱动，新疆80万绿电项目并网后成本优势进一步巩固。云南神火2025年有望首次全年满负荷生产。电池铝箔新产能加速释放，打开成长空间。多家券商给予"买入"评级。

## 五、投资亮点

### 核心亮点

- ✅ **煤铝双核驱动**：电解铝+煤炭双主业，业绩互补，抗周期能力强
- ✅ **成本优势突出**：新疆成本行业第一梯队，云南电价偏低；80万绿电项目并网进一步降本
- ✅ **现金流充沛**：2025年前三季度经营现金流68.98亿元，三费占比仅2.5%
- ✅ **股东回报**：2024年拟10派5元；持续推进股份回购，截至2025年9月累计回购2.55亿元
- ✅ **电池铝箔成长**：铝箔产能14万吨/年，电池箔认证推进中，云南二期规划11万吨（择机建设）
- ✅ **Q3盈利弹性释放**：单季净利15.85亿元同比+26.39%，煤炭触底反弹

### 成长驱动因素

1. **电解铝利润弹性**
   - 氧化铝与电解铝价差修复，电解铝利润空间扩大
   - 云南神火2025年有望全年满负荷生产

2. **绿电降本**
   - 新疆80万千瓦绿电项目年内并网，电力成本进一步下降

3. **铝箔产能释放**
   - 电池箔新产能加速释放，高附加值产品占比提升

4. **煤炭价格修复**
   - 2025年Q3煤炭价格明显上涨，盈利能力修复

### 催化剂

- 铝价上涨、氧化铝价格回落
- 煤炭价格企稳反弹
- 电池铝箔认证通过、产能放量
- 绿电项目并网
