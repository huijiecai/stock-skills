## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 云南铜业股份有限公司 |
| 证券代码 | 000878 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1998年5月15日（前身云南冶炼厂1958年成立） |
| 上市时间 | 1998年6月2日 |
| 所属行业 | 有色金属-铜冶炼 |
| 市场地位 | 国内主要铜冶炼企业，阴极铜产能国内第三、全球第四 |

**公司概况**：
云南铜业是中铝集团、中国铜业旗下唯一铜产业上市公司和核心平台，主要业务涵盖铜的勘探、采选、冶炼，贵金属和稀散金属的提取与加工，硫化工以及贸易。拥有6座铜冶炼厂和6座大中型铜矿山，年产阴极铜130万吨、黄金17吨、白银1175吨、工业硫酸500万吨，阴极铜产能在全球排名第4位、国内排名第3位。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **阴极铜** | 产能130-140万吨/年，国内第三、全球第四 | 电力电缆、新能源、建筑、电子 |
| **黄金** | 年产约17吨 | 珠宝首饰、金融投资、电子元件 |
| **白银** | 年产约1175吨 | 光伏、电子、工业 |
| **工业硫酸** | 年产约500万吨 | 化工、化肥、冶金 |
| **稀散金属** | 硒、碲、铂、钯、铼、铟等 | 半导体、光伏、催化剂 |

### 业务模式

公司采用"矿山+冶炼"一体化模式，拥有西南、东南和北方三大冶炼基地，形成从铜矿勘探采选到冶炼加工的产业链。盈利模式以冶炼加工费（TC/RC）为主，辅以铜价波动带来的库存收益；自有矿山（普朗铜矿等）提供部分原料自给。综合回收能力强，稀散金属提取增厚利润。

### 技术路线/核心技术

- **铜冶炼技术**：6座铜冶炼厂，致力于打造"清洁、节能、高效"的铜冶炼产业
- **三大基地布局**：西南、东南、北方冶炼基地，实现铜冶炼产业千亿战略目标
- **综合回收**：贵金属和稀散金属提取与加工，硫化工产业链
- **西南铜业搬迁升级**：2024年6月整体工程进度约84%，电解生产系统已进入带负荷试车阶段

**技术优势**：
公司冶炼产能布局合理，综合回收能力强。西南铜业搬迁升级改造项目将提升冶炼效率和环保水平。作为中铝集团唯一铜上市平台，在资源端有望获得进一步支持。

## 四、行业地位与产业链定位

### 产业链定位

```
有色金属产业链：
矿产资源勘探 → 矿产开采采选 → 冶炼加工 → 【云南铜业】→ 贸易 → 终端应用
                                        ↑
                              矿山+冶炼一体化
                              普朗铜矿+三大冶炼基地
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 阴极铜产能国内第三（仅次于铜陵有色、江西铜业），全球第四 |
| **竞争优势** | 中铝系唯一铜平台、矿山+冶炼一体化、综合回收能力强 |
| **客户结构** | 面向金属贸易商、下游铜加工企业 |
| **供应商** | 自有矿山（普朗铜矿等）提供部分原料，外购铜精矿为主 |

### 竞争格局

**主要竞争对手**：
- **江西铜业**：铜冶炼龙头，产能规模更大
- **铜陵有色**：全产业链布局，阴极铜产量国内第二
- **紫金矿业**：矿产端龙头，产业链环节不同

**公司优势**：
云南铜业为中铝集团旗下唯一铜上市平台，未来在资源端有望获得进一步支持。2025年收购凉山矿业40%股份，矿山铜产量从6万吨增加至10万吨；中铝集团金龙铜矿（20万吨一期产能）将在2028年投产，提供成长空间。

## 五、投资亮点

### 核心亮点

- ✅ **大股东资产注入**：2025年7月公告收购凉山矿业40%股份，矿山铜产量从6万吨增加至10万吨；中铝集团金龙铜矿（20万吨一期产能）将在2028年投产

- ✅ **上游资源优势**：拥有铜资源储量476万吨，主力矿山普朗铜矿年产铜3.5-4万吨，是国内规模最大的地下有色矿山

- ✅ **冶炼产能行业领先**：阴极铜产能140万吨，国内排名第三，冶炼产能布局合理，综合回收能力强

- ✅ **中铝系唯一铜平台**：作为中铝集团旗下唯一铜产业上市公司，未来在资源端有望获得进一步支持

- ✅ **西南铜业搬迁升级**：项目整体工程进度约84%，投产后将提升冶炼效率和环保水平

### 成长驱动因素

1. **资源端扩张**
   - 凉山矿业收购完成，矿山铜产量提升
   - 金龙铜矿2028年投产预期

2. **冶炼产能释放**
   - 西南铜业搬迁升级项目投产
   - 三大冶炼基地协同

3. **加工费修复**
   - 铜精矿TC/RC若企稳回升，冶炼利润将改善

### 催化剂

- 凉山矿业收购完成并整合
- 铜精矿加工费（TC/RC）企稳回升
- 西南铜业搬迁升级项目投产
- 中铝集团进一步资产注入

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **加工费大幅下降** | 2024年四季度因精矿冶炼加工费大幅下降，归母净利润转亏；毛利率、净利率同比降逾30% |
| ⚠️ **铜价波动** | 盈利与铜价、TC/RC、硫酸价格密切相关；若铜价跌破成本线（如7,000美元/吨）将面临压力 |
| ⚠️ **现金流压力** | 2024年经营现金流净额1.45亿元，同比降97.8%，需关注流动性 |
| ⚠️ **技术替代风险** | 若高温超导等技术实现商业化，或铝代铜实现，估值可能面临下行压力 |
| ⚠️ **资源品位下降** | 全球铜矿品位下降趋势，新项目投产周期长 |

*数据来源：财中社、国信证券研报、公司公告，2024-2025年*
