## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国稀土集团资源科技股份有限公司（原五矿稀土股份有限公司） |

| 上市时间 | 1998年9月11日 |
| 所属行业 | 有色金属-稀土 |
| 市场地位 | 中国稀土集团唯一上市平台，全球中重稀土领军企业，第三梯队潜力企业 |

**公司概况**：
五矿稀土（中国稀土）是中国稀土集团旗下唯一上市平台，主要从事稀土产品的开采、冶炼加工及销售。公司拥有广东梅州、湖南江华、福建宁化、云南陇川等地稀土采矿权，覆盖全国十个省市。2022年国务院批准稀土资产战略性重组，中国稀土集团整合五矿、中铝、赣州稀土等资产，公司作为集团唯一上市平台，享有资产注入预期。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **氧化镨钕** | 轻稀土氧化物，稀土永磁核心原料 | 钕铁硼永磁、新能源汽车电机 |
| **氧化镝** | 中重稀土氧化物 | 高性能钕铁硼、风电、军工 |
| **氧化铽** | 中重稀土氧化物 | 高性能永磁、荧光粉 |
| **稀土氧化物** | 其他稀土氧化物及盐类 | 催化、抛光、储氢等 |
| **稀土金属** | 稀土金属及合金 | 永磁材料、储氢材料 |

### 业务模式

公司采用"采矿+冶炼分离+销售"一体化模式，依托中国稀土集团资源整合优势，拥有南方离子型稀土采矿权和冶炼分离产能。产品主要销售给下游永磁材料、催化材料等企业。盈利模式以稀土氧化物销售为主，受稀土价格波动影响大。

### 技术路线/核心技术

- **离子型稀土冶炼**：南方离子型稀土分离加工，中重稀土特色
- **绿色冶炼**：稀土冶炼环保升级
- **深加工布局**：推进稀土深加工项目，提升高附加值产品占比

**技术优势**：
公司为国内最大的南方离子型稀土分离加工企业之一，中重稀土资源禀赋突出。中国稀土集团拥有全国28.96%冶炼分离指标、29.62%矿产品开采指标，集团资源整合后协同效应显著。

## 四、行业地位与产业链定位

### 产业链定位

```
稀土产业链：
稀土矿开采 → 冶炼分离 → 【五矿稀土/中国稀土】→ 稀土氧化物/金属 → 永磁材料 → 新能源汽车/风电
                    ↑
            中国稀土集团唯一上市平台
            中重稀土领军
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 中国稀土集团拥有全国28.96%冶炼分离指标、29.62%矿产品开采指标；公司为集团唯一上市平台，营收规模10-50亿第二梯队 |
| **竞争优势** | 央企背景、集团资源整合、中重稀土资源、资产注入预期 |
| **客户结构** | 下游永磁材料、催化材料等企业 |
| **供应商** | 集团内稀土矿、自有采矿权 |

### 竞争格局

**行业格局**：
- **北方稀土**：轻稀土龙头，开采配额占全国约70%，冶炼分离占67%
- **中国稀土集团**：中重稀土主导，2026年1月控股广晟有色，南方稀土整合基本完成
- **双雄格局**：北方稀土（轻稀土）+ 中国稀土集团（中重稀土），"南重北轻"

**主要竞争对手**：
- **北方稀土**：轻稀土绝对龙头
- **盛和资源**：稀土业务营收规模最大（百亿级）
- **广晟有色**：中国稀土集团控股，南方中重稀土
- **厦门钨业**：稀土+钨双主业

**公司优势**：
五矿稀土（中国稀土）为中国稀土集团唯一上市平台，享有集团资产注入预期。2026年广晟有色纳入集团后，南方稀土整合完成，形成"南北互补"格局。公司推进稀土深加工项目，积极布局海外市场。

## 五、投资亮点

### 核心亮点

- ✅ **央企唯一上市平台**：中国稀土集团唯一上市平台，集团承诺五年内解决同业竞争，资产注入预期强

- ✅ **中重稀土领军**：南方离子型稀土分离加工龙头，中重稀土（镝、铽等）资源禀赋突出，控制全国超20%镝铽关键资源

- ✅ **2025年扭亏为盈**：稀土价格回升驱动，一季度扭亏，前三季度归母净利1.92亿同比+194.67%

- ✅ **集团整合红利**：2026年广晟有色纳入集团，南方稀土整合完成，战略协同增强

- ✅ **深加工+海外布局**：推进稀土深加工项目提升附加值，积极布局海外市场

### 成长驱动因素

1. **稀土价格回升**
   - 2024年价格大幅下跌后，2025年有所反弹
   - 稀土战略资源属性，政策支撑

2. **资产注入预期**
   - 集团五年内解决同业竞争承诺
   - 广晟有色等资产整合预期

3. **深加工拓展**
   - 提升高附加值产品占比
   - 海外市场拓展

### 催化剂

- 《稀土管理条例》等政策出台
- 集团资产注入进展
- 稀土价格持续回升
- 子公司产能释放
