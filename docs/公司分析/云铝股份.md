## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 云南铝业股份有限公司 |
| 证券代码 | 000807 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1970年（前身为云南铝厂） |
| 上市时间 | 1998年4月8日 |
| 所属行业 | 有色金属-铝 |
| 市场地位 | 国内绿色铝龙头，电解铝产能全国前三 |

**公司概况**：
云铝股份是国内最大的绿色低碳铝供应商，拥有305万吨电解铝产能全部位于云南，水电占比超80%，绿色铝碳排放仅为煤电铝的20%。2018年纳入中国铝业集团管理体系，实际控制人为国务院国资委。公司构建了完整的"铝土矿-氧化铝-炭素制品-铝冶炼-铝加工"一体化产业链。

*数据来源：公司官网、百度百科、东方财富，2025年*

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **电解铝/原铝** | 2024年产量293.83万吨，同比+22.45%，产能利用率96.3% | 建筑、交通、包装、电力 |
| **氧化铝** | 2024年产量140.88万吨，产能利用率100.6% | 电解铝原料 |
| **铝合金及加工品** | 2024年产量125.4万吨，同比+1.75% | 新能源车、光伏、电子 |
| **炭素制品** | 2024年产量80.6万吨，同比+2.85% | 电解铝预焙阳极 |
| **铝土矿** | 自给率51% | 氧化铝原料 |

### 业务模式

公司采用"铝土矿-氧化铝-炭素-电解铝-铝加工"全产业链一体化经营模式。盈利主要来自电解铝及铝加工产品销售，享受铝价上涨带来的利润弹性。铝土矿自给率51%、氧化铝自给率24%、预焙阳极自给率超100%，成本管控能力强。云南电价处于全国低位，单吨电解铝电力成本较新疆低1272元、较山东低2740元。

### 技术路线/核心技术

- **绿色铝技术**：水电占比80%+，碳足迹认证和全产业链ASI认证，受益欧洲碳关税政策
- **循环经济**：铝灰/大修渣100%资源化，赤泥综合利用率25%
- **节能降耗**：2025年前三季度吨铝电耗降低418千瓦时
- **金属镓回收**：建成金属镓回收示范生产线，回收率达84.6%

**技术优势**：
公司是国内首家直接输出ASI（铝业管制倡议）铝供应商，主要产品碳足迹处于国际领先水平。"绿色铝 在云铝"品牌影响力不断扩大，19吨铝合金焊材直接出口欧洲。

*数据来源：公司年报、研报、中国有色网，2024-2025年*

## 四、行业地位与产业链定位

### 产业链定位

```
铝产业链：
铝土矿 → 氧化铝 → 炭素 → 【云铝股份】→ 铝加工 → 终端应用
                          ↑
                    电解铝+氧化铝+铝加工
                    绿色水电铝龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 绿色铝产能305万吨，国内最大绿色低碳铝供应商，电解铝规模全国前三 |
| **竞争优势** | 水电铝成本优势、全产业链一体化、碳足迹认证、ASI认证 |
| **客户结构** | 面向建筑、交通、新能源、电子等下游，产品出口欧洲 |
| **供应商** | 铝土矿自给率51%，氧化铝自给率24%，预焙阳极自给率超100% |

### 竞争格局

**主要竞争对手**：
- **中国铝业**：铝业央企龙头，氧化铝、电解铝规模更大
- **南山铝业**：山东铝业龙头，汽车板等高端加工
- **神火股份**：河南电解铝，煤电铝为主

**公司优势**：
云铝股份以绿色水电铝为核心差异化，受益碳中和政策及欧洲碳关税，低碳铝溢价优势明显。云南电价低位+全产业链自给，成本管控优于多数同行。

## 五、投资亮点

### 核心亮点

- ✅ **绿色铝龙头地位**：国内最大绿色铝供应商，水电占比80%+，碳足迹认证完善，受益欧洲碳关税及国内碳中和政策

- ✅ **成本优势显著**：云南电价全国低位，铝土矿自给率51%、预焙阳极自给率超100%，单吨电力成本较山东低2740元

- ✅ **产能释放+需求增长**：2024年原铝产量293.83万吨同比+22.45%，2025年计划铝产品429万吨；新能源用铝需求持续增长

- ✅ **分红比例抬升**：2024年分红比例32.2%，2025年中期分红比例达40.1%，彰显经营信心

- ✅ **财务结构优化**：资产负债率23.28%，经营性现金流69.77亿元（2025年前三季度同比+25.19%）

### 成长驱动因素

1. **氧化铝价格回落**
   - 氧化铝价格下行有利于电解铝环节利润修复

2. **绿色铝溢价**
   - 碳中和背景下低碳铝需求增长，欧洲碳关税推动绿色铝出口

3. **产能与技改**
   - 金属镓项目建成投产，布局战新产业；昭通太阳坝铝土矿等资源端拓展

### 催化剂

- 氧化铝价格回落、铝价维持高位
- 欧洲碳关税正式实施，绿色铝出口放量
- 新能源、光伏用铝需求超预期
