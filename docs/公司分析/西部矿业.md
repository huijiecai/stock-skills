## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 西部矿业股份有限公司 |
| 证券代码 | 601168 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2000年12月28日 |
| 上市时间 | 2007年7月12日 |
| 所属行业 | 有色金属-采矿与冶炼 |
| 市场地位 | 西部地区矿企龙头，中国第五大铜精矿生产商、第二大铅锌精矿生产商 |

**公司概况**：
西部矿业是西部地区有色金属采选冶龙头企业，控股并经营15座矿山（6座有色金属矿山、8座铁及铁多金属矿山、1座盐湖矿山），拥有8个探矿权和13个采矿权。主力玉龙铜矿为国内第二大单体铜矿，2024年上半年对公司利润贡献超过九成。矿产铜、铅锌、钼产量位居国内前列，矿山均位于国内，地缘政治风险较低。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **矿产铜** | 玉龙铜矿、获各琦铜矿等，2024年矿产铜约15万吨级 | 电力电缆、新能源、电子 |
| **矿产铅锌** | 锡铁山、会东大梁、呷村银多金属矿等 | 镀锌、电池、合金 |
| **矿产钼** | 玉龙铜矿伴生，2023年产量3401吨 | 特种钢、合金 |
| **冶炼铜** | 2024年冶炼铜产量大幅增长 | 铜材加工、下游制造 |
| **铁矿** | 4座铁矿山，计划"十四五"末打造千万吨级铁资源基地 | 钢铁、建筑 |
| **盐湖化工** | 西部镁业镁盐储量3046.25万吨，涉及盐湖提镁、提锂等 | 镁合金、新能源 |

### 业务模式

公司采用"矿山采选+有色冶炼"双轮驱动模式，以自有矿山开采为主，享受金属价格上涨带来的利润弹性；冶炼业务作为补充。盈利模式以矿产金属销售为核心，玉龙铜矿为利润主要来源。通过玉龙三期扩建、铁矿基地建设等持续扩张产能。

### 技术路线/核心技术

- **玉龙铜矿开发**：国内第二大单体铜矿，特大型斑岩和接触交代混合型铜矿床，伴生钼矿；二期已投产，三期扩建推进中
- **多金属综合回收**：铜、铅、锌、钼、银等多金属综合循环利用
- **盐湖提镁/提锂**：西部镁业盐湖化工，涉及氟化工和磷酸铁锂产业链
- **节能降碳**：2024年万元产值能耗同比降21.1%，清洁能源占比提升至67.7%

**技术优势**：
公司矿产铜、矿产钼、冶炼铜产量大幅增产，位居国内特大型矿山企业前列。2024年研发投入6.2亿元，矿山回收率持续提升（玉龙铜矿一车间铜、钼回收率分别提升4.08、5.36个百分点）。

## 四、行业地位与产业链定位

### 产业链定位

```
有色金属产业链：
矿产资源勘探 → 【西部矿业】矿产开采采选 → 冶炼加工 → 下游制造 → 终端应用
                    ↑
              玉龙铜矿+铅锌矿+铁矿
              矿产端为主，冶炼为辅
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 中国第五大铜精矿生产商、第二大铅锌精矿生产商；钼精矿行业主要生产商 |
| **竞争优势** | 西部资源禀赋（西部蕴藏中国59%锌、55%铅、65%铜资源）、玉龙铜矿国内第二大单体铜矿 |
| **客户结构** | 面向冶炼厂、金属贸易商、下游制造企业 |
| **供应商** | 自有矿山为主，资源自给率高 |

### 竞争格局

**主要竞争对手**：
- **紫金矿业**：矿产铜、金龙头，规模更大
- **洛阳钼业**：多金属矿业，铜钴领先
- **江西铜业**：铜冶炼龙头，产业链环节不同

**公司优势**：
西部矿业以矿产端为核心，玉龙铜矿利润贡献超九成，享受金属价格上涨弹性。矿山均位于国内（青海、西藏、四川等），地缘政治风险低。玉龙三期扩建推进中，采选规模将从2280万吨/年增至3000万吨/年，铜金属产量从15万吨增至20万吨。

## 五、投资亮点

### 核心亮点

- ✅ **玉龙铜矿核心驱动**：国内第二大单体铜矿，2024H1利润贡献超九成；二期已投产，三期扩建获自治区批复，采选规模将增至3000万吨/年，铜产量增至20万吨

- ✅ **多矿产品量价齐升**：2025年H1矿产铜、锌、铅、钼产量均实现大幅增长；玉龙铜业净利润34.91亿元，同比+36.99%

- ✅ **高分红政策**：2024年分红比例达81%，共分配23.83亿元，股东回报优厚

- ✅ **地缘风险低**：矿山均位于国内，无海外政治风险

- ✅ **冶炼端利润修复**：2025年H1冶炼铜产量18.22万吨，同比+49.94%；冶炼亏损从2024年H2的7.88亿元收窄至4.26亿元

### 成长驱动因素

1. **玉龙三期扩建**
   - 采选规模从2280万吨/年增至3000万吨/年
   - 铜金属产量从15万吨增至20万吨
   - 已完成九项报告编制并取得评审意见

2. **多矿协同**
   - 铁矿"十四五"末千万吨级基地
   - 铅锌矿资源增储（2023年新增268万吨）

3. **金属价格**
   - 铜、铅、锌、钼价格维持高位有利于业绩

### 催化剂

- 玉龙三期项目获批并开工建设
- 铜、铅、锌、钼价格维持高位
- 冶炼端亏损继续收窄
- 铁矿基地建设进展

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **项目进度不及预期** | 玉龙三期、铁矿基地等扩建项目若延期将影响产能释放 |
| ⚠️ **金属价格下跌** | 铜、铅、锌、锂等金属价格下跌将直接影响矿产端利润 |
| ⚠️ **分红比例下调** | 2024年分红比例81%较高，未来若下调可能影响投资者预期 |
| ⚠️ **产能爬坡不及预期** | 新矿山或扩建项目达产节奏若慢于预期将影响业绩 |
| ⚠️ **安全生产风险** | 矿山开采存在安全风险，事故可能影响生产和声誉 |

*数据来源：西部证券、九方智投研报、公司公告，2024-2025年*
