## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 南京寒锐钴业股份有限公司 |
| 证券代码 | 300618 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 1997年5月12日 |
| 上市时间 | 2017年3月6日 |
| 所属行业 | 有色金属-稀有金属（钴） |
| 市场地位 | 全球第三大、中国最大钴粉生产商，第三梯队潜力企业 |

**公司概况**：
寒锐钴业是国内少数拥有完整钴产业链的企业之一，业务涵盖矿石收购、粗加工、冶炼提纯、钴粉及钴化合物生产。按2020年产量计算，公司为全球第三大及中国最大的钴粉生产商，产品广泛应用于电池材料、硬质合金、高温合金等领域，受益于新能源汽车及电子电器行业发展。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **钴粉** | 核心产品，全球市占率前三 | 硬质合金、高温合金、磁性材料 |
| **氢氧化钴** | 钴中间品 | 电池材料前驱体 |
| **电积钴** | 精炼钴产品 | 电池、合金 |
| **钴化合物** | 钴酸锂、四氧化三钴等 | 锂电池正极材料 |
| **电积铜** | 伴生铜产品，主要盈利来源之一 | 铜管、铜线、铜铸件 |

### 业务模式

公司采用垂直整合的产业链布局，从矿石收购、粗加工、冶炼提纯到钴粉及钴化合物生产形成完整产业流程。盈利模式以钴产品、铜产品销售为主，铜产品因规模较大成为重要利润贡献；产业链布局确保原材料供应稳定和成本竞争力。

### 技术路线/核心技术

- **钴粉制备技术**：钴粉产品在全球市场占有率排名前三
- **完整产业链**：从矿山勘探、采选、冶炼到钴粉生产的垂直整合
- **原材料保障**：以具竞争力的条款取得原材料供应
- **产品多元化**：钴粉、氢氧化钴、电积钴、钴化合物等全系列

**技术优势**：
公司技术研发能力强、产业链完整、国际品牌认可度高，尤其在钴粉领域具有专业化优势。产业链布局使公司能够确保成本及运营效益，获得产业链上更多的附加值。

## 四、行业地位与产业链定位

### 产业链定位

```
钴产业链：
钴矿开采 → 粗加工/氢氧化钴 → 【寒锐钴业】→ 钴粉/钴化合物 → 下游应用
                    ↑
              钴粉、氢氧化钴、电积钴、电积铜
              完整产业链：矿石→冶炼→深加工
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球第三大、中国最大钴粉生产商（按2020年产量）；钴粉领域专业化优势明显 |
| **竞争优势** | 产业链完整、钴粉技术领先、国际品牌认可 |
| **客户结构** | 电池材料、硬质合金、高温合金、磁性材料等下游企业 |
| **供应商** | 自有产业链保障原材料供应，刚果(金)等钴矿来源 |

### 竞争格局

**主要竞争对手**：
- **华友钴业**：国内最大钴产品供应商，2022年产量38,498吨，销量23,058吨
- **格林美**：硫酸钴、氯化钴产量领先
- **腾远钴业**：钴产品竞争者

**公司优势**：
寒锐钴业在钴粉领域具有专业化优势，产销规模虽不及华友钴业，但钴粉细分市场地位稳固。产业链完整确保成本竞争力，技术研发能力强。

## 五、投资亮点

### 核心亮点

- ✅ **钴粉龙头地位**：全球第三大、中国最大钴粉生产商，钴粉细分市场具有专业化优势

- ✅ **完整产业链**：垂直整合从矿石到钴粉的完整产业流程，确保原材料供应稳定和成本竞争力

- ✅ **新能源受益**：钴产品应用于锂电池正极材料，电动汽车行业发展带动需求增长

- ✅ **铜产品贡献**：电积铜为主要盈利产品之一，2024年铜产品毛利率21.66%，利润贡献显著

- ✅ **财务改善**：2024-2025年营收、净利润持续增长，2025年前三季度归母净利润同比+42.57%

### 成长驱动因素

1. **新能源需求**
   - 电动汽车、储能对钴需求增长
   - 三元锂电池对钴的依赖（高镍低钴趋势需关注）

2. **产业链整合**
   - 垂直整合带来成本优势
   - 产品结构优化

3. **产能释放**
   - 钴铜产销增长
   - 产品结构优化

### 催化剂

- 钴价企稳回升
- 新能源汽车销量超预期
- 新产品/新客户拓展
- 产能扩张项目投产

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **钴价波动风险** | 钴价受供需影响波动大，价格下跌将显著影响钴产品毛利率；2025年上半年钴产品毛利率仅5.14% |
| ⚠️ **扣非净利润下滑** | 2025年上半年扣非净利润同比下降17.48%，反映核心盈利能力有所弱化 |
| ⚠️ **应收账款风险** | 应收账款占归母净利润比例高达101.11%，存在回款风险或利润虚增隐患 |
| ⚠️ **流动性风险** | 货币资金/流动负债比率仅90.46%，短期偿债能力偏紧 |
| ⚠️ **高镍低钴趋势** | 锂电池正极材料向高镍低钴方向发展，长期可能影响钴需求 |
| ⚠️ **海外资源风险** | 刚果(金)等钴矿来源国存在政治、政策风险 |

*数据来源：寒锐钴业2025年中报财务分析、证券之星，2025年*
