## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 盛和资源控股股份有限公司 |
| 证券代码 | 600392 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998年7月1日（前身太原理工天成科技） |
| 上市时间 | 2003年5月29日（2012年借壳上市后更名） |
| 所属行业 | 有色金属-稀土矿采选、冶炼分离、锆钛 |
| 市场地位 | 稀土行业第二梯队龙头，国内唯一具有国内外战略布局的上市稀土企业 |

**公司概况**：
盛和资源是一家从事稀土矿开采、冶炼分离和加工的企业，拥有集稀土"矿山、分离、冶炼、回收"为一体的完整产业链。公司拥有四川和江西两个稀土冶炼分离基地，冶炼分离产能约1.5万吨/年；同时布局锆钛矿选矿业务。2025年拟收购澳大利亚Peak公司100%股权，获得坦桑尼亚Ngualla稀土矿（全球最大、品位最高、成本最低的稀土矿床之一），全球化资源布局成效凸显。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **稀土精矿** | 稀土矿采选产品 | 冶炼分离原料 |
| **稀土氧化物** | 氧化镨钕、氧化镧、氧化铈等 | 永磁材料、催化、抛光 |
| **稀土化合物** | 各类稀土盐类 | 功能材料、催化剂 |
| **稀土金属** | 镨钕金属、镧铈金属等 | 钕铁硼永磁、储氢合金 |
| **锆钛产品** | 锆英砂、钛精矿等 | 陶瓷、涂料、钛白粉 |

### 业务模式

公司采用"矿山+分离+冶炼+回收"一体化模式，通过自有稀土矿开采、冶炼分离、金属加工及锆钛矿选矿获取收入。盈利模式以稀土产品销售为主，享受稀土价格上涨带来的利润弹性；锆钛业务协同发展。通过海外并购（Peak公司、Ngualla项目）拓展资源版图。

### 技术路线/核心技术

- **稀土冶炼分离**：四川、江西两大基地，产能约1.5万吨/年
- **稀土废料回收**：回收利用环节完善产业链
- **锆钛选矿**：Fungoni项目产能提升至15万吨/年
- **海外资源开发**：Ngualla项目（坦桑尼亚）预计2026-2027年投产

**技术优势**：
公司拥有相对完善的产业链体系，覆盖采矿、金属冶炼、分离加工、稀土废料回收等多个重要环节。中国唯一一家具有国内外战略布局的上市稀土企业，全球化资源布局成效凸显。

## 四、行业地位与产业链定位

### 产业链定位

```
稀土产业链：
稀土矿开采 → 冶炼分离 → 【盛和资源】→ 稀土金属/功能材料 → 永磁、催化、抛光等
                    ↑
            稀土+锆钛
            国内外双布局
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 稀土行业第二梯队龙头，2025年前三季度营收104.56亿元、净利8.22亿元，行业排名第二（北方稀土第一：营收302.92亿、净利21.07亿） |
| **竞争优势** | 国内外战略布局、完整产业链、Ngualla优质资源、资产负债率32.59%低于行业平均 |
| **客户结构** | 面向永磁材料、催化、抛光等下游客户 |
| **供应商** | 自有矿山+海外并购，Ngualla项目投产后资源自给率提升 |

### 竞争格局

**主要竞争对手**：
- **北方稀土**：稀土行业绝对龙头，掌控白云鄂博稀土资源，2025年前三季度营收302.92亿、净利21.07亿
- **中国稀土**：中重稀土龙头
- **广晟有色**：广东稀土资源

**公司优势**：
盛和资源为国内唯一具有国内外战略布局的上市稀土企业，完成收购Peak公司（Ngualla项目）后，将拥有全球最大、品位最高、成本最低的稀土矿床之一。估值相对北方稀土更低（国元证券给予2025年46.3倍PE），成长性更优。

## 五、投资亮点

### 核心亮点

- ✅ **全球化资源布局成效凸显**：拟斥资7.43亿元收购Peak公司100%股权，获得坦桑尼亚Ngualla稀土矿（资源量461万吨REO，储量88.7万吨，品位4.8%，镨钕占比21.26%），预计2026-2027年投产，年产1.8万吨稀土精矿

- ✅ **稀土价格回升驱动业绩改善**：2025年氧化镨钕价格较年初上涨35.93%；Q1净利同比+178.09%，主要因稀土价格回升、转回存货减值；机构预计2025年稀土价格在供需缺口缩小背景下逐步上涨

- ✅ **锆钛业务协同发展**：Fungoni锆钛项目产能提升至15万吨/年，多元化布局降低单一品种风险

- ✅ **财务结构稳健**：资产负债率32.59%，低于行业平均34.32%；2025年Q1毛利率7.98%同比+7.88pct，净利率5.97%同比+13.82pct

- ✅ **估值相对合理**：国元证券给予2025年46.3倍PE，当前股价22.19元；北方稀土2025年PE约51-59倍，盛和资源估值更具吸引力

### 成长驱动因素

1. **稀土价格上行**
   - 供需缺口缩小，稀土价格逐步上涨
   - 氧化镨钕等产品价格回升

2. **Ngualla项目投产**
   - 2026-2027年投产，年产1.8万吨稀土精矿
   - 全球顶级资源，成本优势明显

3. **锆钛业务放量**
   - Fungoni项目产能15万吨/年
   - 多元化收入来源

### 催化剂

- Ngualla项目收购完成及投产
- 稀土价格维持上涨趋势
- 稀土战略价值提升、政策支持

## 六、风险提示

| 风险类型 | 具体内容 |

| ⚠️ **稀土价格波动风险** | 2024年稀土、锆钛产品价格同比大幅下降导致业绩下滑；价格受供需和宏观影响 |
