## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 洛阳栾川钼业集团股份有限公司 |
| 证券代码 | 603993（A股）/ 03993（H股） |
| 上市板块 | 上海证券交易所（主板）、香港联交所 |
| 成立时间 | 1999年12月22日 |
| 上市时间 | 2007年4月26日（H股）/ 2012年10月9日（A股） |
| 所属行业 | 有色金属矿采选业 |
| 市场地位 | 全球铜钴钼钨铌龙头 |

**公司概况**：
洛阳钼业是全球领先的铜、钴、钼、钨、铌生产商，铜产量全球前十，钴和铌产量全球前二。公司采用"矿山+贸易"双驱动模式，业务分布于亚洲、非洲、南美洲和欧洲，拥有刚果（金）TFM、KFM世界级铜钴矿，2024年营收首次突破2000亿元，净利润首次突破百亿元，位列《财富》中国500强第145位。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **铜** | 电解铜、铜精矿 | 电力电缆、新能源、建筑、电子 |
| **钴** | 氢氧化钴、钴金属 | 三元锂电池、硬质合金、高温合金 |
| **钼** | 钼精矿、氧化钼 | 特钢、石化催化剂、合金 |
| **钨** | 钨精矿、仲钨酸铵 | 硬质合金、国防军工、切削工具 |
| **铌** | 铌铁、铌氧化物 | 特种钢、超导材料 |
| **磷肥** | 磷矿、磷肥 | 农业化肥（巴西地区） |
| **黄金** | 金矿 | 贵金属（厄瓜多尔地区） |
| **金属贸易** | 埃珂森IXM | 全球金属贸易，位居全球前三 |

### 业务模式

公司采用"矿山+贸易"双驱动模式。矿山板块贡献约90%归母净利润，依托刚果（金）TFM/KFM铜钴矿、中国钼钨矿、巴西铌磷矿等全球优质资源；贸易板块通过埃珂森（IXM）开展金属贸易，2024年创历史新高达13.53亿元，同比增48.2%。

### 技术路线/核心技术

- **铜钴矿开发**：TFM、KFM为全球最大高品位铜钴项目之一，铜矿现金成本处全球最低25%分位
- **多金属协同**：铜钴伴生矿综合利用，KFM二期深部硫化矿开发
- **ESG认证**：TFM通过TheCopperMark铜标志再认证，非洲首家获认证矿山；WindESG"AAA"评级

**技术优势**：
刚果（金）TFM、KFM矿山拥有65万吨以上年铜产能，钴产量2024年11.42万吨同比增106%。公司成本控制能力突出，铜矿现金成本处于全球最低分位，具备显著的成本竞争优势。

## 四、行业地位与产业链定位

### 产业链定位

```
矿产资源勘探开采
    ↓
【铜钴矿采选】→ 【洛阳钼业】→ 冶炼加工 → 终端应用
    ↑                    ↑
刚果金TFM/KFM      埃珂森贸易（全球前三）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 钴产量全球第一，铜产量全球前十，铌产量全球前二，钼钨国内龙头 |
| **竞争优势** | 刚果（金）TFM/KFM世界级铜钴矿，成本处全球最低25%分位 |
| **客户结构** | 覆盖全球电池厂商、特钢企业、贸易商，埃珂森贸易网络全球布局 |
| **供应商** | 自有矿山为主，原料自给率高 |

### 竞争格局

**主要竞争对手**：
- **嘉能可（Glencore）**：全球铜钴贸易龙头，全产业链布局
- **华友钴业**：国内钴冶炼龙头，钴镍锂一体化
- **紫金矿业**：全球铜金龙头，海外资源丰富

**公司优势**：
洛阳钼业在钴领域具备全球领先地位，刚果（金）配额制下供应稀缺性凸显。铜钴伴生矿开发成本优势明显，KFM二期、Lumina黄金项目等储备项目提供长期成长空间。

## 五、投资亮点

### 核心亮点

- ✅ **全球钴生产龙头**：2024年钴产量11.42万吨，全球第一，刚果（金）配额3.12万吨，供应稀缺
- ✅ **铜钴共振**：TFM、KFM世界级高品位铜钴矿，2026年规划铜产能80-100万吨
- ✅ **成本优势突出**：铜矿现金成本全球最低25%分位，毛利率持续提升
- ✅ **矿山+贸易双驱动**：埃珂森贸易业务创历史新高，平滑价格波动
- ✅ **产能扩张**：KFM二期投资10.84亿美元，2027年投产新增10万吨铜；Lumina黄金项目2029年前投产

### 成长驱动因素

1. **铜钴价格上行**
   - 2026年全球钴短缺约3.2万吨，刚果（金）配额制限制供给
   - 新能源车、AI算力驱动铜需求，2026年全球铜供需缺口约83万吨

2. **产能释放**
   - KFM二期2027年投产，新增年均10万吨铜
   - 2028年规划铜产能80-100万吨，钴9-10万吨

3. **第二增长极**
   - 收购加拿大Lumina黄金项目，2029年前投产，年产黄金11.5吨、铜1.9万吨

### 催化剂

- 铜价、钴价持续上涨
- KFM二期、Lumina项目进展超预期
- 刚果（金）配额政策利好龙头企业

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **地缘政治风险** | 刚果（金）政局不稳、政策变动（如权益金、出口配额调整）可能影响运营，基础设施薄弱增加成本 |
| ⚠️ **金属价格波动** | 铜、钴、钼等价格受全球经济、供需影响大，价格下跌将冲击盈利 |
| ⚠️ **项目进展不及预期** | KFM二期、Lumina等项目建设周期长，投产进度可能延迟 |
| ⚠️ **汇率风险** | 海外业务占比高，美元、刚果法郎等汇率波动影响业绩 |
| ⚠️ **环保与合规** | 矿山开采环保要求提升，刚果（金）合规成本可能上升 |
