## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 金堆城钼业股份有限公司 |
| 证券代码 | 601958 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2007年5月16日 |
| 上市时间 | 2008年4月17日 |
| 所属行业 | 有色金属-钼 |
| 市场地位 | 国内钼业龙头、全球钼行业重要供应商 |

**公司概况**：
金钼股份是国内钼业龙头和全球钼行业内具有较强影响力的纯钼专业供应商，钼生产经营规模位居全球前列。公司拥有世界六大原生钼矿床之一的陕西金堆城钼矿和河南汝阳东沟钼矿，权益内钼资源逾110万吨金属量，基本实现钼产业链全谱系供应，是A股首家钼产业上市公司。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **钼精矿** | 钼矿采选初级产品 | 钼冶炼原料 |
| **钼炉料** | 氧化钼、钼铁等 | 钢铁添加剂、合金冶炼 |
| **钼化工** | 钼酸铵、钼酸钠等 | 催化剂、颜料、润滑剂 |
| **钼金属** | 钼粉、钼板坯、钼丝等 | 高温炉件、电子、航空航天 |

### 业务模式

公司采用"矿山-冶炼-深加工-贸易"一体化模式，上游自有两大钼矿保障原料供应，中下游覆盖钼炉料、钼化工、钼金属全系列产品，实现钼产业链全谱系供应。产品以内销为主，兼顾出口。

### 技术路线/核心技术

- **钼矿采选**：金堆城、东沟两大世界级钼矿，品位与规模国内领先
- **钼冶炼**：氧化钼、钼铁等炉料产品产能充足
- **钼深加工**：钼化工、钼金属产品延伸产业链

**技术优势**：
资源禀赋突出，钼矿储量和品位全球领先，成本优势明显；产业链完整，产品结构多元，抗周期能力较强。

## 四、行业地位与产业链定位

### 产业链定位

```
钼产业链：
上游矿山 → 【金钼股份】→ 中游冶炼 → 下游深加工 → 终端应用
     ↑
 金堆城+东沟钼矿
 权益资源110万吨+
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内钼业龙头，钼生产经营规模全球前列 |
| **竞争优势** | 资源禀赋突出、产业链完整、成本优势明显 |
| **客户结构** | 钢铁、化工、高端制造等下游客户 |
| **供应商** | 自有钼矿，原料自给率高 |

### 竞争格局

**主要竞争对手**：
- **洛阳钼业**：全球钼、铜、钴多金属龙头，钼产量全球领先
- **紫金矿业**：伴生钼矿，规模较大
- **西部矿业**：有色金属综合企业，含钼业务

**公司优势**：
纯钼专业供应商，资源储量国内领先，产业链全谱系布局，盈利能力强，分红率高（近年平均约75%）。

## 五、投资亮点

### 核心亮点

- ✅ **资源龙头**：权益钼资源逾110万吨，金堆城、东沟为世界级钼矿
- ✅ **纯钼专业**：聚焦钼产业，产业链全谱系，产品结构多元
- ✅ **高盈利高分红**：毛利率37%+、净利率22%+，分红率约75%，股息率约3.64%
- ✅ **供需格局向好**：2024-2026年全球钼供需缺口预计持续扩大（2.88→3.64→4.43万吨）
- ✅ **战略属性**：2025年钼列入出口管制，战略金属定价提升

### 成长驱动因素

1. **钼价中枢上移**
   - 全球钼供需缺口扩大，供给刚性约束
   - 新增矿山有限，品位下降制约产量

2. **需求结构优化**
   - 高端含钼不锈钢、优特钢、高速工具钢需求增长
   - 新能源、航空航天等领域钼应用拓展

3. **政策红利**
   - 出口管制强化国内外价差
   - 战略金属地位提升

### 催化剂

- 钼价持续上涨
- 供需缺口超预期
- 出口管制政策落地

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **钼价波动风险** | 钼价大幅波动直接影响业绩，2024年钼价回落导致净利润略降 |
| ⚠️ **需求下滑风险** | 钢铁、制造业景气度下行影响钼消费 |
| ⚠️ **现金流波动** | 2025Q1经营现金流大幅为负，需关注季节性及营运资金管理 |
| ⚠️ **矿山品位下降** | 长期开采可能导致品位下降，影响成本 |
| ⚠️ **宏观与政策风险** | 地缘政治、环保政策等可能影响供需与价格 |
