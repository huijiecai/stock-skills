## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 盛新锂能集团股份有限公司 |
| 证券代码 | 002240 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001年12月29日 |
| 上市时间 | 2008年5月23日 |
| 所属行业 | 有色金属-锂矿采选、锂盐生产 |
| 市场地位 | 国内锂盐重要企业，第三梯队潜力企业，资源一体化布局 |

**公司概况**：
盛新锂能主要从事锂矿采选、基础锂盐和金属锂产品的生产与销售，另有少量林木业务。公司在全球范围内布局锂资源，拥有四川、津巴布韦、印尼、阿根廷等多地锂矿和盐湖项目，锂盐产能约13.7万吨/年，产能规模位居全球前列。2024年锂精矿产量28.6万吨，同比+65%。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **碳酸锂** | 电池级碳酸锂 | 动力电池、储能电池 |
| **氢氧化锂** | 电池级氢氧化锂 | 高镍三元动力电池 |
| **氯化锂** | 工业级氯化锂 | 石化、制药 |
| **金属锂** | 金属锂 | 锂电池负极、合金 |
| **锂精矿** | 锂辉石精矿 | 锂盐生产原料 |

### 业务模式

公司采用"锂矿开采+锂盐冶炼"一体化模式，通过自有矿山和参股矿山保障原料供应，在国内四川、印尼建设锂盐生产基地，产品直销下游电池厂和材料厂。盈利模式以锂盐销售为主，受锂价波动影响大；资源自给率提升可增强成本优势和抗周期能力。

### 技术路线/核心技术

- **锂辉石提锂**：四川致远、遂宁盛新等基地，锂辉石-硫酸法/碱法提锂
- **盐湖提锂**：阿根廷SDLA盐湖项目独立运营权
- **金属锂**：盛威锂业500吨金属锂产能，规划扩建至2500吨
- **海外布局**：印尼盛拓6万吨锂盐项目（建设中）

**技术优势**：
公司锂盐产能规模全球前列，四川基地成熟运营，印尼项目推进海外布局。木绒锂矿持有52.20%股权，生产规模300万吨/年，2024年10月取得采矿许可证，资源保障能力持续提升。

## 四、行业地位与产业链定位

### 产业链定位

```
锂电产业链：
锂矿开采 → 锂盐冶炼 → 【盛新锂能】→ 正极材料 → 动力电池 → 新能源汽车
                ↑
        锂矿+锂盐一体化
        四川+印尼+津巴布韦+阿根廷
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 锂盐产能13.7万吨/年，位居全球前列；国内第二梯队，低于赣锋、天齐等龙头 |
| **竞争优势** | 全球多资源布局（四川、津巴布韦、印尼、阿根廷），木绒锂矿、萨比星等优质资源 |
| **客户结构** | 覆盖主流动力电池、储能客户 |
| **供应商** | 自有矿山+参股矿山，资源自给率持续提升 |

### 竞争格局

**主要竞争对手**：
- **赣锋锂业**：全球锂业龙头，产业链最完整
- **天齐锂业**：锂矿资源龙头，格林布什等优质矿
- **华友钴业**：镍钴锂一体化，锂盐产能扩张
- **雅化集团、永兴材料**：国内锂盐重要企业

**公司优势**：
盛新锂能资源布局广泛，木绒锂矿（52.20%股权）、津巴布韦萨比星（29万吨锂精矿产能）、印尼6万吨锂盐项目、阿根廷SDLA盐湖，形成多资源保障。锂价底部加大资源投入，逆周期布局，锂价回升时弹性大。

## 五、投资亮点

### 核心亮点

- ✅ **资源一体化布局**：木绒锂矿（300万吨/年规模，2024年10月取得采矿证）、萨比星29万吨锂精矿、印尼6万吨锂盐、阿根廷SDLA盐湖，资源保障能力强

- ✅ **产能规模领先**：锂盐产能13.7万吨/年（四川7.7万吨+印尼6万吨），金属锂500吨，规划扩建至2500吨，产能规模全球前列

- ✅ **锂精矿产量高增长**：2024年锂精矿产量28.6万吨同比+65%，资源端放量

- ✅ **印尼项目推进**：印尼盛拓6万吨锂盐项目产能爬坡，海外布局开始显现

- ✅ **逆周期布局**：锂价底部加大资源开发，木绒锂矿、印尼项目推进，锂价回升时业绩弹性大

### 成长驱动因素

1. **锂价企稳回升**
   - 锂价经历大幅下跌后，供给出清、需求复苏或推动价格企稳
   - 公司高成本产能或逐步出清，行业格局优化

2. **资源项目投产**
   - 木绒锂矿规模化生产
   - 印尼6万吨锂盐产能释放
   - 萨比星锂精矿达产

3. **成本下降**
   - 资源自给率提升降低原料成本
   - 产能利用率提升摊薄固定成本

### 催化剂

- 锂价触底反弹
- 木绒锂矿规模化产出
- 印尼锂盐项目投产
- 新能源汽车销量超预期

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **锂价持续低迷风险** | 锂盐价格处于低位，若继续下跌将加剧亏损，高成本产能面临出清压力 |
| ⚠️ **财务压力** | 有息负债88.41亿元同比+22.19%，货币资金/流动负债仅41%，现金流承压 |
| ⚠️ **汇兑损失风险** | 美元负债及海外项目导致汇兑波动，2025H1财务费用+72.58% |
| ⚠️ **减值风险** | 锂价下跌可能导致存货、在建工程等资产减值 |
| ⚠️ **海外经营风险** | 印尼、津巴布韦、阿根廷项目存在地缘政治、政策变动风险 |
| ⚠️ **产能过剩风险** | 行业产能扩张过快，锂盐供给过剩压制价格 |

|------|----------|------------|------------|
| **乐观** | 锂价显著反弹，木绒+印尼放量 | 扭亏为盈 | 归母净利10亿+ |
| **中性** | 锂价企稳，产能逐步释放 | 亏损收窄或小幅盈利 | 归母净利5-10亿 |
