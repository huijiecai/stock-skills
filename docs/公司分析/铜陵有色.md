## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 铜陵有色金属集团股份有限公司 |
| 证券代码 | 000630 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1992年6月 |
| 上市时间 | 1996年11月20日 |
| 所属行业 | 有色金属-铜冶炼加工 |
| 市场地位 | 国内主要阴极铜生产企业，内资最大铜箔生产企业 |

**公司概况**：
铜陵有色是国内首家铜业上市公司，集铜采选、冶炼、加工、贸易为一体的大型全产业链铜生产企业。2024年阴极铜产量177万吨，占全国14.32%，是国内最大铜箔生产企业之一，拥有从铜采选到加工贸易的完整产业链优势。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **阴极铜** | 2024年产量177万吨，占全国14.32% | 电力电缆、新能源、建筑、电子 |
| **硫酸** | 2024年产量536万吨 | 化工、化肥、冶金 |
| **黄金** | 2024年产量22.91吨 | 珠宝首饰、金融投资、电子元件 |
| **铜箔** | 铜冠铜箔（72.38%）已创业板上市，在建锂电池铜箔产能2.5万吨/年 | 锂电池、PCB、电子 |
| **铜加工材** | 产能约50.8万吨，预计未来总产能57.6万吨 | 铜材加工、下游制造 |

### 业务模式

公司采用"采选冶加"全产业链经营模式，通过自有矿山（米拉多铜矿等）开采、铜冶炼加工及贸易获取收入。盈利模式以冶炼加工为主，米拉多铜矿投产后资源自给率提升，享受金属价格上涨带来的利润弹性。铜箔等高附加值产品扩张为公司提供成长空间。

### 技术路线/核心技术

- **冶炼技术**：阴极铜产能188万吨（含在建），配套硫酸产能449.3万吨，技术水平居行业前列
- **铜箔技术**：国内最大铜箔生产企业之一，在建锂电池铜箔产能2.5万吨/年，预计未来铜箔产能达8万吨/年
- **海外矿山运营**：米拉多铜矿（厄瓜多尔）持股70%，2025年二期投产后铜矿石处理能力达4620万吨/年

**技术优势**：
公司拥有完整的产业链优势，在铜矿采选、冶炼及铜箔加工等领域具有深厚的技术积累。2024年研发投入46.80亿元，同比增长12.38%。

## 四、行业地位与产业链定位

### 产业链定位

```
有色金属产业链：
矿产资源勘探 → 矿产开采采选 → 冶炼加工 → 【铜陵有色】→ 铜加工/贸易 → 终端应用
                                        ↑
                              采选+冶炼+加工全产业链
                              米拉多铜矿+铜冠铜箔
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 阴极铜产量占全国14.32%，预计位列行业第二；内资最大铜箔生产企业 |
| **竞争优势** | 全产业链优势、冶炼规模领先、铜箔高附加值布局 |
| **客户结构** | 面向金属贸易商、下游铜加工企业、新能源电池企业 |
| **供应商** | 米拉多铜矿投产后自给率提升，外购铜精矿为主 |

### 竞争格局

**主要竞争对手**：
- **江西铜业**：铜冶炼龙头，产业链布局相似
- **云南铜业**：中铝系铜冶炼企业，阴极铜产能国内第三
- **紫金矿业**：矿产端龙头，产业链环节不同

**公司优势**：
铜陵有色拥有"采选冶加"全产业链，米拉多铜矿二期投产后资源自给率显著提升；铜箔业务为高附加值延伸，锂电池铜箔产能扩张带来成长空间。冶炼端成本管控能力突出。

## 五、投资亮点

### 核心亮点

- ✅ **米拉多铜矿二期投产**：2025年二期主体全面完工，铜矿石处理能力达4620万吨/年，自产铜产能跃升至31万吨，资源自给率从15%提升至25%，毛利率有望从7.65%提升至10%+

- ✅ **全产业链优势**：从铜采选到冶炼加工、铜箔的完整产业链，抗单一环节波动能力较强

- ✅ **铜箔高附加值扩张**：铜冠铜箔已创业板上市，在建锂电池铜箔产能2.5万吨/年，预计未来铜箔产能达8万吨/年

- ✅ **全球铜供需缺口**：2025年全球铜资源供需缺口预计达84.8万吨，新能源需求爆发背景下公司具备成长空间

- ✅ **冶炼规模领先**：阴极铜产能188万吨（含在建），产量占全国14.32%，行业地位稳固

### 成长驱动因素

1. **资源自给率提升**
   - 米拉多铜矿二期投产后自产铜产能大幅提升
   - 2025年1-7月铜精矿含铜产量完成计划123.56%，创投产以来同期历史新高

2. **铜箔产能释放**
   - 锂电池铜箔需求增长
   - 铜冠铜箔产能扩张

3. **铜价上行**
   - 全球铜供需缺口支撑价格
   - 新能源、电网投资拉动需求

### 催化剂

- 米拉多铜矿二期正式投产并达产
- 铜价维持高位或继续上涨
- 铜箔产能释放、订单饱满
- 加工费（TC/RC）企稳回升

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **冶炼加工费波动** | 铜精矿TC/RC大幅下降将压缩冶炼利润，2024年Q4部分企业因加工费下降利润承压 |
| ⚠️ **应收账款与现金流** | 应收账款同比激增121.71%，经营现金流同比降74.1%，货币资金/流动负债仅38.44%，流动性压力需关注 |
| ⚠️ **资产减值** | 2025年Q3计提资产减值4.5亿元，导致单季净利润大幅下滑 |
| ⚠️ **海外项目风险** | 米拉多铜矿位于厄瓜多尔，地缘政治、环保、社区关系等存在不确定性 |
| ⚠️ **金属价格波动** | 铜价下跌将影响矿山端利润；硫酸价格波动影响冶炼副产品收入 |

*数据来源：财中社、网易财经、公司公告，2024-2025年*
