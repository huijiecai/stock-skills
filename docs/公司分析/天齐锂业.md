## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 天齐锂业股份有限公司 |
| 证券代码 | 002466 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1995年10月16日 |
| 上市时间 | 2010年8月31日 |
| 所属行业 | 有色金属-锂 |
| 市场地位 | 全球锂业前三、国内锂业龙头 |

**公司概况**：
天齐锂业是以锂为核心的新能源材料企业，总部位于四川省射洪市，深耕锂行业超30年。公司控股全球最优质的格林布什锂辉石矿，持有SQM（全球最大盐湖提锂企业）22.16%股权，已实现锂资源100%自给，是全球锂业巨头之一。2024年受锂价大幅下跌影响亏损，2025年锂价企稳后盈利显著改善。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **碳酸锂** | 电池级碳酸锂 | 动力电池、储能电池 |
| **氢氧化锂** | 电池级氢氧化锂 | 高镍三元动力电池 |
| **锂精矿** | 格林布什矿产出 | 锂盐生产原料 |
| **锂化合物及衍生品** | 多种锂盐产品 | 新能源、新材料 |

### 业务模式

公司采用"锂矿开采-锂精矿-锂盐加工"垂直一体化运营模式，从锂矿到锂盐覆盖锂价值链关键环节。格林布什矿供应锂精矿，自有锂盐基地加工为碳酸锂、氢氧化锂等产品，销往全球锂电池产业链客户。同时通过参股SQM、扎布耶盐湖获取盐湖锂资源权益。

### 技术路线/核心技术

- **硬岩锂矿**：格林布什锂辉石矿，氧化锂品位1.9%，全球储量最大、品位最高的在产硬岩锂矿
- **盐湖锂资源**：参股SQM（智利阿塔卡马盐湖）、日喀则扎布耶盐湖（20%）
- **锂盐加工**：碳酸锂、氢氧化锂生产线，张家港3万吨氢氧化锂项目2025年10月达产
- **定价机制**：2024年起锂精矿定价从季度改为按月调整，更贴近现货价格

**技术优势**：
锂资源100%自给，权益锂资源量约1777万吨LCE，锂矿及锂盐产品成本处于行业领先位置。格林布什矿占全球在产锂资源项目供给总量约18%。

## 四、行业地位与产业链定位

### 产业链定位

```
锂产业链：
锂矿开采 → 锂精矿 → 【天齐锂业】→ 锂盐（碳酸锂/氢氧化锂）→ 锂电池 → 新能源车/储能
                ↑
        格林布什矿+SQM+扎布耶
        锂资源100%自给
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球锂业前三，格林布什矿占全球在产锂资源供给约18%，锂盐产能8.88万吨/年 |
| **竞争优势** | 全球顶级锂矿资源（格林布什），参股SQM获取盐湖锂权益，资源100%自给，成本行业领先 |
| **客户结构** | 覆盖全球锂电池产业链头部客户，为二十多个国家和地区提供产品 |
| **供应商** | 锂精矿自供，无上游依赖 |

### 竞争格局

**主要竞争对手**：
- 赣锋锂业：锂电材料一体化，2026年产能10.6万吨LCE，Goulamina项目
- 雅保（Albemarle）：全球锂业龙头，盐湖+硬岩双布局
- SQM：全球最大盐湖提锂企业，天齐持股22.16%
- 盛新锂能：锂盐加工产能扩张快

**公司优势**：
格林布什矿是全球储量最大、品位最高的在产硬岩锂矿，氧化锂品位1.9%，储量1.72亿吨；同时拥有SQM和扎布耶盐湖权益，硬岩+盐湖双布局，资源禀赋行业顶尖。

## 五、投资亮点

### 核心亮点

- ✅ **全球顶级锂矿资源**：控股格林布什锂辉石矿，全球储量最大、品位最高，占全球供给约18%
- ✅ **锂资源100%自给**：权益锂资源量约1777万吨LCE，无上游依赖，成本行业领先
- ✅ **SQM战略投资**：持股22.16%，全球最大盐湖提锂企业第二大股东，分享盐湖锂红利
- ✅ **盈利拐点已现**：2025年扭亏为盈，Q3净利润同比增119%，锂价企稳驱动业绩改善
- ✅ **产能持续扩张**：格林布什总规划产能214万吨/年，张家港3万吨氢氧化锂2025年10月达产

### 成长驱动因素

1. **锂价企稳回升**
   - 2026年全球锂供需略有过剩，2027年预计短缺14.4万吨
   - Q3碳酸锂均价环比+12%，锂精矿环比+14%
   - 新能源车、储能需求长期增长

2. **定价机制优化**
   - 锂精矿定价从季度改为按月，减少与锂盐价格错配
   - 泰利森化学级锂精矿定价周期缩短

3. **产能释放**
   - 格林布什三期52万吨产能在建，建成后总产能超210万吨/年
   - 张家港3万吨+奎纳纳2.4万吨氢氧化锂项目，中期产能超14万吨/年

### 催化剂

- 锂价持续反弹
- 格林布什三期投产
- 张家港、奎纳纳氢氧化锂项目达产
- 2027年全球锂供需短缺预期

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **锂价波动风险** | 锂价大幅下跌将直接冲击盈利，2024年亏损79亿即因锂价暴跌 |
| ⚠️ **需求不及预期** | 新能源汽车、储能增速放缓可能导致锂价下跌 |
| ⚠️ **成本错配** | 锂矿定价周期与锂盐价格错配，库存精矿成本可能拖累业绩 |
| ⚠️ **项目进展风险** | 格林布什三期、奎纳纳等项目投产进度不及预期影响增长 |
| ⚠️ **债务压力** | 有息资产负债率22.22%，需关注债务状况 |
| ⚠️ **汇率波动** | 澳元升值带来汇兑收益，反之则产生汇兑损失 |

## 七、业绩预期与估值

### 业绩预期

| 场景 | 假设条件 | 2025年预期 | 2026年预期 |
|------|----------|------------|------------|
| **乐观** | 锂价持续反弹至10万/吨以上 | 净利润10亿+ | 净利润20亿+ |
| **中性** | 锂价震荡7-9万/吨 | 净利润5-10亿 | 净利润10-15亿 |
| **悲观** | 锂价再次下跌 | 净利润<5亿或亏损 | 业绩承压 |

*机构预测参考：2025年EPS均值0.34-0.53元，净利润3.94-17.91亿元；2025-2027年EPS分别为0.46、0.87、1.22元*

### 估值水平

| 指标 | 当前水平 | 行业平均 | 评估 |
|------|----------|----------|------|
| PE(TTM) | 负值（亏损后恢复中） | - | 盈利恢复后PE有望转正 |
| PB | 1.90-2.92倍 | - | 行业中等水平 |
| PS | 8.33倍 | - | 营收规模支撑 |

*数据来源：东方财富、理杏仁，2025年*

**估值分析**：
公司当前处于盈利恢复期，2024年亏损导致PE为负。随着2025年扭亏为盈，机构预测全年净利润5-18亿元，对应PE区间较大。PB约2倍处于行业中等水平。若锂价持续反弹、产能释放，2026-2027年业绩有望显著改善，估值中枢15-25倍PE较为合理。
