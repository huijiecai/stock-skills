## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广东风华高新科技股份有限公司 |
| 证券代码 | 000636 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1984年 |
| 上市时间 | 1996年 |
| 所属行业 | 电子元器件（被动元件） |
| 市场地位 | 国内MLCC绝对龙头 |

**公司概况**：
风华高科是国内领先的被动电子元器件制造商，专注于MLCC（多层陶瓷电容器）、片式电阻器等产品的研发、生产和销售。公司拥有完整的产品链和大规模生产能力，是国内最大的MLCC生产企业，致力于成为全球一流的电子信息基础产品整合配套供应商。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **MLCC（多层陶瓷电容器）** | 公司核心产品，覆盖常规、高压、车规级等多个系列，月产能300-400亿颗 | 消费电子、汽车电子、AI服务器、通信设备 |
| **片式电阻器** | 包括厚膜、薄膜、热敏、压敏等多种类型，产品线齐全 | 家电、通信、工业控制 |
| **电感产品** | 贴片电感、功率电感等 | 消费电子、汽车电子 |
| **电子材料** | 陶瓷粉体、浆料等关键材料 | 自用及对外销售 |
| **其他被动元件** | 压敏电阻、磁珠等 | 多领域应用 |

### 业务模式

公司采用"研发+制造+销售"一体化模式，拥有从材料到成品的完整产业链。产品主要面向B端客户，通过直销和经销商渠道销售。公司持续推进高端化转型，重点布局车规级、AI服务器等高附加值市场。

### 技术路线/核心技术

- **MLCC技术**：掌握高容量、高压、高频等关键技术，车规级产品通过AEC-Q200认证
- **材料技术**：实现陶瓷粉体等关键材料自主供应，降低对日韩供应链依赖
- **工艺技术**：拥有784项专利，掌握薄层化、高层数等先进工艺
- **智能制造**：推进数字化、自动化生产，提升良率和效率

**技术优势**：
公司是国内少数具备高端MLCC规模化生产能力的企业，车规级产品已完成AEC-Q200认证并实现批量交付，技术壁垒较高。在高温高容MLCC材料领域取得突破，关键材料自主供应能力持续提升。

## 四、行业地位与产业链定位

### 产业链定位

```
电子元器件产业链：
陶瓷粉体/浆料 → MLCC/电阻/电感制造 → 【风华高科】→ 电子终端厂商 → 终端应用
                    ↑
              国内MLCC龙头
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 全球MLCC市占率约5%，国内市场排名第一，国产替代领军企业 |
| **竞争优势** | 国内产品线最全、规模最大的MLCC企业，拥有完整产业链和材料自主供应能力 |
| **客户结构** | 已导入国内头部AI服务器、机器人客户，汽车电子客户覆盖主流车企及Tier1供应商 |
| **供应商** | 关键材料自主供应比例提升，降低对日韩供应链依赖 |

### 竞争格局

**主要竞争对手**：
- **村田（日本）**：全球MLCC龙头，市占率32%，技术领先，主攻高端市场
- **三星电机（韩国）**：全球第二，市占率19%，规模优势明显
- **国巨（台湾）**：全球第三，市占率约10.5%，通过收购补强技术
- **三环集团（国内）**：国内第二大MLCC企业，电子陶瓷一体化龙头

**公司优势**：
- 国内市场规模最大、产品线最全的MLCC企业
- 车规级MLCC已通过AEC-Q200认证，技术壁垒较高
- 关键材料自主供应，成本控制能力强
- 受益国产替代，承接日韩厂商外溢订单

**竞争劣势**：
- 高端产品技术与日韩龙头仍有差距
- 全球市占率仅5%，规模优势不足
- 单位成本高于国际大厂

## 五、投资亮点

### 核心亮点

- ✅ **MLCC超级周期受益者**：行业进入供需失衡的超级周期，高端MLCC订单量达产能2倍，供需缺口显著，公司深度受益
- ✅ **国产替代核心标的**：MLCC国产化率预计2026年突破30%，公司作为国内龙头将承接大量替代订单，市占率有望从5%提升至10%+
- ✅ **AI+车规双轮驱动**：AI服务器MLCC用量是传统服务器8倍，已导入头部AI服务器客户；车规级产品获AEC-Q200认证，汽车电子营收增长66%
- ✅ **涨价周期开启**：2025年11月发布涨价函，产品价格上调5%-30%，行业低价竞争周期结束，进入需求驱动新阶段
- ✅ **材料自主可控**：关键陶瓷粉体等材料实现自主供应，降低供应链风险，成本控制能力提升

### 成长驱动因素

1. **AI服务器需求爆发**
   - 单台AI服务器MLCC用量达3万颗，是传统服务器的8倍
   - 已成功导入国内头部AI服务器及机器人客户
   - AI服务器专用MLCC年复合增长率预计从18%上调至30%

2. **汽车电子高速增长**
   - 新能源汽车MLCC用量是传统车5-10倍
   - 车规级产品已获AEC-Q200认证并批量交付
   - 汽车电子营收同比增长66%，占比持续提升

3. **国产替代加速**
   - 中美科技博弈背景下，国产替代迫切性增强
   - 日韩厂商聚焦高端，中高端订单向国内倾斜
   - 公司产能扩张+技术突破，承接替代订单能力增强

### 催化剂

- MLCC价格持续上涨，涨价幅度超预期
- AI服务器客户订单落地，高端产品占比提升
- 车规级产品在主流车企渗透率提升
- 国产替代政策支持加码

## 六、风险提示

| 风险类型 | 具体内容 |
|----------|----------|
| ⚠️ **原材料成本风险** | 银、锡、铜等金属价格上涨推高成本，虽然公司已涨价传导，但若涨价幅度不足或频率滞后，将压缩毛利率 |
| ⚠️ **行业竞争加剧** | 国内三环集团等竞争对手加速扩张，国际厂商可能调整策略，价格战风险仍存 |
