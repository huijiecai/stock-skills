## 一、公司简介

### 基本信息
- **公司名称**：中芯国际集成电路制造有限公司
- **股票代码**：688981.SH / 00981.HK
- **所属行业**：半导体制造 / 晶圆代工
- **上市时间**：科创板2020年，港股2004年
- **总部位置**：上海
- **员工人数**：约2万人

### 公司定位
中芯国际是中国大陆最大的晶圆代工企业，全球第五大晶圆代工厂，是国内半导体制造的核心企业，承担着国产芯片制造的重任。
