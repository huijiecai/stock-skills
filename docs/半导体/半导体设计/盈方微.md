## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 盈方微电子股份有限公司 |
| 证券代码 | 000670 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1996年（前身） |
| 上市时间 | 1996年12月17日 |
| 所属行业 | 元器件（电子元器件分销+芯片设计） |
| 市场地位 | 电子元器件分销商、影像SoC芯片设计商 |

**公司概况**：
盈方微是一家以电子元器件分销和集成电路芯片设计销售为双主营业务的上市公司。公司通过控股子公司华信科及WorldStyle开展电子元器件分销业务，同时自研影像类SoC芯片应用于智能家居、视频监控等领域。公司目前正在推进重大资产重组，拟收购上海肖克利和富士德中国100%股权。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **电子元器件分销** | 东芝、罗姆、村田等品牌授权分销 | 消费电子、工业控制、通信设备（主要收入来源） |
| **影像类SoC芯片** | 智能处理器、主控处理器、向量处理器 | 智能家居、视频监控、教育机器人 |
| **射频芯片** | 射频前端芯片 | 无线通信设备 |
| **存储芯片** | 内存接口芯片 | 存储设备 |
| **电源芯片** | 电源管理芯片 | 各类电子设备 |

### 业务模式

公司采用"分销+设计"双轮驱动模式：
- **分销业务**：通过华信科及WorldStyle代理东芝、罗姆、村田等国际品牌电子元器件，提供供应链服务
- **芯片设计业务**：自主研发SoC芯片，主打影像处理和智能交互场景

### 技术路线/核心技术

- **RISC-V处理器内核**：自研智能处理器、主控处理器、向量处理器等多系列RISC-V内核
- **影像信号处理（ISP）**：核心技术研发，应用于智能视频场景
- **智能视频算法**：面向边端智能交互与信号处理场景

**技术优势**：
公司坚持自主开发研发理念，在SoC系统设计、图像信号处理等领域积累了一定的技术能力。芯片产品线覆盖消费级和工业级应用场景。

## 四、行业地位与产业链定位

### 产业链定位

```
半导体产业链：
上游材料/设备 → 芯片设计 → 【盈方微】 → 晶圆代工 → 封装测试 → 终端应用
                              ↓
                        电子元器件分销（主要业务）
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 电子元器件分销领域为中小型分销商，芯片设计领域市场份额较小 |
| **竞争优势** | 代理东芝、罗姆、村田等国际品牌，分销渠道稳定 |
| **客户结构** | 消费电子、工业控制、通信设备等领域客户 |
| **供应商** | 东芝、罗姆、村田等国际半导体品牌 |

### 竞争格局

**主要竞争对手（电子元器件分销）**：
- **深圳华强（000062）**：国内电子元器件分销龙头
- **力源信息（300184）**：电子元器件分销商
- **韦尔股份（603501）**：芯片设计+分销

**主要竞争对手（芯片设计）**：
- **全志科技（300458）**：智能应用处理器芯片设计
- **瑞芯微（603893）**：智能应用处理器芯片设计

**公司劣势**：
- 芯片设计业务规模较小，市场竞争力有限
- 分销业务毛利率较低，盈利能力弱
- 持续亏损，财务状况不佳

## 五、投资亮点

### 核心亮点

- ✅ **重大资产重组预期**：正在推进收购上海肖克利和富士德中国100%股权，有望改善业务结构
- ✅ **国产替代概念**：持续推进芯片工艺向国产替代制程迈进，契合国家半导体自主化战略
- ✅ **RISC-V技术储备**：自研多系列RISC-V处理器内核，具备技术积累
- ✅ **经营现金流为正**：尽管亏损，但经营现金流保持正值

### 成长驱动因素

1. **重大资产重组**
   - 拟收购上海肖克利（专业电子元器件分销商）和富士德中国（半导体设备分销）
   - 重组后有望改善盈利能力和业务结构

2. **国产替代趋势**
   - 特朗普加征关税政策可能利好国产替代芯片企业
   - 公司芯片设计业务有望受益

3. **并购重组政策支持**
   - 2025年5月证监会修订《上市公司重大资产重组管理办法》，支持上市公司向新质生产力方向转型升级

### 催化剂

- 重大资产重组方案获得监管批准并完成交割
- 芯片设计业务获得大客户订单
- 国产替代政策加码
