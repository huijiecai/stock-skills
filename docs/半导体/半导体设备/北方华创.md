## 二、主营业务

### 1. 核心产品

| 产品线 | 说明 | 应用领域 |
|--------|------|----------|
| **刻蚀设备** | CCP刻蚀、ICP刻蚀 | 晶圆制造先进制程 |
| **薄膜沉积设备** | PVD、CVD、ALD | 逻辑芯片、存储芯片 |
| **热处理设备** | 立式炉、卧式炉 | 退火、扩散工艺 |
| **清洗设备** | 单片清洗、槽式清洗 | 晶圆清洗 |
| **电子元器件** | 真空组件、电源 | 半导体设备零部件 |

### 2. 业务模式

- **平台化布局**：刻蚀+薄膜+热处理+清洗全品类覆盖
- **国产替代核心**：28nm以上工艺设备成熟量产
- **客户覆盖**：中芯国际、长江存储、长鑫存储等国内主流晶圆厂
