## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 天水华天科技股份有限公司 |
| 证券代码 | 002185 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2003年 |
| 上市时间 | 2007年11月 |
| 所属行业 | 半导体封装测试 |
| 市场地位 | 封测三巨头之一，存储芯片封装领先 |

**公司概况**：
华天科技是国内领先的集成电路封测企业，2024年营收144.62亿元（+28%），净利润6.16亿元（+172.29%）。公司在存储芯片封装、AI算力芯片封装领域具有优势，2.5D封装产线已建设完成，Chiplet技术进入量产阶段。

---

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **先进封装** | 2.5D/3D封装、FOPLP、Chiplet | AI芯片、高性能计算 |
| **存储芯片封装** | DDR5 DRAM封装、Flash封装 | 数据中心、消费电子 |
| **传统封装** | 引线键合、QFN、BGA等 | 消费电子、汽车电子 |
| **晶圆级封装** | WLP、Fan-out等 | 移动设备、物联网 |

### 业务模式

公司采用专业封测代工模式，客户涵盖国内外芯片设计公司。存储芯片封装业务占比约25%，在DDR5 DRAM封装领域处于国内领先地位。先进封装业务占比持续提升。

### 技术路线/核心技术

- **DDR5封装**：已量产，支持高速存储需求
- **2.5D封装**：产线建设完成，良品率98.5%
- **Chiplet技术**：已进入量产阶段，支持多芯片集成

**技术优势**：存储芯片封装国内领先；2.5D封装良率高；成本控制能力强。

## 四、行业地位与产业链定位

### 产业链定位

```
半导体封装产业链：
晶圆制造 → 【华天科技】→ 芯片成品 → 终端应用
(TSMC等)   (封装测试)   (存储/AI芯片) (数据中心)
```

### 市场地位

| 维度 | 说明 |
|------|------|
| **市场份额** | 国内封测企业第三，全球排名第六 |
| **竞争优势** | 存储芯片封装领先，成本控制能力强 |
| **客户结构** | 国内外芯片设计公司，存储芯片客户占比较高 |
| **供应商** | 设备来自国际主流厂商 |

### 竞争格局

**主要竞争对手**：长电科技（国内龙头）、通富微电（国内最大）、日月光（全球龙头）

**公司优势**：存储芯片封装国内领先；2.5D封装良率高；成本控制能力强。

## 五、投资亮点

### 核心亮点

- ✅ **业绩高速增长**：2025年净利润增速超50%
- ✅ **存储芯片封装领先**：DDR5封装量产，受益数据中心需求
- ✅ **2.5D封装量产**：产线建设完成，良率98.5%
- ✅ **Chiplet技术突破**：已进入量产阶段
- ✅ **成本控制能力强**：毛利率持续改善

### 成长驱动因素

1. **存储芯片需求增长**：数据中心对DDR5需求旺盛
2. **2.5D封装放量**：AI芯片封装需求增长
3. **国产替代**：先进封装国产化空间大

### 催化剂

- DDR5封装订单增长
- 2.5D封装新客户拓展
- AI芯片封装订单
- 国产存储芯片放量
