## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海宝信软件股份有限公司 |
| 证券代码 | 600845 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2001 年 |
| 上市时间 | 2001 年 4 月 |
| 所属行业 | AI制造与工业软件 |
| 市场地位 | 钢铁信息化龙头，AI智能制造 |

**公司概况**：
宝信软件是国内钢铁信息化龙头，主营软件开发、系统集成、IDC 服务。公司布局 AI智能制造和工业互联网平台，2024 年 AI制造业务快速推进。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **软件开发** | 钢铁 MES/ERP 系统 | 钢铁企业 |
| **AI智能制造** | AI生产优化系统 | 智能制造 |
| **系统集成** | 自动化整体方案 | 工程项目 |
| **IDC 服务** | 数据中心运营 | 云计算服务 |

### 业务模式

- **软件收入**：软件开发 + 授权费
- **AI服务**：AI优化系统收费
- **集成收入**：系统集成项目收入
- **IDC 收入**：机柜租赁 + 服务费
