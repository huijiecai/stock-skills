## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中公教育科技股份有限公司 |
| 证券代码 | 002607 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1999 年 |
| 上市时间 | 2011 年 1 月 |
| 所属行业 | AI教育与职业培训 |
| 市场地位 | 职业教育龙头，AI智慧教学 |

**公司概况**：
中公教育是国内职业教育龙头，主营公务员招录培训、职业资格培训。公司布局 AI智慧教学和 AI面试模拟，2024 年 AI教育业务稳步推进。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **公务员培训** | 行测 + 申论培训 | 公考考生 |
| **AI智慧教学** | AI个性化学习系统 | 在线学习 |
| **AI面试模拟** | AI面试对练系统 | 面试培训 |
| **职业资格培训** | 教师、医考等 | 职业考试 |

### 业务模式

- **面授培训**：线下培训班收费
- **在线课程**：网课订阅费
- **AI服务**：AI系统使用费
- **图书销售**：教材销售收入
