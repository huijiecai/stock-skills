## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 世纪天鸿教育科技股份有限公司 |
| 证券代码 | 300654 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2004 年 |
| 上市时间 | 2017 年 9 月 |
| 所属行业 | AI教育与数字阅读 |
| 市场地位 | 教辅图书龙头，AI智慧教育 |

**公司概况**：
世纪天鸿是国内教辅图书龙头，主营 K12 教辅图书、数字教育服务。公司布局 AI智慧教育和 AI阅读产品，2024 年 AI教育业务稳步推进。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **教辅图书** | 初中、高中教辅 | K12学生 |
| **AI智慧教育** | AI学习系统 | 学校 + 家庭 |
| **AI阅读产品** | AI阅读辅助工具 | 语文学习 |
| **数字教育** | 在线教育资源 | 数字化学习 |

### 业务模式

- **图书销售**：教材教辅销售收入
- **AI服务**：AI学习系统订阅费
- **数字服务**：在线教育资源收费
- **ToB服务**：学校采购
