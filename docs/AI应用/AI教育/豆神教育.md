## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 豆神教育科技（北京）股份有限公司 |
| 证券代码 | 300010 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2009 年 |
| 上市时间 | 2010 年 1 月 |
| 所属行业 | AI教育与智慧课堂 |
| 市场地位 | 语文教育龙头，AI智慧课堂 |

**公司概况**：
豆神教育是国内语文教育龙头，主营大语文培训、智慧教育产品。公司布局 AI智慧课堂和 AI作文批改，2024 年 AI教育业务快速推进。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **大语文培训** | 线下 + 线上语文课程 | K12学生 |
| **AI智慧课堂** | AI教学系统 | 学校课堂 |
| **AI作文批改** | AI智能批改系统 | 作业辅导 |
| **教育硬件** | 学习机、词典笔 | 家庭教育 |

### 业务模式

- **培训课程**：课时费（线上 + 线下）
- **ToB服务**：学校采购智慧课堂系统
- **AI服务**：AI批改订阅费
- **硬件销售**：教育硬件销售收入
