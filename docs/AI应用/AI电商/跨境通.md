## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 跨境通宝电子商务股份有限公司 |
| 证券代码 | 002640 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2003 年 |
| 上市时间 | 2011 年 9 月 |
| 所属行业 | AI电商与跨境出口 |
| 市场地位 | 跨境电商龙头，AI智能运营 |

**公司概况**：
跨境通是国内跨境电商龙头，主营跨境出口电商、跨境进口电商。公司布局 AI智能运营和 AI供应链，2024 年 AI电商业务快速推进。
