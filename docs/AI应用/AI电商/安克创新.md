| 项目 | 内容 |
|------|------|
| 公司全称 | 安克创新科技股份有限公司 |
| 证券代码 | 300866 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2011 年 |
| 上市时间 | 2020 年 8 月 |
| 所属行业 | AI电商与消费电子 |
| 市场地位 | 跨境电商品牌龙头，AI智能营销 |

**公司概况**：
安克创新是国内跨境电商品牌龙头，主营充电类、音频类消费电子产品的研发和销售。公司布局 AI智能营销和 AI产品设计，2024 年 AI电商业务快速推进。
