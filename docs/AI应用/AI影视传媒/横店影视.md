## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 横店影视股份有限公司 |
| 证券代码 | 603103 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2008 年 |
| 上市时间 | 2017 年 9 月 |
| 所属行业 | AI影视与影院 |
| 市场地位 | 院线第二，AI智慧影院 |

**公司概况**：
横店影视是国内院线第二，主营电影放映、影视制作。公司布局 AI智慧影院和 AI影视拍摄基地，2024 年 AI影视业务稳步推进。
