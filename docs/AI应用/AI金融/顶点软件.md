## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 福建顶点软件股份有限公司 |
| 证券代码 | 603383 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1994 年 |
| 上市时间 | 2017 年 5 月 |
| 所属行业 | 证券 IT与 AI应用 |
| 市场地位 | 证券 IT领先企业，AI交易平台 |

**公司概况**：
顶点软件是国内证券 IT 领先企业，主营证券交易系统、AI 交易平台。公司在机构交易业务系统领域市占率领先，2024 年 AI 业务稳步推进。
