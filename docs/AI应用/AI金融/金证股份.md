## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 深圳市金证科技股份有限公司 |
| 证券代码 | 600446 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1998 年 |
| 上市时间 | 2003 年 12 月 |
| 所属行业 | 金融 IT与 AI应用 |
| 市场地位 | 金融 IT 综合服务商，AI 证券系统 |

**公司概况**：
金证股份是国内金融 IT 综合服务商，主营证券、基金、银行 IT 系统。公司布局 AI 证券系统，2024 年业务稳步推进，是恒生电子主要竞争对手。
