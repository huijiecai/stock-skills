## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 东方财富信息股份有限公司 |
| 证券代码 | 300059 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2005 年 |
| 上市时间 | 2010 年 3 月 |
| 所属行业 | 互联网金融与 AI应用 |
| 市场地位 | 互联网金融龙头，"妙想"大模型领先 |

**公司概况**：
东方财富是国内互联网金融龙头，主营证券、基金销售、金融数据服务。公司推出"妙想"金融大模型，2024 年净利润 96 亿元，充分受益牛市 +AI双轮驱动。
