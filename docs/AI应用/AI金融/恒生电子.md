| 项目 | 内容 |
|------|------|
| 公司全称 | 恒生电子股份有限公司 |
| 证券代码 | 600570 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1995 年 |
| 上市时间 | 2003 年 12 月 |
| 所属行业 | 金融 IT与 AI应用 |
| 市场地位 | 金融 IT龙头，智能投顾领先 |

**公司概况**：
恒生电子是国内金融 IT龙头，主营证券、基金、银行 IT系统。公司布局智能投顾、量化交易系统，2024 年 AI业务稳步推进，机构客户粘性强。
