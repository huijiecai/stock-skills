## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 上海姚记科技股份有限公司 |
| 证券代码 | 002605 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1994 年 |
| 上市时间 | 2011 年 8 月 |
| 所属行业 | AI游戏与扑克牌 |
| 市场地位 | 扑克牌龙头，AI游戏转型 |

**公司概况**：
姚记科技是国内扑克牌龙头，主营扑克牌制造、移动游戏。公司布局 AI游戏研发和 AI营销，2024 年 AI游戏业务稳步推进。
