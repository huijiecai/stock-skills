## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 北京掌趣科技股份有限公司 |
| 证券代码 | 300315 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2004 年 |
| 上市时间 | 2012 年 5 月 |
| 所属行业 | AI游戏与元宇宙 |
| 市场地位 | 手游研发领先，AI+ 元宇宙布局 |

**公司概况**：
掌趣科技是国内手游研发领先企业，主营手游研发发行、元宇宙业务。公司布局 AI游戏研发和 AI+ 元宇宙，2024 年 AI业务稳步推进。
