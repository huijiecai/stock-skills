## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中国科技出版传媒股份有限公司 |
| 证券代码 | 601858 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2011 年 |
| 上市时间 | 2017 年 1 月 |
| 所属行业 | AI语料与出版 |
| 市场地位 | 学术出版龙头，AI语料库领先 |

**公司概况**：
中国科传是国内学术出版龙头，主营科技图书、期刊、数字出版。公司拥有丰富 AI 语料数据资源，2024 年 AI语料业务稳步推进。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **科技图书** | 学术专著、教材 | 科研、教育 |
| **期刊出版** | 学术期刊发行 | 学术交流 |
| **数字出版** | 数据库、知识服务 | 在线学习 |
| **AI语料** | 学术语料库授权 | 大模型训练 |

### 业务模式

- **图书销售**：纸质 + 电子书销售
- **期刊订阅**：机构 + 个人订阅
- **数据库服务**：知识服务平台订阅
- **语料授权**：AI训练数据授权费
