## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 人民网股份有限公司 |
| 证券代码 | 603000 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1997 年 |
| 上市时间 | 2012 年 4 月 |
| 所属行业 | AI内容与媒体 |
| 市场地位 | 官方媒体龙头，AI内容审核 |

**公司概况**：
人民网是国内官方媒体龙头，主营新闻信息服务、内容科技、舆情服务。公司布局 AI内容审核和语料库业务，2024 年 AI业务稳步推进。

|----------|------|----------|
| **新闻服务** | 新闻报道、传播 | 新闻资讯 |

| **舆情服务** | 网络舆情监测 | 政府、企业 |
