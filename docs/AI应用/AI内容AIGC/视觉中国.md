## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 视觉中国集团股份有限公司 |
| 证券代码 | 000681 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2000 年 |
| 上市时间 | 2000 年 12 月 |
| 所属行业 | AI内容与版权服务 |
| 市场地位 | 版权素材龙头，AIGC 合规服务 |

**公司概况**：
视觉中国是国内版权素材龙头，主营图片、视频、音乐版权授权。公司提供 AIGC 合规服务，2024 年 AI训练数据业务稳步推进。

|----------|------|----------|
