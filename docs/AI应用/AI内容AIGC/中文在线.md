## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 中文在线集团股份有限公司 |
| 证券代码 | 300364 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2000 年 |
| 上市时间 | 2015 年 1 月 |
| 所属行业 | AI内容与数字出版 |
| 市场地位 | 数字内容龙头，AI短剧领先 |

**公司概况**：
中文在线是国内数字内容龙头，主营数字阅读、IP 运营、AI短剧。公司布局 AI短剧生产和数字人技术，2024 年 AIGC 业务快速推进。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **数字阅读** | 网络文学平台 | 17K 小说网等 |
| **AI短剧** | AIGC 短剧生产 | 短视频平台 |
| **IP 运营** | 版权运营、改编 | 影视、游戏 |
| **数字人** | 虚拟数字人技术 | 直播、营销 |

### 业务模式

- **阅读付费**：用户订阅 + 单章购买
- **IP 授权**：版权改编授权费
- **AI短剧**：短剧制作 + 分账
- **数字人**：技术服务费
