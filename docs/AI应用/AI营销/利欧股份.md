|------|------|

| 证券代码 | 002131 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 2001 年 |
| 上市时间 | 2007 年 4 月 |
| 所属行业 | AI营销与数字营销 |
| 市场地位 | 数字营销龙头，AI精准投放 |

**公司概况**：
利欧股份是国内数字营销龙头，主营 AI精准营销、机械制造双主业。公司在数字营销领域市占率领先，2024 年 AI营销业务稳步推进。
