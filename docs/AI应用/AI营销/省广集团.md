## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 广东省广告集团股份有限公司 |
| 证券代码 | 002400 |
| 上市板块 | 深圳证券交易所（主板） |
| 成立时间 | 1981 年 |
| 上市时间 | 2010 年 5 月 |
| 所属行业 | AI营销与整合营销 |
| 市场地位 | 国有营销龙头，AI整合营销 |

**公司概况**：
省广集团是国内国有营销龙头，主营整合营销、数字营销、媒介代理。公司布局 AI营销平台，2024 年 AI业务稳步推进，国企背景资源优势明显。
