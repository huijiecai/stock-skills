| 项目 | 内容 |

| 证券代码 | 301171 |
| 上市板块 | 深圳证券交易所（创业板） |
| 成立时间 | 2011 年 |
| 上市时间 | 2022 年 8 月 |
| 所属行业 | AI营销与出海广告 |
| 市场地位 | 海外 AI营销龙头，跨境电商广告领先 |

**公司概况**：
易点天下是国内海外 AI营销龙头，主营跨境电商广告投放、AI精准营销。公司在海外媒体广告领域市占率领先，2024 年 AI营销业务高增长。
