## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 三人行传媒集团股份有限公司 |
| 证券代码 | 605168 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 2003 年 |
| 上市时间 | 2020 年 5 月 |
| 所属行业 | AI营销与整合营销 |
| 市场地位 | 整合营销服务商，AI创意平台 |

**公司概况**：
三人行是国内整合营销服务商，主营数字营销、媒介代理、活动管理。公司布局 AI创意平台，2024 年 AI营销业务稳步推进，客户资源优质。
