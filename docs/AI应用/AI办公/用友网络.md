## 一、公司简介

| 项目 | 内容 |
|------|------|
| 公司全称 | 用友网络科技股份有限公司 |
| 证券代码 | 600588 |
| 上市板块 | 上海证券交易所（主板） |
| 成立时间 | 1988 年 |
| 上市时间 | 1995 年 6 月 |
| 所属行业 | 企业软件与 AI应用 |
| 市场地位 | 企业云服务龙头，YonBIP AI智能体领先 |

**公司概况**：
用友网络是国内企业云服务龙头，主营 ERP 软件、云服务、企业金融。公司推出 YonBIP AI 智能体服务，加速企业智能化转型。2024 年云业务收入占比持续提升，AI 商业化稳步推进。

## 二、主营业务

### 核心产品/服务

| 产品类型 | 说明 | 应用领域 |
|----------|------|----------|
| **YonBIP** | 商业创新平台 | 大型企业数字化 |
| **用友 cloud** | 云 ERP 服务 | 中小企业上云 |
| **AI 智能体** | 企业 AI助手 | 财务、人力、供应链 |
| **企业金融** | 供应链金融 | 中小企业融资 |

- **订阅制收费**：云服务按年订阅
- **软件授权**：传统软件许可 + 维护费
- **金融服务**：利息收入和服务费
