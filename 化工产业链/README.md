# 🧪 化工产业链全景图

> 更新日期：2026年1月22日

---

## 📋 内容说明

> 本文档覆盖 **基础化工**、**精细化工** 与 **化工新材料**。化工行业是国民经济的基础，被称为"工业之母"，其投资逻辑通常围绕 **周期性**、**成本优势** 与 **国产替代（新材料）** 展开。

### 🔑 关键概念：化工行业的分类

| 分类 | 包含内容 | 投资特点 |
|------|----------|----------|
| **基础化工（大宗）** | 石化、煤化、氯碱、化肥 | 强周期、看产能周期与原料价格 |
| **精细化工** | 农药、维生素、添加剂、助剂 | 弱周期、高附加值、看技术壁垒 |
| **化工新材料** | MDI、氟硅材料、电子化学品、特种工程塑料 | 成长性强、国产替代空间大 |

## 🚨 投资观察维度

| 维度 | 说明 | 现状 (2026) |
|------|------|------------|
| **成本为王** | 能拿到低价煤、石油或拥有矿产资源的厂商具有天然优势 | 🔴 资源价格高波动 |
| **产能周期** | 行业扩产到投产的时间差决定了价格波动（猪周期类似） | 🟡 部分领域进入去产能尾声 |
| **国产替代** | 高端材料（如高端聚烯烃、电子化学品）依然依赖进口 | 🟢 替代进程加速 |
| **双碳约束** | 高能耗项目审批严，存量资产价值提升 | 🔴 准入门槛极高 |
