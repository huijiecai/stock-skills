## 🧱 怎么读这套笔记？（统一结构）

每一篇子笔记，尽量都遵循同一套框架，方便快速形成“产业链心智图”：

1.  **它变成了什么？**——先用通俗应用场景把产品“具象化”（衣服、肥料、电池、空调等）。
2.  **产业链怎么走？**——从**上游原料 → 中游中间体 → 下游应用**画清楚路线。
3.  **格局是谁说了算？**——看全球/国内 CR4/CR5、寡头还是分散、小厂还能不能活。
4.  **A 股谁最关键？**——列出核心标的，并用一行话写清楚“为什么是它”。
5.  **投资抓什么变量？**——对应每个子行业，总结 2–3 个最关键的跟踪指标（价差、开工率、矿价等）。

带着上面 5 个问题去看，每个模块都是在回答同一件事：**这条化工链条里，钱到底是怎么赚的？谁在拿大头？**
