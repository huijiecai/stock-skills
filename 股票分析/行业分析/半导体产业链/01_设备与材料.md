## 一、半导体设备

### 1️⃣ 是什么？

> **半导体设备 = 造芯片的"机器"**
>
> 就像做菜需要锅碗瓢盆，造芯片需要各种专业设备。**设备决定了你能造出多先进的芯片。**

| 设备类型 | 作用 | 通俗比喻 |
|----------|------|----------|
| **光刻机** | 把电路图案"印"到硅片上 | 印刷机印报纸 |
| **刻蚀机** | 把不需要的材料"刻掉" | 雕刻刀刻章 |
| **薄膜设备** | 在硅片上镀各种薄膜 | 给蛋糕涂奶油 |
| **离子注入** | 掺入杂质改变性质 | 炒菜加调料 |
| **CMP设备** | 把表面磨平 | 打磨地板 |
| **检测设备** | 检查有没有缺陷 | 质检员 |
| **清洗设备** | 清洗硅片 | 洗碗机 |

## 二、光刻机详解（最核心设备）

### 1️⃣ 是什么？

> **光刻机 = 芯片制造的"照相机"**
>
> 用光把电路图案"曝光"到硅片上，决定芯片能做多小。

```
芯片制造的核心步骤：
1. 在硅片上涂光刻胶
2. 用光刻机把电路图案"曝光"到光刻胶上  ← 这一步决定精度！
3. 显影、刻蚀、沉积...
4. 重复上百次
```

### 2️⃣ 市场规模

```
光刻机市场规模（2025年）：

市场规模：约 250亿美元
EUV光刻机：约 100亿美元（增速最快）
DUV光刻机：约 150亿美元

单台价格：
EUV光刻机：1.5-3.5亿美元/台
ArF浸没式：1亿美元/台
DUV干式：2000-5000万美元/台
```

### 3️⃣ 竞争格局

| 类型 | 波长 | 能做到的制程 | 价格 | 供应商 | ASML市占率 |
|------|------|--------------|------|--------|------------|
| i-line | 365nm | >180nm | 便宜 | 多家 | 低 |
| KrF | 248nm | 180-90nm | 中等 | Nikon/Canon/ASML | ~50% |
| ArF干式 | 193nm | 90-45nm | 较贵 | Nikon/Canon/ASML | ~60% |
| **ArF浸没** | 193nm | 45-7nm | ~1亿美元 | ASML为主 | **80%+** |
| **EUV** | 13.5nm | 7nm以下 | ~1.5亿美元 | **仅ASML** | **100%** |
| **High-NA EUV** | 13.5nm | 2nm以下 | ~3.5亿美元 | **仅ASML** | **100%** |

```
⚠️ ASML是全球唯一的EUV光刻机供应商！
没有EUV，就无法高效量产7nm以下先进制程芯片。
```

### 4️⃣ 国产替代

| 指标 | 国际水平 | 国产水平 | 差距 |
|------|----------|----------|------|
| 最先进制程 | EUV → 3nm | DUV → 28nm | **10年+** |
| 国产龙头 | ASML | 上海微电子 | - |
| 国产进展 | - | 28nm验证中 | 高端完全空白 |

**EUV光刻机为什么难造？**

| 难点 | 说明 |
|------|------|
| 光源 | 用激光打锡滴产生EUV光，效率极低 |
| 反射镜 | 需要纳米级精度的多层反射镜 |
| 真空环境 | EUV会被空气吸收，必须在真空中工作 |
| 供应链 | 需要全球顶尖零部件商配合 |
| 专利壁垒 | 几十年技术积累，专利墙极高 |

## 三、半导体材料

### 1️⃣ 是什么？

> **半导体材料 = 造芯片的"原料"**
>
> 就像做菜需要食材，造芯片需要各种高纯度材料。**纯度差一点点，芯片可能就废了。**

| 材料类型 | 作用 | 纯度要求 | 通俗比喻 |
|----------|------|----------|----------|
| **硅晶圆** | 芯片的"基底" | 99.9999999%（9个9）| 画布 |
| **光刻胶** | 光刻时的"感光材料" | 极高 | 感光相纸 |
| **特种气体** | 各工艺需要的气体 | 极高 | 调料 |
| **CMP材料** | 抛光用的浆料和垫片 | 高 | 抛光膏 |
| **靶材** | 薄膜溅射用 | 高 | 涂料 |
| **湿电子化学品** | 清洗和刻蚀用 | 极高 | 清洁剂 |

## 四、洁净室（无尘室）

### 1️⃣ 是什么？

> **洁净室 = 芯片制造的"手术室"**
>
> 一粒灰尘就能毁掉一颗芯片，所以必须在极度洁净的环境中生产。

```
芯片线宽对比：

头发丝直径：      ~80,000nm
灰尘颗粒：        ~1,000-10,000nm
7nm芯片线宽：     7nm
3nm芯片线宽：     3nm

结论：一粒灰尘 = 几千倍的芯片线宽 → 必须无尘环境
```

### 2️⃣ 市场规模


## 五、投资逻辑总结

### 核心逻辑：国产替代

```
美国制裁 → 国内晶圆厂被迫采购国产设备/材料 → 国产厂商订单暴增
```

### 关注要点

| 要点 | 说明 |
|------|------|
| ✅ 订单增速 | 国产设备厂的订单是否持续增长 |
| ✅ 验证进度 | 是否进入主流晶圆厂供应链 |
| ✅ 技术突破 | 高端产品是否有进展 |
| ⚠️ 估值 | 国产替代预期是否已充分反映 |
| ⚠️ 盈利能力 | 研发投入大，何时能盈利 |

### 投资优先级

```
设备 > 材料 > 洁净室

原因：
• 设备单价高、空间大
• 材料细分领域多、竞争分散
• 洁净室国产化率已较高
```
