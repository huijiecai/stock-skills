## 核心设计原则

### 1. 后端自闭环
数据采集作为后端服务的内置能力，不依赖外部脚本。

### 2. 定时调度 + 查询触发
- **定时调度**：APScheduler 集成在 FastAPI 中，服务启动即自动调度
- **查询触发**：lazy loading，查询时若数据不存在则触发采集

### 3. 数据源优先级
- **adata**：主力，当日分时、日线、指数、概念、资金流向
- **Tushare**：补充，涨跌停数据（连板高度）、历史分钟数据

### 4. API 场景化
- **日线级别 API**：盘前分析、前端展示（按 date 查询）
- **分钟级别 API**：模拟看盘快照（按 time + date 查询）

## 快速开始

### 1. 启动 PostgreSQL

```bash
cd backend
docker-compose up -d
```

### 2. 初始化数据库

```bash
python scripts/pg_init.py
```

### 3. 启动后端服务

```bash
uvicorn app.main:app --reload
```

服务启动后：
- API 文档：http://localhost:8000/docs
- 定时任务自动开始调度
