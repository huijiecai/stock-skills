# Web 前端设计

## 页面概览

| 页面 | 路径 | 核心内容 | 对应 API |
|------|------|---------|---------|
| 首页 | `/` | 三大指数 + 市场情绪 + 热榜 | 指数 API + 统计 API |
| 个股排行 | `/stock-ranking` | 涨跌幅/成交额/换手率排行 | `GET /api/market/stock-ranking` |
| 板块排行 | `/concept` | 板块排行 + 板块内个股 | 概念 API |
| 连板天梯 | `/continuous-board` | 连板分布 + 炸板情况 | `GET /api/market/continuous-board` |
| 涨跌停 | `/limit-list` | 涨停/跌停/炸板股列表 | 涨跌停 API |
| 个股详情 | `/stock/:code` | 分时 + 日线 + 资金流向 | 股票 API |

### 排序切换

| Tab | API 参数 |
|-----|---------|
| 涨幅榜 | `sort=change_pct&order=desc` |
| 跌幅榜 | `sort=change_pct&order=asc` |
| 成交额 | `sort=amount&order=desc` |
| 换手率 | `sort=turnover_rate&order=desc` |
| 大单流入 | `sort=main_inflow&order=desc` |

### 筛选功能

| 筛选器 | API 参数 |
|--------|---------|
| 板块筛选 | `concept_code=BK0612` |
| 日期选择 | `date=2026-03-27` |

### 交互

| 操作 | 效果 |
|------|------|
| 点击排序 Tab | 切换排序方式，重新请求 API |
| 选择板块 | 筛选该板块内个股 |
| 点击个股行 | 跳转个股详情页 |

## 组件结构

```
src/
├── components/
│   ├── DateSelector/       # 日期选择器（复盘历史）
│   ├── MarketSentiment/    # 市场情绪组件
│   ├── IndexCard/          # 指数卡片
│   ├── ChartPanel/         # 图表面板（分时/日K 切换，指数/板块/个股通用）
│   ├── StockTable/         # 个股表格（排行页使用）
│   ├── ConceptTable/       # 板块表格（可展开，含图表面板）
│   ├── ContinuousBoard/    # 连板天梯组件
│   ├── LimitListTable/     # 涨跌停表格（涨停/跌停/炸板）
│   ├── StockDetail/        # 个股详情页
│   └── CapitalFlowChart/   # 资金流向图
├── pages/
│   ├── Home/               # 首页
│   ├── StockRanking/       # 个股排行页
│   ├── ConceptRanking/     # 板块排行页
│   ├── ContinuousBoard/    # 连板天梯页
│   ├── LimitList/          # 涨跌停页
│   └── StockDetail/        # 个股详情页
└── services/
    └── api.ts              # API 封装
```
